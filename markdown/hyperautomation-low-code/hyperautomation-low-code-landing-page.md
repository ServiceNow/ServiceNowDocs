# Hyperautomation and low-code

Use ServiceNow no-code, low-code, and platform developer tools to create and modify apps that run on the ServiceNow AI Platform.

Hyperautomation incorporates multiple automation tools and solutions to create comprehensive automation initiatives. Such tools can include AI, machine learning, Robotic Process Automation, and so on. The ServiceNow AI Platform® App Engine provides a single mobile and web application development platform to build business applications and to extend existing applications that power your digital transformation. Workflow Data Fabric brings automation, low-code, and AI efficiency to your workflows, all on the same platform. Workflow Data Fabric runs on the ServiceNow AI Platform®, a single system of action for the enterprise.
