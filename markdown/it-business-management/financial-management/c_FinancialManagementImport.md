# Financial data import

Financial data resides in the General Ledger Staged Data table.

You can import financial data into the General Ledger Staged Data \[itfm\_gl\_data\_staged\] table by using either of the following methods:

-   Import sets
-   Easy import

See [Import sets](https://www.servicenow.com/docs/access?context=import-sets-landing-page&version=xanadu&pubname=xanadu-integrate-applications&ft:locale=en-US).

**Important:** Be sure that your fiscal calendar can be used with your financial data.

-   **[General ledger](c_GeneralLedger.md)**  
The general ledger contains all expenses for your organization for all fiscal periods.
-   **[View expenses in the general ledger](../task/t_View_ExpensesInTheGeneralLedger.md)**  
You can view records in any of the general ledger tables and make changes if necessary.
-   **[Financial data sources and field maps](data-sources.md)**  
Financial data sources point to the actual raw expense data table used by financial modeling and financial planning.
-   **[Create a financial data source](../task/create-data-sources.md)**  
Database columns referring to raw expense details must be identified to create a new data source in Financial Management.
-   **[Create a data source field map](../task/create-data-source-field-map.md)**  
Field maps create a reference between the data source columns and transaction columns.
