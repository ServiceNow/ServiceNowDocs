# Financial data import

Financial data resides in the General Ledger Staged Data table.

You can import financial data into the General Ledger Staged Data \[itfm\_gl\_data\_staged\] table by using either of the following methods:

-   Import sets
-   Easy import

See [Import sets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/integrate-applications/system-import-sets/import-sets-landing-page.md).

**Important:** Be sure that your fiscal calendar can be used with your financial data.

-   **[General ledger](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-business-management/financial-management/c_GeneralLedger.md)**  
The general ledger contains all expenses for your organization for all fiscal periods.
-   **[View expenses in the general ledger](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-business-management/financial-management/t_View_ExpensesInTheGeneralLedger.md)**  
You can view records in any of the general ledger tables and make changes if necessary.
-   **[Financial data sources and field maps](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-business-management/financial-management/data-sources.md)**  
Financial data sources point to the actual raw expense data table used by financial modeling and financial planning.
-   **[Create a financial data source](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-business-management/financial-management/create-data-sources.md)**  
Database columns referring to raw expense details must be identified to create a new data source in Financial Management.
-   **[Create a data source field map](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-business-management/financial-management/create-data-source-field-map.md)**  
Field maps create a reference between the data source columns and transaction columns.
