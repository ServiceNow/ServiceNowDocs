# Create a service type

Create service types, which are used by IT shared services.

## Before you begin

Role required: cost\_transparency\_admin

## Procedure

1.  Open one of the [IT shared service](t_CreateSharedService.md) records.

2.  Click the lookup icon in the Service type fields.

    The IT Shared Service Types list appears.

3.  Click **New**.

4.  Enter a name.

5.  Click **Submit**.
