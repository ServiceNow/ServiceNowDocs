# Delete a resource plan

Deleting a plan also deletes all associated resource allocations.

## Before you begin

Role required: resource\_user or resource\_manager

## Procedure

1.  Navigate to **All** &gt; **Resource** &gt; **Resource Plans** &gt; **All**.

2.  Open a resource plan and select **Delete** on the form header.

    A confirmation message appears.

    \[Omitted image "DeleteConfirmationMessage.png"\] Alt text: screenshot for Delete confirmation message

3.  Select **Delete**.

    The resource plan and all the associated allocations are removed.
