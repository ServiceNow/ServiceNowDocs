# Save a checklist as a template

After you create a checklist on a project task, you can save it as a template and reuse it on other tasks.

## Before you begin

Role required: it\_project\_manager

## Procedure

1.  Navigate to a project task.

2.  Click the down arrow next to **Checklist**.

3.  Select **Save as Template**.

4.  Enter a name for the checklist.

5.  Click **Save**.
