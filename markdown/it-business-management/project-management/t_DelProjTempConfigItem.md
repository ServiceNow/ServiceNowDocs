# Delete a project template configuration item

Do not delete any of the default template configuration items: project, project task, or project subtask. Doing so affects the ability to create project templates.

## Before you begin

Role required: it\_pps\_admin

## Procedure

1.  Navigate to **All** &gt; **Project Administration** &gt; **Settings** &gt; **Template Config**.

2.  Select a configuration item from the list.

3.  Click **Delete**.
