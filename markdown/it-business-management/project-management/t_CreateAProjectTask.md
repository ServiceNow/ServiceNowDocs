# Project tasks

Create project tasks in several ways, even from other applications in the instance.

## Before you begin

Role required: it\_project\_manager

## About this task

You can create a project task using any of the following options.

## Procedure

-   Use the Project Tasks related list.

    This related list is available from both a project record and a project task record. You can use the [**New** UI action](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-management/t_CreateATaskFromAProject.md) or [insert a new row in the related list](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-management/t_InsertARowIntoTheProjectTasksList.md).

-   Use the [planning console](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-management/c_TheProjectPlanningConsole.md).

-   Use [the project task creator](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-management/t_UseTheProjectTaskCreator.md).

    Use a dialog box to create multiple tasks at one time. The project task creator is available from both a project record and a project task record.

-   By [copying an existing project or task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-management/t_CopyExistingTaskorProject.md).

    This option copies a selected task, or even a whole project, including all child tasks.

-   From [an incident, problem, or change request](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-management/t_CreateTaskIntPrblmChange.md).

    The Project Task related list is also available from the Incident, Problem, and Change Request forms. The tasks you create from these forms makes the project task a child of the incident, problem, or change, rather than a project.
