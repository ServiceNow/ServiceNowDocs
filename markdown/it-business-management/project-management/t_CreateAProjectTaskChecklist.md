# Use a project task checklist

You can create a checklist in a project task to track items that must be completed in the task.

## Before you begin

Role required: it\_project\_user, it\_project\_manager

## Procedure

1.  Navigate to a project task.

2.  To create a checklist for the project task, click **Create new** in the **Checklist** section and specify a name for the checklist.

3.  To remove a checklist from the project task, click the down arrow next to **Checklist** and select **Remove Checklist**.

4.  **Note:** This option appears when there is at least one checklist in the **Checklist** section.

    To save a checklist as a template, click the down arrow next to **Checklist** and select **Save as Template**.
