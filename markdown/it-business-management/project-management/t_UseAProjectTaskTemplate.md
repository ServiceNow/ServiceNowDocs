# Create a task from a project task template

You can save a project task as a template and reuse it when creating a task.

## Before you begin

Role required: it\_project\_manager

You must have a template from an existing project task that has the required field values. See [Templates](https://www.servicenow.com/docs/access?context=c_Templates&version=xanadu&pubname=xanadu-platform-administration&ft:locale=en-US) for more information on this platform feature.

## Procedure

1.  Navigate to **Project** &gt; **Tasks** &gt; **All**.

2.  Click **New**.

3.  If the templates do not appear at the bottom, click the ellipsis \(![Ellipsis icon](../../planning-and-policy/image/gannt_chart_icon.png)\) icon at the top of the form, and select **Toggle Template Bar**.

4.  Click the template link in the template bar at the bottom of the form.

    ![Template link](../image/template_link.png)

    The template is applied to the project.
