# Edit monetary benefit plan

Edit a planned monetary benefit plan to revise the benefits, dates, offset, and so on, to maintain the updated financial records as your work progresses.

## Before you begin

-   [Create monetary benefit plans for your projects in Project Workspace](fin-manage-monetary-benefit-plans-pws.md)
-   Role required: sn\_align\_ws.spw\_financial\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Project Workspace** and open a project.

2.  Select the **Monetary Benefit Plans** related list.

3.  Select the name of a required benefit plan.

4.  In the Details tab, edit the required fields and select **Save**.
