# Edit monetary benefit plan

Edit a planned monetary benefit plan to revise the benefits, dates, offset, and so on, to maintain the updated financial records as your work progresses.

## Before you begin

-   [Manage monetary benefit plans for your planning items in Portfolio Planning](fin-manage-monetary-benefit-plans-ppw.md)
-   Role required: sn\_align\_ws.spw\_financial\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Portfolio Planning Workspace** and select portfolio plan.

2.  From the Prioritization or Roadmap tab, select the planning item for which you want to edit a monetary benefit plan.

    The full details page of the planning item is displayed.

3.  Select the **Monetary Benefit Plans** related list.

4.  Select the name of a required benefit plan.

5.  In the Details tab, edit the required fields and select **Save**.
