# Capture actual monetary benefits

Capture the actual benefit from the planned benefits to measure revenue and calculate the profits.

## Before you begin

-   [Manage monetary benefit plans for your planning items in Portfolio Planning](fin-manage-monetary-benefit-plans-ppw.md)
-   Role required: sn\_align\_ws.spw\_financial\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Portfolio Planning Workspace** and select portfolio plan.

2.  From the list of portfolio plans, select a portfolio plan.

3.  From the Prioritization or Roadmap tab, select the planning item for which you want to capture actual monetary benefits.

    The full details page of the planning item is displayed.

4.  Select the **Monetary Benefit Plans** related list.

5.  Select name of the required benefit plan and select the **Monetary Benefit Plan Breakdowns** list to view the monetary benefit plans breakdown.

6.  Select the required **Entered benefit** field to view the benefit plan form.

7.  You can change the previously captured planned benefit from the **Entered benefit** field.

8.  Enter the actuals benefits resulted from the work execution in the **Actual benefit** field and select **Save**.
