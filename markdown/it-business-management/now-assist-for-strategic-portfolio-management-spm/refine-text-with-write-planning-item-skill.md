# Refine planning items content using write planning item skill

Improve record quality by enabling AI-assisted text refinement in the description fields of Product Idea, Demand, Epic, Project, Capability, Feature, and Story forms in Strategic Planning Workspace. Enable product managers and agile users to refine planning item content efficiently using the Elaborate and Shorten options and create planning items with sufficient information.

## Before you begin

Make sure that the Write planning item skill is active.

Role required: sn\_align\_core.apw\_user

## About this task

Reduce the time spent manually on improving descriptions for Strategic Planning work items such as Product idea, Demand, Epic, Project, Capability, Feature and Story. Within the description field of these work items, use the refine with now assist menu to shorten or elaborate the text.

The write planning item skill helps in improving clarity and completeness of your work item and reduces the rework due to missing or unclear planning item information.

The skill provides content recommendations using the description fields of the planning items. When refining content for a story, the write planning item skill takes into account the description and acceptance criteria fields.

## Procedure

1.  Navigate to **Workspaces** &gt; **Strategic Planning Workspace**.

2.  Open any planning item such as product idea, demand, epic, project, capability, feature, or story.

3.  From the description field, select the **Refine with Now assist** menu and then select **Shorten** or **Elaborate** based on your requirement.

    ![Now Assist generated feedback summary in the Now Assist for Strategic Portfolio Management (SPM).](../images/write-planning-items-skill-sp.png)

    From the description field, you can directly access the **Refine with Now Assist** menu by single-clicking the field.

4.  Select **Replace** to replace the content in the description field.

    ![AI-generated summary for planning items description using the write planning item skill.](../images/write-planning-item-skill-refined-text.png)

5.  From the Refined by Now Assist window, select **Refine** and then select **Shorten** or **Elaborate** to improve the generated text.

6.  When you're finished summarizing the content in planning items, you can replace the existing content with generated content or provide feedback, copy, or view information about it.
