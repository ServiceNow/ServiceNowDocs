# Assign resources to assessment tasks in Configurable Workspace

Once you planned how to migrate resources to the cloud, you can assign the resources to assessment tasks.

## Before you begin

Role required: sn\_cloud\_migration.operator

## About this task

Resources can belong to one assessment task only.

Since applications depend on the servers hosting them, you can assign only the hosting servers to assessment tasks.

## Procedure

1.  From a resource list, select a check box next to the resource you want to add to an assessment task.

    The **Add to assessment** button shows the number of selected resources.

2.  Select **Add to assessment**.

    The **Assign a resource to an assessment task** dialog box opens.

3.  Perform one of the following steps:

    -   To add the resource to an existing task, select the relevant task link in the **Number** column, and then select **Add to CAS**.
    -   To add the resource to a new task, select **Create assessment**.
    The system adds the selected resources to the assessment task and displays the assessment task on a new tab.
