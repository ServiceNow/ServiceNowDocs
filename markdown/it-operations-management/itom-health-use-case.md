# ITOM Health and CSDM tables

ITOM Health leverages and maintains CSDM tables, while various ServiceNow products enhance and contribute value to its operations.

## CSDM tables managed by ITOM Health

ITOM Health manages the following tables:

-   Mapped Application Service table \[cmdb\_ci\_service\_discovered\]
-   Configuration Item table \[ci\_\*\]
-   Dynamic CI Group table \[cmdb\_ci\_query\_based\_service\]
