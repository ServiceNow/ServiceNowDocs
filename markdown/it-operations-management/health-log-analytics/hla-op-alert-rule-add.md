# Define a Log Analytics alert rule in Health Log Analytics

Define a custom Log Analytics alert rule for log data that is not generating alerts automatically. A custom rule enables you to specify the metric, threshold, and alert properties directly.

## Before you begin

Role required: evt\_mgmt\_operator or evt\_mgmt\_admin

**Important:** From the Vancouver release onward, the **Operator Workspace** is deprecated and replaced with Service Operations Workspace. For the new procedure, see the corresponding topic in the Service Operations Workspace for ITOM documentation: [Define a Log Analytics alert rule in Health Log Analytics](hla-op-alert-rule-add-sow.md).
