# Software requirements list in Cloud Migration Assessment

The software requirements list shows the list of applications associated with the assessment tasks.

## Information available in the Software requirements list

|Columns \(from left to right\)|Description|
|------------------------------|-----------|
|Assessment Task|The link to the assessment task.|
|Software Product Name|The application name.|
|Software Publisher|The name of the company which produced the application.|
|Software Version|The application version.|
|BYOL Supported|The name of the Bring Your Own License \(BYOL\).|
|Open Source|The application is developed by Open Source.|
|Open Source Licensing Type|The licensing type of the application developed by Open Source.|
|Application|The application running on a server assigned to the assessment task in the following format: &lt;application name&gt;@&lt;server&gt;.|
