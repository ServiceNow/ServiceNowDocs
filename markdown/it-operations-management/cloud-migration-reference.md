# Cloud Migration Assessment reference

Reference topics provide additional information about mapping and fine-tuning application services using Cloud Migration Assessment lists and forms.

-   **[Components installed with Cloud Migration Assessment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/components-installed-cloud-migration.md)**  
Several types of components are installed with the activation of the Cloud Migration Assessment plugin, including tables, user roles, properties, and scheduled jobs.
-   **[Assessment dashboard in Configurable Workspace of Cloud Migration Assessment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/assessment-dashboard.md)**  
Use the ServiceNow® Cloud Migration Assessment application to plan, organize, and track the process of relocating your enterprise IT resources and workloads to cloud platforms. Analyze the IT resources using dashboards that visualize and summarize information on resources by different categories. Use the Assessment dashboard to review the statistics of the migration tasks and monitor the migration progress.
-   **[Assessment task form in Configurable Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/cloud-migration-assessment-task.md)**  
The form for creating or modifying a Cloud Migration Assessment assessment task displays detailed information about the assessment.
-   **[Server list in Cloud Migration Assessment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/cloud-migration-servers-list.md)**  
Cloud Migration Assessment presents information on server resources in several lists: All servers, Windows servers, and Linux servers.
-   **[Applications list in Cloud Migration Assessment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/cloud-migration-applications-list.md)**  
Cloud Migration Assessment presents information on application resources in the following lists: Applications and Discovered applications.
-   **[VMs on undiscovered servers list in Cloud Migration Assessment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/cloud-migration-vm-undiscovered-list.md)**  
Cloud Migration Assessment presents information on virtual machines hosted by undiscovered servers.
-   **[Software requirements list in Cloud Migration Assessment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/migration-software-requirements-list.md)**  
The software requirements list shows the list of applications associated with the assessment tasks.
