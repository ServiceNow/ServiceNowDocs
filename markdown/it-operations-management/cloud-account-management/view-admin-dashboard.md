# Viewing the admin dashboard

View account statuses and violations, actionable insights, and administrative controls on the Overview dashboard, along with configuration and request policies.

## Access the Overview dashboard

As an admin, you can access the Overview dashboard in either of two ways.

-   Navigate to **All** &gt; **Cloud Workspace** &gt; **Overview**
-   Navigate to **Workspaces** &gt; **Cloud Workspace** &gt; **Overview**

\[Omitted image "cam-dashboard-admin.png"\] Alt text: Admin dashboard view

The table column descriptions are as follows:

-   Account Severity: The account severity is determined by its highest violation.

-   Alias: The system displays the account alias when you configure a scan and execute the Account Alias Policy.

-   Cloud bill this month: Billing happens even if the app isn't set up.

    **Note:**

    If Cloud Cost Management Workspace 8.1 is installed, it automatically gets billing details for accounts with billing jobs configured.
