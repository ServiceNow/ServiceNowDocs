# Create assessment tasks for cloud migration in Configurable Workspace

Create assessment tasks and associate them with migration waves for migration management.

## Before you begin

Role required: sn\_cloud\_migration.operator.

## Procedure

1.  Navigate to **All** &gt; **** &gt; **Cloud Migration** &gt; **Cloud Migration Workspace**.

2.  Select **Assessment dashboard** \[Omitted image "cm-assessment-dashboard-navigation.png"\] Alt text: Assessment dashboard navigation icon, and then select **Create assessment**.

    Alternatively, select **Lists** \[Omitted image "cm-lists-navigation.png"\] Alt text: Lists navigation icon, navigate to**Assessment tasks** &gt; **Tasks**, and select **New**.

3.  Fill in values on the [Create New Cloud Migration Assessment Task form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/cloud-migration-assessment-task.md).

4.  Add attachments.

    1.  Select **Browse** in the **Attachments** section.

    2.  Navigate to a file and select it.

    3.  Select **Upload**.

        The attached file appears under Attachments.

5.  Select **Save**.
