# Create assessment tasks for cloud migration in Configurable Workspace

Create assessment tasks and associate them with migration waves for migration management.

## Before you begin

Role required: sn\_cloud\_migration.operator.

## Procedure

1.  Navigate to **All** &gt; **** &gt; **Cloud Migration** &gt; **Cloud Migration Workspace**.

2.  Select **Assessment dashboard** ![Assessment dashboard navigation icon](../image/cm-assessment-dashboard-navigation.png), and then select **Create assessment**.

    Alternatively, select **Lists** ![Lists navigation icon](../image/cm-lists-navigation.png), navigate to**Assessment tasks** &gt; **Tasks**, and select **New**.

3.  Fill in values on the [Create New Cloud Migration Assessment Task form](../reference/cloud-migration-assessment-task.md).

4.  Add attachments.

    1.  Select **Browse** in the **Attachments** section.

    2.  Navigate to a file and select it.

    3.  Select **Upload**.

        The attached file appears under Attachments.

5.  Select **Save**.
