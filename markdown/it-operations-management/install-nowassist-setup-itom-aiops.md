# Install ITOM AIOps

Install the applications required for ITOM AIOps to enable intelligent issue detection, diagnosis, and automated resolution capabilities.

## Before you begin

Role required: admin

## About this task

**Note:** Applications are installed only if you have the license. If you do not have the required license for an application, it is not installed.

## Procedure

1.  Navigate to your Admin Home page on your instance.

2.  On the IT Operations Management tile, select **View product overview**.

3.  In the Welcome! Let’s get started section, select **Start setup**.

    ![IT Operations Management product overview page showing a first-time setup prompt with a Start setup button.](../image/nowassist-setup-start-setup.png)

4.  On the Installation progress page, select the install icon \(![Install icon.](../image/icon-now-assist-setup-download.png)\) ITOM AIOps.

    ![Installation progress page to install ITOM AIOps.](../image/now-assist-setup-download.png)

    **Note:** Expand the What's included section to view the applications installed with ITOM AIOps.
