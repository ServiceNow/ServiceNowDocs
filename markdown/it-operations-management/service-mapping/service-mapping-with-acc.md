# Service Mapping with Agent Client Collector

Agent Client Collector enables the discovery and mapping of application services without requiring you to set up credentials for the MID Server.

Using Agent Client Collector, you can deploy both top-down and machine learning-based Service Mapping and benefit from a simplified approach to mapping application services.

For information on configuring Service Mapping to work with Agent Client Collector, see [Configure Agent Client Collector for use with Service Mapping](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/service-mapping/configure-acc-for-service-mapping.md).
