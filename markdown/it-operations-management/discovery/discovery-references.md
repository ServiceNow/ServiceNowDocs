# Discovery reference

Reference topics provide additional information about Discovery lists and forms.

-   **[Command Validation Tasks form for pattern-based discovery](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/command-validation-reference.md)**  
**Command Validation Tasks** form displays the fields that you must fill when you create or modify a task for testing a pattern command.
-   **[Command List form for pattern-based discovery](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/commands-list-discovery.md)**  
The **Command List** form contains all commands used in discovery by infrastructure or application type patterns.
-   **[Discovery log details](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/discovery-log.md)**  
Discovery logs contain information on the horizontal discovery process based on probes and patterns.
-   **[Discovery Schedule form reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/discovery-schedule-form.md)**  
The Discovery Schedule form provides details about the configuration items \(CIs\) targeted for discovery, the associated MID Server, and the schedule settings for when and how the discovery runs.
-   **[Cloud Discovery service account form reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/cloud-operations-disco-service-account-form.md)**  
The Cloud Discovery service account form displays detailed information about the selected cloud provider. The Cloud Discovery service account form is available in the Cloud Discovery Workspace.
-   **[Create a new discovery schedule form reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/create-a-new-discovery-schedule-form.md)**  
The Create a new discovery schedule form displays information related to the schedule and the target cloud provider. The Create a new discovery schedule form is available in the Cloud Discovery Workspace.
-   **[Cloud Discovery reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/cloud-discovery-reference.md)**  
Reference topics provide additional information about the lists, forms, and cloud settings used in Cloud Discovery.
-   **[Discovery CLI commands](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/discovery-cli-commands.md)**  
A list of commands that are available to the Discovery Command Line Interface \(CLI\) in the Pattern Designer Enhancements application.
-   **[Discovery Admin Workspace reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/r_discovery-admin-workspace-reference.md)**  
Reference topics provide additional information about the lists, forms, and settings used in Discovery Admin Workspace.
-   **[Queue Registration form reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/r_QueueRegistrationForm.md)**  
The Queue Registration form displays the fields available when creating or modifying a queue.
-   **[Tracked Configuration file form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/discovery/tracked-configuration-file-form.md)**  
The Tracked Configuration file form displays details about a specific configuration file.
