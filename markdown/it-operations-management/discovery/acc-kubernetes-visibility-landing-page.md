# Agent Client Collector for Kubernetes – Visibility

Agent Client Collector for Kubernetes – Visibility detects changes on resources in a Kubernetes cluster. It performs continuous discovery, reports any changes back to your instance, and updates the Configuration Management Database \(CMDB\) with the latest data.
