# Discovery Admin Workspace Tuning

Tuning validates your existing configurations and helps you resolve configuration issues.

## Discovery Admin Workspace Tuning Home page

The Discovery Admin Workspace Tuning Home page provides tools to assess instance health through configured checks and displays failed checks by priority, category, and suite.

## Tuning check results

Use the tuning tools to interrogate your instance for configurations that indicate health issues and identify opportunities to address best practices. Tuning checks your existing configurations and helps you resolving issues. For more information on how to run checks, see [Run Tuning checks](../task/run-tuning-checks.md).

## Roles

The following roles have access to the Discovery Admin Workspace Tuning:

-   admin
-   discovery\_admin
