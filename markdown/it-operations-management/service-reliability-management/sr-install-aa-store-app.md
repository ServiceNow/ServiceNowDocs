# Install the Alert automation rules application

Install the **Alert automation rules** application and any supporting applications.

## Before you begin

Role required: admin

Subscriptions required: ITSM Professional or ITOM Operator Professional subscriptions.

## About this task

Prior to creating management rules, you should have installed and configured the Service Reliability Management application, available in the ServiceNow Store.

## Procedure

1.  Select the link provided to you in the email you received to access the ServiceNow Store and download the application onto your SRM instance.

2.  In your ServiceNow AI Platform instance, navigate to **All** &gt; **System Applications** &gt; **All Available Applications** &gt; **All**

3.  Locate the **Alert Automation Rules** application and select **Install** to activate it.

    Alert automation rules are ready to use within Service Reliability Management.
