# Add or remove multiple services from an integration in SRM

Add multiple services to an integration at once to monitor events from those services all in one place. You can also remove one or more services from the integration when you no longer need to track them.

## Before you begin

Role required: srm\_responder, srm\_manager, and srm\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  Select the integrations launchpad icon \(![Integrations launchpad icon](../../service-operations-workspace-itom/image/integrations-launchpad.png)\).

3.  Select an integration.

4.  Select the **Associated services** tab.

5.  Add or remove multiple services.

## What to do next

When you're adding services, make sure that the configurations are done correctly on the webhook so that the events that come in are connected to the correct service. For more information, see KB article \(add KB permalink!\).
