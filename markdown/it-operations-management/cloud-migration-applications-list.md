# Applications list in Cloud Migration Assessment

Cloud Migration Assessment presents information on application resources in the following lists: Applications and Discovered applications.

## Information available in the Applications and Discovered applications list

Columns \(from left to right\)

</th><th>

Description

</th></tr></thead><tbody><tr><td>

</td><td>

The label of the server that hosts the application.

</td></tr><tr><td>

Name

</td><td>

The application name.

</td></tr><tr><td>

Class

</td><td>

The application class.

</td></tr><tr><td>

Version

</td><td>

The application version.

</td></tr><tr><td>

Operational status

</td><td>

The application operational status: -   Operational
-   Non-operational
-   Repair in Progress
-   DR Standby
-   Ready
-   Retired

</td></tr><tr><td>

Name

</td><td>

The name of the server hosting the application.

</td></tr><tr><td>

Cluster

</td><td>

The name of the cluster in which the server resides. This field is available on the Applications list.

</td></tr><tr><td>

IT Environment

</td><td>

The current IT environment in which the application is deployed: On-premises or Cloud. This field is available on the Applications list.

</td></tr><tr><td>

Category

</td><td>

The category of the server hosting the application. For example, Physical Server.

</td></tr><tr><td>

State

</td><td>

The state of the server hosting the application: Assigned to an assessment task, Unassigned to an assessment task, or Ignored.
