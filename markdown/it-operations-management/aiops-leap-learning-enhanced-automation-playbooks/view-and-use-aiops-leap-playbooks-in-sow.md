# Use AIOps LEAP playbooks in Service Operations Workspace \(SOW\)

AIOps LEAP playbooks are available for use in the SOW interface.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **AIOps LEAP**.

2.  Select the required automation opportunity.

3.  [Generate AIOps LEAP playbooks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/aiops-leap-learning-enhanced-automation-playbooks/generate-playbooks.md).

4.  Open the SOW interface using the Workspaces menu.

5.  Select the required incident, and access the insights' panel.

6.  Select the playbook icon \[Omitted image "playbook-in-sow.png"\] Alt text: playbook icon in SOW insight panel.

7.  If a LEAP playbook is available for the selected incident, then it can be used to resolve the issue.
