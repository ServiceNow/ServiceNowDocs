# Configuring AIOps LEAP

You can download and install AIOps LEAP application from the ServiceNow® Store to manage your IT operations efficiently.

## Configuration overview

1.  [Install AIOps LEAP](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/aiops-leap-learning-enhanced-automation-playbooks/install-aiops-leap-application.md)

    Install AIOps LEAP if you have the admin role.

2.  [Activate AIOps LEAP](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-operations-management/aiops-leap-learning-enhanced-automation-playbooks/activate-aiops-leap.md)

    Activate AIOps LEAP to enable grouping of incidents, create problem records, and generate resolution playbooks.
