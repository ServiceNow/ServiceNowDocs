# Automation details fields

The fields to be configured in the Automation details section of the Respond page, when creating an alert automation.

|Field|Description|
|-----|-----------|
|Order|A number representing the order in which the automation is to run. Automations run in order from the lowest to the highest. Automations for all assignment groups range from 1 to 9999. Automations for single-assignment groups range from 10,000 to 19,999.|
|Automation is managed by|The group that owns, edits, and can delete the automation. The value of this field is determined by the value of the **Assignment group** field when configuring an automation condition in the **If these conditions are met...** section.|
|Description|A description of the automation's functionality.|
