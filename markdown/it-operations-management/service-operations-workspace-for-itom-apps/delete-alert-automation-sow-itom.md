# Delete an automation

Delete an automation to permanently remove an automated alert handling rule from your system.

## Before you begin

Role required: evt\_mgmt\_admin, evt\_team\_operator, or srm\_responder

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  In the primary navigation, select the Alert Automation icon \(\[Omitted image "alert-automations-icon.png"\] Alt text: Alert automation icon\).

3.  On the Alert Automation page, under **Automation types**, select the automation type that you want to delete, such as Ignore, Enrich, Group, or Respond.

    The page for that automation type opens, displaying all the existing automations.

4.  To delete an automation, select the check box next to its name and then select **Delete**.

    \[Omitted image "delete-automation.png"\] Alt text: Delete automation option.

5.  In the confirmation window, select **Delete**.

    The alert is permanently deleted.
