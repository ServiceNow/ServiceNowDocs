# Remediate an alert in Express List

Take a remediation action to fix an issue manually on an alert directly from the Express List pane.

## Before you begin

Role required: evt\_mgmt\_operator, evt\_mgmt\_admin

**Note:** The evt\_mgmt\_user role is view-only and does not have permission to perform this action.

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  From the navigation bar, select the Express List icon \(\[Omitted image "express-list1.png"\] Alt text: Express List icon\).

3.  Select the alert number to open it.

4.  Select the remediation action for the alert from the **Close** drop-down list.

    **Note:** If the list is empty, no remediation actions are available.

    \[Omitted image "remediation-action.png"\] Alt text: Option to remediate an alert.
