# Create alert clustering tags

Alert clustering tags represent an improved way to correlate alerts. Alerts with identical or similar tags \(depending on the configured match method\) are joined together to form a group.

## Before you begin

Role required: evt\_mgmt\_admin

## Procedure

1.  Navigate to **All** &gt; **Event Management** &gt; **Tag Based Alert Clustering Engine** &gt; **Alert Clustering Tags**.

2.  Click **New**.

3.  Fill in the values on the [Event Management tag based alert clustering tag form](../reference/tag-based-alert-clustering-tag-form.md).

4.  Select **Submit**.


## Result

The tag appears on the Tag Based Alert Clustering Tags table.
