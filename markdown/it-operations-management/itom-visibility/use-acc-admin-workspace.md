# Use the Agent Client Collector \(ACC\) admin workspace

Use the Agent Client Collector \(ACC\) admin workspace to monitor the status of your agents.

## Before you begin

Role required: agent\_client\_collector\_admin

## About this task

For details on the information accessible on the ACC admin workspace, see [Agent Client Collector \(ACC\) admin workspace information](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/itom-visibility/acc-admin-workspace-information.md).

## Procedure

1.  Navigate to **Workspaces** &gt; **ITOM Infra Services Workspace**.

2.  Select the **ACC agents** icon \[Omitted image "acc-agents-icon.png"\] Alt text: ACC agents icon.

    The ACC agents page appears, displaying an overview of the agents in your system on the Dashboard tab.

    \[Omitted image "acc-agents-dashboard.png"\] Alt text: ACC agents dashboard

3.  Select the **Agents** tab to view a list of the agents.

    \[Omitted image "acc-agents-workspace.png"\] Alt text: ACC agents page in Workspace

    You can filter the displayed agents by selecting one of the boxes at the top of the page. For example, select **Agents down** to filter the list by agents that are currently down.

4.  Select an agent in the agents list to view details of the agent.

    \[Omitted image "agent-record-page.png"\] Alt text: Agent details page

5.  Select the **Agent actions** button and select from actions to perform on the agent.

    For details on the available actions, see [Agent Client Collector \(ACC\) admin workspace agent actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/it-operations-management/itom-visibility/acc-admin-workspace-agent-actions.md).

    **Note:** The **Agent actions** button displays only when you have installed Agent Client Collector Framework version 6.0.0 or higher.
