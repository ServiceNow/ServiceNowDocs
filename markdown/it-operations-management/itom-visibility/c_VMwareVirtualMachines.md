# VMware virtual machines

Discovery gathers information about virtual machines managed by vCenter.

The Discovery product can extract information from VMware configuration items \(CI\), including their relationships. Discovery is configured to work with the Orchestration VMware Support Plugin, which enables the automatic cloning of virtual machines by an ESX server managed by vCenter. For information about data collected by Discovery on vCenter, see [Data collected for VMware vCenter Server](../reference/r_VCenterDataCollected.md).

-   **[VMware Workstation - Legacy](c_VMWareWinLinWithoutVCenter.md)**  
In the basic VMware system, the VMware Workstation runs on a Windows or Linux host machine.
