# Event severities

The event severity numbers correspond to actual severities on the Metric Anomaly Score to Event Severity Maps page.

|Event Severity Number|Event Severity|
|---------------------|--------------|
|0|Clear \(OK\)|
|1|Critical|
|2|Major|
|3|Minor|
|4|Warning|
