# ACC Error Task page

The fields to be configured on the ACC Error Task page.

|Field|Description|
|-----|-----------|
|Assigned to|Select the person to assign to work on the error.|
|Configuration item|Select the configuration item that the task relates to.|
|Parent|Select the parent task \(if one exists\) that is related to the current task.|
|Short description|Enter a one-line description of the task.|
|Description|Enter a full description of the task.|
