# Agent Client Collector API

Use the Agent Client Collector API to create a flow that executes an `osquery` command on agents and processes the results.

The system supports osquery 5.7.0.

For details on agent APIs, see Enable agent APIs.

For details on enabling policy APIs, see Enable policy APIs.
