# Configure an agent registration key

Configure an agent registration key so that you can deploy Agent Client Collector by using ITOM Cloud Services. With ITOM Cloud Services, you can deploy Agent Client Collector without a MID Server, which saves the MID Server for more persistent resources.

## Before you begin

Install the plugins necessary for ITOM Cloud Services, as described in [Agent Client Collector installation](../concept/acc-installation.md).

Role required: agent\_client\_collector\_admin

## Procedure

1.  Navigate to **All** &gt; **Agent Client Collector** &gt; **Deployment** &gt; **Agent Registration Key**.

2.  Select **New**.

3.  In the **Name** field, enter a descriptive name for the registration key.

4.  Select **Submit**.


## Result

The new agent registration key is ready to use for installation.
