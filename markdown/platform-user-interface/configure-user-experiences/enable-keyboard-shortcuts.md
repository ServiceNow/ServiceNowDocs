# Enable keyboard shortcuts

You can enable or disable keyboard shortcuts from the system settings menu.

## Before you begin

Role required: none

For a list of available keyboard shortcuts, see [Core UI keyboard shortcuts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/platform-user-interface/configure-user-experiences/r_KeyboardShortcuts.md).

## Procedure

1.  Select the gear icon \(\[Omitted image "IconUI16Gear.png"\] Alt text: Gear icon\) in the banner frame to open the System Settings window.

2.  In the General tab, select or clear the **Keyboard shortcuts enabled** option.

3.  Close the system settings window.
