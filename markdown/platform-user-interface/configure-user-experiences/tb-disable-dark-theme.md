# Disable the dark theme in all Next Experience instances

Disable the Next Experience dark theme in all of Next Experience instances.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** and then in the filter navigator, search for `sys_properties.list`.

2.  In the list search bar, search for `glide.ui.polaris.dark_themes_enabled`.

3.  Disable the dark theme in all of your Next Experience instances by setting `glide.ui.polaris.dark_themes_enabled` to **False**.
