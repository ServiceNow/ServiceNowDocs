# Select a theme in Next Experience

Personalize your Next Experience instance by selecting a theme variant or alternate color palette in the user preferences menu.

## Before you begin

**Note:** An administrator must enable at least one theme before you can select one as described in this task. For more information about theme variants, see [Core styles, colors, variants, and alternate color palettes](../reference/difference-themes-variants.md). For more information on defining alternate color palettes, see [Add an alternate color palette](tb-edit-color-palette.md).

Role required: none

## Procedure

1.  In the Unified Navigation banner, select the user menu ![User menu.](../../../get-started/servicenow-overview/image/pol-user-menu.png).

2.  Select **Preferences**.

3.  Select the **Theme** button.

4.  Select an available theme.

    The themes that are displayed in your Theme user preference depend on the customizations that were made by your administrator. You can choose from multiple themes and apply them to your Next Experience instance.

    Each theme card displays the core theme and all variants or alternative color palettes that are associated with that core theme. Hover over the circles beneath the core theme to view a preview of the variant or alternate color palette. A colored border appears around the theme card that has been applied to your instance.

    ![Theme choices.](../image/select-a-theme.png)
