# Update an existing image

You can change an existing image to an updated version.

## Before you begin

Role required: image\_admin or content\_admin

## Procedure

1.  Navigate to **All** &gt; **System UI** &gt; **Images**.

2.  In the Images list, click the name of the image to be replaced.

3.  In the **Image** field, click the **\[Update\]** link.

4.  Click **Browse** and navigate to the desired image file.

5.  Click **Open**, and then click **OK**.

    The new file is uploaded into the instance database, replacing the previous file. It is used in all locations that reference the file name.

    **Note:** If the new image does not appear as expected, clear the browser cache.
