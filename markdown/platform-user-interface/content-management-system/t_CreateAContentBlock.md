# Configure a content block

To configure a content block, define it in the appropriate form.

## Before you begin

Role required: content\_admin or admin

## Procedure

1.  Perform one of the following actions to create a content block.

    -   Navigate to **Content Management** &gt; **Blocks** &gt; **\[Block type\]** &gt; **New**. The form for creating the selected block type opens.
    -   Navigate to **Content Management** &gt; **Blocks** &gt; **All** and select the type of content to create from the list.
    -   Edit a content page and add a stub block of the desired content block type. Click the link in the stub block.
2.  Complete the content block form and save it.

3.  Add the content block [to any content page](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/content-management-system/t_AddContentToAPage.md).
