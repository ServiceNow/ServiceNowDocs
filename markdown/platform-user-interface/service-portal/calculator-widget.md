# Calculator widget

The calculator widget does simple calculations. You can use this base system widget as-is in your portal or clone it to suit your own business needs.

Use the calculator widget as an example of how to pass data between the client and server.

\[Omitted image "WidgetCalculator.png"\] Alt text: Calculator widget containing a math problem

## Instance options

The calculator widget does not include instance options.
