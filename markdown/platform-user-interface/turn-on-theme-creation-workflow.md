# Turn on theme creation workflow for use in the Now Assist panel

Turn on the theme creation workflow to generate themes using the Now Assist panel.

## Before you begin

Install the Now Assist for Creator application. See [Install Now Assist for Creator](https://www.servicenow.com/docs/access?context=install-now-assist-for-creator&version=zurich&pubname=zurich-build-workflows&ft:locale=en-US).

Turn on the Now Assist panel. See [Activate the Now Assist panel standard chat](https://www.servicenow.com/docs/access?context=activate-now-assist-panel&version=zurich&pubname=zurich-intelligent-experiences&ft:locale=en-US)

Role required: admin

## About this task

This workflow is required to use the AI feature within the Now Assist panel.

**Important:** The theme creation workflow requires a separate subscription to Now Assist for Creator.

## Procedure

1.  Navigate to **All** &gt; **AI Agent Studio** &gt; **Overview**.

2.  Select the **Theme Creation Workflow** card.

3.  From the Agentic workflow table, select **Theme Creation Workflow**.

4.  Choose **Select channels and status**.

5.  Turn on **Engage via the Now Assist panel**.

6.  Activate the theme workflow by selecting **Test**.

    The workflow is enabled in the Now Assist panel.
