# Delete a Visual Task Board

You can delete any task board that you own.

## Before you begin

Role required: none

## About this task

Deleting a board does not impact the underlying task records. However, you cannot recover a board after you delete it.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Visual Task Boards**.

2.  Under **Boards you own**, hover over the board and click the delete icon \(**X**\).

3.  In the confirmation dialog, click **Delete**.
