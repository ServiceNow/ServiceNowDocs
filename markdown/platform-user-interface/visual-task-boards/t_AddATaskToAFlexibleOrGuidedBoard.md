# Add a task to a flexible or guided Visual Task Board

Cards in a flexible or guided board represent records on whichever table the board is associated with.

## Before you begin

Role required: none

## Procedure

1.  In a lane header, click the lane menu \(\[Omitted image "IconMoreOptions.png"\] Alt text: Lane menu icon\) and select **Add Card**.

    Alternatively, click **Add Card** at the bottom of a lane.

    Fields are populated based on any filter conditions for the board and, for guided boards, the lane you add the card to.

    Consider the example of a guided board that shows incidents with the **Software** category, with a lane for each assignee. If you add a card to the **ITIL User** lane, the new incident record **Category** and **Assigned to** fields are automatically set to **Software** and **ITIL User**.

2.  Complete the form and click **Submit**.

    A new task card appears in the lane.
