# Using Configurable Workspace

Learn how to use common features in a Configurable Workspace.

-   **[Lists](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/using-lists-configurable-workspace.md)**

    Learn how to use list and list page features in Configurable Workspace.

-   **[Forms](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/using-forms-configurable-workspace.md)**

    Learn how to use form features on a Configurable Workspace record page.

-   **[Activity stream](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/using-activity-stream-configurable-workspace.md)**

    Learn how to use Activity stream features on a Configurable Workspace record page.

-   **[Emails](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/using-compose-configurable-workspace.md)**

    Learn how to send emails on a Configurable Workspace record page.

-   **[Screen Summarization](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/use-screen-summarization.md)**

    Screen Summarization is a feature that supports visually impaired and low-vision users by providing AI-generated summaries of workspace pages and their sections. The page and section summaries can be read aloud with a screen reader.
