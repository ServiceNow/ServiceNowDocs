# Create a customer success choice record

Create a record to set up the categories for the customer success choices.

## Before you begin

Role required: sn\_ti\_core.write

## Procedure

1.  Navigate to **Workspace** &gt; **CSM/FSM Configurable Workspace** and select the **List** icon.

2.  Navigate to the **Customer Success** &gt; **Customer Success Choice** and select **New**.

3.  Enter a name for the Customer Success choice.

4.  Select the category as **Customer Success Definition** from the list.

5.  Select the dependent category from the list.

6.  Enter a description for this record and select **Save**.

    The customer success choice record you create will be available as a sub category when you create success plays.
