# Text Adjust neurodiversity browser extension

Discover the Text Adjust neurodiversity browser extension and its benefits. The extension allows users to change the visual appearance of text on web pages to make them easier to read.

-   **Available in the Chrome web store**

    The Text Adjust Neurodiversity browser extension \(textadjust-neurodiversity-tool\) is a chromium-based, supplemental reading comprehension web tool for users with neurodivergent conditions like Dyslexia, Autism, ADHD, and other cognitive disabilities. This utility enables the user to change the webpage's user interface \(UI\) by replacing the font and adjusting the spacing of the words and paragraphs on the page.

    Features include the ability to adjust the:

    -   Default font on a webpage with a different font family.
    -   Spacing of letters, words, line height, and paragraphs on a webpage.
    This feature also supports 21 language versions based on the user's browser settings. For more information, see the [textadjust-neurodiversity-tool](https://chromewebstore.google.com/detail/textadjust-neurodiversity/cdnbhnlpadniacanbcipaccljlnmpnfj?utm_source=item-share-cb) in the Chrome web store.


-   **Installing the browser extension from the Chrome store**
    1.  Navigate to the [TextAdjust page in the Chrome web store](https://chromewebstore.google.com/detail/textadjust-neurodiversity/cdnbhnlpadniacanbcipaccljlnmpnfj?utm_source=item-share-cb)
    2.  Select "Add to Chrome"
    3.  Select "Add extension"
    4.  To pin to toolbar:
        1.  Hover over the Extension puzzle piece icon
        2.  Scroll to "Text Adjust Neurodiversity browser extension"
        3.  Select the push pin icon next to Text Adjust

-   **Installing the zipped extension in Microsoft Edge**
    1.  1 Navigate to the [Text Adjust announcement in Community](https://www.servicenow.com/community/accessibility-a11y/make-the-web-work-for-you-try-the-text-adjust-neurodiversity/ba-p/3334025) and download the textadjust-neurodiversity-tool.zip
    2.  Unzip the **textadjust-neurodiversity-tool.zip** file and save it to a permanent location on your computer \(e.g. your Documents or Applications folder\).

        **Important:** Do not move or rename this folder after setup, or the tool may no longer work as expected.

    3.  Go to edge://extensions/
    4.  Toggle Developer Mode to 'on.'
    5.  Select the **Load unpacked** button.
    6.  Select the folder you just decompressed in step 1. \(The Text Adjust app should appear under All Extensions.\)
    7.  Select the Extensions icon \[Omitted image "edge-extensions-icon.png"\] Alt text:in the browser to see the extension and select the pin icon \[Omitted image "edge-pin-icon.png"\] Alt text: to pin it to the toolbar.
    8.  To open Text Adjust, select the Text Adjust icon now present in the browser toolbar.
    9.  Review and adjust settings as needed.
