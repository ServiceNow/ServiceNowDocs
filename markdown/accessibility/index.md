# Product accessibility

- [Product accessibility](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/product_accessibility.md) -- Explore product accessibility on the ServiceNow AI Platform. Watch demo videos about accessibility features for ServiceNow products.
  - [Accessibility Conformance Reports](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/available-accessibility-conformance-reports.md) -- Accessibility conformance reports (ACRs) are published for currently supported ServiceNow releases. We use the International Edition of the Information Technology Industry Council's Voluntary Product Accessibility Template (VPAT) to create our ACRs, which are provided in PDF format.
  - [Accessibility by persona](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/r-accessibility-personas.md) -- Specific accessibility features and user preferences can help users with varying access needs. Learn more about accessibility personas and the ServiceNow products and features that can support them.
  - [Text Adjust neurodiversity browser extension](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/text-adjust-extension.md) -- Discover the Text Adjust neurodiversity browser extension and its benefits. The extension allows users to change the visual appearance of text on web pages to make them easier to read.
  - [Product documentation for accessibility](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/available-accessibility-product-documentation.md) -- Discover accessibility features and configuration options in the ServiceNow AI Platform and products. Explore links to additional accessibility information throughout ServiceNow product documentation.
    - [Next Experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/next-experience-accessibility-features-and-updates.md) -- Learn about accessibility features and updates in the Next Experience user interface on the ServiceNow AI Platform.
    - [Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/core-ui-accessibility-features-and-updates.md) -- Explore accessibility features and updates in the Next Experience user interface on the ServiceNow AI Platform.
    - [Portals](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/service-portals-accessibility-features-and-updates.md) -- Explore Accessibility features and updates on the Service Portal feature on the ServiceNow AI Platform.
    - [Mobile Platform](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/mobile-platform-accessibility-features-and-updates.md) -- Explore Accessibility features and updates on the Mobile version of the ServiceNow AI Platform.
    - [Keyboard Shortcuts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/servicenow-keyboard-shortcuts.md) -- ServiceNow strives to incorporate accessibility features into its AI Platform and products. Keyboard shortcuts play a crucial role in ensuring that users can navigate and utilize our tools efficiently. Explore the links in this topic to learn more.
    - [Builders' accessibility features and updates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/builders-accessibility-features-and-updates.md) -- ServiceNow provides accessibility solutions for partners. Builder tools include features that help teams design more accessible interfaces and find and fix issues during the design process.
    - [Now Assist and Accessibility](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/now-assist-and-accessibility.md) -- Now Assist, the generative AI capabilities from ServiceNow, helps users work more efficiently through conversational and proactive experiences. Its capabilities can also support accessibility by reducing cognitive load, offering alternative input methods, and simplifying complex tasks for a broad range of users.
