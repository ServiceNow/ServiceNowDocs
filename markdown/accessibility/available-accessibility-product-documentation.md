# Product documentation for accessibility

Discover accessibility features and configuration options in the ServiceNow AI Platform and products. Explore links to additional accessibility information throughout ServiceNow product documentation.

**Note:** To review accessibility highlights for the Australia release, see [Accessibility and Conformance](https://www.servicenow.com/docs/r/release-notes/r_Accessibility508Compliance.html). For a summary of all accessibility improvements for Australia, see [Release Notes for Accessibility](https://www.servicenow.com/docs/r/release-notes/rn-summary-accessibility.html).

## ServiceNow AI Platform user interfaces

In the ServiceNow platform and products, you can configure accessibility features globally or for specific users. Users can also individually enable user preferences for themselves to meet their specific accessibility needs.

The following topics provide links and resources to help you understand and configure the accessibility features and user preferences available in the different ServiceNow UI.

Discover how to make Next Experience, CoreUI, the Service Portal and ServiceNow Mobile work for everyone.

-   **[Next Experience accessibility features and updates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/next-experience-accessibility-features-and-updates.md)**  
Learn about accessibility features and updates in the Next Experience user interface on the ServiceNow AI Platform.
-   **[Core UI accessibility features and updates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/core-ui-accessibility-features-and-updates.md)**  
Explore accessibility features and updates in the Next Experience user interface on the ServiceNow AI Platform.
-   **[Portal accessibility features and updates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/service-portals-accessibility-features-and-updates.md)**  
Explore Accessibility features and updates on the Service Portal feature on the ServiceNow AI Platform.
-   **[Mobile Platform accessibility features and updates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/mobile-platform-accessibility-features-and-updates.md)**  
Explore Accessibility features and updates on the Mobile version of the ServiceNow AI Platform.
-   **[ServiceNow keyboard shortcuts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/servicenow-keyboard-shortcuts.md)**  
ServiceNow strives to incorporate accessibility features into its AI Platform and products. Keyboard shortcuts play a crucial role in ensuring that users can navigate and utilize our tools efficiently. Explore the links in this topic to learn more.
-   **[Builders' accessibility features and updates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/builders-accessibility-features-and-updates.md)**  
 ServiceNow provides accessibility solutions for partners. Builder tools include features that help teams design more accessible interfaces and find and fix issues during the design process.
-   **[Now Assist and Accessibility](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/nofamily/markdown/accessibility/now-assist-and-accessibility.md)**  
Now Assist, the generative AI capabilities from ServiceNow, helps users work more efficiently through conversational and proactive experiences. Its capabilities can also support accessibility by reducing cognitive load, offering alternative input methods, and simplifying complex tasks for a broad range of users.
