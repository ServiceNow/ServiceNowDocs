# Portal accessibility features and updates

Explore Accessibility features and updates on the Service Portal feature on the ServiceNow AI Platform.

Service Portal allows you to build a mobile-friendly self-service portal experience for your employees or customers. Service Portal is a Now Platform feature that is active by default and interacts with parts of the platform so users can access platform features through portals.

The following table provides links and resources to configure accessibility features in the Service Portal.

|Accessibility Feature|Summary|
|---------------------|-------|
|[Enable text resize in your portal](https://www.servicenow.com/docs/r/platform-user-interface/service-portal/enable-text-resize.html)|Enable text resize to increase the size of your text up to 200% through your browser settings.|
|[Enable dark theme in Service Portal](https://www.servicenow.com/docs/r/platform-user-interface/service-portal/enable-dark-theme.html)|Use the Dark theme in Service Portal to improve focus and provide better accessibility support. This option is commonly used to alleviate eye strain and improve readability.|
