# Creating procedure request cases as a contributor

Create procedure request cases for your patients from a service portal as a user with the contributor profile.

Your administrator can configure the option for creating procedure request cases. By default, the option is available from the Case menu on the Consumer Service Portal page. If you have the [contributor role in CSM](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/customer-service-management/csm-contributor-user-roles.md), you can use the option to create procedure request cases from your Consumer Service Portal.
