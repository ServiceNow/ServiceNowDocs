# Create a support request in Care Team Mobile

Use **Create a request** in Care Team Mobile to create a support request directly from your mobile device.

## Before you begin

Role required: Team member, Team manager

## About this task

![Create a request in Care Team Mobile.](../image/hcls-cto-mobile-create-request.png)

## Procedure

1.  In Care Team Mobile, navigate to **Quick actions** &gt; **Create a request**.

2.  Select the category of request that you want to create.

    For example, **Environmental service request**.

3.  Select the type of request that you want to create.

    For example, **Housekeeping and Sanitation Services** &gt; **Linen and Laundry Services**.

4.  Fill in the fields as needed.

5.  Select **Submit**.

6.  Select **Save as draft** to save the information within the request without submitting it.


## Result

A service request has been created and routed to the associated supporting services department for fulfillment.

## What to do next

[Track a submitted request in Care Team Mobile](cto-mobile-track-requests.md) or [Cancel a submitted request in Care Team Mobile](cto-cancel-submitted-request.md).
