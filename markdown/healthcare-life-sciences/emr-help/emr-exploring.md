# EMR Help

With the ServiceNow® EMR Help application, simplify and streamline the process to submit ServiceNow service requests related to an electronic medical record \(EMR\) system.

![The Health Case request form in the EMR portal.](../image/hcls-emr-healthcare-case.png)

Create service requests from directly within an EMR system that can then be fulfilled using a ServiceNow instance.
