# Creating enrollment cases as a contributor

Create enrollment cases for your patients from a service portal as a user with the contributor profile.

Your administrator can configure the option for creating enrollment cases. By default, the option is available from the Case menu on the Consumer Service Portal page. If you have the [contributor role in CSM](https://www.servicenow.com/docs/access?context=csm-contributor-user-roles&version=xanadu&pubname=xanadu-customer-service-management&ft:locale=en-US), you can use the option to create enrollment cases from your Consumer Service Portal.
