# Pending to-dos widget

The Pending to-dos widget displays a list of to-do items assigned to a patient.

![Screenshot displaying thePending to-dos widget. For the text description, refer to the Key features section.](../image/hcls-widget-pending-to-dos.png "Pending to-dos widget")

## Key features

-   Displayed only for a logged-in user with the personal information completed.
-   By default, shows maximum of two records from the Healthcare Task \[sn\_hcls\_task\] table in ascending order by the date created.
-   Shows all the pending and closed to-do items in the respective sections when a user clicks **View all** on the widget.

The Pending to-dos widget does not include instance options.
