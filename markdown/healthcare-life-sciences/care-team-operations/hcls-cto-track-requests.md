# Track requests in the Care Team Portal

Track requests created by you or your team from directly within the Care Team Portal.

## Before you begin

Role required: Team member, Team manager

![Track requests in the Care Team Portal.](../image/cto-hcls-track-requests.png)

## Procedure

1.  In the Care Team Portal, navigate to **Track a request** &gt; **All services** from the menu bar, or select the **Track requests** from the quick links widget on the landing page.

2.  View requests as they display in the following tabs:

    -   **All** - All service requests created for that category.
    -   **Created by me** - All service requests created by you for that category.
    -   **Closed** - All closed requests.
    Use the lefthand navigation bar to view requests by service type.

    Use the search function within a category to look for a specific request.

    Select the filter icon to filter all requests to those created within the last one, three, six, or 12 months.

3.  Select the request number of a request to see more information about that request as needed.
