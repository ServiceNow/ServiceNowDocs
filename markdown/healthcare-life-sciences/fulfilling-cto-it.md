# Fulfilling service requests in Care Team Operations for Healthcare IT

Incidents created from Healthcare IT cases are resolved using standard IT Service Management incident management functionality in the Service Operations workspace.

For more information, see [Incident Management](https://www.servicenow.com/docs/access?context=c_IncidentManagement&version=zurich&pubname=zurich-it-service-management&ft:locale=en-US).

**Note:** Fulfillers should modify the Incident view Service Operations Workspace and add the related list **Case-&gt;Incident**.

This enables you to see the case associated with see the incident on the Service Operations Workspace under the Related Records section.
