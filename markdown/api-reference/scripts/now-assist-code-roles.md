# Code generation roles

The following roles are installed for use with code generation.

To learn more about managing per-user subscriptions, see [Managing per-user subscriptions in Subscription Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/platform-administration/managing-user-subscriptions-v2.md) and contact your account representative.

## Now Assist Creator \[now.assist.creator\]

Write scripts using AI-powered code generation.

-   **Groups**

    This role is assigned to no groups by default.

-   **Contains Roles**

    This role contains no roles.

-   **Elevated**

    This role is not an elevated role.

-   **Special considerations**

    None
