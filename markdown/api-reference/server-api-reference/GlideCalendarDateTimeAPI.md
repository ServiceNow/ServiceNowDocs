# GlideCalendarDateTime- Scoped \(deprecated\)

The GlideCalendarDateTime class provides methods for performing operations on GlideCalendarDateTime objects, such as instantiating GlideCalendarDateTime objects or working with glide\_date\_time fields.

This API has been deprecated. Please use the [GlideDateTime](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/api-reference/server-api-reference/c_GlideDateTimeScoped.md) API instead.
