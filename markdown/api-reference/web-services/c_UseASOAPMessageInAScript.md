# Preview a SOAP message script

Generate a sample script to send the SOAP message.

After you have developed and tested the SOAP message, click the **Preview script usage** related link in the SOAP Message Function form. The dialog box displays an example of how you can invoke the SOAP message from a script.

You can manipulate the resulting XML response body with [XMLDocument](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/api-reference/scripts/c_XMLDocumentScriptObject.md) or with [gs.getXMLText](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/api-reference/scripts/p_GlideServerAPIs.md) and [gs.getXMLNodeList](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/api-reference/scripts/p_GlideServerAPIs.md).
