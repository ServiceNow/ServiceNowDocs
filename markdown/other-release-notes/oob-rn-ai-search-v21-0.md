# AI Search v21.0 release notes

This release version includes minor bug fixes and improvements.

## Compatibility information

This release is compatible with the Rome release. To learn more, see AI search release notes on the docs site.

## Fixed problems

|Problem category|Problem|Short description|
|----------------|-------|-----------------|
|AI Search Service|PRB1514054|Ingestion of records with attachments causes warnings in the logs|
|AI Search Service|PRB1512211|The AI Search Preview tool highlights text that is not relevant to the query|
|AI Search Service|PRB1513662|Creating a tokenizer is slow, and the system creates one tokenizer for each field of each ingested document if the language is not English|
|AI Search Service|PRB1513883|Min or term count does not discount a stopword in the case of a lemma expansion|
