# MetricBase available versions

Version history release notes for MetricBase releases.

## Compatibility

All MetricBase versions are backwards-compatible and can be used on any ServiceNow family release.

## Installation

For instructions on how to install and configure MetricBase on your on-premise environment, contact your Support Account Manager or sales representative.
