# Machine Learning Engine for on-prem v7.0 release notes

Machine Learning Engine for on-prem v7.0 includes minor bug fixes and improvements.

## Compatibility

Machine Learning Engine v7.0 is compatible with Quebec release.

## Installation

You can access this update along with the download and installation instructions via the [Machine Learning Engine Installation and Configuration for Self-Hosted Customers Knowledge Base](https://hi.service-now.com/kb_view.do?sysparm_article=KB0782052) \(KB0782052\).

For technical assistance and information, please visit ServiceNow Support: [https://support.servicenow.com/support/](https://support.servicenow.com/support/)
