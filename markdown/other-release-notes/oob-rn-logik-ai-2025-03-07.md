# Logik.ai 2025/03/07 release notes

Logik.ai 2025/03/07 release notes include minor bug fixes.

## Schedule

|Sector|Upgrade|
|------|-------|
|Demo|Thursday, Mar 06, 2025, 8 pm CT|
|Test|Thursday, Mar 21, 2025, 8 pm CT|
|Prod|Friday, Apr 04, 2025, 8 pm CT|

Example: an environment with URL https://\[YourLogikURL\].prod.logik.io receives this build Apr 04, 2025.

## Release notes

This release version includes the following fixes.

|Fix|Short description|
|---|-----------------|
|LGK-15543|Product picker parent and select subfield could get out of sync in some scenarios. This has been fixed.|
|LGK-15564|Contains operator was not behaving correctly on single select Product Picker values in certain scenarios. This has been corrected.|
|LGK-15950|In table-based picklists with a large number of picklist options, typing values for the filter did not always correctly filter the results. This has been fixed.|
