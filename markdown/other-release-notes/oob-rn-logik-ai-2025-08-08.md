# Logik.ai 2025/08/08 release notes

Logik.ai 2025/08/08 release notes include minor bug fixes.

## Schedule

|Sector|Upgrade|
|------|-------|
|Demo|Thursday, August 07, 2025, 8 pm CT|
|Test|Thursday, August 21, 2025, 8 pm CT|
|Prod|Friday, September 05, 2025, 8 pm CT|

Example: an environment with URL https://\[YourLogikURL\].prod.logik.io receives this build September 05, 2025.

## Release notes

This release version includes the following fixes.

|Fix|Short description|
|---|-----------------|
|LGK-16892|The Logik RCA Managed Package has been enhanced to add currency support for non-multicurrency organizations. After this enhancement, the currency code from Salesforce will be passed to Logik.|
|LGK-18062|Number fields with precisions defined were not correctly formatting in the Logik UI when clicking out of the field without making edits. This has been fixed.|
|LGK-18410|Existing configurations in read-only mode did not load correctly in the Logik UI. This has been corrected.|
|LGK-18604|In solution config, if the Product label was long, the ellipses to add or delete products were not visible. This is fixed.|
