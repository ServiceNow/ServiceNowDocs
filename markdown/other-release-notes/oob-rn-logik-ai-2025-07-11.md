# Logik.ai 2025/07/11 release notes

Logik.ai 2025/07/11 release notes include new enhancements and minor bug fixes.

## Schedule

|Sector|Upgrade|
|------|-------|
|Demo|Thursday, July 10, 2025, 8 pm CT|
|Test|Thursday, July 24, 2025, 8 pm CT|
|Prod|Friday, August 08, 2025, 8 pm CT|

Example: an environment with URL https://\[YourLogikURL\].prod.logik.io receives this build August 08, 2025.

## Release notes

This release version includes the following new enhancements and fixes.

## New Transaction Manager enhancement: Rule cycle report

The Rule Cycle Report has been extended to Transaction Manager. The report identifies potential loops, providing actionable information for admins to address.

## New Transaction Manager enhancement: Integration settings: set header values

New setting in Integrations to set header key-value pairs, utilize static text, header-level Transaction Manager fields using simple handlebar syntax, or a combination of both.

## Bug fixes

|Fix|Short description|
|---|-----------------|
|LGK-17572|Scroll bars appeared in product pickers when they had a single row, making it difficult to interact with the pickers. This has been fixed for both Windows and Mac users.|
|LGK-17755|The full-screen view on the Logik admin code editor page did not change the layout to full screen as expected. This issue was corrected.|
|LGK-18085|When line order changes, the order of the values in Lookup Aggregate will now update to match.|
