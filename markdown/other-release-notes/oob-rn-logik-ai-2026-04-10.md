# Logik.ai 2026/04/10 release notes

Logik.ai is a high-performance Commerce Logic Engine that complements Salesforce CPQ and eCommerce applications. This document outlines everything you need to know about new features and fixes made in the upcoming Logik.ai release.

## Schedule

|Sector|Upgrade|
|------|-------|
|Demo|Thursday, 09 Apr 2026 8 pm CT|
|Test|Thursday, 07 May 2026 8 pm CT|
|Prod|Friday, 12 Jun 2026 8 pm CT|

Example: an environment with URL, https://\[YourLogikURL\].prod.logik.io, receives this build June 5, 2026.

## Release notes

This release version includes the following fixes.

|Fix|Short description|
|---|-----------------|
|PRB1973057|Fixed issue where associating a blueprint to a configurable product set its External ID to null|
|PRB1986618|Fixed issue where Test Frame hardcoded the LGK namespace|
|PRB1989017|Fixed issue where the Product Picker select-all checkbox allowed invalid options to be selected|
|PRB1997327|Fixed issue where deleting selected rows in a filtered data table unexpectedly removed all rows, not just the selected ones|
|PRB1998923|Fixed issue where tags were adjusting the width of the page|
|PRB1998963|Fixed issue in the Transaction Visual Layout Editor where line-level boolean fields could be set to an invalid display type|
|PRB2001601|Fixed issue with the Export button in the Transaction UI|
|PRB2002477|Fixed issue where Quote AI would ask a question and then disable the cursor|
|PRB2006654|Fixed bug in the Transaction Editor YAML Layout Editor|
|PRB2006936|Fixed issue where header event fields were not refreshing on quote reopen|
|PRB2009515|Fixed issue where a bulk action was setting the product id to null|
