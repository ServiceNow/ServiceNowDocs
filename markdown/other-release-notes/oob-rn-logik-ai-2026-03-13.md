## Release notes

This release version includes the following new enhancements and fixes.

## Transaction Manager: Accessibility improvements

This release includes improvements to keyboard and screen reader accessibility.

## Transaction Manager: Transaction Edit History - new data

An event start time column is added to Transaction Edit History. Use this column to help track the duration of individual transaction events.
