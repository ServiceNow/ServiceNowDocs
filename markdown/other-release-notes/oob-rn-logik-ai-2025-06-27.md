# Logik.ai 2025/06/27 release notes

Logik.ai 2025/06/27 release notes include minor bug fixes.

## Schedule

|Sector|Upgrade|
|------|-------|
|Demo|Thursday, June 26, 2025, 8 pm CT|
|Test|Thursday, July 10, 2025, 8 pm CT|
|Prod|Friday, July 25, 2025, 8 pm CT|

Example: an environment with URL https://\[YourLogikURL\].prod.logik.io receives this build July 25, 2025.

## Release notes

This release version includes the following fixes.

|Fix|Short description|
|---|-----------------|
|LGK-17263|Resolved issue with text wrap in Script Editor|
|LGK-17719|Updated Transaction Manager "Behavior on Idle Timeout" setting timer to reset on start of service call, rather than completion|
|LGK-17757|Addressed Transaction Manager issue that sometimes resulted in Configurator UI not loading|
|LGK-17777|Addressed Transaction Manager issue with rule execution during stage transition|
