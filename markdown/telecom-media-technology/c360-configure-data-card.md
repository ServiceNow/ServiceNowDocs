# Configure the data visualization group variables

Configure the variables in the data visualization card.

## Before you begin

Role required: sn\_telecom\_c360.admin

## Procedure

1.  Navigate to **All** &gt; **Telecom Customer 360** &gt; **Guided Setup** and select **Get Started**.

2.  Select **360 Cards** &gt; **Data visualization group**.

3.  Modify the following variables.

    |Variable name|Description|
    |-------------|-----------|
    |Saved visualizations|A list of saved visualization records to be displayed on the card. The default values are top customer cases and the CSAT score.|
    |PAR visualization filters|Name-value pairs that define the dynamic performance analytics filters used to associate each visualization to the current context record.|
