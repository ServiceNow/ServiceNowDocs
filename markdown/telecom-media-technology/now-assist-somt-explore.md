# Exploring Now Assist for Sales and Order Management for Telecommunications \(SOMT\)

With the Now Assist for Sales and Order Management for Telecommunications \(SOMT\) application, you can use standalone AI agents to create a customer move order capture journey, create enrichment tasks, identify the tasks required for domain orders by using historical data from similar orders.
