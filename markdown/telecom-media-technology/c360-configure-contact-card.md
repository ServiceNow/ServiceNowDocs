# Configure the contact card variables

Configure the variables in the contact card.

## Before you begin

Role required: sn\_telecom\_c360.admin

## Procedure

1.  Navigate to **All** &gt; **Telecom Customer 360** &gt; **Guided Setup** and select **Get Started**.

2.  Select **360 Cards** &gt; **Contact**.

3.  Modify the following variables.

    |Variable name|Description|
    |-------------|-----------|
    |Data table|The table to retrieve customer fields from \(for example, `customer_account`\)|
    |Data table query|Query condition that filters the data table to the current context record.|
    |Fields|Comma separated list of fields to display on the card. Default fields are Mobile phone, Phone, Email, Street, City, State/Province.|
    |Avatar field|The field from the data table to be used as the avatar image for the card.|
    |Show avatar|Controls whether the avatar is displayed.|
