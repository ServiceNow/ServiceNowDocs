# Edit a certification policy

Update an certification existing policy details. You can edit the draft or inactive certification policies.

## Before you begin

Role required: System admin, cmdb admin + analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Data certification page by selecting the Data certification icon \(![data certification icon](../../image/eaw-image/icon-data-cert.png)\).

    The Data certification page opens.

3.  Open draft or an inactive policy by selecting the policy name link.

4.  Select **Edit Policy**.

5.  Edit the policy details as required.

6.  Select **Save &amp; Exit** to save the policy as draft or select **Publish Policy** to publish the draft policy.
