# View publishing status and run log

View publishing run logs to monitor publishing activity and review the publishing history for a TRM catalog.

## Before you begin

Role required: sn\_apm.apm\_admin

## About this task

To see publishing history of a catalog, the catalog must have been published at least once.

Publishing run logs provide a record of all publishing and republishing jobs for a TRM catalog. Use run logs to verify whether a publish or republish operation completed successfully, monitor jobs that are currently running, and review historical publishing activity.

**Note:** Run logs are read‑only. You can monitor job status but cannot modify any details. However, You can use the Refresh button on the run log page while status is running to get latest state.

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Setup page by selecting the Setup icon ![Setup icon](../../image/eaw-image/setup-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to **Publishing Center**.

4.  Select **All**.

5.  Select a TRM catalog to open its publishing configuration.

6.  Select the **Run logs** tab.

    You can see the publishing job history for the selected catalog.

7.  Select **Publish**.
