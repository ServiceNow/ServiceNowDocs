## About this task

Enterprise Modeling and Visualization diagrams often have complex architecture layers like business capabilities, application hierarchy, and process flows. Manual comparison can be time-consuming or error-prone. Using the diagram change analysis Now Assist skill, you can quickly generate a change summary.

You can compare any two diagram versions, not just adjacent versions. For example, you can compare version 3.0 with version 1.0, or version 4 with version 2.

The change summary generated by Now Assist includes an executive summary of changes between the primary version and the comparison version. The summary also includes details of shapes or entities that have been added, modified, or removed.

When you compare two diagram versions, the comparison screen highlights the differences on the canvas:

-   New shapes added in the primary version are highlighted in green.
-   New relationship lines added in the primary version are highlighted in green.
-   Modified shapes are highlighted in yellow.
-   Modified relationship lines are highlighted in yellow.

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Modeling page by selecting the Enterprise Modeling and Visualization icon \(![Enterprise Modeling and Visualization icon.](../../application-portfolio-management/image/eaw-image/modeling/icon-modeling-logo.png)\).

3.  Select the diagram that you want to compare versions for.

4.  Select **Compare versions**.

    The **Compare versions** button isn’t available when you open a diagram that has only one version.

    ![Screenshot displaying Enterprise Modeling and Visualization page with the Compare versions button highlighted.](../image/compare-diag-butn.png)

    The **Compare versions** pop-up window appears.

    ![Screenshot displaying the Compare versions pop-up window.](../image/compare-diag-modal.png)

5.  In the **Compare versions** pop-up window select the versions that you want to compare:

    1.  Select the primary version of the diagram from the **Primary version** drop-down list.
    2.  Select another version of the diagram from the **Version to compare** drop-down list.
    **Note:** You can compare any two versions, not just consecutive versions. For example, you can compare version 3.0 directly with version 1.0.

6.  Select **Compare**.

    The **Compare versions** tab appears and a change summary is generated in the Now Assist side panel. The Now Assist side panel displays the following sections:

    -   **Executive summary**: A Now Assist generated summary of the key changes between the selected versions.
    -   **Diagram changes**: Changes to shapes, entities, or relationships that have been added, modified, or deleted.

        ![Screenshot displaying the version change summary generated by Now Assist](../image/compare-diag-summry.png)

        **Note:**

        -   Select the edit icon \(![Edit icon.](../image/edit-icon.png)\) to change the existing diagram versions and generate a fresh diagram comparison summary.
        -   Select the refresh icon \(![Refresh icon.](../image/refresh-icon.png)\) if you have changed the primary version of the diagram and want to generate a fresh diagram comparison summary.
        -   Select the helpful icon \(![Helpful icon.](../image/like-button-icon.png)\) or not helpful icon \(![Not helpful icon.](../image/dislike-button-icon.png)\) to provide feedback on the change summary generated by Now Assist. This feedback helps Now Assist to generate better change summaries.
        -   Select the copy summary icon \(![Copy summary icon.](../image/copy-summary.png)\) to copy the diagram change summary created by Now Assist.
        -   Select the Now Assist icon \(![Now Assist icon.](../image/now-assist-side-pane-icon.png)\) to close the Now Assist side panel.
