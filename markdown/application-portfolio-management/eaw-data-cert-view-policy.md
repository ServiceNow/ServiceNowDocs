# View certification policy

View details of a published policy.

## Before you begin

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Data certification page by selecting the Data certification icon \(\[Omitted image "icon-data-cert.png"\] Alt text: data certification icon\).

    The Data certification page opens.

3.  Open a published policy by selecting the policy name link and then select the **View Policy** button.

    **Note:** Published policies cannot be edited. If you want to edit the policy details, you must deactivate the policy.

4.  Select **Exit**.
