# Add a strategy for managing applications - Legacy

Demand actions are strategic decisions that you want to execute for an application. Enterprise Architecture provides preconfigured actions that help you enhance the capability of the applications. You can add new demand actions as per your requirements.

## Before you begin

**Important:**

Starting with the Xanadu release, the legacy demand actions module is moved to the Enterprise Architecture Workspace. To learn more, see [Manage demand actions](../concept/eaw-concept/eaw-config-demand-actions.md).

Role required: sn\_apm.apm\_admin

## About this task

Create demand actions that are aligned to the application strategy.

## Procedure

1.  Navigate to **All** &gt; **Enterprise Architecture** &gt; **Administration** &gt; **Demand Actions**.

2.  Click **New**.

3.  Fill in the fields.

    For field information, see [Demand Actions Form](../reference/demand-action-form.md).

4.  Click **Submit**.
