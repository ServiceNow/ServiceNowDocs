# Create a source for an Application TCO

Create a source for an Application Total Cost of Ownership \(TCO\). Specify the source of cost being ingested in Enterprise Architecture Workspace.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace** &gt; **Setup**.

2.  Select **TCO**.

3.  Select **Sources**.

4.  Select **New**.

5.  Enter a name for the source.

6.  Select **Save**.
