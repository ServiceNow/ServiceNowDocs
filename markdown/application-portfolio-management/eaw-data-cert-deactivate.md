# Deactivate a certification policy

Deactivate a published policy to remove it from active certification runs. This process ensures that the policy is no longer active and can't be used for certifications.

## Before you begin

Role required: System admin, cmdb admin + analyst, com.snc\_apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Data certification page by selecting the Data certification icon \(![data certification icon](../../image/eaw-image/icon-data-cert.png)\).

    The Data certification page opens.

3.  Use one of the following options to deactivate a policy.

    -   Select the check box next to the policy that you want to deactivate and then select **Deactivate** from the context menu.
    ![Deactivate a policy using the context menu](../../image/eaw-image/eaw-data-cert-deactivate-policy.png)

    A confirmation message appears.

    -   Open the policy by selecting the policy name link and select the **Deactivate Policy** button.
    ![Deactivate a policy from the record page](../../image/eaw-image/eaw-data-cert-deactivate-from-recordpage.png)

4.  Select **Deactivate Policy**.
