# Create a business hierarchy map form

Creating artifacts for the business capability hierarchy map helps you to update the capability hierarchy directly in the map using Enterprise Modeling and Visualization.

## Create a business hierarchy map form fields

<table id="table_zkv_psq_ycc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Link to artifact

</td><td>

Artifact to which you want to associate the diagram. Use the following options:-   **New artifact**-Select to create an artifact and associate the diagram.
-   **Existing artifact**-Select to associate the diagram to an existing artifact.

</td></tr><tr><td>

Artifact name

</td><td>

Name of the artifact. Enter a name of the new artifact or select an existing artifact name based on the selection in the **Link to artifact** field.

</td></tr><tr><td>

Architectural category

</td><td>

Name of the category for the artifact.

</td></tr><tr><td>

Business application

</td><td>

Name of the business application. Select a business application to create the hierarchy map for it.

</td></tr></tbody>
