# Create new user criteria permissions

Create user criteria permissions to associate them with architectural artifacts.

<table id="table_oh2_d5q_ybc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Document

</td><td>

The architectural artifact associated with the user criteria permission.

</td></tr><tr><td>

Permission

</td><td>

The type of permission linked to the user criteria permission. The available options are:

-   **Reader**
-   **Writer**
-   **Owner**

</td></tr><tr><td>

User criteria

</td><td>

The user group to which the permission is provided.
