# Configure the Enterprise Architecture Workspace

Customize your workspace to display the components that you want to see.

## Before you begin

Role required: sn\_apm.apm\_admin and sn\_apm.apm\_user

## About this task

For the Insights, Overview, and Health sections, you can hide some components if you don't intend to use them.

## Procedure

1.  Navigate to **All** &gt; **Enterprise Architecture** &gt; **Enterprise Architecture Workspace** &gt; **Configuration**.

2.  On the APM EA Configurations page, select an item that you want to hide or show.

3.  In the Active column, select **True** or **False** to show or hide the component in the workspace.

    **Note:** You can create a visualization configuration using the Platform Analytics Workspace. For instructions, see [Create a single score visualization in the Visualization Designer](https://www.servicenow.com/docs/access?context=create-dv-sing-sc-ac&version=yokohama&pubname=yokohama-now-intelligence&ft:locale=en-US).
