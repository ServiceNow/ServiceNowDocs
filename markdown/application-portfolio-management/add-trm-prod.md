# Add a TRM product - Legacy

As an Enterprise Architect, you can add a new TRM product to the TRM library.

## Before you begin

**Important:**

Starting with the Xanadu release, the legacy Technology Reference Model module is moved to the Enterprise Architecture Workspace. To learn more, see [Add a TRM product in Enterprise Architecture Workspace](eaw-task/eaw-create-trm-prod-lifecycle.md).

Role required: sn\_apm\_analyst

## Procedure

1.  Navigate to **All** &gt; **Enterprise Architecture** &gt; **Technology Reference Model** &gt; **Products**.

2.  Click **New**.

3.  On the form, fill in the fields.

    For field information, see [New TRM product form](../reference/trm-product-form.md).

4.  Click **Submit** or **Update**.
