# Add an existing business application to a Technology Reference Model product

You can add existing business application to Technology Reference Model \(TRM\) products in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon \[Omitted image "technology-portfolio-icon.png"\] Alt text: Technology portfolio icon.

3.  In the **TRM catalog** tab, select **TRM product list** from the View list.\[Omitted image "trm-product-view-dropdown.png"\] Alt text: TRM view list drop-down list.

4.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to the TRM product that you want to add existing business applications to.

5.  Select the TRM product.

6.  Select **Business Applications**.

7.  Select **Add**.

8.  In the **Add business application** pop-up window, select the existing business application.

9.  Select **OK**.


## Result

The existing business application is added to the TRM product.
