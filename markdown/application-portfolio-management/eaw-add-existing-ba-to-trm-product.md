# Add an existing business application to a Technology Reference Model product

You can add existing business application to Technology Reference Model \(TRM\) products in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon ![Technology portfolio icon](../../image/eaw-image/technology-portfolio-icon.png).

3.  In the **TRM catalog** tab, select **TRM product list** from the View list.![TRM view list drop-down list.](../../image/eaw-image/trm-product-view-dropdown.png)

4.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to the TRM product that you want to add existing business applications to.

5.  Select the TRM product.

6.  Select **Business Applications**.

7.  Select **Add**.

8.  In the **Add business application** pop-up window, select the existing business application.

9.  Select **OK**.


## Result

The existing business application is added to the TRM product.
