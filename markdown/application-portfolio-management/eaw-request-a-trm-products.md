# Request a TRM product

You can request a TRM product using the TRM catalog to add a new software or hardware to the TRM library.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon ![Technology portfolio icon](../../image/eaw-image/technology-portfolio-icon.png).

3.  Select **Request TRM product**.

4.  On the form, fill in the fields.

    For field information, see [Request TRM product form](../../reference/trm-product-request-form.md).

5.  Select **Create**.
