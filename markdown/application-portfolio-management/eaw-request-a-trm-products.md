# Request a TRM product

You can request a TRM product using the TRM catalog to add a new software or hardware to the TRM library.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon \[Omitted image "technology-portfolio-icon.png"\] Alt text: Technology portfolio icon.

3.  Select **Request TRM product**.

4.  On the form, fill in the fields.

    For field information, see [Request TRM product form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/application-portfolio-management/trm-product-request-form.md).

5.  Select **Create**.
