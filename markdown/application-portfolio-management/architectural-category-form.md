# Architectural category form

Architectural category enables Enterprise architects to categorize and manage artifacts efficiently.

## Architectural category form fields

|Field|Description|
|-----|-----------|
|Name|Name of the category.|
|Description|Description of the category.|
|Parent Category|Hierarchical category. Select the parent category from the Architectural Categories page.|
