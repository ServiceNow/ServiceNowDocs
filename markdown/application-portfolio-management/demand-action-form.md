# Demand Actions Form

Demand actions are strategic decisions that you want to execute for an application. Enterprise Architecture \(formerly Application Portfolio Management\) provides preconfigured actions that help you enhance the capability of the applications.

## Demand Actions Form Fields

|Field|Description|
|-----|-----------|
|Action|Decision taken on the application.|
|Description|Description of the action.|
|Strategy|Plan to implement the action.|
