# View product capabilities associated with a Technology Reference Model product

You can view the list of product capabilities associated with Technology Reference Model \(TRM\) product, in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon \[Omitted image "technology-portfolio-icon.png"\] Alt text: Technology portfolio icon.

3.  In the **TRM catalog** tab, select **TRM product list** from the View list.\[Omitted image "trm-product-view-dropdown.png"\] Alt text: TRM view list drop-down list.

4.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to the TRM product that you want to view the associated product capabilities for.

5.  Select the TRM product.

6.  Select **Product capabilities**.\[Omitted image "trm-product-capability-related-list.png"\] Alt text: Product capabilities tab available as a related entity for a TRM product.


## Result

A list of product capabilities associated with the TRM product is displayed.
