# Request a TRM product lifecycle

You can add a new request to create a life cycle for a TRM product.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon ![Technology portfolio icon](../../image/eaw-image/technology-portfolio-icon.png).

3.  Select the TRM more actions icon ![TRM more actions icon](../../image/eaw-image/trm-more-actions-icon.png) next to **Request TRM product**.

4.  Select **Request TRM product lifecycle**.

5.  On the form, fill in the fields.

    For field information, see [Request TRM product lifecycle form](../../reference/eaw-reference/eaw-create-trm-prod-lifecycle-req-form.md).

6.  Select **Create**.
