# View all certification policies

View the list of all certification policies and manage them in the Enterprise Architecture Workspace.

## Before you begin

Role required: certification\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Setup page by selecting the Setup icon \(![setup icon.](../../image/eaw-image/eaw-icon-setup.png)\).

3.  Select the expand row icon \(![expand row icon.](../../image/ExpandIcon.png)\) next to **Certification Policies**.

4.  Select **All**.
