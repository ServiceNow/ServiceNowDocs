# Create diagram action form

The diagram action form helps you to create action for a custom shape that can be used in the Enterprise Modeling and Visualization.

## Diagram action form fields

|Field|Description|
|-----|-----------|
|Name|Name of the diagram action.|
|Diagram builder configuration|Select the diagram builder configuration for the action to be created.|
|Description|Enter description for the configuration.|
|Active|Select roles to enable the action.|
|Application|System sets this field to Global. This is a non-editable field.|
|Node type|Select node type for the diagram action.|
|Category|Select a category for the diagram action.|
|Sub category|Select a sub category for the diagram action.|
|Description|Enter a description for the diagram action.|
|Icon|Enter name of the icon for which you want to associate the diagram action.|
