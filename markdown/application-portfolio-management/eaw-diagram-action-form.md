# Create diagram action form

The diagram action form helps you to create action for a custom shape that can be used in the Enterprise Modeling and Visualization.

## Diagram action form fields

<table id="table_p4b_24g_c2c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the diagram action.

</td></tr><tr><td>

Diagram builder configuration

</td><td>

Select APM diagram configuration from the list.

</td></tr><tr><td>

Description

</td><td>

Enter description for the configuration.

</td></tr><tr><td>

Active

</td><td>

Select roles to enable the action.

</td></tr><tr><td>

Application

</td><td>

System sets this field to Global. This is a non-editable field.

</td></tr><tr><td>

Node type

</td><td>

Select either of the following options as a node type: -   Image node container
-   Image node container with connectors

</td></tr><tr><td>

Category

</td><td>

Select a category for the diagram action.

</td></tr><tr><td>

Sub category

</td><td>

Select a sub category for the diagram action.

</td></tr><tr><td>

Description

</td><td>

Enter a description for the diagram action.

</td></tr><tr><td>

Icon

</td><td>

Enter name of the icon for which you want to associate the diagram action.
