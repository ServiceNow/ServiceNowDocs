# Create a sub-capability form

Create a sub-capability and add it to the capability hierarchy map.

## Create sub-capability form fields

|Field|Description|
|-----|-----------|
|Name|Name of the sub-capability.|
|Description|A short description of the sub-capability.|
|Parent|Name of the parent capability for the sub-capability that you’re creating.|
