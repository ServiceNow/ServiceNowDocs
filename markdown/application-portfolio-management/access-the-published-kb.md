# Access the published TRM catalog knowledge base

Access the published TRM catalog to view approved technology standards and lifecycle information in the knowledge base.

## Before you begin

Role required: sn\_apm.apm\_user

## About this task

When a TRM catalog is published, it becomes available as knowledge base content. You must have access to the knowledge base where the catalog is published. It depends on the catalog access type. If it is an internal catalog, it requires a sign in. If it is a public catalog, it can be accessed without signing in. The public catalogs can be exposed to anonymous users and can be linked from internet-facing sites depending on portal setup.

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Setup page by selecting the Setup icon ![Setup icon](../../image/eaw-image/setup-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to **Publishing Center**.

4.  Select **All**.

5.  Open the catalog for which you want to access the knowledge base article.

6.  Select the catalog link on the **Details** tab of the Publishing Configuration page.

    ![Access the published TRM catalog knowledge base](../../image/eaw-image/access-cat-link.png)
