# Create diagram for a business capability

An Enterprise architect can create a diagram using Lucidchart for a business capability maps and associate it with an architectural artifact.

## Create diagram form fields

<table id="table_tlq_1zz_k5b"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Diagram Name

</td><td>

Name of the diagram.

</td></tr><tr><td>

Diagram Type

</td><td>

Type of the diagram. It can be either of the following:-   Business Application Hierarchy
-   Business Capability Map

 This field is auto-populated when creating the diagram from a business application or business capability view.

</td></tr><tr><td>

Include Business Applications

</td><td>

Option to include business applications.

</td></tr><tr><td>

All Business Capabilities

</td><td>

Option to include all business capabilities maps in the diagram.

</td></tr><tr><td>

Business Capabilities

</td><td>

List of business capabilities for which you want to create the diagram. You can search and add multiple capabilities.

 **Note:** If you have selected the check box for **All Business Capabilities** in the previous step, then this field does not appear.

</td></tr><tr><td>

Diagramming Tool

</td><td>

The tool using which you are creating the diagram. This field is automatically set to **Lucidchart**.

</td></tr><tr><td>

Link to Artifact

</td><td>

Artifact to which you want to associate the diagram.

 Use the following options:

-   **None**: Select to view the diagram but not associate with any artifact.
-   **New Artifact**: Select to create an artifact and associate the diagram.
-   **Existing Artifact**: Select to associate the diagram to an existing artifact.

</td></tr><tr><td>

Artifact Name

</td><td>

Name of the artifact. This field appears only when **New Artifact** or **Existing Artifact** is selected from the **Link to Artifact** field.

</td></tr><tr><td>

Folder

</td><td>

Name of the Lucid folder. Select the folder where you want to save the diagram. When you select a folder, its sub folders are displayed.

</td></tr></tbody>
