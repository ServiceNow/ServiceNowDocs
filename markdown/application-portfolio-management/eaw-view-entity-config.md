# View all entities

View all entities available for the Enterprise Modeling and Visualization in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm\_mdtl\_com.mdtl\_com\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Setup page by selecting the Setup icon \(\[Omitted image "eaw-icon-setup.png"\] Alt text: setup icon.\).

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: expand row icon.\) next to **Enterprise Modeling and Visualization**.

4.  Select **Entity Configurations**.
