# Application model lifecycle form

The application model lifecycle helps you to better manage lifecycle of a business application.

## Application model lifecycle form fields

<table id="table_mly_1jt_lxb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Model

</td><td>

Model ID of the business application. This field is auto-populated from the **Model ID** field value of the business application form.

</td></tr><tr><td>

Lifecycle type

</td><td>

Type of the lifecycle. Use the following options:-   Internal
-   Publisher

</td></tr><tr><td>

Lifecycle phase

</td><td>

Phase of the lifecycle. Use the following options:-   General Availability
-   End of sale
-   End of support
-   End of extended support
-   End of life

</td></tr><tr><td>

Source

</td><td>

Source of the lifecycle. This field is auto-generated from the business application form.

</td></tr><tr><td>

Phase start date

</td><td>

Start date of the business application lifecycle phase.

</td></tr><tr><td>

Phase end date

</td><td>

End date for the business application lifecycle phase.

</td></tr><tr><td>

Risk

</td><td>

Risk associated with the application lifecycle. Use the following options:-   Very high
-   High
-   Moderate
-   Low
-   None

</td></tr><tr><td>

Active

</td><td>

Option to activate the lifecycle.

</td></tr><tr><td>

Description

</td><td>

Short description of the application lifecycle.

</td></tr></tbody>
