# General information form

Learn about the fields of the General information form. Use this form to provide details for the new policy.

|Field|Description|
|-----|-----------|
|Name|Unique name for the policy.|
|Description|Brief description for the policy.|
|Worknotes|Enter notes for your future reference.|
|Policy type|Type of the policy. By default, this field is set to **Certification**.|
