# Exploring application services

An application service is a set of interconnected applications and hosts which are configured to offer a service to the organization. An application service is a logical grouping of application functions.

Application services enable you to structure your application landscape, specify its function, and associate it with the relevant applications. This helps you to remove redundant applications and optimize your application landscape.

Application services support business capabilities by offering specific application capabilities.
