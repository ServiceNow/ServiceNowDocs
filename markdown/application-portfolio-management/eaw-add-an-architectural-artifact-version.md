# Add an architectural artifact version

Create multiple versions of architectural artifacts in the Enterprise Architecture Workspace.

## Before you begin

Role required: Depends on the role permissions set for a particular architectural artifact.

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon ![Portfolio icon](../../image/portfolio-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to **Information Portfolio**.

4.  Select **Architectural Artifacts**.

5.  Select the architectural artifact that you want to add a new version for.

    A page appears displaying the details of the architectural artifact.

6.  Select **New Version**.

7.  On the **Create new version** form, fill in the fields.

    For a description of the field values, see [Create new architectural artifact version form](../../reference/eaw-reference/eaw-create-new-versions-form.md).

8.  Select **Create**.

    The new version of the architectural decision record is created. You’re also redirected to the newly created architectural artifact page with an updated version number. You’re also redirected to the newly created artifact version details page.
