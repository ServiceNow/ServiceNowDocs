# Delete an architectural document

Delete a draft version of an architectural document that you no longer require.

## Before you begin

Role required: admin

**Note:** Deleting an architectural record will result in the automatic deletion of all related records, such as related entities, artifact versions, and user and group permissions for the record.

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **Information Portfolio**.

4.  Select **Architectural Documents**.

5.  Select the architectural document that you want to delete.

6.  Select **Delete** from the More Actions menu.

    A warning message appears stating this action will delete all related records, such as related entities, artifact versions, and user and group permissions for the record.

7.  Select **Delete** to proceed.

    The architectural document and its related records are deleted.
