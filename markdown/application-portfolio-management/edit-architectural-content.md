# Edit artifact content of an architectural document

Edit artifact content of an architectural document.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon ![Portfolio icon](../../image/portfolio-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to **Information Portfolio**.

4.  Select **Architectural Documents**.

5.  Select an existing architectural document.

    The Architectural Content tab appears for the selected artifact document.![Artifact content](../../image/eaw-image/artifact-content.png)

6.  Make edits to the content as per your requirement.

7.  Select **Save**.
