# Create diagram form for a business application

An Enterprise architect can create a diagram using a diagramming tool for a business application hierarchy and associate it with an architectural artifact.

## Create Diagram form fields

<table id="table_tlq_1zz_k5b"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Diagramming Tool

</td><td>

The tool that you use to create the diagram.-   Enterprise Modeling and Visualization
-   Lucidchart

</td></tr><tr><td>

Diagram Name

</td><td>

Name of the diagram.**Note:** This field appears only when you select Lucidchart in the Diagramming tool field.

</td></tr><tr><td>

Link to Artifact

</td><td>

Artifact to which you want to associate the diagram.

 Use the following options:

-   **None**: Select to view the diagram but not associate with any artifact.
-   **New Artifact**: Select to create an artifact and associate the diagram.
-   **Existing Artifact**: Select to associate the diagram to an existing artifact.

</td></tr><tr><td>

Artifact Name

</td><td>

Name of the artifact. This field appears only when **New Artifact** or **Existing Artifact** is selected from the **Link to Artifact** field.

</td></tr><tr><td>

Diagram Type

</td><td>

Type of the diagram. It can be either of the following:-   Business Application Hierarchy
-   Business Capability Map

This field is auto-populated when creating the diagram from a business application or business capability view.

</td></tr><tr><td>

Business Application

</td><td>

Name of the Business Application for which you are creating the diagram.

 This field is auto-populated when creating the diagram from a business application view.

</td></tr><tr><td>

Entities

</td><td>

Entities that are included in the chart.

 Select the following to include in the diagram:

-   Application Service: Server \(Hardware Model or Software Product\)
-   Business Capability
-   Demand
-   Digital Integration
-   Digital Interface
-   Information Object
-   Project

 **Note:** The Project option is available when the PPM Standard plugin installed.

</td></tr><tr><td>

Folder

</td><td>

Name of the Lucid folder. Select the folder where you want to save the diagram. When you select a folder, its sub folders are displayed.**Note:** This field appears only when you select Lucidchart in the Diagramming tool field.

</td></tr></tbody>
