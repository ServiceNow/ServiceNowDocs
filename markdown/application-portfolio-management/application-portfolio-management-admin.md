# Enterprise Architecture administration

With the sn\_apm.apm\_admin role, you can classify the applications used in the business enterprise. You can also provide privileges to users to do specific tasks, set up indicators to assess application usability, and create bubble charts to help define strategies to maintain applications.

With administrative privileges, you can set up application classification attributes to group applications, users and roles to provide level of access and assign tasks, applications assessment indicators to assess applications, and bubble charts to recommend an application strategy. Such a setup helps you to classify your applications and maintain an inventory, provide privileges to users to do specific tasks, assess the application usability by its indicators, and recommend a strategy to plan and execute organization goals.
