# Configure indicators

Configure application or capability indicators in the Enterprise Architecture Workspace.

You can add a new application or capability indicator or edit existing indicators.

-   **[Add or edit an application indicator](../../task/eaw-task/eaw-create-indicator.md)**  
Add or edit an application indicator to assess the application across dimensions such as cost, quality, technical risk, investments, user satisfaction, and business value.
-   **[Add or edit a capability indicator](../../task/eaw-task/eaw-add-or-edit-a-capability-indicator.md)**  
Add or edit a capability indicator to access business capabilities within the indicator framework.
-   **[Activate or turn off an application or capability indicator](../../task/eaw-task/eaw-enable-or-disable-an-application-indicator.md)**  
Enable or disable application indicators according to your business requirements.
