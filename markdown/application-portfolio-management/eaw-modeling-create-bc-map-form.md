# Create a business capability map form

Creating artifacts for the business capability hierarchy map helps you to update the capability hierarchy directly in the map using Enterprise Modeling and Visualization.

## Create a business capability map form fields

<table id="table_dbj_mky_xcc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Link to artifact

</td><td>

Artifact to which you want to associate the diagram. Use the following options:-   **New artifact**-Select to create an artifact and associate the diagram.
-   **Existing artifact**-Select to associate the diagram to an existing artifact.

</td></tr><tr><td>

Artifact name

</td><td>

Name of the artifact. Enter a name of the new artifact or select an existing artifact name based on the selection in the **Link to artifact** field.

</td></tr><tr><td>

Architectural category

</td><td>

Name of the category for the artifact.

</td></tr><tr><td>

Include business application

</td><td>

Select this check box to include business applications of a capability in the hierarchy map.

</td></tr><tr><td>

Include all business capabilities

</td><td>

Select this check box to include all the business capabilities in the hierarchy map.

</td></tr><tr><td>

Business capability

</td><td>

Name of the business capabilities for which you want to generate the hierarchy map. You can level-0 capabilities from the list.**Note:** This field does not appear when you select the check box for the field **Include all business capabilities**.

</td></tr></tbody>
