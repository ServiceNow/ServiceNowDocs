# Run certification for a policy

Run certification for published policies to generate a new certification instance for an active policy.

## Before you begin

\[Omitted video\] Description: Run certification for a policy in Enterprise Architecture Workspace

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Data certification page by selecting the Data certification icon \(\[Omitted image "icon-data-cert.png"\] Alt text: data certification icon\).

    The Data certification page opens.

3.  Select the check box next to the policy that you want to run certification for.

4.  Select **Run certification** from the context menu or use the **Run certification** button on the page.

    \[Omitted image "eaw-data-cert-run-cert.png"\] Alt text: Run certification

5.  Select **Run**.

    A confirmation message appears with a link to track the progress.\[Omitted image "eaw-data-cert-track-status.png"\] Alt text: Link to track certification status

6.  Select the Track progress link or select the Track progress tab to see the status.
