# Apply filters on the Application Rationalization page

You can apply filters on the Application Rationalization page in the Enterprise Architecture Workspace. Based on the applied filters, the bubble chart and list view are updated so that you can view data for a specific business application. Your filter preferences are saved and applied the next time you visit the page.

## Before you begin

Role required: sn\_apm.apm\_read

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Application Rationalization page by selecting the application rationalization icon \(\[Omitted image "icon-app-rationalization.png"\] Alt text: Application rationalization icon.\).

3.  On the Bubble chart or List view page, select the Application rationalization filter icon \(\[Omitted image "icon-filter.png"\] Alt text: Portfolio overview and health filters icon.\).

    The Application rationalization filter pop-up window appears.

    \[Omitted image "application-rationalization-filter.png"\] Alt text: Application rationalization filter pop-up window.

4.  To filter business applications based on specific data, select any of the following tabs:

    -   **Business applications**
    -   **Indicators**
    -   **Business capabilities**
    You can add filters on all the three tabs simultaneously.

5.  Add conditions to filter the business applications.

    You can add multiple conditions.

    \[Omitted image "application-rationalization-filter-dropdown.png"\] Alt text: Application rationalization filter pop-up window with field drop-down displayed.

6.  Select **Update**.

    The bubble chart and the list view pages are updated based on your applied filters. Also, an indicator is displayed on top of the filter icon to show the number of filters applied.

    **Note:**

    -   You can also clear all applied filters by selecting the **Clear all filters** button.
    -   The List view page displays all business applications, whereas the Bubble chart page displays only assessed applications. As a result, filter results may differ between the two pages.
    -   When you apply a business capability-based filter, business applications associated with the selected capability and also its child capabilities are displayed.
