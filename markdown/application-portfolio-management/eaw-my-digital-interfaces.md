# View your digital interfaces

View the list of all digital interfaces that are created by you in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by clicking the Portfolio icon ![Portfolio icon](../../image/portfolio-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to **My Entities**.

4.  Select **My Digital Interfaces**.
