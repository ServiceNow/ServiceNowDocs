# Storing shapes or images to the database

Upload the images to the database to configure them to show up in the diagrams page of the Enterprise Modeling and Visualization.

## Before you begin

Role required: System administrator

## About this task

Every shape in Enterprise Modeling and Visualization is associated with an image to represent the shape. Users with System administrators role can upload and store images in the database. These images are saved in the Images \[db\_image\] table.

## Procedure

1.  Navigate to **All** &gt; **System UI** &gt; **Images **.

2.  Select **New**

3.  Fill in the details in the new image form.

    For more details, see [New image form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-portfolio-management/enterprise-architecture/eaw-modeling-image-form.md).
