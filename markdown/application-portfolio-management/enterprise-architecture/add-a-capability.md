# Add a business capability

Add a business capability to the capability hierarchy map.

## Before you begin

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Business Capability Hierarchy page by selecting the Business Portfolio icon \(\[Omitted image "business-portfolio-icon.png"\] Alt text: Business Portfolio icon\).

3.  Select **New Capability**.

4.  On the form, fill in the fields.

    For a description of field values, see [Business capability new record form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-portfolio-management/enterprise-architecture/business-capability-new-record-form.md).

5.  Select **Create**.

6.  Create a child capability for the capability that you created.

    1.  Right-click on the hierarchy ID of the capability.

    2.  Select **Create sub-capability**.
