# View all architectural artifacts

You can view the list of all architectural artifacts in the Enterprise Architecture Workspace.

## Before you begin

The permission set at the artifact level also determines if you can view the artifact.

Role required: sn\_apm.apm\_user or sn\_apm.apm\_admin

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **Information Portfolio**.

4.  Select **Architectural Artifacts**.
