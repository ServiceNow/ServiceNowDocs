# View your TRM products

View all your Technology Reference Model \(TRM\) products in the Enterprise Architecture Workspace. TRM allows you to define the standards for your software and hardware products and manage unapproved products in your organization.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by clicking the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **My Entities**.

4.  Select **My Technology Reference Model Products**.
