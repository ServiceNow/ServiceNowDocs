# View all TRM products

You can view all available Technology Reference Model \(TRM\) products in the Enterprise Architecture Workspace.

## Before you begin

You can add or hide the displayed columns according to your requirement by selecting the **List Actions** icon \(\[Omitted image "list-actions-icon.png"\] Alt text: List actions icon.\) and then selecting **Edit columns**.

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon \[Omitted image "technology-portfolio-icon.png"\] Alt text: Technology portfolio icon.

3.  In the **TRM catalog** tab, select **TRM product list** from the View list.

4.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to the TRM product next to a TRM product to view more details.
