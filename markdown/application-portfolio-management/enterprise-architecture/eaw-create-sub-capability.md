# Create a sub-capability

Add a sub-capability to the capability hierarchy map.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Business Capability Hierarchy page by selecting the Business Portfolio icon \(\[Omitted image "business-portfolio-icon.png"\] Alt text: Business Portfolio icon.\).

3.  Select the row context menu icon \(\[Omitted image "icon-menu.png"\] Alt text: Row context menu icon.\) and select **Create sub-capability**.

4.  On the form, fill in the fields.

    For a description of the field values, see [Create a sub-capability form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-portfolio-management/enterprise-architecture/eaw-create-a-sub-capability.md).

5.  Select **Create**.
