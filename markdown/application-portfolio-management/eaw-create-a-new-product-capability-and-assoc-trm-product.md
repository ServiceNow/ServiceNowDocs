# Create product capabilities and associate it with a Technology Reference Model product

You can create product capabilities and associate it with Technology Reference Model \(TRM\) products in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Technology Portfolio page by selecting the Technology Portfolio icon \[Omitted image "technology-portfolio-icon.png"\] Alt text: Technology portfolio icon.

3.  In the **TRM catalog** tab, select **TRM product list** from the View list.\[Omitted image "trm-product-view-dropdown.png"\] Alt text: TRM view list drop-down list.

4.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to the TRM product that you want to create and associate new product capabilities to.

5.  Select the TRM product.

6.  Select **Product capabilities**.

7.  To create a new product capability and associate with the TRM product, select **New**.

8.  In the Create New Product Capability page, fill in the form fields.

    For field information, see [Create product capability form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-portfolio-management/eaw-create-new-product-capability-form.md).

9.  Select **Save**.\[Omitted image "trm-product-capability-related-list-items.png"\] Alt text: Product Capability Maps and Capability Relationship Maps tabs associated with a product capability.


## Result

The product capability is added to the TRM product.
