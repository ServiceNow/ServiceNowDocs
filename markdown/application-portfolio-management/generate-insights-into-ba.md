# Generate insights into business applications

Learn the details of business applications and gather concise and actionable insights on business applications from the summary that is generated by Now Assist for Enterprise Architecture \(EA\).

## Before you begin

Make sure that the Business Application Insights skill is activated. For information, see [Configure Now Assist for Enterprise Architecture \(EA\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/application-portfolio-management/configure-now-assist-ea.md).

**Note:** The Architectural decision record \(ADR\) feature in Enterprise Architecture Workspace uses the ServiceNow Docs component \(sn\_docs\) to create pages in the Artifacts section. The Docs component v6.0.0 is automatically installed with Enterprise Architecture Workspace v3.4.0.

If you’re using an older version of Enterprise Architecture Workspace with Docs component v6.0.0, upgrade the workspace to v3.4.0 to fully use the ADR functionality. For more information, see [KB2017926](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB2017926).

You can use Now LLM Service, Now LLM Long Term Stable models \(LTS\), Azure OpenAI, Google Gemini or Anthropic Claude on AWS as the AI model provider for all Now Assist skills and AI agents. Use the Configuration Controls in [AI Control Tower](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/ai-control-tower/ai-model-providers.md) to define which options are available, then set the skill-level preferences in the [Now Assist Admin console](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/enable-ai-experiences/manage-large-language-models.md). For more information, see [Large language models on the ServiceNow AI Platform®](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/servicenow-large-language-model-now-llm/exploring-large-language-models.md).

Role required: sn\_apm.apm\_user

## About this task

The business application insights help you to make informed decisions without manually reviewing large volumes of data. Help minimize the time that you spend to learn about critical details of business applications like purpose, risk, cost, and recommendations for rationalization.

The business application summary and insights are generated based on certain related fields and related lists. Some examples follow:

-   Fields: Name, Description, Business owner, Install type, Operational status, Platform, IT Application owner, Application family, Application type, Application category, Architecture type, Business criticality, Business Unit, Department, Life Cycle Stage, Life Cycle Stage Status, Publisher, and so on.
-   Related lists: Digital Integration, Digital Interface, CI Scores, Total cost of ownership, and Information Objects.

You can also provide feedback on the business application insights using the thumbs up or thumbs down button. The feedback helps to notify whether the generated business application insights are useful.

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text:.

3.  Select the expand row icon \[Omitted image "ExpandIcon.png"\] Alt text: next to **Application Portfolio**.

4.  Select **Business Applications**.

5.  Select the business application for which you want to generate insights.

6.  Select **Generate insights**.

    \[Omitted image "ba-generate-insights-button.png"\] Alt text: Screenshot showing the business application Generate insights button highlighted.

    The insights for the business application are generated.

    \[Omitted image "ba-summary.png"\] Alt text: Business application insights generated by Agent assist.

7.  Generate a new version of the business application insights by selecting the Refresh icon \[Omitted image "refresh-icon.png"\] Alt text:.

    \[Omitted image "ba-summary-refresh-button-highlighted.png"\] Alt text: Refresh icon highlighted in the Business application insights window.

    **Note:** If you updated the business application details, the **Refresh Summary** button becomes available.

    \[Omitted image "ba-refresh-summary.png"\] Alt text: Screenshot showing the Refresh Summary button available after business application details are updated.
