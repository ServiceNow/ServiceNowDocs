# Elaborate or shorten content in form fields

Elaborate or shorten text in the description or reasoning fields in forms using Now Assist in the Enterprise Architecture Workspace. You can also use prompts to generate content for the **Reasoning** field found in **Planned Disposition** section of a business application form.

## Before you begin

**Note:** Depending on your license, you will have access to certain application features, generative AI skills, agentic workflows, and AI agents. For more information, see ServiceNow product tiers.

**Important:** This Now Assist skill is turned on by default. The skill will be automatically available to appropriate role users for the application. For more information, see Now Assist skills, agents, and agentic workflows on by default.

Role required: sn\_apm.apm\_user

## About this task

Description and reasoning fields often require clear, concise, and contextually accurate information. You can use Now Assist context menu to streamline the process and ensure that the captured information in the fields is comprehensive and easy to understand. The refine text skill is available in the forms of the following entities:

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon ![Portfolio icon](../../application-portfolio-management/image/portfolio-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../application-portfolio-management/image/ExpandIcon.png)\) next to **Application Portfolio**.

4.  Select **Business Applications**.

    **Note:**

    -   For business capability, select **Business Architecture** &gt; **Business Capability**.
    -   For business process, select **Business Architecture** &gt; **Business Processes**.
    -   For value stream stage, select **Business Architecture** &gt; **Value Stream Stages**.
    -   For information object, select **Information Portfolio** &gt; **Information Objects**.
5.  Select the business application whose content you want to elaborate, shorten, or generate.

6.  Select the **Description** field to perform any of the following tasks.

    -   **Elaborate**: Elaborates the selected text.
    -   **Shorten**: Shortens the selected text.
    ![Elaborate or shorten description content.](../image/desc-field-nacm.png)

7.  Select the **Reasoning** field to perform any of the following tasks.

    -   Text box: Use the text box to enter any prompts and derive the required information from the existing business application content.
    -   **Elaborate**: Elaborates the selected text.
    -   **Shorten**: Shortens the selected text.
8.  Select **Save**.

    The business application data is saved.
