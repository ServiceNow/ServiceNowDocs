# Data classification group form

The data classification groups help admin users to categorize the classification tags of an information object.

## Data classification group form fields

|Field|Description|
|-----|-----------|
|Name|Name of the data classification group.|
|Description|Short description of the data classification group.|
|Allow Multi Selection|Option to enable multiple data classifications selection.|
|Application|Application name for which you’re creating the data classification. This field is auto-populated based on the application scope set in the instance.|
