# Add or edit your application services

Add or edit an application service created by you in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by clicking the Portfolio icon ![Portfolio icon](../../image/portfolio-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to **My Entities**.

4.  Select **My Application Services**.

5.  Add or edit an application service.

    -   To add an application service, select **Add**.
    -   To update details of an existing application service, select the record then select **Edit**.
6.  On the form, fill in the fields.

    For field information, see [Application service form](../../reference/eaw-reference/eaw-app-service-form.md).

7.  Select **Save** or **Update**.
