# Add or edit a certification policy

Add or edit a certification policy in the Enterprise Architecture Workspace.

## Before you begin

Role required: certification\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Setup page by selecting the Setup icon \(![setup icon.](../../image/eaw-image/eaw-icon-setup.png)\).

3.  Select the expand row icon \(![expand row icon.](../../image/ExpandIcon.png)\) next to **Certification Policies**.

4.  Select **All**.

5.  Create or update a CMDB Data Management Certification Policy.

    -   To create a certification policy, select **New**.
    -   To update an existing certification policy, select a certification policy, then select **Edit**.
6.  On the CMDB Data Management Certification Policy form, fill in the fields.

    For field information, see [Certification policy form in Enterprise Architecture Workspace](../../reference/eaw-reference/eaw-certification-policy-form.md).

7.  Select **Save**.
