# View all AI dataset digital assets

You can view the list of all available AI dataset digital assets in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon \[Omitted image "portfolio-icon.png"\] Alt text: Portfolio icon.

3.  Select the expand row icon \(\[Omitted image "ExpandIcon.png"\] Alt text: Expand Row icon\) next to **AI Portfolio**.

4.  Select **AI Dataset Digital Asset**.


## Result

A list of AI dataset digital assets is displayed.
