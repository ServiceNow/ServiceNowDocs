# Activate a certification policy

Activate a certification policy to add it to the active certification runs. This process ensures that the policy is active and can be used for certifications.

## Before you begin

Role required: sn\_cmdb\_admin + sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Data certification page by selecting the Data certification icon \(\[Omitted image "icon-data-cert.png"\] Alt text: data certification icon\).

    The Data certification page opens.

3.  Use one of the following options to activate an inactive certification policy.

    -   Select the check box next to the policy that you want to activate and then select **Activate** from the context menu.
    \[Omitted image "eaw-data-cert-activate-context.png"\] Alt text: Activate a policy from the context menu

    -   Open the policy by selecting the link and select the **Activate Policy** button.
    \[Omitted image "eaw-data-cert-activate-record.png"\] Alt text: Activate a policy from the policy record page

4.  Select **Activate Policy**.
