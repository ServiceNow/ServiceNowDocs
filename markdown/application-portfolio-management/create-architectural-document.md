# Create a new architectural document

Create an architectural document in the Enterprise Architecture Workspace. An Architectural document captures and governs architecture information generated from diagrams.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon ![Portfolio icon](../../image/portfolio-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to **Information Portfolio**.

4.  Select **Architectural Documents**.

5.  Select **New**.

6.  Enter a name and select an architectural category.

    **Note:** By default, the format is set to **Document**.

7.  Select **Create**.

    An artifact gets created.

    You can see the following tabs:

    -   Artifact content
    -   Details
    -   Related entities
    ![Architectural document](../../image/eaw-image/modeling/architectural-document.png)
