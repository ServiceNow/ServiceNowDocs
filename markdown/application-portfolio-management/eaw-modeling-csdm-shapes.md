# Common Service Data Model \(CSDM\) shapes

CSDM is a prescriptive data model that defines standardized relationships between service-related objects in the CMDB. The CSDM shapes represent these objects visually in diagrams, enabling architects to model business capabilities, applications, services, and technical components in alignment with the Now Platform.

## CSDM shapes in Enterprise Modeling and Visualization

You can add a shape to the canvas by either selecting the shape or by dragging the shape from the **Shapes**

You can also adjust the size of shapes by selecting a shape and then drag any of its edges or corner handles. Pulling outward increases the shape's dimensions, while dragging inward reduces its size.

**Note:** Connector ports on CSDM shapes are bi-directional. You can use any connector port—top, bottom, left, or right—as either an input or an output when connecting shapes in a diagram.
