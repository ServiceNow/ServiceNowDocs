# Configure categories to display in Group Analysis page

Configure the categories to display in the Categories list of the Group Analysis page.

## Before you begin

Role required: sn\_apm.apm\_analyst

## About this task

Use the choices table \(sys\_choice\) to select or unselect the categories to show or hide in the Group Analysis page.

## Procedure

1.  Navigate to **All** &gt; ** System Definition** &gt; **Choice Lists**.

2.  In the Element column, search for the `group_by` element type.

    The existing group\_by choices appear.

    ![Choices by group](../image/group_by-choice.png)

3.  From the list, select the choice item that you want to show or hide in the Categories list in the Group Analysis page.

4.  Open the record to edit.

5.  Hide or show the item by selecting or unselecting the **Inactive** check box to.

6.  Click **Update**.
