# Publish a certification policy

Publish a draft policy to activate it and enable certifications to be issued under that policy.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Data certification page by selecting the Data certification icon \(![data certification icon](../../image/eaw-image/icon-data-cert.png)\).

    The Data certification page opens.

3.  Open a draft policy by selecting the policy name link.

4.  Select **Edit Policy**.

5.  Edit the policy details as required.

6.  Review all the details on the **Review** page.

7.  Select **Publish Policy**.

    **Note:** Ensure all the mandatory fields are filled in before publishing the policy.
