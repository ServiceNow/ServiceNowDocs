# Add value stream stages to a value stream

You can add value stream stages to a value stream to break it down into discrete steps and establish the foundation for associating business processes and business capabilities with each stage.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspace** &gt; **Enterprise Architecture Workspace**.

2.  Open the Portfolio List view by selecting the Portfolio icon ![Portfolio icon](../../image/portfolio-icon.png).

3.  Select the expand row icon \(![Expand Row icon](../../image/ExpandIcon.png)\) next to **Business Architecture**.

4.  Select **Value Streams**.

5.  Select the value stream that you want to add value stream stages to.

    A new page appears, displaying the details of the value stream.

6.  Select the **Value Stream Stages** tab.

7.  Select **New**.

8.  On the form, fill in the field.

    For a description of the field values, see [Create new value stream stage form](../../reference/eaw-reference/eaw-create-new-value-stream-stage-form.md).

9.  Select **Save**.
