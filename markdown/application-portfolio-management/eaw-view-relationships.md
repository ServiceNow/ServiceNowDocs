# View relationships configurations

View all relationship configurations available for the Enterprise Modeling and Visualization in the Enterprise Architecture Workspace.

## Before you begin

Role required: sn\_apm\_mdtl\_com.mdtl\_com\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Setup page by selecting the Setup icon \(![setup icon.](../../image/eaw-image/eaw-icon-setup.png)\).

3.  Select the expand row icon \(![expand row icon.](../../image/ExpandIcon.png)\) next to **Enterprise Modeling and Visualization**.

4.  Select **Relationships**.
