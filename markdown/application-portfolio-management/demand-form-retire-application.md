# Demand form to retire an application

An Enterprise Architecture \(formerly APM\) user can raise a request to retire an application if the application is no longer in use.

## Demand form fields

|Field|Description|
|-----|-----------|
|Action|Retire Application.|
|Name|Name of the business application to be retired.|
|Category|Operational.|
|Type|Project.|
