# Exploring generating documents from diagrams

Generating documents for a diagram helps you to transform modeling diagrams into governed, reusable architectural documents using predefined templates.

## Generate documents from diagrams overview

This feature connects diagram modeling, document templates, and portfolio‑level document governance into a single, traceable workflow.

Generated documents are stored as architectural documents under Information Portfolio of the Portfolio page. The documents are governed through architectural versions, lifecycle states, and related entities. You can discover, review, and navigate documents independently of the original diagram.

When you generate a diagram from the modeling page, a new architectural artifact gets created and a draft architectural version is created for that document.

Generating documents from diagrams helps you to:

-   Eliminate manual copy‑paste from diagrams
-   Ensure documents are generated from a specific diagram version and capture its associated metadata at the time of creation.
-   Provides portfolio‑level visibility of diagram‑generated documents
-   Supports review, approval, and distribution workflows
