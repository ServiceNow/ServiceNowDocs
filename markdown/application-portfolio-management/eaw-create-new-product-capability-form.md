# Create new product capability form

Use a product capability to specify the feature that a business application or Technology Reference Model product provides to support business objectives.

<table id="table_yqq_xrf_qfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the product capability.

</td></tr><tr><td>

Company

</td><td>

The company, if any, related to the product capability.

</td></tr><tr><td>

Description

</td><td>

A short description of the product capability and its features and purpose.

</td></tr><tr><td>

Type

</td><td>

Categorizes the product capability based on its technical nature. This helps to distinguish between different types of features provided by business applications or Technology Reference Model products.

 Use the following options:

-   **Feature**
-   **Capability**
-   **Technical Service**

</td></tr><tr><td>

Category

</td><td>

A broader classification that groups similar capabilities together.
