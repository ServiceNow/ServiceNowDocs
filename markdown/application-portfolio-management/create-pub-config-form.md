# Create new publishing configuration form

Specify the configuration that is used for publishing a Technology Reference Model \(TRM\) catalog to the knowledge base.

<table id="table_edl_5bz_f3c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Catalog name

</td><td>

Name of the TRM catalog.

</td></tr><tr><td>

Catalog type

</td><td>

Select the catalog type as TRM.

</td></tr><tr><td>

Access type

</td><td>

Select the type of access for the catalog. It can be either of the following:-   Internal- For internal users only. Users must sign in to access the published catalog in the Knowledge Base.
-   Public- For all users. The catalog can be configured to allow guest access to published content.

</td></tr><tr><td>

Status

</td><td>

Status of the catalog. While creating the catalog, this field is set to **Draft**.

</td></tr><tr><td>

Catalog link

</td><td>

Link to access the published TRM catalog. This field is auto-populated with a URL after the catalog is published.

</td></tr></tbody>
