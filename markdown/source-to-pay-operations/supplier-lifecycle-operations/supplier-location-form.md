# Add New Location form

Use the Add New Location form to provide details about the location of a supplier.

<table id="table_l33_pt5_msb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the supplier location.

</td></tr><tr><td>

Street

</td><td>

The street address of the supplier location.

</td></tr><tr><td>

City

</td><td>

The city of the supplier location.

</td></tr><tr><td>

State/Province

</td><td>

The state of the supplier location.

</td></tr><tr><td>

Zip/Postal Code

</td><td>

The zip code of the supplier location.

</td></tr><tr><td>

Country

</td><td>

The country of the supplier location.

</td></tr><tr><td>

Latitude

</td><td>

The latitude coordinates of the supplier location.

</td></tr><tr><td>

Longitude

</td><td>

The longitude coordinates of the supplier location.

</td></tr><tr><td>

Category

</td><td>

The category that the supplier location belongs to. The choices are:-   Contracting address
-   Delivery address
-   Facility \(default\)
-   Headquarters
-   Invoice address
-   Service center

You can select multiple values from the choice list.
