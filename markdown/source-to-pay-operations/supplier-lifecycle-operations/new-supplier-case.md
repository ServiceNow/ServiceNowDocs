# Create New Supplier Case form

Use the Create New Supplier Case form to enter supplier case details and provide resolution information.

<table id="table_xfm_xr5_ksb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td class="sub-head" colspan="2">

Supplier Case

</td></tr><tr><td>

Number

</td><td>

An auto-generated number that uniquely identifies the supplier case.

</td></tr><tr><td>

State

</td><td>

The current state of the case. The choices are:-   **Draft**
-   **Open**
-   **Work in progress**
-   **Awaiting task completion**
-   **Closed completed**
-   **Closed rejected**
-   **Closed canceled**

</td></tr><tr><td>

Case type

</td><td>

The type of case. The choices are:

-   **Supplier support request**
-   **General inquiry**
-   **Banking information change request**
-   **Supplier information change request**
-   **Supplier location change request**
-   **Supplier issue**
-   **Due diligence**
-   **Offboard a supplier**
-   **Conduct sanction screening**
-   **Credit card enablement**

</td></tr><tr><td>

Priority

</td><td>

Indicates how quickly you must complete the case based on its severity.

</td></tr><tr><td>

Supplier

</td><td>

The name of the supplier the case is for.

</td></tr><tr><td>

Assigned to

</td><td>

The name of the agent that the case is assigned to.

</td></tr><tr><td>

Requested by

</td><td>

The person who requested the case.

</td></tr><tr><td>

Due date

</td><td>

The date that the case must be completed.

</td></tr><tr><td>

Created

</td><td>

The date that the case is created.

</td></tr><tr><td>

Assignment group

</td><td>

Group to which the case is assigned to.

</td></tr><tr><td>

Channel source

</td><td>

Channel that was used to request this case; for example, email.

</td></tr><tr><td>

Watch list

</td><td>

Users who are notified when updates are made to the case.

</td></tr><tr><td>

Short description

</td><td>

Short description of the case.

</td></tr><tr><td>

Description

</td><td>

Detailed description of the case.

</td></tr><tr><td class="sub-head" colspan="2">

Summary Details

</td></tr><tr><td>

Parent case

</td><td>

Parent case, if any, from which this case has been created, typically for reference purposes.

</td></tr><tr><td>

Related location

</td><td>

Location of the supplier.

</td></tr><tr><td>

Expected start

</td><td>

Expected start date of the case.

</td></tr><tr><td>

Actual start

</td><td>

Actual start date of the case.

</td></tr><tr><td>

Actual end

</td><td>

Actual end date of the case.

</td></tr><tr><td>

Duration

</td><td>

Duration to complete the case.
