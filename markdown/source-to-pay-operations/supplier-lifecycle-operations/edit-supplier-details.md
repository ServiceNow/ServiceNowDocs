# Edit supplier details from the Source-to-Pay Workspace

Add or modify the details of an existing supplier.

## Before you begin

Role required: sn\_slm.manager, sn\_slm.owner, or sn\_slm.admin

**Note:** The sn\_slm.owner role can edit details only for the suppliers they own.

## Procedure

1.  Navigate to **All** &gt; **Supplier Lifecycle Operations** &gt; **Source-to-Pay Workspace**.

2.  Under My suppliers, select the legal name of the supplier to be edited.

    The supplier Details panel opens.

3.  Select the **Details** tab and update the fields in the Supplier and General sections as needed.

    For descriptions of the fields on the **Details** tab, see [Source-to-Pay Workspace Supplier page](../concept/supp-ws-details-page.md).

4.  Select **Save**.
