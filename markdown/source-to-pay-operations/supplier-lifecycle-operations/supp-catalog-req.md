# Raising requests from the Supplier Collaboration Portal

You can raise requests using the supplier catalog by selecting **Raise a request** in the Supplier Collaboration Portal header.

Raise a request on the Supplier Collaboration portal 

**Important:**

-   The list of suppliers under **My Company** is available from the Xanadu December 2024 release onwards.
-   If M2M mapping between supplier contact and suppliers is enabled, you can select the supplier from the **My Company** drop-down list of suppliers.

The following catalog items are available in the supplier catalog:

-   Add or remove location
-   Enroll a supplier contact
-   Remove a supplier contact
-   Ask a question
-   Submit an idea
-   Submit an issue
-   Update banking details
-   Update profile details
-   Update default supplier \(from Xanadu December 2024 release onwards\)
-   Something else

-   **[Add or remove a supplier location using the supplier catalog](../task/submit-add-remove-supploc.md)**  
Submit a request to add a new supplier location or remove an existing one to keep the supplier information up to date.
-   **[Add a supplier contact using the supplier catalog](../task/enroll-contact-catalog.md)**  
Add the supplier contacts so that they can access the Supplier Collaboration Portal and start working on the assigned tasks.
-   **[Remove a supplier contact using the supplier catalog](../task/remove-contact-catalog.md)**  
Remove a supplier contact that you no longer require.
-   **[Ask a question using the supplier catalog](../task/submit-question.md)**  
Submit a question that you need an answer for.
-   **[Submit an idea using the supplier catalog](../task/submit-idea.md)**  
Submit an idea you would like to share.
-   **[Submit an issue using the supplier catalog](../task/submit-issue.md)**  
Submit an issue that you need help with.
-   **[Update banking details using the supplier catalog](../task/update-supp-banking-info.md)**  
Keep the banking information up to date to ensure timely payments without any delays.
-   **[Update company profile using the supplier catalog](../task/update-supp-profile.md)**  
Update the company profile when the details about your company change.
-   **[Request elevated access](../task/request-elevated-access.md)**  
Submit a request to gain access to the privileges of the primary contact role.
-   **[Update default supplier](../task/update-default-supplier.md)**  
Update the default supplier when multiple suppliers are linked with your profile.
-   **[Request something else using the supplier catalog](../task/submit-general-request.md)**  
Request something else if the available catalog items in the supplier catalog do not meet your needs.
