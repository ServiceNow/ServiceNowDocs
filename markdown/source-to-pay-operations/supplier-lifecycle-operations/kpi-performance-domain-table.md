# Performance Domain table

The Performance Domain \[sn\_kpi\_domain\] table stores the information about the performance domains created for the KPIs.

## Performance Domain \[sn\_kpi\_domain\] table

The Performance Domain \[sn\_kpi\_domain\] table contains the following fields.

|Field|Data type|Description|
|-----|---------|-----------|
|Domain|String|Name of the KPI performance domain.|
|Description|String|Brief description of the KPI performance domain.|
|Active|Boolean|Indicates whether the KPI performance domain record is active. This option is enabled by default.|
