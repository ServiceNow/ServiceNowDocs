# Create new task form

Use the Create new task form to add a new task to a supplier case.

<table id="table_gnl_4vt_2bc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Who is this task for?

</td><td>

Option to select who this task is intended for. The options are:-   **Supplier**
-   **An employee at my company**

</td></tr><tr><td>

Action type

</td><td>

Action type that defines the supplier task. The options for a supplier task are:-   **Complete a form**
-   **Complete checklist**
-   **Complete survey**
-   **Mark complete**
-   **Play video**
-   **Sign document**
-   **Upload documents**
-   **View link**

Action type that defines the employee task. The options for an employee task are:

-   **Mark complete**
-   **Play video**
-   **Upload documents**
-   **View link**

</td></tr></tbody>
</table>-   **[Task form](slo-task-form.md)**  
Use the **Details** tab of the task form to provide details about the supplier or employee task.
