# Vendor Contact table

The Vendor Contact \[vm\_vdr\_contact\] table stores important information about a vendor contact.

## Vendor Contact \[vm\_vdr\_contact\] table

The Vendor Contact \[vm\_vdr\_contact\] table contains the following fields.

|Field|Data type|Description|
|-----|---------|-----------|
|First name|String|First name of the vendor contact.|
|Last name|String|Last name of the vendor contact.|
|Vendor|Reference|The name of the vendor the vendor contact belongs to.|
|Email|Email|Email address of the vendor contact.|
|Primary contact|Boolean|Indicates whether the vendor contact is a primary contact.|
