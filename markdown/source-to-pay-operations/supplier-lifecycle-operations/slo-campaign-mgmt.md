# Smart Assessments

Supplier managers can configure smart assessments in bulk for internal and external users.

**Important:** Check your entitlements to determine whether you have access to Smart Assessments. This feature is available from the March 26 Yokohama release onwards.

Supplier managers use the segmentation rules and assessment templates to create smart assessments in bulk for users. Smart assessments provide a survey-like experience with enhanced UI capabilities. This feature utilizes the capabilities of the Smart Assessment Engine application. Assessments can be created for internal and external users:

-   Internal users \(supplier owners/managers\) respond to the assessment through the Source-to-Pay Workspace.
-   External users \(supplier contacts\) receive emails with a link to the assessment and respond through the Supplier Collaboration Portal.

**Note:** Users can reassign the assessments based on the reassignment configuration. For more information, see [Configure user criteria for reassigning assessments](configure-user-criteria-for-reassigning-assessments.md).

For more information about how to configure smart assessments, see [Configure smart assessments](configure-smart-assessments.md).
