# Update banking details form

Use the Update banking details form to update the banking details of the supplier.

<table id="table_vgg_jk4_psb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Bank Name

</td><td>

Name of the bank.

</td></tr><tr><td>

Primary account

</td><td>

Option that indicates whether this account is the primary account.**Note:** A supplier can have only one primary payment record at a given time.

</td></tr><tr><td>

Beneficiary name

</td><td>

Name of the beneficiary.

</td></tr><tr><td>

Account number

</td><td>

Account number of the beneficiary.

</td></tr><tr><td>

Country

</td><td>

Country the bank is located in.

</td></tr><tr><td>

BSB code \(For Australia\)

</td><td>

Bank State Branch is a six-digit number that is used to identify a bank code and its associated branch in Australia.This field becomes available only if you select **Australia** in the **Country** field.

</td></tr><tr><td>

IFSC code \(For India\)

</td><td>

Indian Financial System Code \(IFSC\) is a unique 11-digit alphanumeric code that is used for online fund transfer transactions in India.This field becomes available only if you select **India** in the **Country** field.

</td></tr><tr><td>

Currency

</td><td>

Currency of the country.

</td></tr><tr><td>

Routing information - Wire/ACH \(For US\)

</td><td>

A unique, nine-digit number used to identify banks and financial institutions.

</td></tr><tr><td>

IBAN

</td><td>

International Bank Account Number used for international payments.

</td></tr><tr><td>

SWIFT code \(BIC\)

</td><td>

Bank Identifier Code \(also known as SWIFT code\) of the bank.

</td></tr><tr><td>

Bank account proof

</td><td>

Option to upload the bank account proof document. Select **Required - Upload** to upload the document.

</td></tr></tbody>
