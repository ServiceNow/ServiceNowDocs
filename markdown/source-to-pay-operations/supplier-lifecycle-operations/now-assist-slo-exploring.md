# Exploring Now Assist for Supplier Lifecycle Operations \(SLO\)

With the Now Assist for Supplier Lifecycle Operations \(SLO\) application, supplier managers can summarize the details of supplier-related cases to keep them informed about their progress and action items.

## Overview of Now Assist for SLO

Using the generative AI capabilities, a supplier manager can generate a summary of a case. This summary provides a supplier manager with a concise overview of the case, actions completed so far, and the next steps that need to be taken. A supplier manager gets the context by gathering information about a case and acting quickly​. Supplier managers can also refresh and post the summary to the work notes or activity stream.

|Now Assist for SLO feature|Skill|
|--------------------------|-----|
|Case summarization|Supplier case summarization|

## Skills

The Now Assist for SLO application includes generative AI skills and features that enable supplier managers to summarize the supplier-related cases within the Source-to-Pay Workspace.

-   **Case summarization**

    Provides a supplier manager with a summary of the supplier-related cases. A supplier manager can see the status of any case in the supply process and take the necessary actions quickly.

    For more information about how to summarize a case, see [Summarize a case by using Now Assist for Supplier Lifecycle Operations \(SLO\) in Source-to-Pay Workspace](../task/now-assist-slo-summarize-case.md).
