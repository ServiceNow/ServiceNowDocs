# Third Party Unit staging table

The Third Party Unit \[sn\_spend\_intg\_third\_party\_uom\] staging table temporarily stores important data on third party units before this data is sent to the primary table.

|Field|Data type|Description|
|-----|---------|-----------|
|Provider|Reference|Reference to the Third Party Registration table.|
|Third party unit|String|Name of the third-party unit.|
|Unit|Reference|Reference to the Unit table.|
