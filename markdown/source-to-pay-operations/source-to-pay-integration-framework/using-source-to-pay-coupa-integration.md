# Use Source-to-Pay integration with Coupa

Use the Source-to-Pay integration with Coupa to process employee information at ServiceNow.

-   **[Trigger a flow or subflow in Coupa](../task/manually-trigger-subflows-coupa.md)**  
Manually trigger a flow or subflow in Coupa.
-   **[View the execution status of a flow](../task/view-executed-flow-status.md)**  
View the status of a flow that has been triggered. If there were any errors during execution, you can view them under the processing message column.
-   **[Use a flow or subflow in Coupa \(outbound\)](../task/use-flow-or-subflow-coupa.md)**  
A flow or subflow can be executed in Coupa using the Workflow Studio.
-   **[Copy a flow or subflow in Coupa](../task/copy-flow-or-subflow-coupa.md)**  
Create a copy of a flow or subflow, make the necessary modifications, and then activate it.
