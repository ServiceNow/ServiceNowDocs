# Using Source-to-Pay integration with Oracle EBS

Understand how you can use Source-to-Pay integration with Oracle EBS to handle sales orders, procurement, finance, and so on in Oracle EBS from your ServiceNow instance.

-   **[Use schedule flows in Oracle EBS](using-schedule-flows-oracle-ebs.md)**  
Use the schedule flows to retrieve information from Oracle EBS, including invoices, cost centers, product models, payment terms, purchasing organizations, departments, GL accounts, currencies, FX rates, invoice payment details, suppliers, plant addresses, and legal entities.
