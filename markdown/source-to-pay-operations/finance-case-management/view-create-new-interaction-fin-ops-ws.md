# Create a new interaction

An interaction record is automatically created when you accept a chat through the chat queue in the Finance Operations Workspace. You can also manually create an interaction from the list page of the Finance Operations Workspace.

## Before you begin

Role required: sn\_fin\_ops.specialist

## Procedure

1.  Navigate to either of the following:

    -   **All** &gt; **Finance Operations** &gt; **Finance Operations Workspace**.
    -   **Workspaces** &gt; **Finance Operations Workspace**.
2.  In the Finance Operations workspace, select the List icon ![](../image/fin-ops-list-icon.png).

3.  From the list page, navigate to either of the following submodules:

    -   **Interactions** &gt; **All**
    -   **Interactions** &gt; **Active**
    -   **Interactions** &gt; **Assigned to me**
4.  Select **New**.

5.  On the form, fill in the fields.

    For more information on the form fields and descriptions, see [Interaction form](../reference/details-tab-fin-ops-interaction.md)

6.  Select **Save**.
