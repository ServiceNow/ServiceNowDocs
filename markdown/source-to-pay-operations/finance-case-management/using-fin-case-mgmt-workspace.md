# Use the Finance Operations Workspace

Finance Specialists can use the Finance Operations Workspace to manage cases, tasks, knowledge articles, or interactions.

-   **[Create a finance case](../task/create-case-from-ws.md)**  
Create a finance case on behalf of a requester to track the progress.
-   **[Assign a finance case to yourself](../task/assign-fin-ops-case-to-yourself.md)**  
As a finance specialist, assign a finance case to yourself from the Finance Operations Workspace. Start resolving it.
-   **[Accept a finance case](../task/accept-fin-case.md)**  
As a finance specialist, if a finance case is assigned to you, accept the case to start resolving it.
-   **[Request additional information for a finance case](../task/req-info-for-fin-inquiry-case.md)**  
Request additional information from the requester to continue working on the finance inquiry case.
-   **[Search for finance cases](../task/search-for-fin-cases.md)**  
As a finance specialist, search for finance cases from the Finance Operations Workspace.
-   **[Initiate sidebar discussions](../task/initiate-sidebar-discussions.md)**  
Start a sidebar discussion to collaborate with stakeholders while working on a finance inquiry case.
-   **[Create a finance task](../task/create-fin-task.md)**  
Create a finance task and associate it to a finance case.
-   **[Create a finance task from a finance case](../task/create-fin-case-task.md)**  
Create a finance task from a finance case. Assign the task to a task owner or group to resolve that case.
-   **[Close a finance case](../task/close-fin-case.md)**  
Close a finance case after all the activities and tasks for resolving the case are complete.
-   **[Create a knowledge base article](../task/create-kb-fcm.md)**  
Create knowledge base articles for finance-related topics.
