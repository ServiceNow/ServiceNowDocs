# Exploring Now Assist for Accounts Payable Operations \(APO\)

With the Now Assist for Accounts Payable Operations \(APO\) application, an accounts payable \(AP\) fulfiller can use generative AI to summarize the details of a case to quickly get the context of the case.

## Now Assist for APOoverview

Using the summarization capability of the Now Assist for APO application, an AP fulfiller can get a concise summary of a case quickly.

A case summary enables an AP fulfiller to gather the context on pending cases or complex cases. These cases can contain many details, including the conversations with the customers or other fulfillers.

## Skills

The Now Assist for APO application includes the generative AI skills that enable an AP fulfiller to understand the context of an invoice case or inquiry case so that they can propose resolutions to the customer more quickly.

-   **Case summarization**

    Provides an AP fulfiller with a summary of an invoice case, including the issue, the actions completed, the status of the case, and the next steps to resolve the case. An AP fulfiller can generate a summary of a case to understand the case context, refresh the summary so that it includes the latest updates to the case, and post the summary to the case work notes.

    The case summarization skill generates a case summary and displays it above the activity stream.

    For invoice processing cases, the case summarization skill generates a case summary by checking the invoice status and the invoice information in the document.

    For inquiry cases, the case summarization skill generates a case summary by checking the short description, description, and tasks that were created for the case.
