# Configure purchase order line mapping

The purchase order line-mapping skill generates the invoice line mapping, changes the invoice line status to review mapping state and provokes the fulfiller to manually review the mapping results.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Now Assist Admin** &gt; **Skills** to access the **Now Assist skills** tab of the Now Assist Admin console.

2.  Select &gt; **Finance and Supply Chain** &gt; **Accounts Payable Operations** &gt; **Purchase order line mapping**.

3.  On the Purchase order line mapping summarization for the APO specialists feature card, select **Turn on**.

    The generative AI skill auto-generates mappings between invoice lines and purchase order lines.
