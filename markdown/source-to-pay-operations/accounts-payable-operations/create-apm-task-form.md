# Create New Invoice task form

Use the Create New Invoice task form to provide details about the Invoice task.

<table id="table_yfx_dwy_lvb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

An auto-generated number that uniquely identifies the task.

</td></tr><tr><td>

Parent Case

</td><td>

Parent case to associate the inquiry task or the exception task with.

</td></tr><tr><td>

Invoice exception

</td><td>

Invoice exception for an invoice.**Note:** This field is displayed only when you create an exception task from within an invoice exception.

</td></tr><tr><td>

Type

</td><td>

This field is set to Exception task by default. This field is read-only.**Note:** This field is displayed only when you create an exception task from within an invoice exception.

</td></tr><tr><td>

State

</td><td>

The current state of the task. At task creation, the State defaults to New.The available task states are:

-   New
-   Assigned
-   Work In progress
-   Closed complete
-   Closed incomplete

</td></tr><tr><td>

Assignment group

</td><td>

Group that you want to assign this task to.

</td></tr><tr><td>

Assigned to

</td><td>

Person that you want to assign this task to.

</td></tr><tr><td>

Short description

</td><td>

Brief description about the task.

</td></tr><tr><td>

Description

</td><td>

Detailed description about the task.
