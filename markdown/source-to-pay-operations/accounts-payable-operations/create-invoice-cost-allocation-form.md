# Create invoice cost allocation form

Field descriptions for the Create invoice cost allocation form used to distribute invoice line costs across cost centers and ledger accounts.

<table id="id_h2d_kbp_vcc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

**Invoice cost allocation** tab

</td><td>

Select the following fields.

</td></tr><tr><td>

Allocation type

</td><td>

Type of cost distribution \(cost center or ledger account\).

</td></tr><tr><td>

Invoice line

</td><td>

Cost allocated to the invoice line

</td></tr><tr><td>

Allocate by

</td><td>

Determines if the cost allocation is made by the following:-   Quantity
-   Quantity percentage
-   Amount
-   Amount percentage

</td></tr><tr><td colspan="2">

**Allocation details**

</td></tr><tr><td>

Cost center

</td><td>

Cost center that incurs the cost of the allocated transaction amount.

</td></tr><tr><td>

Allocation amount

</td><td>

Amount of the cost allocated.

</td></tr><tr><td>

Allocation percentage

</td><td>

Percentage of the cost allocated.This field is visible only if the allocation by is set to Percentage

</td></tr><tr><td>

Allocation quantity

</td><td>

The quantity of goods/services allocated.

</td></tr><tr><td>

Ledger account

</td><td>

Ledger account that incurs the cost of the allocated transaction amount.
