# Configure properties in Finance Common Architecture

Configure properties for various settings used in Sourcing and Procurement Operations application.

## Before you begin

Role required: sn\_fin.finance\_admin

## Procedure

1.  Navigate to **Finance Common** &gt; **Properties**.

2.  Open a property to edit or select **New** to create one.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Unique name of the property.|
    |Description|Short description of the property.|
    |Application|Name of the application to which the property applies.|
    |Value|Value for the property.|

4.  Select **Submit**.
