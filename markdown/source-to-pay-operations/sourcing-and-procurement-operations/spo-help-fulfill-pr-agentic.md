# Help fulfill procurement requests agentic workflow

The Help fulfill procurement requests agentic workflow addresses your procurement needs by providing product recommendations, guided checkout, off-catalog processes, and detailed product information. It also answers questions and tracks related records.

## Help fulfill procurement requests agentic workflow overview

The Help fulfill procurement requests agentic workflow handles and resolves procurement-related inquiries, which include the product information, purchase intent, sourcing new items, or tracking purchasing records.

## Accessing the Help fulfill procurement requests agentic workflow

To access the agentic workflow:

1.  Navigate to **All** &gt; **AI Agent Studio** &gt; **Create and manage**.
2.  Select **Help fulfill procurement requests**.

## Available AI agents in the Help fulfill procurement requests agentic workflow

The following table lists the agents that are used in the Help fulfill procurement requests agentic workflow.

**Important:** In the Define availability screen for the AI agent, make sure that the Status toggle is enabled to activate the AI agent.

|AI agent|AI agent role|
|--------|-------------|
|Procurement inquiry analysis AI agent|Retrieves and displays relevant knowledge articles to address sourcing and procurement queries. If no relevant articles are found, or if intent is unclear, redirects you to a live agent.|
|Track request agent|Provides a summary of either a sourcing request or purchase requisition by using a record number. This agent can also answer any questions that are related to a record or tracking a request as well.|
|Product recommendation and procurement agent|Assists a user to discover products to procure or purchase. After a product is selected, a user can proceed with procurement by initiating the checkout and raising a purchase request. The agent also supports scenarios where a user is looking only for information about a specific product.|

The AI agent decision log displays the AI agents that are working on a request. You can watch their interactions, decisions, and thought processes as they happen in real time.
