# Category analytics

Within the Source-to-Pay Workspace, the Category analytics module includes the Savings dashboard that offers procurement teams a unified view of savings opportunities across the sourcing pipeline.

Open the Source-to-Pay Workspace and select the Category Analytics icon \(![Category analytics icon.](../image/category-analytics-icon.png)\). The Category analytics module opens, displaying the Savings dashboard.

Users with th sn\_spend\_mgmt.sourcing\_category\_manager role can view and access the Savings dashboard in the Category analytics module.

-   **[Savings dashboard](spo-savings-dashboard.md)**  
The Savings dashboard provides procurement teams with a consolidated view of savings opportunities across all pipeline project data.
