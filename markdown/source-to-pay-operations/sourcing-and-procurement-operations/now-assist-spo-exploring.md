# Exploring Now Assist for Sourcing and Procurement Operations \(SPO\)

With the Now Assist for Sourcing and Procurement Operations \(SPO\) application, fulfillers can summarize the details of procurement-related records to keep them informed about their progress and action items.

**Note:** You can install this application only if you're entitled to the Sourcing and Procurement Operations Pro Plus SKU, Source-to-Pay Operations Pro Plus SKU, or the Source-to-Pay and Contract Operations Pro Plus SKU.

## Overview of Now Assist for SPO

Using the generative AI capabilities, a fulfiller can generate a summary of a record that includes an overview, actions completed so far, and next steps. This ​enables them to get the context by gathering information about a record and taking actions more quickly​. It ​also enables them to refresh and post the summary to the work notes or activity stream.

The Now Assist for SPO application includes the generative AI skills and features that enable procurement specialists and fulfillers to view a summary of procurement-related records quickly within the Source-to-Pay Workspace.

-   **Summarization for fulfillers**

    Provides a fulfiller with a summary of procurement-related records. This enables them to get visibility into the status of the record in the procurement process and consequently enables them to take the necessary actions quickly.

    The fulfillers can get a summary of the following procurement records using the Source-to-Pay Workspace.

    -   Sourcing request summarization
    -   Purchase requisition summarization
    -   Procurement case summarization
    For more information, see [Summarize a record by using Now Assist for Sourcing and Procurement Operations \(SPO\) in Source-to-Pay Workspace](../task/now-assist-spo-summarize-record.md).

-   **Conversational intake**

    Now Assist for Sourcing and Procurement Operations \(SPO\) in Virtual Agent uses procurement-related topics to route requesters to the appropriate intake process, and then through conversation, helps to submit their requests. With this application, you don't need to rely on fulfillers, and your requesters can submit their procurement requests accurately with the help of an Artificial Intelligence \(AI\) companion. For more information, see [Request the generative AI capabilities by using the Now Assist for SPO Virtual Agent](now-assist-spo-va-using.md).


For more information on configuring Now Assist for SPO, see [Configuring Now Assist for Sourcing and Procurement Operations \(SPO\)](../task/configure-now-assist-for-spo.md).

As requesters, you can use Now Assist for SPO to submit purchase requisitions, sourcing requests, and even request for products or services that aren't in the catalog. To learn more about how to use Now Assist for SPO, see [Using Now Assist for Sourcing and Procurement Operations \(SPO\)](now-assist-spo-using.md).

For detailed information on Now Assist in Virtual Agent and how to use generative AI skills in your conversational experiences, see [Now Assist in Virtual Agent](https://www.servicenow.com/docs/access?context=now-assist-in-va-landing&version=xanadu&pubname=xanadu-conversational-interfaces&ft:locale=en-US).

Now Assist in Virtual Agent uses large language models \(LLMs\) and generative AI skills to help improve deflection rates and reduce the amount of time-consuming work that Natural Language Understanding \(NLU\) topic discovery requires. To understand how you can interact with the Virtual Agent when a Now Assist capability is active in your instance, and also the benefits of Now Assist in Virtual Agent, see [Exploring Now Assist in Virtual Agent](https://www.servicenow.com/docs/access?context=exploring-now-assist-va&version=xanadu&pubname=xanadu-conversational-interfaces&ft:locale=en-US).

You can also look up [LLM assistants](https://www.servicenow.com/docs/access?context=llm-assistants&version=xanadu&pubname=xanadu-conversational-interfaces&ft:locale=en-US) for more information on the Now Assist in Virtual Agent assistant that is created during the Conversational Interfaces Assistants guided setup.

Domain separation is supported for Now Assist in Virtual Agent. For more information, see [Now Assist in Virtual Agent domain separation](https://www.servicenow.com/docs/access?context=na-in-va-domain-separation&version=xanadu&pubname=xanadu-conversational-interfaces&ft:locale=en-US).
