# Checkout data loss scenarios

The data you enter during checkout are lost in an incomplete checkout process.

This happens if you perform any of the following actions:

-   Log out.
-   Return to your shopping cart page.
-   Navigate to any other page on Shopping Hub.
-   Close the browser window.

However, if you refresh the page, values on the current and previous steps are retained.
