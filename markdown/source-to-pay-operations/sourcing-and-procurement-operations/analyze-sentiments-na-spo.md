# Analyze sentiment in procurement cases

Analyze requester sentiments on active procurement cases to help procurement specialists quickly identify, prioritize, and track sensitive requests.

## Before you begin

Role required: sn\_spend\_gen\_ai.now\_assist\_fulfiller

## Procedure

1.  Navigate to **Workspaces** &gt; **Source-to-Pay Workspace**.

2.  Select the list icon \(\[Omitted image "cases-list-icon.png"\] Alt text: List icon.\).

3.  Navigate to **Lists** &gt; **All work** &gt; **Cases**.

4.  Select any active procurement case.

    The procurement case form view appears.

5.  To see why the sentiment was generated, select the information icon \[Omitted image "icon-more-info.png"\] Alt text: The information icon provides an explanation for why a particular sentiment has been generated..

6.  To refresh the sentiment and trend manually, select the information icon \[Omitted image "icon-more-info.png"\] Alt text: The information icon provides an explanation for why a particular sentiment has been generated. and then select the refresh icon \[Omitted image "icon-refresh.png"\] Alt text: The refresh icon manually refreshes the procurement case sentiment..

    A red dot indicates that the sentiment was updated due to recent activity. Sentiment updates automatically every 30 minutes.
