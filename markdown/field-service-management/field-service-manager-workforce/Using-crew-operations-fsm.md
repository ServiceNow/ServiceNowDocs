# Using Field Service Crew operations

Use Field Service Crew Operations to assign the same set of resources repeatedly to the tasks that requires a crew to work on them.

-   **[Assign work order tasks to crews](../task/assign-wot-crew-operations.md)**  
Assign crews to work order tasks that require a group of agents to work on them.
-   **[Managing crew tasks using the Now Mobile Agent application](executing-crew-tasks.md)**  
After the dispatcher dispatches work order tasks to a crew, tasks are sent to the assigned members of the crew, which includes the crew leader.
-   **[Add a resource to a crew task](../task/add-agent-crew-task.md)**  
Add resources that aren't part of a crew, like agents or pieces of equipment, if you realize they're required after the task has been created.
-   **[Add resource requirements for a work order task](../task/add-resource-requirement-wot.md)**  
Add the resources needed to complete a crew work order task, such as necessary agent skills or equipment items, to avoid delays after the task is assigned.
-   **[Change the time that an agent worked on a crew task](../task/change-agent-task-time.md)**  
Update the time that an agent spent on a crew task if they spent more or less time on the task than the rest of the crew.
