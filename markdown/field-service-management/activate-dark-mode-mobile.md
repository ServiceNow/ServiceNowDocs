# Activate dark mode

Turn on dark mode in the Mobile Agent to alleviate eye strain if the default is too bright.

## Before you begin

Role required: wm\_agent

## Procedure

1.  Open the Mobile Agent.

2.  Tap **Settings**.

3.  Tap **Appearance**.

4.  Tap **Dark theme**.
