# Managing ad hoc filters

Create ad hoc filters in Dispatcher Workspace to filter work order tasks with custom criteria.

You can create an ad hoc filters to be either temporary and unique to a working session, or saved for future use.

There are two types of filters.

<table id="table_oxf_ctc_hwb"><thead><tr><th>

Types

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Temporary Filter

</td><td>

-   Non-persistent, exists only during the current session.
-   Only one temporary filter can be in use at a time. Creating a new temporary filter overwrites the current filter.

</td></tr><tr><td>

Saved Filters

</td><td>

-   Persistent.
-   Multiple filters can be saved for future use.

</td></tr></tbody>
</table>
