# Managing work order tasks using the Field Service Contractor for mobile application

Managers and agents of contractor companies can use the Field Service Contractor for mobile application to view and manage the work order tasks assigned to them.

Organizations can outsource their work order tasks to contractor companies. The contractor company performs assigned jobs by deploying their contractor agents at the customer location.

-   **[Reassign a work order task to yourself on a mobile device](../task/assign-wot-yourself.md)**  
Reassign a work order task to yourself through the Field Service Contractor for mobile feature of the ServiceNow Agent application.
-   **[Reassign a work order task to an agent on a mobile device](../task/reassign-work-order-task.md)**  
As a manager, you can reassign a work order task to an agent through the Field Service Contractor for mobile application.
-   **[Accept or reject a work order task on a mobile device](../task/self-assign.md)**  
Accept or reject a work order task assigned to you through the Field Service Contractor for mobile application.
-   **[Create work orders for a selected asset on a mobile device](../task/create-wot-asset-contrator-mob.md)**  
Create a work order for an asset through the Field Service Contractor for mobile application.
-   **[View knowledge articles related to a work order task on a mobile device](../task/view-articles-field-service-contractor-portal-mobile.md)**  
View knowledge articles related to a work order task through the Field Service Contractor for mobile application to get additional information to help you complete the task.
-   **[Enter incidental expenses directly from a work order task on a mobile device](../task/manage-incidentals.md)**  
Create and track incidental expenses that arise during the execution of a work order task through the Field Service Contractor for mobile application.
-   **[View an agent's profile on a mobile device](../task/view-agent-details-fsc.md)**  
View the profile information of all the agents in your team through the Field Service Contractor for mobile application so you can contact them when necessary.
-   **[View details screen on the Field Service Contractor for mobile in ServiceNow Agent application](../task/view-details-tab-fsc.md)**  
Use Field Service Contractor for mobile application to view work order fields detail.
-   **[Track or add work order task information on a mobile device](../task/view-activity-stream-fsc.md)**  
View activities related to your work order task through the Field Service Contractor for mobile application. You can also add documents, images, or work notes to the work order task record.
-   **[Close a complete work order task on a mobile device](../task/close-complete-wot-fsc.md)**  
Close work order tasks to complete a task for which the issue is fixed or resolved through the Field Service Contractor for mobile application.
-   **[Close an incomplete work order task on a mobile device](../task/close-incomplete-fsc.md)**  
You can close a work order task as incomplete if there is work pending on the task through the Field Service Contractor for mobile application.
-   **[Pause or resume work order task tracking on a mobile device](../task/pause-resume-wot-fsc.md)**  
Record a break from a work order task through the Field Service Contractor for mobile if you are unable to continue the work for any reason.
-   **[View recently closed work order tasks on a mobile device](../task/view-recently-closed-tasks.md)**  
View the work order tasks that are closed in the last seven days in the ServiceNow Agent mobile application.
-   **[Sign on closed work order on a mobile device](../task/preview-sign-wot-contractor-mobile.md)**  
Customers can digitally sign and confirm work order tasks that are closed with the Closed Complete or Closed Incomplete state.
-   **[Request inventory using the Field Service Contractor for mobile application](../task/request-inventory-fsm-contractor-mobile-app.md)**  
Request inventory to receive parts when your stock gets low or you need a part for a particular work order.
-   **[View inventory details using the Field Service Contractor for mobile application](../task/view-inventory-detail-fsm-contractor-mobile.md)**  
View details of available and defective parts in your inventory.
-   **[pick up parts using the Field Service Contractor for mobile application](../task/pick-up-parts-fsm-contrator-mobile.md)**  
Pick up the part by transferring it from an available stockroom when you are at a task site and need a missing part to complete your job.
-   **[Complete questionnaires for work order tasks using the Field Service Contractor for mobile application](../task/questionnaire-wot-fsm-cotractor-mobile.md)**  
Answer to the preset questions associated with a work order task to gather and store the responses for future reference.
