# Navigate a job site in the Mobile Agent application

View a map of the job site and get directions to the specific on-site location.

## Before you begin

To view site maps, administrators must activate the Site Mapping for Field Service Management plugin \(com.\). For more information, see [Activate Site Mapping for Field Service Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/mobile-experience-for-field-service-management-glide-family/activate-sitemap-fsm.md).

Role required: wm\_agent

## Procedure

1.  Select **My work**.

2.  Select a work order task.

3.  Select **Site**.

4.  Select **Get directions**.
