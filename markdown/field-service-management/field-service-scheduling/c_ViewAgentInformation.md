# View agent information

View details about an agent at a particular location by selecting the agent icon on the Dispatcher Workspace map.

The contextual side panel shows the agent's profile that displays the agent schedule and these additional details:

<table id="table_am1_dxp_nhc"><tbody><tr><td>

Name

</td><td>

The agent's name.

</td></tr><tr><td>

Assignment group

</td><td>

Assignment group this agent belongs to.

</td></tr><tr><td>

Schedule status

</td><td>

Possible values are **On time**, **Behind schedule**, and **Ahead**.

</td></tr><tr><td>

Shift status

</td><td>

If the agent is **On shift** or **Off shift**.

</td></tr><tr><td>

Schedule

</td><td>

The schedule of the agent.

</td></tr><tr><td>

Mobile Phone

</td><td>

Contact number for the agent.

</td></tr><tr><td>

Email

</td><td>

Email address for the agent.

</td></tr><tr><td>

Skills

</td><td>

All the skills assigned to the agent.

</td></tr></tbody>
</table>You can also view all tasks for the agent for that day in the **Tasks** list. To display the agent's task route for the day, select **Show Route** and to optimize the route, select **Optimize Route**. The icons are labeled alphabetically based on the start time of the task in the agent's route.
