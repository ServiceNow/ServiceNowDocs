# Using Dispatcher Workspace for crew operations

Use the Dispatcher Workspace to create, modify, and enable crews and assign work order tasks to the crews.

As a dispatcher, you can use Dispatcher Workspace to perform the following activities:

-   Create or modify crews
-   Assign members to the crew
-   Filter and view tasks in the task panel that need a crew
-   Sort crews based on the distance and skills for the tasks that need a crew.

**Note:** The drag and drop feature is not enabled for the tasks requiring a crew.

-   **[Create crews](../task/create-crews-dispatcher-wrkspc.md)**  
As a dispatcher, you can create crews in Dispatcher Workspace to which you can assign work order tasks.
-   **[Create ad hoc crews in Dispatcher Workspace](../task/create-adhoc-crews-disp-wrkspc.md)**  
Create ad hoc crews for a task if no existing crews are available to work on the task or the task requires specific skills to complete the job.
-   **[Create quick crews in Dispatcher Workspace](../task/create-quick-crews-disp-wrkspc.md)**  
Create planned or ad hoc crews faster and more efficiently using a single interface in Dispatcher Workspace. This helps dispatchers to view the agent schedule and availability while creating crews and adding crew members.
-   **[Viewing crews in the Dispatcher Workspace calendar](viewing-crews-dsp-wrkspc.md#)**  
You can view the crew's schedule, availability, and tasks that are assigned to the crew or their work in progress.
-   **[Search for crews on Dispatcher Workspace](../task/search-crews-disp-wrkspc.md)**  
Search for crews with specific attributes on Dispatcher Workspace so you can make appropriate assignments.
-   **[Update crew requirements for the work order task in Dispatcher Workspace](../task/add-crew-size-disp-wrkspc.md)**  
As a dispatcher, you can add or update the number of agents required to perform a particular type of work to complete the task.
-   **[Search for tasks that need a crew on Dispatcher Workspace](../task/search-crew-task.md)**  
Search for work order tasks that require a crew in the Dispatcher Workspace task panel.
-   **[Identify crews and agents for crew tasks on Dispatcher Workspace](../task/sort-crews-for-wot.md)**  
Identify the most eligible crew and agents based on crew skills and distance to assign them to work order tasks that need a crew.
-   **[View crew tasks on Dispatcher Workspace](../task/view-crew-tasks-disp-wrkspc.md)**  
As a dispatcher, you can view the tasks that require crew and are assigned to a crew.
-   **[Assign work order tasks to crews on Dispatcher Workspace](../task/assign-wot-to-crew.md)**  
Assign crews to a work order task that requires a crew of agents in Dispatcher Workspace.
-   **[Assign an ad hoc agent to the crew on a work order task](../task/add-adhoc-agent-crew.md)**  
Add ad hoc members to a work order task that requires a crew.
-   **[Add Field Service crew members in Dispatcher Workspace](../task/add-crew-members-disp-wrkspc.md)**  
Add agents to the Field Service crew as needed in Dispatcher Workspace.
-   **[Add resource requirement for a work order task in Dispatcher Workspace](../task/add-resource-requirement-disp-wrkspc.md)**  
Add or update the resource requirements for a work order task that requires crew, such as number of agents with specific skills or types of equipment required to work on the task.
-   **[Assign required resources to the task-specific crews](../task/assign-required-resources-wot-crew.md)**  
Quickly add resources to task-specific crews if you forgot to add an agent or piece of equipment when you created the crew.
