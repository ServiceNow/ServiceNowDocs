# Flag a work order task

Flag tasks that you want to work on at a later time.

## Before you begin

Role required: wm\_dispatcher

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Dispatching** &gt; **Dispatcher Workspace**.

2.  Click **Dispatcher Workspace**.

3.  Click the Flag icon \(![Flag icon.](../image/flag_tasks.png)\) on the task.


## Result

The work order task is flagged and is displayed when you filter the tasks by the option **Flagged Tasks** in the task panel.
