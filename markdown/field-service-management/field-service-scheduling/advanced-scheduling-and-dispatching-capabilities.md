# Additional scheduling configuration options

Optimize resource allocation and ensure timely service delivery with the advanced scheduling and dispatching capabilities, designed to manage technician workloads and prioritize high-priority tasks.

-   **[Configuring Task Bundling](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/field-service-scheduling/configuring-fsm-task-bundling.md)**  
Bundle work order tasks to keep similar tasks together.
-   **[Configuring Appointment Booking](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/field-service-scheduling/appointment-booking-administer.md)**  
Create or modify and enable the application configuration and the individual service configurations for the appointment booking feature.
-   **[Configuring Multi-day scheduling](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/field-service-scheduling/configuring-multi-day-scheduling.md)**  
Field Service Multi-day task scheduling helps dispatchers manage tasks that require more than a single workday to complete.
-   **[Setting up Soft Booking](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/field-service-scheduling/setting-up-soft-booking.md)**  
Soft Booking allows dispatchers to schedule tasks without triggering immediate notifications or downstream workflows. This feature provides dispatchers with a "planning buffer" to optimize agent schedules before confirming tasks.
-   **[Configuring Intraday schedule automation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/field-service-scheduling/config-intraday-scheduling-steps.md)**  
Intraday schedule automation updates agents schedules to maximize productivity during their shift.
-   **[Configuring Auto Assignment scheduling](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/field-service-scheduling/c_UseAutoAssignment.md)**  
The auto assignment feature can be enabled for requests or tasks, depending on the Service Management \(SM\) application's configuration settings.
