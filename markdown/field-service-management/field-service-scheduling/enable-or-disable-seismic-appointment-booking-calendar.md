# Enable or disable seismic Appointment Booking calendar

Use the seismic appointment booking calendar across all user interfaces to ensure a consistent and seamless scheduling experience.

## Before you begin

Role required: admin

## About this task

Seismic Appointment Booking calendar is enabled by default, but can be disabled if required.

Enhance scheduling support for appointment slot recommendations, customized themes, and an improved user interface.

## Procedure

1.  Navigate to **All** and search for `sys_properties.list`.

2.  Search for the property `sn_apptmnt_booking.use_unified_appt_widget`.

3.  In the **Value** field:

    -   Enter true to **enable** the Seismic Appointment Booking calendar.

    -   Enter false to **disable** the Seismic Appointment Booking calendar.

4.  Select **Update**.


## Result

Users will experience appointment scheduling based on your configuration setting.
