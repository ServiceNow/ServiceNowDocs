# Change the order of subtasks in a bundle

Change the order of subtasks in a bundle if you want the subtasks to be completed in a different order than the one originally assigned.

## Before you begin

Role required: wm\_dispatcher

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Dispatching** &gt; **Dispatcher Workspace**.

2.  Select **List**.

3.  Select **Dispatching** &gt; **My Bundles**.

4.  Select the bundle that you want to reorder.

5.  Select **Bundled Tasks**.

6.  Double-click \(or use the keyboard shortcut\) to select the order number of the subtask that you want to reorder.

7.  Enter a new order number.

8.  Select **Save**.
