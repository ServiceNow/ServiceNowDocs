# Create or modify Agent Efficiency criteria

Create or modify Field Service Agent Efficiency criteria to evaluate agent efficiency and calculate the work duration for a work order task.

## Before you begin

Role required: wm\_admin, wm\_dispatcher

## Procedure

1.  Navigate to **All** &gt; **Agent Efficiency** &gt; **Agent Efficiency Criteria**.

2.  Create a new criterion or update an existing criterion.

    -   To create a new criterion, select **New**.
    -   To modify a criterion, select its name.
3.  On the form, fill in or modify the fields.
