# Use the Now Mobile Agent application to perform tasks

Field service agents and managers use the mobile application to manage their field service tasks from anywhere.

-   **[Prioritize your work using the Now Mobile Agent application](manage-field-service-tasks-mobile.md)**  
Accept or reject tasks, assign tasks to yourself and plan your schedule for the day, view task details, schedule, and task location, or close a work order task using your mobile application.
-   **[Use Time Sheets in the Now Mobile Agent application](timesheet-fsm-mobile.md)**  
You can access and manage your time sheets from a mobile device.
-   **[Manage your inventory using the Now Mobile Agent application](easily-track-parts.md)**  
Field service agents can locate, request, and transfer parts from other agents or warehouses for the current and future work order tasks.
-   **[Manage your crew using the Now Mobile Agent application](manage-crew-mobile-agent-app.md)**  
Add equipment or members to your crew using the Now® Mobile Agent application.
-   **[Actionable push notifications in the Now Mobile Agent application](actionable-push-notifications.md)**  
Assign tasks and send reminders to Field Service agents using push notifications.
-   **[Get help needed to complete jobs](get-help-needed-complete-jobs.md#)**  
Connect with your customers or peers using the mobile app to get more information when you are at a task location.
-   **[Access the information you need using the Now Mobile Agent application](access-information-you-need-mobile.md#)**  
Search relevant information from knowledge articles and see when your team members are on-site and if they are on schedule.
-   **[Complete work order tasks using the Now Mobile Agent application](close-wo-wot-mobile.md#)**  
A work order is automatically closed when all work order tasks associated with it have been closed.
-   **[Field Service mobile offline application](work-offline-fieldservice-mobile.md#)**  
When you are at a task site that does not have an Internet connection, you can work on field service tasks, record time worked on a task, track asset usage, create part requirements, and complete work order and work order tasks using the Now Mobile Agent application.
-   **[Manage events with the Mobile Agent](events-now-mobile-application.md)**  
Field service agents can view events and create and edit personal events in the Mobile Agent.
-   **[Update equipment statuses with the Mobile Agent](update-equipment-statuses-with-the-now-mobile-agent-application.md#)**  
Check in, check out, or report broken equipment using the **My Equipment** applet in the Mobile Agent.
-   **[Navigate a job site in the Mobile Agent application](../task/navigate-job-sites-mobile.md)**  
View a map of the job site and get directions to the specific on-site location.
-   **[Create a work order task using the Now Mobile Agent application](../task/mobile-agent-create-a-wot.md)**  
As an agent, create work order tasks using the mobile application.
-   **[Using location tracking in the Now Mobile Agent application](location-tracking-mobile.md)**  
Use location tracking so that you can keep a record of your location, either for a defined period of time or while you perform tasks.
-   **[Playbooks for the Now Mobile Agent application](playbooks-now-mobile-agent-app.md)**  
Playbooks for the Now Mobile Agent application provide a guided experience for completing work order tasks on a mobile device.
-   **[Managing work order tasks for linear assets using the Mobile Agent application](working-on-linear-asset-wot.md)**  
Manage work order tasks for linear assets to ensure efficient and effective maintenance and inspection processes.
-   **[Using Field Service Management Virtual Conferencing Integration](using-vci-fsm.md)**  
Initiate a conference call from your native mobile application with the help of Field Service Management Virtual Conferencing Integration.
