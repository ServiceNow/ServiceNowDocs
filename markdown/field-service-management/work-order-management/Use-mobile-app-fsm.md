# Completing work on the ServiceNow Agent mobile application

Field service technicians and managers use the ServiceNow Agent mobile application to manage their field service tasks from anywhere.

## Using overview

Completing work order tasks on ServiceNow Agent can include one or more of the following activities.

-   [Time Sheets in ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/timesheet-fsm-mobile.md)

    Create and submit time sheets, log hours, and more all from your mobile device for accurate billing and reporting.

-   [Personal events in ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/events-now-mobile-application.md)

    Create a personal event, like a doctor's appointment, on your mobile device so you don't get a task scheduled during your appointment.

-   [Create a task on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/mobile-agent-create-a-wot.md)

    Field service technicians can create work order tasks from their mobile device in case there is some work that was missed when the work order was created.

-   [Prioritizing on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/manage-field-service-tasks-mobile.md)

    Field service technicians can plan their schedule for the day, view task details including client details, location, and requirements, and more.

-   [Tracking your location on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/location-tracking-mobile.md)

    Use location tracking so the customer knows when you will arrive. This also helps dispatchers quickly see if your travel time has made you behind schedule.

-   [Inventory on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/easily-track-parts.md)

    Field service technicians can locate, request, and transfer parts from other technicians or warehouses for current and future work order tasks, and record any equipment or tools used for billing or inventory purposes.

-   [Equipment on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/update-equipment-statuses-with-the-now-mobile-agent-application.md)

    Check in, check out, or report broken equipment using the Mobile Agent app.

-   [Field Service Contractor for mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/wot-fsc.md)

    Managers and technicians of contractor companies can use the Field Service Contractor for mobile application to view and manage the work order tasks assigned to them.

-   [Crew on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/manage-crew-mobile-agent-app.md)

    Add equipment to crews, manage crew membership, and more on the Mobile Agent app.

-   [Push notifications on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/actionable-push-notifications.md)

    Field service technicians, managers, and dispatchers can use push notifications to receive and acknowledge new task assignments and stay informed of any other updates related to tasks, groups, or individual agents.

-   [Start a Sidebar discussion on the Mobile Agent application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/start-sidebar-mobile.md)

    Connect with your customers or peers using the Mobile Agent app to get more information when you're at a task location.

-   [View job site maps on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/navigate-job-sites-mobile.md)

    View a map of the job site, or get step-by-step directions all from the Mobile Agent app.

-   [Linear assets on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/working-on-linear-asset-wot.md)

    Manage work order tasks for linear assets for an efficient and effective maintenance and inspection processes.

-   [Playbooks on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/playbooks-now-mobile-agent-app.md)

    Playbooks for the Mobile Agent application provide a guided experience for completing work order tasks on a mobile device.

-   [Virtual Conferencing on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/using-vci-fsm.md)

    Schedule or start a Zoom or Microsoft Teams meeting from the Now Mobile Agent application.

-   [Knowledge articles on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/access-information-you-need-mobile.md)

    Search and access troubleshooting guides, manuals, FAQs, and relevant knowledge articles, and see when your team members are on-site and if they are on schedule.

-   [Working offline on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/work-offline-fieldservice-mobile.md)

    Download field service data to the application cache so you can still use the information in a work order task when you're at a task site without an internet connection, and sync data once reconnected.

-   [Closing tasks on Mobile Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/close-wo-wot-mobile.md)

    Close a work order task as complete after you finish the work required for the task.
