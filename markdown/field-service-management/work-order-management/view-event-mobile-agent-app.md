# View events in the Mobile Agent

View events in the Mobile Agent to stay updated on your schedule, and any events planned for the day.

## Before you begin

Role required: wm\_agent

## Procedure

1.  Open the Mobile Agent and tap **My schedule**.

2.  Tap the day of the event.

3.  Tap the event to view it.
