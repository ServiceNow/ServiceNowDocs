# Create a work order task

Managers can quickly create tasks from Field Service Manager Mobile.

## Before you begin

Role required: wm\_manager, sn\_fsm\_tp.fsm\_territory\_resource\_manager

## Procedure

1.  Open Field Service Manager Mobile and tap **Teams**.

2.  Tap the assignment group or territory that the agent you want to assign a task to is part of.

3.  Tap the clipboard icon \(![clipboard](../image/manager-clipboard.png)\) next to an agent.

4.  Tap the more actions icon \(![more actions](../image/more-action-mobile-dark.png)\).

5.  Tap **Create new work**.

6.  On the form, fill in the fields.

7.  Tap **Submit**.

    A work order task is created and is ready to be qualified or dispatched.
