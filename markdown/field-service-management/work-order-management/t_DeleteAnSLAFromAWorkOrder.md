# Delete an SLA from a work order

Remove an SLA from a work order but leave it available for use with other work orders.

## Before you begin

Role required: wm\_admin

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Work Order** &gt; **Work Order SLAs**.

2.  Open a work order.

3.  In the **Task SLAs** related list, select the check box beside the SLA name.

4.  In the **Actions** choice list, select **Delete**.
