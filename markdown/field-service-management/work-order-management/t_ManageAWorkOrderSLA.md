# Manage a work order SLA

View and manage work order SLAs from the Work Order form or the Work Order Task form.

## Before you begin

Role required: wm\_admin, wm\_dispatcher

## About this task

Activities include suspending and resuming SLAs and deleting SLAs from a work order.

## Procedure

1.  To view the list of work orders with SLAs, navigate to **Field Service** &gt; **Work Order** &gt; **Work Order SLAs**.
