# Record the time worked on a work order task

Manage and record time worked on work order tasks and activities. Time recorded entries automatically generate time cards and time sheets for approval by managers.

-   **[Using rate types and labor rate cards](using-rate-types-labor-rate-cards.md)**  
Use rate types and labor rate cards to define different cost rates for different activities recorded by field service agents.
-   **[Record time worked for a task or activity manually](../task/field-service-agent-record-time.md)**  
Agents can record time worked on a work order task as well as time spent on other activities.
-   **[Pause a work order task](../task/automate-time-recording-field-service.md)**  
Record a break from a work order task in the Field Service application if you are unable to continue the work for any reason. You can resume the work order task when you start working on it again. The system automatically tracks and calculates the actual time taken to complete the task even though you worked at different intervals.
-   **[Review time recorded for a task](../task/field-service-agent-review-time.md)**  
Agents can review the time recorded for work order tasks.
-   **[Modify or delete time worked entries](../task/field-service-agent-modify-time.md)**  
Agents can modify and delete time worked entries before a time sheet is approved.
-   **[Review and submit a time sheet](../task/field-service-agent-submit-time.md)**  
Agents can review and submit time sheets to managers for review and approval.
-   **[Review and approve time sheets](../task/field-service-manager-approve-time.md)**  
Managers can review time cards and time sheets for agents in their assignment groups.
