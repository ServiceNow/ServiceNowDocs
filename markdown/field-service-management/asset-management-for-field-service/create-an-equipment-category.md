# Create an equipment category

Create equipment categories to organize types of equipment.

## Before you begin

Role required: wm\_admin

## About this task

Equipment categories or resource categories are types of equipment that can be assigned to crews.

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Equipment Resources** &gt; **Equipment Categories**.

2.  Select **New**.

3.  Enter a name in the **Category name** field.

4.  Enter a description in the **Short description** field.

5.  Display equipment on the Dispatcher Calendar by selecting **Show on schedule**.

6.  Select **Active** to make this category active.

7.  Select **Submit**.


## Result

The equipment category is added to the Resource categories list.
