# Create a work order from a change request in the CSM Configurable Workspace

Create a work order directly from the change request using the CSM Configurable Workspace.

## Before you begin

Roles required: itil, wm\_admin, and sn\_change\_write

## About this task

A change request can accommodate only one work order at a time. You can create another work order for a change request only if an existing work order is canceled.

## Procedure

1.  Log in to the CSM Configurable Workspace.

2.  Click the **List** icon \(![List icon.](../../../administer/workspace/image/list-icon-black.png)\).

3.  In the Lists panel, navigate to **Change** &gt; **All**

4.  Open the change request for which you want to create a work order.

5.  Click **Create Work Order**.


## Result

A work order is created successfully.
