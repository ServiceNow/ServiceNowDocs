# Configure the Now Mobile Agent application

Configure the Now Mobile Agent application and customize it for field service agents and dispatchers.

## Before you begin

Ensure that the mobile plugin is installed.

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Applications** &gt; **Studio**.

2.  In the **Select Application** screen, select **Field Service Mobile**.

3.  Follow these steps to configure the Now Mobile Agent application.

    1.  [Connect to your Field Service data](install-fsm-mobile-app.md)

        Connect the Now Mobile Agent application to your Field Service application instance to manage work on your mobile device.

    2.  [Customizing UI actions for the Now Mobile Agent application](../concept/configure-uiaction-fsm-mobile.md)

        Make it easier for your end users to get things done faster with the Now Mobile Agent application by creating custom UI actions.

    3.  [Configure UI actions in the Now Mobile Agent application](configure-ui-actions-mobile.md)

        Enable or disable the desired UI actions on the mobile device to minimize the load on the mobile resources.

    4.  [Configuring push notifications for task assignment](../concept/push-notifications-tasks-slas.md)

        Assign tasks and send reminders to customer service agents and field technicians using push notifications.

    5.  [Location tracking for mobile](../../../administer/tablet-mobile-ui/concept/mobile-location-tracking.md)

        Location tracking enables you to track the activity and positioning of agents while they perform tasks. You can also use the feature to make sure your agents are safe and can be easily located.

    6.  [Configure scheduled offline caching](configure-scheduled-offline-caching.md)

        Configure system components to automatically download the cache in the background.

    7.  [Configure the recently closed work order tasks list](configure-recently-closed-di.md)

        Configure the number of recently closed work order tasks to display in the **Recently Closed Tasks** list in the Now Mobile Agent application.

    8.  [Configure special handling notes for the Now Mobile Agent application](configure-shn-fsm-mobile.md)

        Use special handling notes to notify users about important information on an individual work task.

    9.  [Enable chat in the Now Mobile Agent application](enable-chat-in-fsm-mobile.md)

        Activate the Chat quick action so that field service agents can chat with a virtual agent using the Now Mobile Agent application.

    10. [Enable dark theme in the Now Mobile Agent application](dark-theme-mobile.md)

        Turn on dark theme to alleviate eye strain and improve readability.
