# Set the default provider for a Custom Map Screen

Configure a map screen to enable site maps on the Mobile Agent application.

## Before you begin

Ensure there is a Custom Map Provider. For more information, see [Create a Custom Map Provider record](create-custom-map-prvdr-rcrd.md).

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Site Mapping** &gt; **Custom Map Screens**.

2.  Select **Indoor Map of Site Details for WOT Site**.

3.  Enter the Custom Map Provider record in **Default Provider**.


## Result

Agents can see the map of the building on the Mobile Agent application.
