# Change Field Service Manager Mobile to use assignment groups or territories

Administrators can change Field Service Manager Mobile to show territories instead of assignment groups.

## Before you begin

Role required: admin

You must have the Field Service Manager Mobile plugin installed before changing from assignment groups to territories. You must have the territory planning plugin installed to use territories with Field Service Manager Mobile.

## About this task

By default Field Service Manager Mobile uses assignment groups.

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Manager Mobile** &gt; **Properties**.

2.  Select the **Show Mobile view by** drop-down list.

3.  Select **Assignment Groups** or **Territories**.

4.  Select **Save**.
