# Manage an appointment from the portal

Book, reschedule, or cancel an appointment for a service from either the Customer or the Consumer Service Portal.

Booking an appointment creates a work order for the selected service. The details for the scheduled appointment appear on the work order and work order task forms.

Reschedule an appointment by selecting it from the list of upcoming appointments or by going directly to the respective work order or work order task.

An appointment cannot be canceled if the current time is within the cancellation time window specified in the service configuration.
