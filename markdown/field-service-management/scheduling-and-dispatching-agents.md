# Scheduling and dispatching work order tasks to agents

Send agents to work on the tasks.

You can:

-   Assign tasks to agents
-   Use dispatch map to monitor agent and task status
-   Dispatch tasks to agents

-   **[Dispatching work order tasks](../../planning-and-policy/concept/c_DispatchWorkOrderTasks.md#)**  
When all of the work order tasks associated with a work order are qualified, you can dispatch the work order tasks. You can dispatch them from either the task form, the task map, or Dispatcher Workspace.
-   **[Field service agent management](field-service-agent-management.md)**  
Field Service managers can view and manage the skills, schedules, and work order tasks of agents in their assignment group.
-   **[Workforce](using-manager-workforce.md)**  
Workforce is a configurable application that enables any users with the wm\_basic role to easily access and view agent schedules, the team calendar, and map from one centralized location.
-   **[View and assign scheduled tasks](../task/view-and-assign-scheduled-tasks.md)**  
Learn about different ways to view tasks in the Scheduled state​.
-   **[Using dynamic scheduling](using-dynamic-scheduling.md)**  
Dynamic scheduling increases the efficiency of Field Service Management by automatically assigning tasks to the agents.
-   **[View scheduling history for work order tasks](../task/view-scheduling-history.md)**  
View the history of the scheduling method of work order tasks that are assigned and scheduled.
-   **[Run the Populate Agents Daily Schedule Table](../task/run-agent-schedule.md)**  
Trigger the Populate Agents Daily Schedule Table to run so dispatchers can see the changes that they made without having to wait for the nightly execution.
-   **[Create a work schedule for an agent or technician](../../customer-service-management/task/create-agent-work-schedule.md)**  
Users with the agent schedule administrator role can create one or more work schedules for a customer service agent or a field service technician.
-   **[Create a schedule to use with the agent calendar](../../customer-service-management/task/create-schedule-agent-calendar.md)**  
Users with the agent calendar administrator role can create a schedule to use with the agent calendar.
-   **[Using Field Service Crew Operations](Using-crew-operations-fsm.md)**  
Use Field Service Crew Operations to assign the same set of resources repeatedly to the tasks that requires a crew to work on them.
-   **[Using Field Service Marketplace](using-fsm-marketplace.md)**  
Field Service Marketplace optimizes contractor management, improves communication, and streamlines task allocation processes.
-   **[Field Service multi-day task scheduling](multi-day-task-fsm.md)**  
With the Field Service multi-day task scheduling capability, you can schedule work order tasks spanning multiple schedule entries or days.
-   **[Using Field Service Territory Planning Console](using-territory-planning-console.md)**  
Use the Territory Planning Console to assign work orders or work order tasks to agents or crews in the territories that best match their location based on conditions that you set.
