# Map iconography in Dispatcher Workspace

Unique map pins and icons in the dispatch map that mark the location of tasks and agents.

|Icon|Description|
|----|-----------|
|![Agent location marker.](../../field-service-management/image/agent-location.png)|An agent's location.|
|![Agent static state.](../../field-service-management/image/agent-static-state.png)|An agent's last known location when their live location can't be determined. For example, if the agent has lost reception.|
|![Agent home location.](../../field-service-management/image/agent-at-home.png)|The agent is at their home location.|
|![Multiple agents marker.](../../field-service-management/image/multiple-agents.png)|Multiple agents are at this location.|
|![Assigned map marker.](../../field-service-management/image/assigned.png)|The task is accepted, assigned, or scheduled.|
|![Assigned multiple.](../../field-service-management/image/assigned-multiple.png)|Multiple tasks have been accepted, assigned, or scheduled at this location.|
|![Assigned P1.](../../field-service-management/image/assigned-P1.png)|A priority-1 task has been accepted, assigned, or scheduled.|
|![Colored marker with 1 on it.](../../field-service-management/image/route-marker.png)|The assigned work order task displayed while viewing agent's route. The number indicates the order in which the task will be completed.|
|![Needs attention marker.](../../field-service-management/image/needs-attention.png)|The task is in the pending dispatch or draft state.|
|![Needs attention multiple.](../../field-service-management/image/needs-attention-multi.png)|Multiple tasks are in the pending dispatch or draft state.|
|![Needs attention P1.](../../field-service-management/image/needs-attention-P1.png)|A priority-1 task is in the pending dispatch or draft state.|
|![In progress.](../../field-service-management/image/in-progress.png)|An agent is on the way to the task, work on the task is paused, or the task is in progress.|
|![In progress multiple.](../../field-service-management/image/in-progress-multiple.png)|An agent is on the way to a location with multiple tasks, work on a task is paused, or the task is in progress.|
|![In progress P1.](../../field-service-management/image/in-progress-P1.png)|An agent is on the way to a priority-1 task, work on the priority-1 task is paused, or the priority-1 task is in progress.|
|![Complete.](../../field-service-management/image/complete.png)|A work order task is complete.|
|![Complete multiple.](../../field-service-management/image/complete-multiple.png)|Multiple tasks are complete at the same location.|
|![Complete P1.](../../field-service-management/image/complete-P1.png)|A priority-1 task is complete.|
|![Incomplete.](../../field-service-management/image/incomplete.png)|A task was closed as incomplete or canceled.|
|![Incomplete multiple.](../../field-service-management/image/incomplete-multiple.png)|Multiple tasks were closed as incomplete or canceled.|
|![Incomplete P1.](../../field-service-management/image/incomplete-P1.png)|A priority-1 task was closed as incomplete or canceled.|
|![Home location tasks.](../../field-service-management/image/home-location.png)|A task is assigned at the agent's home location.|
|![Home location task.](../../field-service-management/image/home-location-multi.png)|Multiple tasks are assigned at the agent's home location.|
|![Co-located multiple.](../../field-service-management/image/colocated-multi.png)|More than one task, agent, or crew is at this location.|
|![Co-located P1.](../../field-service-management/image/colocted-P1.png)|More than one task, agent, or crew is at this location. At least one of the tasks at this location is a priority-1 task.|
|![Completed route.](../../field-service-management/image/completed-route.png)|The lighter-colored line on the dispatch map shows the route that a Field Service agent has completed.|
|![Pending route.](../../field-service-management/image/pending-route.png)|The darker-colored line on the dispatch map shows the route that a Field Service agent is planning to complete.|
