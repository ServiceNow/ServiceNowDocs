# Prioritize your work using the Now Mobile Agent application

Accept or reject tasks, assign tasks to yourself and plan your schedule for the day, view task details, schedule, and task location, or close a work order task using your mobile application.

-   **[Plan your work order tasks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/view-wot-mobile.md)**  
Plan how to execute tasks assigned to you using the mobile app. You can prioritize by accessing SLAs and call customers directly from the app.
-   **[Assign tasks to yourself based on Intelligent Task Recommendation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/recommend-task-fsm-mobile.md)**  
Assign the best available recommended task to yourself so that you can fill a gap in your schedule.
-   **[Plan your schedule](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/view-task-location-schedule-mobile.md)**  
View tasks to be executed as well as your personal events on a particular day to plan your schedule for that day. Add events for yourself and display your availability to team members.
-   **[Pick up an unassigned task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/pick-up-unassigned-task-mobile.md)**  
If you are available, you can assign a task to yourself.
-   **[Optimize a task route](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/optimize-task-route-mobile.md)**  
Get optimal routes to your task locations and use the task map to navigate to the location. You can view the location of tasks assigned to you for the current week and use the app to choose the optimal route to execute the tasks for the current day.
-   **[Accept or reject a work order task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/accept-reject-task-on-mobile.md)**  
Accept a work order task that has been assigned to you. You can reject the task if, for example, you don’t have the required skills or you aren’t available at the required time.
-   **[Start work on a work order task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/work-task-on-mobile.md)**  
You can record both the time you begin traveling to the work site and the time you start the work.
-   **[Activate dark mode](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/activate-dark-mode-mobile.md)**  
Turn on dark mode in the Mobile Agent to alleviate eye strain if the default is too bright.
-   **[Create a sales opportunity](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/create-opportunity.md)**  
Create sales opportunities for services, products, or assets identified during field service visits using the Now Mobile Agent application.
-   **[View sales opportunities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/view-sales-opportunities.md)**  
View, sort, and filter sales opportunities using the Now Mobile Agent application.
-   **[Complete a questionnaire for work order task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/field-service-management/complete-questionnaire-mobile-app.md)**  
Complete the answers to questions associated with a work order task and store the responses for future reference. For example, you can create questionnaires for verifying the condition of an equipment before it is repaired.
