# Manage your inventory using the Now Mobile Agent application

Field service agents can locate, request, and transfer parts from other agents or warehouses for the current and future work order tasks.

-   **[Track inventory](../task/view-available-parts-inventory.md)**  
View a map that shows stockrooms with available parts relative to your location.
-   **[Request inventory](request-inventory.md#)**  
Request inventory when your stock gets low or you need a part for a particular work order.
-   **[Accept or reject part requests from peer agents](../task/accept-reject-part-request-mobile.md)**  
Review the part requests that you receive from peer agents.
-   **[Picking up parts](picking-up-parts.md#)**  
You can view a summarized list of transfer orders for all the assets that are waiting to be picked up physically.
-   **[Request rejected parts from other stockrooms](../task/request-rejected-parts-from-other-stockrooms.md)**  
Request parts from other stockrooms that have been rejected by your peer agents.
-   **[Duplicate part request](../task/duplicate-part-request.md)**  
Create a duplicate part request to request the same list of parts that you requested in the past.
-   **[View transfer order lines for requested parts](../task/source-part-mobile.md)**  
View all transfer order lines for the parts that are requested by you.
-   **[View schedule and track the parts for drop-off](../task/drop-off-parts-mobile.md)**  
You can view the parts that you have scheduled for drop off from your personal stockroom to another stockroom.
-   **[Analyze all work orders for a selected asset](../task/identify-defect-asset-mobile-app.md)**  
Scan an asset to view its work history and upcoming work orders. You can also create a work order for an asset.
-   **[Track an asset for a work order task from a mobile device](../task/use-create-asset-mobile.md)**  
Track the status of an asset that you used to execute a work order task or an asset that you removed or replaced by using the Now Mobile Agent application on a mobile device. You can track the status of these assets as you're doing the task from a task location.
-   **[Register new assets](../task/register-new-assets.md)**  
Register new assets that are supported for customers and have not been registered previously using the Now Mobile Agent application.
-   **[Move defective assets through the transfer process](../task/move-defective-assets-mobile-app.md)**  
Add defective parts from your stockroom to the drop-off list. Generate transfer orders to move these items to the stockroom where it needs to be dropped off.
-   **[Move excess assets through the transfer process from a mobile device](../task/move-excess-assets-transfer-process.md)**  
Move your excess consumable parts from your stockroom to the drop-off list by using the Now Mobile Agent application on your mobile device. You can generate transfer orders to move these items to the stockroom where it must be dropped off.
-   **[View install base information on Now Mobile Agent application](../task/view-install-base-fsm-mobile.md)**  
View information about install base items in work order tasks on the Now Mobile Agent mobile application to track the status of the questions and issues raised for the purchased products or services.
