# Analytics and reporting for Field Service Management

Field Service Analytics and Reporting Solutions contain preconfigured dashboards with actionable data visualizations to improve your business processes and practices.

-   **[Use the Field Service Management Overview dashboard](../../../use/dashboards/application-content-packs/Use_field_service_management_overview_dashboard.md)**  
Use the Field Service Management Overview dashboard to review the work orders by their order of priority. You can also view tasks by their assignment groups.
-   **[Monitoring operations in real time](monitoring-operations-in-real-time.md)**  
Proactively monitor critical field service trends and track cost allocations with dashboards.
-   **[Contractor dashboard](../../../use/dashboards/application-content-packs/contractor-dashboard.md)**  
Use this dashboard to review the status of work order tasks, agents, agent feedback, and appointments of contractor companies.
-   **[Contractor dashboard in the configurable workspace](../../../use/dashboards/application-content-packs/contractor-dashboard-configurable-workspace.md)**  
Use this dashboard to review the status of work order tasks, agent rating, and work distribution by type and state.
-   **[Field Service Territory Capacity Analytics dashboard](../../../use/dashboards/application-content-packs/capacity-dashboard.md)**  
Use the Field Service Territory Capacity Analytics dashboard to track capacity trends with interactive charts and graphs, perform gap analysis with a customizable pivot table, and apply filters for targeted data viewing and informed decision-making.
-   **[Scheduling Health dashboard](../../../use/dashboards/application-content-packs/scheduling-health-dashboard.md)**  
Use this dashboard to view technician metrics, task metrics, and Schedule Optimization configuration details.
-   **[Field Service Safety dashboard](use-safety-dashboard-fsm.md)**  
Review the status of agents, tasks, and assets using the Field Service Management Covid19 map. Monitor the compliance reports of agents through the Field Service Safety dashboard.
-   **[Field Service Territory Performance Analytics dashboard](../../../use/dashboards/application-content-packs/territory-planning-dashboard.md)**  
Use this dashboard to view and analyze performance analytics such as open work order tasks, agent utilization on work order tasks, and average travel time across territories.
-   **[View work order trends topics](../task/view-wo-trends.md)**  
Analyze the backlog of support work orders and identify clusters of work orders that point to similar underlying issues by using Predictive Intelligence for Field Service Management.​
