# View allocations for an opportunity

View allocation details and distributed amounts for an opportunity.

## Before you begin

Role required: sn\_opty\_mgmt\_core.opportunity\_allocation\_viewer

At least one allocation type must be active. If none are configured, contact your system administrator.

## Procedure

1.  In the CSM Configurable Workspace, select the **List** \[Omitted image "list-outline-24.svg"\] Alt text: view.

2.  Navigate to **Opportunity** &gt; **All** and select the opportunity you want to view.

3.  Select the **Allocations** tab to view all the allocations for this opportunity.

    The number in brackets displays the total count of active allocations. Each allocation group appears separately with its own set of rows for easy identification:

    -   Revenue: Distributes the total opportunity amount across eligible participants for ownership, forecasting, and credit.
    -   Overlay: Distributes product or line-level opportunity value for shared credit and forecasting.
    **Note:** You can customize active allocation types as needed.

    When no allocations are associated with this opportunity, the list appears empty.


## Result

The allocations for the opportunity are displayed in a read-only list and grouped by allocation type.
