# Export opportunities

Export a list of all opportunities in a specified file format, such as JSON or Excel, in Opportunity Management and provide it to a customer.

## Before you begin

Role required: sales\_agent

## Procedure

1.  In the CSM Configurable Workspace, navigate to the **List** view. ![](../image/Lists.png) and select **Opportunity** &gt; **All**.

2.  Select **Export**.

3.  In the Export dialog box:

    1.  Select the **File Type**:

        -   Excel
        -   CVS
        -   JSON
        -   PDF
    2.  Select the **Delivery Type**:

        -   Email - Enter a valid email address.
        -   Download - Downloads the export to a local drive.
    3.  Select **Export**.

    The list of opportunities is exported according to the parameters set.
