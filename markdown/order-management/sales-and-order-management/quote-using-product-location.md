# Filter catalogs by location in a quote

Agents can use the location option on the catalog tab to filter products by location in Quote Management.

## Before you begin

Role required: sales\_agent

## Procedure

1.  In the CSM Configurable Workspace, select the **List** ![](../image/Lists.png) view.

2.  Navigate to **Quote** &gt; **All-Quotes**.

3.  Select the quote that you’re working with.

4.  Select the **Catalog** tab.

5.  In the **Select a location** field, select a location.

    ![](../image/som-filter-catalog-by-location.png)The products in the catalog are filtered according to the selected location.
