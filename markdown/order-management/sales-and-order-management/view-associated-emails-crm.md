# Track emails linked from Microsoft Outlook

View emails that have been linked to CRM records using the Microsoft Outlook add-in for auditing or troubleshooting purposes.

## Before you begin

Role required: staging\_emails\_admin

## Procedure

1.  Navigate to **All** &gt; **User Mailboxes** &gt; **Staged Emails**.

2.  View a record by selecting the associated Information icon ![](../image/field-more-info.png), and then select **Open Record**.


## Result

The associated email is displayed with additional information such as the target table and the UID. For more information, see [Staged Email table](../reference/staged-email-fields-outlook.md).
