# Configure Program Segment Mapping

Establish a mapping between a program and a segment on the program segment mapping \(sn\_prm\_program\_segment\_mapping\) table to determine which segment belongs to which partner program.

## Before you begin

Role required: sn\_prm.partner\_ui

## Procedure

1.  Navigate to **All** &gt; **Partner Relationship Management** &gt; **Program Segments**.

2.  Select **New**.

3.  On the form, fill in the fields.

    To learn about the fields on the program segment mapping \(sn\_prm\_program\_segment\_mapping\) table, see [Program segment mapping table fields](../reference/program-segment-mapping-table-fields.md).

4.  Select **Submit**.
