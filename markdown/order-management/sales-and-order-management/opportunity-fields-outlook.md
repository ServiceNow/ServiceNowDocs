Field descriptions for viewing or editing an opportunity record from the ServiceNow CRM for Outlook add-in.

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

System-generated unique identifier for the opportunity record starting with the prefix OPTY.

</td></tr><tr><td>

Short Description

</td><td>

Brief summary of the opportunity.

</td></tr><tr><td>

Sales Cycle Type

</td><td>

Type of sales cycle for this opportunity. The available options are:-   NEWCUST
-   RENEW
-   UPSELL

</td></tr><tr><td>

Industry

</td><td>

Industry associated with the opportunity. The available options are:-   Manufacturing
-   Healthcare
-   Technology Services
-   Banking
-   Bio-Technology
-   Telecommunications

</td></tr><tr><td>

Account

</td><td>

Account associated with the opportunity. You can use the search icon to find an existing account.

</td></tr><tr><td>

Stage

</td><td>

Current stage of the opportunity in the sales pipeline. The available options are:-   Qualify
-   Develop
-   Propose
-   Negotiate
-   Closed - Won
-   Closed - Lost

</td></tr><tr><td>

Estimated Deal Size

</td><td>

Estimated monetary value of the opportunity. Includes currency selection.

</td></tr><tr><td>

Owner

</td><td>

Sales representative responsible for the opportunity.

</td></tr><tr><td>

Opportunity Group

</td><td>

Group or team associated with managing the opportunity.

</td></tr><tr><td>

Probability %

</td><td>

Likelihood of closing the opportunity, expressed as a percentage.

</td></tr><tr><td>

Description

</td><td>
