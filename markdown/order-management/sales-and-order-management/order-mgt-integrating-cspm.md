# Order Management diagnostics capability

Agents can use ServiceNow® Order Management diagnostics capability to diagnose and resolve service issues related to order tasks and domain orders.

Install these plugins:

-   Sales Customer Relationship Management plugin
-   Service test management plugin

    For more information about service test management, see [Setting up a test group](https://www.servicenow.com/docs/access?context=setting-test-group&version=zurich&pubname=zurich-proactive-service-exp-workflows&ft:locale=en-US) and [Setting up test definitions](https://www.servicenow.com/docs/access?context=settingup-test-specifications&version=zurich&pubname=zurich-proactive-service-exp-workflows&ft:locale=en-US).
