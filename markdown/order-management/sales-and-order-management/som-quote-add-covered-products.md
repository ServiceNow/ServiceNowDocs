# Add covered products to a quote

Agents can add a covered product to line items in a quote in Quote Management.

## Before you begin

Role required: sales\_agent or sales\_manager

## About this task

Covered products are products that are added to a warranty or entitlement. Once you have added a warranty or entitlement to your quote line items, the covered product option becomes available.

## Procedure

1.  In the CSM Configurable Workspace, select the **List** ![](../image/Lists.png) view.

2.  Navigate to **Quote** &gt; **All-Quotes** and select a quote that you want to add covered product too.

3.  From the **Quote line items** tab, select the quote line item.

4.  Select **More** &gt; **Covered product**.

5.  In the **Covered Product** tab, select **New**.

6.  In the **Create New Covered Product** form, select the covered product line item you want to add.
