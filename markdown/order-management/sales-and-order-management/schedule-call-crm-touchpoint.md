# Schedule a call from a touchpoint

Plan and track customer calls to organize and manage follow-ups and outreach and maintain a complete interaction history.

## Before you begin

The Omnichannel Callback for Customer Service Management \(com.sn\_omnichannel\_callback\) application must be installed to enable the click-to-call feature so agents can call customers from the touchpoint record. For more information, see [Install Omnichannel Callback for Customer Service Management](https://www.servicenow.com/docs/access?context=install-csm-omni-callback-app&version=australia&pubname=australia-customer-service-management&ft:locale=en-US).

Role required: sn\_crm\_touchpoint.touchpoint\_writer

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace**.

2.  Select the List icon ![](../../../reuse/icons/product-icons/list-outline-24.svg).

3.  Navigate to **Touchpoints** &gt; **All Touchpoints**.

4.  Open the touchpoint record.

5.  Navigate to the **Scheduled calls** tab.

6.  Select **New**.

7.  On the Create New Callback form, fill in the fields.

    For a description of the field values, see [Create new callback form](../reference/create-call-touchpoint-form.md).

8.  Select **Save**.


## Result

The call is scheduled and can be tracked from the touchpoint's **Scheduled calls** tab.
