# Create a quote from an opportunity

Create a quote directly from an opportunity using the Quotes tab.

## Before you begin

Ensure that you have captured and provided all the information for your opportunity, such as opportunity details, line items, and competitor's information.

Role required: sales\_agent

## Procedure

1.  Navigate to the **List** view ![](../image/Lists.png) and select **Opportunity - All**.

2.  Select the opportunity that you want to work with and select the **Quotes** tab.

3.  Select **Create Quote** to create a quote from the selected opportunity.
