# Create new appointment form

Field descriptions for editing or reviewing details on the Create New Appointment form in the CRM Touchpoints and Opportunity Management modules.

<table id="table_apt_form_33c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

From

</td><td>

Email ID of the user sending or organizing the appointment. This field is automatically set to the logged-in user.

</td></tr><tr><td>

To

</td><td>

Email ID of the recipient or attendee of the appointment.

</td></tr><tr><td>

Activity state

</td><td>

Current state of the appointment activity. The default state is Ready.

</td></tr><tr><td>

Start time

</td><td>

The date and time the appointment begins.

</td></tr><tr><td>

End time

</td><td>

The date and time the appointment ends.

</td></tr><tr><td>

Reminder \(minutes\)

</td><td>

How many minutes before the appointment start time a reminder notification is sent. The available options are:-   None
-   15 \(default\)
-   30
-   60
-   120

</td></tr><tr><td>

Subject

</td><td>

A brief description of the purpose of the appointment.

</td></tr><tr><td>

Location

</td><td>

The physical or virtual location where the appointment takes place.

</td></tr><tr><td>

Message

</td><td>

Additional details or notes about the appointment. This field supports rich text formatting.
