# Create a quote

Create a quote in Quote Management to start the quote building process.

## Before you begin

Role required: sales\_agent

## About this task

When you start a quote, a pop-up prompts you for quote information. You can choose to create a quote for a customer account or consumer. The pop-up changes depending on your choice.

-   Account – The quote is created for an existing customer account.
-   Consumer – The quote is created for an existing consumer.

## Procedure

1.  In the CSM Configurable Workspace, select the **List** ![](../image/Lists.png) view.

2.  Navigate to **Quote** &gt; **All-Quotes** and select **New**.

3.  In the **Create new quote** dialog box, fill in the fields.

    -   To create a quote for an account, select the **Account**.
    -   To create a quote for a consumer, select the **Consumer** name.
