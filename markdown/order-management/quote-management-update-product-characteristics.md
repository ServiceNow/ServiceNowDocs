# Update product characteristics

Use the Quote Characteristics tab to make updates to product characteristics.

## Before you begin

Role required: sales\_agent

## Procedure

1.  In the CSM Configurable Workspace, select the **List** \[Omitted image "Lists.png"\] Alt text: view.

2.  Select **Quotes** &gt; **All**.

3.  Select the line item for the product characteristic change.

4.  Select the Quote Line Characteristics tab.

5.  Select the quote characteristic that you want to update.

6.  Make the characteristic change and select **Save**.
