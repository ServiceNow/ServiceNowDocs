# Associate a field with a blueprint

Associate a field with a blueprint so that the field can be used in a configuration experience.

## Before you begin

Role required: Admin

## Procedure

1.  In the CPQ navigation pane, click **Blueprints**.

    \[Omitted image "cpq-blueprints.png"\] Alt text: Menu

2.  On the Blueprints page, click the name of the blueprint with which you intend to associate your field.

    \[Omitted image "cpq-blueprints-admin-click-name.png"\] Alt text: Blueprints

3.  On the Blueprint Admin page, in the Associated Fields tab, click **Associate fields**.

    \[Omitted image "cpq-blueprints-associate-fields-arrow.png"\] Alt text: Associate fields

4.  In the Associate Additional fields window, select the fields to associate with the blueprint, and then click **Done**.

    \[Omitted image "cpq-blueprints-associate-additional-fields.png"\] Alt text: Associate additional fields

    The fields associated with the blueprint are listed on the Fields tab of the blueprint administration page.

    \[Omitted image "cpq-blueprints-admin-fields-tab.png"\] Alt text: Admin fields tab
