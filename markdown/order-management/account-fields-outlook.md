# Account form in the ServiceNow CRM for Outlook add-in

Field descriptions for viewing or editing an account record from the ServiceNow CRM for Outlook add-in.

|Field|Description|
|-----|-----------|
|Name|Name of the account or company.|
|Website|Company website URL.|
|Number|System-generated unique identifier for the account record starting with the prefix ACCT.|
|Phone|Primary phone number for the account.|
|Primary Contact|Main contact person for the account.|
|Street|Street address of the account.|
|Parent Account|Parent account if this account is a subsidiary.|
|City|City where the account is located.|
|State / Province|State or province where the account is located.|
|Zip / Postal code|Postal or ZIP code for the account address.|
|Country|Country where the account is located.|
