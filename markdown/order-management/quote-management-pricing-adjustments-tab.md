# View manual pricing adjustments

Use the Pricing Adjustments tab in Quote Management to view and update manual pricing adjustments made to order line items in your quote.

## Before you begin

Role required: sales\_agent

## About this task

To create a pricing adjustment, see [Add pricing adjustment to a quote line item](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/order-management/quote-management-add-pricing-adjustment.md).

## Procedure

1.  In the CSM Configurable Workspace, select the **List** \[Omitted image "Lists.png"\] Alt text: view.

2.  Select **Quotes** &gt; **All**.

3.  Select the quote that you want to view in the **Quote List**.

4.  Select the **Pricing Adjustment** tab.

5.  Select the price adjustment line item that you want to update.

6.  Make the pricing adjustment.

7.  Select **Save**.

    The pricing adjustment is updated and reflected on the overall quote pricing field.
