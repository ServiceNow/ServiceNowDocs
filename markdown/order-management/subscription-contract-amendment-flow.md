# Capturing data from a configuration when amending a subscription contract

How to capture data from a prior configuration of a subscription contract.

When amending a pre-existing subscription contract, you may want to capture data relating to the prior configuration. Through additional fields added to the Configuration Quote Line and the CPQ Quote Line objects, you can access prior quantity and prior price for configuration items.
