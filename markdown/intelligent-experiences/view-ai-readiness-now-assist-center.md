# View your AI readiness assessment in Now Assist Center

Review the readiness assessments that the Now Assist Readiness Evaluation has identified for your instance.

## Before you begin

Before you can review your readiness assessments in Now Assist Center, you must first run the AI readiness assessment job. For more information, see Run the AI readiness assessment job in Now Assist Center.

Role required: sn\_na\_center.nac\_admin

## About this task

Follow these steps to review the AI readiness assessments based on your instance data.

## Procedure

1.  Navigate to **All** &gt; **Now Assist Center** or **Workspaces** &gt; **Now Assist Center**.

2.  Review the AI Readiness section of the home page to see the overall AI readiness assessment for the instance.

    The card displays a description of the assessment, the number of findings sorted by status, and a link to the AI Readiness Assessment page.

    \[Omitted image "now-assist-center-home-ai-readiness-overall.png"\] Alt text: Card showing the summary results of the overall AI readiness assessment.

3.  Select **View assessment** in the card or select AI readiness assessment \(\[Omitted image "icon-now-assist-center-nav-readiness.png"\] Alt text: AI readiness assessment icon.\) in the side navigation bar to view the complete list of AI readiness assessments.

    The AI Readiness Assessment tab opens showing a general assessment summary and a list of assessments for individual product areas.

    Each card shows the number of assessment findings, the readiness status for each assessment category, and a link to view the assessment details.

    \[Omitted image "now-assist-center-ai-readiness-page.png"\] Alt text: AI Readiness Assessment tab showing a general assessment summary and a list of assessments.

4.  Select the **Instance health assessment** card to view information about your instance status.

5.  Select **View assessment** in the card of an individual assessment.

    The View Assessment tab opens showing the detailed results of a readiness assessment for the specified area of your instance. It shows the summary results of the checks performed and lists the important tasks to achieve AI implementation readiness.

    \[Omitted image "now-assist-center-ai-readiness-details.png"\] Alt text: View Assessment tab showing the detailed results of a readiness assessment.

6.  Filter the list of tasks as needed.

    -   Select the category box and choose a category to filter by category.
    -   Select any combination of the filter option buttons to filter by readiness status and task type.
7.  Select the caret-down-outline icon \(\[Omitted image "icon-now-assist-center-expand.png"\] Alt text: Caret-down-outline icon.\) to view the task details.


## What to do next

Resolve the reported issues to improve AI implementation readiness.
