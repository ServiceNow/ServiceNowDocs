# Now Assist skills in the Platform workflow

Most Now Assist generative AI products include skills in the Platform workflow, such as product navigation. Some Now Assist products include skills for the conversational user and platform experience, as well as knowledge article recommendations.

![Platform skills are available in Now Assist Features when you install any Now Assist plugin.](../images/na-platform-skills.png "The Platform skills tile in the Platform workflow of the Now Assist Admin console")

-   **[Now Assist conversational experience](now-assist-platform-conv-experience.md)**  
Conversational experience generative AI skills can improve the user self-service experience. Platform Now Assist skills generate search summaries, order catalog items from the chat window, and create Virtual Agent topics that use large language model \(LLM\) topic discovery.
-   **[Knowledge content recommendation](now-assist-platform-knowledge.md)**  
Knowledge generative AI skills on the ServiceNow AI Platform provides recommendations for editing a knowledge article. Once activated, this skill is available on the Now Assist context menu.
-   **[Platform skills](../../virtual-agent/concept/na-platform-skills.md)**  
Now Assist includes special generative AI skills, such as the ability to search for records using plain language, available with the activation of any Now Assist plugin.
