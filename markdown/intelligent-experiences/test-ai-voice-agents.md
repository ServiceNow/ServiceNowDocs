# Test AI voice agents

Test your AI voice agents associated with a voice assistant from the Assistant Designer. You can make browser-based voice calls and view real-time analysis of AI voice agents invoked and tools executed during the conversation.

Use the voice testing experience in Assistant Designer to evaluate how your voice agents handle real conversations before deploying them. See [Test a voice assistant](https://www.servicenow.com/docs/access?context=test-a-voice-assistant&version=australia&pubname=australia-conversational-interfaces&ft:locale=en-US) for more information on testing voice agents.
