# Platform skills

Now Assist includes special generative AI skills, such as the ability to search for records using plain language, available with the activation of any Now Assist plugin.

To use these skills, activate one or more Now Assist plugins, such as Now Assist for Customer Service Management \(CSM\) or Now Assist for IT Service Management \(ITSM\).

\[Omitted image "na-platform-skill-navigation.png"\] Alt text: Navigation and other Platform skills are automatically activated. You can deactivate each skill manually.

-   **[Navigation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-global-navigation.md)**  
Use the navigation skill in Now Assist to take you where you want to go on the ServiceNow AI Platform.
