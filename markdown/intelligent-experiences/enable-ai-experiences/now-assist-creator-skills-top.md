# Now Assist skills in the Creator workflow

Now Assist for Creator includes a number of skills that can make developing on the ServiceNow AI Platform more efficient.

\[Omitted image "na-creator-skill-cards.png"\] Alt text: The Now Assist for Creator skill tiles are shown in the Now Assist Admin Creator workflow.

For more information, see [Now Assist for Creator](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/build-workflows/now-assist-for-creator/now-assist-for-creator-landing.md).

-   **[App generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-app-generation-skill.md)**  
App generation enables developers to kick start application building through conversation.
-   **[Catalog item generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-catalog-item-generation.md)**  
Use the Catalog item generation skill to create catalog items of your choice using Now Assist.
-   **[Intelligent code recommendations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-text-to-code.md)**  
Code generation is designed to empower developers to write scripts quickly with AI-generated code based on text or code prompts and can improve time to value for applications.
-   **[Data visualization generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-data-visualization-generation.md)**  
Data visualization generation lets users create Platform Analytics data charts through conversation.
-   **[Flow generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-flow-generation-skill.md)**  
Use the flow generation skill in Workflow Studio to create new flows.
-   **[Flow recommendations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-flow-recommendations-skill.md)**  
Generative AI recommendations can help you author flows by recommending possible components based on what you've entered.
-   **[Playbook generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-playbook-generation-skill.md)**  
Now Assist for Creator includes the playbook generation skill. Playbook generation gives generative AI capabilities to playbook authors, who can provide text directions to create a playbook outline with placeholder activities.
-   **[Refine content](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-refine-content.md)**  
Use the refine content skill to shorten, elaborate, rephrase, or generate content in the fields by using Now Assist.
-   **[Robotic Process Automation \(RPA\) bot generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-rpa-bot-generation.md)**  
Use this skill to create automations, activities, and automation logic from text instructions and preview options.
-   **[Spoke generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-spoke-generation-skill.md)**  
Generate spoke and create actions using Now Assist in Workflow Studio.
-   **[UI generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/intelligent-experiences/enable-ai-experiences/now-assist-ui-generation-skill.md)**  
Use this generative AI skill to quickly create experiences by describing what you want using natural language.
