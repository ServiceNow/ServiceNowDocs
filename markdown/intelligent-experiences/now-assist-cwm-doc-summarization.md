# Docs summarization in Collaborative Work Management

Gain insights into the page content by summarizing it or improve content quality by refining it in Docs for Collaborative Work Management \(CWM\).

Quickly summarize the content of a page in CWM Docs. Whether you're reviewing long documents or preparing for meetings, Doc summarization skill helps you stay informed and efficient.

With the Doc summarization skill, you can also elaborate and shorten selected content, resulting in improved content quality within the CWM Doc pages. Quickly shorten lengthy paragraphs, paraphrase bullet points, or expand on key points ensuring that your content is tailored to specific needs.

![Doc summarization in Collaborative Work Management using Now Assist.](../../../product/now-assist-cwm/images/cwm-na-doc-summarization.png)

The Doc summarization skill is supported starting with the Xanadu Patch 9 release and Collaborative Work Management v6.0.0.
