# AI asset inventory tab in AI Control Tower

The AI asset inventory tab contains all the AI-related assets used by an organization, including AI models, AI systems, prompts, and datasets.

## AI asset inventory dashboard

**Prerequisites**

Role required: AI steward \[sn\_ai\_governance.ai\_steward\]

The **AI asset inventory** tab provides a comprehensive view of the widgets. You can view the data in detail on each widget.

![AI asset inventory tab that displays all the metric types for AI systems.](../image/aict-ai-asset-inventory.png)

-   **AI systems by provider**

    This section shows the number of AI systems, categorized by the providers using a donut chart.

    **Note:** The AI system by provider donut chart doesn’t display the **Other** category when there are more than five providers, which can cause a mismatch between the number of metric labels and the items shown in the list when selecting a metric label.

-   **AI systems by type**

    This section shows all the AI systems categorized in

    -   Agentic AI
    -   Generative AI
    -   Classic AI
-   **All AI systems**

    This section shows all the AI systems classified into four stages:

    -   Steward review
    -   Approved for development
    -   Ready for deployment
    -   Deploy
-   **AI asset inventory by department**

    This section shows the number of AI systems classified in the **Providers** and **Components** using bar graph.
