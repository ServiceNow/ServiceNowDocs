# AI Gateway

Explore the AI Gateway page and its tab in the AI Control Tower Configurations.

AI Gateway page enables you to pause and resume AI Gateway transactions and monitor the AI connections. The AI Gateway page has the following tabs:

**Settings**- Allows to pause and continue the AI Gateway transactions.

**AI connections**- View the list of allowed AI connections.

**Global MCP clients**- View the list of Global MCP clients \(CIMD support\), Where CIMD refers to Client ID Meta Data. These Global MCP clients once created can be used across all MCP severs. You can Add Global MCP clients from the tab. To know about adding the MCP client, see .

You can **Export** the list into PDF, JSON, Excel, and CSV format. The list of allowed AI connections are categorized into.

-   Name
-   Allowed on AI Gateway
-   Last updated by
-   Updated

For information on AI Gateway tab on the AI Control Tower Home, see [AI Gateway tab in AI Control Tower](ai-gateway-tab.md)
