# Agentic AI - Assessment dashboard tab

The Now Assist Readiness Evaluation Dashboard's Agentic AI - Assessment tab helps determine agentic AI readiness for IT Service Management \(ITSM\) and Customer Service Management \(CSM\) by delivering automated, data-driven insights.

**Note:** For results to appear in this dashboard, ensure that you have completed the configuration steps first. For more information on configuration, see [Configuring Now Assist Readiness Evaluation](configuring-now-assist-readiness-evaluation.md).

The following list describes details of the Agentic AI - Assessment dashboard tab.

-   **Total Issues**

    This area of the dashboard shows the total issues identified.

-   **Issues - Trends**

    This area of the dashboard shows the trends over time in a bar chart for the products that need the most attention before implementation.

-   **Total Estimated Effort \(Days\)**

    This area of the dashboard shows the total estimated work effort in development days.

-   **Estimated Effort - Categorized**

    This area of the dashboard shows the estimated work effort needed to correct the issues for each product, depending on the issue category, in a bar chart.

-   **Issues - Categorized**

    This area of the dashboard shows the percentage of each issue category for each product in a bar chart.

-   **Estimated Effort - Trends**

    This area of the dashboard shows the trends over time in a bar chart for each products' estimated work effort.
