Learn about the ServiceNow AI Lens landing page.

</td><td>

Action

</td><td>

Preconfigured Lens actions that define behavior of ServiceNow AI Lens. Only the Lens actions that are configured to be triggered from the desktop and are assigned to you are visible on the landing page. If there are no Lens actions assigned to you, you will not see any Lens action on the landing page.

</td></tr><tr><td>

3

</td><td>

Call-to-action button

</td><td>

-   **Proceed with Lens** - Button to initiate the Lens session using the selected Lens action. This button is visible when a Lens action is selected.
-   **Capture with Lens** - Button to initiate the Lens session in a standalone mode. This button is visible when there is no Lens action selected.

</td></tr><tr><td>

4

</td><td>

Action card options

</td><td>

-   Pin and unpin - Pin option to make the Lens actions favorite on the landing page. The last pinned Lens action appears first in the list.
-   Details - Shows Lens action details, such as Name, Description, Default additional directions, and Table.
