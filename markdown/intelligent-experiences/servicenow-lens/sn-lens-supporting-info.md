# Supporting information for ServiceNow AI Lens

Get a quick overview of the important information that is related to the ServiceNow AI Lens application.

## Supported versions

ServiceNow AI Lens is supported from Yokohama Patch 6 and Zurich Patch 1.

## Supported user interfaces

To scan artifacts such as images, screens, websites, and applications using Now Assist, install the ServiceNow AI Lens application and then activate the ServiceNow AI Lens skill. For more information, see [Configure ServiceNow AI Lens](../task/install-sn-lens.md).

The ServiceNow AI Lens application requires a Now Assist for Creator or Now Assist for Platform license.

## Application information

Activate the ServiceNow AI Lens store app \(com.sn\_ai\_lens\) to use the ServiceNow AI Lens skill.

This store app has the following dependencies:

-   ServiceNow AI Lens Core \(sn\_app\_lens\_core\)
-   Now Assist for Platform \(sn\_genai\_platform\)
