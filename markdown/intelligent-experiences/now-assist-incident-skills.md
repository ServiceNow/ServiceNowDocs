# Now Assist content and record summary skills

Now Assist skills can provide generative AI summarization capabilities for cases, incidents, and other records on the ServiceNow AI Platform.

**Note:** If you're looking to add summarization to custom or child tables, you can create a new skill in Now Assist Skill Kit. See [Create a skill](../../now-assist-skill-kit/task/create-new-skill.md).

![Incident skill tiles are available in some Now Assist Admin workflows.](../images/incident-skills-tile-na-admin.png "Example Incident skills tile in Now Assist Admin")

![The summarization for fulfillers skill tile is available in Now Assist for Sourcing and Procurement Operations (SPO) in the Finance & Supply Chain workflow.](../images/spo-fulfiller-summarization-tile.png "Example summarization skill tile for Now Assist for Sourcing and Procurement Operations (SPO)")

![Content summary skill tiles are available in Now Assist for Strategic Portfolio Management (SPM) in the Technology workflow.](../images/spm-genai-skill-tiles.png "Example content summary skill tiles for Now Assist for Strategic Portfolio Management (SPM)")

-   **[Record summarization](now-assist-case-summary.md)**  
You can quickly learn the details of a record from record summarization generated by Now Assist. By reading a Record summarization, you can get details about a case or incident and can save your requester from having to repeat the same information that the requester has already provided.
-   **[Change request summarization](now-assist-change-request-summarization.md)**  
Use the change request summarization skill to quickly capture important details of the change request, including the current status.
-   **[Chat summarization](now-assist-chat-summary.md)**  
You can quickly learn the details of a chat from a chat summarization generated by Now Assist. By reading a chat summarization, you can get enough details about the chat and can save your requester from having to repeat the same information that the requester already provided to Virtual Agent.
-   **[Configuration item \(CI\) summarization](na-cmdb-config-item-summarization.md)**  
Configuration items \(CIs\) are a central element in numerous features such as Incident Management and Event Management, and they are associated with the respective incidents and alerts. This skill gathers all of those incidents, alerts, security vulnerabilities, and associated records into a single summary.
-   **[Correlation insights for records in Security Operations](now-assist-correlation-insights-security-operations.md)**  
Generate correlation insights from the Now Assist panel to help you connect past events to the security incidents you are working on.
-   **[Docs summarization in Collaborative Work Management](../../now-assist-skills/concept/now-assist-cwm-doc-summarization.md)**  
Gain insights into the page content by summarizing it or improve content quality by refining it in Docs for Collaborative Work Management \(CWM\).
-   **[Feedback and multi feedback summarization](now-assist-feedback-summary.md)**  
Summarize one or multiple feedbacks received using the feedback summarization or multi feedback summarization that is generated by Now Assist. By reading a feedback summary, your product managers can get enough details to improve the product features, usability, and performance.
-   **[Fulfiller summarization for Sourcing and Procurement Operations](now-assist-summarization-fulfillers-spo.md)**  
Summarize requests and their related records to quickly update fulfillers on their progress and open action items.
-   **[Gen AI Docs](now-assist-genai-doc-skills.md)**  
Summarize content in documents using Gen AI Docs that is generated by Now Assist. By reading a document summary, you can get enough details to improve the features, usability, and performance of a product or project.
-   **[Email project summary](now-assist-email-summary-skill.md)**  
Schedule the project summary emails to help Project Managers track key elements such as milestones, resource assignments, tasks, financials, RIDAC \(Risk, Issue, Decision, Action, Change request\), and overall project metrics.
-   **[Sidebar discussion summarization](now-assist-sidebar-discussion-summarization.md)**  
Generate a summary of Sidebar discussions between agents, requesters, and subject matter experts using the Sidebar discussion summarization skill.
-   **[Test summarization](now-assist-test-summarization.md)**  
Use this skill to create a test run summary after the test is executed. It includes the main points covered during the test execution, including the test output, test interpretation, and other defined test parameters.
-   **[Work order task closure summarization](now-assist-work-order-task-summary.md)**  
Field service agents can create detailed notes and close work order tasks faster with help from Now Assist.
