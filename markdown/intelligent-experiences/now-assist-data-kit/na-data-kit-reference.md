# Now Assist Data Kit reference

Reference topics for Now Assist Data Kit.

-   **[Now Assist Data Kit roles \(sn\_data\_kit.analyst\)](../reference/data-kit-analyst-role.md)**  
This user can edit and save ground truth in Now Assist Data Kit. This role has read only access within Now Assist Data Kit.
-   **[Now Assist Data Kit roles \(sn\_data\_kit.admin\)](../reference/data-kit-admin-role.md)**  
This user can create, update, and publish datasets in Now Assist Data Kit. This role is mandatory to use Now Assist Data Kit.
