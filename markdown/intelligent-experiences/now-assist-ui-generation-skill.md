# UI generation

Use this generative AI skill to quickly create experiences by describing what you want using natural language.

To generate a UI experience, describe the table, chart type, and navigation your users will be working with. For best results, include as much detail as possible.

Once generated, you can review the generated experience and accept or reject it. Once accepted, you can modify the experience description to achieve the desired results.

![The Incident Management main page for a new experience shows that it was generated by Now Assist.](../../ui-generation/image/ui-generation-main-page.png "Example main page for a new experience")

For more information, see [Experience generation](https://www.servicenow.com/docs/access?context=generate-ui&version=xanadu&pubname=xanadu-application-development&ft:locale=en-US).
