# Configure Task Intelligence

Configure and install the Task Intelligence Admin Console and its related applications.

## Configuration overview

The Task Intelligence Admin Console is installed along with Task Intelligence for CSM or Task Intelligence for ITSM. See [Install Task Intelligence Admin Console](../task/install-task-intelligence.md) for more details.

-   **[Configure Task Intelligence for ITSM](https://www.servicenow.com/docs/access?context=configure-task-intelligence-for-itsm&version=yokohama&pubname=yokohama-it-service-management&ft:locale=en-US)**

    Configure Task Intelligence for ITSM so that agents can use the application to fulfill their requirements efficiently.

-   **[Configure Task Intelligence for Customer Service](https://www.servicenow.com/docs/access?context=configure-task-intelligence&version=yokohama&pubname=yokohama-customer-service-management&ft:locale=en-US)**

    Install the Task Intelligence for Customer Service application and configure the different features: case categorization, language detection, sentiment analysis, and Document Intelligence for Customer Service.
