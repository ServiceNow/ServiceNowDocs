# Manage machine learning models with Task Intelligence

Create, edit, train, and assess machine learning models to make predictions for customer service cases and incidents.

-   **[Managing Task Intelligence for ITSM models](https://www.servicenow.com/docs/access?context=using-task-intelligence-itsm&version=yokohama&pubname=yokohama-it-service-management&ft:locale=en-US)**

    Use the machine learning capabilities of Task Intelligence for ITSM to predict and recommend incident field information and help with incident categorization.

-   **[Use Task Intelligence for Customer Service](https://www.servicenow.com/docs/access?context=use-task-intelligence&version=yokohama&pubname=yokohama-customer-service-management&ft:locale=en-US)**

    Use the Task Intelligence features to create field prediction and sentiment models for cases, and review Task Intelligence analytics and prediction history.
