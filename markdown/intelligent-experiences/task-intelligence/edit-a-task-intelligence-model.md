# Edit a Task Intelligence model

Retrain your machine learning models or edit what they predict to better align with your business goals.

-   **[Edit a model to predict incident fields](https://www.servicenow.com/docs/access?context=edit-your-model&version=zurich&pubname=zurich-it-service-management&ft:locale=en-US)**

    Edit the incident prediction model that has already been trained and deployed. Change the model configurations, view the updated training results, and redeploy the model.

-   **[Create and edit a case field prediction model](https://www.servicenow.com/docs/access?context=csm-task-intel-create-cat-solution&version=zurich&pubname=zurich-customer-service-management&ft:locale=en-US)**

    Create, train, and edit a model to predict fields for customer service cases.

-   **[Create a model to predict case sentiment](https://www.servicenow.com/docs/access?context=csm-task-intel-create-sentiment-solution&version=zurich&pubname=zurich-customer-service-management&ft:locale=en-US)**

    Edit and test the pre-trained sentiment model to predict sentiment for customer service cases.

-   **[Create a model to detect case language](https://www.servicenow.com/docs/access?context=csm-task-intel-create-language-solution&version=zurich&pubname=zurich-customer-service-management&ft:locale=en-US)**

    Edit and test the pre-trained model to detect the language used to create customer service cases.
