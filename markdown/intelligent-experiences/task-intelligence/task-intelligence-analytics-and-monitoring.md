# Task Intelligence Analytics and Monitoring

Use the Task Intelligence Admin Console to monitor, analyze, and assess the performance of your machine learning model.

-   **[Task Intelligence for ITSM Analytics and Monitoring](https://www.servicenow.com/docs/access?context=task-intel-monitor-analystics&version=australia&pubname=australia-it-service-management&ft:locale=en-US)**

    View the impacts of your Task Intelligence models. Monitor model performance overtime, track business value, and view what predictions your agents did and didn't use.

-   **[View Analytics for Task Intelligence for CSM](https://www.servicenow.com/docs/access?context=csm-task-intel-case-monitoring&version=australia&pubname=australia-customer-service-management&ft:locale=en-US)**

    View Task Intelligence for Customer Service analytics and see how your machine learning models are working.

-   **[View Task Intelligence prediction history](https://www.servicenow.com/docs/access?context=csm-task-intel-prediction-history&version=australia&pubname=australia-customer-service-management&ft:locale=en-US)**

    View details about historical predictions in the Predictor Results for CSM models.
