# Playbook generation

Now Assist for Creator includes the playbook generation skill. Playbook generation gives generative AI capabilities to playbook authors, who can provide text directions to create a playbook outline with placeholder activities.

Playbook generation is a skill installed with the Now Assist for Creator \(sn\_now\_creator\) application. You can install this application from the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

Access the playbook generation skill when creating a new playbook in Workflow Studio.

![You have the option to build a playbook with Now Assist in Workflow Studio, or to build one from scratch.](../../process-automation-designer/images/new-playbook-now-assist.png)

For more information, see [Playbook Assist](https://www.servicenow.com/docs/access?context=playbook-assist-landing&version=xanadu&pubname=xanadu-build-workflows&ft:locale=en-US).

**Note:** Playbook authors must be assigned the **now.assist.creator** role to use playbook generation. See [Playbook Assist roles](https://www.servicenow.com/docs/access?context=playbook-assist-roles&version=xanadu&pubname=xanadu-build-workflows&ft:locale=en-US) for more information.
