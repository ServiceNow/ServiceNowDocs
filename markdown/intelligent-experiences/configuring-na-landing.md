# Configuring Now Assist settings and features

Use the Now Assist Admin console to activate the various Now Assist applications and skills that you’re entitled to.

## Configuration overview

Use the Settings page in the Now Assist Admin console to activate the plugins, turn on the Now Assist panel, and view the account settings. You must install at least one Now Assist application before you can configure any skills.

The following example shows the Settings page with four available plugins to install, including Now Assist for HRSD and Now Assist for Creator.

![Now Assist settings page that lets you install plugins, activate the Now Assist panel, and view account details.](../image/config-now-assist-1.png "Now Assist Admin console settings page")

-   **[Install Now Assist plugins](../task/install-now-assist-feature-plugins.md)**  
Install Now Assist plugins to enable generative AI on your instance.
-   **[Turn on the Now Assist panel](../task/activate-now-assist-panel.md)**  
Turn on the Now Assist panel to enable your agents to use Now Assist skills, such as task summarization, recommendation or navigation, in a side panel on the user interface.
-   **[Activate a Now Assist skill](../task/configure-a-now-assist-skill.md)**  
Configure the triggers, settings, and display locations for Now Assist skills to enable generative AI capabilities across the ServiceNow AI Platform.
-   **[Edit a Now Assist skill](../task/edit-a-now-assist-skill.md)**  
Edit the configuration of a Now Assist skill to choose the inputs or triggers and the display location of the skill output.
-   **[Make a copy of a Now Assist skill](../task/make-a-copy-of-a-now-assist-skill.md)**  
The 'Make a copy' feature enables you to create a copy of a Now Assist skill so that you can experiment with skill settings and configure the skill to fit your business needs.
-   **[Troubleshoot a Now Assist skill](../task/troubleshoot-a-now-assist-skill.md)**  
Run diagnostics for a skill on the Now Assist Admin console and get information about the status of your skill configuration.
-   **[Enable translation for Now Assist applications](../task/enable-dynamic-translation-for-now-assist-applications.md)**  
Turn on multi-language support for user-entered text with Dynamic Translation in Now Assist applications.
-   **[Review Now Assist account information](../task/review-now-assist-account-information.md)**  
Review your Now Assist license details on the Account page of the Now Assist Admin console to make sure that you're up to date on what's available to you.
-   **[Activate offensiveness protection for generative AI](../task/activate-offensiveness-protection-for-generative-ai.md)**  
Turn on offensiveness protection to log and add the option to block offensive content in AI-generated text and conversations.
-   **[Configure prompt injection attack protection](../task/configure-prompt-injection-attack-protection.md)**  
Activate or deactivate prompt injection attack protection for AI-generated text and conversations.
-   **[Configure sensitive topic filters](../task/configure-sensitive-topic-filters.md)**  
Set up filters to redirect users to a different topic if certain subject material is detected in a Virtual Agent conversation.
-   **[Export Now Assist Guardian logs](../task/export-now-assist-guardian-logs.md)**  
Export logs from Now Assist Guardian to get insights into how often different guardrails are being detected and used.
-   **[Manage large language models](../task/manage-large-language-models.md)**  
Access and select the LLM \(large language model\) used for various Now Assist skills. The selection impacts all the skills within the capability.
