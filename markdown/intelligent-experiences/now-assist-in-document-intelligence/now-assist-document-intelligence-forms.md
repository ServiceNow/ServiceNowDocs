# Now Assist in Document Intelligence forms

Use forms to view and update Now Assist in Document Intelligence information.

-   **[Question form for document Q&amp;A setup](document-qna-single-field-form.md)**  
The Question form for document Q&amp;A enables you to define a question you want to ask about the document.
-   **[Field form for document extraction setup](document-extraction-single-field-form.md)**  
The Field form enables you to define a single field for extraction.
-   **[Table form for document extraction setup](document-extraction-table-form.md)**  
The Table form enables you to define a table for extraction.
