# Field types in Now Assist in Document Intelligence

The field type specifies the information that is retrieved from a document with Now Assist in Document Intelligence.

The following field types are available to administrators when configuring fields for use cases.

**Note:** Some field types convert the extracted value into a standard format. For more information, see [Data normalization](../concept/data-normalization.md).

<table id="table_nql_cxs_12c"><thead><tr><th>

Field type

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Boolean \(True/False\)

</td><td>

Holds a true/false value. In document Q&amp;A questions, the value is displayed as a yes or no answer. The answer can be followed by an explanation generated by Now Assist, if enabled by the admin.

 This field type is available for document Q&amp;A use cases.

 ![Boolean question field type.](../image/docintel-na-field-question.png)

</td></tr><tr><td>

Date

</td><td>

Date, displayed in the date format extracted from the document.

 This field type is available for document extraction use cases.

 ![Date field type.](../image/docintel-na-field-date.png)

</td></tr><tr><td>

Decimal

</td><td>

A number with up to two decimal places \(for example, 12.5 or 12.55\).

 This field type is available for document extraction use cases.

 ![Decimal field type.](../image/docintel-na-field-decimal.png)

</td></tr><tr><td>

Float \(floating point number\)

</td><td>

A number with up to seven decimal places \(for example, 12.0 to 12.0000000\).

 This field type is available for document extraction use cases.

 ![Float field type.](../image/docintel-na-field-float.png)

</td></tr><tr><td>

Integer

</td><td>

Whole number \(for example, 12\).

 This field type is available for document extraction use cases.

 ![Integer field type.](../image/docintel-na-field-integer.png)

</td></tr><tr><td>

Reference

</td><td>

A reference field stores a reference to a field on another table. For example, the Caller field on the Incident table is a reference to the User \[sys\_user\] table.

 For more information, see [Reference field type](https://www.servicenow.com/docs/access?context=c_ReferenceField&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US).

 This field type is available for document extraction use cases.

 ![Reference field type.](../image/docintel-na-field-reference.png)

</td></tr><tr><td>

Text

</td><td>

Text field.

 This field type is available for document extraction and document Q&amp;A use cases.

 ![Text field type.](../image/docintel-na-field-text.png)
