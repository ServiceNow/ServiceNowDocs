# Reviewing a legal document with Now Assist in Document Intelligence

Use Now Assist in Document Intelligence so that generative AI can provide you with a document analysis for contract reviews.

Now Assist uses generative AI capabilities to identify the specified clauses in the document text and provide recommendations for the standardized text.

For the Xanadu release, Now Assist in Document Intelligence includes the contract metadata extraction skill and the contract analysis skill. These skills are available with the Now Assist in Contract Management workflows. For more information, see [Now Assist in Contract Management](https://www.servicenow.com/docs/access?context=cncore-now-assit-landing&version=xanadu&pubname=xanadu-employee-service-management&ft:locale=en-US).
