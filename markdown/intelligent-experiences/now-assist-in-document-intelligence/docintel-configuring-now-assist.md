# Configuring Now Assist in Document Intelligence

If you have the admin role, you can configure the Now Assist in Document Intelligence feature, enabling agents to use its generative AI skills within their application workspace.

For the Xanadu release, there is no stand-alone configuration of Now Assist in Document Intelligence. The contract metadata extraction skill and the contract analysis skill contain the Now Assist in Document Intelligence feature. These skills are available with the Now Assist in Contract Management workflows. For more information, see [Now Assist in Contract Management](https://www.servicenow.com/docs/access?context=cncore-now-assit-landing&version=xanadu&pubname=xanadu-employee-service-management&ft:locale=en-US).
