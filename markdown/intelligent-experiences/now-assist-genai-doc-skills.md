# Gen AI Docs

Summarize content in documents using Gen AI Docs that is generated by Now Assist. By reading a document summary, you can get enough details to improve the features, usability, and performance of a product or project.

The Now Assist for Strategic Portfolio Management \(SPM\) application includes the skills and features of generative AI that enable you to generate a summary from the documents. With the Gen AI Docs skills, you can eliminate the need to analyze documents manually. The following skills are available:

-   EAP Teams Gen AI Docs
-   Planning item Gen AI Docs
-   Project Gen AI Docs

**Note:** Now LLM Service is the default provider for these Now Assist skills. You can also choose the Azure OpenAI AI provider. For more information, see [Configuring Generative AI Controller](../../generative-ai-controller/task/configure-generative-ai-controller.md).
