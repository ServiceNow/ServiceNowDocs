# Supporting information for Agentic Desktop

Get a quick overview of the important information that is related to the Agentic Desktop application.

## Supported versions

Agentic Desktop is supported from Zurich Patch 4.

## Supported user interfaces

To design and run AI-powered desktop actions using Now Assist AI agents, install the Agentic Desktop application. For more information, see [Configure Agentic Desktop](../task/configure-agentic-desktop.md).

## Licensing requirements

The Agentic Desktop application requires a ServiceNow Pro Plus or Enterprise Plus license.

## Application information

This app has the following dependencies:

-   Agentic Desktop Core \(sn\_desktop\_core\)
-   Now Assist AI agents
