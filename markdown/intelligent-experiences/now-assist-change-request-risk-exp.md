# Change request risk explanation

Use this skill to show the explanation of the calculated risk rating for a change request.

Use the Now Assist icon ![Now assist icon.](../../../common/image/icon-ai-sparkle.png) on the Risk field of the change request to calculate the risk. Results are low, medium, or high.

![Change request risk explanation in Service Operations Workspace.](../../../product/now-assist-itsm/image/now-assist-chg-risk-explanation.png)

For more information, see [Explain the risk of a change request by using Now Assist for IT Service Management \(ITSM\)](https://www.servicenow.com/docs/access?context=change-risk-exp-now-assist&version=xanadu&pubname=xanadu-it-service-management&ft:locale=en-US).
