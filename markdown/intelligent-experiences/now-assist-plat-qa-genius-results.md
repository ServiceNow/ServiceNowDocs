# Now Assist Q&amp;A Genius Results

Use this skill to help users find answers to their questions in a conversational format. Now Assist summarizes the Q&amp;A Genius Result answers with references.

Now Assist in AI Search uses the Now LLM to extract actionable Q&amp;A Genius Result answers from knowledge article results found in Service Portal, Virtual Agent, Employee Center, and global searches. When a user submits a question in search, AI Search retrieves the top knowledge article result and passes it to the Now LLM for answer generation. Answers augment the user's search results, displaying as actionable Now Assist Q&amp;A Genius Result answer cards. For reference, each answer card includes a link to its source knowledge article.
