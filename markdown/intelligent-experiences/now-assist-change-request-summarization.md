# Change request summarization

Use the change request summarization skill to quickly capture important details of the change request, including the current status.

A change request summary provides you with a concise summary of a change request. The summary is based on the change request state and is generated from the information in the related table.

![The change request summary is shown at the top of the form. This example includes the Objective, Plan, and Risk summaries.](../../../product/now-assist-itsm/image/now-assist-change-summary.png "Change request summary in Now Assist for IT Service Management (ITSM)")

## Generating a change request summary

You can generate a change request summary in the following products.
