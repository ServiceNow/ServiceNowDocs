# Delete a document extraction use case

Delete a use case when it’s no longer needed for your documents.

## Before you begin

Role required: sn\_docintel.manager

## Procedure

1.  Navigate to **All** &gt; **Document Intelligence** &gt; **Document Data Extraction Administration** &gt; **Use Cases**.

2.  In the list, select the display name of the use case that you want to delete.

3.  On the use case screen, select the options icon \(![Options icon](../image/icon-docintel-field-options-menu.png)\) and select **Delete**.

4.  In the confirmation box, select **Delete**.


## Result

The use case is deleted, along with the related fields, field groups, integrations, flows, and machine learning \(ML\) models.
