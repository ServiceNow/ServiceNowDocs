# Now Assist content management skills

These Now Assist skills provide generative AI capabilities for chat features on the ServiceNow AI Platform.

The Contracts skills in the Employee workflow area are available when you install Now Assist for Legal Service Delivery \(LSD\). These skills work in Contract Management Pro.
