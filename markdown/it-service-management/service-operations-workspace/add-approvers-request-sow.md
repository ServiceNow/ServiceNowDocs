# Add approvers to approve a request in SOW

Assign one or more approvers to the request or requested items to expedite the process and ensure that the user's request is handled quickly.

## Before you begin

Role required: sn\_request\_write, admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  Navigate to **List** &gt; **Requests** &gt; **Open requests**.

3.  Select a request item.

4.  Select the **Related records** tab.

5.  In the navigation menu, select **Approvers**.

6.  Select **Add**.

    A modal appears that contains the list of users.

7.  Select the user who must be added as an approver.

8.  Select **Add**.
