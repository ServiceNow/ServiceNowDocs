# Reorder the navigation pane modules in Service Operations Workspace for ITSM

Quickly access the navigation pane modules in Service Operations Workspace for ITSM. For example, Home, List, Inbox, and so on.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Now Experience Framework** &gt; **UI Builder**.

2.  From the **Experiences** list, select **Service Operations Workspace**.

3.  On the Service Operations Workspace page, select **Settings** at the top.

4.  On the Experience Settings: Service Operations Workspace page, navigate to the **Side navigation** section.

    This section displays all modules left navigation pane modules in Service Operations Workspace for ITSM.

5.  Drag the modules in the required order.
