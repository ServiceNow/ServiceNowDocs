# Exploring Password Reset

The Password Reset for Service Operations Workspace application enforces strong and secure passwords by enabling end users to reset or change their passwords either by using the self-service process or by requesting the service desk agent.

To configure service desk assisted password reset, see [Setting up Service Desk assisted Password Reset in Service Operations Workspace](setup-password-reset-sow.md).
