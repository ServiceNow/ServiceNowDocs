# CSDM framework for HAM

Common Service Data Model \(CSDM\) is a standard and consistent set of terms and definitions to adopt across all ServiceNow products on the ServiceNow AI Platform.

Using CSDM standard and consistent set of terms and definitions helps to effectively track assets through their life cycle transitions. To know more about CSDM, see .

To set up CSDM environment for HAM, see Set up the CSDM environment.

**Note:** It is optional to adopt CSDM. You can either adopt CSDM or continue using the existing model.
