# Review a change template

Review a change template proposed for common change requests.

## Before you begin

Role required: change\_manager

You must have role configured with access to approve the templates associated with the change model.

## About this task

The approval process for change templates is determined by the publish approval flow set by administrators. For more information, see [Change template management flows](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-service-management/change-management/change-template-management-flows.md).

## Procedure

1.  Navigate to **All** &gt; **Change** &gt; **Models** &gt; **Change Templates**.

2.  Select the template assigned to you to approve.

3.  Review a proposed template.

    -   To approve a proposed template, either select **Approved** in the **State** column or navigate to the **Approvers** tab, select the template in the Requested state and then select **Approve**.

        After approval, the template state changes to Published.

    -   To reject a proposed template, either select the **Rejected** option in the **State** column or navigate to the **Approvers** tab, select the template in the Requested state and then select **Reject**.

        After rejection, the template state changes to Draft.
