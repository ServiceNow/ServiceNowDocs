# Change Management workflows

The Change Management processes for a change request are controlled by workflows defined for a specific change request type. You can use the graphical Workflow Editor to modify one of the default workflows, or to create additional change request workflows.

By default, six change request workflows are provided.

|Workflow|Description|
|--------|-----------|
|Change Request - Emergency|The workflow process that an emergency change request follows.|
|Change Request - Emergency change tasks|The child workflow process that handles the creation and management of the change tasks for the emergency change request workflow.|
|Change Request - Normal|The workflow process that a normal change request follow.|
|Change Request - Normal change tasks|The child workflow process that handles the creation and management of the change tasks for the normal change request workflow.|
|Change Request - Standard|The workflow process that a standard change request follow.|
|Change Request - Standard change tasks|The child workflow process that handles the creation and management of the change tasks for the standard change request workflow.|
