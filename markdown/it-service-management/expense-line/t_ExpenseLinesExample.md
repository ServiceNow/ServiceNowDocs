# View an expense

Expense lines can be used in various ways, for example, how to view expenses that are associated with a given contract.

## Before you begin

Role required: asset or contract\_manager

## About this task

## Procedure

1.  Navigate to **Contract** &gt; **Contract** &gt; **All**.

2.  Select a contract.

3.  View the **Expense Lines** related list.

    All the expense lines for the contract are listed. The total cost of the contract and its current value are also shown.

4.  Select an expense line to view its details.
