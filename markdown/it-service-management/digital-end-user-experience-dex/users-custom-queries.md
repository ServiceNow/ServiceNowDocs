# Custom queries

Use the custom queries to access operating system information using SQL, enabling in-depth system analysis and issue resolution.

Troubleshoot problems or collaborate with administrators by utilizing predefined Operation system instrumentation framework queries \([OSQuery](https://osquery.io/)\) or write a custom query if necessary.

You can choose a query from the **Queries** drop-down list and select **Run** to obtain the most recent response from the device.

**Note:** Custom queries are stored in the Device Custom Query \[dex\_ci\_device\_custom\_query\] table. When you execute a query, the list displays a maximum of 100 entries retrieved from that table. For the complete set of response entries, refer to the Device Custom Query Response \[dex\_ci\_device\_custom\_query\_response\] table.
