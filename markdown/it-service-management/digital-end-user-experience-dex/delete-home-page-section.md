# Delete a section from the Home page

Delete a section when you no longer need it in.

## Before you begin

Role required: sn\_dex\_desktop.admin

## Procedure

1.  Navigate to **Desktop Assistant** &gt; **Configuration** &gt; **Home**.

2.  Under Desktop Assistant Section, select the check box that appears before the name of the section that you want to remove.

3.  From the Actions on selected rows drop-down list, select **Delete**.

    \[Omitted image "desktop-exp-delete-section.png"\] Alt text: The option to delete appears in the Actions on selected row drop-down list.

4.  In the Confirmation window, select **Delete**.


## Result

The section is removed from the home page.
