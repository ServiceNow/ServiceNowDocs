# Users — Application Performance

View the 10 applications with the most memory and CPU usage and the performance and usage data associated with a specific application.

Application performance provides valuable insights into how the application is used, and how well it's performing. Use this information to decide where you can make improvements to enhance the user experience and achieve business objectives.

For more information on the application performance and usage cards, see [Application performance pages](../reference/user-application-metrics-cards.md).
