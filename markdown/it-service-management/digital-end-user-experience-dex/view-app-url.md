# View the domain or URL for a web application

View the domain or web address of a web application to access the application and use it effectively.

## Before you begin

Confirm that the DEX plugin \(sn\_dex\) is installed.

Role required: sn\_dex.user or ITIL

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  In the primary navigation pane, select the DEX Administration icon \(\[Omitted image "icon-administration.png"\] Alt text: The DEX Administration icon.\).

3.  Select **Configure applications** on the Application management card.

    The list of monitored applications appears.

4.  Select the application for which you want to view the application URL.

    The URLs appear in the **What domains should be monitored?** field.
