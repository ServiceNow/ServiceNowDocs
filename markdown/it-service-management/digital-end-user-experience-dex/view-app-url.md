# View the domain or URL for a web/SaaS application

View the domain or web address of a web/SaaS application to access the application and use it effectively.

## Before you begin

Confirm that the DEX plugin \(sn\_dex\) is installed.

Role required: sn\_dex.user or ITIL

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace** and from the left pane, select the DEX Administration icon \(\[Omitted image "icon-administration.png"\] Alt text: The DEX Administration icon.\).

2.  Select the **Manage applications** card.

    The list of monitored application appears.

3.  Select the application for which you want to view the application URL.

    The URL appears in the **Domain/URL** field.
