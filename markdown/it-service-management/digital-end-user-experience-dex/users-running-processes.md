# Running processes

View the device CPU and memory usage, idle CPU percentage, and number of processes along with other data points on the performance and behavior of running processes. By monitoring these metrics, you can gain insights into how running processes are performing and identify performance bottlenecks or issues that might be impacting the digital experience.

The system updates the running processes data using policies every 72 hours. Additionally, you can manually populate the data on demand by selecting the refresh icon \(![Refresh icon used to refresh the page.](../image/icon-dex-refresh.png)\). To learn more about the policies, see [DEX policies for macOS](../reference/dex-policies-mac.md) and [DEX policies for Windows](../reference/dex-policies-windows.md).

For more information on each card, see [Running processes page](../reference/user-running-process-cards.md).
