# Installed application — Number of crashes section

View the number of crashes to gauge the application's stability, identify performance trends, and make informed decisions to enhance its reliability and user experience.

Current number of crashes

</td><td>

Measure of the application's stability and reliability. A lower count indicates a smoother and more dependable device, as it suggests that the application experiences fewer unexpected failures or interruptions.

 **Note:** The metric supports the Zoom application as well as applications that emit a window app crash event \(event id = 1000\) on crashing such as Microsoft OneDrive, Microsoft Teams, Microsoft Excel, Microsoft PowerPoint, Microsoft OneNote, Microsoft PowerPoint, Microsoft Outlook, and Microsoft Word.

</td></tr><tr><td>

Number of crashes - &lt;time&gt;**Note:** The &lt;time&gt; range of the performance data can cover the past 2 hours, 24 hours, or 15 days, depending on the timeframe you select from the provided drop-down list in the Performance section.

</td><td>

Trend of crash count over a specific duration.The trend reflects the application's stability and reliability, with fluctuations indicating changes in performance.
