# Change Home page title and description

Change the title and description of the Home page, as required.

## Before you begin

Confirm that the DEX Desktop Assistant \[sn\_dex\_desktop\] application has been installed. For more information, see [Install Application and Device Health](install-app-device-health.md) and [Download and install Desktop Assistant](download-desktop-experience.md).

Role required: sn\_dex\_desktop.admin

## Procedure

1.  Navigate to **Desktop Assistant** &gt; **Configuration** &gt; **Application**.

    ![The Desktop Assistant page that provides options to modify your home page title or brief description, as well as activate notifications and the chat icon.](../image/dex-da-home-page-form.png)

2.  On the form, edit the required fields.

    |Field|Description|
    |-----|-----------|
    |Title|Title of the Home page.|
    |Description|Brief overview of the content and purpose of the product.|
    |Employee Center URL|URL to the Employee Center where employees can access important resources, information, and tools related to their work.|
    |Notifications Enabled|Option to enable or disable the notification bell icon.|
    |Use Virtual Agent \(must be configured\)|Option to enable or disable the chat icon.|

3.  Select **Update**.


## Result

The Home page information is updated.
