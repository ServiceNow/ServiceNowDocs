# Monitor system performance

Monitor how your organization's systems are performing. Tracking CPU and memory usage helps prevent bottlenecks that slow down applications and helps ensure that the system can handle workloads without running out of resources. Disk performance monitoring identifies potential storage issues, such as slow read or write speeds, while I/O monitoring helps detect delays in data transfer.

## Before you begin

Confirm that the DEX plugin \(sn\_dex\) is installed.

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  Select the Insights icon \(\[Omitted image "icon-insights.png"\] Alt text: Insights icon\) in the primary navigation pane.

3.  Select **System performance**.

    For more information on the system performance metrics, see [DEX Insights reports](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-service-management/digital-end-user-experience-dex/insights-reports.md).
