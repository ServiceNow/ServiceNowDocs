# Access

View essential information regarding the user login details. This tab provides the user name, user type, login type, and the timestamp of the last login.

The user name serves as a unique identifier within the system, enabling users to identify themselves. The user type field indicates whether the user is active. The login type field reveals the method or protocol used for the login process, such as a standard user name and password. Finally, the last login timestamp indicates the date and time of the user's most recent login activity, empowering users to track and effectively manage their login information.

The system updates the login data using policies every 72 hours. Additionally, you can populate the data manually on demand by selecting the refresh icon \(![Refresh icon used to refresh the page.](../image/icon-dex-refresh.png)\). To learn more about the policies, see [DEX policies for macOS](../reference/dex-policies-mac.md) and [DEX policies for Windows](../reference/dex-policies-windows.md).

**Note:** The **Host** field is visible only when you access the device remotely via RDP \(Windows\) or SSH \(Mac\). The **User profiles** section is exclusive to Windows OS.

For more information on the user logins card, see [Access page](../reference/user-logins-card.md).
