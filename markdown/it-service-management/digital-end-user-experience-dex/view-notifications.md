# View notifications

View notifications to stay up to date with important information and events in real time.

## Before you begin

Confirm that the DEX Desktop Assistant \[sn\_dex\_desktop\] application has been installed. For more information, see [Install Application and Device Health](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-service-management/digital-end-user-experience-dex/install-app-device-health.md) and [Download and install Desktop Assistant](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-service-management/digital-end-user-experience-dex/download-desktop-experience.md).

Role required: sn\_dex\_desktop.user or sn\_dex\_desktop.admin

## Procedure

1.  On the Desktop Assistant home page, select the notifications bell icon \(\[Omitted image "icon-notification-bell.png"\] Alt text: The Notifications bell icon\).

    When you select the notification bell icon \[Omitted image "icon-notification-bell.png"\] Alt text: The Notifications bell icon, a drop-down list appears, displaying the list of notifications.\[Omitted image "View\_DA\_Notifications.png"\] Alt text: Desktop Assistant notifications list

    **Note:** With DEX Washington DC release, if a user logs in to Desktop Assistant on a new device, any existing unexpired notifications don't appear in the notifications list.
