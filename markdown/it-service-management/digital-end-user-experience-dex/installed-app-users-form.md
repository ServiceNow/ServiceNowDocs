# Installed application — Users section

The All users and Impacted users lists capture data regarding the active users, impacted users, the last logged in device, and the location of the user. This information helps with security and access control, as well as providing insights into user behavior and application usage.

<table id="table_lsb_btf_4wb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Active user who is connected to DEX using the monitored devices.

</td></tr><tr><td>

Last logged in device

</td><td>

Digital devices like laptops and desktops that users most recently used.

</td></tr><tr><td>

Last logged in location

</td><td>

Location of the device that the user most recently used.**Note:**

The device location is important in an outage or emergency situation. When there’s an outage or emergency, it’s important to be able to locate affected devices and users to assist and support. Device location data can be used to identify the extent and severity of the outage, prioritize restoration efforts, and direct resources to the affected areas.

</td></tr><tr><td>

App last accessed

</td><td>

Date and time when the application was last accessed by the user with respect to a device.Only All users page has this field.

**Note:** User can access the application using different devices. The **App Last accessed** field displays the date and time for a specific device.

</td></tr></tbody>
</table>Selecting the user name takes you to the details page of that user. For detailed information on a user and associated applications, see [Device details](../concept/dex-workspace-user-details.md).

To return to the Installed application details page, see [Installed application details](../concept/installed-application-details.md).
