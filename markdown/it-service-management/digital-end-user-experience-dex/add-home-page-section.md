# Add a section in the Home page

Add a section so that you can organize related cards under that section for easy navigation.

## Before you begin

Confirm that the DEX Desktop Assistant \[sn\_dex\_desktop\] application has been installed. For more information, see [Install Application and Device Health](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-service-management/digital-end-user-experience-dex/install-app-device-health.md) and [Download and install Desktop Assistant](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-service-management/digital-end-user-experience-dex/download-desktop-experience.md).

Role required: sn\_dex\_desktop.admin

## Procedure

1.  Navigate to **Desktop Assistant** &gt; **Configuration** &gt; **Home**.

2.  Select the name of the home page for which you want to add a section.

3.  In the Tab to Sections Mappings section, select **New**.

    \[Omitted image "desktop-exp-section-mapping.png"\] Alt text: Page where you can map a specific section to a specific home page.

4.  Perform any of the following actions:

    -   If you want to select an already existing section, perform the following steps:
        1.  In the **Desktop Assistant Section** field, select the magnifying glass icon \(\[Omitted image "icon-magnifying-glass-blue.png"\] Alt text: The magnifying glass icon to search for existing sections.\).
        2.  Select the section name and select **Submit**.

            The section is created on the Desktop Assistant home page.

    -   If you want to create a new section, perform the following steps:
        1.  In the **Desktop Assistant Section** field, select the magnifying glass icon \(\[Omitted image "icon-magnifying-glass-blue.png"\] Alt text: The magnifying glass icon to search for existing sections.\) and then select **New**.
        2.  In the **Name** field, enter the name of the section.
        3.  Select **Submit**.

            You are directed to the Tab to Section Mapping page where you can see the section name in the **Desktop Assistant Section** field.

        4.  Select **Submit**.

## Result

The section is created on the Desktop Assistant home page.
