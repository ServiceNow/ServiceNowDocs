# Device health

View the overall operational status and performance of a user's device. Assess the performance using key metrics and indicators such as system uptime, pending updates and patches, recent crashes or blue/black screens, battery health, and application stability.

The system updates the device data using policies every 72 hours. Additionally, you can manually populate the on-demand data by selecting the refresh icon \(![Refresh icon used to refresh the page.](../image/icon-dex-refresh.png)\). To learn more about the policies, see [DEX policies for macOS](../reference/dex-policies-mac.md) and [DEX policies for Windows](../reference/dex-policies-windows.md).

## Device health overview

The Overview section provides device details such as type, name, OS, model ID, current location, CPU, and RAM specifications. Device Metrics show key info such as pending updates, battery health, BSOD causes, and crashes, offering a comprehensive view of device performance and health. Device events in the last 24 hours include activities like software installations, providing insights into usage patterns and performance history. Pending system updates offer version details and necessary actions for improved functionality and security, while OS details provide information on the device's operating system.

## Device health alerts

Navigate the Alerts section to discover the open device alerts. Each alert is distinguished by a unique identifier \(Number\) and features a summary of the issue, severity, and the timestamp of its creation \(Created\).

## Device health incidents

Look up the incidents for device. Each incident has a unique identifier \(Number\) and features a summary of the incident, priority, and the timestamp for when it was created \(Opened\).

## Device health security

The Security section applies to Windows OS only, providing crucial insights into the device protection, specifically focusing on Windows disk encryption tool Bitlocker. The section highlights key specifics like volume type, protection status, mount point, capacity, volume status, and encryption percentage. The section provides a comprehensive overview of the device's encrypted volumes and security status.

## Device health battery

The Battery health assesses the overall condition, efficiency, and potential lifespan of the device battery. The section offers insights into its current capacity, cycle count, and remaining charge percentage. Use the information to optimize performance, prolong lifespan, and promote uninterrupted device usability.

## Device health hardware

The Hardware section provides key details about the hard drive, offering insights into its crucial specifications. The section includes the following details:

-   Drive assigned name for identification purposes
-   Status denoting its operational condition
-   Manufacturer indicating the producing company
-   Media loaded showing if any storage media is inserted
-   Model number specifying the drive specific version
-   Partitions showcasing the number of distinct sections on the drive
-   Size revealing the total storage capacity available on the hard drive

These details collectively offer a comprehensive overview of the hard drive characteristics and functionalities and are essential for maintenance, troubleshooting, compatibility, capacity planning, and overall system management.

For information on each field, see [Device health page](../reference/user-health-card.md).
