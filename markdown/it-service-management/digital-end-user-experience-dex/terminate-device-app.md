# Terminate a process on your device

Terminate a process on your device to boost performance and free up valuable resources for smoother operation.

## Before you begin

Role required: sn\_dex.engineer or sn\_dex.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace** and from the primary navigation pane, select the Devices icon \(![Devices icon](../image/icon-devices.png)\).

2.  Select the device for which you want to terminate a process.

3.  Select **Running processes**.

4.  From the contextual side panel, select the Action library icon \(![Actions library icon](../image/icon-action-library.png)\).

5.  In the End process card, select **Start**.

6.  Select the End process card that now shows the process state as **New**.

7.  Under Get user approval card, select the **I received approval to trigger this action** check box and select **Submit**.

8.  In the **Process ID** field, enter the corresponding process ID obtained from the PID column.

9.  Select **Submit**.

    The process is terminated.


## What to do next

On the contextual side panel, select the Playbook icon \(![Playbook icon](../image/icon-playbook.png)\) to access the current action of the device and also to review the device's action execution history.
