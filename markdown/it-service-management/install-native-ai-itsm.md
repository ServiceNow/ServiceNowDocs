# Install AI-native IT Service Management

Set up and get started with AI-native IT Service Management \(AI-native ITSM\) on your instance.

## Before you begin

Role required: admin

-   Request for the entitlement of the Now Assist for Setup application \(sn\_ia\) from [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) and install it. See [Set up Now Assist with Setup Hub](https://www.servicenow.com/docs/access?context=ia-setup-now-assist&version=australia&pubname=australia-platform-administration&ft:locale=en-US) and [Understand the Configuration page flow in Setup Hub](https://www.servicenow.com/docs/access?context=ia-configure-il&version=australia&pubname=australia-platform-administration&ft:locale=en-US).
-   Request for entitlement of the AI-native IT Service Management application \(sn\_ai\_itsm\_cont\) from [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home).

## About this task

The following items are installed with AI-native IT Service Management.

-   AI agents and agentic workflows simplifying ITSM workflows, supporting a simplified Employee Center portal, and a simplified administrator, employee, and fulfiller experience. For information on AI agents, see [AI agents and agentic workflows in AI-native IT Service Management](../reference/agents-ai-native-it-service-desk.md).
-   Commonly requested catalog items. For information on catalog items, see [Catalog items installed with AI-native IT Service Management](../reference/catalog-items-base-system.md).

## Procedure

1.  From the **All** menu, navigate to **Admin Center** &gt; **Application Manager** &gt; **Available for you**.

2.  Search for the AI-native IT Service Management application \(sn\_ai\_itsm\_cont\).

3.  Review the application listing for information on dependencies, licensing or subscription requirements, and release compatibility.

4.  Select **Install**.

5.  Apply default configurations post installation of the AI-native IT Service Management application \(sn\_ai\_itsm\_cont\).

    1.  From the header of your ServiceNow instance, navigate to **Admin** &gt; **Admin Home**.

    2.  From the **Manage your products** section, select **View product overview** for AI-native IT Service Management.

        The Product Hub page for AI-native IT Service Management is displayed.

        ![Set up AI-native ITSM application](../image/getStartedInstall.png)

    3.  In the **Installation progress** section, from the **Not Installed** tab, select **Apply default configurations** for AI-native IT Service Management.

        ![Applying default configurations for AI-native IT Service Management](../image/ai-native-itsm-install-product-hub.png)

        **Important:** Do not change the current scope while presets are being configured.

    4.  In the **Apply default configurations** pop-up window, select **Confirm**.

6.  Start configuring AI-native IT Service Management.

    1.  In the **Now you're ready to configure** pop-up window, select **Configure**.

        ![Start configuring AI-Native IT Service Management](../image/ai-native-itsm-configure.png)

        The Configure AI-native IT Service Management page is displayed.

    2.  In the **Get started with configuration** pop-up window, review the configuration instructions and select **Got it**.

        ![Reviewing configuration instructions for AI-native IT Service Management](../image/ai-native-itsm-config-instructions.png)
