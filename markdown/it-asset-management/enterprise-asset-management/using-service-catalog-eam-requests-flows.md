# Using Service Catalog for Enterprise Asset Management requests and flows

Use the Service Catalog application to create catalog requests for the Enterprise Asset Management application.

-   **[Use an enterprise asset request flow](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/create-eam-assetrequest.md)**  
Use an enterprise asset request flow for requesting, sourcing, and deploying catalog items from the Service Catalog application.
-   **[Request an enterprise asset refresh](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/request-eam-assetrefresh.md)**  
Use the Enterprise Asset Refresh flow to track the assets that are nearing the end of their life cycle and replace them with new assets.
-   **[Create an asset resale flow](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/enterprise-asset-resale-flow.md)**  
Create an asset resale flow for an asset to be resold in order to reduce waste and save cost.
-   **[Create a catalog request for onboarding multiple assets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/create-asset-onboard-catalog-req.md)**  
Create a catalog request for onboarding multiple assets on a single model.
-   **[Create a catalog request to reclaim enterprise assets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/create-enterprise-catalog-request.md)**  
Create a catalog request to efficiently reclaim enterprise assets when an employee leaves an organization or moves to a different role.
