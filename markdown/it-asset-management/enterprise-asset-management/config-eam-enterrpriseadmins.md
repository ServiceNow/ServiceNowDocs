# Configuring Enterprise Asset Management by enterprise administrators

The enterprise administrator role has complete access to the Enterprise Asset Management application thereby allowing this role to create multi-component assets and models along with other related configurations.

Enterprise administrators have the capability to configure the following:

-   Opt-in to Enterprise Asset Management
-   Create multi-component models to specify one or more model components based on the same model
-   Create multi-component assets from multi-components model
-   Create and import classification codes
-   Configure model and asset tabs
-   Import enterprise models and assets in bulk
-   Import and export content data
-   Use Advanced Shipment Notification
-   Revert normalization
-   Create linear asset types
-   Create configuration values for risk likelihood, risk impact, and risk score
-   Create shipping carrier records and view integration profiles for third-party shipping carriers
-   Create knowledge articles
-   Create calibration categories for enterprise asset calibrations
-   Create a calibration attribute library
-   Create a quality standard for your enterprise asset calibrations
-   Create a service frequency for your enterprise asset calibrations
-   View license report for the OT Asset Management application

-   **[Opt-in to Enterprise Asset Management Content Service](../task/optin-cs-eam.md)**  
Opt in to the Enterprise Asset Management Content Service to share unnormalized enterprise model data such as enterprise models, enterprise model lifecycles, and custom enterprise product models from your organization with ServiceNow to improve the normalization process.
-   **[Create a multi-component model](../task/create-multicomponent-model.md)**  
Create a multi-component model to define more than one model component based on the same product model.
-   **[Create a multi-component asset](../task/create-multicomponent-asset.md)**  
Track and manage your assets by creating a multi-component asset from a multi-component model.
-   **[Create sites in the Enterprise Asset Management](../task/create-sites-eam.md)**  
Create a site for asset groups in the Enterprise Asset Management application.
-   **[Create classification codes](../task/create-classification-codes.md)**  
Create classification codes and assign them to enterprise models and model categories to increase the efficiency of enterprise models.
-   **[Import classification codes](../task/import-class-codes-eam.md)**  
Import multiple classification codes at one go to your ServiceNow instance.
-   **[Import enterprise models and assets in workspace](../task/bulk-import-eam.md)**  
Import multiple enterprise models and assets at one go in the Enterprise Asset Workspace.
-   **[Configure tabs in the Enterprise Asset Workspace](../task/configure-model-asset-class-tabs.md)**  
Configure the options to hide or display various tabs, including tabs that are associated with specific model categories, in the Enterprise model management view and Enterprise asset estate view of the Enterprise Asset Workspace.
-   **[Import and export content data for Enterprise Asset Management](../task/import-export-data-eam.md)**  
Import and export content data to the ServiceNow Enterprise Asset Management Content Service to improve the normalization process. On-premise users can use the Manage Enterprise Library module to import or export data via a zip file.
-   **[Use Advanced Shipment Notification in Enterprise Asset Management](../task/asn-eam.md)**  
Use Advanced Shipment Notification \(ASN\) to automate and create enterprise asset records for when your assets are in transit.
-   **[Revert normalization](../task/revert-norm-eam.md)**  
You can revert the normalization of enterprise models in the Enterprise Asset Workspace.
-   **[Create linear asset types](../task/create-linear-asset-types.md)**  
Create linear asset types to categorize linear assets.
-   **[Create configuration values for risk likelihood](../task/risk-likelihood-config.md)**  
Use the Risk Likelihood module to create configuration values for the likelihood vector.
-   **[Create configuration values for risk impact](../task/risk-impact-config.md)**  
Use the Risk Impact module to create configuration values for the impact vector.
-   **[Create configuration values for risk scores](../task/risk-score-config.md)**  
Use the Risk Score module to create configuration values for risk score bands.
-   **[Create a shipment carrier record in Enterprise Asset Workspace](../task/create-shipment-carrier-record-eam.md)**  
Create a shipping carrier record in the Enterprise Asset workspace to associate the carrier with an integration profile.
-   **[View integration profiles for third-party shipping carriers in the Enterprise Asset Workspace](../task/view-shipping-carrier-integration-profiles-eam.md)**  
View the integration profiles for your third-party shipping carriers in the Asset operations view of the Enterprise Asset Workspace. These integration profiles enable you to connect to third-party shipping carrier applications so that you can retrieve shipment details for your enterprise assets.
-   **[Create a knowledge article for Enterprise Asset Management](../task/create-eam-knowledge-article.md)**  
Create and edit knowledge articles within a knowledge base to share information across your organization.
-   **[Create a calibration category for your enterprise asset calibrations](../task/create-eam-calibration-categories.md)**  
Create a calibration category to indicate the type of calibration that you need to perform on your enterprise assets.
-   **[Create a quality standard for your enterprise asset calibrations](../task/create-eam-calibration-quality-standard.md)**  
Create a quality standard to indicate the requirements and guidelines that you must follow while performing your enterprise asset calibrations.
-   **[Create a service frequency for your enterprise asset calibrations](../task/create-eam-calibration-service-frequency.md)**  
Create a service frequency to specify how often you must perform an enterprise asset calibration.
-   **[Create a calibration attribute library](../task/create-eam-calibration-attribute-library.md)**  
Create a library of common calibration attributes that are not associated with any enterprise models or enterprise assets. Use these attributes to define the details and requirements of your commonly used calibrations.
-   **[View the license report for the OT Asset Management application](../task/view-license-report-otam.md)**  
View details of the subscriptions consumed by your organization for the OT Asset Management application by using the ITAM License Report.
-   **[Manage failure and resolution codes](manage-failure-res-codes-eam.md)**  
Improve maintenance efficiency by managing the failure and resolution codes in the Admin center of the Enterprise Asset Workspace.
-   **[Configure settings to monitor asset key performance indicators \(KPIs\) in the Enterprise Asset Workspace](configure-kpi-settings-eam.md)**  
Set up the necessary configurations in the Admin Center to enable the tracking of KPIs for your enterprise assets. These settings enable you to monitor and manage the performance your assets.
