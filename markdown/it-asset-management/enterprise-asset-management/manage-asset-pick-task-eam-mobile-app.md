# Manage enterprise asset pick tasks using the Mobile Agent application

Work on the Asset pick tasks assigned to you by using the Mobile Agent application.

-   **[Start working on an enterprise asset pick task using the Mobile Agent application](../task/start-work-pick-task-mobile-eam.md)**  
Indicate in the Mobile Agent application when you start work on an asset pick task assigned to you.
-   **[Record time worked on an enterprise asset pick task using the Mobile Agent application](record-time-pick-task-mobile-agent-eam.md)**  
Manage and record time worked on enterprise asset pick tasks by using the Mobile Agent application. After you start working on a task, you can pause and resume work.
-   **[Locate and pick the enterprise assets in your stockroom using the Mobile Agent application](../task/locate-and-pick-enterprise-asset-mobile-app.md)**  
Locate an enterprise asset placed in a specific aisle and space in the stockroom by using indoor maps. After locating the asset, pick and drop off the asset to another location within the stockroom using the Mobile Agent application.
