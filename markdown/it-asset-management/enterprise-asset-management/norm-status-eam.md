# Normalization status for enterprise models

Description of the normalization statuses for enterprise models.

<table id="table_eg4_myc_cjb"><thead><tr><th>

Status

</th><th>

Description

</th></tr></thead><tbody><tr><td>

New

</td><td>

The model is created and has not yet run through the normalization process.

</td></tr><tr><td>

Normalized

</td><td>

After normalization, values for normalized manufacturer, product, model name, model number and model type are populated.Under specific conditions, certain fields that are typically read-only can be edited.

</td></tr><tr><td>

Partially Normalized

</td><td>

After normalization, values are populated for normalized manufacturer, product, and model type and the model number is blank.

</td></tr><tr><td>

Manufacturer Normalized

</td><td>

After normalization, values for normalized manufacturer is populated and the rest of the columns are blank after normalization.

</td></tr><tr><td>

Match Not Found

</td><td>

The normalization process could not match any of the fields of the model.**Match Not Found** status could occur if a normalization rule for the software does not exist.

</td></tr></tbody>
