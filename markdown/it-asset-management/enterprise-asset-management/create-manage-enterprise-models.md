# Create and manage enterprise models

Create and manage your enterprise models to keep track of your enterprise assets.

-   **[Create enterprise models](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/create-model-eam.md)**  
Create a model based on a top tier Enterprise Asset Management model category. Use models to manage and keep track of your assets.
-   **[Create enterprise model components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/create-model-components-eam.md)**  
Create model components and add them to a model. A model comprising of one or more model components is a multi-component model.
-   **[Create model categories](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/create-custom-model-category.md)**  
You can create model categories for the Enterprise Asset Management application.
-   **[Add a substitute model for an enterprise model](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/add-enterprise-model-substitute.md)**  
On an enterprise model record, specify the related models that you can substitute the enterprise model with.
-   **[Add calibration attributes to an enterprise model](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/add-calibration-attributes-enterprise-model.md)**  
Add calibration attributes to an enterprise model to specify and manage the calibrations that are required for the associated enterprise assets throughout their life cycles.
-   **[Clone calibration attributes to an enterprise model](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/clone-calibration-attributes-enterprise-model.md)**  
Clone an existing calibration attribute to an enterprise model to add a calibration attribute with similar settings.
-   **[View the enterprise model hierarchy in the Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/view-model-hierarchy-eam.md)**  
View the hierarchy details of pre-assembled and user-assembled enterprise models in a tree-like format in the Enterprise Asset Workspace.
-   **[Manage the lifecycle of enterprise models with calculated lifecycle templates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/it-asset-management/enterprise-asset-management/manage-eam-lifecycle-temp.md)**  
Manage the complete lifecycle of enterprise models by creating calculated lifecycle templates and associating these templates with the models.
