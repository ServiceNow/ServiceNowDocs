# Manage enterprise asset put away using the ServiceNow Agent application

Scan and put away the assets from the receiving bay in the designated aisle-space using the ServiceNow Agent application.

**Important:** The Asset put away feature is available with Enterprise Asset Management version 9.0 onwards.

-   **[Put away enterprise assets using the ServiceNow Agent application](../task/perform-put-away-mobile-agent-eam.md)**  
Scan available assets in the inventory and perform asset put away in the designated scanned drop-off location.
