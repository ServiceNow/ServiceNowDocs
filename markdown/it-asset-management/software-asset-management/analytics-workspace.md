# Software asset analytics view

The unified and streamlined dashboards in the Software asset analytics view in the Software Asset Workspace enable you to track your performance and view your analytics.

You can access the Software asset analytics view by navigating to **Software Asset Workspace** &gt; **Software asset analytics**.

![Software asset analytics view](../image/software-analytics-workspace.png "Software asset analytics view")

The Software asset analytics view includes the following dashboards:

-   SaaS Overview: A single Oracle dashboard showing key KPIs for both pure Oracle as well as Microsoft 365 and Adobe Cloud. For more details, see [SaaS overview dashboard in workspace](../reference/saas-dashboard-workspace.md).
-   Discovered inventory: View your licensable discovered software and devices. For more details, see [Discovered inventory dashboard](discovered-inventory-dashboard.md).
-   Normalization and content: View normalization and Content Service trend charts. For more details, see [Normalization and content dashboard](../reference/norm-content-dashboard-workspace.md).
-   Lifecycle management: Review and track the end of life \(EOL\) software products in your asset estate. For more details, see [Lifecycle management dashboard](lifecycle-management-dashboard.md).
-   Optimization and savings: View metrics on how to maximize and save costs on your software assets. The dashboard also displays licensing optimizations for third-party publishers such as Microsoft, Red Hat, Adobe, and SAP. For more details, see [Optimization and savings dashboard in workspace](../reference/optimization-dashboard-workspace.md).
-   Engineering License Overview: View all your engineering applications license position and usage. For more details, see [Engineering License overview dashboard in workspace](../reference/engineering-license-dashboard-workspace.md).
-   Cloud cost simulator: Compare and evaluate the estimated costs of migrating your on-premises resources to the cloud for each cloud environment. For more details, see [Cloud simulator dashboard](cloud-simulator-dashboard.md).
