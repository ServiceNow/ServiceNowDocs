# Execute actions in Automation Center

Execute an action to fix common repeated issues efficiently.

## Before you begin

Role required: sn\_ac.automation\_business\_user, sn\_ac.automation\_technical\_user, catalog\_admin, or sn\_ac.automation\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Automation Center Workspace**.

    The Automation Center dashboard is displayed.

2.  Select the Action Center icon \(![Action Center icon](../images/action-center-icon.png)\) on the left.![Action Center](../images/action-center-page.png)

    -   In the **Recents** section, all actions that have been executed are displayed.
    -   You can view all the actions that have been executed by selecting the **View list** button in the **Recents** section.
    -   In the **All actions** section, all available actions are displayed. You can search for any action or filter and sort your results.
3.  In the **All actions** section, select **Execute** for any available action that you want to run.

    A new page is displayed with additional information.

4.  Provide all required information, and select **Run**.

    The action is run and displayed in the **Recents** section.
