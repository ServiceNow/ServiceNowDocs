# Create an automation request

Create an automation request and a process diagram or requirement diagram to create an automation from your diagram.

## Before you begin

Role required: sn\_ac.automation\_business\_user, sn\_ac.automation\_technical\_user, playbook.write, playbook.activity\_def\_read, connection\_admin, or cmdb\_read

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Service Portal Home**.

2.  From the Service Portal Home page, you can search for `Submit an automation request`, and select the option from the result.

    Alternately, you can select **Request Something** &gt; **Can we help you?** &gt; **Submit an automation request** on the Service Portal Home page.

3.  Fill in the details, and select **Submit**.

    For more information, see [Submit an Automation Request form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/automation-center/submit-automation-request-form.md).

    The next page will enable you to create a process diagram.

    **Note:** You get the option to create your requirement diagram only from the Service Portal Home page.

4.  Select **Create a diagram**.

    Create your requirement diagram. For more information about editing or creating diagrams, see .

    When creating a diagram, we need not set a trigger. A trigger will be set when the automation is created from the requirement diagram.
