# Request diagram

After creating an automation request, if a diagram is not created, request for a diagram. A diagram is a visual representation of the automation that you want to create.

## Before you begin

Role required: sn\_ac.automation\_technical\_user, playbook.write, playbook.activity\_def\_read, connection\_admin, or cmdb\_read

## Procedure

1.  Navigate to **Workspaces** &gt; **Automation Center Workspace**.

2.  On the **Lists** tab \(![List icon](../../automation-center/images/autocenter-list-icon.png)\), under **Build**, select **All Automation Requests**.

3.  Select an automation request for which you want to request the diagram.

4.  Select **Request diagram**.

    A diagram is requested to the business user.

    **Important:** This button is available only if the business user has not created a requirement diagram when creating an automation request.

    A notification is sent to the creator of the automation request to provide a requirement diagram.
