# Shared Parameters

You can set parameters and their values that are shared across bot processes in the RPA Hub. The Shared Parameters components enable you to get the values of one or more such parameters.

-   **[Use the GetSharedParameters component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/use-rpahub-sharedparameters-getsharedparameters.md)**  
Get the values of multiple shared parameters set up for multiple bot processes on the RPA Hub using the GetSharedParameters component.
-   **[Use the SetSharedParameter component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/use-rpahub-sharedparameters-setsharedparameter.md)**  
Update the value of a specific shared parameter that you had set for multiple bot processes on the RPA Hub by using the SetSharedParameter component in the RPA Desktop Design Studio.
-   **[Properties of the shared parameters](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/properties-rpahub-shared-parameters.md)**  
Learn about the properties of the shared parameters in RPA Desktop Design Studio.
