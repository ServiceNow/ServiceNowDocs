# JavaScript

The JavaScript connector enables you to execute custom JavaScript by writing custom classes and methods and include them as part of you automation in the RPA Desktop Design Studio.

To use the connector, you must first configure it by writing and validating custom classes and methods. After that, you must use the method the connector exposes. Use the links.

-   To use the JavaScript connector, see [Use a connector in RPA Desktop Design Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/use-connector.md).
-   To configure the JavaScript connector, see [Configure the JavaScript connector](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/configure-javascript-connector.md).
-   To expose the methods, see [Use connector method](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/use-connector-method.md).
-   To use the methods, see [Use a component in RPA Desktop Design Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/configure-components.md).

-   **[Configure the JavaScript connector](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/configure-javascript-connector.md)**  
Configure the JavaScript connector by writing custom script and validating custom classes and functions. Then you execute the script by providing values for the functions.
-   **[JavaScript connector methods](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/integrate-applications/rpa-hub/connector-javascript-methods.md)**  
Execute custom JavaScript with the Execute method as part of an automation in the RPA Desktop Design Studio.
