# Parameters of the XmlTemplate connector methods

Learn about the various parameters of the XmlTemplate connector methods in RPA Desktop Design Studio.

|Parameter name|Description|Mandatory?|
|--------------|-----------|----------|
|FilePath|Path to the PDF or XML document that is used to create templates.|Yes|
|Status|Boolean response to whether the template was loaded correctly.|Not applicable|
|Template|Template name as configured in the XmlTemplate connector.|Yes|
