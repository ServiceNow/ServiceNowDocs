# Using Unattended Robot

You can use the Unattended Robot application to execute unattended automations that are triggered either by a schedule, API, or manually.

**Triggered By** field on the Process Job form determines if a bot process is triggered via Schedule, Manual or API​. For more information, see [Process job form in RPA Hub](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/integrate-applications/rpa-hub/process-job-form.md).
