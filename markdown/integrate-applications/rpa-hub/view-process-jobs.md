# View RPA Hub process jobs for all bot processes

View a list of execution logs for all bot processes.

## Before you begin

Role required: sn\_rpa\_fdn.rpa\_business\_user, sn\_rpa\_fdn.rpa\_support\_user, sn\_rpa\_fdn.rpa\_developer, sn\_rpa\_fdn.rpa\_robot, or sn\_rpa\_fdn.rpa\_admin

## About this task

A process job represents the execution of a bot process by a robot.

## Procedure

1.  Navigate to **All** &gt; **Robotic Process Automation** &gt; **RPA Hub Workspace**.

2.  Select the list icon \(![List icon.](../image/rpahublist-icon.png)\).

3.  On the **Lists** tab, under **Monitor**, select **Process Jobs**.

4.  Open a process job to view the details.

5.  In the **Activity** and **Compose** fields, view the activity of the form, work notes, and additional comments.
