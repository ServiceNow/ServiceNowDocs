# Properties of the Reflection components

Learn about the properties of the Reflection components in RPA Desktop Design Studio.

|Property|Description|
|--------|-----------|
|Search|Option to enable a search for a property.|
|Delay After Execution|The number of seconds to pause after running.|
|Delay Before Execution|The number of seconds to pause before running.|
