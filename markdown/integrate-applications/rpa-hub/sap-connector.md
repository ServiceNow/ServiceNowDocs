# SAP connector

The SAP connector automates workflows on the SAP Graphical user interfaces \(GUI\). For example, create and save a vendor entry. Create custom automations with the SAP connector on the RPA Desktop Design Studio. The SAP Connector is built by Bristlecone Inc.

To create automations using the SAP Graphical user interfaces \(GUI\), you must first configure the SAP connector.

To use the SAP connector, do the following steps:

-   Enable the SAP connector. See [Use a connector in RPA Desktop Design Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/integrate-applications/rpa-hub/use-connector.md).
-   Configure the SAP connector. See [Configure the SAP connector](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/integrate-applications/rpa-hub/configure-the-sap-connector.md).
-   Expose the methods. See [Use connector method](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/integrate-applications/rpa-hub/use-connector-method.md).
-   Use the methods. See [Use a component in RPA Desktop Design Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/integrate-applications/rpa-hub/configure-components.md).
