# Set up the Smartsheet spoke

Set up the Smartsheet spoke to enable integration between the ServiceNow® instance and Smartsheet.

## Before you begin

-   Request an Integration Hub subscription.
-   Activate the Smartsheet plugin.
-   Create an alias for Smartsheet Application Programming Interface \(API\) key.
-   Role required: admin.

## About this task

Setting up involves generating an API key on the Smartsheet and creating a connection and credential record on the ServiceNow instance.
