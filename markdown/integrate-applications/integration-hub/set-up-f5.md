# Configure a connection for the F5 BIG-IP spoke

Integrate the ServiceNow instance and F5 using basic authentication to authenticate ServiceNow requests.

## Before you begin

-   Request Integration Hub subscription
-   Activate F5 BIG-IP spoke
-   Role required: admin

## Procedure

1.  Navigate to **Process Automation** &gt; **Flow Designer**.

2.  Click the **Connections** tab.

3.  Locate the **F5\_BIG\_IP** connection alias and click **View Details**.

4.  Click **Edit**.

5.  On the **Connection** form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Connection Name|Name to identify the F5 BIG-IP connection alias record.|
    |Connection URL|URL of the host machine where the F5 BIG-IP server is installed.|
    |User name|Your F5 BIG-IP user name.|
    |Password|Your F5 BIG-IP user name.|

6.  Click **Submit**.
