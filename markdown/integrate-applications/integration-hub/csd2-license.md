# Create entitlements for distributed software using SAM

Entitlements are associated with the software model and must be created if you want to track the license for software deployed by CSD 2.0 using SAM.

## Before you begin

Role required: sam\_admin or sam\_user

For information about software reconciliation, see [Software reconciliation for compliance](https://www.servicenow.com/docs/access?context=c_SAMReconciliation&version=zurich&pubname=zurich-it-asset-management&ft:locale=en-US).
