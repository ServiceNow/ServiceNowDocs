# Create a folder structure in Lucidchart

Ensure that you have at least one folder created in the My documents folder of your computer.

## Before you begin

-   Lucidchart account.
-   Role required: Lucidchart admin.

## About this task

You must have a folder structure in place in the Lucidchart portal for the Create Diagram UI action to work. If you do not have a folder created in the Lucidchart, follow the below steps to create a folder.

## Procedure

1.  Navigate to [https://lucid.app/documents\#/documents%3Ffolder\_id=home](https://lucid.app/documents#/documents%3Ffolder_id=home)

2.  Log in to your Lucidchart account using the same credentials that you used to create OAuth 2.0 Client.

3.  In the left navigation, select **+New** button, then select **Folder**.

    ![Create folder in the Lucidchart](../image/lucid-create-folder.png)

4.  Enter a name for the folder.

5.  Ensure the folder is created under the Documents.
