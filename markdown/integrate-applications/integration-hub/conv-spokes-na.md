# Now Assist in Conversational Spokes

Use Now Assist in Conversational Spokes application to start utilizing the conversational ability of Integration Hub Spoke actions.

## Request apps on Store

Visit the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website to view all the available apps and for information about submitting requests to the store.

## Capabilities

Perform business actions through conversational interface like Now Assist. Unlock the interactive conversational experience to automate your business workflows. This feature requires an Integration Hub subscription. For more information, see [Legal schedules - IntegrationHub overview](https://www.servicenow.com/content/dam/servicenow-assets/public/en-us/doc-type/legal/snc-addendum-integrationhub.pdf).

For information about configuring the conversational settings, see . For information about the Integration Hub Spoke actions that are currently conversational ready, see .

## Supported versions

The Now Assist in Conversational Spokes application is supported on Xanadu Patch 3 and later releases.

## Activation of the conversational actions

Now Assist in Conversational Spokes application offers generative AI capabilities to make the spoke actions conversational. This application requires these subscriptions:

-   Now Assist for Platform \(sn\_genai\_platform\) subscription
-   One of these Workflow Data Fabric subscriptions:
    -   Integration Hub Professional V4 SKU
    -   Workflow Data Fabric Professional V5 SKU
    -   Workflow Data Fabric Enterprise V4 SKU
    -   Workflow Data Fabric Enterprise UU V4 SKU

After installing the Now Assist in Conversational Spokes application, you can configure the available conversational spoke actions.
