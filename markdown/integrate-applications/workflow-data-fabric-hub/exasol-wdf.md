# Exasol

The Exasol connector provides read-only access to data and metadata in an Exasol database.

A connection admin can create a connection to Exasol and give data stewards access to that connection in Workflow Data Fabric Hub. A data steward can select the established connection to create a data fabric table and map data from Exasol. This allows users to retrieve Exasol data through the table list view or GlideRecord scripts. For details on creating data fabric tables and mapping data, see [Managing data fabric tables in Workflow Data Fabric Hub](managing-data-fabric-tables-wdf.md).
