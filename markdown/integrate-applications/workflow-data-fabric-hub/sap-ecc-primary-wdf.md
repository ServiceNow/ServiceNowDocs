# SAP ECC

The SAP ECC connector provides read-only access to data and metadata from an SAP ECC system.

A connection admin can set up a connection to SAP ECC in the Workflow Data Fabric Hub and grant data stewards access to this connection. Data stewards can then use the established connection to create a data fabric table and map data from SAP ECC. This allows users to access SAP ECC data through the table list view or by using GlideRecord scripts.
