# Primary connectors

Connect to external data sources using primary connectors in Workflow Data Fabric Hub.

Primary connectors are developed, made available, and supported by ServiceNow.

|Primary connector|Minimum compatibility|
|-----------------|---------------------|
|[Databricks](databricks-wdf.md)|Yokohama Patch 5|
|[Google BigQuery](bigquery-wdf.md)|Yokohama Patch 5|
|[Oracle](oracle-wdf.md)|Yokohama Patch 5|
|[Snowflake](snowflake-wdf.md)|Yokohama Patch 5|
|[Amazon Redshift](amazon-redshift-wdf.md)|Yokohama Patch 5|
|[SAP ECC](sap-ecc-primary-wdf.md)|Yokohama Patch 5|
|[SAP S/4HANA](sap-s4hana-primary-wdf.md)|Yokohama Patch 5|
|[Teradata](teradata-wdf.md)|Yokohama Patch 5|
|[Amazon S3 Tables](amazon-s3-tables-wdf.md)|Yokohama Patch 5|
|[Apache Iceberg](apache-iceburg-primary-wdf.md)|Yokohama Patch 5|
|[Cloudera](cloudera-wdf.md)|Yokohama Patch 5|
|[Microsoft OneLake](microsoft-onelake-wdf.md)|Yokohama Patch 5|
|[Prometheus](prometheus-wdf.md)|Yokohama Patch 5|
|[ServiceNow Remote Instance](servicenow-remote-instance-wdf.md)|Yokohama Patch 5|

For additional information on data types and authentication methods that aren't supported with primary connectors, see [KBB0010487](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KBB0010487).
