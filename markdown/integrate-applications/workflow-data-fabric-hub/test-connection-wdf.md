# Test an established connection

Test the connection to an external data source from an established connection.

## Before you begin

Role required: df\_connection\_admin

## Procedure

1.  Navigate to the **Established connections** tab in Workflow Data Fabric Hub in one of the following ways:

    -   Navigate to **All** &gt; **Workflow Data Fabric Hub** &gt; **Established connections**.
    -   Navigate to **Admin** &gt; **Workflow Data Fabric Hub** &gt; **Established connections**.
2.  Select the connection that you want to test.

3.  Select **Test connection**.


## Result

The test connection succeeds or fails. If the connection fails, consult your database administrator to verify the connection attributes and authentication details. Make any necessary changes, and then try again.
