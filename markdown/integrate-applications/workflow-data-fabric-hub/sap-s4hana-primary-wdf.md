# SAP S/4HANA

The SAP S/4HANA connector provides read-only access to data and metadata from an SAP S/4HANA system.

A connection admin can create a connection to SAP S/4HANA and give data stewards access to that connection in Workflow Data Fabric Hub. A data steward can select the established connection to create a data fabric table and map data from SAP S/4HANA. This allows users to retrieve SAP S/4HANA data through the table list view or GlideRecord scripts.

For details on implementation, refer to [Zero Copy Connector for ERP](../../../build/erp-integration/concept/erp-integration-overview.md).
