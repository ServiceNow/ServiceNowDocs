# Delta Lake

The Delta Lake connector provides read-only access to data and metadata in the Delta Lake format.

A connection admin can create a connection to Delta Lake and give data stewards access to that connection in Workflow Data Fabric Hub. A data steward can select the established connection to create a data fabric table and map data from Delta Lake. This allows users to retrieve Delta Lake data through the table list view or GlideRecord scripts. For details on creating data fabric tables and mapping data, see [Managing data fabric tables in Workflow Data Fabric Hub](managing-data-fabric-tables-wdf.md).
