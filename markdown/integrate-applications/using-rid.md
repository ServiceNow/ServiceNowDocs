# Using Requirement Intake Diagram

Use Requirement Intake Diagram to design the automation requirements visually from a diagram.

-   [Create an automation request](../task/create-aut-req.md)

    Create an automation request and a process diagram to create an automation from your requirement diagram.

-   [Request diagram](../task/request-diagram.md)

    After creating an automation request, if a diagram is not created, request for a diagram. A diagram is a visual representation of the automation that you want to create.

-   [View diagram](../task/view-diagram.md)

    After creating a diagram, you can view it to edit or continue creating an automation from it. A diagram is a visual representation of the automation that you want to create.

-   [Build automation](../task/build-automation.md)

    Use the requirement diagram to build an automation.

-   [View automation](../task/view-automation.md)

    View the automation created in Workflow Studio.
