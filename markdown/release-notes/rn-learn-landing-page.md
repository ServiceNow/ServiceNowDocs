# Learn about the Yokohama release

The Yokohama release includes new features and improvements built on the ServiceNow AI Platform®.

## Release dates

-   Early availability: January 30, 2025
-   General availability: March 12, 2025

## Release notes for upgrades

Before you upgrade, review release notes from your current version to the Yokohama release. The release notes offer valuable information about new functionality, notable changes, and available fixes. They help you determine whether the upgrade contains functionality you need and fixes that resolve any issues affecting your instance. The release notes can also help you determine whether items you previously customized are being upgraded.
