# Learn about the Xanadu release

The Xanadu release includes new features and improvements built on the ServiceNow AI Platform®.

## Release dates

-   Early availability: August 1, 2024
-   General availability: Targeted for late Q3 2024

## Release notes for upgrades

Before you upgrade, review release notes from your current version to the Xanadu release. The release notes offer valuable information about new functionality, notable changes, and available fixes. They help you determine whether the upgrade contains functionality you need and fixes that resolve any issues affecting your instance. The release notes can also help you determine whether items you previously customized are being upgraded.
