## Learn about specific themes for multiple Xanadu products

Every new or updated feature and product has its own individual release notes topic, which includes information such as upgrade tasks, new features, changed features, browser requirements, and other specifications. These product-specific release notes are listed in [Features and changes by product](new-features-changes.md).
