# Analytics, Intelligence, and Reporting release notes

The ServiceNow Analytics, Intelligence, and Reporting application presents analytics on Core UI dashboards through reports and Performance Analytics widgets and enables you to explore Performance Analytics indicators on the Analytics Hub. Analytics, Intelligence, and Reporting was enhanced and updated in the Xanadu release.

## Analytics, Intelligence, and Reporting highlights for the Xanadu release

-   Core UI Reporting and Dashboards functionality deprecated in the Xanadu release
-   Several enhancements made for accessibility

## Important information for upgrading Analytics, Intelligence, and Reporting to Xanadu

If you are upgrading, you can use the Platform Analytics Migration Center to take advantage of a single set of visualizations and unified filters for all data sources.

## Deprecations

-   Core UI Reporting functionality is deprecated in favor of [Data visualizations in Platform Analytics](https://www.servicenow.com/docs/access?context=analytics-center-data-visualizations&version=xanadu&pubname=xanadu-now-intelligence&ft:locale=en-US) for new instances and instances that have migrated to the Platform Analytics experience.
-   Core UI Dashboards functionality is deprecated in favor of [Dashboards in Platform Analytics](https://www.servicenow.com/docs/access?context=analytics-center-dashboards&version=xanadu&pubname=xanadu-now-intelligence&ft:locale=en-US) for new instances and instances that have migrated to the Platform Analytics experience.

## Activation information

Analytics, Intelligence, and Reporting functionality is a ServiceNow AI Platform feature that is active by default. Some data sources, such as indicators, may require a subscription.

## Accessibility information

Reflow is supported for low-vision users. The following changes have been made to provide this support:

-   At 400% zoom, legends move to the bottom of the screen.
-   At 400% zoom, zoom is enabled by default on bubble visualizations.
-   At 400% or higher zoom, tooltips fit inside the screen.
-   Tooltips are shown when hovering over data points on bar and time series visualizations.
-   On gauge visualizations, range labels are shown as legends.

## Related ServiceNow applications and features

-   **[Platform Analytics Migration Center](https://www.servicenow.com/docs/access?context=data-migration&version=xanadu&pubname=xanadu-now-intelligence&ft:locale=en-US)**

    The Platform Analytics Migration Center provides a single set of visualizations and unified filters for all data sources.

-   **[Platform Analytics experience](https://www.servicenow.com/docs/access?context=par-workspace&version=xanadu&pubname=xanadu-now-intelligence&ft:locale=en-US)**

    The Platform Analytics experience is a unified interface for visualizing data from multiple data sources.
