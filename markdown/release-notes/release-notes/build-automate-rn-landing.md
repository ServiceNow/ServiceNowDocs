# Hyperautomation and Low-code release notes

The ServiceNow AI Platform® has new and updated hyperautomation and low-code features in the Xanadu release.

-   **[App Engine Studio release notes](app-engine-studio-rn.md)**  
The ServiceNow® App Engine Studio \(AES\) application enables creators of varying skill levels to build applications that meet the immediate needs of your organization. App Engine Studio was enhanced and updated in the Xanadu release.
-   **[Creator Studio release notes](creator-studio-rn.md)**  
The ServiceNow® Creator Studio product is a guided application development experience that enables business process experts to create request-based applications without the barriers of traditional low-code development. Creator Studio was enhanced and updated in the Xanadu release.
-   **[ERP Customization Mining release notes](erp-customization-mining-rn.md)**  
The ServiceNow ERP Customization Mining \(ERP-CM\) product enables ERP \(Enterprise Resource Planning\) consultants and architects to identify customizations and recommend candidates for replatforming from the ERP system onto the Now Platform. ERP Customization Mining was enhanced and updated in the Xanadu release.
-   **[ERP Data Hub release notes](erp-canvas-rn.md)**  
The ServiceNow® ERP Data Hub application \(formerly known as ERP Canvas\) enables you to connect to the ERP \(Enterprise Resource Planning\) system of record, query remote tables, and build data models to use ERP data on the ServiceNow AI Platform. ERP Data Hub was enhanced and updated in the Xanadu release.
-   **[Integration Hub release notes](integration-hub-rn.md)**  
The ServiceNow® Integration Hub application extends the ServiceNow® Workflow Studio automation capabilities by enabling you to integrate your instance data with data in external systems. Integration Hub was enhanced and updated in the Xanadu release.
-   **[Next Experience release notes](../now-platform-user-interface/next-experience-rn.md)**  
The ServiceNow® Next Experience application delivers an intuitive, personalized experience that enables your organization to drive productivity, improve engagement, and inspire learning with the ServiceNow AI Platform®. Next Experience was enhanced and updated in the Xanadu release.
-   **[Next Experience Developer \(NED\) Tools release notes](next-experience-developer-ned-tools-rn.md)**  
The ServiceNow® Next Experience Developer \(NED\) Tools application provides support for creating components to track and enhance performance and overall improve productivity. NED Tools was enhanced and updated in the Xanadu release.
-   **[Now Assist for Creator release notes](now-assist-for-creator-rn.md)**  
The ServiceNow® Now Assist for Creator application includes generative AI skills that can make developing on the ServiceNow AI Platform more efficient. Now Assist for Creator was enhanced and updated in the Xanadu release.
-   **[RPA Hub release notes](rpa-rn.md)**  
The ServiceNow® RPA Hub application enables end-to-end automation for your organization. RPA Hub was enhanced and updated in the Xanadu release.
-   **[ServiceNow IDE release notes](servicenow-ide-rn.md)**  
The ServiceNow® integrated development environment \(IDE\) application enables developers to create scoped applications in source code in an IDE based on Visual Studio Code for the Web on the ServiceNow AI Platform. ServiceNow IDE is a new application in the Xanadu release.
-   **[ServiceNow SDK release notes](servicenow-sdk-rn.md)**  
The ServiceNow® software development kit \(SDK\) enables developers to create scoped applications in source code locally in Visual Studio Code Desktop and upload changes to a ServiceNow instance. ServiceNow SDK was enhanced and updated in the Xanadu release.
-   **[ServiceNow Studio release notes](servicenow-studio-rn.md)**  
The ServiceNow® ServiceNow Studio application provides a unified experience for all ServiceNow® development activities, enabling admins and developers to extend base system solutions and easily create custom apps. ServiceNow Studio is a new application in the Xanadu release.
-   **[Stream Connect dashboard release notes](stream-connect-dashboard-rn.md)**  
The ServiceNow® Stream Connect dashboard application provides a central place where you can view and manage your Stream Connect integrations. The Stream Connect dashboard is a new application in the Xanadu release.
-   **[UI Component CLI Extension release notes](ui-component-cli-extension-rn.md)**  
The ServiceNow® UI Component CLI Extension provides an abstraction around various component development workflows including project scaffolding, starting a local development server, and deploying a component into a ServiceNow instance. UI Component CLI Extension was enhanced and updated in the Xanadu release.
-   **[UI Builder release notes](ui-builder-rn.md)**  
The ServiceNow® UI Builder application is a web user interface builder for building pages for workspaces and portals or custom web experiences with Next Experience Components. UI Builder was enhanced and updated in the Xanadu release.
-   **[Workflow Studio release notes](workflow-studio-rn.md)**  
The ServiceNow® Workflow Studio application provides a single location to access all process automation applications. Workflow Studio was enhanced and updated in the Xanadu release.
