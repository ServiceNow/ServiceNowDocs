# Conversation Improvement Themes release notes

The ServiceNow® Conversation Improvement Themes application transforms conversation evaluations from AI Control Tower into long-term performance insights by analyzing conversation quality data over time. Conversation Improvement Themes is a new application in the Zurich release.

## Conversation Improvement Themes highlights for the Zurich release

-   Use automated theme detection with a domain-specific large language model \(LLM\) theme classification framework to tag Knowledge Base articles, catalog items, Virtual Agent \(VA\) topics, and agents into themes.
-   Receive thematic insights by breaking down poor and good-quality conversations into top underlying themes.
-   Compare themes based on whether they lead to good or bad conversations.

See [Conversation Improvement Themes](https://www.servicenow.com/docs/access?context=conv-impr-themes-landing&version=zurich&pubname=zurich-intelligent-experiences&ft:locale=en-US) for more information.

**Important:** Conversation Improvement Themes is available in the ServiceNow Store. For details, see the "Activation information" section of these release notes.

## Conversation Improvement Themes features

-   **[Determine effective and ineffective conversations](https://www.servicenow.com/docs/access?context=conv-impr-themes-use&version=zurich&pubname=zurich-intelligent-experiences&ft:locale=en-US)**

    Analyze conversation quality data over time and classify them into effective and ineffective conversations.

-   **[Categorize conversations into themes](https://www.servicenow.com/docs/access?context=conv-impr-themes-use&version=zurich&pubname=zurich-intelligent-experiences&ft:locale=en-US)**

    Identify recurring patterns linked to low or high conversation quality and categorize user requests into actionable themes.
