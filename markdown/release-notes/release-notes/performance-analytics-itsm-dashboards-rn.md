# Performance AnalyticsITSM dashboards release notes

The ServiceNow® Performance Analytics ITSM dashboards provide several Platform Analytics solutions that contain preconfigured dashboards. Performance Analytics ITSM dashboards was enhanced and updated in the Xanadu release.

## Performance Analytics ITSM dashboards highlights for the Xanadu release

Experience the enhanced Performance Analytics ITSM dashboards in the Xanadu release.

See [Legacy: Performance Analytics ITSM Dashboards](https://www.servicenow.com/docs/access?context=itsm-dashboards-content-pack&version=xanadu&pubname=xanadu-it-service-management&ft:locale=en-US) for more information.

## New in the Xanadu release

-   **Next Experience [Legacy: IT Executive dashboard](https://www.servicenow.com/docs/access?context=it-executive-dashboard&version=xanadu&pubname=xanadu-it-service-management&ft:locale=en-US)**

    Get insights into the IT Service Management \(ITSM\) data using the ITSM Success Dashboard. This dashboard provides metrics for executives and managers to take informed decisions. Agents can use the [Service Operations Workspace for ITSM](https://www.servicenow.com/docs/access?context=sow-landing-page&version=xanadu&pubname=xanadu-it-service-management&ft:locale=en-US) Agent dashboard to get information about their work. These dashboards replace the existing Performance Analytics dashboards.


## Related ServiceNow applications and features

-   **[Performance Analytics \(Indicator data sources\)](https://www.servicenow.com/docs/access?context=pa-overview&version=xanadu&pubname=xanadu-now-intelligence&ft:locale=en-US)**

    Use ServiceNow® Performance Analytics to create reports on key performance indicators \(KPIs\) to monitor and analyze metrics.

-   **[Success Dashboard](https://www.servicenow.com/docs/access?context=success-dashboard-indicator-landing&version=xanadu&pubname=xanadu-it-service-management&ft:locale=en-US)**

    The ServiceNow® Success Dashboard enables the leadership team and process owners to gain insights into the performance of their ServiceNow® IT Service Management \(ITSM\) implementation using the KPIs defined by the ServiceNow AI Platform®.

-   **[Service Operations Workspace for ITSM](https://www.servicenow.com/docs/access?context=sow-landing-page&version=xanadu&pubname=xanadu-it-service-management&ft:locale=en-US)**

    Improve your ITSM agent experience and manage your IT fulfillment volume more efficiently by using ServiceNow® Service Operations Workspace. Manage your schedule and complete training by using ServiceNow® Workforce Optimization for ITSM in the Service Operations Workspace.
