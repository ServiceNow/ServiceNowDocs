# Table Builder release notes

The ServiceNow® Table Builder application is a centralized way to build tables, forms, and display logic. Table Builder was enhanced and updated in the Zurich release.

## UI changes

-   **[Accessing Table Builder](https://www.servicenow.com/docs/access?context=accessing-form-builder&version=zurich&pubname=zurich-application-development&ft:locale=en-US#section_ulg_1wr_w2c)**

    In the search input, enter `tablename.builder`, `tablename.view`, `tablename.sheet`, or `tablename.flow` to access the table directly.


## Browser requirements

Internet Explorer isn’t supported.

## Related ServiceNow applications and features

-   **[App Engine Studio](https://www.servicenow.com/docs/access?context=aes-overview&version=zurich&pubname=zurich-application-development&ft:locale=en-US)**
