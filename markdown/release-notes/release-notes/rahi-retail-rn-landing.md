# Retail release notes

Retail has new and updated features in the Yokohama release.

**Note:** The Retail mobile application and the case types added in Zurich are now available in Yokohama also. For more information on the Retail mobile application and the case types, see [https://www.servicenow.com/docs/bundle/zurich-release-notes/page/release-notes/retail/retail-rn.html](https://www.servicenow.com/docs/bundle/zurich-release-notes/page/release-notes/retail/retail-rn.html).

-   **[Retail Core release notes](retail/retail-core-rn.md)**  
The ServiceNow® Retail Core application empowers frontline managers and associates to seamlessly manage day-to-day store operations by automating tasks, resolving issues quickly with self-service, and freeing up time to better serve customers. Retail Core was enhanced and updated in the Yokohama release.
-   **[Retail Task Management Core release notes](retail/retail-task-management-rn.md)**  
The ServiceNow® Retail Task Management Core application optimizes the planning, organizing, and assigning of tasks to staff in retail environments. Retail Task Management Core is a new application in the Yokohama release.
