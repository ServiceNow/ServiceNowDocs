# Legal Virtual Agent Conversations release notes

The ServiceNow® Legal Virtual Agent Conversations application uses machine-learning algorithms for the predicting Subcategory and Assignment group for legal requests. This application is currently supported for the Zurich release, but it will be deprecated in a future release.

## Plugin information

-   **Deprecated plugins**

    The following plugins are planned for deprecation in a future release:

    [Legal Virtual Agent Conversations](https://www.servicenow.com/docs/access?context=legal-va-conversations&version=zurich&pubname=zurich-employee-service-management&ft:locale=en-US) \(sn\_lg\_virtual\_agent\): Planned for deprecation in C release.There is no replacement for this plugin.
