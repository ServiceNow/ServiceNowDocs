# Hiring Experiences release notes

Hiring Experiences is a suite of applications that offers a flexible and interactive workspace where hiring managers, recruiters, interviewers, and applicants can seamlessly perform their respective hiring tasks. Hiring Experiences has new applications and features in the Zurich release.

-   **[Applicant Center release notes](applicant-center-rn.md)**  
The ServiceNow® Applicant Center application provides your applicants with a dedicated portal that has a seamless process and supports effective engagement with the stakeholders in the hiring process. Applicant Center is a new application in the Zurich release.
-   **[Hiring release notes](hiring-tab-rn.md)**  
The ServiceNow® Hiring application in the Manager Hub streamlines the hiring process and enhances collaboration, accountability, and communication for hiring managers. Hiring is a new application in the Zurich release.
-   **[Interview management release notes](interview-management-rn.md)**  
The ServiceNow® Interview management application streamlines the interview setup and scheduling process by minimizing manual tasks and improving efficiency. Interview management is a new application in the Zurich release.
-   **[Recruitment workspace release notes](recruiter-workspace-rn.md)**  
The ServiceNow® Recruitment workspace application provides a flexible workspace for you as a recruiter to manage, prioritize, and perform your work. Recruitment workspace is a new application in the Zurich release.
-   **[Talent profile release notes](talent-profile-rn.md)**  
The ServiceNow® Talent profile application helps you get an aggregated view of different talent types at different touch points in the hiring workflow. Talent profile is a new application in the Zurich release.
