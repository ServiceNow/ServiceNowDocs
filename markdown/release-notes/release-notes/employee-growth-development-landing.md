# Talent Development release notes

The ServiceNow® Talent Development application enables managers and employees to identify and track career growth and development opportunities. It supports career growth through conversations, talking points, and ongoing development tracking. Talent Development was enhanced and updated in the Yokohama release.

-   **[Talent Development Core release notes](talent-development-core-rn.md)**  
The ServiceNow® Talent Development Core application supports an end-to-end process for individuals and managers to take control of their career growth and development plans. Talent Development Core was enhanced and updated in the Yokohama release.
-   **[Mentoring release notes](mentoring-rn.md)**  
The ServiceNow® Mentoring application offers improved networking opportunities and enables individuals to forge meaningful connections. Mentoring was enhanced and updated in the Yokohama release.
-   **[Skills Foundation release notes](skills-intelligence-skill-harmonization-rn.md)**  
The ServiceNow® Skills Foundation application provides an overview of skills-related data and enables admins to work on Job architecture data from a single location. Skills Foundation was enhanced and updated in the Yokohama release.
-   **[Talent Feedback release notes](talent-feedback-rn.md)**  
The ServiceNow® Talent Feedback application allows managers to request feedback on employee skills from the employee’s collaborators. Talent Feedback is a new application in the Yokohama release and is compatible with Xanadu and Washington DC releases.
-   **[Opportunity Marketplace release notes](opportunity-marketplace-rn.md)**  
The ServiceNow® Opportunity Marketplace application transforms internal mobility, providing employees with a unified hub to explore career opportunities. Opportunity Marketplace was enhanced and updated in the Yokohama release.
