# Growth Experiences release notes

The ServiceNow® Growth Experiences suite enables managers and employees to identify and track career growth and development opportunities. It supports career growth through conversations, talking points, and ongoing development tracking. Growth Experiences was enhanced and updated in the Zurich release.

-   **[Career Conversations release notes](career-conversations-rn.md)**  
The ServiceNow® Career Conversations application supports career growth through created conversations, talking points, and ongoing development tracking. Career Conversations was enhanced and updated in the Zurich release.
-   **[Mentoring release notes](mentoring-rn.md)**  
The ServiceNow® Mentoring application is a platform that connects individuals seeking mentorship with experienced mentors. It creates a seamless experience for both mentees and mentors. Mentoring was enhanced and updated in the Zurich release.
-   **[Skills Foundation release notes](skills-intelligence-rn.md)**  
The ServiceNow® Skills Foundation application provides an overview of skills-related data and enables admins to work on Job architecture data from a single location. Skills Foundation was enhanced and updated in the Zurich release.
