# Security Operations release notes

Security Operations has new and updated features in the Xanadu release.

-   **[Application Vulnerability Response release notes](secops-app-vuln-resp-rn.md)**  
The ServiceNow® Application Vulnerability Response application brings security and IT together to enable you to remediate your most critical vulnerabilities more quickly and efficiently. Application Vulnerability Response is included as part of the ServiceNow® Vulnerability Response application. Application Vulnerability Response was enhanced and updated in the Xanadu release.
-   **[Configuration Compliance release notes](secops-cc-rn.md)**  
The ServiceNow® Configuration Compliance application enables you to prioritize and remediate the most critical configuration-related vulnerabilities in your environment quickly and efficiently. Configuration Compliance was enhanced and updated in the Xanadu release.
-   **[Container Vulnerability Response release notes](secops-container-vuln-resp-rn.md)**  
The ServiceNow® Container Vulnerability Response application brings security and IT together to enable you to remediate your most critical vulnerabilities more quickly and efficiently. Container Vulnerability Response was enhanced and updated in the Xanadu release.
-   **[Now Assist for Security Operations release notes](secops-now-assist-security-operations-rn.md)**  
The ServiceNow® Now Assist for Security Operations application permits security analysts to resolve security incidents faster with intelligent workflows and generative AI skills. Now Assist for Security Operations is a new application in the Xanadu release.
-   **[Security Incident Response release notes](secops-sir-rn.md)**  
The ServiceNow® Security Incident Response \(SIR\) application helps your organization connect security and IT teams, respond faster and efficiently to threats, and view your organization's security posture. Security Incident Response was enhanced and updated in the Xanadu release.
-   **[Security Posture Control release notes](secops-security-posture-control-rn.md)**  
The ServiceNow® Security Posture Control application provides cybersecurity teams with visibility into security tool coverage gaps and deviations from security tool configuration for their enterprise assets. Security Posture Control was enhanced and updated in the Xanadu release.
-   **[Threat Intelligence Security Center release notes](secops-tisc-rn.md)**  
The ServiceNow® Threat Intelligence Security Center \(TISC\) application empowers your organization to connect security and IT teams so you can respond faster and more efficiently to threats.
-   **[Vulnerability Response integrations release notes](secops-vuln-resp-integrations-rn.md)**  
Third-party integrations with the ServiceNow® Vulnerability Response application help enrich your vulnerability data. The ServiceNow applications and third-party integrations that are compatible with the Vulnerability Response application were enhanced and updated in the Xanadu release.
-   **[Vulnerability Response release notes](secops-vuln-resp-rn.md)**  
The ServiceNow® Vulnerability Response application brings security and IT together to enable you to remediate your most critical vulnerabilities more quickly and efficiently. Vulnerability Response was enhanced and updated in the Xanadu release.
