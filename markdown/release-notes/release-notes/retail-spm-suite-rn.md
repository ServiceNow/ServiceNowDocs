# Retail Strategic Portfolio Management Suite release notes

The ServiceNow® Retail Strategic Portfolio Management Suite gives retail organizations a structured, repeatable way to plan and track store life cycle projects such as new store openings, relocations, refurbishments, technology refreshes, and closures. Retail Strategic Portfolio Management Suite is a new application in the Australia release.

## Retail Strategic Portfolio Management Suite highlights for the Australia release

-   Manage store life cycle projects such as new store openings, relocations, refurbishments, technology refreshes, and closures from a single platform.
-   Use predefined project templates to set up retail projects quickly and consistently, with prepopulated tasks and dependent task sequences.
-   Define the key dates to categorize and manage retail projects in the Project Type list.
-   Guide project teams through each stage of a store life cycle using stage-gate playbooks that define activities, approvals, and team handoffs.

See [Explore Retail Strategic Portfolio Management Suite](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/explore-spm-retail-suite.md) for more information.

## Retail Strategic Portfolio Management Suite features

-   **[Create a retail project](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/spmr-create-retail-projects.md)**

    Set up a retail project from a predefined template that matches the type of store work, such as a new store opening, relocation, refurbishment, technology refresh, or closure. Each template creates a project with prepopulated tasks and dependent task sequences.

-   **[Use playbooks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/use-playbooks-spm-retail.md)**

    Use the stage-gate playbook to guide a retail store project through each stage of its life cycle, from initiation to go-live or closure. Each stage includes prescribed activities, approvals, and team handoffs to confirm that required information is captured and governance processes are followed.


## UI changes

-   **[Retail Strategic Portfolio Management Suite playbooks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/use-playbooks-spm-retail.md)**

    The **Playbooks** option is available in the L-2 navigation menu of the Project Workspace for retail projects created from a template.

-   **[Project type fields](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/spmr-project-type-fields.md)**

    New Project Type form fields to mark important dates based on the created project type.


**Important:** Retail Strategic Portfolio Management Suite is available in the ServiceNow Store. For details, see the "Activation information" section of these release notes.

## Activation information

Install Retail Strategic Portfolio Management Suite by requesting it from the ServiceNow Store. Visit the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) to view all the available apps, and for information about submitting requests to the store. For cumulative release notes information for all released apps, see the [ServiceNow Store version history release notes](https://www.servicenow.com/docs/bundle/store-release-notes/page/release-notes/store/sn-store-release-notes.html).

## Plugin information

-   **New plugins**

    The following plugins are new in Australia:

    Retail Strategic Portfolio Management Suite \(com.snc.spm\_retail\): Provides predefined project templates, stage-gate playbooks, and roles for managing retail store life cycle projects in the Project Workspace.


## Additional requirements

The Project Workspace application must be installed before installing the Retail Strategic Portfolio Management Suite.
