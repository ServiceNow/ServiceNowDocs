# Now Assist for Vault release notes

The ServiceNow® Now Assist for Vault application generates custom data patterns, checks role access for an encrypted column, and schedules data discovery jobs. Now Assist for Vault can make it easier for admins to run common tasks without going to multiple systems. Now Assist for Vault is a new application in the Zurich release.

## Now Assist for Vault highlights for the Zurich release

[Zurich Patch 4](../quality/zurich-patch-4.md) The generate custom data pattern skill now uses the Now LLM Service as the default provider. You can switch to another provider as needed.

[Zurich release notes](../family-release-notes.md) The Now Assist for Vault application generates custom data patterns from text descriptions to streamline your workload, checks role access for an encrypted column to monitor your instance’s encryption access posture, and schedules data discovery jobs to detect sensitive data.

See [Now Assist for Vault](https://www.servicenow.com/docs/access?context=now-assist-vault-landing&version=zurich&pubname=zurich-platform-security&ft:locale=en-US) for more information.

**Important:** Now Assist for Vault is available in the ServiceNow Store. For details, see the "Activation information" section of these release notes.

## Now Assist for Vault features

-   **[Generate a custom data pattern by using Now Assist for Vault](https://www.servicenow.com/docs/access?context=generate-custom-data-pattern-now-assist-vault&version=zurich&pubname=zurich-platform-security&ft:locale=en-US)**

    The generate custom data pattern skill now uses the Now LLM Service as the default provider. You can switch to another provider as needed.


-   **[Using Now Assist for Vault](https://www.servicenow.com/docs/access?context=using-now-assist-vault&version=zurich&pubname=zurich-platform-security&ft:locale=en-US)**

    The Now Assist for Vault application generates custom data patterns from text descriptions to streamline your workload, checks role access for an encrypted column to monitor your instance’s encryption access posture, and schedules data discovery jobs to detect sensitive data. Enable these skills in Now Assist Admin console:

    -   Generate custom data pattern
    -   Check role access for encrypted column
    -   Schedule Data Discovery job
