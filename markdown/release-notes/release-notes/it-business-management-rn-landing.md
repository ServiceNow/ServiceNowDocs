# Strategic Portfolio Management release notes

Strategic Portfolio Management \(formerly IT Business Management\) has new and updated features in the Xanadu release.

-   **[Collaborative Work Management release notes](collaborative-work-management-rn.md)**  
The ServiceNow® Collaborative Work Management \(CWM\) application is an industry agnostic application that provides a central hub to plan, visualize, and collaborate on work with team members across the organization. CWM was enhanced and updated in the Xanadu release.
-   **[Goal Framework release notes](goal-framework-rn.md)**  
The ServiceNow® Goal Framework application enables your business to create goals, set targets for them, and evaluate the progress of the goals and targets to accomplish your organizational plans and drive business outcomes. Goal Framework was enhanced and updated in the Xanadu release.
-   **[Goal Framework for SPM release notes](goal-framework-for-spm-rn.md)**  
The ServiceNow® Goal Framework for Strategic Portfolio Management \(SPM\) application enables you to automate the actual value of your targets for goals that are defined using the ServiceNow® Goal Framework application. Goal Framework for SPM was enhanced and updated in the Xanadu release.
-   **[Microsoft Azure DevOps Integration for Agile Development release notes](microsoft-azure-devops-integration-for-agile-development-rn.md)**  
The ServiceNow® Microsoft Azure DevOps Integration for Agile Development application enables bidirectional synchronization of records between Azure DevOps and Agile Development 2.0. This application was enhanced and updated in the Xanadu release.
-   **[Now Assist for Strategic Portfolio Management \(SPM\) release notes](now-assist-for-spm-rn.md)**  
The ServiceNow® Now Assist for Strategic Portfolio Management \(SPM\) application introduces generative AI skills into Strategic Portfolio Management. You can summarize feedback or content in Docs, create stories for epics, generate concise project summaries, create demands through conversations, and quickly gain insights into projects. Now Assist for Strategic Portfolio Management \(SPM\) is updated in the Xanadu release.
-   **[Portfolio Planning release notes](portfolio-planning-rn.md)**  
The ServiceNow® Portfolio Planning application helps you enhance traditional product and portfolio management by visualizing the alignment of work with organizational objectives. Portfolio Planning was enhanced and updated in the Xanadu release.
-   **[Project Portfolio Management release notes](project-portfolio-management-rn.md)**  
The ServiceNow® Project Portfolio Management application enables you to create and manage a wide range of planning items ranging from a few small tasks to large portfolios of projects. Project Portfolio Management was enhanced and updated in the Xanadu release.
-   **[Project Workspace release notes](project-workspace-rn.md)**  
The ServiceNow® Project Workspace application provides an intuitive user experience that enables project managers to plan and manage their projects. Project Workspace was enhanced and updated in the Xanadu release.
-   **[Resource Management Workspace release notes](rsrc-mgmt-wrkspc-rn.md)**  
The ServiceNow® Resource Management Workspace application provides resource managers with a more efficient way to work with a dedicated workspace to view and manage resource assignments. Resource Management Workspace was enhanced and updated in the Xanadu release.
-   **[Scaled Agile Framework \(SAFe\) release notes](scaled-agile-framework-rn.md)**  
The ServiceNow® Scaled Agile Framework \(SAFe\) applications help you to manage strategic and operational work all from one place by using lean and agile principles. Various SAFe applications are planned for deprecation in the Xanadu release.
-   **[Strategic Planning release notes](strategic-planning-rn.md)**  
The ServiceNow® Strategic Planning application helps you accomplish end-to-end planning using a single workspace. Strategic Planning was enhanced and updated in the Xanadu release.
