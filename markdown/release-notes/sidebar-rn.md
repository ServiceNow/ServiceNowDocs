# Sidebar release notes

The ServiceNow® Sidebar application enables agents and subject matter experts to collaborate on user issues. Sidebar was enhanced and updated in the Zurich release.

## Sidebar highlights for the Zurich release

Prevent Sidebar discussions from becoming cluttered by using threaded replies when responding to messages.

See [Sidebar](https://www.servicenow.com/docs/access?context=sidebar-landing&version=zurich&pubname=zurich-conversational-interfaces&ft:locale=en-US) for more information.

**Important:** Sidebar is available in the ServiceNow Store. For details, see the "Activation information" section of these release notes.

## New in the Zurich release

-   **[Sidebar](https://www.servicenow.com/docs/access?context=sidebar-landing&version=zurich&pubname=zurich-conversational-interfaces&ft:locale=en-US)**

    Use threaded replies to answer Sidebar messages without adding clutter to the main discussion.


## Activation information

Sidebar is a ServiceNow AI Platform feature part of the Omni-Experience Standard Feature Set. For details, see [Installing Sidebar](https://www.servicenow.com/docs/access?context=install-sidebar&version=zurich&pubname=zurich-conversational-interfaces&ft:locale=en-US).
