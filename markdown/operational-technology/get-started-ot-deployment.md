# Get started with OT Discovery components

To get started with the Discovery for Operational Technology \(OT\) solution, you can try any of the following, suggested tasks.

## Tasks to get started

**Note:** See [Use the Discovery Console for OT interactive configuration wizard](../task/console-onboarding-wizard.md) for more information on how to complete these tasks.

**Service Graph Connector for ServiceNow OT Discovery**

Review these steps to get started with the Service Graph Connector for ServiceNow Operational Technology \(OT\) Discovery.

1.  [Install the Service Graph Connector for ServiceNow Operational Technology Discovery](../../sgc-ot-discovery/task/install-sgc-ot-discovery.md).
2.  Go through the [Service Graph Connector for OT Discovery guided setup](../task/sgc-ot-discovery-guided-setup.md).
3.  [Generate imports and validate the connections](../task/generate-imports-validate-connections.md).
4.  [Configure the OT Discovery connections &amp; credentials](../task/configure-the-ot-discovery-connections-credentials.md).

Discovery Console for OT

Review these steps to get started with the Discovery Console for OT component after installation. For more information about how to install the Discovery Console for OT, see [Install the Discovery Console for OT](../task/ot-discovery-console-installation.md).

1.  [Generate a certificate](../task/generate-new-certificate-discovery-for-ot.md).
2.  Verify the web Discovery Console for OT is running.
3.  Reboot the Discovery Console for OT.

**Discovery Sensor for OT**

To install the Discovery Sensor for OT, see [Install the Discovery Sensor for OT](../task/install-discovery-sensor-for-ot.md). For more steps with the Discovery Sensor for OT try:

1.  [Register the Discovery Sensor for OT](../task/registering-ot-sensor.md).
2.  Use the [Device Management Interface](dmi.md).

**OT Discovery Collector**

Review the following suggestions to get started with the OT Discovery Collector component after installation.

1.  [Use the OT Discovery Collector](../../msi-console/concept/use-ot-collector-tasks.md).
2.  [Restart OT Discovery Collector on a Windows system](../../msi-console/task/restart-discovery-collector-on-windows.md)
3.  [Restart OT Discovery Collector on a Linux system](../../msi-console/task/restart-collector-on-linux.md).
