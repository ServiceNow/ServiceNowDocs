# Setting up Hardware Vulnerability Assessment of OT devices using guided setup

Use Industrial Workspace Admin Guided setup to walk you through configuring the Hardware Vulnerability Assessment feature available on the Industrial Workspace menu.

## Before you begin

Role required: admin

## About this task

Use the Industrial Workspace Admin guided setup to assign required user roles, configure system property, and schedule jobs to perform hardware vulnerability assessment of OT devices.

## Procedure

1.  Navigate to **All** &gt; **Industrial Workspace Admin** &gt; **Guided Setup** &gt; **Operational Technology Vulnerability Response**.

2.  Select **Get Started** for the Operational Technology Vulnerability Response application.

3.  Select the **Hardware Vulnerability Assessment** task.

4.  Select the following task tabs, then select **Configure** to complete the configuration tasks.

    |Task|Purpose|
    |----|-------|
    |**User Roles assignment**|Assign users or user groups with roles that enable them to use the Hardware Vulnerability Assessment feature.|
    |**Configure Vulnerability Assessment properties**|Select the property that enables automatic creation of vulnerable items based on the hardware vulnerability assessment.|
    |**Schedule Vulnerability Assessment Jobs**|Execute scheduled jobs that enable vulnerability assessment on OT devices in the inventory.|


-   **[Assign roles for Hardware Vulnerability Assessment](assign-hva-roles.md)**  
Assign roles to users so that they can configure properties, and use hardware vulnerability assessments features, capabilities, and data.
-   **[Configure property for Hardware Vulnerability Assessment to create VIT automatically](configure-hva-properties.md)**  
Configure this property to automatically create vulnerable items \(VIT\) based on the hardware vulnerability assessment.
-   **[Run scheduled jobs to perform Hardware Vulnerability Assessment](jobs-hwd-vul-assessment.md)**  
Execute scheduled jobs to perform hardware vulnerability assessment.
