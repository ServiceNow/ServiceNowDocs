# Use the Discovery Console for OT interactive configuration wizard

The Discovery Console for OT provides an interactive configuration wizard that steps you through setting up your Console, Sensor, Collector, assets, queries, and sites.

## Before you begin

Role required: admin

## About this task

After you have installed and setup the Discovery Console for OT, the Discovery Sensor for OT, and the OT Discovery Collector on your system, the Console's interactive configuration wizard is available for use. The wizard steps through the initial tasks for using the Console, setting up Sensors and Collectors, and gives you an overall introduction to using the OT components.

## Procedure

1.  On the Console Home page, select the wand at the top of the page \(Tools\).

2.  When displayed, select the **Onboarding** tool.

    ![Onboarding tab](../images/onboarding-tab.png)

3.  The configuration wizard opens in the Console.

    ![Configuration wizard](../images/wizard-intro-screen.png)![]()

4.  You can collapse the wizard to the right side of the Console screen by selecting the ![Collapse](../images/move.png) icon in the top, right corner of your screen.

    Move the wizard to the right \(collapse\) to improve Console visibility.

    To expand and center the wizard on the screen, select the ![expand](../images/widen.png) icon.

    ![Wizard collapsed](../images/wizard-collapsed-to-right.png)

5.  The wizard steps you through tasks such as configuring your Console, connecting the Sensor and Collector, creating a site, and creating Auto Queries.

6.  Just follow the steps in the wizard and when you complete each one, mark them as **Done**.

7.  The wizard provides a **Skip** button so you can skip any of the optional steps.

8.  You can open the configuration wizard on any Console page by selecting the **Tool** wand.
