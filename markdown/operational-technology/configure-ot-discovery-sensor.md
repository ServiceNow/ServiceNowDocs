# Configure the Discovery Sensor for OT

Configuring the Discovery Sensor for OT provides discovery of all assets in your OT environment. The Sensor requires a Linux operating system that is installed either as a virtual machine or on a Bare-metal computer that is x86.

The workflow for configuring the Discovery Sensor for OT is as follows.

1.  Download and install the Discovery Console for OT package.
2.  Download the Discovery Sensor for OT package.
3.  Unzip the Sensor package and install the ISO image.
4.  Install the Discovery Sensor for OT.
5.  Use the DMI URL displayed in the final screen of the Sensor installation. Or use an URL specific to your IP address.
6.  Register the Discovery Sensor for OT using the DMI and the Discovery Console for OT.
7.  After the Discovery Sensor for OT is installed and registered, install the OT Discovery Collector.

You can start with [Install the Discovery Sensor for OT](install-discovery-sensor-for-ot.md).
