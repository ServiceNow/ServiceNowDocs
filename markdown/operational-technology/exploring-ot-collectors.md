# Explore the OT Discovery Collector

The ServiceNow OT Discovery Collector provides you with enhanced features to create an inventory of your assets in your OT environment.

## OT Discovery Collector overview

The Discovery OT Collector is a lightweight software application that is used to discover devices within segments of the OT network where a Sensor cannot reach. It is deployed deep within the network zones in your OT environment. The Collector delivers native protocol support without requiring any appliance, making it fast to implement and easy to scale.
