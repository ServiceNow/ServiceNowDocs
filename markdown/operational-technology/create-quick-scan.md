# Create a Quick Scan

Create a **Quick Scan** from the Auto Query page.

## Before you begin

Role required: admin

## About this task

A Quick Scan has fewer options to choose from, but it also runs quickly at approximately 5 minutes. After the scan runs, select the View Results in the Results column.

## Procedure

1.  Navigate to the Auto Query page.

2.  Select the **Action** button.

    ![Quick Scans](../../../images/action-button-quick-scan.png)

3.  From the Action drop-down menu, select **Quick Scans**.

    The Quick Scans page opens.

4.  Select the add icon ![](../../../../msi-console/image/add-icon-msi.jpg).

    Fill in each step and then select the **Next** button to move to the next step.

5.  In the **Identification** field, add a unique name or use the one provided.

6.  From the available selections, select a **Target**.

    Target selection includes:

    -   **Assets**: You can select more than one and search for an Asset from the list.
    -   **IP Addresses**: You can add one or more IP addresses to scan.
    -   **IP Address Ranges**: Enter an IP Range. Select the add icon ![](../../../../msi-console/image/add-icon-msi.jpg) to add additional IP ranges.
7.  In the **Sensors** step, select the box next to one or more listed Sensors.

8.  Select a **Scan Type**.

    You can choose from the Simplified Query Type selection or toggle the **Advanced** button and select from the advanced query types. See the [Auto Query page](../../auto-query-console.md) for additional information about Query Types.

9.  On the Confirmation page, select the **Run Quick Scan** button to start the scan.

    On the Confirmation page, a warning is displayed.

    ![Confirmation warning](../image/quick-scan-warning.png)
