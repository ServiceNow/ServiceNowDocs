# Deactivate a user

Deactivating a user ensures that they cannot access the Discovery Console for OT.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to the Users page.

2.  Next to the user you want to deactivate, select the Deactivate User icon ![](../../msi-console/image/delete-icon-ot-console.png).

3.  On the confirmation page, select **Deactivate User**.

    ![Deactivate user window](../../msi-console/image/deactivate-user-ot-console.png)
