# Home page

The Home page on the Discovery Console for OT provides a dashboard overview of the assets and sites on your current system.

In the navigation menu, you can select the Assets and Sensors pages to reveal sub-menus. You can select Sites, Users, and the Settings items for more data.

The Console opens to the Home page.

![Home page](../images/homepage.png)

The **Home** page is made up of a left-hand, expandable navigation menu and a main area; that is, the Assets Dashboard. The dashboard is divided into the following sections.

-   Assets section
-   Assets by Location section
-   KPI cards section

## Assets

The Assets section gives you an overview of the assets and sites available in your system, organized by widgets. The widgets are the following:

-   **All**: List of all the assets in your system.
-   **New**: List of the most recently discovered assets.
-   **Modified**: List of the most recently edited or updated assets.
-   **Sites**: List of all current sites.

By selecting the **All** widget, you can view the list of all the assets in your system. The **New** widget takes you to a list of the most recently discovered assets. The **Modified** widget takes you to a list of the most recently edited or updated assets. The **Sites** widget lists all current sites available in the Console.

Underneath these widgets are two graphs: **New Assets** and **Modified Assets**. The graphs display asset information organized by date.

Under these graphs are the **By Brand** and **By Port** graphs. The **By Brand** graph breaks up your assets according to brand. The **By Port** graph breaks up your assets based on the available ports that they're connected to. Selecting a section on either of these images takes you to the asset list filtered by the selected brand or port.

The following image shows an example of the data that you can find in the Assets section.

![Assets overview on the Home page dashboard](../../msi-console/image/assets-ot-console.png)

## Assets by Location

The Assets by Location section provides the number of assets according to the available site. The location of an asset is dependent on the Site's IP ranges that you define when creating a site. For more information about creating a site, see [Add a Site](ot-discovery-console-user-guide/task/add-site.md).

The **Top Locations** widget ranks each location based on its number of assets. The **New Assets** widget calls out any recently discovered assets by Site. The **Modified Assets** widget calls out recently edited or updated assets by site.

The following image shows an example of the data that you can find on the Assets by Locations section.

## Keyswitch / Protection

The Keyswitch / Protection section displays any key switch security or protection that you can have for your system assets. The following image shows an example of the data that you can find in the KeySwitch / Protection section.

**Note:** The Keyswitch / Protection section is only available if you have data available in your system.
