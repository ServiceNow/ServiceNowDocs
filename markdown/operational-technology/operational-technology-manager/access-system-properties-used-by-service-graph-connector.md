# Access system properties used by the Service Graph Connector

As an admin, view system properties related to the Service Graph Connector.

## Before you begin

View a filtered list of all the scripts that are used by the Service Graph Connector.

Role required: admin, cmdb\_inst\_admin, or cmdb\_ot\_admin

## Procedure

1.  Navigate to **All** &gt; **Industrial Workspace Admin** &gt; **OT Manager Admin** &gt; **Import OT Devices - System Properties**.

    The list shows records whose names begin with **sn\_otsm\_sgc.excel**.

2.  View the list of scripts.
