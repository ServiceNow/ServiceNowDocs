# Edit the protection policy for an OT View Rule

Edit the protection policy for an Operational Technology \(OT\) View Rule to edit the View Rule as needed.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System UI** &gt; **View Rules**.

2.  To only view the OT View Rules, apply the following filter:

    **\[Name\] \[contains\] \[OT\]**

3.  For the OT View Rule that you want to edit, select the **Read-only** value in the **Protection policy** column.

4.  Change the **Protection policy** column value to **--None--**.


## Result

Changing the **Protection policy** column value from **Read-only** to **--None--** allows users assigned the **admin** role to edit the OT View Rule.
