# Capability Details Page

View the various details for a capability using the Capability details page.

You can access capability details by selecting a capability from a product adoption roadmap or the a capabilities map.

The Capability details page displays information for a capability including description, relevant product adoption guidance, and it's alignment to business objectives.

You can use the **About** section to view detailed capability information to support informed decisions and **Related accelerators** to find relevant tools to help you implement and the capability.
