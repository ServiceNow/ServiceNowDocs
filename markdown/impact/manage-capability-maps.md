# Manage Capabilities Map

Explore and manage the usage status for the capabilities you are entitled to use.

## Before you begin

Role required: Impact Platform Owners, Impact Admins, Impact Executives, and Impact Users

## About this task

Update usage status view related resources, edit notes, hide, or display the capabilities from your instance using Capability Maps. The product Adoption Summary information on the page also displays the number of capabilities by their usage status and the capability recommendations from the Impact Squad. You can also add a custom capability.

## Procedure

1.  Navigate to **Impact Workspace ** &gt; **Product Adoption** &gt; **Capabilities map**.

2.  In **Capabilities** section, view more information about each capability by expanding the ServiceNow workflows.

    The grid displays the important information such as usage status and last updated date along with the access to the Notes for the capabilities: ![Capabilities for every workflow along with the necessary information such as usage status and last updated date](../image/store-capabilities-section.png)

3.  Navigate the workflows and select a product.

    All capabilities for the selected workflow and product are displayed in a table with color-coded status indicators![List of Capabilities indicating Usage status such as Needs Validation and Not in Use.](../image/store-capabilities-selected.png)

4.  Manage the capabilities by using the following features:

    -   **[Update capability usage status](../reference/manage-capability-features-for-store.md#update-capability-usage-status)**

        Modify the capability Usage Status for one or many capabilities

    -   **[View related resources](../reference/manage-capability-features-for-store.md#manage_capabilities_view_related_resources)**

        View all supporting resources.

    -   **[Hide and Unhide capabilities](../reference/manage-capability-features-for-store.md#section_hide_unhide_capabilities)**

        Show or Hide capabilities in the grid.

    -   **[Edit Capability notes](../reference/manage-capability-features-for-store.md#section_edit_capabilities_note)**

        Modify information related to a capability.

    -   **[Add a new Capability](../reference/manage-capability-features-for-store.md#section_add_new_capability)**

        Add a new capability manually.

    -   **[Capability Details Page](../reference/capability-details.md)**

        To view and edit details for a capability.
