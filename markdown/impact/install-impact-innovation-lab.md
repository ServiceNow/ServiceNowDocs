# Install the Impact Store Application from the ServiceNow Store

Follow these instructions to install the Impact Store Application.

## Before you begin

-   Versions supported: Yokohama and Xanadu Patch 2 and above
-   Impact Store Application ServiceNow entitlement \(sn\_entitlement\) version 4.1.2 or higher.

    **Note:** Applications that you purchase from the ServiceNow Store are added to the Application Manager automatically. Install the application from the Application Manager to activate it in an instance. Review the Impact main application listing in the ServiceNow Store for information on dependencies, licensing or subscription requirements, and release compatibility.

-   A named administrator must be allocated as the contact to perform the installation and implementation of the Impact Store Application and receive the registration email.

Role required: Any Impact role, admin

## Procedure

1.  Navigate to the ServiceNow Store.

2.  Select **Apps and Solutions**.

3.  Find Impact using the filter criteria and search bar.

    If you cannot find the application, contact your Impact Squad for additional assistance on procuring the Impact Store Application.

4.  Select **Buy** to procure the Impact Store Application.

5.  Log in to your ServiceNow® instance.

6.  Navigate to **All** &gt; **Application Manager**.

7.  Search for Impact.

    You may need to refresh the sync with the ServiceNow Store with the **Sync now** option if Impact doesn't display in the list.

8.  Select **Install** to complete the installation.

9.  Select **Configure**.

    ![Configure option for Impact in the Application Manager.](../image/configure-impact-store.png)

    You are redirected to the Impact Guided Setup.
