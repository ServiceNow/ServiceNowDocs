# Value Journey

Use Value Journey to get a closer look at your **Value blueprint**, and **Business Value Reports**.

## Value Journey overview

The value journey constitutes a continuous loop of the end-to-end process of managing the value lifecycle, inclusive of envisioning value, creating value, ongoing management, optimization, and validation of value realized. By default, this cycle is executed on a yearly basis, although deviations of this time frame are possible by exception.
