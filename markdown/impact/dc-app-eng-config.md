# Configure Data Collection for App Engine

Configure Data Collection for App Engine.

## Before you begin

You must update Manual Indicators periodically \(monthly, for example\) using the scoresheet function.

Role required: admin, pa\_power\_user, pa\_admin, or pa\_data\_collector

## Procedure

1.  Navigate to **Performance/Platform Analytics &gt; Scoresheet**, and then select **Impact VM - Former LCAP systems run-rate**.

2.  Enter the data point in the relevant month cell.

    ![Example with Jul 2024 cell selected with an Indicator score of 5,000.](../image/dct_config_appeng.png)
