# Impact Value Management Data Collection Content Pack for HAM

Install and configure the Data Collection Content Pack for HAM.

Perform the steps in the following order:

1.  [Install the Data Collection Pack for HAM](../task/dc-ham-install.md)
2.  [Configure Data Collection for HAM](../task/dc-ham-config.md)
3.  [Activate Data Collection for HAM](../task/dc-ham-activate.md)
4.  [Review HAM artifacts](../reference/dc-ham-install-artifacts.md)

You can also modify the configuration at any point, and maintain customizations when you are ready to upgrade. For more information, see [Modifying the Data Collection configuration](dc-all-modify.md) and [Upgrading the Data Collection Content Pack](dc-all-upgrade.md).
