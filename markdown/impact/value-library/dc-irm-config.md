# Configure Data Collection for IRM

Configure Data Collection for IRM.

## Before you begin

For the Manual Indicators, you must add a new data point every month. For example, given the fixed nature of the Impact VM - Legacy IRM Systems Annual Run-Rate indicator, you only need to enter this data point once.

**Important:** If you do not have full access to Performance/Platform Analytics through a Pro or Enterprise subscription, you are required to enter this data point every month

Role required: admin, pa\_power\_user, pa\_admin, or pa\_data\_collector

## Procedure

1.  Navigate to **Performance/PlatformAnalytics** &gt; **Scoresheet**, and then select **Impact VM - Legacy IRM Systems Annual Run-Rate**.

2.  Enter the data point in the relevant month cell.
