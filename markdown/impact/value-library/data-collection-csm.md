# Impact Value Management Data Collection Content Pack for CSM

Install and configure the Data Collection Content Pack for CSM.

**Important:** You should test system configuration changes in a sub-production environment before you deploy them into production.

Perform the steps in the following order:

1.  [Install the Data Collection Pack for CSM](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/value-library/dc-csm-install.md)
2.  [Configure Data Collection for CSM](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/value-library/dc-csm-config.md)
3.  [Activate Data Collection for CSM](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/value-library/dc-csm-activate.md)
4.  [Review CSM artifacts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/value-library/dc-csm-install-artifacts.md)

You can also modify the configuration at any point, and maintain customizations when you are ready to upgrade. For more information, see [Modifying the Data Collection configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/value-library/dc-all-modify.md) and [Upgrading the Data Collection Content Pack](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/value-library/dc-all-upgrade.md).
