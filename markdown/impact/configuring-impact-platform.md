# Configuring the Impact Store Application

The tasks are outlined that are required to install and configure the Impact Store Application.

1.  [Install the Impact Store Application from the ServiceNow Store](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/install-impact-innovation-lab.md)
2.  [Initiate the connection to the Impact Delivery Instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/initiate-the-connection-impact-delivery-instance.md)
3.  [Connect to the provider instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/connect-instance-impact-store-app.md)
4.  [Use Guided Setup to synchronize and migrate data to the Impact Store Application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/guided-setup-impact-in-app.md)

**Note:** The configuration and connection steps must be performed for each instance to connect to the Impact Delivery Instance for multi-instance configurations. See [Impact multi-instance-support](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/impact-multi-instance-support.md) for additional configuration information.

## Using Guided Setup to implement the Impact Store Application

Guided setup provides a sequence of tasks that help you configure the Impact Store Application in your ServiceNow instance. To open guided setup for Impact, navigate to **All** &gt; **Adoption Services** &gt; **Guided Setup** &gt; **Configure products** &gt; **Impact Guided Setup**.

For more information, see [Use Guided Setup to synchronize and migrate data to the Impact Store Application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/impact/guided-setup-impact-in-app.md).
