# Instance Observer user configurable dashboard

Users can create their own dashboard according to their preferences of metrics that help users to analyze the health of their system in a snapshot.

The default base system dashboard, the All instance health dashboard, is the Instance Observer default homepage that provides a snapshot of the overall health of your instances with key insights, such as performance, reliability, and availability at a glance with the following widgets:

-   Alerts: The alerts for that account for the last 24 hours
-   Instance availability: The availability across all instances
-   Open cases: All open cases for that account
-   Known Issues \(ePRBs\): All known issues with ServiceNow
-   Clusters: Cluster details of all instances in the account
-   KPIs: Important KPIs for all instances of the account

Users can select the auto-refresh frequency for the dashboard.

**Note:** The All Instance health dashboard dashboard is not editable, deletable, or cloneable. Create a new dashboard to explore configuration options. See [Create a dashboard](customize-instance-observer-dashboard.md) for details.
