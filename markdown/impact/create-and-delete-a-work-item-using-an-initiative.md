# Create and delete a work item using an initiative

Create an actionable Strategic Planning Workspace \(SPM\) or Collaborative Work Management \(CWM\) work item using free-form initiatives.

## Before you begin

Role required: Impact Platform Owner, Impact Portfolio Owner.

-   An appropriate role with write and delete permissions to create or delete work items.
-   A Collaborative Work Management role and a personal workspace to create a CWM work item.

## Procedure

1.  Navigate to **All** &gt; **Impact** &gt; **Accelerators and initiatives** &gt; **Catalog**.

2.  Select the initiative.

3.  Select **My work items** from the contextual side panel.

4.  Select **Convert to work item**.

    \[Omitted image "create-workitem-from-initiative.png"\] Alt text: Select Convert to work item under My work items in an initiative.

5.  On the work items form, fill in the fields.

    For more information, see [Planning item type form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/it-business-management/scenario-planning-in-spw/new-planning-item-planning-org-form.md).

6.  Select **Convert**.

    A new work item is created, and you can open it by selecting the link in the success message.

7.  Open the new work item by selecting the link in the success message that appears.

    **Note:** You can also access the work item under **my work items tab** in the recommendation or on your SPM or CWM dashboard.
