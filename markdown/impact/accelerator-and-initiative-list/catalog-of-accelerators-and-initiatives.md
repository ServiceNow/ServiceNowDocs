# View the Initiatives and Accelerators catalog

Learn about available Accelerators and initiatives that you can request to help achieve your business outcomes for your instance.

\[Omitted image "catalog\_accelertors\_initatives\_new.png"\] Alt text: Catalog of Accelerators and Initiatives with the All tab selected

## Usage by category

The catalog page displays the number of accelerators and initiatives in use along with the entitled usage limits. For example, 1/2 in use indicates that the instance uses one accelerator out of the permissible limit of two. You can choose to browse the catalog of accelerators and request another suitable accelerator to make the best use of the entitlement in the package.

**Note:** The number of Accelerators and initiatives that you're entitled to is based on your subscription package. See  for additional information.

## All Accelerators and initiatives

The **All** tab displays flash card views of all the available accelerators and initiatives. Each flash card displays the estimated duration of delivery and a short description that serves as a quick reference.

**Note:**

The **Recommended** label on a flash card indicates Accelerators that have improved the business outcomes for many other customers. Therefore, you might want to check those cards first because they might be useful when you explore new accelerators for your business. The Impact squad also recommends some of these accelerators based on data.

The tabbed view of Accelerators and initiatives offers the following views of the accelerators and initiatives:

-   Filtered view: View selective flash cards based on filters such as Bookmarked, Recommended, Request History, or Type by using the Quick Filter icon \(\[Omitted image "icon-filter-list.png"\] Alt text: Quick Filter\).
-   Sorted view: View organized flash cards sorted by alphabetical order, creation time, and duration by using the Sort icon \(\[Omitted image "icon-sort-list-catalog.png"\] Alt text: Sort icon\).
-   Searched view: View a specific accelerator flash card by using the Search icon \(\[Omitted image "icon-search-cataloglist.png"\] Alt text: Search icon\).
-   Show on-demand accelerators only: View only the available on-demand accelerator flash cards from the catalog.

## Bookmark an Accelerator

You can mark an accelerator flash card as a favorite for a quick reference for citations, discussions, decisions, and delivery by selecting the bookmark icon \(\[Omitted image "bookmark-catalog.png"\] Alt text: Bookmark icon\).

You can see a filtered view of bookmarked flash card by using the Quick Filter icon.
