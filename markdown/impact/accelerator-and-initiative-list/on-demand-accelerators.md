# On-demand accelerators

Customers can complete the full accelerator fulfillment process independently, without requiring squad assistance.

## Accelerator Overview

On-demand accelerators provide a fully self-serve experience for Impact customers with a 90-day action plan. Engagement-type filtering, progress tracking, and questionnaire persistence are accessible from the catalog.

## Self-serve accelerator fulfillment workflow

The workflow consists of consecutive tasks across the following activity types:

-   **Questionnaire**

    Helps assess the customer requirement.

    The UI presents questions specific to each accelerator.

-   **Transformation**

    Helps identify what needs to change by analyzing how processes work and where gaps exist.

    This activity addresses the following tasks:

    -   Prioritize gaps: Filter items that are struggling or have a gap that needs to be addressed.
    -   Select the path: Select the items to address and decide the path for the workflow.
-   **Component**

    Helps create an action plan.

    After the accelerator workflow completes, an action plan is generated based on the inputs provided in earlier steps. You can export the action plan as an MS Excel file for stakeholder presentations. You can configure the report to other time frames, such as weekly, monthly, or daily.
