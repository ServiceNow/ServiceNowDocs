# Consumption Report

The Consumption Report provides an overview of your Impact tier features and their usage.

With the Consumption Report, you can identify under utilized or not yet used benefits. The Consumption Report filters usage to an adjustable specific period of time.

**Note:** The Consumption Report defaults to a year long range.
