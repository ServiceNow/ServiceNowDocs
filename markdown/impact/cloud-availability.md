# Cloud availability

Check specific ServiceNow data center availability, service degradation, or for interruption.

The cloud availability is displayed in a world map view with colored status indicators representing the availability of each data center.

-   Available: Green
-   Service degradation: Yellow
-   Service outage: Red

![Display of the world map with colored status indicators representing the availability of each data center.](../image/data-center-status-map.png)

See which data center is available and if there is an outage or a specific trend of actvity. Select a specific data center from the map to expand the details.

![Shows the data center details page after expanding from the cloud map.](../image/data-center-details.png)

Go directly to Instance Observer, view summaries of the availability, and details of the last 90 days service issues.
