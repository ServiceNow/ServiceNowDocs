# Create and delete a work item using a recommendation

Create an actionable Strategic Planning Workspace \(SPM\) or Collaborative Work Management \(CWM\) work item using impact squad-based or rule-based recommendations.

## Before you begin

Role required: Impact Platform Owner, Impact Portfolio Owner

## Procedure

1.  Navigate to **All** &gt; **Impact** &gt; **Impact Workspace** &gt; **Home**.

2.  Select the recommendation from the **Recommended next steps** section.

3.  To view the complete list of recommendations, select **View full list**.

4.  Convert the recommendation to a work item.

    -   If you are sure you want to convert the recommendation, from the **View Details** drop-down list in the recommendation entry, select **Convert to work item**.
    -   If you want to see more details about the recommendation and then decide to convert the recommendation, select the **View Details** button and then in the dialog box, select the **Convert to work item** button.
5.  Select **Convert to work item** in a recommendation using one of the following:

    -   Select **View Details** for a recommendation.
    -   Select more actions in the **View Details** list.
6.  Select a work item from the list, and then select **Next**.

    **Note:** If you create a work item using a recommendation, it appears unavailable \(greyed out\) in the list.

7.  On the work item form, fill in the fields.

    For more information, see [Planning item type form](https://www.servicenow.com/docs/access?context=new-planning-item-planning-org-form&version=yokohama&pubname=yokohama-it-business-management&ft:locale=en-US).

8.  Select **Convert**.

9.  Open the new work item by selecting the link in the success message that appears.

    **Note:** You can also access the work item under the **My work items** tab in the recommendation or on your SPM or CWM dashboard.
