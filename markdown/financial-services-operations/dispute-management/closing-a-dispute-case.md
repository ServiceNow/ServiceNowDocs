# Closing a dispute case

After you close a transaction in the playbook, the case is set to the **Closure** state.

The dispute agent finalizes disputed transactions and moves them to closure. This process involves filling in closure details for the case and handling any necessary card reissuance in fraud cases.

|Activity|Description|
|--------|-----------|
|Closure details for the case|Fill in the necessary closure details and close the case.|
|Block and reissue card|For fraud-related disputes, block and reissue the card before closing the case.|
