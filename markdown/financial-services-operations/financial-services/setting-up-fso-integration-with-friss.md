# Setting up Financial Services Operations Integration with FRISS

Integrate your FRISS account with your ServiceNow® instance so that you determine whether the claim is legitimate or fraudulent.

## Application setup

Set up your integration:

1.  Request an Integration Hub subscription. To learn how, see [Request Integration Hub](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/integrate-applications/integration-hub/request-ih-overview.md).
2.  Activate Financial Services Operations Integration with FRISS from ServiceNow Store, which automatically activates the FRISS spoke.
