# Integrate applications

Financial Services Operations \(FSO\) integrations provide seamless client experiences with automated verification and acceptance.

This section provides information about our FSO integration with spokes and applications built using these integrations.

-   **[Integrating with spokes](spokes.md)**  
By integrating spokes with the Financial Services Operations \(FSO\) applications, flow designers can provide the actions within Workflow Studio for specific applications.
-   **[Integrating with third-party applications](fso-embedded-spokes-overview.md)**  
Provides information about the integrated spokes and tasks that must be performed or requirements that are needed, to use the spoke. These requirements are specific to the third-party application.
-   **[Integrating with other ServiceNow applications](fso-integrated-applications-overview.md)**  
Includes shared applications that are complete and are ready for use on other instances. You can extend the capabilities of Financial Services Operations and integrate with other ServiceNow applications.
