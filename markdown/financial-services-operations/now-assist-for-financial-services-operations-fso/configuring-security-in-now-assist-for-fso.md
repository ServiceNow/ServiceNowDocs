# Configuring security in Now Assist for Financial Services Operations \(FSO\)

You can enable security implementation on Now Assist for FSO skills through access control lists \(ACLs\) and user identities. These ACLs determine which users have permissions to discover and invoke these features.

Predefined ACLs exist in Now Assist for FSO for case summarization skills.

See [Configure an ACL rule](https://www.servicenow.com/docs/access?context=t_CreateAnACLRule&version=xanadu&pubname=xanadu-platform-security&ft:locale=en-US) for more information on configuring ACLs.
