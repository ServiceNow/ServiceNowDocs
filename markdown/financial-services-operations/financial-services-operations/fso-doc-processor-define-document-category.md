# Define a document category

Define document categories to categorize the different document types that are used across Financial Services Operations applications. An example of a document category could be Proof of Identity.

## Before you begin

Role required: sn\_doc\_processor.admin

## Procedure

1.  Navigate to **Document Processor** &gt; **Administration** &gt; **Document Categories**.

2.  In the list, click **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the category.|
    |Active|Option to activate this category.|

4.  Select **Submit**.
