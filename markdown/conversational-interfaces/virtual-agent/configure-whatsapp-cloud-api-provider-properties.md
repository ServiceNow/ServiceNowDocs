# Configure WhatsApp Cloud API provider properties

You can customize the behavior of the Conversational Integration with WhatsApp \(WhatsApp Cloud API\) application by configuring its provider properties.

## Before you begin

Role required: admin

## Procedure

1.  In the filter navigator, enter `sys_cs_provider.list`.

2.  In the Connections list, select **VA Whatsapp Adapter Provider**.

3.  Select the **Provider Properties** tab.

4.  In the Custom Adapter Properties list, select the property that you want to update.

    **Note:** If a message appears about the application scope, select here to be able to edit the record.

5.  Update the **Value** field and select **Update**.


## Result

The provider property is updated with the new value.
