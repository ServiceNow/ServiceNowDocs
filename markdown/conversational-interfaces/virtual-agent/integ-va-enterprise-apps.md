# Integrating Virtual Agent with enterprise messaging apps

Integrating enterprise messaging apps such as Slack, Microsoft Teams, and Workplace with ServiceNow® Virtual Agent associates the app with your ServiceNow instance to facilitate conversations across the enterprise.

Conversational integrations are available for Slack, Microsoft Teams, and Workplace from the ServiceNow Store.
