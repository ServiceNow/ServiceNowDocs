# Unsupported Virtual Agent features for Conversational IVR

Refer to the unsupported features of Virtual Agent in Conversational IVR with Amazon Connect.

## Unsupported features

Chat Surveys: Conversational IVR with Amazon Connect supports surveys but it cannot read the contents of the survey summary card, which is a card type control that IVR does not support currently.

To effectively use a survey with Conversational IVR, you must ensure to turn off the **Show Summary** check box when creating or configuring a survey that is applicable to IVR conversation.
