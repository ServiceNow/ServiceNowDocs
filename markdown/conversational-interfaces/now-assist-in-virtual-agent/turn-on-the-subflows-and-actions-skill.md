# Turn on the subflows and actions skill

Turn on the subflows and actions skill to run subflows and actions from Now Assist conversations.

## Before you begin

-   Configure the Now Assist in Virtual Agent application. See [Configuring Now Assist in Virtual Agent](../../now-assist-in-va/task/configure-now-assist-va.md).
-   Role required: virtual\_agent\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **Now Assist Admin** &gt; **Now Assist Features**.

2.  In the workflow list, select **Platform**.

    ![Platform skills are available in Now Assist Features when you install any Now Assist plugin.](../images/na-platform-skills.png)

3.  In the Conversational experience card, select **View details**.

4.  Select **All available Conversational experience skills** &gt; **Subflows and actions** &gt; **Turn on skill**.

5.  In the Conversational experience card, verify that the **Subflows and actions** skill is activated under **All Conversational experience skills**.
