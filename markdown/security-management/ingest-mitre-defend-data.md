# Ingest MITRE DEFEND data

Ingest MITRE D3FEND data \(tactics, techniques, and artifacts\) from the MITRE website to integrate with SIR.

## Before you begin

Role required: sn\_ti.admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Scheduled Jobs**.

2.  Select **MITRE DEFEND ingestion** from the list of scheduled jobs.

3.  Select **Execute Now** to start the data ingestion.
