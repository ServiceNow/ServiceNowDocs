# Security Operations Integrations - Get Network Statistics workflow

The Security Operations Integrations - Get Network Statistics workflow retrieves a list of active network connections from a host or endpoint.

## Before you begin

Role required: sn\_si.analyst

## About this task

This workflow runs automatically when a configuration item is added to a security incident.
