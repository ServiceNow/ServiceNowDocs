# Download TISC application from ServiceNow Store

Before you run the Threat Intelligence Security Center application in your instance, you must download it from the ServiceNow® Store and install it.

## Before you begin

Role required: sn\_sec\_tisc.admin

## Procedure

1.  Follow the instructions for [downloading an application from the ServiceNow Store](../../security-incident-response/reference/download-app-first-time.md) and install the **Threat Intelligence Security Center Security Operations** application.

2.  After installation is complete, you can open the Threat Intelligence Security Center Workspace by navigating to **Workspaces** &gt; **Threat Intelligence Security Center**

    ![Download and install TISC app from the workspace.](../image/tisc-download-app.png)


## What to do next

After you successfully downloaded the application, verify that you’ve completed all the tasks mentioned here in this section, [Set up Threat Intelligence Security Center](set-up-threat-intelligence-security-center.md) for a smooth integration.
