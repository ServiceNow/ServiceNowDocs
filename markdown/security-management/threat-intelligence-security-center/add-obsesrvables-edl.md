# Add Observables to EDL

Add observables such as IPs, domains, and hashes to External Dynamic List \(EDLs\).

## Before you begin

Role required: sn\_sec\_tisc.analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center**.

2.  Click **Threat Analyst Workbench** icon.

3.  Go to **Observables** &gt; **All Observables**.

4.  Open any observable record.

5.  Click **Add to EDL**.

    The Add to EDL modal screen is displayed.
