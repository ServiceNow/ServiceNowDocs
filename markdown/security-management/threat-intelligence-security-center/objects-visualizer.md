# View details in Relationship Graph

Relationship Graph for observables, indicators, and various other entities in TISC has associated relationship graphs. Using the Visualizer, you can view the relationships between observables, indicators, and entities which provides context for you to further investigate. The Visualizer uses colors and icons to illustrate various information about the objects.

Role required: sn\_sec\_tisc.analyst

The relationship graphs displays two different levels of nodes association as part of its related records.

![Objects Visualizer](../image/tisc-visualizer.png "Two levels of Relationship Graph")

You can do the following:

-   Zoom into any object, fit into the screen, drag the nodes, or pin them to focus on a particular relationship.
-   Click on any object to open the details pane. On the details pane, you can learn more information that is associated with the object. You can view the node properties such as Object, Object Name, Description, and Sources. By default the details section of each graph node is not displayed and you have to click on each node in the graph to view the details of the selected node.

    **Note:** The Visualizer is available for all the STIX Domain Objects \(SDOs\) except the Observed Data objects. Click **Refresh** button to refresh the visualizer.

-   You can also click on the pop out icon to view the graph in a much larger window.

-   **[Link node actions in Relationship Graph](../task/tisc-link-nodes.md)**  
Add new related records to link the nodes.
