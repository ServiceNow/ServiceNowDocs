# Create users in TISC instance

Users can be created in the ServiceNow TISC instance with any valid user role \[`sn_sec_tisc.api_obs_read_access]`.

## Before you begin

Role required: Splunk admin.

## Procedure

1.  Log in to **Splunk**.

2.  Navigate to **Apps** &gt; **Manage Apps** or select the **Manage** icon.

3.  Search for the **Threat Intelligence Security Center for Splunk** app and install it.

4.  After installation, the application appears in the list of apps.

    For more information, see [Assign a role to a user](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/t_AssignARoleToAUser.md).
