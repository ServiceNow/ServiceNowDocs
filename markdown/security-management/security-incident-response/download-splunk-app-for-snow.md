# Download the ServiceNow Security Operations application

The first step in setting up the ServiceNow-to-Splunk integration is to download the ServiceNow Security Operations application from Splunkbase.

## Before you begin

Role required: admin

## Procedure

1.  Open Splunkbase.

2.  Search for [ServiceNow Security Operations Integration](https://splunkbase.splunk.com/apps/#/page/1/search/servicenow/order/relevance).

3.  Download the application.
