# Integrate MSIM Conference Calls with Cisco Webex

The MSIM Conference Call Cisco Webex integration enables you to manage and initiate a Cisco Webex conference call directly from a major security incident.

## Key Features

-   Start and end Cisco Webex meetings directly from major security incidents.
-   Manage a Cisco Webex meeting by adding or removing participants.

## Prerequisites

Verify you’ve installed the following Integration Hub plugins on your ServiceNow instance.

-   Notify plugin \(com.snc.notify\)
-   Notify Cisco Webex Connector \(sn\_notify\_webex\)
-   Cisco Webex Teams Spoke \(sn\_cisco\_teams\_spk\)
-   Notify UI Components for Configurable Workspaces

For more information, see [Request IntegrationHub](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/request-ih-overview.md).
