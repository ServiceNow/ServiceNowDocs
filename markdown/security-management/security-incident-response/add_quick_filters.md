# Add or modify quick filters

Add or modify quick filters for security incidents or response tasks within the list view.

## Before you begin

Role required: sn\_si.basic

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspace**.

2.  Click the list \(\[Omitted image "listview-icon.png"\] Alt text: list icon\) icon.

3.  For example, go to **Lists** &gt; **Security Incidents** &gt; **Quick Filters** section.

    \[Omitted image "quick-filters1.png"\] Alt text: quick filters

4.  Click **Edit** icon to make the changes.

    **Note:** The change that you make will be persisted, and when the user returns back to this page the next time then the persisted preference is displayed.

    \[Omitted image "esit-quick-filters.png"\] Alt text: Edit quick filters

5.  Click **Save**.
