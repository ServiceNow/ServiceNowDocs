# View Proofpoint Analytics Dashboard

This Proofpoint analytics dashboard provides details of VAP Users and Top Clickers.

## Before you begin

Role required: sn\_si\_admin

## Procedure

1.  Navigate to **All** &gt; **Workspaces** &gt; **Security Incident Response Workspace**.

2.  Select **SIR Dashboards**.

3.  Select **Proofpoint Analytics Dashboard** from the drop-down list.

    The Proofpoint Analytics Dashboard appears.![Proofpoint Analytics Dashboard](../image/proofpoint-analyst-dashboard-sir.png)

4.  Select **VAP Users** tab to view the list of VAP users.

    Select a VAP user to view its details.

5.  Select **Top Clickers** tab to view the list of Top Clickers.

    Select a top clicker to view its details.
