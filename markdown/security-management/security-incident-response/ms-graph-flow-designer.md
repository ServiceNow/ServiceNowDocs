# Flow Designer and Integration Hub usage in Microsoft Graph Security API integration

Using the Flow Designer and Integration Hub functionality, several subflows and actions have been built as part of the Microsoft Graph Security API integration.

The following subflows are available:

-   Microsoft Graph Security API Configuration Validation: This is used in the configuration tile to validate the credentials in the initial setup.
-   REST Call For Microsoft Graph Security API: This is used for any interactions with Microsoft Graph security API using REST calls.
-   Retrieve Sample Alerts: This is used in the Mapping section of the profile setup to fetch sample data. This sub-flow is triggered asynchronously.
