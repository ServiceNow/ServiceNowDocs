# Example 1: Select implementations: Threat Lookup

The example below demonstrates the implementation\(s\).

## Example 1: Only Screen 1: Threat Look Up

You can perform **Run Threat Lookup** related integration capabilities action using the **Investigation** tab from the SIR Workspace.

1.  On the **Investigation** tab, navigate to the **Entry Points Lists** section displayed on the left side of the page.
2.  Select any entry point and run the integration capability action.

\[Omitted image "investation-tab-related-list.png"\] Alt text: Investigation tab related records list

**Note:** You can also navigate to the **Related Records** tab on the workspace to perform the integrations capabilities action.

**Threat Look Up** is one of the example which is described in detail on how to perform the integrations.
