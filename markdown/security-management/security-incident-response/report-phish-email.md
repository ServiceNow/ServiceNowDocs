# Report Phish Email

Report phishing emails from the lists view.

## Before you begin

Role required: sn\_si.analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspace**.

2.  Click the list \(\[Omitted image "listview-icon.png"\] Alt text: list icon\) icon.

3.  Go to **User Reported Phishing Emails** &gt; **All**.

    The user reported phishing emails list is displayed.

    \[Omitted image "phish-email-list.png"\] Alt text: Phishing email list

4.  Click **Report Phish Email** button.

    Report a phishing email by uploading it.

5.  Click **Add File** to add attachments.

6.  Click **Upload**.

    **Note:** Report a Phish Email by uploading the attachments. You can upload only one phishing email at once and should be in .eml format.

7.  Click **Submit**.

    The phishing email to log a security incident is created.
