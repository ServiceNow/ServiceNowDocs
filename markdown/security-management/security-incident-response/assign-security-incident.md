# Assign Security Incidents

Assign security incidents.

## Before you begin

Role required: sn\_si.basic

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspace**.

2.  Click the list \(\[Omitted image "listview-icon.png"\] Alt text: list icon\) icon.

3.  Click **Lists**.

4.  For example, go to **Security Incidents** &gt; **All Open**.

5.  Select one or more security incidents.

6.  Click **Assign**.

7.  Fill the details such as Assignment group, Assigned to, and add Work notes if required.

    The incident will then be assigned to the user.

8.  Click **Save**.
