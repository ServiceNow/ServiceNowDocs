# Create a category

Create an security incident category that you want to use to classify security incidents.

## Before you begin

Role required: sn\_si.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspaces** &gt; **Administration**.

2.  Select **Category Management**, and then **Category**.

3.  Select **New**.

    The Details page appears. The Table, Element and Language fields are preconfigured for the current user.

4.  In the **Sequence** field, enter the sequence number.

5.  Select the **Inactive** check box to enable or disable the category.

6.  In the **Label** field, enter a category name.

7.  In the **Value** field, enter a category value.

8.  Enter a value in **Dependent value**, if this category is dependent on any field.

9.  In the **Hint** field, add a hint for this category.

10. Select **Save** to save the category.


## Result

The new category is saved and appears on the Category page.
