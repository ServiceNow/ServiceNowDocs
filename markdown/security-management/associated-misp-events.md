# Associated MISP events

You can use the associated MISP events list view to view the events that have been created manually or automatically in the context of a security incident.

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Security Incident

</td><td>

SIR security incident.Click the security incident to view the security incident.

</td></tr><tr><td>

MISP Event

</td><td>

MISP event ID. Click the MISP event to view the MISP event in the ServiceNow AI Platform.

</td></tr><tr><td>

MISP Source

</td><td>

MISP source where the event is created.Click the MISP source to view the source.

</td></tr><tr><td>

Event Creation Mode

</td><td>

Status of whether the event was created manually or automatically with the event creation profile.Click the MISP event to view the associated MISP event.

</td></tr><tr><td>

Automatic Event Creation Profile

</td><td>

Event creation profile that is used to create the automatic event.Click the profile to view the Automatic MISP Event Creation Profile.
