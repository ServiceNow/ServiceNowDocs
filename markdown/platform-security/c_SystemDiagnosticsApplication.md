# System Diagnostics module

The System Diagnostics application provides logs that relate to the platform.

These logs are available:

-   **Upgrade History**

    Tracks every upgrade to an instance.

-   **Slow Queries**

    Provides insight into how queries affect platform performance. See [Use a slow query log](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/platform-administration/platform-performance/t_UseASlowQueryLog.md).
