# Use Access findings

Use Access findings to view the potential vulnerabilities that are discovered the last time access checks were ran on your ServiceNow AI Platform.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Access Management** &gt; **Access Findings**.

    **Note:** You can also select **Access Findings** from the Access Management Console.

2.  Select the following for more details:

    -   Access findings: The potential vulnerabilities that were discovered the last time access checks were ran.
    -   Access checks: The checks produce findings and are ran every 24 hours.
3.  Select each record on the respective pages to learn more about the vulnerability and detailed resolutions.
