# Data Discovery policy

Use Data Discovery policies to define what sensitive data patterns should be identified and how they should be handled.

The **Policy** tab of Data Discovery reviews current discovery policies. View additional details of a policy by selecting the **View details** button. Edit, delete, and switch the state of current policies by selecting the three dots ![Three dots expandable](../image/dds-dot-icon.png). To create a new policy see [Create new policy](../task/dds-new-policy.md).
