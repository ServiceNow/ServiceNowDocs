# Explore Now Assist for Vault

With the Now Assist for Vault application, you can automate many tasks in Vault. These tasks include securing custom applications to improve your security posture, checking role access for an encrypted column to monitor your instance’s encryption access posture, and others.

## Ask Now Assist in ServiceNow Vault console

The Ask Now Assist area in ServiceNow Vault console lists common security tasks. Selecting a task opens a conversational interface that lets you accomplish the task through prompts. For more information about the Now Assist conversational interface, see [Now Assist panel](https://www.servicenow.com/docs/access?context=now-assist-panel-overview&version=australia&pubname=australia-intelligent-experiences&ft:locale=en-US).
