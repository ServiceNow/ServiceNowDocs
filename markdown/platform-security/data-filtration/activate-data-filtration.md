# Activating data filtration

Learn how to activate data filtration on your instance.

## Before you begin

Role required: security\_admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Plugins**.

2.  Use the search field to find the Data Filtration \(com.glide.data\_filtration\) plugin.

3.  Click **Install**, and then in the Activate Plugin dialog box, click **Activate**.
