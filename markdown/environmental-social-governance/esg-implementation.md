# ESG Management implementation

Use the steps in the ServiceNow® ESG Management application checklist to download the ESG Management from the ServiceNow Store, and get it ready for operation. Mandatory and optional setup steps, as well as an implementation checklist are provided to simplify the setup.

The flow of mandatory steps in the checklist is illustrated here.

![Steps for ESG implementation.](../images/esg_implementation_guide_workflow-png.png "Implementation workflow for ESG Management")

-   **[Install ESG Management](../task/install-esg.md)**  
You can install the ESG Management application \(sn\_esg\) if you have the sys\_admin role. The application includes demo data and installs related ServiceNow® Store applications and plugins if they are not already installed.
-   **[Complete the entitlements for ESG Management](../task/complete-the-entitlements-for-esg-management.md)**  
After you install the ESG Management application from the ServiceNow® Store , you must complete the entitlements to gain complete access to the application and use it.
-   **[Setup checklist for the ESG Management application](../task/setup-checklist-esg.md)**  
This checklist includes the set up tasks that you are required to complete in your ServiceNow AI Platform® instance. When you have completed these tasks, the base system is ready for operation.
