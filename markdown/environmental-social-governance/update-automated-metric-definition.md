# Update metric definition

Modify a metric definition when your data collection requirements change or when you want to associate new goals, targets, entities, or citations with the definition.

## Before you begin

-   A metric definition must exist. For more information on the types of metrics definitions and the procedures to create them, see the following topics:
    -   [Create a manual metric definition](create-manual-metric-definition.md)
    -   [Create an automated metric definition](create-metric-definition.md)
    -   [Create a calculated metric definition](create-composite-metric-definition.md)
-   Role required: sn\_esg.metric\_manager

## Procedure

1.  Navigate to **All** &gt; **Operational Sustainability Management** &gt; **Operational Sustainability Workspace**.

2.  In the sidebar, select the List ![](../../esg-management-reporting/images/icon-lists.png) icon.

3.  Filter the list by selecting a metric definition type from the Metrics tab: Automated metric definitions, Manual metric definitions, or Calculated metric definitions.

4.  Select the metric definition from the list.

5.  Select **Details** in the navigation panel.

6.  Update the fields you want to change.

7.  Select **Save**.

8.  Update associated records.

    1.  Select **Goals**, **Targets**, **Citations**, or **Thresholds** in the navigation panel.

    2.  Select the associated entity.

    3.  Modify the fields you want to change.

    4.  Select **Save**.

    For details regarding the sections and related lists, see [Metric definitions sections and related lists](../concept/metric-definition-sections-and-related-lists.md).

9.  Select **Save**.
