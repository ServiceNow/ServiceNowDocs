# Recalculate emission factor

When an emission factor value is updated, you can initiate the recalculation of the calculated metric definitions to which the emission factor is associated to. This ensures that metrics data remains consistent and accurate, reflecting the latest changes.

## Before you begin

Role required: sn\_esg.program\_manager, sn\_esg.admin

## About this task

For immediate updates, you can manually execute a CMD by opening it and selecting 'Recalculate', it is added to a process queue where a scheduled job automatically recalculates the data.

## Procedure

1.  Navigate to **All** &gt; **Environmental, Social, and Governance** &gt; **ESG Workspace** &gt; **Lists** &gt; **Emission Factor Library** &gt; **Factor**.

2.  Open the emission factor that is updated and for which you want to recalculate the data.

3.  Select **Recalculate**.

    To recalculate data for all calculated metric definitions linked to an emission factor, use the 'Recalculate' button on the emission factor's detail page.
