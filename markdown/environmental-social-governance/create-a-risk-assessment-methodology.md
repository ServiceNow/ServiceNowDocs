# Configure a risk assessment methodology

Configure the risk assessment methodologies \(RAMs\) that are provided by default. A RAM is a configuration or a record in the risk assessment engine.

## Before you begin

Role required: sn\_esg.admin

## About this task

You can use a RAM to assess either the risks or objects in your organization. For example, you can configure a RAM to specify the types of risk assessments and the entities on which a risk assessment is performed. There are two RAMs that are provided by default with the domain area set to **ESG**.

## Procedure

1.  Navigate to **All** &gt; **Environmental, Social, and Governance** &gt; **Risk administration** &gt; **Risk assessment methodologies**.

2.  Select the RAM that you want to modify or configure.

3.  On the form, modify the fields.

    For information on the RAM fields, refer to [Risk Assessment Methodology form](https://www.servicenow.com/docs/access?context=risk-assessment-methodology-form&version=xanadu&pubname=xanadu-governance-risk-compliance&ft:locale=en-US).
