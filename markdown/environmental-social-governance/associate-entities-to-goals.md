# Associate entities to goals

After you create the goals, you must associate those entities, which you want to track, to the goals.

## Before you begin

Role required: sn\_esg.program.manager

## Procedure

1.  Navigate to **All** &gt; **Environmental, Social, and Governance** &gt; **ESG Workspace** &gt; **Lists** &gt; **Program Setup** &gt; **Goals**.

2.  Open the goal for which you want to add the entities.

3.  Select the entities related list.

4.  Select **Add**.

    1.  Select the entities to add.

    2.  Select **Add**.
