# Run the entity onboarding job for SAP Concur

Run the specified scheduled job to get new entities that may have been added in Concur. Running this job ensures that the entity mapping table is updated.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Scheduled Jobs**.

2.  Search for the **SAP Concur entity onboarding** scheduled job.

3.  Select **Execute Now**.
