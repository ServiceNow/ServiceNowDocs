# Corrective action charges form

Corrective action charges form include the expenses related to addressing a non-conformance like labor charges.

<table id="table_ok4_nmg_xgc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Corrective action

</td><td>

Corrective action number that is automatically generated. The number starts with CA.

</td></tr><tr><td>

Type

</td><td>

Corrective action charges type. Options are:-   Labor: **Labor code** is displayed
-   Part: **Part** and **Main part** is displayed
-   Miscellaneous
-   External services

</td></tr><tr><td>

Labor code

</td><td>

Name of the labor code.

</td></tr><tr><td>

Part

</td><td>

Select the part name to be replaced.

</td></tr><tr><td>

Main part

</td><td>

Select the main part to designate the part as main part.

</td></tr><tr><td>

Quantity

</td><td>

Quantity of the item to be replaced or repaired.

</td></tr><tr><td>

Unit of measure

</td><td>

Unit of measure is configurable. Available options are:-   Box
-   Bundle
-   Carton
-   Case
-   Days
-   Each
-   Kit
-   Month
-   Pack
-   Year

</td></tr><tr><td>

Unit price

</td><td>

The unit price on the corrective action charges doesn't include taxes and other costs. **Note:** The unit price is used to determine the base cost for the part \(unit cost \* quantity\). Taxes and other charges are mentioned on the claim.

</td></tr><tr><td>

Impacted asset applicability

</td><td>

List of assets that are impacted.

</td></tr><tr><td>

Description

</td><td>

Brief detail about the corrective actions charges.
