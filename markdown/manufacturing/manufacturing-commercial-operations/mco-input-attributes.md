# Create an MCO input attribute

Create the MCO input attributes for sales promotion campaign.

## Before you begin

Role required: manufacturing operations admin \(sn\_mfg\_cmn.manufacturing\_operations\_admin\)

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Sales Promotion Setup** &gt; **Promotion Questionnaire** &gt; **MCO Input Attributes**.

2.  Select **New**.

3.  Fill in the fields on the [MCO input attributes](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/manufacturing/manufacturing-commercial-operations/mco-input-atrributes_form.md)form, as appropriate.

4.  Select **Save**.
