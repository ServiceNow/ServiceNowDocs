# Partially approve a claim

As an OEM claims agent, partially approve the claims job expenses raised by a dealer.

## Before you begin

Role required: claims agent

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Repair**.

    **Note:** To learn how to create and navigate to the Review &amp; approve form, see [Create repair claim](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/manufacturing/manufacturing-commercial-operations/mco-create-repair-claim.md).

2.  Select **Review &amp; approve** from **Activities**.

3.  From the drop-down list, select **Partially approved**.

    **Note:** You can edit the claim before approving or rejecting.

4.  Process the claim.

    |Action|Description|
    |------|-----------|
    |**Approve the claim**|Select **\[Omitted image "check-outline-24.svg"\] Alt text: Check icon**.|
    |**Reject the claim**|Select **\[Omitted image "close-outline-24.svg"\]**.|
    |**Delete the claim**|Select **\[Omitted image "Form\_DeleteIcon.png"\]**.|
    |**Approve a lesser amount than the requested one**|Enter an amount in **Approved amount**.|
    |**Delete the claim**|Select **\[Omitted image "Form\_DeleteIcon.png"\]**.|

5.  View the repair claim summary details by selecting **Claim Summary**.

6.  Select **Submit**.

7.  Select **Comment** and **Confirm**.
