# Sales promotion management

The Agent workspace enables the OEMs to create the promotion campaign for a specific set of customers.

|Tasks|Description|
|-----|-----------|
|[My promotions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/manufacturing/mco-my-promotions.md)|Displays all the sales promotion claims that are assigned to the person logged in to the workspace.|
|[View all promotions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/manufacturing/mco-all-promotions.md)|Displays all the sales promotion claims.|
