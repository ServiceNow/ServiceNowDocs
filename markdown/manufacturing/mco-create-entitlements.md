# Create an entitlement

Create an entitlement for a Manufacturing Commercial Operations entity, such as an account, consumer, or product.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **MCO Setup** &gt; **Warranty Contracts** &gt; **Entitlements**.

2.  Select **New**.

3.  Fill in the fields on the Entitlement form.

4.  Select **Submit**.
