# Phase task form

Phase task form enables you to add the details of the recall phase.

<table id="table_cjs_shk_wfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

Recall campaign phase task number. This field is automatically generated. The number starts with `TASK`.

</td></tr><tr><td>

Assign to

</td><td>

Name of the person to assign the recall campaign phase task.

</td></tr><tr><td>

Recall campaign phase

</td><td>

Recall campaign phase.

</td></tr><tr><td>

Priority

</td><td>

Priority of the recall phase task. Options are:-   1-Critical
-   2-High
-   3-Moderate
-   4-Low
-   5-Planning

</td></tr><tr><td>

State

</td><td>

State of the campaign task. Options are:-   Pending
-   Open
-   Work in Progress
-   Closed Complete
-   Closed Incomplete
-   Closed Skipped

By default, Open is displayed.

</td></tr><tr><td>

Active

</td><td>

Active or inactive state of the phase task.

</td></tr><tr><td>

Short description

</td><td>

Short description of the phase task.

</td></tr><tr><td>

Description

</td><td>

Detailed description of the phase task.
