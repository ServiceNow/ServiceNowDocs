# Install Service Bridge for Providers and Consumers for Manufacturing Commercial Operations

To set up and configure the Service Exchange for Providers application for Manufacturing Commercial Operations, follow these steps.

**Note:** Service Exchange 2.x.x that is being released with the Australia release does not support migration of the Service Exchange \(Legacy\) versions. If you are using a Service Bridge \(Legacy\) version, before you upgrade to the Australia release, you must follow instructions in the [Service Bridge for Providers \(Legacy\) - Migration Utility \(KB1499823\)](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB1499823) to migrate your configuration data.
