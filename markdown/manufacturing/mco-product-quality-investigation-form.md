# Product quality investigation form

The PQI form enables you to add the investigation details.

<table id="table_v5g_52g_xhc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

Product quality investigation case task number is automatically generated. The number starts with PQI and incremented for every new report.

</td></tr><tr><td>

Issue severity

</td><td>

Choose the severity of the product quality investigation from the options:-   Critical
-   Major
-   Moderate
-   Minor

</td></tr><tr><td>

Issue type

</td><td>

Choose the category of the product quality investigation, which helps classify the nature of the issue for accurate tracking and resolution.-   Design
-   Process
-   Supplier
-   Material
-   Logistics
-   Documentation

</td></tr><tr><td>

Owner

</td><td>

Select the user responsible for the product quality investigation.

</td></tr><tr><td>

Product model

</td><td>

Select the product model.

</td></tr><tr><td>

Problem solving methodology

</td><td>

Select the type of methodology to solve the problem:-   8D: A comprehensive, team-based approach for complex issues \(plan, team, describe, contain, root cause, correct, prevent, recognize\)
-   4D: A phased approach for solving issues \(Describe, contain, correct, close\)
-   Other

</td></tr><tr><td>

State

</td><td>

Choose the state of the PQI.-   Draft
-   New
-   In progress
-   Closed complete
-   Closed canceled
-   Closed incomplete

</td></tr><tr><td>

Priority

</td><td>

Choose the priority of the PQI.

</td></tr><tr><td>

Assignment group

</td><td>

Organizational units or teams responsible for handling specific tasks, cases, or issues.

</td></tr><tr><td>

Assigned to

</td><td>

User to whom the non-conformance case has been assigned.

</td></tr><tr><td>

Total estimated cost

</td><td>

Enter the total estimated cost of the PQI case.

</td></tr><tr><td>

Total actual cost

</td><td>

Enter the total actual cost of the PQI case.

</td></tr><tr><td>

Short description

</td><td>

Enter a short description on the PQI case.

</td></tr><tr><td>

Description

</td><td>

Enter the detailed description of the PQI.

</td></tr><tr><td>

Escape point

</td><td>

Identify the escape point, which is the nearest stage in the process where the root cause could have been detected but was overlooked.

</td></tr><tr><td>

Report summary

</td><td>

Enter the summary of the PQI report.
