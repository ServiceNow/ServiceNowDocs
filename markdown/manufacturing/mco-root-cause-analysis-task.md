# Create a root cause analysis

Create a root cause analysis \(RCA\) to identify the underlying cause of an issue and implement effective, long-term solutions.

## Before you begin

Role required: Quality Issue Management Admin or product non-conformance resolver \(sn\_mfg\_qm.product\_non\_conformance\_case\_resolver\)

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **List** &gt; **Product Non Conformance Case** &gt; **Root Cause Analysis Task**.

2.  Select **New**.

3.  On the Root Cause Analysis Task form, fill in the fields.

    For a description of the field values, see [Root cause analysis task form](../reference/mco-root-cause-analysis-task-form.md).

4.  Select **Save**.

5.  Select **Discuss**.

    The Start a Sidebar discussion window is displayed.

    For more information on the sidebar discussion window, see [Create a Sidebar discussion for a record](https://www.servicenow.com/docs/access?context=sidebar-discussions-record&version=australia&pubname=australia-conversational-interfaces&ft:locale=en-US).

6.  Select **Save**.

Associate a cause to a RCA for the root cause analysis of the product non-conformance.
