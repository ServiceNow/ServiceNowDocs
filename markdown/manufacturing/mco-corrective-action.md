# Create a corrective action

Create a corrective action for the quality investigation case.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **List** &gt; **Product Quality Investigation** &gt; **Corrective Action**.

2.  Select **New**.

3.  On the Corrective Action form, fill in the fields.

    For a description of the field values, see [Corrective action form for PQI](../reference/mco-corrective-action-form-for-pqi.md).

4.  Select **Save**.
