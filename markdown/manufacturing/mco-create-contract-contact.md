# Create a contact

Create a contact, and enter the contact number for that contact.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **MCO Setup** &gt; **Warranty Contracts** &gt; **Contract Contact**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Contract|Contract number list.|
    |Contact|Contact number|

4.  Select **Save**.
