# Claim details form

The Claim details form enables the dealer to enter the claim details for the repair work performed.

<table id="table_bzb_hqf_sfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Asset

</td><td>

Asset or product on which repair action is performed.

</td></tr><tr><td>

Requested by

</td><td>

User who is raising the repair claim request.

</td></tr><tr><td>

Consumer

</td><td>

Customer information that is auto-populated based on the install base.

</td></tr><tr><td>

Description

</td><td>

Short description on the repair claim.

</td></tr><tr><td>

Dealer

</td><td>

Dealer who is raising the repair claim request.

</td></tr><tr><td>

Reported date

</td><td>

Date on which the claim is being raised.

</td></tr><tr><td>

External work order

</td><td>

Claim work order number that is a string value.

</td></tr><tr><td>

Unit of measure

</td><td>

Quantity of assets that are raised for repair claim. Unit of measure is configurable. Available options are:-   Box
-   Bundle
-   Carton
-   Case
-   Days
-   Each
-   Kit
-   Month
-   Pack
-   Year

</td></tr><tr><td>

Product usage

</td><td>

Enter the number of Asset used. It provides insights into how customers use a product.
