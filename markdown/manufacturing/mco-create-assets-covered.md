# Create an asset

Create an asset and Associate it with contacts.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **MCO Setup** &gt; **Warranty Contracts** &gt; **Asset Covered**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Contract|Contract name.|
    |Asset|Asset that is covered under the contract.|
    |Date added|Date when the asset is added to the contract.|
    |Date removed|Date when the asset is removed from the contract.|

4.  Select **Submit**.
