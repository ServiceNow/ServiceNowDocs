# All campaign phases

Agent workspace enables OEMs to create or view all the recall campaign phase claims assigned to logged in to the workspace.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Recall Management** &gt; **All Campaigns** &gt; **All Campaign phases**.

2.  Select **New**.

3.  Create recall campaign phase by referring to.[My campaign phases](mco-rc-my-campaign-phases.md)
