# Create a dealer

As an OEM, create a dealer role.

## Before you begin

Role required: manufacturing operations admin \(sn\_mfg\_cmn.manufacturing\_operations\_admin\)

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **MCO Setup** &gt; **Dealer**.

2.  Select **New**.

3.  On the form, fill in the fields.
