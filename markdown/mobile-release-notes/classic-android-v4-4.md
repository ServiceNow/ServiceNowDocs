# Mobile Classic Android v4.4

The Mobile Classic Android v4.4 release provides problem fixes.

## Download the latest Mobile Classic version

To download the latest release, visit the [Google Play store](https://play.google.com/store/apps/details?id=com.servicenow.servicenow&hl=en).

**Important:** There are several new ServiceNow mobile apps, including Mobile Agent, Now Mobile, Mobile Onboarding, and Now Support. Check out these new apps for the latest features and capabilities.

 PRB1324484

Users with the ServiceNow Classic mobile Android app with SSO are unable to log in

</td><td>

This issue affects the Mobile Classic Android mobile app when connected to an instance with a SSO configuration, the application is terminated, and the user waits for both the session and refresh tokens to expire.

 When the user relaunches the application and attempts to log in, they are presented with the ServiceNow instance logout screen, and the users are unable to continue.
