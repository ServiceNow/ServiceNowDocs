# Mobile Classic mobile app version history

The version history release notes contain details on what's new fixed in each version.

## New mobile apps are available

**Important:** Classic UI is going away on mobile!  In Paris and future releases, the ServiceNow Classic app no longer receives enhancements or non-priority bug fixes. Migrate to ServiceNow® mobile to take advantage of our improved mobile features. For more details, see Migrate from Classic mobile to ServiceNow mobile.

ServiceNow Classic is the legacy mobile app by ServiceNow. There are several new ServiceNow mobile apps, including Mobile Agent, Now Mobile, Mobile Onboarding, and Now Support. Check out these new apps for the latest features and capabilities.

To read the product documentation for the ServiceNow mobile apps and mobile platform, navigate to the [ServiceNow Product Documentation site](https://docs.servicenow.com). Under the **Explore products** section, select **Mobile configuration and navigation**.
