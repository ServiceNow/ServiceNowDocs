# Now Support app version history

The version history release notes contain details on what's fixed in each version.

Now Support gives you control and flexibility to conduct key customer support tasks anywhere and anytime – all from a modern mobile app powered by the ServiceNow AI Platform®. Empowering you as a customer to resolve issues your way.

**Tip:** The mobile version history release notes contain summaries of fixes in each mobile version. For more detailed information about new, changed, and removed features, refer to the [mobile release notes](https://www.servicenow.com/docs/access?context=mobile-rn) for each ServiceNow family release. To read the product documentation for the ServiceNow mobile apps and mobile platform, navigate to the [ServiceNow Product Documentation site](https://docs.servicenow.com). Under the **Explore products** section, select **Mobile configuration and navigation**.
