# ServiceNow Classic for Android v5.0

The Android v5.0 release provides updates for the application.

## New mobile apps are available

ServiceNow Classic is the legacy mobile app by ServiceNow.

**Important:** There are several new ServiceNow mobile apps, including Mobile Agent, Now Mobile, Mobile Onboarding, and Now Support. Check out these new apps for the latest features and capabilities.

## Updated in this release

In Paris and future releases, the ServiceNow Classic app no longer receives enhancements or non-priority bug fixes.

Users upgrading from the London release or earlier can transition from ServiceNow Classic to the new ServiceNow mobile platform. For details on the migration process, see Migrate from Classic mobile to ServiceNow mobile.

Administrators can now prevent users with the ServiceNow Classic app from logging in. For more details, see Disable the ServiceNow Classic mobile app.
