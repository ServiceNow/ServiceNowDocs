In addition to the mobile app updates on the iOS App and Google Play stores, which are recorded here, you can also read the family release notes and product documentation for the ServiceNow mobile apps and mobile platform.

-   To read the mobile release notes for each ServiceNow family release, navigate to the [ServiceNow Product Documentation site](https://docs.servicenow.com). Under the **Release notes and upgrades** section, select your release family.
-   To read the product documentation for the ServiceNow mobile apps and mobile platform, navigate to the [ServiceNow Product Documentation site](https://docs.servicenow.com). Under the **Explore products** section, select **Mobile configuration and navigation**.
