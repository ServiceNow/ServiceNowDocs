# Activate Case Action Status

Activate the Case Action Status plugin to use the case action status feature.

## Before you begin

Role required: admin

## About this task

Case action status provides a Customer Service-specific application of the Action Status Automation feature, which tracks blocking tasks created for customer service cases and updates the action status indicators.

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Plugins**.

2.  Search for the plugin com.snc.csm\_action\_status.

3.  Click **Activate**.
