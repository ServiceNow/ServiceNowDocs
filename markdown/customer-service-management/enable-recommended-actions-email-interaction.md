# Enable recommended actions for an email interaction

Help the agent to find relevant resources based on the context of an email by enabling recommended actions.

## Before you begin

Role required: admin

## About this task

For more information, see [Configuring Recommended Actions](../concept/nba.md).

## Procedure

1.  Navigate to **All** &gt; **Recommended Actions ** &gt; **Contexts**.

2.  Select  **CSM Interaction context**.

3.  Set the **Active** column as true.
