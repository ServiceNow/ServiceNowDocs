# Configure Customer Central properties

Set how many days you want to display data from in Customer Central.

## Before you begin

Role required: admin

## About this task

This property applies to both the Customer Information and Customer Activity views.

## Procedure

1.  Navigate to **All** &gt; **Customer Central** &gt; **Administration** &gt; **Properties**.

2.  Enter the number of days you want to display data from in Customer Central.

    The default is set to 30.
