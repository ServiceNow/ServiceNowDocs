# Configure activity type templates for the Customer Activity view

Create an activity type template to define how to display the activity type data on each of the activity tiles in the activity feed.

Learn more about configuring activity type templates from the following video tutorial.Configure activity type templates

## Before you begin

Role required: admin

## About this task

You can define which fields are displayed, where it is displayed on the tile, and the order of the display.

## Procedure

1.  Navigate to **All** &gt; **Customer Central** &gt; **Feed Configuration** &gt; **Activity Type Templates**.

2.  Select **New**.

3.  Fill out the fields, as required.

    |Field|Description|
    |-----|-----------|
    |Name|Enter a name for the template.|
    |Activity Type|Select the activity type to add the template for.|
    |Icon|Select an icon to use for the activity type.|

4.  Select **Submit**.

5.  Once you have created an activity type template, select it from the Activity Type Template list to configure the fields and order of fields to display.

6.  Select New to create template fields for the activity type.

7.  Fill out the fields, as required.
