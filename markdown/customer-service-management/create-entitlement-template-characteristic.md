# Create a characteristic

Create a characteristic that can be associated with an entitlement or service contract line.

## Before you begin

Role required: sn\_customerservice\_manager

## About this task

A characteristic is used for defining the service coverage that is provided to customers. These characteristics might include the scope of the service, the time period, and any limitations or exclusions such as geographical location and other attributes.

## Procedure

1.  Navigate to **All** &gt; **Product Catalog Management** &gt; **Characteristics** &gt; **Characteristics**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|The name of the characteristic for the entitlement.|
    |Input type|The type of input for the coverage value related to this characteristic. For example, Choice and CheckBox.|
    |Description|The short overview of the entitlement characteristic.|

4.  Select **Submit**.
