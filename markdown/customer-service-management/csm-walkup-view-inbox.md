# View Walk-up Experience inbox

View a personal inbox of automatically assigned Walk-up Experience interactions using the CSM Configurable Workspace feature interface.

## Before you begin

Role required: sn\_csm\_walkup.walkup\_technician

## About this task

There are two ways technicians can view the walk-up inbox. Technicians primarily access CSM Configurable Workspace for all Walk-up Experience related information, including the inbox. However technicians can also access CSM Walk-up Experience in the application navigator, or left navigation bar, to link to and view the inbox in CSM Configurable Workspace.

## Procedure

1.  To view the walk-up inbox, open CSM Configurable Workspace.

2.  Select **Inbox** in the left navigation bar of the interface.

    Automatically pushed interaction assignments appear in the inbox based on agent schedules and assignment rule configurations in the Advanced Work Assignment feature. Technicians can set an inbox status of **Available**, **Away**, and **Offline**.

3.  Alternatively, to view the walk-up inbox via the application navigator, navigate to **CSM Walk-up Experience** &gt; **Technician** &gt; **My Walk-up Inbox**.

    You are linked directly to your personal inbox in CSM Configurable Workspace. Automatically pushed interaction assignments appear in the inbox based on agent schedules and assignment rule configurations in the Advanced Work Assignment feature.
