# Add content types to a permission

Content types for a permission determine the access that users have to certain content in a forum.

## Before you begin

Role required: sn\_communities.forum\_admin

## Procedure

1.  Navigate to **All** &gt; **Community** &gt; **Administration** &gt; **Permissions**.

2.  In the **Permissions** list, click the permission you created.

3.  Click the **Content Type Access** related list.

4.  Click **New**.

5.  Select a content type and an access type.

6.  Click **Submit**.
