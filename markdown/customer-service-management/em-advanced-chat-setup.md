# Chat Setup configuration form

Description of the Chat Setup configuration form fields.

<table id="table_rqd_md4_ndb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the Live Agent chat setup configuration. The default name is `Chat setup`.

</td></tr><tr><td>

Live chat enabled

</td><td>

Option to activate a live chat.

</td></tr><tr><td>

Allow transcript download

</td><td>

Option to enable the user to download a transcript of the conversation.

</td></tr><tr><td>

Show agent names and avatars

</td><td>

Option to display agent names and avatars in the Agent Chat client when a live agent enters, transfers, or exits a chat.

</td></tr><tr><td>

Enable notifications for all users

</td><td>

Option to activate the delivery of notifications.

</td></tr><tr><td>

Agent Whisper

</td><td>

Option to activate Agent Whisper, which enables agents to message supervisors without the requester knowing.For more information, see [Agent Whisper](https://www.servicenow.com/docs/access?context=ci-agent-whisper&version=xanadu&pubname=xanadu-conversational-interfaces&ft:locale=en-US).

</td></tr><tr><td>

Emoji Enabled

</td><td>

Option to activate the ability to include emojis in Agent Chat. For more information, see [Using emojis in Agent Chat](https://www.servicenow.com/docs/access?context=ci-agent-chat-using&version=xanadu&pubname=xanadu-conversational-interfaces&section=ac-using-emojis-agent-chat&ft:locale=en-US).

</td></tr><tr><td>

No Agent Available message

</td><td>

Message that appears when a live agent isn’t available. The default message is `There are no agents available at the moment. Please try again later.`

</td></tr><tr><td>

Application

</td><td>

The application that contains this configuration record. This field is automatically set to Global.

</td></tr><tr><td>

Live chat wait status

</td><td>

Displays the wait time for Live Agent support in the Chat menu if **Wait Time** has been selected.

</td></tr></tbody>
</table>
