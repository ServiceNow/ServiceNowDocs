# Activate Customer Service Management with Request Management

Activate the CSM with Request Management plugin to use the integration with Request Management.

## Before you begin

Instead of doing the configuration through this procedure, consider using guided setup. For more information, see [Integrate with IT Service Management using Guided Setup](configure-csm-sm-integration.md).

Role required: admin

## Procedure

1.  Navigate to **System Definition** &gt; **Plugins**.

2.  Search for the Customer Service with Request Management plugin \(com.sn\_cs\_sm\_request\).

3.  Click **Activate**.
