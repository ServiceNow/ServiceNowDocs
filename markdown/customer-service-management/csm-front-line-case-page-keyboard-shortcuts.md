# Front-line case page keyboard shortcuts

You can use keyboard shortcuts to quickly perform create comments, work notes, and emails when using the Front-line case page in CSM Configurable Workspace.

When composing an email, using the shortcut key enables the user to compose an email in a sub-tab.

When composing a comment or work note:

-   If the user has a work note in progress, using the keyboard shortcut displays the existing work note.
-   If the user has a comment in progress, using the keyboard shortcut displays the existing comment.

|Action|Windows keyboard shortcut|Mac keyboard shortcut|
|------|-------------------------|---------------------|
|Compose an email|Control+Alt+e|Command+Option+e|
|Compose a comment|Shift+Alt+c|Shift+Option+c|
|Compose a work note|Shift+Alt+w|Shift+Option+w|

For more information, see [Next Experience keyboard shortcuts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/platform-user-interface/configure-user-experiences/next-experience-keyboard-shortcuts.md).
