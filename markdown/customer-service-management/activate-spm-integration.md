# Activate Customer Service Management with Service Portfolio Management

Activate the CSM with Service Portfolio Management \(SPM\) plugin to use the integration with Service Portfolio Management.

## Before you begin

Instead of doing the configuration through this procedure, consider using guided setup. For more information, see [Integrate with Service Portfolio Management using Guided Setup](../concept/spm-csm-integration.md).

Role required: admin

## Procedure

1.  Navigate to **System Definition** &gt; **Plugins**.

2.  Search for the plugin sn\_csm\_spm.

3.  Click **Activate**.
