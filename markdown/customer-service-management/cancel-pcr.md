# Cancel post case review

Cancel a post case review that hasn’t been approved, rejected, or no longer requested state.

## Before you begin

Role required: sn\_customerservice\_manager or admin

## Procedure

1.  Navigate to **All** &gt; **Case Digest** &gt; **My Post Case Reviews**.

2.  Select and open a post case review that you want to cancel.

3.  Select **Cancel**.

    **Note:** If the post case review state was awaiting approval, approvers are notified by email that the post case review has been canceled and approval is no longer required.

    The post case review document is now in **Canceled** state.
