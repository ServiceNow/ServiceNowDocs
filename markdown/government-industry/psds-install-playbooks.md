# Install and configure playbooks in Public Sector Digital Services

Install one or more playbook applications available for use with Public Sector Digital Services. You can then configure the features available for submitting requests and routing requests to agents.

The following applications are available with Public Sector Digital Services that enable you to create and use playbooks. For more information on installing and configuring each playbook, see the following corresponding pages.

-   **[Social Benefits Playbook](configuring-social-benefit-playbook.md)**

    Provides an end-to-end workflow for handling applications for social benefit programs.

-   **[License and Permit Playbook](configuring-license-permit-playbook.md)**

    Provides an end-to-end workflow for handling license and permit requests.

-   **[Information Request Playbook](configuring-information-request-playbook.md)**

    Provides an end-to-end workflow for handling public record and information requests.

-   **[Service Request Playbook](configuring-service-request-playbook.md)**

    Provides an end-to-end workflow for handling non-emergency service requests.
