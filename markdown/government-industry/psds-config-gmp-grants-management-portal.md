# Configure the Grants Management Portal

As an admin, you can configure various elements of the Grants Management Portal.

## About this task

You can customize the Grants Management Portal by following these procedures or by using [guided setup for Public Sector Digital Services Core application.](../task/psds-use-gs.md)

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Service Portal Configuration**.

    Some of the components that you can customize include:

    -   **Branding**: [Configure the portal branding](https://www.servicenow.com/docs/access?context=c_BrandingEditor&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US) to give the Grants Management Portal its own look and feel. For example, you can give your portal a name and add your logo.
    -   **Menus**: [Configure the portal header menu](https://www.servicenow.com/docs/access?context=configure-header-menu&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US) to set up the main menu, one of the main navigation controls for your users.
    -   **Homepage**: [Assign a homepage to the portal](https://www.servicenow.com/docs/access?context=assign-portal-homepage&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US).
    -   **Widgets**: Use [Service Portal widgets](https://www.servicenow.com/docs/access?context=service-portal-widgets&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US) to define the content in your portal.
    -   **Roles**: [Configure page security by role](https://www.servicenow.com/docs/access?context=configure-page-security&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US) to either set up pages to be public \(no login required\) or to filter them by user role.
