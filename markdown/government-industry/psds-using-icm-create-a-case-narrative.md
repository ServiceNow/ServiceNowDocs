# Create an investigative case narrative in Investigative Case Management

Add supporting details to the case narrative to organize ongoing updates and investigative context. Information from a police report, witness complaint, or lead may go in the case narrative in order to organize pertinent information into one place.

## Before you begin

Role required: icm.investigator

## Procedure

1.  Navigate to the CSM Configurable Workspace and select **My active cases**.

    Alternatively, you can select a case using the CSM Configurable Workspace Lists menu by selecting **Lists** &gt; **Investigative Cases** &gt; **All** and selecting the case.

2.  Select the case you wish to add narrative details to.

3.  Select the **Narrative** tab.

4.  In the Case Narrative field, enter the narrative details.

    You can refine the text entered in this field using the Now Assist for PSDS Case Narrative refinement skill. For more information on this skill and how to use it, see [Refine an investigative case narrative using the Now Assist for PSDS Investigative case narrative refinement skill](psds-using-na-refine-icm-case-narrative.md).

    ![case narrative refinement skill view](../image/psds-narrative-skill-activated.png)

5.  Select **Save**.


## Result

The case narrative has been created. A log of edits attached to the users name appears in the activity stream, where team members assigned to the case can see a running log of what information was added and when. This is helpful if there are multiple investigators and other personas involved in the case. The case narrative can be edited and added to at any time as new details, leads, and context become available.
