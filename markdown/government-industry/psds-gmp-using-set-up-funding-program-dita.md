# Create a funding program using Grants Management program setup for Public Sector Digital Services

As a grant program manager at a government service agency, use the Public Sector Digital Services Grants Management program setup​ to either create a new funding program, or create one from an existing configuration.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **CSM Configurable Workspace**.

2.  Navigate to **Lists** &gt; **Funding Programs** and choose **New**.

    Here, you have the option to create a new funding program, or start by copying data and configurations from an existing funding program, using the funding program ID.

    -   To create a new funding program without using any pre-existing configurations, choose **New**.
    -   To copy data and configurations from a pre-existing funding program, choose **Create a new program from an existing one**. Choose the existing funding program from the drop-down menu and select **Continue**.

        You can review which fields are copied over to the new funding program record, including default fields. You can configure the fields as necessary.

3.  On the **Details** form, fill in the details of the funding program such as the program information, timeline, and budget.

    |Name|Description|
    |----|-----------|
    |Funding organization name|The name of the agency or institution that provides or owns the funds for the funding program.|
    |Category|The domain area the funding program is intended to support.|
    |Funding program ID|The Funding Program ID is a unique identifier assigned to each funding program to distinguish it from other programs and to support traceability across the grants management lifecycle. Each grant program is associated with a single funding program via the funding program ID.|
    |Funding source ID|A unique identifier that represents the specific source from which the funds originate. This ID helps agencies track the original source of the funds—such as a government body or a research foundation and ties the funding program back to its originating entity.|
    |Owner|The grant program manager or the grant program director.|
    |Name|A descriptive name or a program tile for the funding program.|
    |Timeline|The program start and end dates represent the full duration during which the funding program is active and within which all its associated grant programs must operate.|
    |Budget|Represents the overall amount of money allocated to that program for its entire active duration. All the linked grant programs share this budget.|

4.  Choose **Save**.

    The **Grant programs** tab appears adjacent to the **Details** tab from where you can create grant programs linked to the funding program.

    Choose **new** to create a grant program.

    For more information about creating a grant program, see [Create a grant program using Grants Management program setup for Public Sector Digital Services](psds-gmp-using-set-up-grants-management-program.md).
