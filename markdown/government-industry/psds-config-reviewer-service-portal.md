# Configure Reviewer Service Portal

Configure the Grants Management Reviewer Service Portal for merit reviewers to track, score, and review grants proposals.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Service Portal Configuration**.

    Some of the components that you can customize include:

    -   Branding: [Configure the portal branding](https://www.servicenow.com/docs/access?context=c_BrandingEditor&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US) to give the reviewer service portal its own look and feel. The Coral Theme is available for the reviewer service portal.
    -   Menus: Update the base system menu or add a new one. For details, see [Configure the portal header menu](https://www.servicenow.com/docs/access?context=configure-header-menu&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US).
    -   Home page: Update the default home page by updating the widget configuration or assign a custom home page. For details, see [Assign a homepage to the portal](https://www.servicenow.com/docs/access?context=assign-portal-homepage&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US).
    -   Widgets: Use configurable portal widgets to define the content in your portal. For details, see [Configurable Portal widget library](https://www.servicenow.com/docs/access?context=configurable-portal-widgets-library&version=yokohama&pubname=yokohama-customer-service-management&ft:locale=en-US).
    -   Header: Update the program header configurations according to your agency's needs. For details, see [Configure a custom program record header in the Reviewer Service Portal](psds-config-gmp-rsp-program-header.md).
    -   List pages: Browse, search &amp; filter through records of cases, products, orders, and others. For details, see [Add and configure the Portal Data List widget](https://www.servicenow.com/docs/access?context=config-data-list-widget-inst-optns&version=yokohama&pubname=yokohama-customer-service-management&ft:locale=en-US)
    -   Roles: [Configure page security by role](https://www.servicenow.com/docs/access?context=configure-page-security&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US) to either set up pages to be public \(no login required\) or to filter them by user role.
    -   Knowledge base: Configure a knowledge base to house the merit review framework for users in the portal to view the associated knowledge articles. For details, see [Configure the Merit Review Scoring Framework for a Grant Program in the Reviewer Service Portal](psds-config-gmp-scoring-framework.md).
