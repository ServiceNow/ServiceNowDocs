# Rally spoke release notes

Version history for the Rally spoke integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - May 2023**

    Fix for bringing back all user subscriptions.

-   **Version 1.0.0 - November 2021**

    The Broadcom Rally spoke provides actions to view and analyze meaningful usage data for Broadcom Rally software subscriptions. Analyze usage for your Rally account to determine which licenses are stale so that you can reclaim these licenses and realize your potential savings.
