# Integrations for Manufacturing Commercial Operations release notes

Version history for the ServiceNow® Integrations for Manufacturing Commercial Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.2.0 - June 2026**

    No release notes.

-   **Version 1.1.0 - June 2026 \(Zurich\)**

    No release notes.

-   **Version 1.0.4 - April 2026**

    No release notes.

-   **Version 1.0.3 - March 2026**
    -   Overview
        -   Integrations for Manufacturing Commercial Operations enables automotive and heavy equipment OEMs to digitize their warranty repair claims process on the Now Platform. The application provides a STAR-compliant integration layer that allows dealer networks to submit repair claims and receive service advisories — all through standardized, industry-recognized data formats.
        -   By bridging dealer management systems with ServiceNow's Manufacturing Commercial Operations \(MCO\) module, OEMs gain a centralized, auditable claims lifecycle without requiring dealers to change their existing tools or workflows.
