# Analytics Pack for Contract Management Pro release notes

Version history for the Analytics Pack for Contract Management Pro application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - April 2026**

    Fixed: UI improvements on Contracts Dashboard

-   **Version 1.2.0 - May 2025**

    Fixed: UI issues.

-   **Version 1.1.0 - August 2024**

    Fixed: The contract requests in Draft and Cancelled state were appearing in the Contract request trend report. The filters have been updated to hide the contract requests in Draft and Cancelled state.

-   **Version 1.0.2 - May 2024**

    Gain insight, analyze trends, and drill down into the details of contract operations from the Contracts Dashboard. The application uses ServiceNow Performance Analytics and is included with a full Contract Management Pro license.
