# Virtual Agent for Sourcing and Procurement Operations release notes

Version history for the Virtual Agent for Sourcing and Procurement Operations on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.8.0 - May 2025**

    New: Provides a virtual agent topic to create finance cases for customers using Finance Case Management.

-   **Version 3.6.8 - August 2024**

    Minor fixes.

-   **Version 3.6.6 - May 2024**

    Minor fixes.

-   **Version 3.6.0 - February 2024**

    Minor fixes.

-   **Version 3.1.0 - November 2023**

    New: Cost center managers can now provide their approval decision with the assistance of virtual agent, with updated approval content that displays the allocated amount that the cost center manager needs to approve. Note: Existing and new customers would only see these new changes if they have defined the new approval rules for cost center manager approval.

-   **Version 3.0.0 - August 2023**

    Minor fixes.

-   **Version 2.4.4 - May 2023**

    Changed: Renamed this application to Virtual Agent for Sourcing and Procurement Operations. This application was formerly named Virtual Agent Experience Pack for Procurement Service Management.

-   **Version 2.4.2 - February 2023**
    -   Changed: Modified the Virtual Agent topic to re-direct shoppers to the new Shopping Hub configurable portal.
        -   Note: If you are a new customer, the Shopping Hub configurable portal will be your default experience. If you are an existing customer and want to use this new portal, you will need to run a fixed script to update existing links to be redirected to the new Shopping Hub configurable portal.
-   **Version 2.3.4 - November 2022**

    Minor fixes.

-   **Version 2.3.1 - August 2022**

    Minor fixes.

-   **Version 2.2.1 - May 2022**

    Minor fixes.

-   **Version 2.1.0 - February 2022**

    A conversational bot platform for providing user assistance through conversations within a messaging interface for ShoppingHub.
