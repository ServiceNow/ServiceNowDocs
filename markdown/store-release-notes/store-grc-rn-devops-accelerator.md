# GRC: DevOps Accelerator release notes

Version history for the GRC: DevOps Accelerator on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 16.0.0 - February 2023**

    Fixed: Reduction in installation size of the application.

-   **Version - August 2022**

    The DevOps Accelerator integrates IRM Policy &amp; Compliance and DevOps to embed governance early in the app development lifecycle and automate processes to eliminate the delays caused by manual activities, while also mitigating unnecessary compliance and technology risk. All that is required is for either a GRC admin to assign control objectives to PaCE policies or a PaCE admin to assign PaCE policies to GRC control objectives. Then the automated preventative controls within the DevOps Accelerator can be used to continuously monitor for compliance and evidence collection without developer intervention. Integration of governance into the development process also enables exception handling so developers can meet compliance obligations and support audit requirements without sacrificing productivity and slowing innovation.
