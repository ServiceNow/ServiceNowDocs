# External Agent Management Util Pack release notes

Version history for the External Agent Management Util Pack application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - August 2025**

    Bug fixes.

-   **Version 1.1.0 - August 2024**

    Changed: Added domain column in agent mapping table for use by domain separated apps.

-   **Version 1.0.0 - February 2024**

    External Agent Management Util Pack allows Amazon Connect integration to sync Agent ID between Amazon Connect and ServiceNow.
