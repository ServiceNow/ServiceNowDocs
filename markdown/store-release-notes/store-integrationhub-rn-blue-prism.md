# Blue Prism spoke release notes

Version history for the Integration Hub Blue Prism spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.2 - September 2022**

    Patch version of Blue Prism Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.1 - August 2020**

    Fixed: This version includes a fix to enable actions to work in Paris.

-   **Version 1.0.0 - May 2020**

    Provides actions to automate the management of Jobs and Processes for Blue Prism.
