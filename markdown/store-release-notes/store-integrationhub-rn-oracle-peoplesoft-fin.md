# Oracle Peoplesoft Financial spoke release notes

Version history for the Oracle Peoplesoft Financial spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - June 2026**

    Fixed: Streamline ACLs

-   **Version 1.1.0 - March 2024**

    Added authentication template for one-click connectivity setup.

-   **Version 1.0.7 - September 2022**

    Patch release of Oracle Peoplesoft Financial Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.5 - December 2021**

    Fixed: Patch release of Oracle Peoplesoft Financial Spoke for IntegrationHub. This version adds a fix to duplicated content in the GLIDE-INF folder and includes security-related changes.

-   **Version 1.0.3 - March 2021**

    Patch release of Oracle Peoplesoft Financials Spoke.

-   **Version 1.0.1 - January 2021**
    -   The Oracle Peoplesoft Financial spoke provides a list of actions that interact with PeopleSoft SOAP APIs. It provides the foundation to synchronize ServiceNow and PeopleSoft, such as create PO, update PO, create a journal entry, create invoices, create goods receipts, and so on.
    -   Given that PeopleSoft is a highly customizable system, Oracle Peoplesoft Financial spoke also showcases the design pattern that can support more PeopleSoft APIs beyond what are out of the box.
