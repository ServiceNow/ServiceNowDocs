# Career Assessment release notes

Version history for the Career Assessment Application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.3.0 - June 2026**

    Added Non-Glide Cobalt Raven ACLs for all the Exam tables.

-   **Version 3.2.0 - December 2025**

    Changed: Read only directive is uptaken and marked all the read only fields' 'strict read only' attribute to true.

-   **Version 3.1.0 - May 2025**

    New: Coral theme support.

-   **Version 3.0.4 - February 2025**

    Fixed accessibility issues.

-   **Version 3.0.1 - January 2024**

    Fixed: Security issues.

-   **Version 3.0.0 - September 2023**

    The Career Assessment application works as an exam engine that improves the learning experience of users in Employee Center. Learning administrators can create exams and learning paths usingCareer Assessmentto create customized learning journeys for users that help with career advancement and help test a learner's knowledge of a subject.
