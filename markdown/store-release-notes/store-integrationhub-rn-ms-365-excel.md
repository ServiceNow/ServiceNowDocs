# Microsoft 365 Excel spoke release notes

Version history for the Microsoft 365 Excel Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - July 2023**

    Fix to handle drive-id that has more than 40 characters.

-   **Version 1.0.1 - December 2020**

    Patch release of Microsoft 365 Excel spoke for IntegrationHub.

-   **Version 1.0.0 - September 2020**

    New: Provides actions to automate worksheet, table, and cell management in Microsoft 365 Excel.
