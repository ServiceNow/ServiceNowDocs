# Service Graph Connector for SolarWinds release notes

Version history for the ServiceNow AI Platform capabilities Service Graph Connector for SolarWinds on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.6.0 - August 2025**

    New: Onboarded the connector to SGC Central.

-   **Version 2.5.0 - March 2025**

    New: Added support for air gap SolarWinds system.

-   **Version 2.4.1 - December 2022**

    Fixed security issue in the guided setup.

-   **Version 2.3.0 - May 2022**
    -   New:
        -   Add Multi-instance support
        -   Add Cleanse rule for "NONE" in cmdb\_serial\_numbers
-   **Version 2.2.1 - February 2021**

    New: Added statement in guided setup for customer to acknowledge they are aware of recent SolarWinds security advisory. This statement must be acknowledged through guided setup for both new installs and upgrades.

-   **Version 2.1.7 - September 2020**

    This is a certified integration application that enables asset inventory data \(hardware and software\) from SolarWinds to be populated in the CMDB. It enables a customer to ingest data from an external source into the ServiceNow CMDB.
