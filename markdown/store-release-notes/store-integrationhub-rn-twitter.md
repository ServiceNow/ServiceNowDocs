# X spoke release notes

Version history for the Integration Hub X spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.4.0 - June 2026**

    Fixed: Streamline ACLs.

-   **Version 2.3.0 - September 2025**

    Added required ACLs and Roles as per the AI Security Directive guidelines.

-   **Version 2.2.2 - July 2025**

    Fixed: Twitter Spoke AI Agents - March Release - Agents Renaming and Output transformation strategy set to None.

-   **Version 2.2.0 - March 2025**

    Minor release with new AI agents.

-   **Version 2.1.0 - October 2024**

    Spoke name changed from "Twitter Spoke" to "X Spoke".

-   **Version 1.2.2 - September 2023**

    Fixed: The license requirements.

-   **Version 1.2.1 - August 2023**

    Fixed the Tweet Message action.

-   **Version 1.2.0 - April 2023**
    -   New: Authentication template for easier spoke setup
    -   Fixed:
        -   Improve installation performance of spoke
        -   KMF error in Tweet action
-   **Version 1.1.2 - October 2021**

    Fixed: Security bug.

-   **Version 1.1.1 - July 2021**

    Patch release of Twitter spoke for IntegrationHub. This version adds KMF modules for additional security of the password2 fields and Rome compatibility.

-   **Version 1.0.1 - October 2019**

    Provides actions to automate posting messages and media to a corporate Twitter news feed.
