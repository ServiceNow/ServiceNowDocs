# Amazon Alexa spoke release notes

Version history for the Integration Hub Amazon Alexa spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 1.2.0 - February 2026**

    New: Added Authentication Template for Amazon Alexa Spoke.

-   **Version 1.1.0 - November 2024**

    Fixed: Sandbox access for client.

-   **Version 1.0.3 - September 2023**

    Fixed: The license requirement.

-   **Version 1.0.2 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.1 - August 2021**

    This is a patch release for Amazon Alexa Spoke. This version includes a fix at execution of Update Interaction Model action and documentation for Alexa Security configuration.

-   **Version 1.0.0 - November 2020**

    Provides actions to automate the management of Amazon Alexa Skills.
