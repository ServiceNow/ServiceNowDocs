# Universal Request integration with Microsoft Teams release notes

Version history for the Universal Request integration with Microsoft Teams on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.1 - December 2021**

    Fixed: Added the keywords "Request Help", "Create Universal Request", and "Employee Help Request" in the keywords field for the "Employee Help Request \(Teams Template\)" topic.

-   **Version 1.0.0 - September 2021**

    The ServiceNow® Universal Request integration with Microsoft Teams application extends the Now Virtual Agent integration with Microsoft Teams to enable employees to more effectively request and receive service from within Microsoft Teams.
