# Automation Anywhere spoke release notes

Version history for the Integration Hub Automation Anywhere spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 1.2.1 - May 2025**

    Fixed: Auto refresh of token.

-   **Version 1.2.0 - March 2025**

    Added 2 new actions and support v2 Automation Anywhere API.

-   **Version 1.1.2 - September 2022**

    Patch release of Automation Anywhere Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.1.1 - October 2021**

    Fixed: Security bug.

-   **Version 1.0.23 - July 2021**

    Patch release of the Automation Anywhere spoke for IntegrationHub. This version adds KMF modules for additional security of the password2 fields and Rome compatibility.

-   **Version 1.0.2 - April 2021**

    Patch release of the Automation Anywhere spoke for IntegrationHub.

-   **Version 1.0.1 - August 2020**

    Provides actions to automate the management of bots and devices in Automation Anywhere.
