# Knowledge Graph release notes

Version history for the Knowledge Graph application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.2 - March 2025**
    -   The Knowledge Graph application enables organizations to transform fragmented enterprise data into a connected, intelligent framework, enhancing the capabilities of Now Assist in Virtual Agent, AI agents, and generative AI skills across ServiceNow. By seamlessly integrating structured and unstructured data including ServiceNow records, knowledge bases, and external sources
    -   Knowledge Graph delivers context-aware insights and personalized recommendations.
    -   With a dedicated, no-code UI, administrators can effortlessly design and manage knowledge graph schemas, including configuring nodes \(tables\), properties \(columns\), and relationships. The application streamlines the entire process, from schema creation and data ingestion to performance monitoring and results analysis, ensuring a scalable, flexible, and intuitive approach to knowledge modeling.
    -   By unifying knowledge across platforms and integrating with ServiceNow generative AI applications, the out-of-the-box integrations of the Knowledge Graph will drive productivity, improve decision-making, and unlock the full potential of enterprise data while maintaining robust data governance and permission controls.
