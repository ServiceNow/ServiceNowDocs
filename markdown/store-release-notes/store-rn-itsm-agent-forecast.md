# Agent Forecast release notes

Version history for the Agent Forecast application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.7.2 - June 2026**

    Fixed minor defects as part of this release

-   **Version 5.7.1 - March 2026**

    Fixed minor defects as part of this release.

-   **Version 5.7.0 - December 2025**

    Minor bug fixes.

-   **Version 5.6.3 - June 2025**

    Fixed: Minor issues.

-   **Version 5.6.2 - May 2025**

    Fixed: Minor issues.

-   **Version 5.6.0 - November 2024**
    -   Fixed:
        -   Minor issues
        -   Security improvements
-   **Version 5.4.0 - May 2024**
    -   Minor fixes
    -   Addressed accessibility issues
-   **Version 5.2.0 - November 2023**

    Minor fixes.

-   **Version 5.1.0 - August 2023**

    Fixed: Minor fixes.

-   **Version 5.0.0 - May 2023**
    -   Changed:
        -   Create volume forecast at daily interval up to a year \(52 weeks\).
        -   Option to create generate volume forecast at 15, 30, and 60 minute intervals.
-   **Version 4.2.0 - February 2023**
    -   Major fixes
    -   Changed: Default algorithm set to Seasonal Trend Loess \(STL\)
-   **Version 3.0.2 - March 2022**

    New: Initial release for San Diego

-   **Version 3.0.0 - February 2022**

    Demand forecasting for workforce optimization
