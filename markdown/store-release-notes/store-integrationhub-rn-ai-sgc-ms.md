# AI Service Graph Connector for Microsoft release notes

Version history for the ServiceNow® AI Service Graph Connector for Microsoft application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.1.0 - June 2026**

    New: Released Azure Foundry Tenant Wide Discovery support

-   **Version 2.0.0 - March 2026**

    This integration connects AI Control Tower’s AI Discovery capabilities with Azure Foundry\(New and old\) and Copilot studio, enabling automated discovery and governance of AI assets across enterprise Microsoft environment
