# Now Assist for Software Asset Management \(SAM\) release notes

Version history for the Now Assist for Software Asset Management \(SAM\) application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 8.0.1 - June 2026**

    New feature: Entitlement creation error handling - This feature uses AI to suggest resolutions when users upload the standard Excel import template for entitlement creation.

-   **Version 4.4.10 - May 2026 \(Yokohama\)**
    -   Resolve SaaS connection errors - This feature enables diagnosing and resolving connection errors occurring in SaaS integrations within ServiceNow SAM, either at the validate connection step during initial setup or during the normal integration flow. Investigate root causes such as authentication failures, API endpoint changes, or credential expiry. It helps in providing actionable error details at the point of failure
        -   whether during setup or live sync
        -   reducing time spent on manual troubleshooting.
    -   Contract entitlement data extraction - Reduce manual effort by leveraging AI to automatically create entitlements in SAM by extracting licensing information from software contracts.
-   **Version 7.0.0 - April 2026 \(Zurich &amp; Australia\)**
    -   New features:
        -   Resolve SaaS connection errors - This feature enables diagnosing and resolving connection errors occurring in SaaS integrations within ServiceNow SAM, either at the validate connection step during initial setup or during the normal integration flow. Investigate root causes such as authentication failures, API endpoint changes, or credential expiry. It helps in providing actionable error details at the point of failure — whether during setup or live sync — reducing time spent on manual troubleshooting.
        -   Contract entitlement data extraction - Reduce manual effort by leveraging AI to automatically create entitlements in SAM by extracting licensing information from software contracts.
-   **Version 6.1.1 - March 2026 \(Australia\)**

    New features: Automate user resolution with AI for SaaS license management to increase efficiency and accuracy in subscription management.

-   **Version 5.3.3 - March 2026 \(Zurich\)**
    -   New:
        -   Microsoft 365 license assignment AI Agent:
            -   The M365 Software License Assignment Agent enables end-to-end automated license fulfillment for Microsoft 365 requests submitted through the ServiceNow Service Catalog.
            -   When users request a license in the ServiceNow catalog, the AI agent evaluates the request, verifies license availability, and automatically assigns the appropriate license to the requesting user on the Microsoft 365 Admin Center with no manual intervention. This improves fulfillment speed, reduces manual effort, and helps maintain accuracy and compliance.
-   **Version 4.4.5 - January 2026 \(Yokohama\)**

    New features: Automate user resolution with AI for SaaS license management to increase efficiency and accuracy in subscription management.

-   **Version 5.2.4 - December 2025 \(Zurich\)**

    New features: Automate user resolution with AI for SaaS license management to increase efficiency and accuracy in subscription management.

-   **Version 5.1.0 - October 2025 \(Zurich\)**

    Performance improvements and bug fixes.

-   **Version 4.1.0 - October 2025 \(Yokohama\)**

    Performance improvements and bug fixes.

-   **Version 5.0.0 - September 2025 \(Zurich\)**
    -   New features:
        -   Accelerate software fulfillment as AI automates entitlement allocation and purchase order creation.
        -   Expand reclamation coverage as AI recommends tailored rules to reduce unused software costs.
        -   Gain confidence in software removal as AI evaluates the right candidates for reclaiming licenses.
-   **Version 4.0.0 - September 2025 \(Yokohama\)**
    -   New features:
        -   Accelerate software fulfillment as AI automates entitlement allocation and purchase order creation.
        -   Expand reclamation coverage as AI recommends tailored rules to reduce unused software costs.
        -   Gain confidence in software removal as AI evaluates the right candidates for reclaiming licenses.
-   **Version 3.3.0 - August 2025 \(Zurich\)**
    -   Zurich compatibility for:
        -   Publisher compliance summarization
        -   Product compliance summarization
        -   Recommended actions
-   **Version 2.3.0 - August 2025 \(Yokohama\)**

    New third-party Al model provider options available for Now Assist: Google Gemini 2.0 Flash, Google Gemini 2.5 Pro, and AWS Claude 3.7 Sonnet are available for Now Assist skills and Al agents in addition to Now LLM Service and Azure OpenAl GPT-4.1 and GPT-4.1 mini.

-   **Version 2.0.0 - May 2025**
    -   New:
        -   Use agentic workﬂow to automate software asset sourcing to improve operational efficiency
        -   Gain insights into product compliance and optimization with AI-powered product summaries

    -   Yokohama Early Availability: Gain insights into your publisher license compliance by using Now Assist for SAM
-   **Version 1.0.0 - February 2025**

    With the Now Assist for Software Asset Management \(SAM\) application, SAM managers can use generative AI to summarize important publisher information and gain insights on the software estate and license position, making it easier to manage compliance with publisher contracts.
