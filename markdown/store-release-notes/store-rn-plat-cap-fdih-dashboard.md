# FDIH Dashboard release notes

Version history for the FDIH Dashboard application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 25.0.5 - November 2024**

    This is a patch release with one defect fix.

-   **Version 25.0.3 - May 2024**

    A few problem fixes.

-   **Version 25.0.0 - February 2024**
    -   Added below 2 widgets.
    -   Top 10 max Flow Designer Runtimes \(ms\) last 14 days
    -   Top 10 avg Flow Designer Runtimes \(ms\) last 14 days
-   **Version 23.0.10 - March 2023**

    Fixed: Flow Usage header background is now dark in Dark theme.

-   **Version 23.0.8 - February 2023**

    Operations Dashboard for usage tracking and debugging of Flow Designer and Integration Hub.
