# AI Control Tower Core release notes

Version history for the AI Control Tower Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.4 - May 2025**
    -   New:
        -   Enhancements to landing page dashboards for AI Steward
        -   Support for managing Agentic AI assets in AI inventory
        -   Multi-instance implementation to sync preferences and AI asset inventory across sub-prod and prod instances
        -   Enabled controls for blocking approval workflows for Now Assist \[custom skills\]
        -   Provision for opt-in/opt-out for Data overflow processing in event of traffic spikes
    -   Changed: Consolidated list view of AI asset Inventory and Approvals
