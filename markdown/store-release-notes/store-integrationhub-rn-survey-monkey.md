# SurveyMonkey spoke release notes

Version history for the SurveyMonkey spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.6 - August 2024**
    -   Fixed:
        -   Look up Team Members Stream - Added new output provided by vendor
        -   Look up Users - Rate limits
-   **Version 2.0.5 - December 2022**

    Patch release of SurveyMonkey Spoke for IntegrationHub. This version includes localization fixes.

-   **Version 2.0.4 - October 2022**

    Fixed: Patch release of SurveyMonkey Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 2.0.3 - November 2021**

    Fixed: Patch release of the SurveyMonkey spoke for IntegrationHub. This version fixes errors observed at data stream actions \(for example, Look up Surveys\).

-   **Version 2.0.2 - July 2021**

    Patch release of SurveyMonkey Spoke for IntegrationHub. This version fixes an issue to fetch collectors from the survey.

-   **Version 2.0.0 - May 2021**
    -   Minor release for the SurveyMonkey Spoke to include new actions to:
        -   Download subscriptions and
        -   Update user activity
-   **Version 1.0.0 - December 2020**
    -   New:
        -   Collector Management
            -   Create Collector
            -   Delete Collector
            -   Get Collector Details
        -   Contact Management
            -   Add Contacts To Contact List
            -   Create Contact List
            -   Delete Contact List
            -   Look up Contact Lists
            -   Look up Contacts in Contact List
            -   Update Contact List
        -   Folder Management
            -   Create Folder
            -   Look up Folders
        -   Response Management
            -   Look up Survey Responses
            -   Look up Survey Trends
        -   Survey and Question Management
            -   Create Question
            -   Create Survey
            -   Delete Question
            -   Delete Survey
            -   Look up Questions
            -   Get Survey Details
            -   Look up Survey Pages
            -   Look up Surveys
            -   Update Question Update Survey
        -   User Management
            -   Get User Details
