# Twilio spoke release notes

Version history for the Integration Hub Twilio spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - February 2023**

    New: Authentication template for spoke setup.

-   **Version 1.1.3 - September 2022**

    Patch version of Twilio Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.1.2 - September 2020**

    Patch release of Twilio spoke for IntegrationHub.

-   **Version 1.0.0 - November 2019**
    -   Automate account, call, messaging service, notification service, short code, and text messaging management of your Twilio instance from the ServiceNow AI Platform.
    -   Create a new account
