# Jamf spoke release notes

Version history for the Jamf spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.0 - June 2026**

    Non Glide Cobalt Raven ACLs added.

-   **Version 1.4.0 - April 2026**

    Added 3 new external triggers.

-   **Version 1.3.2 - March 2026**

    Fixed: \[Security Bug\] Missing OOB 'report\_view' ACLs for multiple tables

-   **Version 1.3.1 - January 2026**

    New: Added spoke actions and subflows for CSD agents.

-   **Version 1.2.1 - December 2025**

    New: Added Spoke Actions required for CSD agents.

-   **Version 1.1.0 - November 2025**

    New: Support revocation of installed softwares in SAM table.

-   **Version 1.0.8 - July 2024**

    Fixed: Added bearer token authentication support for classic APIs.

-   **Version 1.0.7 - May 2024**

    Security fix.

-   **Version 1.0.6 - September 2023**

    Fixed: The license requirement.

-   **Version 1.0.5 - April 2023**

    Patch release of Jamf Spoke for Integration Hub. This version includes deduplication changes.

-   **Version 1.0.4 - June 2022**

    Removed: Patch release of Jamf Spoke. This version removes invalidated properties from plugin.xml.
