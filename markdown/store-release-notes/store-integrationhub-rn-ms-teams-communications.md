# Microsoft Teams Communications spoke release notes

Version history for the Integration Hub Microsoft Teams Communications spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.1 - March 2026**

    Fixed - Action: Look up Meeting Recording Content

-   **Version 1.5.0 - December 2025**

    New Spoke Actions: Look up meeting data, Look up transcripts, Look up transcripts data, Create Meeting Event, Create Notification Subscription, Look up Meeting Recording Content

-   **Version 1.4.1 - November 2024**

    Fixed: Issue related to autoCallRecording for Create Meeting action.

-   **Version 1.4.0 - September 2024**
    -   Changed: Action: Create Online Meeting - Accepts additional parameter to allow for automatic recording of the meeting
    -   Fixed: Action: Get User Details
-   **Version 1.3.2 - March 2024**

    Fixed: Start Group Call action to allow passing in Bot Name.

-   **Version 1.3.1 - September 2023**

    Fixed: Look up User IDs Stream action.

-   **Version 1.3.0 - May 2023**
    -   New: Look up User IDs Stream action
    -   Changed: Create User Online Meeting action
-   **Version 1.2.4 - January 2023**

    Fixed: Error that occurs when you start a teams conference call.

-   **Version 1.2.3 - December 2022**

    New: Localization framework support.

-   **Version 1.2.2 - November 2022**

    Patch version of Microsoft Teams Communications Spoke for IntegrationHub. This version adds a fix to enable meeting presenter and chat mode options.

-   **Version 1.2.0 - February 2022**

    New: Minor release of Microsoft Teams Communications Spoke for IntegrationHub. This version adds support for starting conferences with different lobby bypass settings and surfacing audio join information.

-   **Version 1.1.3 - November 2021**

    Fixed: Patch release of Microsoft Teams Communications spoke for IntegrationHub. This version fixes meeting extensibility issues.

-   **Version 1.1.2 - September 2021**
    -   Fixed:
        -   Microsoft Teams Meeting Extensibility, which now relies on a newer API for creating Online Meetings.
        -   Uptake of new API recommended by Microsoft for creating online Meetings with support for Application Access policy.
-   **Version 1.1.1 - June 2021**

    Patch release of the Microsoft Teams Communications spoke for IntegrationHub. This version includes a fix to an error observed in Flow Designer while using Add Tab to Meeting.

-   **Version 1.1.0 - February 2021**

    Minor release of Microsoft Teams Communications spoke for IntegrationHub. This version includes changes to support Servicenow's App - Meeting Extensions for Microsoft Teams.

-   **Version 1.0.0 - October 2020**

    Provides actions to automate the management of group calls and online meetings.
