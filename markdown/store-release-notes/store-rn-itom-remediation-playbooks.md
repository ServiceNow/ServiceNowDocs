# Remediation Playbooks release notes

Version history for the Remediation Playbooks application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - November 2022**

    Minor product fixes.

-   **Version 1.0.0 - February 2022**

    This component helps customers to monitor and remediate basic installed app issues. The playbook that we are launching has content that helps customers to identify ZCC \(ZScaler Agent\) error state and remediate with ACC \(ServiceNow Agent Client Collector\).
