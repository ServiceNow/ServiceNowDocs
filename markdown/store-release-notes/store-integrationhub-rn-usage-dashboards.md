# Integration Hub Usage Dashboards release notes

Version history for the Integration Hub Usage Dashboards application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.1.0 - June 2026**

    Fixed: Streamline ACLs.

-   **Version 3.0.0 - August 2025**

    New: Added transaction information for External Content Connectors and Orchestration under the "Transactions by Type" chart.

-   **Version 2.1.0 - May 2025**

    Enable users to explore detailed information from the Data Egress charts, view the Transaction Log, and Export History data.

-   **Version 2.0.3 - December 2024**

    Backend fixes.

-   **Version 2.0.1 - August 2024**

    Integration Hub Usage Dashboard enables the customer to have enhanced visibility of Integration Hub transaction usage data and view egress usage from inbound API requests and URL exports.

-   **Version 1.1.1 - May 2024**

    Integration Hub Usage Dashboard enables the customer to have increased visibility of the transaction usage data and view the egress usage from the inbound REST and SOAP requests.

-   **Version 1.0.7 - February 2024**

    Integration Hub Usage Dashboard enables the customer to have improved visibility of the transaction usage data. The reference to the legacy dashboard has been removed.

-   **Version 1.0.3 - December 2023**

    Support for Subscription Management v2.

-   **Version 1.0.2 - March 2023**

    Fixed: Fixed an issue on the chart for the Custom Spoke Transaction Usage.

-   **Version 1.0.1 - November 2022**

    This dashboard shows an overview of Integration Hub usage metrics to show customers how their organization uses Integration Hub and to support license compliance.
