# Resizable panes component release notes

Version history for the Resizable panes component application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 29.0.2 - June 2026**

    Fixed various defects

-   **Version 29.0.1 - March 2026**

    New: Left &amp; right with split-stack mode that lets users to toggle and stack left and right panes

-   **Version 28.0.3 - December 2025**
    -   New: Ability to activate resize by double-click on handle in addition to dragging.
    -   Fixed: Resizable panes don't go back to previous position when coming out of stack mode \(PRB1903147\).
-   **Version 27.1.0 - August 2025**

    Fixed various accessibility and cosmetic issues.

-   **Version 27.0.9 - May 2025**
    -   New:
        -   Keyboard shortcuts to expand/collapse resizable panes
        -   New UIB properties to support responsive authoring
    -   Fixed: Multiple bug fixes
-   **Version 27.0.5 - February 2025**

    Displays two content slots, one on the left and one on the right, that users can resize horizontally or vertically. Optionally, the component can display an action bar that enables users to set which panes display.
