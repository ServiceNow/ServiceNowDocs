# Care Team Work Management release notes

Version history for the Care Team Work Management on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.0 - June 2026**
    -   This release delivers platform-wide accessibility \(WCAG 2.2 AA, 400% zoom/reflow\), security-directive access-control hardening, and subscription-alignment role corrections for Care Team Work Management.
        -   Security and Compliance
            -   Automated WCAG 2.2 AA accessibility testing: Enabled across all Industry Verticals applications in this bundle.
            -   WCAG 2.2 AA compliance and 400% zoom/reflow support: Delivered across Care Team Operations forms, lists, and agent workspaces.
            -   Non-Glide access control hardening: Audited and merged non-Glide ACL changes across Care Team Operations.
            -   Subscription alignment — administrator role inheritance: Care Team Work Management administrator role inheritance corrected to align with subscription entitlement boundaries.
            -   Care Team accessibility checkers: Added WCAG 2.2 AA accessibility checkers to Care Team Operations.
-   **Version 1.4.0 - March 2026**
    -   New features:
        -   Healthcare Operations Orchestration: Enables operational leaders and care team managers to coordinate complex, multi-department healthcare operations through orchestration cases, task plan templates, and playbook-driven workflows integrated with Planned Work Management \(PWM\).
        -   Task Plan Template Integration: Care team managers can now create task plan templates with configurable playbook activities. The task plan configuration supports both rounding and care team activity types, with the correct templates auto-populating based on playbook selection.
        -   Operational Leader Read Access: New CTWM Viewer role grants operational leaders read access to all care team cases and tasks with both "Allow if" and "Deny unless" ACLs.
    -   Fixed: Care Team Manager &amp; Agent Task Plan Access: Fixed an issue where Care Team Managers and Agents were unable to access sn\_cto\_case Task Plan Templates.
    -   Security &amp; Compliance: CTWM Workspace Defect Fix: Fixed workspace landing page, care team case/task form view, and list view issues.
-   **Version 1.1.0 - December 2025**
    -   New:
        -   A new landing page user experience has been added for Care Team Agents and Care Team Agent Managers.
        -   Smart assessments can now be completed in the workspace as part of the care team task plan process.
-   **Version 1.0.4 - August 2025**

    The ServiceNow Care Team Work Management application is built on the Healthcare Operations Core platform and enables hospital care teams such as nurses, unit coordinators, and department leads to create, assign, and track structured work plans that support operational workflows across care teams. Care teams can monitor the progress of work plan requests and tasks through the CSM/FSM Configurable Workspace or the Care Team Portal, ensuring visibility and accountability from initiation to completion.
