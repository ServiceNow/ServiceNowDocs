# Now Assist for Sales Customer Relationship Management release notes

Version history for the Now Assist for Sales Customer Relationship Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.3 - June 2026**

    New dependant plugin - Opportunity Management AI Features added.

-   **Version 1.1.1 - April 2026**

    Maintenance release. Contains internal code updates with no impact to existing functionality or user-facing behavior.

-   **Version 1.1.0 - March 2026**

    Name of application is changed from "Now Assist for Sales and Order Management \(SOM\)" to "Now Assist for Sales Force Automation \(SFA\)".

-   **Version 1.0.7 - March 2026**

    Name of application is changed from "Now Assist for Sales and Order Management \(SOM\)" to "Now Assist for SFA".

-   **Version 1.0.6 - January 2026**

    Changed: Making it compatible with latest version of Platform and SDR AI Agent.

-   **Version 1.0.5 - October 2025**

    Extended the support for GPT-OSS model

-   **Version 1.0.4 - September 2025**

    New: Support for Virtual SDR AI agent

-   **Version 1.0.1 - May 2025**

    Now Assist for Sales and Order Management \(SOM\) delivers AI-powered insights and automation to drive faster, smarter sales and order processes.
