# Flow Diagramming release notes

Version history for the Flow Diagramming application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 25.1.2 - February 2024**
    -   Added subtle warning to actions and flow logic that contain empty required fields
    -   Added support for datastream actions
    -   Added support for Exit Loop flow logic \(also known as Break\)
    -   Added support for Skip Iteration flow logic \(also known as Continue\)
    -   Added support for Placeholder flow logic to support flow generation
-   **Version 24.3.2 - November 2023**
    -   New:
        -   Added support for Try/Catch flow logic
        -   Added support for annotations
        -   Added support for step details
-   **Version 24.1.1 - August 2023**

    New: This release adds support for 'Do the Following Until'.

-   **Version 23.2.1 - May 2023**

    This release contains fixes for performance.

-   **Version 22.1.5 - March 2023**

    Fixed: PRB1639711 Improve performance when looking up large numbers of flows and subflows.

-   **Version 23.2.0 - February 2023**
    -   The following features are added as part of store release:
        -   Support for 'Make a Decision' flow logic
        -   Export flow diagram
        -   Support for flow triggers:
            -   asynchronous REST API
            -   Kafka
            -   MetricBase
        -   Open action in Action/Subflow Designer from action or context menu
        -   Duplicate action
-   **Version 22.1.4 - August 2022**

    New: Support for Do the following in parallel flow logic.

-   **Version 21.1.2 - March 2022**

    Minor fixes.

-   **Version 21.1.1 - February 2022**
    -   New:
        -   Build new flows as a diagram
        -   Toggle to see flows as in the diagram or natural language views
        -   Edit flow content in a panel
        -   Follow the paths that your flow creates
        -   Zoom in and out of your flow
