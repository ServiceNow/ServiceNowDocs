# Next Experience Readiness Checker release notes

Version history for the Next Experience Readiness Checker application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 24.0.0 - February 2024**

    Fixed: Minor updates to context text for Vancouver+ instances.

-   **Version 23.0.7 - January 2023**
    -   Changed:
        -   Added responsive layout to interface
        -   Large count values are abbreviated
        -   Count value for Connect Support now shows number of conversations instead of individual messages
        -   Updated application messages to accurately reflect state as of latest patch and familiy release
    -   Fixed:
        -   Fixed spelling errors
        -   Applications without any usage data returned will not show links to data lists
-   **Version 23.0.6 - November 2022**

    The Next Experience Readiness checker tool enables admin users to examine their instance to see if agents and other users are currently using applications that are not compatible with Next Experience. Based on the usage found, the tool makes a recommendation regarding turning on Next Experience immediately or doing additional due diligence before turning it on.
