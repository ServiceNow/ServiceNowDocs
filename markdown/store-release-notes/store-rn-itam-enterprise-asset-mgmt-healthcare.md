# Enterprise Asset Management for Healthcare release notes

Version history for the Enterprise Asset Management for Healthcare application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.2 - March 2026**

    App is certified with the latest EAM 10.0.

-   **Version 1.1.0 - August 2024**
    -   -   New:
    -   Added new roles
        -   sn\_eamhc.medical\_asset\_manager
            -   Full access to Medical asset workspace, Mobile Agent app, all EAM features, including all EAM workflows and tasks, and creating models and assets in the respective classes.
        -   sn\_eamhc.medical\_asset\_technician
            -   Limited access to Medical asset workspace and Mobile Agent app and ability to work on select EAM workflows and tasks limited to the respective model and asset classes.
-   **Version 1.0.0 - February 2024**

    Enterprise Asset Management for Healthcare is an application that provides medical asset managers and medical asset technicians with access to Enterprise Asset Management workflows and features to manage the full lifecycle of medical models and assets.
