# GRC: Virtual Agent release notes

Version history for the GRC: Virtual Agent on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 19.1.0 - November 2024**

    Fixed: Report risk event now available only for IRM Enterprise users and no longer appears for IRM Professional users.

-   **Version 16.0.1 - May 2023**

    Fixed: Reduced the size of the application jar file.

-   **Version 12.0.0 - March 2021**

    Fixed: Fixed a bug in the Quebec release due to which the virtual agent topics were not visible for customers

-   **Version 9.0.0 - November 2019**

    Initial version of GRC: Virtual Agent.
