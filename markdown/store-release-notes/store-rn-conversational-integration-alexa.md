# Conversational Integration with Alexa release notes

Version history for the Conversational Integration with Alexa integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.5 - August 2024 \(Xanadu\)**

    Fixed: Unable to run Alexa due to changes on provider user id in request payload from Alexa.

-   **Version 1.4.2 - August 2022**

    Fixed: Voice input error.

-   **Version 1.4.1 - May 2022**

    Fixed: Pre-chat survey and localization issues

-   **Version 1.3.0 - February 2022**
    -   New: Unified Logging has been enabled.
    -   Fixed:
        -   Issue with 'Say repeat to hear the options again' on voice-only devices
        -   Issue with People card
        -   Technical issues with column header in a topic
        -   Issue with updates to 'number\_indicator\_msg' bot message
        -   Issue with new built-in intents that are not supported
        -   The intermittent hanging of pre-chat survey and its missing summary
        -   Issue with 'boolean\_msg bot' message
        -   Issue with the prompt and scrolling on the pre-chat survey
        -   The issue with bot response
        -   The issue with logs caused due to incorrect property in the script
-   **Version 1.0.3 - October 2021**

    ServiceNow® Conversational Integration with Alexa enables ServiceNow Virtual Agent to interact with requesters through Alexa voice interface using the Virtual Agent topics.
