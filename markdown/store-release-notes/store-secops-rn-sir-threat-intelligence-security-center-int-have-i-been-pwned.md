# Threat Intelligence Security Center integration with Have I been pwned? release notes

Version history for the ServiceNow® Threat Intelligence Security Center integration with Have I been pwned? application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.1.0 - June 2026**

    Changed: Implement Query Range ACL.

-   **Version 2.0.0 - March 2026**

    "Have I Been Pwned" integration with Threat Intelligence Security Center integration enables Threat Intelligence Analyst to enrich observables, such as email addresses and domains, with compromise status information by querying external breach and exposure data sources.
