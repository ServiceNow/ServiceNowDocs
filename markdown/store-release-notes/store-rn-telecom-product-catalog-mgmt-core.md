# Product Catalog Management Core release notes

Version history for the Product Catalog Management Core on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.2 - March 2022**
    -   Changed:
        -   Updated the data model to be extensible
        -   Added missing domain and class name field to support domain separation and table extension
-   **Version 1.4.1 - October 2021**

    Changed: Minor ACL changes

-   **Version 1.1.0 - March 2021**
    -   New:
        -   A new Specification Category table will drive product model creation
        -   The Product Offering Characteristic table has been updated with to allow mapping to the source specification
        -   An external id field has been added to support integration with external order systems
-   **Version 1.0.0 - December 2020**

    This application contains the base product catalog data model and functionality to define product, services, and resource specifications, and product offerings based on those specifications.
