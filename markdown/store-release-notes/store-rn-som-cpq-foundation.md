# CPQ - Foundation release notes

Version history for the ServiceNow® CPQ - Foundation application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.1 - April 2026**
    -   CPQ - Foundation delivers AI skills and agents purpose-built for configure, price, and quote workflows. Designed for enterprises processing high volumes of orders, invoices, and returns, this solution enables B2B customers and service teams to resolve requests faster with less manual effort.
    -   Built on the ServiceNow AI Platform, CPQ - Foundation includes prebuilt AI agents that automate order case intake, assist order operations, handle invoice dispute intake, and support return merchandise authorization \(RMA\) requests, reducing resolution times and improving customer satisfaction.
