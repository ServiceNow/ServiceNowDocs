# Flow Template Builder release notes

Version history for the Flow Template Builder application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.9 - May 2023**

    Fixed: Error on flows that have no data pills defined.

-   **Version 1.0.8 - June 2022**

    Fixed: Patch release of Flow Template Builder. This version includes a fix for removal of hard dependency on AES.

-   **Version 1.0.6 - May 2022**

    Fixed: Patch release of Flow Template Builder. This version includes fixes for long string text wrap and console errors while adding empty pills at inputs.

-   **Version 1.0.4 - February 2022**

    Flow Templates guide flow authors to create new flows for common use cases. Flow Template Builder allows citizen developers to create their own flow templates.
