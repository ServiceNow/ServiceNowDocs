# Workspace Builder for App Engine release notes

Version history for the AES Workspace Builder application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 28.2.0 - December 2025**

    Changed: Maintenance release

-   **Version 28.1.1 - August 2025**

    Maintenance update.

-   **Version 27.2.2 - May 2025**

    No updates.

-   **Version 27.1.1 - February 2025**

    Maintenance update.

-   **Version 26.2.1 - November 2024**

    Fixed: Wizard for workspace creation showing wrong or unexpected tables.

-   **Version 25.1.0 - May 2024**
    -   New: Adding refresh icons to the toggle components across workspace builder.
    -   Fixed:
        -   Search configuration issues on workspace settings modal.
        -   Breadcrumb issue on the search page from portal UI template.
        -   When sys\_ux\_applicability is not available, auto creation functionality from workspace builder creates duplicate record is now fixed.
-   **Version 25.0.0 - February 2024**

    New: When opening an existing workspace without defined roles, a new modal appears and prompts the user to enter roles before proceeding.

-   **Version 24.2.0 - January 2024 \(Vancouver\)**

    Addressed a security issue.

-   **Version 23.3.0 - January 2024 \(Utah\)**

    Addressed a security issue.

-   **Version 22.3.0 - January 2024 \(Tokyo\)**

    Addressed a security issue.

-   **Version 24.1.2 - November 2023**
    -   New: Introducing keyboard-based drag-and-drop functionality to support accessibility.
    -   Changed:
        -   Search configuration in workspace settings changed from drop-down list to type-ahead with pagination.
        -   Workspace settings data is now fetched on-demand when you select the Settings button.
    -   Fixed: Correct role and selection status appears for side navigation items to support accessibility.
-   **Version 24.0.1 - August 2023**
    -   New: Disabled Contextual sidebar and playbook for old record pages opened in the Workspace Builder.
    -   Changed:
        -   Colors and fonts in Workspace Builder theme.
        -   The performance of the Workspace Builder has been updated, resulting in a reduced loading time.
    -   Fixed:
        -   URL fixed for custom company code.
        -   Workspace Builder sometimes fails to load in instances with large datasets.
-   **Version 23.1.0 - May 2023**
    -   New:
        -   Added ability to dot walk when managing list columns.
        -   When a custom or high-code dashboard is included in a workspace, developers will get an educational modal to guide them to UI Builder to edit it.
        -   With updated keyboard accessibility, developers can navigate through Workspace Builder using the tab key.
    -   Changed: Colors and fonts in building experience.
-   **Version 23.0.1 - February 2023**
    -   New:
        -   A developer can reorder list categories by dragging them into a new position.
        -   List platform metadata is automatically cleaned up when a developer deletes a list category or filtered list in Workspace Builder.
        -   When creating a workspace, filtered list or record page tables used for backend purposes are hidden.
    -   Fixed:
        -   List usability issues.
        -   Components on the home page may have appeared shrunken in TS2. The width on those components has been reset, so new workspaces won't experience this effect.
-   **Version 22.1.3 - November 2022**
    -   Workspace Builder for App Engine is a no-code application that allows you to manage dashboards, build lists, and customize record pages for workspaces created in App Engine Studio. App Engine licensed customers enjoy exclusive access to this premium feature.
    -   Developers can drive value with custom workspaces faster than ever with clicks instead of code. The newly streamlined experience allows developers to use a single builder to create custom data visualizations, filtered lists, and record pages all in one place.
