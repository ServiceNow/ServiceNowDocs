# product configurator release notes

Version history for the product configurator application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - March 2026**

    Changed: Localization and Internationalization changes

-   **Version 1.0.1 - November 2024**

    New: When creating a Sales Agreement Quote and adding a configurable offer, the Configurator UI is launched. It allows the user to multi-select all pre-negotiated options of the configurable product offer, as agreed upon with the customer in the Sales Agreement.

-   **Version 1.0.0 - February 2024**

    The product configurator application allows businesses to streamline the process of creating customized quotes or orders for complex products and services. It provides a user-friendly interface for sales representatives, customers, or internal teams to dynamically configure and personalize products according to specific requirements. Also provides visibility into pricing changes during the configuration process.
