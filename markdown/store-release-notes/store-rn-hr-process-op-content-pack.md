# HR Service Delivery Process Mining Content Pack release notes

Version history for the HR Service Delivery Process Mining Content Pack application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.1 - February 2024**

    No new features or fixes. This major release now supports Process Mining v25.6.3+.

-   **Version 3.0.1 - August 2023**

    This version is compatible with Vancouver and Process Mining v24.4.1. There are no net new features or fixes.

-   **Version 2.0.0 - February 2023**

    Changed: Content updates to support Process Mining v23.x

-   **Version 1.0.0 - August 2022**

    Configuration to support the usage and analysis of HRSD workflows with Process Mining
