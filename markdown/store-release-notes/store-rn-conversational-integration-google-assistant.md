# Conversational Integration with Google Assistant release notes

Version history for the Conversational Integration with Google Assistant integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.6 - August 2022**

    General fixes.

-   **Version 2.0.5 - February 2022**

    New: Unified Logging has been enabled for the San Diego release. Note: This version is backward compatible with the Rome release, which does not have Unified Logging functionality.

-   **Version 2.0.4 - December 2021**
    -   Fixed:
        -   The display of white image issue when the device is in Dark Mode.
        -   L10N issues.
        -   The additional 'Please refer to the screen' speech output issue.
        -   The missing policy URL for KB card in screen device.
        -   People card for voice only devices by adding the department attribute.
-   **Version 2.0.3 - October 2021**

    ServiceNow® Conversational Integration with Google Assistant enables ServiceNow Virtual Agent to interact with requesters through Google Assistant voice interface using the Virtual Agent topics.
