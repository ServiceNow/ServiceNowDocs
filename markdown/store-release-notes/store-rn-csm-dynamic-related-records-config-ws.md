# Dynamic Related Records for Configurable Workspace release notes

Version history for the Dynamic Related Records for Configurable Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 25.7.0 - June 2026**

    Non-Glide Cobalt Raven ACLs in Product Code - Security enhancements to query access controls for Dynamic Related Records

-   **Version 25.5.2 - February 2026**

    Improved Dynamic Related Record \(DRR\) reliability by gracefully handling invalid Related Record Definitions so all valid definitions continue to load.

-   **Version 25.5.1 - December 2025**
    -   New: Contains new granular admin roles for enhanced security.
    -   Changed: Certain fields were made to be Read Only for security purposes.
-   **Version 25.4.4 - June 2024**

    Security bug.

-   **Version 25.4.3 - May 2024**

    Minor accessibility and security fixes.

-   **Version 25.4.2 - March 2024**

    Minor localization improvements.

-   **Version 25.4.1 - February 2024**
    -   Displays related records in the contextual side panel in Configurable Workspace that dynamically change based on the context of the current record or playbook activity.
    -   The existing family release plugin \(com.snc.uib.sn\_dyn\_rel\_rec\) is moved to a store release starting in Washington DC. Any new changes to the dynamic related records feature will be delivered via this store app.
