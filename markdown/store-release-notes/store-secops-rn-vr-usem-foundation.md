# USEM - Foundation release notes

Version history for the ServiceNow® USEM - Foundation application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.0 - June 2026 \(USEM\)**

    No new features with this release. Updated the application to support bundling with other USEM applications.

-   **Version 1.0.2 - April 2026**

    Base \(Foundation\) tier app for licensing Unified Security Exposure Management \(USEM\) AI capabilities.
