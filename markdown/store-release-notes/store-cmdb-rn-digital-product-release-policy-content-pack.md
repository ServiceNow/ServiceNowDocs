# Digital Product Release Policy Content Pack release notes

Version history for the Digital Product Release Policy Content Pack application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.0 - August 2025**

    New: Added a new policy: Validate change requests. This policy validates whether all changes associated with the release are in a particular state.

-   **Version 2.1.0 - May 2025**

    Changed: Updated policies to run appropriately for Product and Service releases.

-   **Version 2.0.0 - February 2025**

    Changed: Updated policy content for multi-product releases.

-   **Version 1.1.0 - November 2024**

    Fixed: Minor bug fixes.

-   **Version 1.0.0 - February 2024**

    Digital Product Release Policy Content Pack contains data collectors and policies that enable you to automate phase gate verification using scripted or low-code policies.
