# Plivo spoke release notes

Version history for the Integration Hub Plivo spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 1.2.0 - November 2025**

    New: Added read-only fields for security measures.

-   **Version 1.1.0 - February 2024**

    New: Added authentication templates, to help in the spoke setup using the Connections dashboard.

-   **Version 1.0.4 - September 2022**

    Patch version of Plivo Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.2 - December 2021**

    Changed: Patch release of Plivo Spoke for IntegrationHub. The version includes security-related changes.

-   **Version 1.0.1 - October 2021**
    -   Fixed:
        -   Missing action descriptions for multiple actions.
        -   Issue with Look up Message action.
        -   Empty string validation error for recipient phone numbers within Send Bulk SMS action.
        -   Empty error message for unauthorized error.
        -   Limited records being retrieved for Look up action records.
-   **Version 1.0.0 - July 2020**

    Provides actions to manage calls, conferences and SMS in Plivo.
