# Cloud Storage release notes

Version history for the Cloud Storage application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.1 - November 2024**

    Patch for bug fixes in error handlings.

-   **Version 1.0.0 - August 2024**

    This application helps capture all the Cloud capacity usage information from both instance and account level information to be populated in central instance to be used for apps like Subscription Management.
