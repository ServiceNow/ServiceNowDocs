# Audio player component release notes

Version history for the Audio player component application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 27.3.1 - March 2026**
    -   New:
        -   Enhanced Audio Playback Controls: Allows users to jump ahead/back and control playback speed on voicemail recordings, improving navigation and listening efficiency.
        -   Designed to address user requests for more flexible audio playback.
        -   Internationalization and Cross-Browser Support: Ensures the audio player displays content in the user's selected language and functions consistently across supported browsers, addressing global accessibility and compatibility needs.
        -   Accessibility Enhancements: Improves accessibility by supporting keyboard navigation, event interception, and compact mode, enabling users with different needs to interact with the audio player effectively.
-   **Version 23.0.0 - August 2024**

    Changed: UIB icon change.

-   **Version 22.1.0 - November 2022**

    New: Add support for Internationalization and localization

-   **Version 22.0.0 - August 2022**

    Configure and play audio files available on your instance or other locations.
