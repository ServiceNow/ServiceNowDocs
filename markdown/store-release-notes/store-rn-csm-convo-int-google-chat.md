# Conversational Integration with Google chat release notes

Version history for the Conversational Integration with Google chat application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.2 - June 2026 \(Zurich, Australia\)**
    -   Fixed: File upload is now working properly
    -   Enhancement\(s\): Enhanced picker experience
-   **Version 3.0.1 - April 2026**
    -   New: Support Virtual Agent deployment via Google Workspace Marketplace app listing
    -   Fixed:
        -   Admin experience issues when deleting the integration
        -   Missing image cards from catalog items
        -   Missing live agent prefix in messages
        -   Fixed markdown formatting and parsing issues
-   **Version 2.0.5 - January 2026 \(Yokohama, Zurich\)**
    -   Support for Now Assist
    -   Support for Synthesized Response
    -   Streaming
-   **Version 2.0.4 - December 2025**
    -   Support for Now Assist
    -   Support for Synthesized Response
    -   Streaming
-   **Version 1.0.0 - August 2025**

    Enable end users to create and follow up on ServiceNow requests, receive notifications, and engage with Virtual Agent and live agents directly from within the Google Chat messaging app.
