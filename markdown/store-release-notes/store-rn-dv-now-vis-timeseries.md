# @servicenow/now-vis-timeseries release notes

Version history for the @servicenow/now-vis-timeseries application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 23.0.8 - March 2023**

    Bug fixes

-   **Version 23.0.5 - February 2023**
    -   Timeseries is a part of Data Visualization component. It presents multiple values of ServiceNow or custom data over time.
    -   Timeseries data visualizations can be used in the Next Experience and are available from Next Experience Dashboards, the UI Builder, and the Visualization Designer.
    -   This application is installed automatically with the "Data Visualization Core" application.
    -   Fixed: Bug fixes.
