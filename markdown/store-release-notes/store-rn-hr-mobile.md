# HR Service Delivery for mobile release notes

Version history for the HR Service Delivery for mobile on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 21.3.0 - June 2026**

    Fixed: Survey-type HR tasks on the Mobile Employee Service Portal now support multi-page navigation and complete reliably.

-   **Version 21.2.9 - December 2025**

    Fixed: Minor issues in displaying badges.

-   **Version 21.2.7 - May 2025**

    Fixed: Minor changes related to UI theming.

-   **Version 21.2.6 - February 2025**

    Fixed: Minor bug fixes.

-   **Version 21.2.5 - December 2024**

    Fixed: The push notification for an HR case in Now Mobile shows incomplete message.

-   **Version 21.2.4 - August 2024**
    -   The Subject person field in the HR case on Now Mobile will refer to the Subject person profile tofetch the data.
    -   Attachments added from Now Mobile to HR cases for Lifecycle event cases are associated with the Lifecycle event case table.
-   **Version 21.2.3 - May 2024**

    An authorization issue is fixed.

-   **Version 21.2.2 - February 2024**

    This version of Now Mobile app enables the next experience for the Mobile along with some important fixes.

-   **Version 21.2.0 - August 2023**

    This version of Now Mobile app enables the next experience for the Mobile along with some important fixes.

-   **Version 21.1.1 - May 2023**

    Fixed: This version includes a fix to enable new versions of i10n/i18n tools.

-   **Version 21.1.0 - February 2023**

    New: This version of Now Mobile app enables the next experience for the Mobile along with some important fixes.

-   **Version 25.0.1 - May 2022**
    -   New:
        -   Improved navigation with the Quick links section on the home screen.
        -   Added content recommendations with the Recommended for you section on the support tab.
        -   Introduced cross-channel favorites feature; access all favorites, favorite articles, and requests from the Now Mobile app.
-   **Version 21.0.5 - November 2021**
    -   Fixed:
        -   Plugin Repair issue in multi node environment is fixed
        -   Hide section if empty checkbox in My onboarding record of Now mobile and Now onboarding Nav is fixed
        -   Cosmetic issue related to request form page is fixed
-   **Version 21.0.4 - July 2021**
    -   New:
        -   New Allows employees to search or browse for HR services
        -   Allows employees to submit HR requests
        -   Employees can chat with a virtual agent
        -   Can view and complete assigned tasks
