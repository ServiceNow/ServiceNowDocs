# Approvals Hub integration with Workday release notes

Version history for the Approvals Hub integration with Workday application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - June 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 2.0.1 - August 2025**

    Configuration details screen for Workday approvals in Now Mobile.

-   **Version 1.0.9 - May 2024**

    No new feature was delivered in May '24.

-   **Version 1.0.4 - February 2024**

    Added out-of-the-box configurations to enable approvers to filter Workday approvals and show Workday approvals in Manager Hub.

-   **Version 1.0.2 - November 2023**

    This app allows you to bring Workday approval tasks directly into ServiceNow Employee Center Pro - providing a centralized interface for seamless approvals.
