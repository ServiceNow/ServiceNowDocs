# Alignment Planner Workspace integrations for PPM and Agile 2.0 release notes

Version history for the Alignment Planner Workspace integrations for PPM and Agile 2.0 application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.0 - February 2022**
    -   New:
        -   Introducing a new table map for SAFe Epic to support integration of SAFe Epics between APW and SAFe plugins.
        -   Guided setup is now available for configuring APW Integration with PPM / Agile2.0 / SAFe
    -   Fixed: Project milestone integration with APW is now supported for Teamspace projects.
-   **Version 2.0.0 - November 2021**
    -   New:
        -   Supports integration of project milestones from Project Workspace to APW.
        -   Maps primary goal field for projects, demands, scrum epics between APW and PPM / Agile 2.0.
        -   New table map template for mapping milestones.
    -   Changed: Adds planning organization field on scrum epic. This field is used in the scrum epic table map to map planning organization fields. In the previous release, Top portfolio field was mapped to planning organization field.
