# Document display component release notes

Version history for the Document display component application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 27.5.2 - March 2026**

    New: new property Search text to highlight text in PDF

-   **Version 27.0.0 - May 2025**

    New: Document comparison feature within document display component.

-   **Version 26.0.1 - August 2024**
    -   Changed:
        -   Updated the component icon in UI builder.
        -   Updated the dependency to use the latest HTML editor with TinyMCE v6.
-   **Version 25.0.0 - February 2024**

    Fixed: UI fixes to update accessibility \[WCAG 2.1 AA\] support.

-   **Version 22.1.0 - November 2022**

    New: Added support for internationalization and localization

-   **Version 22.0.0 - August 2022**

    Display documents to users with this component. It can display PDF or image files from the attachments table or html documents. HTML documents can be configured to allow users to edit. Other file types can only be downloaded to view.
