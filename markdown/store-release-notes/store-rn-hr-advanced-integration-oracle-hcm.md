# HR Service Delivery Advanced Integration with Oracle HCM release notes

Version history for the HR Service Delivery Advanced Integration with Oracle HCM application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - June 2026**

    Changed: Access control lists \(ACLs\) have been updated to align with current platform security standards.

-   **Version 1.3.0 - December 2025**

    Minor bug fixes.

-   **Version 1.2.2 - May 2025**

    Minor bug fixes.

-   **Version 1.2.1 - November 2024**
    -   View personalized information about Direct Deposit, Total Rewards, Holiday Calendar, and Time Off information from ServiceNow.
    -   Raise a case for any query/help via the widget.
-   **Version 1.1.4 - August 2024**
    -   View personalized information about Direct Deposit, Total Rewards, Holiday Calendar, and Time Off information from ServiceNow
    -   Raise a case for any query/help via the widget
-   **Version 1.1.3 - May 2024**
    -   View personalized information about Direct Deposit, Total Rewards, Holiday Calendar, and Time Off information from ServiceNow.
    -   Raise a case for any query/help via the widget.
-   **Version 1.0.6 - February 2024**

    Minor fixes.

-   **Version 1.0.8 - December 2023**

    Fixed: Minor performance improvements.

-   **Version 1.0.5 - December 2023**

    Fixed: Minor performance improvements.

-   **Version 1.0.4 - July 2023**
    -   This patch version has been certified with Oracle HCM Cloud Spoke 3.3.0.
-   **Version 1.0.3 - November 2022**

    Minor optimization and refactoring

-   **Version 1.0.1 - February 2022**
    -   Fixed: Error message for payslip feature
    -   Changed: Action name for use cases 'Total Rewards' and 'Payroll Discrepancy' have been renamed.
-   **Version 1.0.0 - November 2021**
    -   New:
        -   Submit a query related to your direct deposit account in the Oracle HCM system.
        -   Submit a query related to your time off balance system in the Oracle HCM system.
        -   View your holiday calendars in the Oracle HCM system in the configured portal in a ServiceNow instance.
        -   View your total rewards information in the configured portal in a ServiceNow instance.
