# Password Reset for Virtual Agent release notes

Version history for the Password Reset for Virtual Agent on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.0.6 - August 2025**

    Fixed minor bugs

-   **Version 5.0.5 - May 2025**

    Fixed minor bugs.

-   **Version 5.0.4 - August 2024**
    -   Fixed:
        -   Resolved issue with Change Password topic in Now Mobile App
        -   Resolved issue where Reset Password topic was not publishing properly
-   **Version 5.0.3 - February 2024**
    -   New: Added support for Soft PIN token expiration
    -   Changed: Soft PIN tokens issued to end-users will now expire
-   **Version 5.0.2 - November 2023**

    Minor fixes.

-   **Version 5.0.1 - February 2023**

    New: Added the enrollment topic support for Password Reset for Virtual Agent.

-   **Version 4.1.1 - November 2022**
    -   New:
        -   Password Reset topic
            -   IVR support
            -   Auto-generated passwords for end users
-   **Version 4.0.2 - August 2022**

    New:

    -   Extended asynchronous password reset support to the Virtual Agent topics.
    -   Updated existing topics to address the user enumeration issues for parity with the web experience.
-   **Version 3.1.1. - May 2022**
    -   New:
        -   Added custom components for the change password conversation to extend support to third-party credential stores in the Virtual Agent web client.
        -   Added NLU models for additional languages.
-   **Version 3.0.5 - February 2022**
    -   New:
        -   Topics republished for San Diego compatibility
        -   Actionable notifications for password expirations
-   **Version 2.0.0 - December 2021**
    -   Deflection tracker added to topics to improve Virtual Agent Analytics
    -   Language support added for conversational flows and NLU models for the following:
        -   Brazilian Portuguese
        -   Chinese Simplified
        -   Italian
        -   Japanese
        -   Korean
        -   Spanish
-   **Version 1.3.2 - September 2021**

    Fixed: Removed duplicate configuration records for Virtual Agent issue auto-resolution.

-   **Version 1.3.1 - August 2021**
    -   New:
        -   Added French and German language support for virtual agent topic flows
        -   Added French and German language support for NLU intents
        -   Increased NLU model confidence threshold to improve accuracy performance
-   **Version 1.2.0 - June 2021**
    -   New:
        -   Updated Password Change topic to allow additional attempts for incorrectly inputted password
        -   Minor verbiage changes to Password Change and Password Reset topics
-   **Version 1.1.0 - April 2021**
    -   New:
        -   Password Reset for Virtual Agent includes several predefined topic conversations designed to help your users complete credential and account related issues. These topics focus on the below use cases:
            -   Password Reset
            -   Password Change
            -   Account Unlock
-   **Version 1.0.1 - March 2021**
    -   New:
        -   Password Reset for Virtual Agent includes several predefined topic conversations designed to help your users complete credential and account related issues. These topics focus on the below use cases:
            -   Password Reset
            -   Password Change
            -   Account Unlock
