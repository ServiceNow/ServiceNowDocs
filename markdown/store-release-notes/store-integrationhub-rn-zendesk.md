# Zendesk spoke release notes

Version history for the Zendesk Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.9.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.8.0 - November 2025**

    Security patch: Improve sample webhook registry security.

-   **Version 1.7.1 - June 2025**

    Fixed: Upload attachment other than JSON types.

-   **Version 1.7.0 - November 2024**

    Evaluated and modified sandbox access for client-callable script includes as false.

-   **Version 1.6.2 - January 2024**

    Fixed: Look up Users Stream action does not bring in all users using pagination.

-   **Version 1.6.1 - August 2023**

    In v1.6.1, the Look up Custom Role By ID action is fixed.

-   **Version 1.6.0 - May 2023**

    Passed SAM Application testing and relist this Spoke with about 40+ new actions that align with Zendesk ticket management business.

-   **Version 1.5.0 - September 2022**
    -   Added 50+ more actions
    -   Added bi-directional integration design pattern
    -   Added remote table
    -   Added authentication template
    -   Added sample outbound flow
-   **Version 1.0.0 - August 2021**

    New: Manage Zendesk accounts using your ServiceNow instance. Analyse the data usage for Zendesk and reclaim the idle accounts.
