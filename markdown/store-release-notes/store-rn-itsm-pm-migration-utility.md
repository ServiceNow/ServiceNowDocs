# Problem Management Migration Utility release notes

Version history for the ITSM Problem Management Migration Utility on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.0 - March 2026**

    Changed: Updates to read only dictionary attributes, no functional changes.

-   **Version 2.2.0 - May 2025**

    No changes.

-   **Version 2.1.7 - November 2024**

    Minor fixes.

-   **Version 2.1.5 - February 2024**

    Changed: Version updated for dependency version update.

-   **Version 2.1.4 - February 2023**

    Fixed: UI-related localization issues.

-   **Version 2.1.2 - November 2022**
    -   New: Added a clean-up step to remove all unused widgets, widget indicators, reports, and report sources.
    -   Fixed: Architectural changes \(no functional impact\)
-   **Version 2.1.0 - May 2022**

    New: Introduced a new Clean-Up task for the Old Process Flow Formatters.

-   **Version 2.0.1 - September 2021**
    -   Fixed:
        -   Modified details for Backup instance guided setup task
        -   Minor fixes
-   **Version 2.0.0 - April 2021**

    New: Listed modifications on a few Core ITSM Business rules under the Warnings section as they impact the Problem States.

-   **Version 1.3.1 - February 2021**

    Fixed: minor issues fixed.

-   **Version 1.3.0 - January 2021**

    New: Added a new guided setup clean up step which guides the admin to remove or deactivate the old filters on their instance.

-   **Version 1.2.0 - October 2020**

    New: Added a new guided setup step which guides the admin to add the problem management roles to the users on their instance

-   **Version 1.1.0 - July 2020**

    Minor issues fixed

-   **Version 1.0.6 - May 2020**

    The Problem Management Migration Utility is a guided setup application that helps upgrade customers to adopt the base version of problem management using the guided problem life cycle \(Problem Management Best Practice - Madrid - State Model\).
