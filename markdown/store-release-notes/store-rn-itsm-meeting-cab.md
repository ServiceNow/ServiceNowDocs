# Meeting CAB release notes

Version history for the Meeting CAB on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 9.2.0 - June 2026**
    -   Changes:
        -   Updated plugin dependencies to ensure compatibility with the ServiceNow Australia and later releases.
        -   Minor defect fixes.
-   **Version 9.0.1 - March 2026**

    Updated version for compatibility with Service Operations Workspace 9.0.

-   **Version 2.0.2 - December 2025**

    No changes.

-   **Version 1.0.7 - August 2025**
    -   Fixed:
        -   Schedule page for a Change should be more performant as a trigger to get related data has been removed as it is no longer required
        -   The Upcoming CAB Meetings button should always redirect users to the list of upcoming CAB Meetings
        -   Pagination controls for the CAB Meeting are fixed to the bottom of the page. This functionality is managed using UI Builder
-   **Version 1.0.6 - May 2025**
    -   Change Advisory Board \(CAB\) for Service Operations Workspace supports the CAB manager experience for defining and creating CAB meetings. It guides them through the process of scheduling a reoccurring meeting, inviting appropriate attendees and dynamically populating agenda items for each meeting.
    -   CAB Workspace for SOW allows CAB managers to run CAB meetings and for CAB attendees to discuss agenda items. CAB attendees can easily add notes on each item facilitating discussion, view the change on which the agenda item is based and authorize the change if appropriate. CAB managers can write up meeting notes and share them with stakeholders when the CAB meeting completes.
    -   CAB for Service Operations Workspace allows CAB managers, change managers and CAB attendees to work together to ensure changes are reviewed and approved efficiently and in real-time.
