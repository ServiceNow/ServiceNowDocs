# Customer Engagement Sequences release notes

Version history for the Customer Engagement Sequences application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.0 - June 2026**

    Fixed: Enhanced security by introducing non-glide Query Range ACLs.

-   **Version 2.1.0 - April 2026**

    Fixed: Enhancements aimed at improving the calling experience from the playbook callback activity during runtime.

-   **Version 2.0.1 - December 2025**
    -   New:
        -   Add multi-trigger sequences with decision branches between stages.
        -   Leverage schedule call activity for structured telesales workflows within sequences.
        -   Enable click-to-call integration with platforms like Amazon Connect on sequences call activity.
        -   Implement granular role framework for sequence admins, writers, executors, and readers.
        -   Enforce runtime permissions for fine-grained access control of sequences.
-   **Version 1.0.0 - August 2025**

    Customer Engagement Sequences application automates personalized outreach and follow-up activities to engage customers throughout their entire lifecycle.
