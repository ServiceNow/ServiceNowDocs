# OT Enhanced Access Control release notes

Version history for the ServiceNow® OT Enhanced Access Control application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - June 2026**

    Enhanced Access Control for Operational Technology \(OT\) utilizes deny unless access control rules \(ACLs\) to help enforce IT and OT separation and site-based access. Installing this plugin turns on Enhanced Role Based Access Control for Operational Technology Manager.
