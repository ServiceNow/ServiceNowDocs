# Hiring Core release notes

Version history for the Hiring Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - May 2024**

    New: Added additional security constraints on the Job Req table for Opportunity Marketplace use case.

-   **Version 1.0.0 - February 2024**

    The Hiring Core application serves as the foundation for talent acquisition and internal mobility within your organization. By extracting relevant data from the foundational data model, it efficiently populates job requisitions and provides a display of application status. Hiring Core encompasses essential data models and shared components, seamlessly supporting both Talent Acquisition and Internal Mobility processes.
