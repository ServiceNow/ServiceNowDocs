# Atlassian Jira integrations Common release notes

Version history for the Atlassian Jira Integrations Common on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - June 2021**

    Changed: Simplified process of importing issues from Jira to ServiceNow.

-   **Version 2.0.0 - September 2020**

    Changed: Simplified process of importing issues from Jira to ServiceNow.

-   **Version 1.1.0 - July 2020**

    Introducing support for project-level mapping between Jira and ServiceNow.
