# Workplace Space Mapping release notes

Version history for the Workplace Space Mapping application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.20.5 - June 2026**

    Fixed: Fixed accessibility issues.

-   **Version 1.20.0 - March 2026**

    No updates.

-   **Version 1.19.4 - December 2025**

    No updates.

-   **Version 1.19.0 - August 2025**

    Changed: Use the new neighborhood tables, which are part of Workplace Core.

-   **Version 1.18.1 - May 2025**

    No updates.

-   **Version 1.17.1 - February 2025**

    No updates.

-   **Version 1.16.0 - November 2024**
    -   New:
        -   Employees can now select a space to see its details from the location directory.
        -   Admins can indicate whether the filter settings in the location directory are persistent between sessions using system properties.
    -   Fixed:
        -   Security fixes.
        -   Accessibility improvements.
-   **Version 1.15.0 - August 2024**
    -   New:
        -   Property to show reservation and occupancy data on Location Directory Maps
        -   Property to set auto refresh time \(to fetch latest reservation and occupancy data\) on Location Directory Maps
    -   Fixed: The interactive floor map widget from the catalog page did not load when multiple campuses were involved. This has been fixed.
-   **Version 1.13.0 - May 2024**
    -   Fixed:
        -   Not able to select a space from the floor map in the Request furniture record producer.
        -   Accessibility fixes
-   **Version 1.10.10 - February 2024**

    Fixes only.

-   **Version 1.10.7 - November 2023**

    No significant updates.

-   **Version 1.10.5 - May 2023**

    Fixed: Earlier, selecting a space on the interactive map returned an error. This has been fixed now.

-   **Version 1.7.2 - December 2021**
    -   Changed
        -   Generic improvements have been made on the loading time for viewing and using the floor map while making a reservation.
        -   The floor map legend map is now collapsible to reduce any map obstruction. You can configure the legend entry color and label.
        -   The floor picker and the legend now also display the total capacity statistics.
    -   Fixed
        -   Using a voiceover tool did not read all the mandatory fields when using the OOB record producer.
        -   The location directory showed inactive users.
        -   In some cases, while searching for available spaces using the floor map, a filter needed to be applied multiple times to show the correct locations.
-   **Version 1.6.1 - October 2021**

    New: Additional support has been added to support all the 22 languages that are currently available across the ServiceNow AI Platform.

-   **Version 1.5.2 - September 2021**
    -   New: A new card-only view mode has been introduced to view locations when there is no map for a building/floor.
    -   Changed: The employee details will now surface on the map card along with location information.
-   **Version 1.4.1 - June 2021**
    -   New: You can also search for users in the Location directory.
    -   Changed: Improved way-finding experience on a floor map in the Location directory.
-   **Version 1.3.2 - March 2021**

    New: Minor fixes.

-   **Version 1.2.3 - January 2021**
    -   New:
        -   When Reservation Management is installed:
            -   Reservation admins can enable Mappedin floor maps on the Make a reservation portal page. They can do this by turning on the Show map view checkbox for any reservable module that should include maps.
            -   On the Make a reservation portal page, workplace users can use Mappedin floor plans \(in addition to the card and schedule views\) to make reservations.
-   **Version 1.1.1 - December 2020**
    -   New:
        -   When Reservation Management is installed:
            -   Reservation admins can enable Mappedin floor maps on the Make a reservation portal page. They can do this by turning on the Show map view checkbox for any reservable module that should include maps.
            -   On the Make a reservation portal page, workplace users can use Mappedin floor plans \(in addition to the card and schedule views\) to make reservations.
-   **Version 1.0.2 - November 2020**
    -   Workplace Space Mapping provides employees with an accurate and interactive floor map to reserve a workspace, conference room, or parking spot based on availability and view reserved or unavailable spaces using an intuitive color legend. Workplace managers have the ability to import space administration tables, define reservable and non-reservable spaces, and track reservations.
        -   New: Feature parity with Workplace Safety Management mapping solution
