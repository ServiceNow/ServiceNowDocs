# Opportunity Marketplace release notes

Version history for the Opportunity Marketplace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.7.0 - June 2026**

    Minor release. Green light code changes, no major functionality changes

-   **Version 2.6.1 - March 2026**

    Data from Opportunity Marketplace is now available via the Manager Insights API.

-   **Version 2.5.0 - December 2025**

    Change summary: The scope for change for this OPM release it is mostly implementation of directives, Test failure fixes, and Minor fixes for defect.

-   **Version 2.4.0 - August 2025**

    Minor fix: implementing role selection drop down when creating as aspiration from an active job description.

-   **Version 2.3.1 - May 2025**

    Added approval flows for opportunity creation.

-   **Version 2.2.2 - February 2025**

    Opportunity Marketplace has been improved to support users who are also leveraging systems where work is being defined and executed. In this release we are providing integration with Strategic Portfolio Management to simplify the creation of projects in the marketplace by leveraging records that exist in SPM. Users who have under resourced projects in SPM can select those projects during opportunity creation in the marketplace and publish them to employees in the marketplace. Employees can view and apply for the projects in the same process as with any other marketplace opportunity.

-   **Version 2.1.3 - December 2024**

    This release has focused on efficiencies in recommendation processing and search in order to reduce page load times throughout Opportunity Marketplace.

-   **Version 2.1.1 - November 2024**

    Internal Jobs feature has been improved to allow customers more flexibility in their job posting process. We have introduced the new Hiring Core Job Posting table which will store the job posting data and will be the point of integration with customer ATS solutions. This update provides customers leveraging Job Requisitions the flexibility to create multiple job postings per requisition. It also allows customers to create job postings without any reference to Job Requisitions. Job Posting IDs are now available to be presented to employees on the Job Posting UI.

-   **Version 2.0.4 - August 2024**

    New: Introducing custom types of opportunities, where owners can create opportunities based on types like Gig, Project, Volunteer etc.

-   **Version 1.1.4 - May 2024**

    AI Search can now be enabled to support internal jobs both within the Opportunity Marketplace as well as when doing a global search for Internal Jobs. Customers that have enabled AI Search are now able to enable it for Opportunity Marketplace as the default search if they choose to.

-   **Version 1.0.6 - February 2024**

    Opportunity Marketplace transforms internal mobility, providing employees with a unified hub to explore career opportunities. With customizable match preferences and seamless ATS integration, it changes engagement and retention by facilitating transparent and efficient career development within your organization.
