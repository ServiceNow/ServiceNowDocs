# GRC: Business User - Lite release notes

Version history for the GRC: Business User - Lite application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 18.0.0 - February 2024**

    Fixed: Report view ACL fixes.

-   **Version 16.0.0 - May 2023**

    Fixed: Migrated scoped app packaging to new parent pom to remove duplicated content.

-   **Version 15.0.3 - August 2022**

    New: Move the sn\_irm\_shared\_cmn related files in GRC: Business user lite to if folder

-   **Version 14.2.0 - May 2022**

    Fixed: Issue causing Security Admin to be unable to remove the business user lite role

-   **Version 13.0.5 - September 2021**

    This application installs the 'GRC Business User - Lite' role which can be assigned to users. This role enables the users to respond to or create certain types of tasks. This helps in ensuring that the first line employees are a part of risk and compliance thus making GRC more pervasive throughout the organization. The 'GRC Business User - Lite' is available only to customers with 'Risk Lite Operator' license.
