# Advanced Work Assignment for Sourcing and Procurement Operations release notes

Version history for the Advanced Work Assignment for Sourcing and Procurement Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.1.1 - December 2025 \(Zurich\)**

    Changed: Users with the Admin role can now assign individual users or user groups to Purchase Order Exceptions based on conditions defined on the exception or related PO fields.

-   **Version 2.3.0 - May 2024**

    Minor fixes.

-   **Version 2.5.0 - February 2024**

    Minor fixes.

-   **Version 2.0.1 - November 2023**

    Minor fixes.

-   **Version 2.0.0 - August 2023**

    Minor fixes.

-   **Version 1.0.2 - May 2023**
    -   Initial release: Provides configurations to support automatic routing, queuing, and assignment of procurement cases, emails, and live agent chat conversations.
        -   Procurement administrators can configure service channels to capture the incoming work. For example, an administrator can configure or enable live agent chat. This allows shoppers to chat with procurement agents. Similarly, a procurement agent can enable case creation from email. In such cases, when shoppers send their queries in emails, case are automatically created.
        -   Procurement administrators can configure queues for a service channel. This gives the administrator the ability to route the incoming work to an appropriate agent who can then respond to the queries and resolve the issues.
        -   Procurement administrators can define a custom service channel, add an existing table, like for example Purchase Requisition, and route the incoming purchase requisitions to appropriate agents or fulfillers. This capability is not restricted to support just default configured service channels but can also be extended to custom service channels.

            **Note:** Existing customers will not be impacted as this is a new functionality that they can choose to use.
