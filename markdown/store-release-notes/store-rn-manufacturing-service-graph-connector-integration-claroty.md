# Service Graph Connector Integration for Claroty CTD release notes

Version history for the Service Graph Connector Integration for Claroty CTD on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.8 - May 2025**
    -   New:
        -   Support creation of firmware records in firmware table.
        -   Prevent import of empty rack slots into CMDB.
        -   Introduce new table to store class mappings for Claroty CTD SGC
        -   Update the SGC to support the new ire\_criterion\_attribute on the OT entity table.
        -   Supports Claroty CTD 5.1 APIs as well.
-   **Version 2.1.5 - February 2025**

    New: Fix to clean the serial record entries from the sys\_object\_source table using a fix script.

-   **Version 2.1.4 - November 2024**

    Fixed: Updated with various bug fixes.

-   **Version 2.1.3 - September 2024**
    -   New:
        -   Map the Hardware and OT Control Module classes into columns with the updated transform map configuration
        -   Certified for Xanadu
-   **Version 2.1.2 - August 2024**
    -   Map the Hardware and OT Control Module classes into columns with the updated transform map configuration
    -   Certified for Xanadu
-   **Version 2.1.0 - February 2024**
    -   Added enhanced troubleshooting capabilities
    -   Certified for Washington DC
-   **Version 2.0.4 - December 2023**

    Fixed: Issue with the flow action when users select a specific MID server while establishing the connection.

-   **Version 2.0.2 - August 2023**
    -   Certified for Vancouver
    -   ServiceNow Operational Technology \(OT\) Certified
-   **Version 2.0.1 - February 2023**

    New: Operational Technology \(OT\) Certified for Tokyo and Utah.
