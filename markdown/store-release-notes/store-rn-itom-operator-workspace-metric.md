# Operator Workspace Metric Configuration release notes

Version history for the Operator Workspace Metric Configuration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 19.1.3 - January 2021**
    -   The metrics tab displays the metrics on the CI. The user is able to view:
        -   Metric values in a time range before and after the alert creation
        -   Dynamic threshold as calculated by the system
        -   Mutiple metrics on the same chart
    -   The metric details tab is configurable. The Event Management admin is able to modify the metrics which are visualized for the CI type.
