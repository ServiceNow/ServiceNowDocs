# Digital Signature API release notes

Version history for the Digital Signature API application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 26.0.0 - August 2024**

    Fixed: Security fixes.

-   **Version 25.0.0 - February 2024**

    Fixed various minor issues.

-   **Version 22.1.0 - November 2022**

    No customer-impactful changes to Digital Signature API. Digital Signature Component has added support for internationalization and localization.

-   **Version 22.0.2 - August 2022**

    A common API that can be used to get and set saved user signatures from the now platform. The behavior of the API can be customized through script extension points. Default behavior is provided when e-signature plugin is installed.
