# Client Software Distribution 2.0 release notes

Version history for the Client Software Distribution 2.0 on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.7.3 - June 2026**

    Non Glide Cobalt Raven ACLs added.

-   **Version 1.6.0 - April 2026**

    New: Agents and Agentic workflows for MECM

-   **Version 1.5.1 - February 2026**

    New: AI Agents and Agentic workflows for group management and catalog item creation.

-   **Version 1.4.0 - December 2025**

    New: Dot walking directive for CSD 2.0.

-   **Version 1.3.0 - November 2025**
    -   New: Granular read-only controls
    -   Fixed: Subflow changes for revoke installed client software for Intune &amp; Jamf
-   **Version 1.2.3 - July 2024**
    -   Fixed: Security defect on Table ACL Bypass via GlideRecord Encoded Query
    -   Fix for OOB non-client callable script include: CSDUtil is calling current object.
-   **Version 1.2.2 - March 2024**
    -   Fixed:
        -   Extend Lease UI Action displaying error message
        -   Lease start Date throwing an error message with valid data
-   **Version 1.2.1 - September 2023**
    -   Fixed:
        -   End-user is not able to revoke software installation.
        -   Label in flow Revoke Client Software" steps are not shown properly.
        -   Date/Time formats not handled in SP Catalog Client Scripts.
-   **Version 1.2.0 - January 2023**

    New: This version includes support to uninstall apps for SCCM, that were installed without using Client Software Distribution 2.0.

-   **Version 1.1.1 - December 2022**

    Patch release of Client Software Distribution 2.0 for IntegrationHub. This version adds localization support for the application.

-   **Version 1.1.0 - September 2022**

    Changed: Minor release of Client Software Distribution 2.0. This version includes the integration changes for third-party software provider Microsoft Intune.

-   **Version 1.0.3 - July 2022**

    Fixed: Patch version of Client Software Distribution. This version includes a fix with the scheduled job "Reconcile Requested Software".

-   **Version 1.0.2 - February 2022**

    Requests for desktop and mobile software applications are one of the most common IT tickets, burdening IT support with manual, repetitive work. Client Software Distribution 2.0 \(CSD 2.0\) enables administrators to stop the productivity drain by automating these requests from the service catalog using third-party management systems to unlock IT support productivity and deliver fast, 24x7 self-service resolution. Client Software Distribution 2.0 \(CSD 2.0\) is a major new release that modernizes and expands support to automate software requests on a broader range of Microsoft endpoints and now, Apple devices with out-of-the-box support for: Microsoft Endpoint Configuration Manager for Windows 10/11, Windows 365, and Azure Cloud VMs Jamf for MacOS and iOS CSD is better together with Software Asset Management \(SAM\) Pro – automate software installation and removal with CSD; manage risk and compliance and control costs with SAM!
