# Oracle Compute Engine spoke release notes

Version history for the Oracle Compute Engine Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.4 - December 2022**

    Changed: App dependency requirement.

-   **Version 1.0.3 - September 2022**

    Patch version of Oracle Compute Engine Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.2 - March 2022**

    New: Includes missing dependency on Oracle Cloud IAM Spoke.

-   **Version 1.0.1 - July 2020**

    New: Patch release of the Oracle Compute Engine spoke for IntegrationHub. This version adds KMF modules for additional security of the password2 fields and Rome compatibility.
