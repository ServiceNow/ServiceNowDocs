# Oracle Object Storage Management spoke release notes

Version history for the Oracle Object Storage Management Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.2 - March 2022**

    New: Includes missing dependency on Oracle Cloud IAM Spoke.

-   **Version 1.0.1 - July 2021**

    Patch release of the Oracle Object Storage Management spoke for IntegrationHub. This version adds KMF modules for additional security of password2 fields and Rome compatibility.

-   **Version 1.0.0 - December 2020**
    -   New:
        -   Bucket Management
            -   Create Bucket
            -   Delete Bucket
            -   Get Bucket
            -   Look up Buckets
            -   Re-Encrypt Bucket
            -   Update Bucket
        -   Namespace Metadata
            -   Get Namespace
        -   Object Management
            -   Copy Object
            -   Create or Update Object
            -   Delete Object
            -   Get Object
            -   Look up Objects
            -   Re-Encrypt Object
            -   Rename Object
            -   Restore Object
