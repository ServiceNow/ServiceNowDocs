# Google Cloud SQL spoke release notes

Version history for the Integration Hub Google Cloud SQL spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.2 - September 2023**

    Fixed: The license requirements.

-   **Version 1.0.1 - September 2022**

    Fixed: Patch version of Google Cloud SQL Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.0 - July 2020**

    Provides actions to automate the management of Google Cloud SQL.
