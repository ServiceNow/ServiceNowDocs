# Retail Task Management Core release notes

Version history for the Retail Task Management Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - February 2025**
    -   The Retail Task Management core contains all the task management features from a retail perspective.
    -   The latest version introduces the capability for a Multi Store Case Creation framework.
    -   The Multistore Case Creation framework enables the creation of multiple child cases for separate stores under a parent case. This is designed for executing tasks such as inventory counting or product price changes, where each store needs to perform a similar set of tasks to achieve a common objective. It facilitates seamless two-way communication and visibility between stores and headquarters, enhancing customer experience, improving staff productivity, and reducing costs.
