# Sales Agreement Data Model release notes

Version history for the CSM Sales Agreement Data Model application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 8.1.0 - June 2026**

    Introduced new Query range ACLs

-   **Version 8.0.0 - December 2025 \(Yokohama, Zurich\)**

    New: Introduced granular admin roles for enhanced security.

-   **Version 7.1.0 - August 2025 \(Yokohama, Zurich\)**

    New: Enabled access to sales agreements data on portals for external users

-   **Version 6.1.0 - August 2025 \(Xanadu\)**

    New: Enabled access to sales agreements data on portals for external users.

-   **Version 7.0.0 - May 2025 \(Yokohama\)**

    Bug fixes.

-   **Version 6.0.0 - May 2025 \(Washington DC, Xanadu\)**

    Bug fixes.

-   **Version 4.0.0 - February 2025**

    New: REST APIs to enable importing of sales agreements from external systems.

-   **Version 2.0.0 - November 2024**

    New: Minor user experience changes.

-   **Version 1.0.1 - August 2024**

    The application provides the data model to store sales agreements.
