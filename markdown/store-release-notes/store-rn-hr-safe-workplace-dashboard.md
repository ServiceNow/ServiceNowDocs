# Safe Workplace Dashboard release notes

Version history for the Safe Workplace Safe Workplace Dashboard application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

**Note:** To automatically install all the Safe Workplace applications at once, download the ServiceNow® Safe Workplace suite application.

## Version history

    Fixed: Security fixes.

-   **Version 1.40.0 - February 2024**

    Removed: The Workplace PA dashboard has been deprecated. Customers who have downloaded the 1.39.2 or any older version will continue to have the dashboard and can continue to use it. Customers who have downloaded the 1.39.3 or newer version of the app will NOT have the dashboard.
