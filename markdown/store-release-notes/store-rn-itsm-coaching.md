# Coaching release notes

Version history for the ITSM Coaching application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 9.8.1 - March 2026**

    Fixed: Granular admin roles.

-   **Version 9.8.0 - December 2025**
    -   Bug fixes
    -   Security uptakes
    -   Granular role uptakes
-   **Version 9.7.1 - August 2025**

    Minor bug fixes.

-   **Version 9.7.0 - May 2025**

    Minor bug fixes.

-   **Version 9.6.2 - November 2024**

    No Major feature changes. Contains minor changes and bug fixes.

-   **Version 9.5.0 - August 2024**

    Fixed accessibility issues in this release.

-   **Version 9.4.0 - May 2024**

    Fixed: WCAG bugs.

-   **Version 9.2.0 - February 2024**
    -   Changed:
        -   Migrating coaching dashboards to UIB
        -   Migrating Trainee dashboards to UIB
    -   Fixed: WCAG issues
-   **Version 9.1.0 - November 2023**
    -   New:
        -   Uptook SRP WFO for Contextual sidebar
        -   Coaching Assessment \(SOW\) Accessibility issues Migrated Coach assessment card list to Layout 3.0.
    -   Fixed: 'Skills' and 'Courses' in coaching assessments are not displayed is fixed.
-   **Version 9.0.1 - August 2023**
    -   New:
        -   Migrate to Page layout 3.0
        -   Limit skills to 50 as default and make Skills filter persistent
        -   Uptake Applicable groups on Coaching
    -   Fixed: Skill matrix scaling and the low performance issues
-   **Version 8.1.5 - August 2023**

    Fixed: Skill matrix scaling issue and; performance improvement.

-   **Version 8.3.0 - February 2023**

    New: Create demo data for Assessments, Coaches, surveys, and learning content

-   **Version 8.1.4 - September 2022**

    New: Added dynamic error message for Skill matrix, so users who encounters payload size issues can apply filters.

-   **Version 8.1.3 - August 2022**
    -   The ServiceNow Coaching application lets you facilitate the coaching of employees using coaching assessments and coaching opportunities that you can configure for a critical moment in a process.
    -   Coaching consistently improves employee performance and processes by providing real-time bi-directional improvement feedback as well as automated contextual recommendations.
