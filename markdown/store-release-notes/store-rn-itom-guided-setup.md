# ITOM Guided Setup - New release notes

Version history for the ITOM Guided Setup - New application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 27.3.1 - June 2026**

    New: The Learn phase of the Discovery Guided Setup now includes a task, Set up ITOM faster with the new Configuration Console, that introduces the ITOM Configuration Console. The Configuration Console offers a faster way to install and update apps, with interactive configuration pages, task assignment, and automated implementation using Now Assist.

-   **Version 27.2.1 - December 2025**
    -   New: Added videos to show the instance setup process for Event Management and Discovery
    -   Changed:
        -   Improved content for clarity and better linking
        -   Moved menu items for the guided setup to the respective folders for Event Management and Discovery
-   **Version 27.0.2 - October 2025**
    -   The ITOM Guided Setup application provides a structured onboarding experience to help users quickly implement and realize value from IT Operations Management \(ITOM\). It guides users through setup steps and recommends best practices, such as starting with an initial service and scaling capabilities over time to cover more advanced use cases and a broader deployment.
    -   Path to Success: Learn, Crawl, Walk, Run
        -   Learn: Understand foundational ITOM concepts, including key benefits, common use cases, and where to find resources.
        -   Crawl: Begin initial configuration tasks, such as setting up your instance to support each application by installing plugins and assigning key roles.
        -   Walk: See the tools in action and make them your own including creating discovery schedules, triaging alerts using Express List and more.
        -   Run: Optimize the applications for your business by improving key performance indicators and enabling advanced use cases, then deploy to production.
