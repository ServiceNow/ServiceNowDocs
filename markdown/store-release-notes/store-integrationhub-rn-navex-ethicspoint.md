# Navex EthicsPoint spoke release notes

Version history for the Navex EthicsPoint Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - December 2025**

    Minor bug fixes.

-   **Version 1.0.2 - February 2024**

    Fixed: Attachments are not getting fetched in Navex Integration in Washington instances.

-   **Version 1.0.1 - December 2023**
    -   Fixed:
        -   Missing MAP's causing connection failure for navex with error - "Access Denied to cryptographic module 'global.com\_snc\_core\_automation\_connection\_credential\_glideencrypter".
        -   All metadata actions should be marked active false.
-   **Version 1.0.0 - August 2022**

    Navex Ethicspoint Spoke enables you to pull employee relation cases from a Navex Ethicspoint instance into a ServiceNow instance.
