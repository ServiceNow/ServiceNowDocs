# Threat Intelligence Security Center integration with Shodan release notes

Version history for the Threat Intelligence Security Center integration with Shodan application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - June 2026**

    Changed: Implemented Query Range ACL.

-   **Version 1.0.8 - March 2026**

    Fixed: Resolved an issue that caused observable enrichment to fail when processing URLs containing an IP address.

-   **Version 1.0.7 - May 2025**

    Fixed: Shodan integration logo was not displaying correctly.

-   **Version 1.0.6 - March 2025**

    Changed the application name.

-   **Version 1.0.5 - November 2024**

    Changed: The application encounters error codes such as 404 from Shodan API, the user experience is now improved by appending meaningful messages such as 'No Observables found in Shodan'.

-   **Version 1.0.3 - August 2024**

    New: Pushed changes to support localization.

-   **Version 1.0.2 - May 2024**

    Shodan is a search engine for Internet-connected devices that analyzes service banner information from connected devices all around the globe. Service banners include information about a computer system, such as host name, device type, operating system, geographic location, and connected ISP. When integrated with Threat Intelligence Security Center, this service banner information provides additional enrichment data and insight for Observable investigations.
