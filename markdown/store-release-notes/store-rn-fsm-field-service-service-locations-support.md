# Field Service with Service Locations support release notes

Version history for the ServiceNow® Field Service with Service Locations support application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 30.1.1 - May 2026**

    Fixed minor issues to enhance the user experience across the workspace and Service Portal.

-   **Version 30.1.0 - April 2026**

    Migration to store.

-   **Version 30.0.3 - March 2026**
    -   Field Service with Service Locations Support lets users create a new service location directly from a work order or work order task when the location doesn't already exist in the system. Work continues without interruption, and no one has to leave the workflow to set up the location separately before proceeding.
    -   Service locations created in context are immediately available for planning and execution, keeping location data complete and accurate without requiring pre-configuration or admin intervention.
