# Metric Explorer for Operator Workspace release notes

Version history for the Metric Explorer for Operator Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.0.1 - May 2023**
    -   Fixed issues:

        -   Show alerts on CI doesn't work properly if you are in Q4 range \(Oct - Dec\)
        -   Unable to get confidence band
        -   POL\_FIT\_FINISH POL OFF Metric title is always "metrics"
-   **Version 19.0.7 - October 2021**

    Metric Explorer for Operator Workspace enables operators to investigate metric behavior and compare and correlate different metrics across hosts, applications and services, in order to troubleshoot issues and find their root cause.
