# Content Pack for CMDB release notes

Version history for the Content Pack for CMDB application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.0 - March 2026**
    -   New:
        -   Created new tables to hold SGC Mapping Data content \(precomputed metadata content for IH-ETL mappings of SGCs\).
        -   Added SGC Mapping Data content to the tables.
        -   Added scripts to populate content into the tables and to allow script-based retrieval of content.
-   **Version 1.0.1 - May 2025**

    Minor bug fixes.

-   **Version 1.0.0 - November 2024**

    Content Pack for CMDB includes a collection of knowledge articles that are utilized by Generative AI to provide recommendations for diagnosing issues within the Service Graph Connectors from the Now Assist panel.
