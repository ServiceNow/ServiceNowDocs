# Roadmunk spoke release notes

Version history for the Roadmunk Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.7.1 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.6.6 - March 2026**

    Fixed: Connection validation.

-   **Version 1.6.5 - July 2024**

    Fixed: Import Roadmunk ideas and Roadmunk ideas modules.

-   **Version 1.6.4 - May 2024**

    Incorporated API change for Account action.

-   **Version 1.6.3 - August 2023**

    In V1.6.3, the Deactivate user action is fixed.

-   **Version 1.6.1 - May 2023**

    Fixed: Defined "report\_view" ACL on columns containing "sensitive" information, limiting the audience capable of using these columns as aggregate in reports.

-   **Version 1.6.0 - March 2023**
    -   Added 50+ new actions
    -   Added remote table
    -   Added authentication template
    -   Added sample outbound flow
-   **Version 1.0.2 - September 2022**

    Fix in Roadmaps Stream action.

-   **Version 1.5.0 - September 2022**
    -   Added 50+ new actions
    -   Added remote table
    -   Added authentication template
    -   Added sample outbound flow
-   **Version 1.0.1 - August 2021**

    New: Initial release of the Roadmunk spoke. Use this spoke to analyze usage so that you can reclaim subscriptions and determine your potential savings.
