# Sourcing and Procurement Operations Integration with Oracle release notes

Version history for the Sourcing and Procurement Operations Integration with Oracle application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - December 2025**
    -   Changed: Added granular roles to system properties in accordance with new directives
    -   Fixed: Removed hard-coded timeout values to improve configurability and system flexibility.
-   **Version 1.0.1 - August 2025**

    Fixed: Update all error messages in the script include to use "gs.getMessage" for translation.

-   **Version 1.0.0 - October 2024**
    -   This application provides you with the ability to:
        -   Create or update or cancel purchase order in Oracle EBS
        -   Create goods receipt in Oracle EBS
        -   Oracle EBS - Update Plugin.xml for SPO to include the dependencies
