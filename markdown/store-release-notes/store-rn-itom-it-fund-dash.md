# ServiceNow IT Fundamentals Dashboard release notes

Version history for the ServiceNow® IT Operations Management - IT Fundamentals Dashboard on the ServiceNow Store.

**Important:** Starting with the Rome release, IT Fundamentals Dashboard is being prepared for future deprecation. It will be hidden and no longer activated on new instances but will continue to be supported. For details, see the [Deprecation Process \[KB0867184\]](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB0867184) article in the Now Support knowledge base.

## Version history

-   **Version 1.0.0 - March 2018**
    -   Health report for CMDB which includes newly created CIs, accuracy of CIs, potential problems of CIs, CMDB Health scorecard \(showing Completeness, Correctness and Compliance of CIs\), and Operational CIs by Discovery Source.
    -   Health report of Incidents which includes critical/high incident count, incident types, average MTTR for P1/P2, and total incidents with and without CIs.
    -   Health report of Change Requests which includes emergency change rate, incidents created from change request and total change requests with and without CIs.
