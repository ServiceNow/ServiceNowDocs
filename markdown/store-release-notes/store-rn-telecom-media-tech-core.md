# Telecommunications, Media and Technology Core release notes

Version history for the Telecommunications, Media and Technology Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.3.0 - February 2025**
    -   Updated columns in Inbound queue 'sys\_ux\_list' lists in workspace
    -   Added "Inbound queue read" roles to above lists.
    -   Updated the display field for IBQ record
-   **Version 6.0.0 - November 2024**
    -   Data model Changes: Table Relationship Diagram
    -   ACL Changes: ACLs / Roles
    -   Archival Policies
-   **Version 5.1.0 - August 2024**

    Changed: API helper classes.

-   **Version 5.0.0 - February 2024**
    -   New:
        -   Use inbound queue table to support producer notification framework
        -   Localization support
    -   Fixed: Null exception handling in api response
-   **Version 4.0.0 - November 2023**
    -   New:
        -   Enable the inbound queue table to support multiple records types.
        -   Disable updating order date in product and service order patch API.
    -   Fixed: External id validation for related party in POST and PATCH order API.
-   **Version 3.0.0 - August 2023**
    -   New
        -   Build data model to support TMF 641 outbound request and common functionality of TMF 641
    -   Fixed
        -   List API is returning 404 for query related to account.sn\_tmt\_core\_external\_id query param
-   **Version 2.0.0 - February 2023**

    Changed: Internal optimization to align to NOW engineering standards

-   **Version 1.0.1 - November 2022**

    Changed: Internal optimization to align to NOW engineering standards

-   **Version 1.0.0 - August 2022**

    New: Core application for Service Management applications for Telecom, Media and Technology
