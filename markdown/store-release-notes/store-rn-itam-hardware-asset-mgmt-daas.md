# Hardware Asset Management for DaaS release notes

Version history for the Hardware Asset Management for DaaS application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 12.1.1 - March 2026**

    In this version, DaaS HAM – Read-Only Column Handling columns set as read-only in DaaS HAM would have their Read-Only attribute clearly defined as either "Client Script Modifiable" or "Strict Read Only".

-   **Version 12.0.0 - February 2025**

    In this version, the security for client callable scripts is improved.

-   **Version 11.0.0 - August 2024**

    Device as a Service \(DaaS\) offerings provide assets to customers on a subscription basis, delivering greater flexibility to scale up and down versus traditional leasing. DaaS providers cangain significant advantages to their internal operations by managing assets on a single platform. DaaS providers can optimize their asset inventory from sourcing through disposal to support their DaaS subscription models. By consolidating all DaaS activities and information onto one platform, DaaS providers can improve efficiency and, thereby, create better customer experiences.
