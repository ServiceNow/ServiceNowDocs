# Rich Text Editor Component for Security Operations release notes

Version history for the Rich Text Editor Component for Security Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.1 - June 2026 \(Australia\)**

    Fixed: Resolved a WCAG 1.4.3 contrast ratio issue where hyperlinks in rich text content were not meeting the 4.5:1 minimum contrast requirement in dark themes.

-   **Version 2.0.0 - December 2025**

    Fixed: Number is shown instead of display value when selecting records using slash commands.

-   **Version 1.2.21 - September 2025**

    Fixed: Count click for Link Opening under Analyst Workspace tab.

-   **Version 1.2.2 - August 2025**

    New: Rte component incorporating start date and end date, count feature, dynamic record mention.

-   **Version 1.1.2 - August 2025**

    New: Rte component incorporating start date and end date, count feature, dynamic record mention.

-   **Version 1.2.1 - May 2025 \(Xanadu\)**

    Fixed: One bug has been addressed and resolved as part of this release.

-   **Version 1.1.1 - May 2025 \(Washington DC\)**

    Fixed: One bug has been resolved as part of this release.

-   **Version 1.2.0 - February 2025**

    Changed: Updates for Security Incident Response - Daily status reporting.

-   **Version 1.1.0 - February 2025**
    -   Changed:
        -   Updates for Now Assist for Security - Now Assist for Context Menu \(NACM\) changes.
        -   Updates for Security Incident Response reporting changes.
-   **Version 1.0.5 - November 2024**

    Fixed: An issue where users were unable to insert security incidents/ tasks and their fields with auto-complete on the ServiceNow AI Platform Xanadu release.

-   **Version 1.0.4 - October 2024**

    Fixed: On Xanadu, the RTE component was not loading.

-   **Version 1.0.2 - August 2024**

    Changed: This version has fixes for the issues in the Xanadu environment.

-   **Version 1.0.0 - May 2024**

    Rich Text Editor Component for Security Operations is a dependency-only plugin that provides the Rich Text Editor Component for Workspace.
