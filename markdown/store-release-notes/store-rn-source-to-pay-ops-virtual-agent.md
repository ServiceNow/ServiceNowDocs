# Virtual Agent for Source-to-Pay Operations release notes

Version history for the Virtual Agent for Source-to-Pay Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.11.0 - August 2025**

    Includes pre-built Virtual Agent topics for sourcing and procurement. Employees can search for products, raise sourcing requests, access procurement knowledge, and complete related tasks through a conversational interface.
