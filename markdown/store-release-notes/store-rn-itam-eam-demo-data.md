# EAM Demo Data release notes

Version history for the ServiceNow® EAM Demo Data application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.2.1 - May 2026**
    -   Fixed: Demo data for EAM calibration attributes has been migrated from the deprecated sn\_asset\_cmn\_service\_attribute table to the correct sn\_eam\_calibration\_attr table.
    -   Demo data enhancements:
        -   Parking Garage Stockroom updates — The EAM Parking Garage East and EAM Parking Garage West stockrooms now have the Pick Task Required option enabled. M2M channel and location relationship records have also been added for these stockrooms.
        -   Expanded location dataset — The location dataset now includes a full indoor map hierarchy, covering the Americas and the North America campus with building, floor, and room-level locations.
-   **Version 1.2.0 - March 2026**

    EAM Demo Data 1.2.0 contains data for work plans and work order templates to demonstrate the work order planning and scheduling process for the Australia release.

-   **Version 1.0.1 - December 2025**

    Quickly set up your demo environments with preconfigured enterprise asset records available in all major EAM workspaces. This application speeds up proof-of-concept and training scenarios by offering curated data sets, enabling realistic demos without manual setup. For internal use only.

-   **Version 1.0.0 - November 2025**
    -   The EAM Demo Data app provides pre-seeded, ready-to-use records that enable quick and effective demonstrations of ServiceNow Enterprise Asset Management \(EAM\) capabilities.
    -   It includes sample asset, model, and stockroom data designed to highlight key product workflows such as calibration, condition inspections, procurement, inventory, and end-of-life management.
    -   This app is intended for partners, Solution Consultants \(SCs\), and customers to install directly from the ServiceNow Store for internal demos, enablement, and evaluation purposes.
    -   The EAM Demo Data app installs on an existing EAM instance and may modify current configurations. Therefore, it should only be used for demo purposes by internal stakeholders and is not recommended for installation in production or sub-production environments. We do not take responsibility for data loss or overwriting existing data if installed without proper consideration.
