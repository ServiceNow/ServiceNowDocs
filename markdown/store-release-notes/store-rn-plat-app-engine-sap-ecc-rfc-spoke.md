# SAP ECC RFC Spoke Dependencies release notes

Version history for AP ECC RFC Spoke Dependencies on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.4.0 - April 2024**

    Added more actions to SAP ECC RFC Spoke.

-   **Version 1.0.1 - September 2023**

    Fixed: Updated the SAP Transport Request files to allow the SAP ECC RFC Spoke v2.0 or beyond to integrate with SAP ECC version EHP0 - EHP8.

-   **Version 1.0.0 - August 2023**
    -   This dependency file is built by Bristlecone Inc.
    -   To fully get advantage of the latest SAP ECC RFC Spoke capabilities, customers will be required to import these dependency files into SAP ECC system before using the SAP ECC RFC Spoke v2.0 or beyond. Simply follow the steps below:
        -   Download and unzip the attached zip file
        -   Import a Transport Request into SAP that involves ingesting:
            -   A CO file: K904097.EH8
            -   A Data file: R904097.EH8
