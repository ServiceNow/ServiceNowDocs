# BigFix Inventory spoke release notes

Version history for the Integration Hub BigFix Inventory spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.0 - June 2026**

    Non Glide Cobalt Raven ACLs added.

-   **Version 1.5.4 - September 2023**

    Fixed: Read access to sys\_language not granted with missing x-scopes.

-   **Version 1.5.3 - April 2023**

    Added localization support to the Spoke.

-   **Version 1.5.2 - March 2023**

    Changed: Added localization support to the Spoke.

-   **Version 1.5.1 - November 2022**
    -   General revision to the Spoke with the latest design patterns
    -   Added sample remote table
    -   Added sample outbound flow
    -   Added sample inbound flow
    -   Added authentication template for easy one-step setup
-   **Version 1.0.2 - September 2022**

    Patch version of BigFix Inventory Spoke for IntegrationHub. This version includes fixes to improve the installation performance of the spoke.

-   **Version 1.0.1 - February 2021**

    HCL Bix Fix 1.0.1 is a patch release to the List Hardware Inventory DataStream action \(This action earlier wasn't returning IP address correctly for a device coming from thee BigFix computers API\).

-   **Version 1.0.0 - May 2020**

    Provides actions to list and gather information from BigFix Inventory.
