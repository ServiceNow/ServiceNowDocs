# Healthcare Computerized Maintenance Management System release notes

Version history for the Healthcare Computerized Maintenance Management System on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 7.0.0 - May 2024**

    New: Improved the Onbourd Medical Device record producer to be conformant with PID uptake.

-   **Version 6.1.0 - February 2024**

    New: Minor enhancements to true up to the Washington DC release.

-   **Version 4.0.0 - November 2023**
    -   New:
        -   Updated clinical engineer workspace experience now includes:
            -   Persona specific landing page
            -   Clinical Device Management overview
            -   Clinical Device Model Managment overview
            -   Dashboard access from workspace
-   **Version 1.1.1 - February 2023**
    -   New:
        -   Changes to the workflows
        -   Improvements to user experience
