# Customer Central release notes

Version history for the Customer Central application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 30.4.1 - June 2026**

    Search enhancements for Customer History

-   **Version 30.3.1 - March 2026**

    Minor bugs fixed.

-   **Version 30.0.3 - December 2025**
    -   Customer History:
        -   Predefined Date filters are added along with Custom date filter to narrow down the results basis time period.
        -   Search Icon added for users to click and search text in search bar. \(By default search bar is hidden\)
        -   Click on search icon can display or hide search bar.
    -   Customer Information: Bug Fixes
-   **Version 28.0.9 - October 2025**

    Defect Fixes.

-   **Version 29.0.1 - August 2025**
    -   Real-Time updates in Customer History
    -   Configure background colour for icon in activity type
    -   Configure activity grouping time period to quarterly or yearly
    -   Display activities with activity creation date via configuration
    -   Customer Activity has been renamed to Customer History
    -   Security bug fix in Customer Information
-   **Version 28.0.4 - June 2025**

    Bug fixes.

-   **Version 28.0.2 - May 2025**

    Guided setup available for Customer Activity configuration

-   **Version 27.3.1 - February 2025**

    Customer Central provides important customer context into one central view for an agent. Configurable customer activities and information ensure an optimized workspace tailored to customer interactions. With all essential data readily accessible, agents can focus on resolving issues, understanding context, and taking decisive action efficiently.
