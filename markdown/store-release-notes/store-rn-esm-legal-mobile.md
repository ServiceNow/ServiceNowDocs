# Legal Mobile release notes

Version history for the Legal Mobile on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.6.1 - June 2026**

    Fixed: Minor UI fixes.

-   **Version 5.6.0 - December 2025**
    -   Changed:
        -   Added the ability to enter comments before closing a legal request. Comments are now mandatory to ensure proper documentation and audit trail.
        -   Added lookahead search capability for matter owner to search and select users when assigning legal matter tasks, eliminating the need to manually scroll through the list of users.
    -   Fixed: Legal matter and matter task assignments in Mobile Agent is now restricted user selection by assignment group.
-   **Version 5.5.1 - August 2025**

    Fixed: Added role validation during request assignment to fulfillers.

-   **Version 5.5.0 - August 2023**
    -   Changed: Dark theme support
    -   Fixed: Polaris theme issues
-   **Version 5.3.0 - February 2023**
    -   New:
        -   Legal Mobile app interfaces now match the ServiceNow AI Platform application
        -   After configuration, the Artificial Intelligence search works on mobile
-   **Version 5.2.0 - November 2022**

    Changed: Streamlined Mobile User Experience with updated themes, colors, icons, and typography.

-   **Version 5.1.0 - August 2022**

    New: Added the Create Legal Request button in the Universal Request page for the routing agents to create a legal request from a universal request to get support from the Legal department.

-   **Version 5.0.0 - May 2022**

    Fixed: Minor fixes

-   **Version 4.0.2 - February 2022**

    Changed: Legal: Mobile has been renamed Legal Mobile

-   **Version 4.0.1 - September 2021**

    Fixed: Minor fixes

-   **Version 3.3.0 - March 2021**
    -   New:
        -   Employees can use the Now Mobile app to submit and track legal requests
        -   Legal department teams can use the Now Agent mobile app to manage legal requests and matters, and approvals
        -   Both applications are native to the iOS and Android operating system
    -   Now Mobile and ServiceNow Agent apps can be downloaded on an iOS platform from the Apple App Store or on an Android platform from the Google Play Store.
