# Field Service Manager Workforce release notes

Version history for the ServiceNow® Field Service Manager Workforce application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.1.0 - April 2026**

    Fixed: External group visibility for any user

-   **Version 1.0.5 - March 2026**
    -   Field Service Manager Workforce gives field service managers a single place to see agent schedules, team calendars, work order tasks, and real-time location data so they stop piecing together team availability from multiple systems and start managing from a complete picture.
    -   Most managers span several tools just to answer basic questions: Who's available? Where are they? Is this territory covered? Workforce consolidates that across assignment groups and territories, so managers can spot conflicts, balance workloads, and confirm coverage before problems reach the dispatch queue.
    -   When Workforce Optimization for Field Service is active, managers can filter the team calendar by event type — Appointments, Case Tasks, or Work Order Tasks — keeping the view focused on what's actually relevant to their operation.
