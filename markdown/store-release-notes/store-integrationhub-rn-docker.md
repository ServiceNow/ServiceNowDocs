# Docker spoke release notes

Version history for the IntegrationHub Docker spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.4.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 2.3.4 - July 2025**

    Fixed: Deleted stale ACLs and app module.

-   **Version 2.3.3 - July 2023**

    Fixed: Issue regarding zboot errors in DMT.

-   **Version 2.3.2 - March 2023**

    Fixed: Security bug fix for Stored XSS in Docker webhook registry.

-   **Version 2.3.0 - January 2023**
    -   New: Authentication templates for spoke setup
    -   Fixed: Location changes
-   **Version 2.1.4 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 2.1.3 - February 2022**

    Changed: Aligned the Spoke input data type with that in Docker API.

-   **Version 2.1.2 - December 2021**

    Fixed: Patch release of Docker Spoke for IntegrationHub. This version includes a fix within 'Create a Volume' action and adds missing vendor information when the plugin belongs to ServiceNow.

-   **Version 2.1.1 - November 2021**

    Patch release for the Docker spoke.

-   **Version 2.0.3 - September 2021**

    Patch release for the Docker spoke.

-   **Version 1.0.3 - February 2020**
    -   Fixed:
        -   A 'Page Not Found' error has been fixed
        -   Other minor bug fixes
-   **Version 1.0.2 - August 2019**

    Licensing patch for the Docker spoke for Integration Hub
