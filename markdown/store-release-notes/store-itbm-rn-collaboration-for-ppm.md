# PPM Collaboration release notes

Version history for the PPM Collaboration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - November 2024**
    -   Changed: Client callable Script includes to be more secure.
    -   Fixed: Minor bug.
-   **Version 2.0.1 - May 2023**

    Updated for localization support.

-   **Version 2.0.0 - January 2021**

    New: Support for Microsoft Teams as a project collaboration tool to facilitate active and timely communication about the project.

-   **Version 1.0.0 - August 2020**

    Initial release. PPM Collaboration utilizes Slack as a project collaboration tool to facilitate active and timely communication about the project. The project manager, users, and any other stakeholders added to the project channel can easily stay up to date with the project status and chat with other members.
