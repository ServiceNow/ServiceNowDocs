# ITAM Health Check release notes

Version history for the ITAM Health Check application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.5 - March 2026**

    In this release, minor defects have been resolved to enhance stability.

-   **Version 3.0.4 - December 2025**
    -   In this release:
        -   A single consolidated check, 'Oracle Database Server Missing Relationship for On-Premise VMs', has been introduced. This replaces multiple individual checks for VMware, Solaris Zones, Solaris LDOM, and Nutanix.
        -   The redundant check 'Oracle Database Server Solaris Zone Hard Partition Relationship Discovery' has been removed, as its logic is now included in the unified check.
        -   Minor bugs have been resolved as part of routine stability improvements.
-   **Version 3.0.3 - October 2025**

    The results are now displayed by product in the Product results section of the License Workbench.

-   **Version 3.0.1 - August 2025**

    In this version, the ITAM Health Check application is updated so that results are displayed by product in the Product results section of the License Workbench.

-   **Version 2.3.2 - February 2025**
    -   Added new Health checks for the following configurations:
        -   Microsoft 365
        -   General
-   **Version 2.2.2 - August 2024**

    Fixed: Issues on checks for Microsoft SQL server and Microsoft 365 are fixed.

-   **Version 2.2.1 - May 2024**
    -   Added new Health checks for the following configurations:
        -   Microsoft SQL Server
        -   Oracle Java
-   **Version 2.1.6 - February 2024**
    -   Added new Health checks for the following configurations:
        -   Microsoft Windows Server
        -   Microsoft SQL Server
-   **Version 2.1.1 - August 2023**

    Created a new CMDB suite and added Config and CMDB suite checks for Microsoft SQL and Windows servers.

-   **Version 2.0.4 - February 2023**
    -   New: Added new Health checks for the following configurations:
        -   Oracle Database Server
        -   SaaS - Microsoft 365
        -   Microsoft SQL Server
-   **Version 1.0.1 - August 2022**

    Perform Software Asset Management configuration scans to determine issues with configuration or discovery, that could impact software License compliance results. Additionally, get recommendations to fix the issues and thereafter create tasks for successful resolution.
