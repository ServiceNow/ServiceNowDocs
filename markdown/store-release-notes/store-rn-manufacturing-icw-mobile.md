# ICW Mobile release notes

Version history for the ServiceNow® ICW Mobile application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.0.0 - June 2026**

    No updates or changes for this version.

-   **Version 1.0.2 - May 2026**

    This patch updates internal references to an upcoming Innovation Labs release of Industrial Loss Elimination.

-   **Version 1.0.0 - March 2026**
    -   Industrial workers perform required operations and configuration activities in a shop floor environment. Being able to keep an accurate task list and complete activities when needed without having to find a laptop or go to the control room is a key enabler of productivity for both industrial workers and the processes they support.The ICW Mobile Experience enables workers using the Now Agent app on iOS and Android mobile devices to:
        -   View outstanding tasks for the area where they work
        -   Execute assigned Industrial Guided Tasks and other workflows directly on their mobile devices
        -   Create ad-hoc task workflows based on Industrial Standards
