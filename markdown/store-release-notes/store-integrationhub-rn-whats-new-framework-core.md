# What's New Framework Core release notes

Version history for the What's New Framework Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.3 - August 2025**
    -   Fixed:
        -   Support video content from YouTube and Vimeo link
        -   Fixed the 400 API Error for the What's New Menu Page
        -   Allow Xanadu to be set as the minimum supported version of the What's New content record
-   **Version 1.1.0 - February 2025**
    -   New: What's New is now supported on Classic pages. With this release, users will be able to view What's New content \(subject to content availability\) for those application areas/pages.
    -   Fixed:
        -   Issues preventing the notifications for Whats New content availability on "?" Icon.
        -   Incorrect availability information present in the "Available" tag for each feature
-   **Version 1.0.4 - May 2024**

    Security fix.

-   **Version 1.0.3 - February 2024**

    With this app, ServiceNow pushes content about the latest features of the products, periodically, allowing practitioners to view information about the new features without the need to go through the release notes.
