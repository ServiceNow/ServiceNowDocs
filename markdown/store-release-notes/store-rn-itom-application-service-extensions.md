# Application Service Extensions release notes

Version history for the ITOM Application Service Extensions application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - March 2026**

    Enables the creation of a service type that support a unified service model. Facilitating multi-source Service Mapping by integrating data from various sources into a cohesive representation.

-   **Version 1.1.7 - November 2024**
    -   All content of reducing redundant recomputation was removed.
    -   Fixed a problem with colors on unified map.
-   **Version 1.1.5 - August 2024**

    The application offers extended features for application service functionality.
