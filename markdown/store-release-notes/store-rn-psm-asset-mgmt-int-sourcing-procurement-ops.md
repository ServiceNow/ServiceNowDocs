# Asset Management Integration for Sourcing and Procurement Operations release notes

Version history for the Asset Management Integration for Sourcing and Procurement Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.4 - June 2026**

    Defect fixes.

-   **Version 1.5.2 - March 2026**

    Changed: Improved interaction with asset management and sourcing and procurement integration by simplifying swimlane text, enhancing no‑price items, redirecting submissions to ITAM PO tables, and adding notifications.

-   **Version 1.1.1 - December 2025**

    Changed: Updated validation messages during supplier selection in the ITAM-SPO better together flow to improve clarity and accuracy.

-   **Version 1.0.1 - August 2025**

    Enables asset managers to request goods from the Shopping Hub catalog, and allows the procurement team to work on those requests.
