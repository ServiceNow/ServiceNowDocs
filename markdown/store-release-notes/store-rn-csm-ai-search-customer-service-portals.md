# AI Search integration for customer and consumer service portals release notes

Version history for the AI Search integration for customer and consumer service portals on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - December 2025**

    Changed: Added support for custom icons in the knowledge search template, allowing customizable icon rendering within GenAI search results.

-   **Version 1.0.5 - May 2023**

    Fixes.

-   **Version 1.0.2 - April 2021**

    New: Support for AI Search on CSM and CSP portals
