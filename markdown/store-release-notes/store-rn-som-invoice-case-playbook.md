# Invoice Case Playbook release notes

Version history for the ServiceNow® Invoice Case Playbook application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.1.0 - May 2026**

    New: Resume and submit invoice cases in draft state using the Process tab on the Invoice case details page.

-   **Version 1.0.4 - April 2026**
    -   Fixed:
        -   The Enter Details section was blank on invoice case creation, preventing customers from completing the form. The fields now display correctly.
        -   Modified security access for declarative actions to ensure declarative actions require customer access.
-   **Version 1.0.3 - April 2026 \(Yokohama\)**
    -   Fixed:
        -   The Enter Details section was blank on invoice case creation, preventing customers from completing the form. The fields now display correctly.
        -   Modified security access for declarative actions to ensure declarative actions require customer access.
-   **Version 1.0.1 - March 2026**

    Provides a secure and streamlined playbook experience to create an invoice case on the ServiceNow Business Portal. Empowers customers to create and manage invoice-related cases effortlessly.
