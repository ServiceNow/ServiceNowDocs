# Participant Suggestions release notes

Version history for the Participant Suggestions application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.1 - August 2024**

    Backend changes to support dynamic switching of expert finder services.

-   **Version 1.0.3 - February 2024**

    When you select participants to join an existing conversation, Sidebar displays a list of participants who may be helpful in solving a customer's issue.
