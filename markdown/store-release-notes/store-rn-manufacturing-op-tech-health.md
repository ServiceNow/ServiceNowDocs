# Operational Technology Health release notes

Version history for the Operational Technology Health application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.0 - June 2026**

    New: Certified for Australia Patch 3

-   **Version 3.1.0 - March 2026**
    -   New:
        -   You can mark an OT alert with an OT tag
        -   Certified for Australia
-   **Version 3.0.0 - December 2025**

    Modified app inter dependencies.

-   **Version 1.0.1 - August 2025**
    -   ServiceNow's Industrial Process Health product is a comprehensive solution designed to improve the performance, availability, and reliability of Industrial assets and processes. It provides real-time monitoring and analysis of the Industrial environment, enabling proactive insights and alerts that help prevent equipment downtime and production disruptions.
    -   Key features include event management, which aggregates data from various monitoring tools to identify and prioritize incidents, and production impact analysis using unified maps.
    -   By leveraging automation and AI-driven capabilities of OT Service Management, it streamlines incident management processes, reduces mean time to resolution \(MTTR\), and ensures that critical business processes remain operational and efficient, ultimately driving better asset performance and improved productivity.
