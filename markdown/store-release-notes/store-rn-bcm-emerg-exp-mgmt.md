# Emergency Exposure Management release notes

Version history for the Emergency Exposure Management on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.27.2 - June 2026**

    Fixed: Remediated non-Glide ACLs

-   **Version 1.27.0 - August 2025**

    Changed: Minor ACL updates

-   **Version 1.26.0 - November 2024**

    Security fixes.

-   **Version 1.25.0 - May 2024**
    -   Security fixes.
    -   Removed: The Field Service Safety dashboard has been deprecated. Customers who have downloaded the 1.24.0 or any older version will continue to have the dashboard and can continue to use it. Customers who have download 1.25.0 or newer version of the app will NOT have the dashboard.
-   **Version 1.24.0 - August 2022**
    -   Fixed:
        -   Minor bug
        -   Quality improvement
-   **Version 1.23.0 - January 2022**

    New: Localization for 22 languages other than English. Translations are available for features released up until the December 16, 2021 release.

-   **Version 1.22.1 - December 2021**

    Changed: Updates to the translation engine

-   **Version 1.21.0 - November 2021**

    Fixed: Minor bug

-   **Version 1.20.0 - October 2021**

    Fixed: Security bug

-   **Version 1.19.0 - September 2021**

    New: Ability to view the Field Service Safety Dashboard with new theming

-   **Version 1.18.0 - August 2021**

    Fixed: Minor bug

-   **Version 1.17.0 - July 2021**
    -   Changed: Updates to the translation engine
    -   Fixed: Security bug
-   **Version 1.16.0 - April 2021**

    New: Localization for features from the March 25 release

-   **Version 1.15.0 - March 2021**

    Fixed: Minor bugs

-   **Version 1.14.1 - March 2021**
    -   New: Localization for features from the February 18 release
    -   Fixed: Minor bug
-   **Version 1.13.0 - February 2021**

    New: Localization for features from the February 4 release

-   **Version 1.12.0 - February 2021**

    New: Localization for features from the January 21 release

-   **Version 1.11.0 - January 2021**

    New: Support for domain separation

-   **Version 1.10.0 - December 2020**

    New: Localization for features from the November 19 release

-   **Version 1.9.0 - November 2020**
    -   New: Localization for features from the November 5 release
    -   Fixed: For users in an Australian timezone, Diagnostic Request end dates now work as expected
-   **Version 1.8.0 - November 2020**

    New: Localization for features from the October 15 release

-   **Version 1.7.0 - October 2020**
    -   New: Localization for features from the October 1 release
    -   Fixed: Minor bug fixes
-   **Version 1.6.2 - October 2020**
    -   New:
        -   The new Tracing System module is a central location for admins to set up new data sources and all aspects of how to retrieve and use contact tracing data.
        -   To allow existing Emergency Exposure Management users to migrate to the new "bring your own data source" feature, v1.6.2 includes an upgrade fix script.
        -   Localization for features from the September 16 release.
    -   Changed: The Diagnostic Request form's checkboxes were replaced with a multi-select lookup field.
-   **Version 1.4.0 - September 2020**

    New: Localization for the September 3 release features

-   **Version 1.3.0 - September 2020**
    -   New:
        -   Cisco DNA Spaces error handling for Emergency Exposure Management
        -   Localization for features from the August 20 release
-   **Version 1.2.0 - August 2020**
    -   New:
        -   Support for the new ServiceNow Contact Tracing application's integration with Cisco DNA Spaces
        -   Localization for features from the August 6 release
-   **Version 1.1.0 - August 2020**

    New: Localization for features from the July 23 release

-   **Version 1.0.11 - July 2020**
    -   New:
        -   Localization for features from the July 9 release
        -   Compatible with ServiceNow instances on the Paris release
-   **Version 1.0.10 - July 2020**

    New: Localization for features from the June 18 release

-   **Version 1.0.6 - June 2020**
    -   New: Localization for Emergency Exposure Management v1.0.5 features
    -   Fixed: Unexpected behavior of 'start date' and 'end date' in Emergency Exposure Management
-   **Version 1.0.5 - June 2020**
    -   New:
        -   Integration with workplace potentially exposed user data
        -   Updated localization for:
            -   Japanese
            -   French
            -   German
            -   Spanish
            -   Italian
            -   Brazilian Portuguese
            -   Korean
            -   Simplified Chinese
-   **Version 1.0.4 - April 2020**
    -   New: The Emergency Exposure Management UI is now available in the following languages.
        -   Brazil/Portuguese
        -   French
        -   German
        -   Italian
        -   Japanese
        -   Korean
        -   Simplified Chinese
        -   Spanish
-   **Version 1.0.3 - April 2020**
    -   New:
        -   General improvements to the ‘Use Outlook meetings’ feature, including the ability to check attendee details for recurring meetings
    -   Fixed:
        -   On the Diagnostic Request form, the blue banner to ‘Select start and end dates’ no longer appears after the dates have been selected
        -   In the ‘Use Outlook meetings’ feature’s start and end dates, users can now select future dates
        -   When a user downloads a diagnostics report, the downloaded Excel spreadsheet now includes the ‘From same location’ attribute
        -   Other minor bug fixes
-   **Version 1.0.2 - March 2020**

    Enables organizations to identify employees who may have been exposed to infectious diseases, such as the COVID-19 virus, by analyzing an impacted employee's meetings and locations.
