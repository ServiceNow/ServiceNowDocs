# Status Report UI Component for MSIM Workspace release notes

Version history for the Status Report UI Component for MSIM Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.6 - June 2026**

    Fixed: Fixed the free form text fields in the preview section.

-   **Version 1.0.2 - November 2024**

    Fixed: Accessibility issues in the MSIM Workspace in tabs such as, Status Report, Collaboration, etc.

-   **Version 1.0.1 - February 2024**

    Changed: Migrated Status Report workspace page to Layout 3.0 to support accessibility.

-   **Version 1.0.0 - March 2022**

    This is a UI dependency-only plugin that provides the UI component for the status reports feature in the Major Security Incident Management workspace.
