# Now Assist for Workplace Service Delivery \(WSD\) release notes

Version history for the Now Assist for Workplace Service Delivery \(WSD\) application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.13 - June 2026**

    None.

-   **Version 1.1.8 - March 2026**

    The release notes of the related plugins can be viewed on the respective plugin's release notes.

-   **Version 1.1.5 - December 2025**

    The release notes of the related plugins can be viewed on the respective plugin's release notes.

-   **Version 1.1.3 - October 2025**

    Fixed issues.

-   **Version 1.0.5 - October 2025**

    Fixed issues.

-   **Version 1.1.2 - September 2025**
    -   New:
        -   The introduction of Workplace AI Agents
            -   Optimize cleaning schedules \(cancel or create cases\) based on expected attendance rates.
            -   Identify sources automatically and suggest floors when importing floorplans to reduce manual intervention.
    -   Fixed: Security fixes
-   **Version 1.0.3 - June 2025**

    Supported for Xanadu Patch 9.

-   **Version 1.1.0 - May 2025**
    -   New:
        -   The introduction of Workplace AI Agents
            -   Support Event planners move planned reservations when a space becomes unavailable.
            -   Provide space suggestions to Event Planners when an employee requests changes to a managed reservation.
-   **Version 1.0.0 - February 2025**

    Now Assist for Workplace Service Delivery elevates your ServiceNow Workplace Services experience with generative AI capabilities. Transform the employee experience by enabling seamless interaction with physical workspaces through generative AI technology.
