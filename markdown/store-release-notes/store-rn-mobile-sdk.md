# Mobile SDK release notes

Version history for the Mobile SDK application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.0 - June 2026**
    -   Fixed: Corrected query\_range ACLs across Mobile SDK tables
    -   New: Added backend support for NowVoice SDK module to retrieve active Now Assist voice deployment configuration
    -   Removed: Removed NowAnalytics
-   **Version 2.2.0 - March 2025**

    Added ability to enable Now Assist in SDK.

-   **Version 2.0.0 - November 2023**

    The ServiceNow Mobile SDK enables you to use the capabilities of the ServiceNow platform within your standalone mobile apps.
