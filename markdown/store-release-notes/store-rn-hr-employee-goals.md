# Employee Goals release notes

Version history for the Employee Goals application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - June 2026**

    Reviewed Non-Glide Cobalt Raven ACLs

-   **Version 1.2.0 - March 2026**

    Fixed a defect to add boolean for addEncodedQuery while using GlideRecordSecure

-   **Version 1.1.1 - December 2025**

    This release contains bug fixes. No customer-facing changes are included.

-   **Version 1.1.0 - May 2025**
    -   New: API to retrieve the employee's goals summary for inclusion in the Employee Summary as part of Employee Persona Now Assist feature
    -   Changed: Coral Theming improvements
-   **Version 1.0.5 - February 2025**
    -   Fixed:
        -   RCA on creating a draft growth plan when employee goals plugin is installed.
        -   White background fix behind the goals title in the flyout
-   **Version 1.0.3 - August 2024**

    Fixed: Improved scope security.

-   **Version 1.0.1 - May 2024**

    Employee Goals provides employees and their managers visibility into employee personal and business goals, in context of Talent Development activities.
