# Oracle EBS Spoke Dependencies for Oracle Database 19c or later spoke release notes

Version history for the Integration Hub Oracle EBS Spoke Dependencies for Oracle Database 19c or later spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 1.0.0 - April 2026**

    This is the required dependency application to enable OEBS Spoke actions to interact with OEBS application on Oracle Database 19c or later accordingly. This dependency application creates an OEBS application package that contains the stored procedures, which expose the associated tables and their columns as web services for OEBS Spoke actions to interact with.
