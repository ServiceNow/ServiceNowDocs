# Risk Assessments Integration for Sourcing and Procurement Operations release notes

Version history for the Risk Assessments Integration for Sourcing and Procurement Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.9.0 - March 2026**

    New: Introduced support for read-only dictionary entries and overrides across Spend PPM, Category Management, VRM, and ITAM modules.

-   **Version 1.5.5 - December 2023 \(Utah\)**

    Minor fixes.

-   **Version 1.5.4 - November 2023**

    Minor fixes.

-   **Version 1.5.2 - May 2023**

    Changed: Renamed this application to Risk Assessments Integration for Sourcing and Procurement Operations. This application was formerly named Purchase Automation Integration with Risk Assessment.

-   **Version 1.5.1 - August 2022**

    Minor fixes.
