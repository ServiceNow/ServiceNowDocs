# Cybersecurity Executive Dashboard release notes

Version history for the Cybersecurity Executive Dashboard application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.7.0 - June 2026**

    Changed: Reduced excessive administrative permissions from Vulnerability Response role profiles to better align with least-privilege access control principles.

-   **Version 30.1.1 - March 2026 \(USEM\)**

    Fixed: Made minor performance improvements, including latency optimizations.

-   **Version 2.4.5 - January 2026**

    Fixed: Security Score now accurately reflects changes to Security Posture Rollup weightages in the Cybersecurity Executive Dashboard.

-   **Version 30.1.0 - January 2026 \(USEM\)**

    Fixed: Security Score now accurately reflects changes to Security Posture Rollup weightages in the Cybersecurity Executive Dashboard.

-   **Version 2.4.4 - December 2025**

    Minor fixes for this release.

-   **Version 2.4.3 - May 2025**

    Fixed: The Targets section now automatically refreshes to show updated targets whenever they are created, updated, or deleted. In the past, this had to be done manually. This improvement allows users to see the latest changes in real-time, improving the efficiency and accuracy of target management.

-   **Version 2.4.1 - February 2025**

    Fixed: This release features minor user experience improvements to improve usability and provide a better experience.

-   **Version 2.3.2 - November 2024**

    Minor fixes for this release.

-   **Version 2.2.1 - August 2024**

    New: Updated the PA indicators of Configuration Compliance to populate data based on newly added fields in Test Result table.

-   **Version 2.1.3 - June 2024**
    -   New:
        -   GRC Metrics Integration: Included key metrics from Governance, Risk, and Compliance, offering a comprehensive view of your organizations cybersecurity posture. A new GRC dashboard can be easily accessed from the Cybersecurity Executive Dashboard, providing seamless navigation and integration of critical risk and compliance data.
        -   Enhanced User Experience: The user experience has been completely reimagined to facilitate a more intuitive understanding of the dashboard. This enhancement ensures that security functions are easily distinguishable and the interface remains scalable to meet growing organizational needs.
        -   Operational Technology Metrics: Expanded vulnerability metrics to include those specific to Operational Technology \(OT\). This addition provides a more holistic view of security across both IT and OT environments, ensuring comprehensive risk management and monitoring.
        -   Accurate and Reliable Metrics: Several corrections have been made to existing metrics to enhance accuracy and reliability. These improvements ensure that all data reflected in the dashboards is precise, aiding in better decision-making and strategic planning.
-   **Version 2.0.1 - February 2024**

    Gain comprehensive insights into your organization's security posture with the Cybersecurity Executive Dashboard. The dashboard integrates data from all products in the Security Operations product suite, including Vulnerability Response, Application Vulnerability Response, Container Vulnerability Response, Configuration Compliance, Security Incident Response, and Major Security Incident Response. Additionally, it incorporates employee readiness data by seamlessly integrating with leading third-party phishing simulation tools such as KnowBe4 and Microsoft Defender for Office365.
