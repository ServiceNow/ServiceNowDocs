# Software Asset Management integration with Workday release notes

Version history for the ITAM Software Asset Management integration with Workday application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.14 - March 2026 \(Yokohama\)**

    In this version, support for OAuth integration and event-based processing is added.

-   **Version 1.0.12 - February 2025**

    Issue in processing contingent workers if employee type field is empty is fixed.

-   **Version 1.0.8 - August 2024**

    The SAM Workday license calculator is updated to remain compatible with the latest Reconciliation Framework changes implementation.

-   **Version 1.0.7 - May 2024**

    In this version, Workday Download Subscriptions is modified to change the data paths from personal information data to employment information data.

-   **Version 1.0.4 - January 2024**

    In this version, Workday Download Subscriptions scheduled job issue is fixed.

-   **Version 1.0.3 - November 2023**

    In this version, a few upgrade issues are fixed.

-   **Version 1.0.2 - February 2023**

    New: Compatible with Utah grouping enhancement APIs.

-   **Version 1.0.1 - August 2022**
    -   The Software Asset Management - Workday Integration is built by Bristlecone, Inc.
    -   The Software Asset Management - Workday Integration provides subflows to download subscriptions for Workday HCM and Workday financials.
