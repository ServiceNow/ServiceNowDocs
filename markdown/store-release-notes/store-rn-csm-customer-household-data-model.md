# Customer Household Data Model release notes

Version history for the ServiceNow® Customer Household Data Model application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.2.1 - May 2026**

    Defect fixes

-   **Version 2.1.0 - April 2026**

    Fixed: Defect fixes.

-   **Version 2.0.6 - March 2026**
    -   Customer Household Data Model provides a structured foundation for representing households as logical groupings of individual consumers. It enables organizations to create households, define household members, and establish relationships between members providing a comprehensive representation of household structure.
    -   By providing a consolidated view of household dynamics, the application empowers service teams to deliver more personalized, coordinated, and efficient support. It enhances visibility into shared services, improves case handling across related customers, and drives better customer outcomes through contextual awareness.
