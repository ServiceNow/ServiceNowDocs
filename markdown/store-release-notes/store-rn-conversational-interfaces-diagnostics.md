# Conversational Interfaces - Diagnostics release notes

Version history for the Conversational Interfaces - Diagnostics application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.1 - November 2024 \(Xanadu\)**

    Fixed: Dynamic Translation Support Topic is deleted and will not show in Virtual Agent anymore.

-   **Version 2.2.0 - May 2024 \(Washington DC\)**

    Minor fixes.

-   **Version 2.1.6 - February 2024 \(Washington DC\)**

    Fixed minor issues.

-   **Version 2.1.5 - August 2023**

    Fixes.

-   **Version 2.1.2 - May 2023**

    Fixed: Welcome message for Diagnsotic Pre Chat topic.

-   **Version 2.1.1 - February 2023**

    Changed: Added plugin for diagnostic search. The diagnostic content related to a plugin that isn't installed won't appear to the user in Virtual Agent.

-   **Version 2.0.2 - August 2022**

    New: Processes modified.

-   **Version 1.1.0 - May 2022**
    -   Fixed:
        -   MS Teams errors and troubleshooting flows.
        -   CI Diagnostics topic blocks.
        -   Record creation issues.
-   **Version 1.0.2 - October 2021**

    The Conversational Interfaces Diagnostic Tool enables admins and technical support engineers to diagnose common configuration and plugin setup issues by interacting with a diagnostic chatbot via Virtual Agent.
