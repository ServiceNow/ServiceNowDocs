# Operational Sustainability Management with Concur integration release notes

Version history for Operational Sustainability Management with Concur integration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.3.1 - June 2026 \(Australia\)**

    Changed: This release includes security enhancements that strengthen access controls across the application.

-   **Version 22.0.0 - March 2026**

    Changed: Updated name of the application to 'Operational Sustainability Integration with Concur'

-   **Version 21.1.1 - December 2025**
    -   New: Minor improvement to expose 'Active' field configuration for Choice options.
    -   Fixed: Resolved an issue affecting localization functionality
-   **Version 21.0.0 - August 2025**

    Changed: Refreshed demo data to better align with current feature capabilities and use case scenarios.

-   **Version 20.1.0 - May 2025**

    Fixed: Important security fixes.

-   **Version 19.1.0 - November 2024**

    Fixed: This release contains important security fixes.

-   **Version 16.0.2 - May 2023**
    -   New:
        -   Ability to configure connection and inbound webhook details to enable the integration
        -   Activate the metric definitions for collecting travel details to calculate emissions
        -   Ability to stage the incoming trip detail and process the entity and metric data
