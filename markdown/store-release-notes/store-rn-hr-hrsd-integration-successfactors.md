# HRSD integration with SuccessFactors release notes

Version history for the HRSD integration with SuccessFactors application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.1 - October 2025**

    Installing the HR Service Delivery Integration with SAP SuccessFactors application now automatically installs the required Human Resources Scoped App: Core dependency.

-   **Version 1.5.0 - May 2025**

    New: Added Subflow for VA Topic: Update Personal Details - Employees can update personal details like address, phone number, email etc through ServiceNow Virtual Agent

-   **Version 1.2.1 - November 2024**
    -   Pull employee profile data from SAP SuccessFactors to ServiceNow HR Service Delivery
    -   Sync to-dos across both SAP SuccessFactors and ServiceNow HR Service Delivery to provide more efficient service delivery and enable employees to access and complete their to-dos from either system
-   **Version 1.0.5 - February 2024**

    Minor fixes.

-   **Version 1.0.4 - November 2023**

    Fixed: Retrieve the Effective Workers Service subflow fetching duplicate records in a delta load.

-   **Version 1.0.3 - May 2023**

    Changed: This patch version has been certified with SAP SuccessFactors spoke which only supports OAuth SAML grant-type authentication.

-   **Version 1.0.2 - December 2022**

    Fixes related to HR profile and To-dos

-   **Version 1.0.1 - October 2022**

    Fixed: Included fixes for SAP SuccessFactors

-   **Version 1.0.0 - September 2022**
    -   The HR Service Delivery Integration with SAP SuccessFactors was built by BristleCone Inc.
    -   This integration synchronize employee profiles and employee to-dos between HR Service Delivery and SAP SuccessFactors.
