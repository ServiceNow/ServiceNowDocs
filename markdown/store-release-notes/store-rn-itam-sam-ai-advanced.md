# Software Asset Management AI Advanced release notes

Version history for the ServiceNow® Software Asset Management AI Advanced application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.1 - June 2026**

    New feature: Entitlement creation error handling - This feature uses AI to suggest resolutions when users upload the standard Excel import template for entitlement creation.

-   **Version 1.0.4 - April 2026**
    -   Plugin for AI features for Software Asset Management AI Advanced \(SAM AI Advanced\).
    -   SAM Advanced is ServiceNow's software asset management product for organizations that want AI built into their day-to-day SAM operations. It includes everything in SAM Pro — plus AI capabilities.
    -   This plugin has been introduced as part of the adoption of product tiers across ServiceNow product lines. This establishes a consistent approach to how AI capabilities are packaged, entitled, and delivered on the Now Platform.
