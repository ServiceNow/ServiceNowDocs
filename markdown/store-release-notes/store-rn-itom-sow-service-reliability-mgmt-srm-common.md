# Service Operations Workspace Service Reliability Management \(SRM\) Common release notes

Version history for the Service Operations Workspace Service Reliability Management \(SRM\) application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 7.0.0 - June 2026**

    Fixed a security vulnerability in Query Range Access Control Lists \(ACLs\).

-   **Version 6.6.0 - March 2026**
    -   New: Email notifications for autonomous SLO creation
    -   Fixed: MS Teams channel integration fixes
-   **Version 6.5.3 - December 2025**
    -   Changed:
        -   Remove unneeded services from SRM to keep your monitoring data relevant.
        -   Renamed application service and technical service to service instance and technology management service, aligning with current naming standards.
        -   Removed the Integrations Launchpad and Alert Automation icons from the sidebar. You can now access them under the ITOM Admin Experience icon in the Service Operations Workspace.
    -   Fixed: Fixed team governance link visibility.
-   **Version 6.4.0 - August 2025**

    Fixed: Service Operations Workspace Admin Center -- fixed SRM configuration issue.

-   **Version 6.3.3 - June 2025**

    Fixed: Addressed several post-import issues affecting Dynamic CI services.

-   **Version 6.3.2 - May 2025**

    Changed: The service registration workflow has been streamlined to make onboarding easier and now requires less effort to get services ready for operation.

-   **Version 6.0.3 - March 2025**

    Fixed: Changes to prevent overwriting existing conditional ACLs.

-   **Version 6.2.0 - February 2025**

    New: Support for Technical Service Offerings.

-   **Version 6.1.0 - November 2024**

    Fixed: Issue preventing the creation of new technical services in SRM.

-   **Version 6.0.1 - September 2024**

    Fixed: This release addresses a configuration issue in scheduled jobs to allow more control over SRM jobs execution.

-   **Version 6.0.0 - August 2024**

    Delivers common configurations for Service Reliability Management, which are stored globally, such as access control changes. This configuration is installed only as a result of enabling the Service Reliability Management application.
