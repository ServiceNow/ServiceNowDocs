# Cloud Insights Billing release notes

Version history for the Cloud Insights Billing application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.0.1 - April 2024**

    Plugin version refresh to 6.0.1 apps.

-   **Version 6.0.0 - February 2024**

    The plugin version is refreshed based on 6.0 version.

-   **Version 5.0.3 - January 2024**

    Plugin version refresh to 5.0.3 version.

-   **Version 3.1.3 - January 2024**

    Plugin version refresh for 3.1.3 release.

-   **Version 3.1.2 - June 2023**

    Fixes to address couple of customer issues.

-   **Version 3.1.1 - March 2023**

    This is a patch release which incorporates fixes.

-   **Version 4.0.0 - February 2023**
    -   New:
        -   Cloud Database Optimization
        -   Custom thresholds for optimization rules
-   **Version 3.1.0 - November 2022**

    This is a minor release which incorporates Azure API change and fixes.

-   **Version 3.0.0 - June 2022**

    New: Support for Google cloud

-   **Version 2.2.0 - November 2021**

    New: A separate app for the Cloud Provisioning and Governance product to download the billing data. Users are not required to install the full-fledged Cloud Insights app.

-   **Version 2.1.0 - September 2021**

    New: A separate app for cloud provisioning and goverance product to download the billing data. Now customers are not required to install the full fledged cloud insights app.
