# Playbooks for Field Service Management release notes

Version history for the Playbooks for Field Service Management on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.5 - August 2025**

    Fixed the issue where "View playbooks" button is not displayed on work order task cards

-   **Version 2.0.1 - February 2024**

    Changes: Avoid multiple loading signs when loading the playbook.

-   **Version 2.0.0 - August 2023**
    -   New
        -   Added ability to showcase Playbook on Mobile
        -   Built an OOTB playbook for Work Order Task Closure as an example playbook, which showcases the ability of showing playbook in mobile
-   **Version 1.0.1 - April 2021**

    New: Enables creation of playbooks for Field Service Management workflows
