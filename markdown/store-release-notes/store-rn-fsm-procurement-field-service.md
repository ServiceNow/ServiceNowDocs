# Procurement for Field Service Management release notes

Version history for the Procurement for Field Service Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.0 - March 2026**

    New.

-   **Version 2.0.0 - February 2025**

    The ServiceNow Procurement for Field Service application provides seamless integration between Procurement Service Management and Field Service Management applications. Use this capability to enable field service mobile workers to order required parts and assets via Procurement Service Management. Allow Procurement Service Management to then fulfill the order and deploy the procured assets via the parts and inventory Field Service Management workflows running on the Mobile Agent app.
