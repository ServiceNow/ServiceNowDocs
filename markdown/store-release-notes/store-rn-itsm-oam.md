# Outlook Actionable Messages release notes

Version history for the ITSM Outlook Actionable Messages app on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.7.0 - June 2026 \(Australia\)**

    Updated to support the latest version of the dependent apps.

-   **Version 4.6.1 - December 2025**

    Updated to support the latest version of the dependent app.

-   **Version 4.3.3 - May 2025**

    Fixed minor issues with translations in the Outlook customer satisfaction survey.

-   **Version 4.3.2 - August 2024**

    Improved support for experience &amp; service feedback capability improvements.

-   **Version 4.2.1 - May 2024**

    Capture user feedback directly in email notifications for service feedback utilizing Surveys.

-   **Version 4.1.5 - February 2024**

    Capture user feedback directly in email notifications for service feedback.

-   **Version 4.0.6 - November 2023**

    Minor fixes.

-   **Version 4.0.2 - June 2023**

    Changed: Pre-built integration with SAP Concur allows approvers to:

    -   View expense report.
    -   View line items for an expense report.
    -   View attachments for a report.
    -   Approve or send back the expense report with comments.
    -   Requires customer to be on Tokyo or a higher version.
-   **Version 3.1.3 - November 2022**

    Fixes

-   **Version 3.1.2 - August 2022**

    Minor fixes

-   **Version 3.1.0 - May 2022**

    Minor fixes

-   **Version 3.0.0 - January 2021**

    Outlook Actionable Messages has been updated with the following new features: Customization support for approval emails via formatting blocks Graphical Approval Emails Image Support for 1-Click Surveys Advanced editor mode allowing for admins to directly customize the underlying JSON.

-   **Version 2.0.1 - July 2019**
    -   Outlook Actionable Messages enables rich interactive emails for approvals and surveys in Microsoft Outlook. With Actionable Messages, it’s easier and faster to approve or complete a survey when you can take action right from the email message.
    -   New: Support for Signed Cards, Support for Outlook Mobile
