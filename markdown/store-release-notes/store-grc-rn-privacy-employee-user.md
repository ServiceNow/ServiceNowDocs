# Privacy Employee User release notes

Version history for the Privacy Employee User application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 19.0.1 - August 2024**

    This application installs the 'Privacy Employee User' role that enables the users to respond to and create certain types of tasks. The Employee role ensures that employees can report, create, or update privacy records like Privacy case, Privacy issue, and other types of privacy records. This means more people in your organization can help manage privacy regulations better. The 'Privacy Employee User' plugin is available only to customers with 'PRM Operator' license.
