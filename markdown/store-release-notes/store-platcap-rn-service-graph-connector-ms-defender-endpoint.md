# Service Graph Connector for Microsoft Defender Endpoint release notes

Version history for the Service Graph Connector for Microsoft Defender Endpoint application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - May 2025**

    Fixed: Mapped the devices to the computer class, and implemented Windows server classification.

-   **Version 1.1.0 - March 2025**
    -   New: Changed the install\_status of devices to handle Life Cycle Management.
    -   Fixed:
        -   Mapped the onboarded status to the onBoardingStatus field in the extension table.
        -   Changed the first\_seen and last\_reported fields from string to date/time.
-   **Version 1.0.0 - February 2024**

    Microsoft Defender helps enterprise networks prevent, detect, investigate, and respond to advanced threats. Service Graph Connector for Microsoft Defender Endpoint imports device details from Azure on which Microsoft Defender agents are deployed into the ServiceNow CMDB quickly, seamlessly, and securely. This integration is part of ServiceNow-developed Service Graph Connectors. The Plug-and-play Service Graph Connectors simplify setup and facilitate quick integration.
