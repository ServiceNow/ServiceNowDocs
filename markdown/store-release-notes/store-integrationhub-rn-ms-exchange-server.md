# Microsoft Exchange Server spoke release notes

Version history for the Integration Hub Microsoft Exchange Server spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.6.0 - June 2026**

    Non Glide Cobalt Raven ACLs added.

-   **Version 2.5.1 - November 2025**

    Fixed: Added authentication parameter in the Powershell step.

-   **Version 2.5.0 - August 2023**
    -   Added delta sync to detect changes
    -   Added action to retrieve recurring event occurrences
    -   Updated Look up Calendar Item by ItemID action to fetch additional details like UID
-   **Version 2.4.0 - February 2023**

    New: Authentication template.

-   **Version 2.3.3 - October 2022**

    Fixed: Patch release of Microsoft Exchange Server Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 2.3.2 - May 2022**

    New: Patch release of Microsoft Exchange Server Spoke. This version includes credType exchange in powershell actions.

-   **Version 2.3.0 - December 2021**
    -   Fixed: Minor release of Microsoft Exchange Server Spoke for IntegrationHub. This version includes support for Kerberos Authentication and includes the below fixes.
        -   XML Injection in Outbound Web Service Calls.
        -   'Transaction cancelled' error for the 'Delete Address List' action in Quebec.
        -   Error at Create Attachment.
        -   UTC Time conversion issue.
        -   Power shell execution error.
-   **Version 2.2.0 - November 2021**

    New: This release adds 4 new actions \(Update a Calendar Meeting, Delete Calendar Event for User, Look up Meetings Stream, Look up Folder ID\).

-   **Version 2.1.1 - June 2021**

    This is a patch release for the Exchange Server spoke with fixes.

-   **Version 1.7.0 - December 2020**

    Minor release of Microsoft Exchange Server spoke for Integration Hub. This version includes a new action under Mailbox Management.

-   **Version 1.6.1 - November 2020**

    Minor release of the Microsoft Exchange Server spoke for Integration Hub.

-   **Version 1.0.0 - August 2019**

    Provides actions to automate address lists, mailboxes, and calendar events for user management in Microsoft Exchange Server.
