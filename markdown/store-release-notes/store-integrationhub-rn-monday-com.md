# monday.com Spoke release notes

Version history for the Integration Hub monday.com Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.1.5 - June 2025**
    -   Fixed:
        -   Look up Item
        -   Look up Group by ID
        -   Look up Files by Board ID
        -   Look up Column Values
        -   Look up Items by IDs
        -   Look up Items Stream by Board ID
        -   Add File to File Column
        -   Add File to Update
-   **Version 1.1.4 - November 2023**

    Fixed: Add last activity to a user query.

-   **Version 1.1.3 - August 2023**

    In V1.1.3, Look up Boards Stream action is fixed for user's last activity.

-   **Version 1.1.2 - May 2023**
    -   Added localization changes to support multi-language
    -   Added "report view" ACLs to the table columns with sensitive info
-   **Version 1.1.1 - February 2023**
    -   New: Added actions beyond user management and subscription management
    -   Fixed: Bug fixes
-   **Version 1.0.1 - August 2021**

    Made changes to support internationalization
