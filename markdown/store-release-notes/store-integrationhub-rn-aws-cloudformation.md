# AWS CloudFormation spoke release notes

Version history for the Integration Hub AWS CloudFormation spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.4 - March 2024**

    Added help content for the actions where it was missing.

-   **Version 1.1.3 - September 2023**

    Fixed: The license requirements.

-   **Version 1.1.2 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 1.1.0 - May 2022**

    Patch release of AWS CloudFormation spoke for IntegrationHub.

-   **Version 1.0.0 - July 2020**

    Provides actions to automate the management of stacks, stack sets and change sets in Amazon CloudFormation.
