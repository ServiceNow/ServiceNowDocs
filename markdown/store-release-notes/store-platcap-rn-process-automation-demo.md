# Process Automation Experience Demo release notes

Version history for the Process Automation Experience Demo application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 28.0.1 - April 2026 \(Zurich\)**

    Fixed: The interaction\_closed\_complete declarative action did not enforce write access before performing the close interaction operation.

-   **Version 24.1.4 - October 2024 \(Vancouver\)**

    Fixed: Added ACL check in the sys\_ux\_screen\_conditions scripts as part of Playbook Workspace Demo.

-   **Version 24.1.3 - September 2023 \(Vancouver\)**

    Fixed: Corrected the trigger for the process Application Approval Demo.

-   **Version 24.1.1 - August 2022 \(Vancouver\)**

    Changed: Updated to demonstrate building in the diagram view and with new start rules. See the family release notes for more information.

-   **Version 21.4.1 - November 2022 \(San Diego\)**
    -   Changed: XML files for the checklist template are now in the Plugin updates folder.
    -   Removed: Onboarding demo process.
-   **Version 22.1.1 - November 2022 \(Tokyo\)**
    -   Changed: XML files for the checklist template are now in the Plugin updates folder.
    -   Removed: Onboarding demo process.
-   **Version 22.04 - August 2022**
    -   New: Playbooks now auto advance to the next stage, without users having to manually navigate.
    -   Fixed: Key performance and accessibility improvements, usability changes, and fixes.
-   **Version 21.3.2 - August 2022 \(San Diego\)**

    Changed: Minor changes to demo content

-   **Version 21.2.2 - May 2022**
    -   New:
        -   New button for Incident List and Add Application Approval in workspace demo
        -   Adoption of new proxied data broker in Critical Incident Activity UI
            -   Note: users should update any existing custom Activity UIs to adopt the new data broker
    -   Changed: Converted Process Automation Experience Demo to Layout3
    -   Removed:
        -   "Open List" demo DA once OOB has been added
        -   Dummy data knowledge article; replaced with reference to OOB knowledge article
-   **Version 21.1.3 - February 2022**
    -   New:
        -   Demonstrate how to provide a new record query into Playbook Record Generator in Agent Workspace
        -   Add Optional Activities to a running Playbook in the Incident Response Demo
        -   Adopt Unified Navigation in Playbook Workspace Demo
    -   Fixed: Incident Response Demo now works in Agent Workspace
-   **Version 20.3.3 - October 2021**

    New: Includes an example of a User Form showcasing the new "Mark Complete" functionality within the Playbook Experience Demo.
