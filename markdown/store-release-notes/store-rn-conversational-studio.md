# Conversational Studio release notes

Version history for the Conversational Studio application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.6 - November 2024**

    Conversational Studio is our up-leveling of the existing Virtual Agent Designer tool, adding more configurability, added components, and a cleaner user experience consistent with the other designers on the ServiceNow platform.
