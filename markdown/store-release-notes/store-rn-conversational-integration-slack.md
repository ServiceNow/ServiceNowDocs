# Conversational Integration with Slack release notes

Version history for the Conversational Integration with Slack integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.0.7 - June 2026 \(Australia\)**

    Defect fixes:  PRB2013415, PRB2014752, PRB2011095d

-   **Version 6.0.6 - March 2026 \(Australia\)**
    -   Implement Feedback api for Slack
    -   Fixed a few defects
-   **Version 6.0.5 - December 2025 \(Zurich\)**
    -   People card support for Slack
    -   Read only directive for Slack
    -   Support for agentic response with Slack
    -   Support @mention functionality in Slack group chats.
    -   Fixed bugs.
-   **Version 6.0.2 - October 2025**

    Fixed issues.

-   **Version 6.0.0 - August 2025 \(Yokohama\)**
    -   Improvements:
        -   Support Streaming in Slack
        -   Enhanced Security Model Adoption Zurich Directive for Slack
    -   Bugs:
        -   Fixing the issue with team\_id for slack OOB bot DEF0671272: People card support for Slack
        -   Slack Pre Link Job doesn't link accounts daily
        -   VASlackResponseBuilder.getImageUrl\(\) currently uses a hardcoded domain and should be replaced with 'glide.servlet.uri'
        -   Outbound messages from Slack not getting delivered when wait card is enabled
-   **Version 5.0.15 - May 2025 \(Yokohama\)**

    Fixed minor bugs.

-   **Version 5.0.13 - March 2025 \(Yokohama\)**

    Fixed minor bugs.

-   **Version 5.0.12 - February 2025 \(Yokohama\)**
    -   New: Synthesized Response support for Slack
    -   Fixed:
        -   Fix for Skip button issue
        -   Added translations for the store app
-   **Version 5.0.10 - November 2024 \(Xanadu\)**
    -   Fixed the text user input control issue: When the user is prompted to input text and sends their response via Slack, VA will say "I'm having technical issues and won't be able to continue this conversation."
    -   Removed the extra AI message: During a Q&amp;A card conversation we see the text "Answer generated by...Review for accuracy" repeated 3-times.
    -   Fixed the notification message issue: Notification received with notification message as "undefined"
-   **Version 5.0.8 - August 2024 \(Xanadu\)**
    -   \[ Slack \] Pagination and search support in Slack.
    -   GenAIQnA Adapter Card without feedback.
    -   Handling system messages / legal disclaimer
    -   Fixed the issue with HTML tags appearing when the user switches to a catalog item.
    -   Fixed issue with Pre-Link Job scheduled job thathas problem with picking the users in case there are unused sys\_cs\_provider\_application records.
-   **Version 5.0.5 - February 2024 \(Washington DC\)**

    Fixed issues.

-   **Version 5.0.4 - August 2023 \(Vancouver\)**

    Minor fixes.

-   **Version 5.0.2 - May 2023**
    -   New: Improved support for multi-attachment upload.
    -   Minor fixes.
-   **Version 5.0.1 - February 2023**
    -   New: Slack integration with Self-configured bot.
    -   Changed: Improved Admin UI for Slack integrations.
    -   Fixed: Minor bug fixes.
-   **Version 4.1.0 - November 2022**
    -   New: Display of end user-friendly channel names and destination types names.
    -   Fixed:
        -   File sharing between Live Agent and Slack User.
        -   Image upload issue via link for the file picker topic.
-   **Version 3.0.8 - November 2022**
    -   Fixed:
        -   File sharing between Live Agent and Slack User.
        -   Image upload issue via link for the file picker topic.
-   **Version 4.0.0 - August 2022**
    -   New:
        -   Virtual Agent now enables customization and branding of the Slack bot's name and icon.
        -   Slack now displays the Promoted topics to end users during conversation.
    -   Fixed:
        -   Branding issues.
        -   Message ordering.
        -   Topic execution.
        -   System property issue.
        -   Technical issues: Account-linking, Table output control.
        -   User interface display: I'm done button.
        -   Controls: Multi flow control, Catalogue Genius Card.
        -   Notification issues: Actionable and Show notification.
-   **Version 3.0.5 - May 2022**
    -   New: Virtual Agent now enables pre-linking of Slack bot users to ServiceNow instance.
    -   Fixed:
        -   Bot responses
        -   Reference choice control
-   **Version 3.0.2 - February 2022**
    -   New: Unified logging framework has been enabled.
    -   Fixed:
        -   The issue with 'Bye' keyword directly ending the conversation.
        -   The unified logging backward compatibility condition.
        -   The Intent to return the utterances with characters in bold.
        -   The issue with live agent name display.
        -   The outage topics by removing the extra link in service disruption.
-   **Version 3.0.1 - December 2021**
    -   New: Estimated wait time for connecting to a live agent is now displayed.
    -   Fixed:
        -   Undefined non-actionable notification message.
        -   Issue with 'Show More' option on the bot.
-   **Version 3.0.0 - October 2021**
    -   New: Make all card templates in the Conversational Integration with Slack available in the same template framework/table.
    -   Fixed:
        -   Security fix
        -   L10N issues
-   **Version 2.0.3 - January 2022**
    -   Fixed:
        -   The outage topics by removing the extra link in service disruption.
        -   The Intent to return the utterances with characters in bold.
        -   The issue with 'Bye' keyword directly ending the conversation.
-   **Version 2.0.0 - August 2021**
    -   New:
        -   YouTube URL unfurling.
        -   Support for rich text using markdown in notification messages.
        -   URL redirect to Service Portal page.
    -   Fixes
-   **Version 1.0.1 - February 2021**
    -   New:
        -   Automatic account linking for Virtual Agent users
        -   Notifications that support:
            -   Subscription management
            -   Image content
            -   Continuous conversations from notification receipt to bot interaction
            -   Actionable content \(action buttons\)
        -   AI Search results in Virtual Agent that support:
            -   Image cards
            -   Catalog cards
            -   Q&amp;A cards
