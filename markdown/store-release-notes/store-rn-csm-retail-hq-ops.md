# Retail HQ Operations release notes

Version history for the Retail HQ Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.3.0 - June 2026**
    -   Changed:
        -   IView/Track HQ Case page with detailed case information.
        -   HQ Case state management, details, and field configuration.
        -   HQ Task state management, details, and fields.
        -   View All Plans page with plan list and Plan Details – Track Plan tab view.
-   **Version 4.4.0 - June 2026 \(Zurich\)**
    -   Changed:
        -   HQ Case state management, details, and field configuration.
        -   View/Track HQ Case page with detailed case information.
        -   HQ Task state management, details, and fields.
        -   View All Plans page with plan list and Plan Details – Track Plan tab view
-   **Version 5.1.0 - May 2026**
    -   New:
        -   Enables HQ Operations Agents, Managers and Regional Managers to track and monitor published HQ Communication store plans
        -   Added HQ case fulfilment capability in workspace for store plan
        -   Added capability to track status of all HQ cases in the store plan
        -   Added capability to track status of store cases generated within each HQ case
-   **Version 4.3.0 - May 2026 \(Zurich\)**
    -   New:
        -   Enables HQ Operations Agents, Managers, and Regional Managers to track and monitor published HQ Communication store plans
        -   Added HQ case fulfillment capability in workspace for store plan
        -   Added capability to track the status of all HQ cases in the store plan
        -   Added capability to track the status of store cases generated within each HQ case
-   **Version 4.1.0 - March 2026 \(Australia\)**

    New: Enables HQ Operations Agents, Managers and Regional Managers to create structured operational store plans with defined cases and tasks.

-   **Version 3.3.0 - March 2026 \(Zurich\)**

    New: Enables HQ Operations Agents, Managers and Regional Managers to author structured operational store plans with defined cases and tasks.

-   **Version 2.1.0 - August 2025**

    The Retail HQ Operations application enables effective coordination between HQ teams. It facilitates the execution of assigned work and allows HQ teams to monitor progress.
