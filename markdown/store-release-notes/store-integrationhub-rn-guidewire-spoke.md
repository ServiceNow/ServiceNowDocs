# Guidewire Spoke release notes

Version history for the Guidewire Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - March 2026**

    Updated: Replaced direct references to the admin role with feature-specific granular admin roles to align with least-privilege access principles.

-   **Version 1.2.1 - August 2025**

    New: Added datastream and other IntegrationHub apps dependency.

-   **Version 1.2.0 - November 2023**
    -   New:
        -   Look Up Fixed Property Incidents on Claim by Claim ID Stream
        -   Look Up Notes on Claim by Claim ID Stream
        -   Look Up Vehicle Incidents on Claim by Claim ID Stream
        -   Look Up Claim Documents by Claim ID Stream
        -   Look Up Exposures on Claim by Claim ID Stream
        -   Look Up Claim Document Content by Document ID
        -   Look up Claim Contact on Claim by Contact ID
        -   Look Up Injury Incidents on Claim by Claim ID Stream
-   **Version 1.1.0 - August 2023**

    New:

    -   Added spoke actions to validate and close exposure
    -   Added spoke actions to update draft auto claim
    -   Added spoke actions to retrieve policy details from policy center
    -   Added spoke actions to retrieve product definitions details
    -   Added spoke actions to support policy cancellation, bind, issue, and quote a policy
    -   Added localization-related changes
    Fixed:

    -   Updated input parameters order in Look up Company Accounts Stream spoke action.
    -   Updated 'Query Parameter' input param label to 'Include'.
    -   Updated 'Loss date' and 'Policy number' as mandatory fields for Create draft claim spoke.
-   **Version 1.0.0 - May 2023**

    Guidewire Software, Inc. \(GWRE\) is the global market leader for core technology in property and casualty \(P&amp;C\) insurance. Guidewire's ClaimCenter, PolicyCenter, and BillingCenter make up the Guidewire InsuranceSuite.The Guidewire Spoke is integrated with the cloud version of Guidewire InsuranceSuite. The spoke application encapsulates granular actions which can serve as the building blocks for insurance cases requiring integration with Guidewire. Although this spoke addresses certain primary APIs, customers can expand the spoke to support any Cloud APIs available in Guidewire.
