# Oracle Boot Volume spoke release notes

Version history for the Oracle Boot Volume Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.6 - December 2022**

    Changed: App dependency requirement.

-   **Version 1.0.5 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.4 - March 2022**

    New: Includes missing dependency on Oracle Cloud IAM Spoke.

-   **Version 1.0.3 - October 2021**
    -   Fixed:
        -   Issues with actions - Get Boot Volume Backup, Look up Boot Volume Backups, Update Boot Volume, Delete Boot Volume, Look up Boot Volumes, Update Boot Volume Backup, Attach Boot Volume, Get Boot Volume Attachment, Change Boot Volume Backup Compartment, Look up Boot Volume Attachments, Get Boot Volume, Change Boot Volume Compartment, Delete Boot Volume KMS Key, Delete Boot Volume Backup, Get Boot Volume KMS Key, Update Boot Volume KMS Key, Detach Boot Volume, Create Boot Volume Backup
        -   Missing link with OCI SHA256WithRSA Signature not available at the credential picker.
-   **Version \(1.0.2\) - June 2021**

    Patch release of Oracle Boot Volume spoke for IntegrationHub. This version adds KMF modules for additional security of the password2 fields.
