# Gremlin spoke release notes

Version history for the Integration Hub Gremlin spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.2 - May 2021**

    Patch release of the Gremlin spoke for IntegrationHub. This version includes a fix for the 'Response size exceeds limit' error using Look Up Attacks action.

-   **Version 1.0.1 - December 2020**

    Patch release of the Gremlin spoke for IntegrationHub.

-   **Version 1.0.0 - July 2020**

    Provides actions to manage attacks and schedules in Gremlin.
