# Alignment Planner Workspace integration with Azure DevOps \(ADO\) release notes

Version history for the Alignment Planner Workspace integration with Azure DevOps \(ADO\) on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - May 2022**
    -   New:
        -   Introduced guided setup and added support for domain separation, localization, custom table map, boolean field map, bi-directional syncing of attachments, explicit unlinking of work items from Azure DevOps, refresh areas, handling of changes in Azure DevOps projects/areas.
        -   Ability to bulk import work items
        -   Added scheduled jobs to fetch Azure DevOps areas
-   **Version 1.0.0 - February 2022**

    Enable bidirectional synchronization of work items between ServiceNow Alignment Planner Workspace \(APW\) and Microsoft Azure DevOps. Plan your work in APW and execute it in Azure DevOps. The bidirectional sync of records ensures that your progress is up to date on both the platforms.
