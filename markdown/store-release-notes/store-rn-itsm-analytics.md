# ITSM Analytics release notes

Version history for the ITSM Analytics on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.1.2 - June 2026**
    -   Fixed:
        -   'Open problems classified as Known error' KPI
        -   'Average wait time', 'Average service time' display units changed to hours from raw milliseconds
        -   Condition field for 'Password Enrollment Snapshot' indicator
-   **Version 4.1.1 - April 2026**
    -   Fixed:
        -   Data collection job warnings
        -   Incorrect breakdown mappings
        -   Accessibility issues
-   **Version 4.0.0 - December 2025**

    Granular admin role support in Success Dashboard and Benchmarks.

-   **Version 3.1.2 - August 2025**

    Fixed in this release: Fixed issues with generating breakdowns associated to Requested Item indicator source \(sc\_catalog\).

-   **Version 3.1.0 - May 2025**

    Fixed: Issues with Data Collection jobs for Walk-up Experience indicators.

-   **Version 1.0.0 - May 2024**

    ITSM Analytics is a package of all the ITSM related operational KPIs, data visualization and dashboard. Helps you understand the performance of your key processes defined around ITSM capabilities. Base system has dashboard template for Incident, Problem, Change, Request, Service Catalog, Major Incident, Interaction, On-call, Walk-up and Password Reset.
