# Password Reset integration for Microsoft Active Directory

Version history for the Integration Hub Password Reset integration for Microsoft Active Directory application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.0 - August 2025**

    New: Migrated to the latest Microsoft Active Directory actions from the AD v2 spoke, moving away from the older, deprecated Microsoft Active Directory actions.

-   **Version 3.0.0 - November 2024**
    -   Changed:
        -   Renamed the spoke from "Microsoft AD Spoke for Password Reset" to "Password Reset integration for Microsoft Active Directory"
        -   Bug fixes
-   **Version 2.1.1 - January 2024**

    Minor fixes.

-   **Version 2.1.0 - November 2023**

    Minor fixes.

-   **Version 1.0.1 - July 2021**
    -   Password Management subflows include:
        -   Password Lock State
        -   Password Change
        -   Password Reset
        -   Password Unlock
