# Google Chat Spoke release notes

Version history for the Google Chat Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - September 2025**

    Fixed - Required ACL\(s\) or Role\(s\) for agents.

-   **Version 1.1.2 - August 2025**

    Fixed - AI Agents - Renaming and Output transformation strategy

-   **Version 1.1.1 - March 2025**

    Minor release with new AI agents.

-   **Version 1.0.1 - August 2024**

    Google Chat is chat based collaboration tool in a company. Individuals within a company collaborate with each other using direct messages. Teams manage group chat using Spaces. Space owners can add or remove members from a Space. Individuals can chat via threads and messages. ServiceNow, as a system of engagement, can facilitate these actions using workflows easily. All Space, Space member, message management actions cab be facilitated using Actions within Flow Designer. Provides Actions to automate the management of Spaces, Space members, and messages.
