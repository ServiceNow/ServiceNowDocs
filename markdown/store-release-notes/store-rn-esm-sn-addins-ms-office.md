# ServiceNow Add-in for Microsoft 365 Office release notes

Version history for the ServiceNow Add-in for Microsoft 365 Office application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 7.5.0 - June 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 7.4.3 - March 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 7.4.0 - December 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 7.1.3 - May 2025**

    Added an ACL to determine the endpoint of the manifest file download to avoid security concerns.

-   **Version 7.1.2 - February 2025**

    Employees can self-serve across multi-department workflows from within Microsoft Outlook or other Microsoft 365 apps supporting add-ins.
