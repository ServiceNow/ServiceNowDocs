# User Experience Redirection release notes

Version history for the ServiceNow® User Experience Redirection application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.1 - March 2026**
    -   To support the overall workspace adoption
        -   For new customers, SOW redirection will be ON by default, meaning all users will be redirected to SOW.
            -   This setting can be switched only with ServiceNow consultation
        -   For existing customers, they will have a configuration setting available at the record level.
            -   Once turned ON at record level, all users of that record type should be available in SOW only
            -   Existing customers should be able to toggle the setting as required.
