# Microsoft Defender for Office365 Integration for SecOps release notes

Version history for the Microsoft Defender for Office365 Integration for SecOps application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.4 - December 2024**

    Minor fixes for this release.

-   **Version 2.3.1 - November 2024**

    Fixed: Minor fixes for this release.

-   **Version 2.2.3 - May 2024**

    The Microsoft Defender for Office365 Integration for SecOps application integrates Microsoft Defender for Office365 with the Cybersecurity Executive Dashboard providing detailed insights into your organization's phishing preparedness. This allows cybersecurity executives to see how the organization is performing in phishing simulations and identify areas where additional training may be needed.
