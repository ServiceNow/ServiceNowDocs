# SAP integration for Finance Close Automation release notes

Version history for the SAP integration for Finance Close Automation in ancillary software on the ServiceNow Store.

**Important:** The Finance Close Automation application has been deprecated and is no longer maintained.

## Version history

-   **Version 7.0.0 - May 2020**
    -   New: New Extension Point to add logic before saving journal entries on SAP side
    -   Changed: Journal entries post queue maintained at JE level irrespective of journal entries using JE or Task workflow
    -   Fixed: Derivation of COPA cost center issue Failure in saving journal entries when posting
-   **Version 6.0.0 - April 2020**

    Fixed: The latency issue is fixed in processing Journal Entry transactions
