# Security Operations Health Analytics release notes

Version history for the Security Operations Configuration Compliance application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 30.2.0 - April 2026**
    -   Changed: Auto-Close Health Dashboard configurations for Vulnerability Response, Container Vulnerability Response, and Configuration Compliance are now applied consistently during upgrades.
    -   Fixed: The Collect Health Dashboard Metrics job now collects all expected information reliably.
-   **Version 2.5.0 - April 2026**
    -   Changed: Auto-Close Health Dashboard configurations for Vulnerability Response, Container Vulnerability Response, and Configuration Compliance are now applied consistently during upgrades.
    -   Fixed: The Collect Health Dashboard Metrics job now collects all expected information reliably.
-   **Version 2.4.3 - December 2025**

    Minor fixes for this release.

-   **Version 2.4.2 - May 2025**

    Minor fixes for this release.

-   **Version 2.3.3 - November 2024**

    New: New widgets have been added in the Remediation Healthtab to improve the visualization of discovered items' \(DI\) health. These improvements provide detailed insights into the status of the DIs.

-   **Version 2.2.1 - August 2024**

    New: Updated the reports in the Health dashboard to incorporate the newly added 'Active' field of Test Result table.

-   **Version 2.1.8 - May 2024**

    Fixed: The Health Analytics dashboard is now independent of the Vulnerability Response application.

-   **Version 2.1.2 - February 2024**

    Changed: Removed the dependency on Vulnerability Response.

-   **Version 2.1.1 - November 2023**

    Minor fixes for this release.

-   **Version 2.0.3 - August 2023 \(Vancouver\)**
    -   This dashboard provides the health metrics and overall health posture of Vulnerability Response applications.
    -   New:
        -   Monitor the overall health of the Vulnerability Response application.
        -   View individual scores that make up the aggregated scores across the pillars of implementation, configuration, integration, process, and data.
