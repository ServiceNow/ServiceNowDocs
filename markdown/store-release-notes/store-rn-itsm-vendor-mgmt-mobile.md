# Vendor Management Mobile release notes

Version history for the Vendor Management Mobile application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - August 2023**

    New: Added dark theme.

-   **Version 1.2.1 - February 2023**
    -   New: Next Experience UI update for Mobile Agent ITSM - Vendor Configure AI Search for Vendor mobile app Migrate from legacy icon section to new icon section for Vendors
    -   Changed: Update icons used by Vendor applet launcher to be Next Experience compatible Update item card with Next Experience styles using Mobile Card Builder
-   **Version 1.1.1 - February 2022**

    Provides a mobile interface for vendor managers to be able to view relevant information filtered by the vendors they are responsible for, as well as all other vendors.
