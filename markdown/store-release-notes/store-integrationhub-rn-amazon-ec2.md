# Amazon EC2 spoke release notes

Version history for the Integration Hub Amazon EC2 spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.1 - April 2026**
    -   Fixed:
        -   1. Issue where the Get Instance Attribute action returned empty values for certain inputs
        -   2. Issue where the Update Instance Attribute action did not work for Block Device Mapping configurations
-   **Version 1.4.0 - September 2025**

    Fixed: Required ACL\(s\) or Role\(s\) for agents.

-   **Version 1.3.0 - March 2025**

    Added 2 new conversational spoke actions and subflow samples.

-   **Version 1.2.1 - September 2022**

    Fixed: Patch version of the Amazon EC2 Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.2.0 - November 2020**

    New action: Start Instances.

-   **Version 1.0.2 - April 2020**

    Provides actions to automate the management of instances and images in Amazon EC2.
