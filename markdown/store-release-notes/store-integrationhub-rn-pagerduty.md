# PagerDuty spoke release notes

Version history for the PagerDuty spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.5.1 - January 2026**

    Fixed: Create Incident action's From Field's &lt;email&gt; tag malform error

-   **Version 1.5.0 - October 2025**

    Added 12 AI Agents.

-   **Version 1.4.1 - June 2025**

    Fixed:

    -   Look up Business Services Stream action
    -   Look up Users Stream action
-   **Version 1.4.0 - November 2024**

    Evaluated and modified Sandbox Access for Client-Callable Script Includes as false.

-   **Version 1.3.2 - August 2024**
    -   Fixed: ACL 'PagerDutyCreateWebhookSubscription' Script Include
    -   Fixed: Look up Users action's pre-processing variable.
-   **Version 1.3.1 - June 2023**
    -   Added about 50 actions to align with Pagerduty's core business functions
-   **Version 1.0.3 - January 2021**

    PagerDuty spoke provides actions to see and understand meaningful usage for PagerDuty software subscriptions. Analyze the on-call users to see which licenses are considered stale and reclaim these licenses to realize potential savings.
