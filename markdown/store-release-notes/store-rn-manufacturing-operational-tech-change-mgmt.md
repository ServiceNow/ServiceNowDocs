# Operational Technology Change Management release notes

Version history for the Operational Technology Change Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.0 - June 2026**

    New: Certified for Australia Patch 3

-   **Version 3.1.0 - March 2026**

    New: Certified for Australia.

-   **Version 3.0.0 - December 2025**

    New: Certified for Zurich.

-   **Version 2.0.7 - August 2025**

    New: Certified for Zurich.

-   **Version 2.0.6 - February 2025**
    -   New:
        -   The Incidents Fixed by Change related list was added to the OT change request record in the Industrial Workspace to list OT incidents that need an OT change request for remediation.
        -   The Incidents Caused by Change related list was added to the OT change request record in the Industrial Workspace to list OT incidents needed because of an OT change request.
-   **Version 2.0.5 - November 2024**

    Fixed: Updated with various bug fixes.

-   **Version 2.0.2 - September 2024**
    -   New:
        -   OT Change Management now includes Industrial Process Manager and can be used with or without Operational Technology Manager
        -   The OT change record in the Industrial Workspace was updated so that all existing related lists are included under the Related Records tab
        -   Certified for Xanadu
-   **Version 2.0.1 - August 2024**
    -   OT Change Management now includes Industrial Process Manager and can be used with or without Operational Technology Manager
    -   The OT change record in the Industrial Workspace was updated so that all existing related lists are included under the Related Records tab
    -   Certified for Xanadu
-   **Version 1.0.1 - February 2024**

    Certified for Washington DC.

-   **Version 1.0.0 - August 2023**

    The ServiceNow Operational Technology Change Management application enables your team members to work collaboratively on changes to Operational Technology or industrial equipment configurations. These changes include any optimizations, alterations in the production process, or vulnerability fixes.
