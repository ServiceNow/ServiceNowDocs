# Amazon Connect spoke release notes

Version history for the Amazon Connect spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - March 2025**

    Added more output fields for Get User Details action.

-   **Version 1.1.3 - February 2024**

    Fixed: Get Current Metrics Data, Get Metrics Data Flow Action not working from Script Include.

-   **Version 1.1.2 - December 2022**

    Changed: Internal license check processing.

-   **Version 1.1.1 - September 2022**

    Fixed: Improve installation performance for the spoke

-   **Version 1.1.0 - February 2022**
    -   New: Minor release of the Amazon Connect spoke for IntegrationHub. This version includes 3 new actions - Get Contact Attributes, Update Contact Attributes, and Look up Agent Status Stream.
    -   Fixes
-   **Version 1.0.0 - January 2021**

    Provides actions to automate the management of Amazon Connect.
