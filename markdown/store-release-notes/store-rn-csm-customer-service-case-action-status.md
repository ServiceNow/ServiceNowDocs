# Customer Service Case Action Status release notes

Version history for the ServiceNow® Customer Service Case Action Status application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.0.2 - June 2026**

    Defect Fix \(Keep major case related list consistent in app-csm-action-status and app-major-issue-management\) - no new feature has been delivered.

-   **Version 2.0.0 - March 2026**

    No new features have been delivered; migrated the application from family to store. Provides visual action status indicators on customer service cases so agents can quickly identify cases that need attention or are blocked, and prioritize their work accordingly.
