# AWS IAM spoke release notes

Version history for the Integration Hub AWS IAM spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 1.1.0 - July 2023**
    -   Changed:
        -   Action names as per naming standards
        -   Added path output to actions: Look up Group, Look up Group Users, Look up User, Look up Groups for User, Look up Roles
-   **Version 1.0.4 - October 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.3 - August 2021**

    New: Patch release AWS IAM spoke for IntegrationHub. This version adds KMF module for additional security at password2 fields.

-   **Version 1.0.2 - May 2020**

    New: Updated release of AWS IAM spoke for IntegrationHub. A new List Roles action is created in this release.

-   **Version 1.0.0 - February 2020**

    Provides actions to automate the management of Users, Groups, and Policies for AWS Identity and Access Management \(IAM\).
