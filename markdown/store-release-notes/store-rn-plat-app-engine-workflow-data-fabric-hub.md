# Workflow Data Fabric Hub release notes

Version history for the Workflow Data Fabric Hub application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.1.0 - April 2026**

    Australia - Patch1 Compatible release.

-   **Version 3.0.1 - March 2026**

    Australia Compatible Release

-   **Version 2.2.1 - January 2026**

    Zurich Patch 4 and higher.

-   **Version 2.1.0 - October 2025**

    This patch release includes the following key fixes:

    -   Fixed an issue where references for cdata-end="216" data-start="211"&gt;INT and data-end="229" data-start="221"&gt;BIGINT column types were lost when editing a data fabric table.
    -   Addressed missing translations on the Access Control tab to ensure consistent localization.
-   **Version 2.0.1 - August 2025**
    -   Workflow Data Fabric Hub unifies business and technology data from multiple sources, enabling centralized, real-time access to your enterprise data without copying it to your instance.
        -   Create data fabric tables for flexible data sources from zero copy connections and local data.
        -   Retrieve data from external sources in real-time without copying any data to your instance using zero copy connections.
        -   Fuel AI agents and workflows on the ServiceNow AI Platform with external data using data fabric tables.
        -   Provide data to AI-powered features on the ServiceNow AI Platform, expand workflow automation, and enrich metrics and analytics with enterprise data.
-   **Version 1.1.0 - October 2025**

    This patch release includes the following key fix:

    -   Fixed an issue where references for code data-end=""216"" data-start=""211""&gt;INT and data-end=""229"" data-start=""221""&gt;BIGINT column types were lost when editing a data fabric table.
