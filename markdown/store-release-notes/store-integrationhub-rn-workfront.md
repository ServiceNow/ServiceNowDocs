# Workfront spoke release notes

Version history for the Workfront Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.3.0 - November 2025**

    Security patch: Improved sample webhook registry security.

-   **Version 1.2.0 - November 2024**

    Evaluated and modified sandbox access for client-callable script includes as false.

-   **Version 1.1.3 - January 2024**

    Fixed: Misconfiguration of table/field ACLs within 'com.sn.workfront.spoke' plugin.

-   **Version 1.1.1 - May 2023**

    Fixed: Defined "report\_view" ACL on columns containing "sensitive" information, limiting the audience capable of using these columns as aggregate in reports.

-   **Version 1.0.0 - August 2021**

    New: Initial release of the Workfront spoke. Use this spoke to analyze usage so that you can reclaim subscriptions and determine your potential savings.
