# Cloud Migration Assessment release notes

Version history for the Cloud Migration Assessment application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - November 2024**

    Changed: Routine security and policy updates.

-   **Version 1.3.0 - February 2024**

    Some UI improvements.

-   **Version 1.2.2 - February 2023**
    -   New: Added support for localization
    -   Fixed: Package format changes
-   **Version 1.2.0 - May 2022**
    -   Fixed:
        -   vCenter Performance metrics collection
        -   ITOM licensing
-   **Version 1.0.1 - February 2022**

    Are you migrating on-premises workloads to the cloud? Is it mind-boggling and difficult to group the servers, track the migrations and decide on the next steps? In a popular field survey, it was noted that 80% of cloud migration projects are delayed due to a lack of sufficient information. Cloud Migration Assessment is precisely the app you need. Group servers, create migration assessment tasks, educate and entice stakeholders to aid a better decision for migrating to the cloud or not. With this app you can use the collected information on hardware, apps, and server configuration to evaluate the migration approaches: The migration strategies and the number of migration waves you need. Use the combined think-tank of the stakeholders to get guidance towards the best approach.
