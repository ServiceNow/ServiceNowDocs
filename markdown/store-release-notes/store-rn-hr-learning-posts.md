# Learning Posts release notes

Version history for the Learning Posts application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.4.6 - August 2025**

    Fixed: Security improvements to widgets.

-   **Version 5.3.1 - May 2024**
    -   Fixed:
        -   Empty ACL problem.
        -   Translation problem with the Question field on the portal.
-   **Version 5.0.1 - August 2023**

    Maintenance updates.

-   **Version 5.0.0 - August 2022**
    -   Removed:
        -   Learning content in Portal Search
        -   Learning content recommendation widget on Employee Center
-   **Version 4.0.0 - February 2022**

    New: Support for Learning tasks within Journey Accelerator plans

-   **Version 3.0.0 - November 2021**
    -   New:
        -   OOB Integration with SABA
        -   Learning recommendations via turn key predictive intelligence
-   **Version 2.0.0 - September 2021**
    -   New:
        -   Integration with Sumtotal in addition to other content providers
        -   Employees can search for learning content or training in Employee Center using AI search and contexual search.
