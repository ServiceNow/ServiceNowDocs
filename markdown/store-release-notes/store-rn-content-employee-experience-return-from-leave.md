# Employee Experience Pack: Return from Leave release notes

Version history for the Employee Experience Pack: Return from Leave content on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - November 2022**
    -   Please note that version 2.1.0 of this Experience Pack is officially compatible with the San Deigo release only.
    -   These update sets contain knowledge templates and a Lifecycle Event modeled after common implementation practices.
    -   Included content:
        -   Lifecycle Event consisting of 5 Activity Sets and:
            -   30 Activities
            -   2 HR Services
            -   2 Notification
            -   2 Take Survey task
            -   3 Knowledge Articles
-   **Version 2.0.0 - April 2022**
    -   Please note that version 2.x of this Experience Pack is officially compatible with the Rome release only. These update sets contain knowledge templates and a Lifecycle Event modeled after common implementation practices.
    -   Included content:
        -   Lifecycle Event consisting of 5 Activity Sets and:
            -   30 Activities
            -   2 HR Services
            -   2 Notification
            -   2 Take Survey task
            -   3 Knowledge Articles
