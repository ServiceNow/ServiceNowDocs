# Asana Spoke release notes

Version history for the Asana Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - August 2024**

    Fixed the Look up Workspace Task action to calculate the user's last activity.

-   **Version 1.0.2 - September 2023**
    -   Fixed:
        -   RRR report gives cache flush warning for PluginInvalidateProperties
        -   'licensable' field set to be true in plugin.xml
-   **Version 1.0.1 - August 2023**

    Fixed the Look up Stories Stream action to calculate the user's last activity.

-   **Version 1.0.0 - August 2022**
    -   ATTENTION: Use of this product requires a license for either IntegrationHub Professional or Software Asset Management.
    -   Manage Asana accounts using your ServiceNow instance. Analyse the data usage for Asana and reclaim the idle accounts.
