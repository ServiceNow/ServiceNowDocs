# Supplier Lifecycle Operations Integration with Oracle release notes

Version history for the Supplier Lifecycle Operations Integration with Oracle application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - December 2025**
    -   Changed: Added granular roles to system properties in accordance with the new directives.
    -   Fixed:
        -   Added sn\_so \(Supplier Operations\) as a dependent plugin for Supplier Lifecycle Operations Integration with Oracle
        -   Removed hard-coded timeout values to improve configurability and system flexibility.
-   **Version 1.1.2 - May 2025**

    Minor bug fixes.

-   **Version 1.1.0 - October 2024**
    -   New:
        -   Added an integration to create suppliers from ServiceNow Supplier Lifecycle Operations \(SLO\) into Oracle
        -   Added an integration to fetch supplier details from Oracle into ServiceNow Supplier Lifecycle Operations \(SLO\)
-   **Version 1.0.0 - August 2024**

    This application enables bi-directional integration between Supplier Lifecycle Operations to Oracle EBS and Oracle Fusion which allows you to create, update supplier data.
