# Retail Customer Complaint release notes

Version history for the Retail Customer Complaint application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.7.0 - June 2026**

    New: Security and compliance directive remediation \(role cleanup, ACL alignment\).

-   **Version 3.6.0 - June 2026 \(Zurich\)**
    -   New:
        -   Security and compliance directive remediation \(role cleanup, ACL alignment\).
        -   Zurich compatibility updates and dependency alignment.
-   **Version 3.1.0 - March 2026 \(Australia\)**

    Changed: Made fieldsOpened By andState read-only usingUser Interface\(UI\) policy instead of dictionary override.

-   **Version 3.0.0 - March 2026 \(Zurich\)**

    Changed: Made fieldsOpened By andState read-only usingUser Interface\(UI\) policy instead of dictionary override.

-   **Version 2.1.0 - August 2025**

    The Retail Customer Complaint application helps manage and resolve customer feedback related to store experiences. The application allows customers to submit complaints anonymously to encourage honest feedback and help stores improve their service.
