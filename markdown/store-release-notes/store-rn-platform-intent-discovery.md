# Intent Discovery release notes

Version history for the Intent Discovery on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.3.8 - May 2026 \(Yokohama, Zurich, Australia\)**

    Added Deny ACLs for NLU workbench tables

-   **Version 3.3.4 - May 2023**

    Fixed: Internal fixes, no changes in customer facing features.

-   **Version 3.1.1 - May 2022**
    -   Changed: Menu navigation
    -   Minor fixes.
-   **Version 3.0.6 - August 2021**
    -   New: Language Grouping: Add primary and secondary language intents to a model group. Additionally, choose specific language models to add intents to.
    -   Changed: Simplified experience for adding intents to different language models.
-   **Version 2.0.8 - February 2021**

    New: Delivers the Intent Discovery feature, and is dependent on the NLU Workbench - Advanced Features plugin. Intent Discovery identifies user intents from analysing incident/case data. Use this application to help identify which intents to model and build Virtual Agent Conversations to attain maximum deflection.
