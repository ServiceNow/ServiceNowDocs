# CPRO Dashboard release notes

Version history for the ServiceNow® CPRO Dashboard application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.0.5 - December 2025**

    The Chief Procurement Officer Dashboard integrates data from Sourcing and Procurement Operations and Performance Analytics for Sourcing and Procurement Operations to provide clear visibility into spend performance, supplier efficiency, and contract compliance.
