# Connect Hub release notes

Version history for the ServiceNow® Connect Hub application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - June 2026**

    New: One‑Click MCP Connector creation from MCP Registry.

-   **Version 2.0.1 - May 2026**
    -   Connect Hub is a centralized integration experience that provides seamless connectivity between ServiceNow and third‑party systems.
    -   With Connect Hub, organizations can discover, manage, and integrate external platforms such as Atlassian, SharePoint, and Salesforce to streamline workflows and improve cross-platform collaboration.
    -   Users can browse and configure a wide range of connectors, including model context protocol \(MCP\) connectors, for standardized integration patterns and metadata collectors that simplify integration setup and maintenance through configuration rather than custom code.
    -   Connect Hub supports secure authentication and connection management through API keys, OAuth tokens, and access controls, keeping integrations compliant with organizational security standards.
    -   Connect Hub supports bidirectional connectivity for both inbound and outbound communication, allowing systems to exchange data and trigger actions based on defined events and business rules.
