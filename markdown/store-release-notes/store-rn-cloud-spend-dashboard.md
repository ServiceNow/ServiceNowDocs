# Cloud Spend dashboard release notes

Version history for the IT Operations Management Cloud Spend dashboard on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.5 - December 2023**

    In this release version, the Cloud Insights \(sn\_clin\) and Software Asset Management - SaaS License Management \(com.sn\_sam\_saas\_int\) plugin versions are refreshed.

-   **Version 1.0.4 - February 2023**

    Fixed: App dependency fix.

-   **Version 1.0.3 - June 2022**

    Fixed: App dependency fix.

-   **Version 1.0.0 - June 2020**

    This dashboard allows IT organizations to understand their total spend on IaaS, PaaS, and SaaS in a single location and get visibility into opportunities to reduce cost.
