# Performance Analytics content pack for Essential SAFe release notes

Version history for the IT Business Management Performance Analytics content pack for Essential SAFe on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.2 - September 2023**
    -   New: Support for dark theme
    -   Fixed: Issues with Time Elapsed widget
-   **Version 1.4.1 - May 2023**

    Updates to the localization support.

-   **Version 1.4.0 - July 2021**
    -   New:
        -   Support for custom story states in area charts of all Essential SAFe dashboards
        -   Added read-only access to all Essential SAFe dashboards using sn\_safe\_read role
-   **Version 1.3.0 - February 2021**
    -   Enabled support for the following metrics:
        -   Program Predictability Measure for Agile Release Train
        -   Program Predictability Measure for Teams
-   **Version 1.2.2 - January 2021**

    Bug fix for the following issue: Agile 2.0 Sprint Dashboard: Sprint Cumulative Flow Diagram does not properly display stories.

-   **Version 1.2.1 - August 2020**
    -   New: Cycle Time for stories in Team, Epic, Feature, ART, and PI dashboards
    -   Access the Story Cycle Time report by installing these dashboards from Solution Library once you upgrade the application. For more details, see Using Solution Library for Essential SAFe dashboards in the documentation.
-   **Version 1.1.0 - June 2020**
    -   New:
        -   Added the following reports:
            -   Cumulative Flow Diagram for Program Increments, Sprints, and Epics
            -   Sprint Variance for teams
-   **Version 1.0.1 - April 2020**

    Fixed: Bug fixes.

-   **Version 1.0.0 - September 2019**

    The Performance Analytics content pack for Essential SAFe provides dashboards for the ServiceNow Scaled Agile Framework \(SAFe\) - Essential SAFe application. These are designed to provide valuable insight on the way we plan and execute software development work. Dashboards are included for Teams, Sprints, Program Increments, Agile Release Trains, Features, and Epics.
