# Learning release notes

Version history for the Learning application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.7.0 - June 2026**

    Fixed: Added Non-Glide cobalt raven ACL's to all the tables in learning app.

-   **Version 5.6.0 - March 2026**

    New: Extract raw activity data from the Learning application for all employees under a manager, enabling managers to analyze their learning activities.

-   **Version 5.5.0 - December 2025**
    -   Changed: Read-only directive changes.
    -   Fixed: Discover Learning page performance.
-   **Version 5.4.1 - August 2025**
    -   Fixed:
        -   Internalization Issues fixed.
        -   "Learnings - Assigned to Team" Performance improved.
        -   Fixed "Assign Learning" widget which is not showing the Assigned Materials
-   **Version 5.4.0 - May 2025**
    -   Changed:
        -   Coral Theme changes
        -   Employee Persona Assistance API's for Learning
-   **Version 5.3.0 - February 2025**
    -   Fixed:
        -   Fixed A11y violations on learning model, activity page, learning task of type video content and Learning enroll page
        -   Corrected text color in Learning achievements zero state to 181A1F
-   **Version 5.2.0 - November 2024**
    -   New:
        -   Added User skill activity APIs for retrieving skill activity insights based on the below:
            -   Completed and in progress learning activities for a given skill
            -   Developing skills of a user from learning activities
            -   Top learning activities of a specific user group for a given skill
        -   Multi-lingual support for skills
    -   Updated: Implemented security directive for securing GlideAggregate/GlideRecord Encoded Query
-   **Version 5.1.5 - August 2024**

    Minor bug fixes. No major changes.

-   **Version 5.1.3 - May 2024**

    No change required.

-   **Version 5.1.1 - February 2024**

    Small changes.

-   **Version 5.0.1 - January 2024**

    Security fixes.

-   **Version 4.0.0 - August 2023**
    -   Learning is foundational in career development. It is used to build knowledge and gain skills necessary for employees to be effective in their roles and to grow towards other opportunities.
    -   The Learning application brings learning content from disparate LMS systems into the ServiceNow portal experience, allowing users to find, discover, and manage their learning courses in one place.
    -   The ServiceNow University application belongs to the Employee Growth and Development \(EGD\) product family.
