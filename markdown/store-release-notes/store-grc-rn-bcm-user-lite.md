# GRC: Business Continuity Management User - Lite release notes

Version history for the GRC Business Continuity Management User - Lite application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 11.0.0 - June 2026 \(Australia\)**

    Dependent application updated.

-   **Version 10.0.0 - March 2026**

    Fixed application dependency.

-   **Version 5.0.1 - August 2023**

    New: Added new BCM Lite Operator Role.

-   **Version 4.0.0 - February 2023**

    New: App packaging to new parent pom to remove duplicated content

-   **Version 3.0.0 - August 2022**

    The BCM Lite user application enables users to respond to the BIA questionnaires, participate in exercises by working and reporting back status on the tasks, also participate in the review and approval of the BIAs, BCPs, Exercises
