# Order Management for Business Locations release notes

Version history for the Order Management for Business Locations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.0 - June 2026**

    Fixed: Minor defect fixes

-   **Version 1.3.1 - June 2026**

    Changed: Updated the dependent order management app's version.

-   **Version 2.1.0 - April 2026**

    Changed: Renamed Service Model Foundation entities to improve clarity and maintain consistency across the platform.Please refer to ServiceNow Documentation for the label changes.Eg: "Service Organization" is renamed to "Organization Core", "Service Organization Criteria" is renamed to "Organization Criteria".

-   **Version 2.0.0 - March 2026**

    Changed: Replaced the admin role with the order administrator role \(sn\_ind\_tmt\_orm.order\_admin\) in applicable scripts.

-   **Version 1.3.0 - December 2025**

    Fixed: Minor bugs.

-   **Version 1.1.0 - August 2025**

    This application is designed to integrate Service Model Foundation with Order Management to streamline operations and optimize the end-to-end life cycle for order management.
