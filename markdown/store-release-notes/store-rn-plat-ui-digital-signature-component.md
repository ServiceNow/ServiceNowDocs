# Digital signature component release notes

Version history for the Digital signature component application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 27.3.1 - June 2026**

    Fixed: PRB1936043

-   **Version 27.3.0 - February 2026**

    Fixed: The 'Sign' button does not work when the signatory line is auto filled. PRB1962439

-   **Version 27.1.0 - August 2025**

    Fixed cursor and signature color in dark mode.

-   **Version 27.0.3 - May 2025**
    -   New: Add property to hide submit button to eliminate redundancy of duplicate signatures
    -   Fixed: One bug fix: Digital Signature PR failing for Lint Failures/Warnings
-   **Version 27.0.2 - February 2025**
    -   New: Track new event "Clear" signature and added timestamp to events
    -   Fixed: minor bug fixes
-   **Version 26.0.2 - August 2024**

    Changed: Updated the component icon in the UI builder toolbox.

-   **Version 25.0.1 - February 2024**

    Fixed various minor issues.

-   **Version 24.0.2 - August 2023**
    -   New
        -   Added accessibility features to support 2.1AA compatibility.
    -   Fixed
        -   Various fixes to address minor issues.
-   **Version 22.1.0 - November 2022**

    New: Added support for internationalization and localization

-   **Version 22.0.3 - August 2022**

    Get a user signature when required for documents or other artifacts like approvals. A user can type or draw their signature and can optionally display legal text and require user acknowledgement.
