# Now Assist for Spokes release notes

Version history for the Now Assist for Spokes application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.1 - November 2025**

    New: Adding support of Knowledge Graph and People's API for the meeting scheduler AI Agent.

-   **Version 2.1.0 - October 2025**

    Minor release with a name change for the store app.

-   **Version 2.0.1 - March 2025**

    This application enables AI capabilities in Integration Hub Spokes, such as Conversational Spoke Actions and Spoke AI Agents.
