# Site Mapping for Field Service Management release notes

Version history for the Site Mapping for Field Service Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.1 - March 2026**

    Fixed security issue in regards to files having empty roles.

-   **Version 2.0.6 - August 2025**

    Fixed a cross scope issue found during MSI Security Testing: "Execute operation on script include 'SOQueryRuleUtils' from scope 'Site Mapping for Field Service Management' was denied"

-   **Version 2.0.4 - December 2024**

    Fixed minor security bug - ACL bypass via UX Screen Condition.

-   **Version 2.0.3 - August 2024**
    -   New:
        -   Support affected product flow for work order task in mobile
        -   Ability to view indoor map for the affected products
-   **Version 1.0.1 - February 2024**

    ServiceNow Site Mapping for Field Service Management provides a native mobile solution for field service technicians and contractors visiting onsite locations to perform work. Easily visualize work sites, navigate using wayfinding, and locate assets with step-by-step directions.
