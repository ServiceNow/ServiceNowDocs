# Homepage deprecation help tool release notes

Version history for the Homepage deprecation help tool application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.2 - November 2024**

    Fixed: Applied ACLs on the table data information for the Homepage deprecation help tool.

-   **Version 2.0.1 - September 2024**

    Fixed: Support domain separation on the Homepage deprecation help tools tables.

-   **Version 1.2.2 - August 2024**
    -   Changed:
        -   Converted Core UI responsive dashboard to the new Platform Analytics dashboard
        -   Updated menu redirection
-   **Version 1.2.1 - July 2023**

    Fixed: Reports are visible by dashboard\_admin role.

-   **Version 1.2.0 - May 2023**

    No functional changes.

-   **Version 1.1.2 - February 2023**

    New: Homepages can be converted to dashboard tabs. Can be added to a new Dashboard or to existing ones as additional tabs.

-   **Version 1.0.5 - September 2022**

    Fixed: Better support for internationalization on Homepage deprecation help tool

-   **Version 1.0.4 - June 2022**

    Fixed: Under the "Menus and Favorites" Dashboard tab, all Homepages marked as favorite by all users are displayed in the List report.

-   **Version 1.0.3 - April 2022**

    Fixed: Homepage deprecation dashboard, the "Menus and Favorites" tab displays also the Homepages that have been converted or retired

-   **Version 1.0.2 - March 2022**

    The Homepage deprecation help tool identifies all homepages on an instance and enables you to migrate homepages to dashboards, retire homepages, and restore retired homepages as dashboards.
