# Oracle Netsuite Spoke release notes

Version history for the Oracle Netsuite Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.0.3 - September 2025**
    -   Added dynamic output field for Look up Record Details action.
    -   Enhanced security with KMF module access policy.
-   **Version 1.0.1 - August 2025**

    The Oracle NetSuite Spoke offers powerful, ready-to-use Spoke Actions for creating, retrieving, updating, and deleting records, seamlessly wrapping Oracle NetSuite's primary REST APIs. By simply specifying an object name such as Invoice, Receipt, or others, these dynamic Spoke Actions intelligently display the relevant input and output fields, enabling you to build sophisticated workflow automations quickly and intuitively. Customers can further leverage the ServiceNow AI Platform and other Spokes to create more complex workflows beyond imagination.
