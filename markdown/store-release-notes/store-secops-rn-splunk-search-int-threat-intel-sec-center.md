# Threat Intelligence Security Center integration with Splunk Search release notes

Version history for the Threat Intelligence Security Center integration with Splunk Search application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.5 - March 2025**

    Changed the application name from "Splunk Search integration for Threat Intelligence Security Center" to "Threat Intelligence Security Center integration with Splunk Search".

-   **Version 3.0.2 - August 2024**

    New: Supports localization.

-   **Version 3.0.0 - June 2024**

    The Threat Intelligence Security Center \(TISC\) seamlessly integrates with Splunk, by enabling comprehensive searches across the log data within the Splunk deployment. It efficiently incorporates pertinent sighting information into Observables for enhanced security analysis.
