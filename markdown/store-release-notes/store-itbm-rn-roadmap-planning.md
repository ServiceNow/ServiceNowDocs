# Roadmap Planning release notes

Version history for the Roadmap Planning on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.1 - February 2021**

    New: An elegant UI designed specifically for high level planning that enables you to adjust plans with a simple drag and drop. Powerful visualizations enable you to display and group plans based on any criteria or attribute you want.
