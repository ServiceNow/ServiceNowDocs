# GRC: Privacy Lite User release notes

Version history for the GRC: Privacy Lite User application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 19.0.0 - August 2024**

    The GRC: Privacy Lite User application gives users certain roles as lite operators. These roles let the users respond to or create certain types of tasks. The lite role ensures first-line employees can respond to or create or update privacy-related records such as Privacy Assessments, Privacy case tasks, etc. This means more people in your organization can help manage privacy risks or drive privacy compliance. The 'GRC: Privacy Lite User' plugin is available only to customers with 'Privacy Lite Operator' license.
