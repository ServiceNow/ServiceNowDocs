# Hiring Connector release notes

Version history for the Hiring Connector application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 8.0.0 - June 2026**

    No functional changes

-   **Version 7.1.0 - March 2026**

    Addressed minor defects.

-   **Version 7.0.0 - December 2025**

    Provides the integration spoke and the import set to import data from a third-party hiring system into ServiceNow for use within other hiring applications.

-   **Version 6.0.0 - August 2025**

    The application contains the Hiring Connector and other related functionalities. Hiring Connector provides the integration spoke and the import set to import data from a third-party hiring system into ServiceNow for use within other hiring applications.
