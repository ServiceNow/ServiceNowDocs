# GRC: Cybersecurity Controls Accelerator release notes

Version history for the Cybersecurity Controls Accelerator on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.0.6 - March 2026**

    Changed: Updated control objective content records with Record nature field set to Current version and State field set to Published.

-   **Version 18.1.0 - June 2024**

    Fixed: Moved CIS version 8 indicator templates to Technology Controls Monitoring Accelerator.

-   **Version 18.0.1 - February 2024**

    New: This accelerator provides pre-packaged CIS V8 controls and indicator templates to monitor them along with its common controls mapped to ISO 27001/2, PCI DSS 4.0, NIST 800-53, NIST 800-171, CSA CCM 4.0 etc.

-   **Version 16.0.0 - May 2023**

    New: Content for CIS Version 8 authority documents, citations, and control objectives.

-   **Version 15.0.1 - December 2022**

    Fixed: Reduction in installation size of the application.

-   **Version 15.0.0 - August 2022**

    Fixed: Cache flush issue on plugin upgrade.

-   **Version 12.0.0 - March 2021**

    Fixed: Fixed supporting data for indicator templates.

-   **Version 11.0.1 - October 2020**
    -   New:
        -   Content for CIS Controls Authority Document and related Citations
        -   Content for CIS Control Objectives mapped hierarchically to the citations
        -   Pre-defined indicator templates mapped to the CIS Controls for automated controls validation and continuous monitoring
        -   Pre-defined mapping of indicators with equivalent UCF controls
