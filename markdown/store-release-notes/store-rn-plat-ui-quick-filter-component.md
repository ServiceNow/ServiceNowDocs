# Quick filter component release notes release notes

Version history for the Quick filter component on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 27.2.3 - June 2026**

    Fixed various minor defects

-   **Version 27.1.4 - August 2025**

    Fixed various functional, a11y and cosmetic issues.

-   **Version 27.1.3 - May 2025**

    One bug fix: Selected filters shows disabled value fields instead of selected pills as defined in 'selectedFilters' data.

-   **Version 27.1.2 - February 2025**
    -   New:
        -   Support filtering on Date type fields
        -   Show quick filter labels conditionally based on 'showLabels' property
    -   Fixed: minor bug fixes
-   **Version 27.0.1 - November 2024**
    -   New:
        -   Support for filtering on 'strings'
        -   Support for configurable section headers in typeahead dropdown
        -   Support for configurable footer action button in quick filter
        -   Ability to configure placeholder text value for input field
    -   Fixed: Multiple minor bug fixes
-   **Version 26.0.1 - August 2024**

    Fixed: minor 1 defect.

-   **Version 25.0.2 - February 2024**

    Fixed various minor issues.

-   **Version 23.0.9 - August 2023**

    Light condition builder that enables you to create multiple filters quickly from a data source in your page and select the best subset of those filters for your users.
