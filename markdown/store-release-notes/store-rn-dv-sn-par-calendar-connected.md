# @devsnc/sn-par-calendar-connected release notes

Version history for the @devsnc/sn-par-calendar-connected application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 23.0.8 - March 2023**

    Bug fixes

-   **Version 23.0.5 - February 2023**
    -   The Calendar Data Visualization presents data from a ServiceNow table in a calendar view.
    -   The Calendar Data visualization can be used only in Next Experience. It is available in Next Experience Dashboards, UI Builder, and Visualization Designer.
    -   This application is installed automatically with "Data Visualization Core" application.
    -   No changes in the current version.
