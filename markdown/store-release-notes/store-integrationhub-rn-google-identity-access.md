# Google Identity And Access spoke release notes

Version history for the Integration Hub Google Identity And Access spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.1 - September 2022**

    Fixed: Patch version of Google Identity And Access Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.1.0 - October 2021**

    New: This version adds 3 new actions: Look up Roles in Project Stream, Look up Projects Stream, Look up Roles in Organization Stream.

-   **Version 1.0.2 - August 2021**

    Patch release of the Google Identity And Access Spoke for IntegrationHub. This version includes a fix for hard coded secrets in credentials.

-   **Version 1.0.1 - May 2020**

    Provides actions to automate the management of Google Identity and Access.
