# Enterprise Employee Experience Pack release notes

Version history for the Safe Workplace Enterprise Employee Experience Pack on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - February 2021**

    Fixed: Addressed an issue where the Enterprise Employee Experience Pack had a reference to a table that may not exist

-   **Version 1.2.0 - January 2021**

    Fixed: Update to ensure compatibility for an optional integration with the ServiceNow Workplace Core application

-   **Version 1.1.2 - August 2020**

    New: Localization for features from the July 23 release

-   **Version 1.1.1 - July 2020**
    -   This initial release provides the following configurations for a ‘Return to Workplace’ use case for organizations who are reopening locations to employees after a health crisis or similar emergency.
        -   Return to Workplace - Lifecycle event: Guides employees who are returning to work on-site after working from home for an extended period. When other ServiceNow Safe Workplace applications are also installed, additional employee tasks are included in the workflow: reserving a shift or space, completing health verification, requesting PPE, and completing a daily contact log.
        -   Return to Workplace - campaigns: Includes a portal banner and notification email content before and after a return date to keep employees informed and engaged on the changing process. For added flexibility, you can use 2 styled content items instead of portal banners for added flexibility.
        -   Lifecycle event for location readiness: Guides a work site lead and cross-departmental teams through the process of reopening a work location or area. The ServiceNow Workplace Safety Management application is required for this feature.
        -   4 sample knowledge articles for employees.
