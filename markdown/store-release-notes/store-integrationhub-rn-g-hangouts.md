# Google Hangouts spoke release notes

Version history for the Integration Hub Google Hangouts spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

Google Hangouts spoke v1.1.0 is the latest version. The Google Hangouts Spoke is now deprecated and no longer supported or available for new activation.

## Version history

-   **Version 1.1.0 - March 2023**

    New: Authentication template for easier spoke setup.

-   **Version 1.0.4 - September 2022**

    Fixed: Patch version of Google Hangouts Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.3 - November 2020**

    Patch release of the Google Hangouts spoke for IntegrationHub.

-   **Version 1.0.2 - July 2020**

    Patch release of Google Hangouts spoke for IntegrationHub.

-   **Version 1.0.1 - February 2020**

    Provides actions to automate hangouts and attendee management in Google Hangouts.
