# Redox Electronic Health Record Spoke release notes

Version history for the Redox Electronic Health Record Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.4 - January 2024**

    An Integration Hub spoke that is used to facilitate Redox outbound calls. As a hospital, payer \(insurance\), or life sciences organization, you can unlock the care capacity of multiple healthcare systems by enabling bidirectional integrations between them with the Redox Inbound Integration application. You can improve inbound and proactive outbound service scalability and capabilities by using the application to integrate with healthcare systems that use the Redox platform.
