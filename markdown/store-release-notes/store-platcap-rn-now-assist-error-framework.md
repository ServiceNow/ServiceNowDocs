# Now Assist for Error Framework release notes

Version history for the ServiceNow® Now Assist for Error Framework application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - June 2026**

    Now Assist for Error Framework is an AI-powered application that transforms how customers identify and resolve errors. It intelligently surfaces the most critical errors, prioritizes them by business impact, and provides AI-generated summaries to guide faster decision-making. Beyond visibility, the built-in AI agent can autonomously remediate errors — reducing manual effort and accelerating resolution time.
