# Security Posture Control API Connectors release notes

Version history for the ServiceNow® Security Posture Control API Connectors application on the ServiceNow Store.

-   **Version 2.1.1 - June 2026**
    -   New:
        -   Enhancements to the API Connector builder for Security Posture Control to streamline custom connector creation and improve the request/response mapping experience.
        -   Expanded support for additional authentication patterns when building custom API connectors.
    -   Fixed: Enhancements to stabilize API connector execution and data ingestion. You might see performance improvements during data imports.
-   **Version 2.0.1 - April 2026**
    -   Fixed: Saving a new connector no longer creates a duplicate, empty connector record.
    -   Changed: Removed the tar-fs library \(v2.1.3\), which was not required by the SPC Connector Framework application to remediate a known security vulnerability.
-   **Version 2.0.0 - December 2025**

    Provides the framework for the API Connector builder in the Security Posture Control \(SPC\) workspace.
