# Entitlements Verification release notes

Version history for the Entitlements Verification application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.0.0 - December 2025 \(Yokohama, Zurich\)**

    New: Introduced granular admin roles for enhanced security

-   **Version 5.0.0 - February 2025**

    New: Added location for verifying entitlements.

-   **Version 2.0.0 - November 2024**

    New: Minor updates including support data model changes.

-   **Version 1.0.0 - February 2024**
    -   The entitlement verification APIs help in querying customer entitlements to help customer service personnel to seamlessly and accurately identify the level of service or support a customer is promised.
    -   The APIs provide relevant entitlement data in response to a flexible set of input parameters.
