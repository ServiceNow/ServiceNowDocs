# Google Cloud Storage spoke release notes

Version history for the Integration Hub Google Cloud Storage spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.1 - December 2025**

    Fixed: Added missing help descriptions across most Google Cloud Storage Spoke actions to improve usability and clarity.

-   **Version 1.1.0 - October 2025**

    New: Added new AI agents for bucket access control, object access control, object management, bucket management, and improved Agent ACL capabilities.Fixed: Removed subflows from AI Agents.

-   **Version 1.0.4 - September 2024**

    Fixed: security bug.

-   **Version 1.0.3 - February 2024**

    Fixed: Visibility of actions post v1.0.2 install.

-   **Version 1.0.2 - September 2023**

    Fixed: The license requirements.

-   **Version 1.0.1 - April 2023**

    Fixed: Improve installation performance of application.

-   **Version 1.0.0 - May 2020**

    Provides actions to automate the management of Google Cloud Storage.
