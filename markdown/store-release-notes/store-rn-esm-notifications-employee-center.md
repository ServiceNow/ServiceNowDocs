# Notifications for Employee Center release notes

Version history for the Notifications for Employee Center application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.2 - June 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 2.0.9 - December 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 2.0.2 - August 2025**

    Get Notifications for Employee Center feature in the Browser Extension for Employee Center. The Browser Extension for Employee Center let's you stay connected with your Employee Center portal activities even when you're working on external websites. Note: Browser Extension feature requires the Employee Center for Browser Extension plugin.

-   **Version 1.0.3 - May 2025**

    Fixed a bug to support the Now Assist Full page/Dynamic Window experience on portal notifications.

-   **Version 1.0.2 - February 2025**

    A unified notifications experience that provides employees with timely and relevant information across multiple departments promoting engagement and increasing productivity.
