# Process Mining for Telecommunications release notes

Version history for the Process Mining for Telecommunications application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.1.0 - June 2026**

    Changed process mining for telecommunications related to dependent applications

-   **Version 5.0.0 - February 2024**

    Fixed: Change to support platform capabilities.

-   **Version 4.0.0 - August 2023**

    Dependent app.

    Changed label to Process Mining and set content pack as true for process optimization.

-   **Version 3.0.0 - February 2023**

    Changed: Internal optimization to align to NOW engineering standards.

-   **Version 2.0.1 - November 2022**

    Changed: Internal optimization to align to NOW engineering standards

-   **Version 2.0.0 - August 2022 \(compatible with Tokyo\)**

    Changed: Dependency apps

-   **Version 1.0.1 - August 2022 \(compatible with San Diego\)**

    Changed: The underlying version of Process Mining has been updated to the most recent version of Process Mining.

-   **Version 1.0.0 - September 2021**

    Provides insight to order management process owners about order fulfillment process inefficiencies.
