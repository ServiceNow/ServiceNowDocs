# Customer Service Document Templates release notes

Version history for the ServiceNow® Customer Service Document Templates application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.1.1 - June 2026**

    Non-Glide Cobalt Raven ACLs in Product Code - Security enhancements to query access controls for Customer Service Document Template.

-   **Version 2.0.0 - March 2026**

    The Customer Service document template feature enables users to configure the record fields that are included in generated documents.
