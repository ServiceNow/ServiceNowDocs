# Conversational Integration with Workplace from Facebook release notes

Version history for the Conversational Integration with Workplace from Facebook on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.0.1 - February 2025 \(Yokohama\)**

    Added translations for the store app.

-   **Version 4.0.4 - November 2023**

    Updated support for Facebook Workplace adapter input transformations on zbooted instances.

-   **Version 4.0.3 - August 2023 \(Vancouver\)**

    Fixed: Bug fixes.

-   **Version 4.0.2 - May 2023**
    -   Fixed:
        -   Date input and time topics.
        -   Redirection to Workplace login page after installation.
        -   Minor fixes.
-   **Version 4.0.1 - February 2023**
    -   New: Improved Admin UI for Workplace from Facebook integration.
    -   Fixed: Minor Bug fixes.
-   **Version 3.0.4 - November 2022**

    Fixed: Minor bugs.

-   **Version 3.0.3 - February 2022**
    -   New: Unified logging framework has been enabled.
    -   Fixed:
        -   Language issue with Live Agent.
        -   Issue with image type file picker.
        -   The unified logging backward compatibility condition.
-   **Version 3.0.2 - December 2021**
    -   New: Notification actions now display as buttons and not quick actions.
    -   Fixed:
        -   Issue with actionable and simple notifications.
        -   Issue with All type file picker not accepting files other than image files.
        -   Bot response failure due to malformed URL exception.
-   **Version 3.0.0 - October 2021**
    -   New: Make all card templates in the Conversational Integration with Workplace from Facebook available in the same template framework/table.
    -   Fixed:
        -   Email as text control input
        -   Errors on image input
        -   Rendering of table output
        -   L10N issue
        -   Action content not recognized as input
        -   Rendering of virtual machine details
-   **Version 2.0.3 - January 2022**
    -   Fixed:
        -   Language issue with Live Agent.
        -   Issue with image type file picker.
-   **Version 2.0.0 - August 2021**
    -   New:
        -   YouTube URL unfurling.
        -   Support for rich text using markdown in notification messages.
        -   URL redirect to Service Portal page.
    -   Fixes
-   **Version 1.0.1 - February 2021**
    -   Fixed:
        -   Support for custom portal URL in email notifications
        -   Guest walk-up appointment booking and reschedule is failing after upgrading
