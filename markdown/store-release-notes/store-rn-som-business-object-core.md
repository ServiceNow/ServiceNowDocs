# Business Object Core release notes

Version history for the Business Object Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.0 - June 2026**

    Fixed: Enhanced security by introducing non-glide Query Range ACLs.

-   **Version 2.1.0 - December 2025**

    Some of the current fields have their property value set to read\_only. According to the platform’s security directive, these fields must be updated to use the new value strict\_read\_only to ensure compliance with enhanced security requirements. This change does not affect functionality.

-   **Version 1.0.0 - August 2025**

    Primary data model for Lead-to-Cash Process Management application.
