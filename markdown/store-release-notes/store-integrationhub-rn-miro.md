# Miro Spoke release notes

Version history for the Miro Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.4.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 3.3.1 - December 2025**

    Fixed: Error during Update User action

-   **Version 3.3.0 - September 2025**

    Added required ACLs and Roles as per the AI Security Directive guidelines

-   **Version 3.2.1 - August 2025**

    Fixed: Miro Spoke AI Agents May Release: Tools - Output response includes anchor tags that are not rendered as clickable link

-   **Version 3.2.0 - May 2025**

    Minor release with new AI agents.

-   **Version 3.0.8 - July 2024**

    Fixed: Type matching defect in util scripts.

-   **Version 3.0.7 - April 2024**

    Fixed: Issue related to scheduled import for all users.

-   **Version 3.0.6 - September 2023**
    -   Fixed:
        -   Deactivate user action output pills
        -   Error while executing Create Team action
        -   The license requirements
-   **Version 3.0.5 - March 2023**

    Fixed: A Look up Organization Member stream action to allow default values too for Active, License and Role input fields.

-   **Version 3.0.4 - January 2023**

    Fixed - Subscription job.

-   **Version 3.0.3 - October 2022**

    Fixed: Patch version of Miro Spoke for Integration Hub. This version includes fixes for missing input validation for a few actions \(including Look up User\) and an update on documentation to use the least permissions.

-   **Version 3.0.2 - August 2022**
    -   New:
        -   Added User Management actions
        -   Added Team Management actions
        -   Added Board Management actions
        -   Added Board Item Management actions
        -   Added 2 modules: Miro Users, Miro Teams
        -   Added sample subflows and data imports
    -   Removed: Deprecated 4 user management actions that did not make use of the latest error handling mechanism
-   **Version 2.0.1 - January 2022**

    Fixed: In 2.0.1 version, fixes are made to support backward compatibility to Quebec.

-   **Version 2.0.0 - August 2021**

    In this release, updates are made to incorporate the Miro API end points.

-   **Version 1.0.3 - June 2021**

    Minor release of the Miro spoke for IntegrationHub.

-   **Version 1.0.2 - May 2021**

    New: Use this spoke to view and analyze usage data for Miro software. Take actions, such as reclaiming stale licenses, to determine your potential savings.

-   **Version 1.0.0 - March 2021**

    New: The Miro spoke provides actions to see and understand meaningful usage data for Miro software subscriptions. With this spoke, you can analyze licensing information for Miro.
