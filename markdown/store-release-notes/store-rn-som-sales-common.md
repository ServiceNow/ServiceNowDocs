# Sales Common release notes

Version history for the ServiceNow® Sales Common application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 6.9.1 - June 2026**

    Fixed: Minor security fixes

-   **Version 6.8.1 - May 2026**

    ACLs were added for Organization Quote Viewer and Organization Order Viewer to allow quote/order line level access.

-   **Version 6.6.3 - April 2026**

    Updates in this release contain library level functions to support the price ramp feature in Quote Management and Order Management.

-   **Version 6.5.0 - March 2026**

    Fixed: Minor defects fix.

-   **Version 6.3.2 - January 2026**

    Fixed a minor defect.

-   **Version 6.3.0 - December 2025**
    -   The Sales Common Application gives access to the roles listed below and other shared utilities for opportunity and quote. These roles are responsible for creating opportunities and quotes.
        -   Sales Agent
        -   Sales Manager
        -   Sales Ops Specialist
