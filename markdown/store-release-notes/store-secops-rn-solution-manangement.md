# Vulnerability Solution Management release notes

Version history for the Security Operations Vulnerability Solution Management on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.4.1 - December 2022**

    Minor fixes for this release.

-   **Version 10.4.1 - November 2022**

    Minor fixes for this release.

-   **Version 10.4.0 - March 2022**
    -   New:
        -   Generic framework based on the Common Vulnerability Reporting Framework \(CVRF\) has been introduced so that you can build the integrations for faster information exchange and processing. Major vendors including Oracle, Cisco, Red Hat and Microsoft provide CVRF format for describing vulnerabilities.
        -   The solution data can be imported through file import or by using an API.
-   **Version 10.3.0 - June 2020**
    -   New:
        -   Recertified for version 10.3.0.
        -   Added Red Hat Solution Intelligence Integration for Security Operations. The Red Hat solution intelligence integration correlates the vulnerabilities in your environment with the Linux solutions recommended to remediate them.
-   **Version 8.0.1 - June 2019**

    Refer to Vulnerability Response release notes for product changes and updates in the Madrid release.
