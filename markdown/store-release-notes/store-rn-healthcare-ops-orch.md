# Healthcare Operations Orchestration release notes

Version history for the ServiceNow® Healthcare Operations Orchestration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.2.0 - June 2026**

    Version bump only

-   **Version 1.1.0 - May 2026**
    -   New: Data Archival Policies: Archival policy records for Orchestration Cases \(ORC\) and Orchestration Tasks \(ORT\) enable automated data lifecycle management, supporting record retention and data governance requirements.
    -   Enhanced:
        -   Playbook Review Stage
            -   Plan Review Summary Layout: The plan review summary page now displays task template items with an updated layout, improving readability and navigation during the playbook review workflow.
            -   Edit and Publish: From within the playbook review activity, plan authors and agent managers can edit template details inline and publish the template directly, streamlining the review-to-publish workflow.
        -   Associated Organizations on Template Items: Template item records now display a related list of associated service organizations, giving plan authors visibility into which organizational units a template item applies to.
        -   Care Team Portal URL Routing: URL mapping for the Care Team Portal ensures navigation links route correctly across portal pages.
-   **Version 1.0.0 - March 2026**

    Healthcare Operations Orchestration is an application that enables operational leaders and care team managers to coordinate complex, multi-department healthcare operations through orchestration cases, task plan templates, and playbook-driven workflows integrated with Planned Work Management \(PWM\).
