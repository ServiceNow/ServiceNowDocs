# Workplace Service Delivery Integration with Microsoft Places release notes

Version history for the Workplace Service Delivery integration with Microsoft Places application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.8 - March 2026**

    Security fixes.

-   **Version 1.2.5 - December 2025**

    No updates.

-   **Version 1.2.3 - August 2025**

    Increasing version number to align with the releases.

-   **Version 1.2.0 - May 2025**
    -   Fixed:
        -   The Open Requests widget was showing a higher-than-expected count of requests due to a display discrepancy.
        -   Accessibility improvements.
-   **Version 1.1.1 - February 2025**

    Fixed: Bugs related to translations and internationalization.

-   **Version 1.1.0 - November 2024**

    New: Employees can raise 5 new service requests - Space Arrangement, General Request, Desk Change, Employee Parking, Furniture Request

-   **Version 1.0.3 - August 2024**

    ServiceNow Workplace Service Delivery integration with Microsoft Places application extends the Workplace Case Management integration with Microsoft Places to enable employees to more effectively request and receive services from within Microsoft Places or Microsoft Teams. When used with Workplace Reservation Management and Workplace Reservations for Microsoft Outlook Add-in, users are able to raise service requests and associate them to the existing reservations. Technicians and fulfillers can continue to service and resolve the workplace cases using Workplace Case Management.
