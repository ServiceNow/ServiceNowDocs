# Workforce Optimization for CSM Configurable Workspace release notes

Version history for the Customer Service Management Workforce Optimization for CSM Configurable Workspace on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.0.1 - June 2026**
    -   Added interoperable route for CSM record pages in Manager Workspace
    -   Removed CSM default record page interoperable
    -   Fixed padding issues and minor defects.
-   **Version 5.0.0 - March 2026**
    -   Enhance the Manager Dashboard to view:
        -   New AI Insights tab on the Manager Dashboard in the Manager Workspace \(Enterprise SKU\).
        -   The AI-powered widgets: Sentiment Analysis, Trending Topics, and Auto QA. These widgets provide deeper, real‑time visibility into customer sentiment signals, trending interaction themes, and quality evaluations.
    -   Fixed minor bugs.
-   **Version 4.5.0 - December 2025**

    Security enhancement and minor bug fixes.

-   **Version 4.4.0 - August 2025**

    Minor bug fixes.

-   **Version 4.3.3 - May 2025**

    Fixed: Minor issues.

-   **Version 4.3.1 - March 2025**

    Minor fixes.

-   **Version 4.3.0 - February 2025**

    The new Manager Workspace landing page provides managers with real-time alerts and key metrics that reflect the overall health and performance of the organization. By consolidating work and resource management data, managers can effectively oversee team performance, optimize agent utilization, and drive continuous improvement.

-   **Version 3.9.3 - November 2024**

    Fixed: Security and minor issues.

-   **Version 3.9.0 - August 2024**

    Minor fixes.

-   **Version 3.8.1 - May 2024**

    Minor fixes.

-   **Version 3.7.1 - May 2024**

    Minor fixes.

-   **Version 3.6.2 - March 2024**

    New: Agent shift sign-up feature is now available for CSM/FSM configurable workspace.

-   **Version 3.6.1 - February 2024**
    -   New:
        -   New out-of-the-box metrics for the Amazon Connect voice channel, Whatsapp, Line, SMS and Facebook messenger
        -   Outlook integration with shift planning events
    -   Minor fixes.
-   **Version 3.5.0 - November 2023**

    New: Manage work of all your teams from a central location using the Work Scheduler.

-   **Version 3.3.1 - November 2023**

    Fixed: Pushed fixes.

-   **Version 3.3.0 - May 2023**

    Changed:

    -   Create volume forecast at daily interval up to a year \(52 weeks\).
    -   Option to create generate volume forecast at 15/30/60 minutes interval.
-   **Version 3.2.0 - February 2023**
    -   New
        -   Events for User Experience Analytics added to
            -   Schedules &gt; Forecast Tab
            -   Team Performance and user profile page
            -   Shifts and Schedules
        -   Telemetry for Agent self enroll and recommendation of courses and paths - My Learning page, Overview and Discover pages
        -   Added Table Definitions for Coaching and Learning, Skills,Forecasting,
-   **Version 3.1.1 - November 2022**
    -   Support for 15 minute forecast intervals while generating volume forecasts
    -   Performance improvements
    -   Major fixes
-   **Version 3.0.0 - August 2022**
    -   New
        -   Auto-rejection of duplicate/overlapping time off requests
        -   Resource and volume forecasts at 15 minutes granularity
    -   Fixed: Localization support and minor bug fixes
-   **Version 2.3.0 - May 2022**
    -   New
        -   The manager can override the agents' presence state for their team.
        -   Added navigation menu items required for the manager role.
        -   Demo data is now available as part of this app.
    -   Changed
        -   The agent availability scorecard on the homepage will have the 'Service channel' column on drill-down.
        -   After editing and saving 'time worked', the success notification will have the agent name hyperlinked to the profile page.
    -   Fixed: Fixed a few UI and accessibility issues.
-   **Version 2.1.0 - March 2022**

    New: New configurable workspace experience for the WFO manager persona

-   **Version 1.0.0 - May 2021**
    -   New:
        -   Optimize shift planning by allowing team members to manage their shifts and schedules through day/week/month views
        -   Streamline the submission of leave and shift swap requests by team members
        -   Trigger standard or conditional approval logic configured in Flow Designer after team members submit requests
