# Product Catalog Advanced release notes

Version history for the Product Catalog Advanced on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.3.0 - March 2026**

    Fixed: Product Offering TMF API to query and create new product offerings

-   **Version 10.2.0 - December 2025**

    Changed: Generate rule set input variables, when characteristic options are added on a published product specification.

-   **Version 10.0.0 - August 2025**
    -   New:
        -   Support of additional data types for product characteristics. New types supported are Decimal, Integer, Date, Date Time, Array, and Object
        -   Support for Complex conditions and attributes of Object data type in all the fulfillment rules: compatibility, decomposition and attribute propagation
-   **Version 9.1.0 - May 2025**

    Changed: Bundle offers support for product catalog API in POST, GET, CHANGE, and DELETE operations and submit.

-   **Version 8.0.2 - February 2025**

    Fixed: Minor internal bugs.

-   **Version 8.0.1 - November 2024**

    New: Ability to control display order of optional specifications and characteristic options in the Configurator UI.

-   **Version 8.0.0 - August 2024**

    New: Changes made to support bundling of product offers that have references to spec or spec hierarchy.

-   **Version 7.0.1 - May 2024**

    Minor fixes.

-   **Version 6.0.0 - February 2024**
    -   New: Product Offering Category API support
    -   Changed: Product Offering API changes to accept category information
-   **Version 5.0.0 - November 2023**

    New: Move API code to the new telecommunications application \(sn\_om\_tmt\).

-   **Version 4.0.0**
    -   New:
        -   Support network slicing for 5G service \(Connected Navigation\)
        -   Jeopardy Management
-   **Version 3.0.0 - February 2023**
    -   Changed: Internal optimization to align to NOW engineering standards
    -   Fixed: Resolved various reported issues.
-   **Version 2.0.2 - November 2022**

    Changed: Internal optimization to align to NOW engineering standards

-   **Version 2.0.0 - August 2022**

    New: Support catalog versioning for service, resource specification and product inventory

-   **Version 1.0.1 - June 2022**

    Changed: Dependency application version has been updated based on family true up version

-   **Version 1.0.0 - February 2022**

    Product Catalog Advanced contains updated capabilities in the product catalog including service specifications, resource specifications, and specification relationships.
