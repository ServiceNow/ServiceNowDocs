# Talent Feedback release notes

Version history for the Talent Feedback application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - June 2026**

    Reviewed cross-scope access checks for the getRefRecord\(\) API..

-   **Version 1.2.0 - March 2026**
    -   For the Mar'26 release, the Talent feedback app was enhanced to include support for feedback templates in the Employee feedback collection AI agent.
        -   Using the Employee feedback collection AI agent, managers can collect feedback on their reportees.The agent recommends collaborators using data from employee profile, Talent development applications, employee journeys and optionally, from collaboration data from Microsoft Graph
        -   In the previous release, the agent supported only a single type of feedback request \(an open ended feedback request where a manager can add a custom note to the feedback provider and request a text response\).
        -   From Mar'26, managers can select from a list of available feedback templates \(configured by the admin using Platform Surveys tool\), improving relevance and actionability of feedback
        -   Managers can also see a summary of the recent feedback collected for an employee and navigate to a feedback page in Manager Hub to view all collected feedback
-   **Version 1.1.0 - December 2025**

    The Talent feedback data model has been enhanced to support newly introduced feedback types: The Employee Feedback Collection AI Agent now captures additional feedback types enabled by the updated data model, allowing managers to gather more structured and context-specific insights.

-   **Version 1.0.2 - August 2025**

    No new customer-facing changes in this release.

-   **Version 1.0.1 - May 2025**

    Talent feedback application enables managers to gather comprehensive feedback on employees' skills, allowing for a thorough assessment and validation of their skills. By collecting feedback alongside skill activity insights, it provides managers with valuable input to validate employees' skills.
