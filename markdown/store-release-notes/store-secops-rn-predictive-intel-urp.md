# Predictive Intelligence for User Reported Phishing release notes

Version history for the Predictive Intelligence for User Reported Phishing application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.5.0 - June 2026**

    Changed: Updated Non-Glide Cobalt Raven ACLs for Security Incident Response.

-   **Version 10.3.7 - September 2024**
    -   Fixed:
        -   Error when saving sn\_sir\_ml\_urp\_config with custom close code
        -   Predictive Intelligence for phishing emails not working in some cases
-   **Version 10.3.6 - May 2023**

    Fixed: Predictive Intelligence URP ML - Feature record gets created for Aggregated Phishing records.

-   **Version 10.3.3 - August 2022**

    Changed: Created usage metrics records and updated the solution store.

-   **Version 10.3.1 - June 2020**

    New: Ability to get close codes for the training data set from custom fields.
