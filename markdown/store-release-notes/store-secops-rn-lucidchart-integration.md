# Lucidchart Integration release notes

Version history for the Security Operations Lucidchart Integration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.4.1 - February 2025**
    -   Fixed:
        -   Order issue of Business Capabilities
        -   Disabled the create diagram action for APM read role
-   **Version 2.4.0 - November 2024**

    Fixed: Access control fixes.

-   **Version 2.3.0 - August 2024**

    Changed: Modified underlying Data Model to support common framework for Enterprise Modeling and Visualization.

-   **Version 2.2.1 - January 2024**
    -   New: Individual users can view and select a folder name while creating a Lucidchart diagram.
    -   Fixed: Individual users can authenticate themselves and use their own Lucid folder where the Lucidchart diagram can be saved.
-   **Version 2.2.0 - August 2023**
    -   New: Added a Folder ID field to create the Lucid diagram
    -   Changed: Changed - In Entities, for the Business Application hierarchy map, added Digital Integration and Digital Interface
-   **Version 2.0.0 - November 2022**

    Now, you have the option to include or exclude related business applications in the Business Capability diagram.

-   **Version 1.0.0 - August 2022**

    Enables integration of Application Portfolio Management with Lucidchart. Allows Enterprise Architects to create Lucidchart diagrams for Business Application hierarchy and Business Capability maps.
