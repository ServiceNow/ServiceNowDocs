# Adobe Experience Platform release notes

Version history for the Integration Hub Adobe Experience Platform on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.0 - January 2025**

    Enhanced to OAuth 2.0 due to JWT deprecation.

-   **Version 2.1.2 - April 2023**

    Fixed 2 actions that had an extra space in the headers of rest action.

-   **Version 2.1.1 - September 2022**

    Patch version of Adobe Experience Platform Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 2.0.0 - June 2020**
    -   New:
        -   Added new actions and handlers for AEP Sandboxes.
        -   Upgraded to use authentication templates.
    -   Removed: Removed the JSON response blobs and used object data pills.
    -   Fixed: Fixed a token expiry mismatch.
-   **Version 1.0.0 - December 2019**

    Provides actions to automate dataset, entity, and metadata retrieval management in Adobe Experience Platform.
