# Mobile Card Builder release notes

Version history for the Mobile Card Builder on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 27.13.1 - June 2026**

    New: Removed obsolete ACL role.

-   **Version 26.12.0 - August 2025**
    -   New:
        -   Integrated Mobile App Builder into ServiceNow Developer Studio for a seamless development experience.
        -   Added support for Dark Theme and Mako Theme to improve visual customization.
-   **Version 26.11.2 - June 2025**

    Fixed: Added the missing IDs to display the correct icons.

-   **Version 26.11.0 - February 2025**

    Fixed: Accessibility audit bugs.

-   **Version 25.10.0 - November 2024**
    -   New: Predefined system button type support for button configuration.
    -   Fixed: Added missing style variables to improve consistency and visual coherence across components.
-   **Version 24.9.1 - August 2024**
    -   New:
        -   Create a card from scratch in Mobile Card Builder.
        -   Open Mobile Card Builder within the same browser tab as Mobile App Builder to streamline transitions between the tools.
    -   Fixed: Resolved issues to improve usability, and performance.
-   **Version 24.8.3 - February 2024**

    Fixed: UX quality improvements to ensure compatibility with the Washington release.

-   **Version 24.8.2 - January 2024**

    Updated instance security.

-   **Version 24.8.0 - November 2023**
    -   New:
        -   Updated user experience with the inclusion of the following changes:
            -   Implementation of a feature to notify users when a new version of Mobile Card Builder is ready for installation.
            -   Improvement of the preview capability due to the inclusion of color variables.
            -   Added more options to text and button element typography styles.
        -   Ability to create an alternative text screen reader label for buttons labelled as icon only.
    -   Fixed: Resolved issues to improve customer experience, usability and performance.
-   **Version 23.7.1 - August 2023**

    Fixed: Removed dependency on the plugins that are not used in Mobile Card Builder.

-   **Version 23.7.0 - May 2023**
    -   New:
        -   Changes to UI Rule configuration -Timeago formatting customizations &amp;Condition builder support for consuming card fields &amp; mobile scratchpad script
        -   Updated Mobile Card Builder to use Next Experience branding colors and logos.
    -   Fixed:
        -   Resolved issues to improve customer experience, usability, and performance.
        -   Making the application more accessible by fixing a11y related issues.
-   **Version 21.6.1 - February 2023**
    -   New:
        -   Improved open screen previews, which are more representative of the type of cards that users view on their mobile device.
        -   A built-in condition builder to assist with UI rule configuration, enabling users to visually create conditions without the need to define and manually enter condition strings.
        -   Added support for the following UI rule options:
            -   Display the date as a time period. For example, 2 hours ago and 3 weeksago.
            -   Display of a text decorator, which is an icon that can be added next to text to emphasize thelabel.
    -   Fixed:
        -   Resolved issues to improve customer experience, usability and performance.
        -   Making the application more accessible by fixing a11y related issues.
-   **Version 21.5.0 - August 2022**
    -   Changed:
        -   Support Next Experience and UX refresh design features, including:
            -   Typography
            -   Adding icon video elements to cards
        -   Updated MCB elements to support following features:
            -   Ability to add image as a background for a container.
            -   Updated color picker for new theme colors.
            -   Theme colors on buttons.
    -   Fixed:
        -   Resolved issues to improve customer experience usability.
        -   Fixed a11y related issues to make the application more accessible.
-   **Version 21.4.0 - May 2022**
    -   Changed: Upgraded to use the latest version of Now Design System \(NDS\) UI components.
    -   Fixed: Resolved usability, accessibility and performance issues in Mobile Card Builder
-   **Version 19.3.3 - February 2022**
    -   New: Added support to configure UI rules and UI rule actions.
    -   Changed: Updated error handling to bubble up and display meaningful error messages to users.
    -   Fixed: Resolved issues to improve customer experience and usability.
-   **Version 1.2.0 - July 2021**

    Minor fixes.

-   **Version 1.1.1 - March 2021**
    -   New:
        -   Support for Card Action functionality
        -   Accessibility updates
-   **Version 1.0.6 - January 2021**
    -   New:
        -   Create and edit templates for use on mobile cards
        -   Choose/change the template used on a mobile card
        -   Choose the data you want to see on your mobile card
