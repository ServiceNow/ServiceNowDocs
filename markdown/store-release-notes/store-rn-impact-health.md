# Impact Health release notes

Version history for the Impact Health application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.3 - December 2025**
    -   Changed:
        -   The Proactive Code Check UI action has been moved to the Related Links section on the Update Set form.
        -   This app will be available only for the existing customers
-   **Version 1.3.3 - September 2025 \(Zurich\)**

    No updates.

-   **Version 1.3.2 - September 2025 \(Xanadu and Yokohama\)**

    No updates.

-   **Version 1.3.1 - August 2025 \(Zurich\)**

    New: Localisation support for a few languages

-   **Version 1.3.0 - August 2025 \(Yokohama and Xanadu\)**

    New: Localisation support for a few languages

-   **Version 1.2.0 - June 2025**

    Minor bug fixes.

-   **Version 1.1.1 - March 2025**

    The Impact Health App improves platform health by enabling proactive management of instance quality, compliance, and performance. As a companion to the Impact Store App, it is designed to support platform health and ensure adherence to best practices.
