# Project Costing for Sourcing and Procurement Operations release notes

Version history for the Project Costing for Sourcing and Procurement Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.1 - June 2026 \(Australia\)**

    Changed: Remediated non‑Glide Cobalt Raven ACLs in product code \(DIRS0000421\).

-   **Version 2.14.0 - March 2026**
    -   Changed:
        -   Enhanced the shopping experience by enabling shopping on behalf of others, with improved visibility and management of these purchases.
        -   Users can select delegates without the delegate route, view business owner purchases in My Purchases, and modify purchases made on behalf of others.
-   **Version 2.11.0 - December 2025 \(Yokohama\)**

    Changed: Enhanced security by replacing the broad admin role with multiple granular sub-administrative roles, enabling more precise permission assignments for specific administrative tasks. This update includes changes to system properties, ACLs, roles, and flows.

-   **Version 2.10.0 - May 2025**

    Add deny unless ACLs for SPO apps.

-   **Version 2.8.7 - February 2025**

    Minor bug fixes.

-   **Version 2.8.1 - November 2024**

    Evaluated sandbox access for client-callable script.

-   **Version 2.5.6 - May 2024**

    Minor fixes.

-   **Version 2.5.0 - February 2024**

    Changed: Introduced minor content changes to the intake form that an employee or requester fills in with project details to be captured in the integration experience with the Strategic Portfolio Management \(SPM\) product. Note: These project questions are only available on the intake form if the employee or requester has access to SPM. There is no impact on existing customers.

-   **Version 1.2.7 - December 2023 \(Utah\)**

    Minor fixes.

-   **Version 2.0.1 - November 2023**

    Minor fixes.

-   **Version 2.0.0 - August 2023**

    Changed: In Shopping Hub, only shoppers who have explicit access to the project tables will now be able to associate a project with a request. Previously, this was implicit and users were automatically enabled to select project information. This was causing fulfiller license overage issues. Note: Impacts customers whose shoppers were not given explicit access to the project tables.

-   **Version 1.2.6 - June 2023**

    Fixed: Minor bug fixes.

-   **Version 1.2.5 - May 2023**

    Changed: Renamed this application to Project Costing for Sourcing and Procurement Operations. This application was formerly named Procurement with Project Management.

-   **Version 1.2.3 - February 2023**
    -   Changed: Modified the Procurement with Project Management better together workflow to address a new state value introduction in the Invoice table.
        -   **Note:** This is not a business functionality change, just a technical change. Only existing customers with the Procurement with Project Management application will see these changes.

-   **Version 1.2.2 - November 2022**

    Minor fixes.

-   **Version 1.2.1 - August 2022**

    Minor fixes.
