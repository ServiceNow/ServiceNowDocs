# Team Performance release notes

Version history for the ITSM Team Performance application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 202603.0.0 - March 2026**

    Fixed: PRB1990164 - When adding KPIs to a KPI group of type 'teams' the search icon shows KPIs of all types, instead of only showing KPIs where assigned\_to or assignment\_group is added as a break down.

-   **Version 202601.0.2 - January 2026**
    -   Changed: Deprecated process mining / optimization from WFO \(workforce optimization\)
    -   Fixed:
        -   PRB1916913 - team performance manager role are now able to add KPIs to KPI groups
        -   PRB1940347 - KPI records for the monthly frequency indicator are now available in DPM
-   **Version 202507.0.2 - August 2025**

    Minor bug fixes.

-   **Version 7.2.3 - February 2025**

    Fixed: On a KPI record, the display aggregate fields are now hidden when latest score = true, as latest score is never an aggregate.

-   **Version 7.2.1 - November 2024**

    Minor bug fixes.

-   **Version 7.1.5 - August 2024**

    Minor bug fixes and family compatibility upgrades.

-   **Version 7.1.0 - February 2024**

    Minor fixes.

-   **Version 7.0.1 - November 2023**
    -   Changed:
        -   Reduce number of process mining projects, by creating only one project for one KPI group and table combination.
        -   Minor performance improvements.
    -   Minor fixes.
-   **Version 5.1.15 - September 2023 \(Utah\)**

    Fixed: Issue where Applicable groups were not honored on the Teams module.

-   **Version 6.0.1 - August 2023**

    Changed: Using a default template for Process Mining.

-   **Version 5.1.14 - August 2023**

    Fixed: Issue related to the latest score for Digital Portfolio Management.

-   **Version 5.1.8 - May 2023**
    -   New: Users can associate multiple KPI groups to one or more assignment groups.
    -   Minor fixes.
-   **Version 5.1.5 - February 2023**

    New: Minor UI change

-   **Version 5.1.2 - November 2022**
    -   New: Added the Records, Metrics, and Process Analysis tabs for all the parent and child KPIs.
    -   Minor fixes.
-   **Version 5.1.1 - August 2022**

    Get insights into your teams' performance and analyze anomalies or bottlenecks within the process when you drill down into KPIs.
