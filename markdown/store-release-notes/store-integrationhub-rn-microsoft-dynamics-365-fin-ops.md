# Microsoft Dynamics 365 for Finance and Operations spoke release notes

Version history for the Microsoft Dynamics 365 for Finance and Operations spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.5.0 - June 2026**

    Security patch on non-glide ACLs

-   **Version 2.4.3 - January 2026**

    Fixed: Triggering multiple get/refresh metadata causing multiple attachments to data source.

-   **Version 2.4.1 - November 2025**

    Security patch: Improved sample webhook registry security.

-   **Version 2.3.1 - October 2025**

    Created Contain Role and updated the script include as per the security guidelines.

-   **Version 2.3.0 - September 2025**

    Created Contain Role and updated the script include as per the security guidelines

-   **Version 2.2.0 - November 2024**

    Evaluated and modified Sandbox Access for Client-Callable Script Includes as false.

-   **Version 2.1.0 - May 2024**
    -   Updated the existing Auth Template to ensure OAuth token retrieval from other caller scope
    -   Added a sample inbound subflow
-   **Version 2.0.2 - September 2023**

    Fixed: 'licensable' field set to be true in plugin.xml.

-   **Version 2.0.1 - July 2023**

    Optimized performance.

-   **Version 1.1.0 - November 2021**

    New: Added 4 new actions.

-   **Version 1.0.1 - October 2021**

    The Microsoft Dynamics Finance and Operation Spoke provides a list of actions that wraps around the Dynamics primary REST APIs.
