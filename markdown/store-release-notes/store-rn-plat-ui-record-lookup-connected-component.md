# Record lookup connected component release notes

Version history for the Record lookup connected component application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 28.0.2 - June 2026**

    Fixed various minor defects

-   **Version 28.0.1 - December 2025**
    -   New: icons for lookup card actions
    -   Fixed: various cosmetic issues
-   **Version 27.1.6 - August 2025**

    Fixed performance issue caused by redundant data resource call

-   **Version 27.1.4 - August 2025**

    Fixed performance issue caused by redundant data resource call.

-   **Version 27.1.3 - May 2025**
    -   New:
        -   Domain separation for lookup
        -   Linked card header update
        -   aria-live announcements within lookup
        -   New UIB property panel updates
        -   Ability to enable a "Verify" button and show a responsive highlighted "Verified" tag on selected lookup card
    -   Changed: Linked card header functionality improved in collapsed mode
    -   Fixed: Multiple bug fixes
-   **Version 27.1.1 - February 2025**
    -   Changed: Linked card now auto-refreshes when there is a backend update
    -   Fixed: various minor bugs and cosmetic changes
-   **Version 27.0.3 - November 2024**
    -   New:
        -   Ability to hide record header
        -   Increased flexibility to configure search results in lookup
        -   Configure lookup to render highlighted values in the linked card
        -   Ability to qualify search results using specific conditions \(reference qualifiers\)
        -   Ability to hide fields with empty values
        -   Ability to hide link button
        -   Ability to expand/collapse card
    -   Fixed: Multiple minor bug fixes
-   **Version 26.0.1 - August 2024**
    -   New: true-up this store app in Xanadu
    -   Fixed: minor defects
-   **Version 25.0.1 - February 2024**

    When handling a case, sometimes not all necessary information is available. The 'Lookup' component simplifies this process. It allows for quick searches of existing contacts, products, accounts or adding new ones. Once a user searches for a contact, they can preview multiple options without leaving their current context, select the correct contact from the results, and link them to the case.
