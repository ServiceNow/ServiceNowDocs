# Skills Intelligence release notes

Version history for the Skills Intelligence on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.0 - August 2023**

    New:

    -   Support LinkedIn import for employee skills data.
    -   Support importing skills data through a resume in docx format.
    -   Validate career data before importing.
    -   Validate skills in Manager Hub for Growth Conversations.
    -   Auto-populate required Proficiency levels across the roles by skill level type.
    -   Automatic mapping of the role level to the employee profile.
    -   Update and validate skills frequently using Proactive Prompts.
-   **Version 2.0.0 - May 2023**

    New:

    -   Skills Profile -As an employee, you can view, add, and edit your skills and related information in a central place. The skills profile will help you plan your growth and development more effectively by providing a clear overview of your current skill set.
    -   Suggest skills based on user's job title/job codes
    -   Job architecture schema design - Customers can recreate their job architecture data relations within the ServiceNow platform for skills uses cases
    -   API returning valid skills from cmn based on string
    -   API for Algorithm to determine employee skill gap
    -   API for Fuzzy skill matching for adjacent/related skills
    -   Import HCM profile data
