# Security Incident Response Mobile app release notes

Version history for the Security Incident Response Mobile on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.4.2 - June 2026**

    Fixed: The color contrast of the "2 - High" text in toast messages has been improved for accessibility. The text color now uses \#B00000 to meet WCAG guidelines.

-   **Version 10.4.1 - March 2026**

    Fixed: Fixed and issue where using the "Assign to me" function on a Security Incident record modified the Assignment Group field.

-   **Version 10.4.0 - November 2024**

    Fixed: You can now reopen closed Security Incidents \(SI\) or Security Incident Tasks \(SIT\) on SIR Mobile.

-   **Version 10.3.7 - September 2024**

    Fixed: Outcome Field on Security Operations Mobile in the Mobile Agent App does not function as a choice field like it does on Desktop.

-   **Version 10.3.6 - February 2024**

    Fixed: Clicking on notifications on Security Incident Response incidents or tasks assigned to you results in an internal server error.

-   **Version 10.3.5 - July 2023**

    Fixed: The edit action is missing on the latest Security Incident Response Mobile app.

-   **Version 10.3.4 - May 2023**

    Fixed: Next Experience dark theme issues.

-   **Version 10.3.2 - November 2020**

    Changed: As part of the inclusive language initiative, the Master Item label has been changed to Primary Item label.

-   **Version 8.0.6 - August 2019**

    Fixed:

    -   Sub category options update when category is selected
    -   Fixed push notification issues for incident assignment
    -   Fixed discrepancy for assigned to options in mobile versus desktop client
-   **Version 8.0.5 - July 2019**
    -   As a security analyst, you can use the Security Incident Response Mobile app to access a ServiceNow AI Platform® instance from your Android or iOS mobile device.
    -   You can receive notifications when security incidents or security response tasks are assigned to you or to your assignment groups and begin remediation without being tied to the desktop.
    -   Monitor the security incidents the Security Operations Center \(SOC\) or your assignment groups are responding to with search criteria that you enter.
