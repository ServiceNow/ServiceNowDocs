# Carbon Black Integration for Security Operations release notes

Version history for the Carbon Black Integration for Security Operations on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.3.7 - November 2024**

    Changed: Migration of Workflows to Flow Designer flows.

-   **Version 10.3.4 - February 2023**

    Fixed: Supports Type changes to display on the Security Incident Response workspace.

-   **Version 10.3.2 - July 2022**

    Fixed: Isolate host issue resolved and which enables customer to do host isolation without any issues.

-   **Version 10.3.1 - June 2020**

    New: Implementation Flow to support the new capability framework \(v2.0\).

-   **Version 9.0.0 - November 2019**

    Fixed: Minor bug fixes.

-   **Version 8.0.8 - July 2019**
    -   New: Recertified for New York
    -   Fixed: Bug fixes
