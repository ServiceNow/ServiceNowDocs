# Core Business Suite Foundation for Source to Pay release notes

Version history for the ServiceNow® Core Business Suite Foundation for Source to Pay application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.5 - June 2026**

    Shipped out-of-the-box assignment rules for the Core Business Suite Foundation for Source to Pay. The goal is to give S2P admins immediate visibility into existing rule configurations without manual setup, so they can review and understand which rules are active by default.

-   **Version 3.0.0 - April 2026**

    The Core Business Suite Foundation for Source to Pay provides pre-configured workflows for streamlined case and knowledge management, along with foundational AI capabilities for the Source to Pay department.
