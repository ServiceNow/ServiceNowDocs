# Service Graph Workspace - Content release notes

Version history for the ServiceNow® Service Graph Workspace - Content application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.0 - March 2026**

    This application contains content for the Data Model Navigator application used by the Explore feature in the Service Graph Workspace. It currently works in conjunction with the CMDB Workspace v9 and above, where the full Service Graph Workspace can be found.
