# Voice Controls Simulator Tool release notes

Version history for the ServiceNow® Voice Controls Simulator Tool application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.5.0 - June 2026 \(Australia\)**
    -   Fixed in this release:
        -   The following ARIA-label attribute accessibility fixes were made:
            -   Close button
            -   Toggle buttons for Queue, Agent, External, and Transfer tabs
            -   Info button icon
            -   Transfer call button
-   **Version 1.4.1 - May 2026 \(Australia\)**

    Plugin available for the current bundle for Interaction Controls Component \(ICC\).

-   **Version 1.3.0 - April 2026 \(Australia\)**

    New in this Release: None - Inclusion for packaging Interaction Controls Component only.

-   **Version 1.2.0 - March 2026**
    -   New in this release:
        -   The Voice Controls Simulator Tool supports the transfer list when the application is zoomed up to 400%.
        -   The Voice Controls Simulator Launch page displays without requiring a browser refresh after closing or ending the simulation.
        -   When configuring the Voice Simulator for outbound calls, a message displays informing users that an interaction will be created.
-   **Version 1.1.0 - December 2025**

    TheVoice Controls Simulator tool enables CCaaS partners and customers to simulate and test voice control UI flows within theCSM Configurable Workspace, providing a comprehensive environment for validating call-related interactions.
