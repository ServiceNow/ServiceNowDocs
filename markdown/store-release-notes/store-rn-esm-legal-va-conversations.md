# Legal Virtual Agent Conversations release notes

Version history for the Legal Virtual Agent Conversations on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.1 - August 2022**

    Changed: If Universal Request integration is available, then the Legal Request virtual agent topic generates a Universal Request number instead of Legal Request number.

-   **Version 1.2.2 - December 2021**

    Changed: Virtual Agent conversation topics for Legal Service Delivery are grouped under the Legal Service Delivery category.

-   **Version 1.2.1 - September 2021**

    Fixed: Fixes and minor changes.

-   **Version 1.1.1 - June 2021**

    New: Option to connect to a live agent in the Legal department

-   **Version 1.0.1 - March 2021**
    -   New:
        -   Automates handling of repetitive requests
        -   Reduces the response time to your employee needs
        -   Integrates with legal services to create requests on behalf of employees
