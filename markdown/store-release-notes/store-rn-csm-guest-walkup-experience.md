# Guest Walk-up Experience for Customer Service release notes

Version history for the Customer Service Management Guest Walk-up Experience for Customer Service on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - June 2026**

    Defect fixes

-   **Version 2.0.1 - March 2026**

    Bug fixes.

-   **Version 1.4.0 - December 2025**

    Bug fixes.

-   **Version 1.3.0 - August 2025**

    Bug fixes.

-   **Version 1.2.0 - May 2025**

    Bug fixes.

-   **Version 1.1.9 - February 2025**

    Fixes.

-   **Version 1.1.4 - August 2023**

    Fixes.

-   **Version 1.1.3 - February 2023**

    Fixed: bug fixes.

-   **Version 1.1.2 - November 2021**

    Fixed: Performance improvements to appointment booking flow.

-   **Version 1.1.1 - August 2021**

    Fixed: bug fixes.

-   **Version 1.1.0 - June 2021**
    -   Fixed:
        -   WCAG 2.1 fixes
        -   Functional bug fixes
-   **Version 1.0.6 - February 2021**

    Fixed: Support for custom portal url in email notifications Guest walk-up appointment booking and reschedule is failing after upgrading to PP6/ PP4 HF4.

-   **Version 1.0.5 - November 2020**

    Fixed: Guest user appointment booking fails after any pending appointment is cancelled.

-   **Version 1.0.4 - October 2020**
    -   Fixed:
        -   Appointment booking for guest user not honoring the location time zone configuration
        -   When a registered user books an appointment using guest flow, 2 appointment confirmation emails are sent to the user
-   **Version 1.0.2 - August 2020**

    This application extends the Walk-up Experience feature for guest users. Walk-up Experience \(for registered users\) for Customer Service Management was released in Paris.
