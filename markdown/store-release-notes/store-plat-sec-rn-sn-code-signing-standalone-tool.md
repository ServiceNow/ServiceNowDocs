# ServiceNow Code Signing Standalone Tool release notes

Version history for the ServiceNow® Code Signing Standalone Tool application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.0 - February 2025**

    The Code Signing Standalone Signing Tool is your go-to solution for signing ServiceNow application records locally with your private key securely, efficiently, and independently of any ServiceNow instance. Designed for seamless cross-platform compatibility, it supports Windows, MacOS, and Linux environments, providing an intuitive and streamlined installation process. Whether you're enhancing your code's integrity or ensuring compliance, this tool delivers a user-friendly experience backed by robust functionality.
