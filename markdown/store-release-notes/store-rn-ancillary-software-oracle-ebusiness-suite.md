# Oracle E-Business Suite Spoke Dependencies release notes

Version history for the ServiceNow Oracle E-Business Suite Spoke Dependencies in ancillary software on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.11.0 - September 2024**

    Modified package to support the improvement of the existing 9 actions.

-   **Version 1.10.0 - June 2024**

    Modified package to support the enhancement or addition of 12 actions.

-   **Version 1.9.0 - May 2024**

    Modified package to support the enhancement/addition of 17 actions.

-   **Version 1.8.0 - April 2024**

    Modified package to support the addition of 21 actions.

-   **Version 1.7.0 - March 2024**

    Modified package to support the change of 9 new actions.

-   **Version 1.6.0 - February 2024**

    Modified package to support the change of Create Receipt action.

-   **Version 1.1.0 - September 2023**

    Fixed: Added more packages to the dependency file.

-   **Version 1.0.0 - May 2021**

    New: This is the required dependency application to enable OEBS Spoke actions to interact with OEBS application accordingly. This dependency application creates an OEBS application package that contains the stored procedures, which expose the associated tables and their columns as web services for OEBS Spoke actions to interact with.
