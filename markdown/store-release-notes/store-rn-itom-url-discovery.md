# ITOM URL Discovery release notes

Version history for the ServiceNow® ITOM URL Discovery application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.1.1 - June 2026**
    -   Fixed:
        -   URLs can now be removed from the Targeted URL Discovery monitoring list.
        -   The Product and Application Service fields are now optional when adding a URL to the Targeted URL Discovery list.
-   **Version 1.1.0 - May 2026**
    -   New:
        -   Remove Targeted URLs:
            -   You can now remove individual URLs from the Targeted URL Discovery monitoring list directly from the Discovery Admin Workspace Settings page, giving you greater control over which URLs are actively monitored as your organization's needs change.
    -   Enhanced:
        -   Bulk upload URLs:
            -   The Targeted URL Discovery list now supports bulk URL uploads using .csv or .xlsx files, simplifying configuration for large-scale monitoring.
-   **Version 1.0.2 - April 2026**
    -   ITOM URL Discovery is a ServiceNow application that gives IT administrators visibility into URLs accessed across the organization through a browser extension. Surfaced inside the Discovery Admin Workspace, it enables organizations to monitor web activity, measure usage metrics, and feed discovered URLs into downstream products like Service Mapping and Synthetic Monitoring.
    -   The app supports two complementary monitoring modes:
        -   Targeted URL Discovery — tracks curated URLs with deep metrics including total usage time, last accessed timestamps, and per-device access details.
        -   Broad URL Discovery — passively captures all user-visited URLs and page hit counts across the organization.
