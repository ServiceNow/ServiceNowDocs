# Legal Stock Preclearance release notes

Version history for the Legal Stock Preclearance on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.10.1 - June 2026**

    Changed: Security enhancements.

-   **Version 4.9.0 - December 2025**

    Changed: Security enhancements.

-   **Version 4.8.3 - August 2025**

    Fixed proposed transaction date validation in the intake form.

-   **Version 4.8.0 - May 2025**
    -   Changed: Improved Legal Stock Preclearance icon for improved space utilization in the intake form.
    -   Fixed: UI issues
-   **Version 4.7.3 - March 2025**

    New: Legal Stock Preclearance application is also available as part of the Legal Simple Compliance suite.

-   **Version 4.7.2 - February 2025**

    Fixed: Security issues.

-   **Version 4.5.0 - November 2024**
    -   Fixed:
        -   In the Stock Pre-clearance intake form, the date in the information message was not displayed as per the logged-in user's date format preference. This is now fixed.
        -   Date displayed in the activity stream for a Stock Pre-clearance request was not displayed as per the logged-in user's date format preference. This is now fixed.
-   **Version 4.3.1 - July 2024**

    Fixed: When an employee is submitting a stock pre-clearance request using Employee Center, an error prevented the request from being submitted. This has been fixed.

-   **Version 4.2.7 - February 2023**

    Changed: Added Legal Practice Apps core dependency.

-   **Version 4.2.6 - November 2022**

    Fixed: Fixed an issue where the stock preclearance requests were not being approved when there were no approvers in the assignment group.

-   **Version 4.2.2 - May 2022**

    Fixed: Minor fixes

-   **Version 4.1.1. - February 2022**

    Changed: Legal: LPA: Stock Preclearance has been renamed Legal Stock Preclearance

-   **Version 4.0.2 - December 2021**

    New: Added new email notifications for various stock preclearance request actions

-   **Version 3.3.3 - October 2021**

    Changed: Updated UI for adding access persons.

-   **Version 3.3.2 - September 2021**

    New: Request legal services related to stock preclearance from Employee Center portal

-   **Version 3.2.0 - June 2021**
    -   New: The Stock Preclearance legal practice application enables the legal staff to:
        -   Manage employee requests to sell stocks within the open trading window.
        -   Configure a company’s access persons list, trading window, and trading holidays.
