# Idea Manager dashboard release notes

Version history for the IT Business Management Idea Manager dashboard on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.1 - May 2026**

    Fixed: Fixed filter for "Average age open ideas" to follow "PPM - Idea State" and "PPM - Idea Category"

-   **Version 2.2.0 - December 2025**
    -   New: Provided access to idea manager module to "idea\_manager" and "idea\_manager\_admin" roles.
    -   Removed: Removed access to the idea manager module to "admin" role.
-   **Version 2.1.0 - August 2024**

    Fixed: Few widgets shows 'report\_view field ACL' error for the users with 'idea\_manager' role.

-   **Version 2.0.0 - May 2024**
    -   New: Idea manager dashboard moved to next experience platform analytics.
    -   Removed: Old user experience has been removed for new customers.
-   **Version 1.0.0 - June 2020**

    This plugin packages dashboard with summary views of idea metrics and trends such as ideas in different states, age of ideas, categories, and ideas converted to different work entities.
