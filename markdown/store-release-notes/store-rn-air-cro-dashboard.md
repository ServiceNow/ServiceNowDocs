# CRO Dashboard release notes

Version history for the ServiceNow® CRO Dashboard application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.0.10 - December 2025**

    The Chief Risk Officer Dashboard consolidates data from GRC, including Profiles, Policy and Compliance Management, Audit Management, and Risk Management, to provide a unified view of enterprise risk, compliance posture, and audit performance.
