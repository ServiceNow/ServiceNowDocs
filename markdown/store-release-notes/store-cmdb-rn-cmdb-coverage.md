# CMDB Coverage release notes

Version history for the ServiceNow® CMDB Coverage application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.1 - May 2026**
    -   Fixed:
        -   Fixed the Discovery Gap scheduled job to update the gap number correctly in the widget.
        -   Fixed the IP-to-CI widget to show the list view of IPs with a single click.
-   **Version 1.0.0 - March 2026**
    -   CMDB Coverage helps organizations identify missing data and highlight gaps to ensure compliance with enterprise standards. By improving the coverage of the CMDB, teams can gain better visibility, reliable asset data, and stronger IT operational control.
    -   Key benefits:
        -   Gain a clear understanding of the infrastructure environment.
        -   Identify discovery gaps and create targeted discovery schedules.
        -   Use the coverage dashboard to highlight areas for improvement and ensure CMDB completeness.
