# @servicenow/now-vis-bar release notes

Version history for the @servicenow/now-vis-bar application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 23.0.8 - March 2023**

    Bug fixes

-   **Version 23.0.5 - February 2023**
    -   Fixed: Bug fix. Bar visualization is a part of the Data Visualization component. It presents ServiceNow or custom data in vertical or horizontal bars.
    -   The Bar data visualization can be used in the Next Experience and is available from Next Experience Dashboards, the UI Builder, and the Visualization Designer.
    -   This application is installed automatically with "Data Visualization Core" application.
