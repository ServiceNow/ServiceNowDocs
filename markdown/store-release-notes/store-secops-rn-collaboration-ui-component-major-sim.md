# Collaboration UI Component for Major Security Incident Management Workspace release notes

Version history for the Collaboration UI Component for Major Security Incident Management Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - November 2024**

    Fixed: Accessibility issues in the MSIM Workspace in tabs such as, Status Report, Collaboration, etc.

-   **Version 1.2.0 - March 2022**

    Fixed: Fixes related to the Next Experience theme.

-   **Version 1.1.0 - December 2021**

    Collaboration Component for Security Operations is a depedency plugin that provides the Collaboration Component for Major Security Incident Management Workspace.
