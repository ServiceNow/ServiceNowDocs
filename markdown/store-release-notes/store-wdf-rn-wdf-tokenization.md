# WDF Tokenization release notes

Version history for the ServiceNow® WDF Tokenization application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 3.1.0 - June 2026**

    Introduced backend enhancements to support the unified credits dashboard.

-   **Version 2.0.3 - March 2026**

    Includes some performance updates.

-   **Version 2.0.2 - December 2025**

    Consolidate all Workflow Data Fabric consumption into a single Fabric Credit balance, gaining complete visibility into how credits are spent daily. Each credit represents a unit of consumption under the unified meter; for example, a managed record or asset, a discrete transaction \(API call\), a megabyte of data throughput, or a minute of runtime, depending on the underlying product. Drill down by product to see which workloads drive usage and monitor daily trends over a rolling 365-day window.
