# SAP S4 HANA IDoc spoke release notes

Version history for the SAP S4 HANA IDoc spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - April 2026**

    Enhanced MID Server JAR file and SapConnector Class to avoid conflicting Name, Version, and Filename

-   **Version 1.2.0 - October 2024**

    Security updates on MID script includes.

-   **Version 1.1.2 - December 2023**

    Fixed: Mismatching in schedule segment's spelling in Update Purchase Order \(HANA IDOC\) action.

-   **Version 1.1.1 - April 2023**

    Improved installation performance.

-   **Version 1.1.0 - February 2022**
    -   New: Added 2 actions
        -   Create Customer Account
        -   Update Customer Account
-   **Version 1.0.2 - September 2021**

    Changed: Updated SapConnector jar

-   **Version 1.0.1 - January 2021**
    -   The SAP S/4HANA IDOC spoke provides a list of actions that interact with SAP standard IDOC. It provides the foundation to create and update from ServiceNow to SAP S/4HANA.
    -   Given that SAP S/4HANA is a highly customizable system, SAP S/4HANA IDOC spoke also showcases the design pattern that can support standard IDOCs and custom IDOCs beyond what is out of the box.
