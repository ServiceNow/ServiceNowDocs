# Accounts Payable Operations Integration with Coupa release notes

Version history for the Accounts Payable Operations Integration with Coupa application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - June 2026**
    -   New:
        -   Added Create or cancel invoice flow in Coupa.
            -   Note: This flow is inactive until changes for theRemit to field are made available in the workspace UI.
    -   Fixed: Integration service job tracker records are not marked as failed when flows encounter an internal server error.
-   **Version 1.4.0 - December 2025**

    Changed: Added granular roles to system properties in accordance with new directives.

-   **Version 1.3.0 - August 2025**

    This application enables bi-directional integration between Accounts Payable Operations to Coupa and allows you to Fetch, create, update invoices.
