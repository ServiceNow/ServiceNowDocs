# GRC: Workbench release notes

Version history for the GRC: Workbench on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 21.1.0 - December 2025 \(Zurich\)**

    Removed a redundant tooltip to improve clarity and user experience.

-   **Version 20.1.0 - May 2025**

    Fixed: Resolved a security issue to improve overall application security.

-   **Version 19.0.1 - August 2024**

    Fixed: UI flickering issues are resolved.

-   **Version 18.1.0 - June 2024**

    New: The application is no longer dependent on other apps. It can be installed as a standalone application.

-   **Version 16.0.0 - May 2023**

    Fixed: We migrated the parent POM of the scoped app packaging to prevent duplication of content in the built artifact.

-   **Version 14.2.1 - May 2022**

    Changed: Updated JQuery verison

-   **Version 11.0.0 - October 2020**

    This application is compatible with Orlando and Paris. See Supporting Links &amp; Docs for release notes and product documentation.

-   **Version 9.0.0 - November 2019**

    This feature was updated for New York.
