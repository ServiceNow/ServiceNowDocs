# Google Meet Spoke release notes

Version history for the Google Meet Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - November 2025**

    New - AI Agents.

-   **Version 1.1.0 - September 2025**

    Fixed - Required ACL\(s\) or Role\(s\) for agents.

-   **Version 1.0.0 - February 2025**

    Google Meet is a secure video conferencing service that allows users to join virtual meetings with audio, video, chat and screen sharing. Individuals within a company collaborate with each other using meetings, record these meetings, and even download the meeting transcripts. Teams manage group meeting using Spaces. Anyone can create a meeting and invite participants. ServiceNow, as a system of engagement, can facilitate these actions using workflows easily. All Space, Conference, Participant, and Artifacts like recordings and transcripts actions can be facilitated using actions within Workflow Studio.
