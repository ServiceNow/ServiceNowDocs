# Advanced Approval Management release notes

Version history for the ServiceNow® Advanced Approval Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.2.3 - June 2026 \(Australia\)**
    -   Approvers can act on approval requests directly from consolidated email notifications by approving or rejecting individual requests or all requests.
    -   Approval Reminders — Email reminders sent to approvers after a configurable pending period.
    -   Approval Escalations — Pending approvals auto-escalate to a designated approver after a configurable period, with email notification.
    -   Override Approval — New capability and user role to bypass the standard approval flow.
    -   Ad-Hoc Approver Assignment — Approvers and requesters can add an ad-hoc approver mid-approval cycle.
-   **Version 2.2.2 - June 2026 \(Zurich\)**
    -   New:
        -   Consolidation emails that support approve or reject actions per individual request or bulk approve or reject actions for all requests.
        -   Server script include API to fetch approval status.
-   **Version 2.0.0 - April 2026**
    1.  Enables approvers to add ad-hoc approvers to the workflow.
    2.  Consolidate email approval requests for approvers.
    3.  Provide the ability to override an approval step.
-   **Version 1.0.0 - March 2026**

    This application delivers a comprehensive Quote Approval framework to mitigate financial, legal, and compliance risks before quotes reach customers. It enables Sales Reps to identify when and why quotes need approval and ensure the quotes provided to customer adhere to all company's policies and guidelines.
