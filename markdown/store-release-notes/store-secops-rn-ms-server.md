# Microsoft Exchange Server for Security Operations release notes

Version history for the Security Operations Microsoft Exchange Server integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

This integration is for on-premises Microsoft Exchange servers. For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.3.1 - February 2022**

    Fixed: Fixed a performance issue on corrupt mailboxes.

-   **Version 8.0.8 - July 2019**

    New: Recertified for New York.

-   **Version 7.0.2 - March 2019**
    -   Perform Email Search and Deletion workflow
    -   Get Email Details from Exchange Server workflow activity
    -   Search/Delete Threat Email in Exchange workflow activity
