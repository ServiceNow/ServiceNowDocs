# Document Service Framework for Google Drive release notes

Version history for the Document Service Framework for Google Drive application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.2.0 - May 2026**

    Changed: Extended personal authentication for list API.

-   **Version 3.1.0 - December 2025**

    Fixed: Added compatibility support for Glide changes

-   **Version 3.0.0 - August 2025**
    -   Personal authentication support
    -   New subflow for updating document permissions
    -   New subflow for deleting document permissions
-   **Version 2.0.0 - May 2025**

    Changed: All existing flows for Google Drive have been extended to support Shared drives.

-   **Version 1.1.1 - October 2024**
    -   This release doesn't have any code changes
    -   Updating dependency version
    -   Enabling subscription
-   **Version 1.1.0 - August 2024**
    -   Changes for Google Doc Integration to MultiProvider
    -   Minor changes for integrating MultiProvider with DMS
-   **Version 1.0.2 - March 2024**

    App name change.

-   **Version 1.0.1 - October 2023**

    Fixes.

-   **Version 1.0.0 - August 2023**

    This spoke provides the capability to support Multi Provider Document Services \(released as part of the Quebec release\) integration with Google Drive. It creates all the necessary tables and scripts, used by sub-flows in this spoke for seamless integration.
