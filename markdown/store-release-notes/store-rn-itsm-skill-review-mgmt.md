# Skill Review Management release notes

Version history for the Skill Review Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.1 - December 2025**

    Minor bug fixes

-   **Version 1.5.0 - November 2024**

    No Major feature changes. Contains minor changes and bug fixes.

-   **Version 1.4.0 - November 2023**

    New: Uptook SRP changes for WFO SOW Assign category to sys\_notification files in Skill Review.

-   **Version 1.3.0 - August 2023**

    New: Localization build changes WFO applicable groups in skill review requests.

-   **Version 1.2.0 - February 2023**

    A few fixes.

-   **Version 1.1.3 - August 2022**
    -   This can only be used if you have Workforce Optimization. Use this application to manage skills across your team and empower your team members to manage their skill profile.
    -   Includes the ability for a team member to self-assess their skill set and propose changes to their manager. Manager can then approve or reject skill proposals.
