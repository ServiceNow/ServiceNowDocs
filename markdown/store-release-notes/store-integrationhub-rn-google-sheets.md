# Google Sheets spoke release notes

Version history for the Integration Hub Google Sheets spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.7 - September 2023**

    Fixed: The license requirements.

-   **Version 1.0.6 - April 2023**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.5 - September 2022**

    Patch version of Google Sheets Spoke for Integration Hub. This version includes a fix for encoding issue within Add Row action.

-   **Version 1.0.4 - August 2021**

    Patch release of the Google Sheets spoke for IntegrationHub. This version includes a fix of encoding "/" within Get Cell Values action.

-   **Version 1.0.3 - June 2021**

    Patch release of the Google Sheets spoke for IntegrationHub. This version includes a fix for dynamic choice within actions.

-   **Version 1.0.2 - October 2020**

    Patch release of the Google Sheets spoke for IntegrationHub.

-   **Version 1.0.1 - July 2020**

    Provides actions to read and update cells, and manage sheets in Google Sheets.
