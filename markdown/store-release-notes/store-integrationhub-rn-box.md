# Box Spoke release notes

Version history for the Integration Hub Box spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.7.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 3.6.0 - April 2026**

    Added 10 new actions, primarily around Box Hub, Box Hub Items, and Box AI.

-   **Version 3.5.1 - January 2026**

    Fixed: ACL enhancement on multiple tables.

-   **Version 3.5.0 - December 2025**

    Added 10 external triggers

-   **Version 3.4.0 - September 2025**

    Added required ACLs/Roles as per the AI Security Directive guidelines.

-   **Version 3.3.1 - July 2025**

    Fixed: look up the right file ID when existing same name files in different path at Box.

-   **Version 3.3.0 - May 2025**

    Minor release with new AI agents.

-   **Version 3.2.2 - March 2024**

    Fix in code for Action Look up File ID and Look up Folder ID actions are not loaded with latest code.

-   **Version 3.2.1 - April 2023**

    Patch release of Box Spoke for Integration Hub. This version includes deduplication changes.

-   **Version 3.2.0 - January 2023**

    Fixed: Added Content-Type property in Headers for the missing actions.

-   **Version 3.1.3 - September 2022**

    Fixed: Patch version of Box. This version includes a fix to improve the installation performance of the spoke.

-   **Version 3.1.1 - May 2022**

    Patch release for Box Spoke. This version includes fixes for errors 'Provided username doesn't exist at Action 'Look up User ID' and 'Folder path doesn't exist at Action 'Look up Folder ID'.

-   **Version 3.1.0 - October 2021**

    Fixed: Security bug

-   **Version 3.0.1 - June 2021**

    Patch release of Box Spoke for IntegrationHub. This version adds KMF modules for additional security of the Password2 fields, and a fix for connection URL in the 'Upload File' action.

-   **Version 3.0.0 - April 2021**
    -   Fixed:
        -   JSON response body is now a drag-n-droppable data pills so that end user can build the flow in Flow Designer 10x easier.
        -   Updated with legal hold policy and retention policy
        -   Updated with webhook for inbound integration from Box to ServiecNow
        -   Sample remote tables to view legal hold policies and retention policies
-   **Version 2.0.4 - February 2021**

    Fixed: Fixes for 6 issues are provided as part of this patch release.

-   **Version 2.0.3 - February 2020**

    Fixed: Clarified a confusing error message

-   **Version 2.0.2 - August 2019**

    Licensing patch for the Box spoke on Integration Hub

-   **Version 1.3.8 - March 2019**

    The Integration Hub Box spoke provides native integration to Box to easily manage files and folders from ServiceNow.

    -   Automate creation and management of Box files and folders for collaboration
    -   Use Box as a collaboration tool to facilitate document management
    -   Control permissions by using ServiceNow as a system of engagement to add or remove collaborators for faster outcomes
    -   Manage folders to organize case documents in Box
    -   Manage groups and users in Box
    -   Share document or folder links with collaborators
