# SumTotal spoke release notes

Version history for the Integration Hub SumTotal spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - March 2023**

    New: Authentication template for easier spoke setup.

-   **Version 1.0.2 - December 2022**

    Patch release of SumTotal Spoke for IntegrationHub. This version includes fixes for localization and to improve install performance of the application.

-   **Version 1.0.1 - September 2022**

    Patch version of SumTotal Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.0 - September 2021**

    Provides actions to integrate the SumTotal courses and user activity with ServiceNow.
