# Agent Client Collector Spoke release notes

Version history for the Agent Client Collector Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 1.1.5 - January 2024**

    Fixed for ACCUtils "Error: ACC Agent\_xxx is down, Restart the agent.", if there are multiple records for an Agent.

-   **Version 1.1.4 - September 2023**

    Fixed: The license requirements.

-   **Version 1.1.3 - July 2023**

    Fixed: Issue in definition for GlideRecord and CIDR notion in Linux.

-   **Version 1.1.2 - September 2022**

    Fixed: Patch version of Agent Client Collector Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.1.0 - September 2021**
    -   New:
        -   Two new actions \(Install Agent on Linux and Install Agent on Windows\)
        -   Two new sub-flows \(Install Agents on Linux and Install Agents on Windows\)
-   **Version 1.0.1 - March 2021**

    Patch release of the Agent Client Collector spoke for IntegrationHub.

-   **Version 1.0.0 - February 2021**

    New: ACC spoke provides a framework to write customized workflows or automations to execute on Agents\(ACC\) or Configuration Items \(CI\).
