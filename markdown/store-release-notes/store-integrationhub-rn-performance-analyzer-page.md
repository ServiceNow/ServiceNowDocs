# Performance Analyzer Page release notes

Version history for the Performance Analyzer Page application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.1.0 - March 2026**
    -   Page Load Performance Insights:
        -   View page load times by page variant to better understand performance differences across experiments, configurations, or layouts.
        -   Analyze page load times for a specific user, helping you pinpoint individual performance issues and troubleshoot real‑world scenarios more effectively.
    -   Deeper Visibility into Batch Calls:
        -   Inspect calls executed within batch requests, including a detailed breakdown of each call.
        -   See execution times per call, making it easier to identify bottlenecks and optimize performance.
    -   Easy Sharing &amp; Collaboration: Bookmark or share URLs to quickly return to specific views or share insights with teammates and stakeholders.
    -   UI Improvements: Various minor UI refinements to improve usability, clarity, and overall experience.
-   **Version 2.1.0 - October 2025**

    Important bug fixes to improve experience.

-   **Version 2.0.1 - October 2025**

    Important bug fixes to improve experience.

-   **Version 2.0.0 - August 2025**

    Now including support X and Y Family releases

-   **Version 0.0.8 - May 2025**

    New: We're excited to introduce the Performance Analyzer, a powerful new tool that brings end-to-end visibility into how your Next Experience pages are performing.
