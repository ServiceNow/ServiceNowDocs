# Agent Messaging Component release notes

Version history for the Customer Service Management Agent Messaging Component on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.4 - January 2021**
    -   New:
        -   Certified for Quebec
        -   Added support for agent-initiated messaging for multiple messaging channels
-   **Version 1.0.0 - July 2020**

    New: This application contains workspace UI components for composing agent initiated SMS.
