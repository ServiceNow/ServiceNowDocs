# Conversational Integration with Facebook Messenger release notes

Version history for the Conversational Integration with Facebook Messenger on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.8 - December 2025 \(Zurich\)**

    Fixed bugs for messenger

-   **Version 3.0.7 - October 2025**

    Error while sending a message in Facebook messenger as the meta API has changed.

-   **Version 3.0.6 - February 2025 \(Yokohama\)**

    Added translations for the store app.

-   **Version 3.0.4 - August 2024 \(Xanadu\)**

    Fix: Messenger users are not able to upload files via link.

-   **Version 3.0.2 - August 2023 \(Vancouver\)**

    New: Facebook Messenger will be part of Admin Console with an automated version of installation as feature.

-   **Version 3.0.1 - February 2023**

    Fixed: Minor fixes.

-   **Version 2.0.4 - May 2022**

    Fixed: Bug fixes and improvements.

-   **Version 2.0.2 - December 2021**

    Fixed: Implemented several fixes.

-   **Version 1.0.3 - April 2021**
    -   New:
        -   Enables users to engage in conversations with your company’s virtual agents and live agents using the Facebook Messenger app.
        -   If a bot transfers the conversation to a live agent, the agent can respond to the Facebook Messenger messages in the ServiceNow Agent Workspace or Configurable Workspace application.
        -   The following user input controls are supported: Text, Static Choice, Reference Choice, Boolean, Carousel, and File Picker.
        -   The following bot responses are supported: Text, Image Output, Link, and Card
        -   Rich text and emojis can be used in the Virtual Agent setup to update conversations in the Facebook Messenger app.
        -   Live agents can use the Text, Card, and Image Output response controls to reply to users.
