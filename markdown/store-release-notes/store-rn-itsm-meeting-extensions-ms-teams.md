# Meeting Extensions for Microsoft Teams release notes

Version history for the Meeting Extensions for Microsoft Teams on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.8.0 - December 2025**
    -   New:
        -   Introduced granular Admin Roles to have more control on the Meeting Extensions.
        -   Change of Microsoft SDK.
-   **Version 1.7.0 - May 2025**

    New: The "deny-unless" authentication ACL allows only authenticated users to access the tables related to collaboration services. This ACL significantly improves the security of the data and prevents unauthorized access.

-   **Version 1.6.5 - February 2025**

    Fixed: Avoided impersonation when application ID for extension app isn't populated.

-   **Version 1.6.3 - October 2024**

    Fixed: Marked the "private" property as false.

-   **Version 1.6.2 - August 2024**

    Fixed: Changes in internal authentication mechanisms.

-   **Version 1.6.1 - May 2024**

    Changed: User authentication flows.

-   **Version 1.3.6 - December 2023**
    -   Changes to authentication mechanism.
    -   Note: Some users may see errors when loading Meeting Extensions for Microsoft Teams. For more information, refer to https://support.servicenow.com/kb?id=kb\_article\_view&amp;sysparm\_article=KB1577588
-   **Version 1.3.5 - August 2023**

    Fixed: Incorrect impersonation while executing a block.

-   **Version 1.3.4 - May 2023**

    Changed: Build process improvements.

-   **Version 1.3.3 - February 2023**

    Fix for issues accessing meeting extensions from conference calls initiated from Major Incident Workbench.

-   **Version 1.3.2 - November 2022**

    Fixes.

-   **Version 1.3.1 - August 2022**

    Changed: Updating dependency on Notify Connector for Teams

-   **Version 1.2.10 - May 2022**

    Minor fixes

-   **Version 1.2.8 - February 2022**

    Changed: Localization of the app is now updated to be released with the Meeting Extensibility for Microsoft Teams app.

-   **Version 1.2.4 - November 2021**

    Fixes for stabilisation

-   **Version 1.2.1 - September 2021**

    Fixed: Microsoft Teams Meeting Extensibility, which now relies on newer API for creating Online Meeting.

-   **Version 1.1.1 - June 2021**

    New: Added support for Multi-Tenant Notify BOT provided by ServiceNow. The users can use the Azure apps shipped by ServiceNow instead of creating their own apps for integration.

-   **Version 1.0.5 - February 2021**

    New: Meeting Extensions for Microsoft Teams enables ServiceNow to be embedded directly within a Microsoft Teams meeting kicked off from a Major Incident.
