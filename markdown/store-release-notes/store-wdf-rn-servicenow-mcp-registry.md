# ServiceNow MCP Registry release notes

Version history for the ServiceNow® MCP Registry application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.9 - June 2026**
    -   New: New MCP Servers
    -   Fixed: MCP One-Click Setup
-   **Version 1.0.7 - May 2026**
    -   Every enterprise adopting AI agents faces the same challenge: How do you connect agents to the right tools and data sources without losing control?TheServiceNow MCP Registry solves this by giving you a single, governed catalog of every MCP server your organization uses — whether built internally, sourced externally, or pulled from the broader MCP ecosystem.
    -   Teams can discover and connect to approved MCP servers instantly, while maintaining full visibility and control over what's available, who can access it, and which versions are in use.
    -   Because it's built on the open MCP Registry specification, you get full compatibility with standard MCP tooling and SDKs. You get the security, access control, and operational governance layered on top that enterprise environments demand.
