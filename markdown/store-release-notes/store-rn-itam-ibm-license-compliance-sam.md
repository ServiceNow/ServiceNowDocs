# IBM License Compliance for Software Asset Management release notes

Version history for the IBM License Compliance for Software Asset Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.0.7 - March 2026 \(Australia\)**

    Minor defect fixes.

-   **Version 5.0.7 - December 2025 \(Yokohama\)**

    In this release, minor defects related to IBM IASP have been fixed.

-   **Version 4.0.7 - September 2025 \(Xanadu\)**

    Performance fixes related to the 'SAM - Sync IBM Install to Classification' job

-   **Version 6.0.3 - August 2025 \(Zurich\)**

    Fixed minor bugs related to IBM reconciliation.

-   **Version 4.0.6 - July 2025**

    Fixed a reconciliation issue related to IBM License Metric Tool \(ILMT\) product usage.

-   **Version 5.0.5 - June 2025**

    Fixed a minor bug related to the IBM integration.

-   **Version 5.0.4 - February 2025**

    Added support for IBM Cloud Paks running in VMware, IBM LPAR virtual environments, and public cloud providers \(such as AWS, Azure and GCP excluding containers and Kubernetes\)

-   **Version 4.0.3 - November 2024**

    Minor bug fixes related to IBM ILMT.

-   **Version 4.0.0 - August 2024**

    Added support for IBM software deployed on public cloud environments like AWS, Azure and GCP.

-   **Version 3.1.1 - March 2024**

    In this release, we've made some minor bug fixes.

-   **Version 3.1.0 - November 2023**

    In this release, we've made some minor bug fixes.

-   **Version 3.0.0 - August 2023**

    New: Support for ILMT v2 APIs.

-   **Version 2.0.0 - May 2023**

    Added support for IBM software deployed in IBM LPAR virtual environments.

-   **Version 1.0.7 - March 2023**

    In this release, we've made some minor bug fixes.

-   **Version 1.0.0 - February 2023**
    -   Reconcile Processor Value Unit \(PVU\), Resource Value Unit \(RVU\), and Virtual Processor Core \(VPC\) licenses for IBM products using the Servicenow Discovery and Software Asset Management applications.
    -   Integrate the Software Asset Management publisher pack for IBM with an IBM Authorized SAM Provider \(IASP\) platform to track and manage your IBM licenses. With this integration, you can track and manage IBM licensing directly without having to integrate with the IBM License Metric Tool \(ILMT\) or BigFix Inventory.
