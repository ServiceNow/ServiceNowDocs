# Software Asset Management Playbooks and Guided Setups release notes

Version history for the IT Asset Management Software Asset Management Playbooks and Guided Setups application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 7.0.1 - March 2026 \(Australia\)**

    In this version, software lifecycle reporting and compliance management are streamlined through a guided playbook.

-   **Version 3.0.9 - May 2024**

    In this release, the product type column name is updated in the Software model form for Playbook.

-   **Version 2.0.7 - August 2023**
    -   Added Guided Setup to help you configure Microsoft SQL Server on your ServiceNow instance and created the below categories:
        -   Prerequisites
        -   Software Asset Management Configurations
        -   Verify Software Asset Management; Configurations
-   **Version 1.3.0 - March 2023**

    Issue fix

-   **Version 1.2.0 - March 2023**

    Issue fix

-   **Version 1.1.9 - February 2023**

    Removed: In this release, the sn\_pad\_demo plugin dependency is removed.

-   **Version 1.1.8 - January 2023 \(compatible with Tokyo\)**

    Removed the sn\_pad\_demo plugin dependency

-   **Version 1.0.2 - January 2023 \(compatible with Rome\)**

    Removed the sn\_pad\_demo plugin dependency

-   **Version 1.1.5 - August 2022**

    Fixes

-   **Version 1.1.2 - February 2022**

    Fixes

-   **Version 1.0.1 - July 2021**

    New: New Guided walk-through for creation of an entitlement. Guided walk-through in new workspace view. All on-premise as well as SaaS product type included. For each field details playbook card user interface Aggregate details of all steps in last.
