# Chat integration with Security Incident Management release notes

Version history for the Chat integration with Security Incident Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.10 - August 2025**

    Fixed: Implemented robust handling for HTTP 503 \(Service Unavailable\) errors in Microsoft Teams actions.

-   **Version 1.2.5 - May 2025**

    Fixed: Security Bugs related to Slack Chat Connector for Security Incident Management.

-   **Version 1.2.4 - May 2024**
    -   Fixed:
        -   Empty ACLs addressed
        -   Report view ACLs for all existing tables
        -   Posting a message in the Microsoft Teams channel when the task is assigned
-   **Version 1.2.3 - March 2024**

    Changed: Added a navigation link to Microsoft Teams on the chat channel modal of the workspace.

-   **Version 1.2.0 - March 2022**

    Fixed: Updated the Default Template record table.

-   **Version 1.2.0 - March 2022**

    Fixed: Security bug fixes.

-   **Version 1.1.0 - December 2021**

    Allows you to track chat activity for third-party applications such as Microsoft Teams or Slack.
