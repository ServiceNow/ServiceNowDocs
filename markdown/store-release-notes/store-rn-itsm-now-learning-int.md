# ServiceNow University Integration release notes

Version history for the ServiceNow University Integration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - February 2024**
    -   This integration enables the ServiceNow University content from the ServiceNow instance.
    -   -   This spoke imports Course Items and Learning paths from the ServiceNow University instance to the ServiceNow instance.
-   This spoke imports and updates user course progress and completion history in the ServiceNow instance.
    -   When used with workforce optimization:
    -   -   As a manager, you can assign course items to employees from the Manager Workspace.
-   As an IT employee, you can enroll in the course items from the Service Operations Workspace.
