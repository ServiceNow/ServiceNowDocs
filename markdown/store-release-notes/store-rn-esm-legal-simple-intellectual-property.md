# Legal Simple Intellectual Property release notes

Version history for the Legal Simple Intellectual Property application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.7.0 - June 2026**

    Changed: Security enhancements

-   **Version 1.6.1 - May 2026**

    Fixed: Fixed an issue where installing the "sn\_lg\_ops" plugin incorrectly changed the workspace view scope from Global to Legal Request Management.

-   **Version 1.6.0 - December 2025**

    Changed: Security enhancements.

-   **Version 1.5.4 - August 2025**

    Fixed issues related to reward tasks in invention disclosure workflow.

-   **Version 1.5.3 - May 2025**
    -   Changed: Improved Legal Simple Intellectual Property icon for better space utilization.
    -   Fixed:
        -   For a legal request, the "Contract request" tab is not displayed. This issue has been fixed.
        -   Generate PDF option is not working in Invention disclosure requests. This issue has been fixed.
-   **Version 1.4.2 - February 2025**

    Fixed: Security bugs.

-   **Version 1.3.0 - November 2024**
    -   Fixed:
        -   Record producers created for Legal Simple Intellectual Property application were not displayed to users having Catalog Editor role. This is now fixed.
        -   Security fixes
-   **Version 1.2.0 - August 2024**

    Security fixes.

-   **Version 1.1.1 - May 2024**
    -   Fixed:
        -   Invention Declaration Form legal requests are not getting closed even after the Legal Matter is closed.
        -   The owner of invention disclosure matter is not able to add the legal name of inventors.
        -   The following PDF generation related issues are fixed:
            -   Confirmation message wasn't displayed when the pdf was successfully generated.
            -   Generated PDF wasn't formatted properly.
            -   Inventors name was appearing incorrectly.
            -   When generating the PDFs multiple times, the file names for the generated PDF weren't appended with versions.
        -   For invention disclosure request, the reward amount of rejected reward task wasn't displayed when moving to patent filed stage.
        -   In the invention disclosure related matter, the Application Status field in the header section and Proceed to Application action button weren't displayed.
-   **Version 1.0.7 - January 2024**

    Security fixes.

-   **Version 1.0.5 - November 2023**
    -   Legal Simple Intellectual Property is a legal practice app included with Legal Service Delivery that provides organizations the ability to submit and track IP requests throughout the entire lifecycle. The workflow process helps boost the patent team's productivity by using purpose-built phases and tasks, including invention disclosure.
    -   The intake process can be configured to meet organizational needs while ensuring full transparency is maintained. The patent team can quickly engage outside counsel and configure rewards for the different stages of the patent application through completion.
