# Workplace Lease Administration release notes

Version history for the Workplace Lease Administration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.8.0 - June 2026**
    -   Changed: Unified record page experience. Lease custom record pages have been migrated to the standard record page layout for a consistent experience across Workplace products.
    -   Fixed: Record creation from within Workplace Central workspace lists now works as expected \(Lease Management Module\).  \{PRB2019165\}
-   **Version 1.7.10 - March 2026**
    -   Fixed:
        -   Improved support for 200% zoom to prevent unnecessary scrolling when creating a new Workplace Contract.
        -   The Welcome banner was not displayed with certain screen dimensions or when using 400% zoom.
-   **Version 1.7.6 - December 2025**

    Security fixes.

-   **Version 1.7.3 - August 2025**

    Fixed: The approval flow is not triggered when creating a lease contract with Approvers on an instance running Zurich.

-   **Version 1.6.1 - May 2025**
    -   Changed: Workplace Lease Administration dashboard list views have been relocated to the common page within Workplace Central.
    -   Fixed: Translations have been fixed.
-   **Version 1.5.0 - February 2025**
    -   New:
        -   3 new tables to capture more data related to lease contract - Workplace Contract Responsibilities, Workplace Contract Clauses, and Workplace Contract Options.
        -   Added more fields to existing Workplace Contract table.
-   **Version 1.4.0 - November 2024**

    Fixed a few Accessibility issues.

-   **Version 1.2.3 - August 2024**

    Minor bug fixes.

-   **Version 1.2.1 - May 2024**

    Fixed: Issues related to localization have been fixed.

-   **Version 1.2.0 - February 2024**

    New: Introduced Appsee Analytics user journey tracking for New Lease Contract creation flow.

-   **Version 1.1.3 - August 2023**

    Fixed: Issue related to security. Added report\_view ACLs on the tables.

-   **Version 1.1.0 - May 2023**

    New: New Lease Administration dashboard using which Workplace Manager can track workplace contract expiry based on contract end date.

-   **Version 1.0.2 - February 2023**

    ServiceNow Workplace Lease Administration application is used to track all workplace lease contracts and assets, services, locations that are part of the contract.
