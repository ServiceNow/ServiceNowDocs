# Sourcing and Procurement Operations Integration with Coupa release notes

Version history for the Sourcing and Procurement Operations Integration with Coupa on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - June 2026**

    New: Added Create or cancel receipt flow to support the creation and cancellation of receipts in Coupa.

-   **Version 1.0.0 - August 2025**
    -   This application provides you with the ability to:
        -   Create a sourcing event from a Sourcing Request
        -   Award Purchase in NOW and update in the target system
        -   Look up and pull the Purchase Order information
        -   Look up Shipment details by PO Line ID
        -   Look up Shipment details after an updated date
        -   Look up the Receipt details
