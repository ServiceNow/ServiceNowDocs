# Security Operations 'Have I been pwned?' Integration release notes

Version history for the ServiceNow® Security Operations 'Have I been pwned?' Integration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.5.2 - June 2026**
    -   Have I been pwned? is a free resource used to assess if users have been put at risk due to their online account being compromised or "pwned" in a data breach.
    -   Have I been pwned? integration for Security Operations enables you to submit lookups on domain names and email addresses to determine if a user's personal data has been compromised by a data breach.
    -   Have I been pwned? integration is now available only on the ServiceNow® Store.
