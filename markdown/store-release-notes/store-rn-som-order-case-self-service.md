# Order Case Self Service release notes

Version history for the Order Case Self Service application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.4 - June 2026 \(Australia\)**

    Changed: Functionality to support Manage order operations AI agent.

-   **Version 1.4.3 - June 2026 \(Zurich\)**

    Changed: Functionality to support Manage order operations AI agent.

-   **Version 1.4.2 - March 2026 \(Zurich\)**
    -   Changed: The Order Case line details are displayed on a dedicated details page instead of a pop-up.
    -   Fixed:
        -   Account and contact fields are now placed next to each other on Order details page
        -   The Scope of request is the last field on order details page
    -   Removed: Removed the ability to search an order case using the order number on the Order details page to boost performance
-   **Version 1.4.1 - March 2026 \(Yokohama\)**
    -   Changed: The Order Case line details are displayed on a dedicated details page instead of a pop-up.
    -   Fixed:
        -   Account and contact fields are now placed next to each other on Order details page
        -   The Scope of request is the last field on order details page
    -   Removed: Removed the ability to search an order case using the order number on the Order details page to boost performance
-   **Version 1.3.2 - January 2026 \(Zurich\)**

    Changed: Restricted permissions so that Admin, Customer Admin, and Partner Admin personas can no longer submit order case.

-   **Version 1.3.1 - January 2026 \(Yokohama\)**

    Changed: Restricted permissions so that Admin, Customer Admin, and Partner Admin personas can no longer submit order cases from the Business Portal.

-   **Version 1.3.0 - December 2025 \(Zurich\)**

    Changed: Restricted permissions so that Admin, Customer Admin, and Partner Admin personas can no longer submit order cases from the self-service portal.

-   **Version 1.2.0 - July 2025**

    View order cases using the self-service portal.
