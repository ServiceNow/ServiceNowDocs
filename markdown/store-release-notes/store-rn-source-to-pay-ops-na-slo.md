# Now Assist for Supplier Lifecycle Operations \(SLO\) release notes

Version history for the Now Assist for Supplier Lifecycle Operations \(SLO\) application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 8.0.0 - June 2026**
    -   New:
        -   Ability to link emails to the created cases and the supplier records
        -   Ability to create assessment templates and error handling while cancelling created assessments
-   **Version 7.2.2 - May 2026**

    Minor defect fixes to ensure stable operation across supported environments.

-   **Version 7.1.0 - April 2026**
    -   AI-native initiatives:
        -   Pricing and packaging updates including revised tier structures
        -   Minor defect fixes to enhance system stability and user experience
-   **Version 7.0.0 - March 2026**
    -   New:
        -   Banking details validation from supplier portal
        -   Email triage where the incoming emails are automatically understood, classified, and converted into structured cases
        -   Sentiment analysis capability has been added to supplier cases to surface the emotional tone of the inquiry based on available case communication
        -   Email response generation capability enables supplier managers and agents to quickly compose context‑aware emails from supplier cases using GenAI.
        -   Text‑to‑Analytics capability allows supplier managers to create reports, insights, and visual analytics on demand by using natural language prompts. This removes the need for technical reporting skills and speeds up data-driven decision-making.
        -   New 3p model offering enablement:
            -   Small model track
                -   OpenAI GPT-5 mini
                -   Claude Haiku 4.5
                -   Gemini 3.0 Flash
            -   Large model track:
                -   OpenAI GPT-5.2
                -   Claude Sonnet 4.5
                -   Gemini 3.0 Pro
-   **Version 6.0.1 - December 2025**

    New: In the registration step of supplier onboarding, an agent is created to dynamically generate document collection tasks based on the supplier details.

-   **Version 5.5.0 - October 2025**

    Consolidation of tools and agents to improve the performance of the AI agents for the Coordinate supplier onboarding flow.

-   **Version 4.5.0 - October 2025**

    Fixed:

    -   Consolidation of tools and agents to improve the performance of the AI agents for the Coordinate supplier onboarding flow.
-   **Version 1.5.0 - October 2025**
    -   Security-related changes implemented for supplier case summarization in the Xanadu release.
    -   Supporting Apriel model for supplier case summarization.
-   **Version 5.0.0 - September 2025**
    -   New:
        -   The AI agents automate the supplier activation and rejection flow.
        -   A new AI agent is introduced to perform supplier data validation.
-   **Version 4.1.0 - September 2025 \(Yokohama\)**
    -   New:
        -   The AI agents will automate the supplier activation and rejection flow.
        -   A new AI agent is introduced to perform supplier data validation.
-   **Version 4.0.0 - August 2025**

    For this release two new agents are included to perform following actions: We will be supporting 3P models \(Gemini, Claude, GPT\) for supplier case summarisation

-   **Version 3.0.6 - June 2025**

    In the June patch we are fine tuning the prompt instructions to improve the outcome and accuracy of the agent response.

-   **Version 3.0.1 - May 2025**
    1.  Supplier onboarding has traditionally been a slow and resource-heavy process for procurement and sourcing teams. In this release, we're introducing an AI agent-powered onboarding solution "Coordinate supplier onboarding" designed to automate and streamline the supplier registration process. These intelligent agents guide supplier managers through data submission, validate documentation, orchestrate tasks across internal teams, and integrate seamlessly with backend systems. The result: faster onboarding, reduced manual effort, and a dramatically improved supplier experience.
    2.  Now Assist Panel for summarization will allow fulfillers to summarize supplier cases and converse with Now Assist without needing to navigate away from their current work. The Now Assist Panel can be activated within in the navigation bar in the Platform experience. System administrators can enable this experience simply within the Now Assist Admin Console.
    3.  Enhanced supplier summarization feature with multi-language support powered by Now Assist. Summaries are now available in multiple languages beyond English, enabling broader accessibility and improved global collaboration.
-   **Version 2.0.4 - February 2025**

    Conversational intake flows for suppliers provide an easy, automated and faster way to get queries answered and request status updates.

-   **Version 2.0.4 - February 2025**

    Conversational intake flows for suppliers provide an easy, automated and faster way to get queries answered and request status updates.

-   **Version 1.0.1 - November 2024**

    This feature enables supplier managers to view a summary of completed tasks and upcoming steps for complex cases, such as onboarding. Users, fulfillers, or supplier managers can gain this insight without navigating to multiple screens or performing extra actions.
