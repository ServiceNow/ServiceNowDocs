# Scrum Common release notes

Version history for the IT Business Management Scrum Common on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.4 - April 2024**

    Fixed the issues with order of states in the Cumulative Flow Diagram in Agile 2.0 and SAFe dashboards.

-   **Version 1.3.3 - December 2023**

    Fixed: Fixed the issues with Forecast Scores in the Sprint/Release/Epic Burndown charts. Forecast scores are now replaced with the Trend line scores, that will display the forecast based on the selected sprint/Release/Epic.

-   **Version 1.3.2 - September 2023**
    -   New: Support for dark theme
    -   Fixed: Time elapsed calculation for Sprint and Release dashboards
-   **Version 1.3.1 - May 2023**

    Updates to the localization support.

-   **Version 1.3.0 - July 2021**
    -   New:
        -   Support for Business Stakeholder roles sn\_agile\_read for Agile Development 2.0 and sn\_safe\_read for Essential SAFe and Portfolio SAFe.
        -   Support for custom story states in area charts of all Agile 2.0 and Essential SAFe dashboards.
    -   Fixed: Bug fixes.
-   **Version 1.2.2 - January 2021**

    Fixed: Bug fix.

-   **Version 1.2.1 - August 2020**

    Added support for Cycle Time reports.

-   **Version 1.1.0 - June 2020**

    New: Contains changes to support Performance Analytics content pack for Agile 2.0 v1.1.0 and Performance Analytics content pack for Essential SAFe v1.1.0.
