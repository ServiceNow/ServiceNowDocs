# Collaboration applications - common release notes

Version history for the Collaboration applications - common on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.0 - August 2020**
    -   Initial release.
    -   This app will be utilized as a common app by other BUs as well to manage the channel id on integrating their application with collaboration applications such as Slack &amp; MS Teams.
    -   This will not be available for customers as a standalone download application.
