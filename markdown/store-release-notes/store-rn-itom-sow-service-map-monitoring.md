# Service Operations Workspace Service Map Monitoring release notes

Version history for the Service Operations Workspace Service Map Monitoring application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 26.5.0 - August 2025**

    Fixed: Impacted path alerts panel - Fix incorrect mapping of alerts that are part of an alert group to the parent alert's service

-   **Version 26.4.0 - May 2025**
    -   Fixed:
        -   Accessibility improvements
        -   Unexpected behavior when navigating from the Service Dashboard fixed
-   **Version 26.3.0 - February 2025**
    -   New:
        -   Adding the option to pin an alert at the top to the Impact path panel, so users can focus on it and find it easily.
        -   Allowing users to navigate from a specific alert in Express List to the impact path of that specific alert in the Service Map Monitoring \(using the link for the Service map from the impacted service found on the Preview Panel\)
    -   Changed: Performance improvements
-   **Version 26.1.0 - November 2024**

    Fixed: Performance improvements.

-   **Version 26.0.2 - August 2024**

    Enable operators to visualize relationships between Configuration Items \(CIs\) and alerts with real-time updates and detailed impact paths. Enhance troubleshooting and proactive management by quickly identifying root causes and dependencies for both discovered and application services.
