# Security Operations Setup Assistant release notes

Version history for the Security Operations Setup Assistant application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.4.1 - May 2025**

    Fixed: One bug has been resolved as part of this release.

-   **Version 10.4.0 - February 2025**

    Changed: Updated the existing Workflows with Flow Designer Flows in Setup Assistant \(Security Incident Playbook Settings\).

-   **Version 10.3.10 - November 2024**

    Setup Assistant walks you through setting up Security Incident Response and related third-party integrations for your environment.
