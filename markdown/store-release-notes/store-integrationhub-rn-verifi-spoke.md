# Verifi Spoke release notes

Version history for the Verifi Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - April 2026**
    -   Changed:
        -   Added activity currency output field in case lookup responses to capture the currency code used for case resolution activities.
        -   Updated request builders with resource path and HTTP method outputs for improved API endpoint handling.
        -   Added Verifi Skyflow integration changes for enhanced data security capabilities.
-   **Version 1.0.0 - August 2024**

    Verifi Spoke is an application designed to address the complexities of card dispute resolution by facilitating merchant collaboration. In the dispute resolution process, various stakeholders, including cardholders, issuing banks, card networks like Visa/Mastercard, acquiring banks, and merchants, play crucial roles. Collaborating with merchants is a key step in deflecting chargebacks and saving time and effort for all parties involved.
