# Oracle integration for Finance Close Automation release notes

Version history for the Oracle integration for Finance Close Automation in ancillary software on the ServiceNow Store.

**Important:** The Finance Close Automation application has been deprecated and is no longer maintained.

## Version history

-   **Version 1.0.0 - April 2020**

    New: The Oracle integration for Finance Close Automation enables the posting of journal entries from Finance Close Automation to Oracle E-Business Suite R 12. Supported Oracle versions include 12.1.3 to 12.2.8.
