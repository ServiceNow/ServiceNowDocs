# HR Success Dashboard indicators release notes

Version history for the HR Success Dashboard indicators application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.15 - May 2025**

    Uptake of latest version of the dependent apps.

-   **Version 1.0.10 - November 2023**

    Uptake the latest version of dependent apps.

-   **Version 1.0.8 - May 2023**

    Fixed: Bug fixes.

-   **Version 1.0.5 - November 2022**

    The HR Success Dashboard provides insights to your HR leadership team and process owners to measure the performance of their HRSD implementation. Using the KPIs defined by ServiceNow, you can visualize metrics and interactions to better understand call deflections, self-service, and other key metrics to drive continual improvements and make data-informed decisions.
