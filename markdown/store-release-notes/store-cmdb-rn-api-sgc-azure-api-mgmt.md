# API Service Graph Connector for Azure API Management release notes

Version history for the API Service Graph Connector for Azure API Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.1 - June 2026**

    Security fix.

-   **Version 2.3.0 - April 2026**
    -   New:
        -   Enhanced security controls
        -   Minor additional data mappings
-   **Version 2.2.1 - November 2025**

    Fixed: Tag processing loop issue.

-   **Version 2.2.0 - August 2025**

    Fixed: Performance improvements and minor bug fixes.

-   **Version 2.1.0 - May 2025**
    -   Populate the CMDB data model for APIs and API Gateways using a prebuilt Azure API Management integration.Configuring this API Service Graph Connector will retrieve the following objects from an Azure API Management service:
        -   Azure API Management Services
        -   APIs
        -   Operations
        -   Backends
        -   Subscriptions
        -   API Products
        -   API Consumers \(users\)
