# Firewall Audits and Reporting release notes

Version history for the IT Operations Management Firewall Audits and Reporting on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.10.0 - June 2026**

    Fixed: Manage access through predefined query access controls for all Firewall Audits and Reporting tables

-   **Version 1.9.4 - April 2026**
    -   Panorama object discovery:
        -   New: Discovery now automatically identifies and populates Address Objects and Service Objects defined in PaloAlto Panorama into the ServiceNow CMDB, scoped per device group and firewall manager.
    -   AI Agent support:
        -   New: Use AI Agents to automate firewall rule tasks in Palo Alto Networks Panorama, including creating, validating, and committing rules. This feature requires the AI Agent for Discovery application.
-   **Version 1.8.0 - December 2025**

    Changed: Applied security fixes and updates.

-   **Version 1.7.0 - November 2024**

    Changed: Applied security fixes and updates.

-   **Version 1.6.0 - May 2024**

    Changed: Firewall classic dashboards are removed, all old links point to the dashboard page in the Workspace instead.

-   **Version 1.5.0 - November 2023**

    Changed: Component Accessibility WCAG 2.1.

-   **Version 1.4.0 - August 2023**

    What's New:

    Workspace for firewall administrators: View discovered firewall Inventory and insights with brand new workspace for managing firewall policies.

-   **Version 1.3.0 - February 2023**

    Minor fixes

-   **Version 1.1.3 - May 2022**
    -   Fixed:
        -   Fixes in pattern for fetching firewall security policies.
        -   Spelling corrections for "Palo Alto" manufacturer.
        -   Fixes to ACLs on cmdb\_ci\_appl and cmdb\_ci\_group introduced in the initial version of this app.
-   **Version 1.0.2 - September 2020**
    -   Palo Alto - PAN integration via Patterns.
    -   CMDB data model for Firewall. Discovery of Palo Alto firewall. Service Mapping and Discovery Patterns App has patterns to discover Fortinet, Cisco, and Juniper firewalls.
    -   Firewall request management and task creation via workflows.
    -   Firewall audit framework for discovered firewall policies.
    -   PA dashboards for a single pane of glass views.
-   **Version 1.0.1 - July 2020**
    -   Palo Alto - PAN integration via Patterns.
    -   CMDB data model for Firewall. Discovery of Palo Alto firewall. Service Mapping and Discovery Patterns App has patterns to discover Fortinet, Cisco, and Juniper firewalls.
    -   Firewall request management and task creation via workflows.
    -   Firewall audit framework for discovered firewall policies.
    -   PA dashboards for a single pane of glass views.s
