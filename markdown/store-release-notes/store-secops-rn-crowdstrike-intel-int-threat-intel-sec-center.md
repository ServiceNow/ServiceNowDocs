# Threat Intelligence Security Center integration with CrowdStrike Intelligence release notes

Version history for the Threat Intelligence Security Center integration with CrowdStrike Intelligence application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.4 - March 2025**
    -   Fixed: Issue related to threat lookup intermittent failures.
    -   Changed: Changed application name from "CrowdStrike Intelligence Integration for Threat Intelligence Security Center" to "Threat Intelligence Security Center integration with CrowdStrike Intelligence".
-   **Version 3.0.3 - August 2024**

    New: Pushed changes to support localization.

-   **Version 3.0.0 - May 2024**

    Changed: The Base URL is newly added for Crowdstrike premium feed in the Threat Intelligence Security Center Catalog.

-   **Version 2.0.0 - February 2024**

    The CrowdStrike Intelligence Integration for Threat Intelligence Security Center\(TISC\) serves as an observable enrichment solution. It leverages CrowdStrike Falcon Intelligence to recognize and scan for malware, enriching observables through automated threat lookups based on matching results.
