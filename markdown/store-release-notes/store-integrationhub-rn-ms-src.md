# Microsoft Security Response Center Spoke release notes

Version history for the Microsoft Security Response Center Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - September 2025**

    Fixed - Required ACL\(s\) or Role\(s\) for agents.

-   **Version 1.2.2 - August 2025**
    -   Fixed:
        -   AI Agents: Renaming and Output transformation strategy set
        -   Actions: Get Security Update Details, Submit Abuse Report
-   **Version 1.2.1 - March 2025**

    Minor release with new AI agents.

-   **Version 1.1.0 - June 2023**

    New - Authentication template to the spoke so that spoke setup is easier.

-   **Version 1.0.1 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version \(1.0.0\) - December 2020**
    -   New:
        -   Abuse Management
            -   Submit Abuse Report
        -   Security Management
            -   Get Security Update Details
            -   Look up Security Updates
            -   Look up Security Updates By Key
