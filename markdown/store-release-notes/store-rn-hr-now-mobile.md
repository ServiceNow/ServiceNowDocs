# Now Mobile release notes

Version history for the Now Mobile application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 30.2.2 - June 2026**

    Enhanced A11y support.

-   **Version 30.1.3 - March 2026**

    Compatibility support for the Australia release.

-   **Version 30.1.2 - March 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 30.0.1 - August 2025**

    New: Welcome banner image in Coral theme.

-   **Version 29.3.2 - February 2025**

    Fixed bugs related to user interface and accessibility.

-   **Version 29.2.1 - November 2024**

    Fixed user interface issues.

-   **Version 29.1.10 - May 2024**

    No new feature was delivered in May '24.

-   **Version 29.1.1 - February 2024**

    Minor fixes.

-   **Version 29.0.1 - November 2023**

    Added Employee News browse and search experience to Now Mobile.

-   **Version 28.1.5 - August 2023**

    Streamlined the Mobile Employee Service Portal \(MESP\) topic screen and multi-select experience in Now Mobile. See Webviews for mobile release notes for additional highlights.

-   **Version 28.1.2 - June 2023**

    New: Pre-built integration with SAP Concur allows approvers to:

    -   View expense reports.
    -   View line items for an expense report.
    -   View attachments for a report.
    -   Approve or send back the expense report with comments.
-   **Version 27.1.0 - February 2023**

    New: Delivered the new Next Experience theme, applied consistently across all screens, for a modern and consistent look and feel across the Now Mobile application.

-   **Version 26.0.2 - September 2022**

    Fixes to embedded web view for surveys.

-   **Version 26.0.1 - August 2022**

    Fixed: Multiple My Items sections were showing up on the Home tab for customers taking the May 2022 upgrade.

-   **Version 25.0.1 - May 2022**
    -   New:
        -   Improved navigation with the Quick links section on the home screen.
        -   Added content recommendations with the Recommended for you section on the support tab.
        -   Introduced cross-channel favorites feature – access all favorites, favorite articles, and requests from the Now Mobile app.
    -   Fixed: Fixed MESP bundle.
