# Whois integration for Security Operations release notes

Version history for the Security Operations Whois integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.4.1 - June 2026**

    Fixed: Moved the configuration tile validation endpoint to a configurable system property

-   **Version 10.4.0 - August 2024**

    Changed: Migrated workflows to flow designer.

-   **Version 10.3.5 - May 2023**

    Fixed: Optional tag was missing in the capability filter xml file, which is now fixed.

-   **Version 10.3.4 - February 2023**

    New: Updated to support Security Incident Response workspace.

-   **Version 10.3.1 - June 2020**

    Fixed: Bug fix for observable enrichment.

-   **Version 10.0.1 - April 2020**
    -   Changed: Some Whois entries columns are hidden when data is not available from the WhoisXML API
    -   Fixed: Formatting changes to populate data in Whois entries
-   **Version 10.0.0 - March 2020**

    New: Implementation Flow to support Integration capability framework v2.0


## Version history - Madrid

-   **Version 8.0.9 - April 2020**

    Fixed: Formatting changes to populate data in Whois entries

-   **Version 8.0.8 - July 2019**
    -   New: Recertified for New York
    -   Fixed: Query performance fixes
