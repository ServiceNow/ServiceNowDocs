# Google Cloud Datastore spoke release notes

Version history for the Integration Hub Google Cloud Datastore spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - September 2023**

    Fixed: The license requirements.

-   **Version 1.0.2 - September 2022**

    Patch release of Google Cloud Datastore Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.1 - June 2021**

    Patch release of the Google Cloud Datastore spoke for IntegrationHub. It includes a fix for the value of a variable not being populated in the output of some actions.

-   **Version 1.0.0 - July 2020**

    Provides actions to automate the management of Google Cloud Datastore.
