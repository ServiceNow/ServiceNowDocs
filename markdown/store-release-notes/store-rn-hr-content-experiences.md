# Content Experiences release notes

Version history for the Content Experiences application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 33.0.0 - June 2026 \(Australia\)**

    Updated to support the latest version of the dependent apps.

-   **Version 32.3.1 - June 2026 \(Zurich\)**

    Updated to support the latest version of the dependent apps.

-   **Version 32.3.0 - May 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 32.2.2 - January 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 32.2.1 - December 2025**

    Added additional configurations for the widget header and widget alignment to make Content Experiences more flexible.

-   **Version 32.1.11 - August 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 32.1.4 - May 2025**
    -   The following user interface bugs are fixed:
        -   Fixed video alignment and display.
        -   Improved the layout of the right-to-left languages.
        -   Add audience, as a non-admin, to publish plans in an instance that has the 'Allow Ownership for Audiences' property enabled.
        -   Get topic level security in My Favorites widget and Conversational LLM support to resolve tasks and approvals in Employee Center.
        -   Get new Employee Communications capabilities with an Workplace Service Delivery integration, a calendar widget view, RSVP/Wait listing tools, events notifications and more;
        -   Get improvements on experience feedback and feedback configurations; leverage notifications in news content with Employee Center Pro.
-   **Version 32.1.3 - February 2025**
    -   Fixed bugs related to the user interface, including:
        -   Layout support for right-to-left languages
        -   Enabled news preview from the Campaign Builder Experience
        -   Content order now adheres to the set order value
-   **Version 32.0.5 - November 2024**

    Fixed bugs related to user interface and accessibility.

-   **Version 32.0.2 - August 2024**

    Added demo data for commonly used Campaigns such as new hire campaigns, recurring birthday and anniversary campaigns, tax time, and benefits re-enrollment.

-   **Version 31.0.3 - February 2024**

    Minor fixes.

-   **Version 30.0.2 - November 2023**
    -   Reformatted the usability of Dynamic and Recurring campaigns to make them easier to configure.
    -   Added "hints" to automatically generate plain text descriptions for what the dynamic / recurring campaigns will accomplish.
    -   Updated how the Campaign targets users retroactively to give more control over how far back a campaign will pull in users to target content.
-   **Version 29.0.2 - August 2023**

    The release primarily focuses on addressing minor issues and bug fixes.

-   **Version 28.0.4 - March 2023**
    -   Fixed:
        -   Content schedule on UIB workspace is how when when CG is installed.
        -   Rich content widget no longer errors on upgrade from the November/December 2022/Utah true up versions to the March 23 version.
        -   Content Library \(SCA\) now enforces ownership restrictions.
        -   Audience user criteria and user type can now be changed after creation.
-   **Version 28.0.0 - February 2023**

    See Content Publishing release notes for updates made to the Rich Content Editor for improved Topic Microsite experience.

-   **Version 27.0.3 - November 2022**

    Updated Campaigns functionality to include support for recurring campaigns \(for example work anniversaries\).

-   **Version 26.0.1 - August 2022**

    See Content Publishing and Content Governance release notes for release highlights.

-   **Version 25.0.5 - June 2022**

    Fixed: All of the Content Experience ATF tests are failing due to missing RCA

-   **Version 24.0.4 - February 2022**
    -   New:
        -   Adds support for a new “Styled Content” format called “Video” allowing you to publish banners and announcements within supported widgets with embedded videos.
        -   Provides the ability to launch a full-portal preview of any portal content created within the Content Library.
        -   Adds new supported portal content types within the Content Library including: Events, Rich-text, and Image-based Link.
        -   Provides system properties and configuration capabilities that allow you to designate ownership for Topic pages within EC Pro.
        -   Adds a new configuration option for Link Content allowing the content author to select where the link loads \(within the current browser page or in a new tab\).
-   **Version 23.0.4 - December 2021**

    Minor fixes.

-   **Version 23.0.3 - November 2021**

    Fixes and minor changes.

-   **Version 22.0.1 - September 2021**
    -   Changed:
        -   Added ability to assign Styled Content content type to the Content Experiences Widget
        -   Updated the Content Experiences Widget to display a gradient under the text rather than a blurry shadow box
