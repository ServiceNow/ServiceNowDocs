# Notify Webex Connector release notes

Version history for the Notify Webex Connector application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - December 2025**
    -   Fixed:
        -   Title of the conference modal now displays short description.
        -   Improved handling of duplicate user emails to prevent errors.
-   **Version 1.3.0 - May 2025**

    New: The "deny-unless" authentication ACL allows only authenticated users to access the tables related to collaboration services. This ACL significantly improves the security of the data and prevents unauthorized access.

-   **Version 1.1.1 - April 2024**

    Fixed: Delete webhook action fails due to truncation of webhook ID.

-   **Version 1.1.0 - August 2023**

    New: Support for Notify UIB plugin.

-   **Version 1.0.1 - December 2022**

    New: Support to Tokyo release.

-   **Version 1.0.0 - October 2021**

    Notify provides support for SMS and voice channels on the NOW Platform for rapid communication with internal team members, customers, and contractors. It provides workflows that shorten the timeline for mobilizing resources to begin working on an incident or task by alerting the right people for action.
