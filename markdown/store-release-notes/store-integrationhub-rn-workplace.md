# Workplace from Facebook spoke release notes

Version history for the Integration Hub Workplace from Facebook spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.2.1 - November 2025**

    New: Added read-only fields for security measures.

-   **Version 4.1.2 - September 2023**

    Fixed: The license requirements.

-   **Version 4.1.1 - August 2023**

    Fixed: Look up Organization Members Stream and Look up Conversations Stream actions for pagination issue.

-   **Version 4.1.0 - May 2023**

    Minor release of Workplace from Facebook Spoke for Integration Hub. This version includes an Authentication Template which helps in setting up the spoke easily from Flow Designer.

-   **Version 4.0.6 - December 2022**

    Patch release for Workplace from Facebook Spoke for IntegrationHub. This version adds localization support for the application.

-   **Version 4.0.5 - September 2022**

    Patch version of Workplace from Facebook Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 4.0.4 - March 2022**

    New: In this release, actions related to Message Notification and User Management have been added.

-   **Version 4.0.1 - November 2021**

    New: In this release, actions related to Software Asset Management have been added.

-   **Version 3.0.3 - July 2021**

    Patch version of Workplace from Facebook Spoke for IntegrationHub. This version adds KMF modules to add additional security to password2 fields and Rome compatibility.

-   **Version 3.0.2 - December 2020**

    Patch release of the Workplace from Facebook spoke for IntegrationHub.

-   **Version 3.0.1 - November 2020**
    -   Provide actions to easily automate notifications and group management in Workplace from Facebook
    -   Improves collaboration across task management such as incident, problem, and change
