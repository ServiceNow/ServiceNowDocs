# Financial Services Operations integration with Jack Henry Associates jXchange release notes

Version history for the Financial Services Operations integration with Jack Henry Associates jXchange application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - August 2025**

    New: As part of this release, Deny ACLs were introduced on the existing tables. These Deny ACLs will restrict access for unauthenticated users on CRUD operations.

-   **Version 1.1.2 - November 2023**
    -   Updated:
        -   'Create or update loan product model' flow to use 'sn\_ent\_loan\_account\_model' table instead of 'sn\_bom\_loan\_account\_model'
        -   'Create Deposit Product Model' flow to use 'sn\_ent\_deposit\_account\_model' table instead of 'sn\_bom\_deposit\_account\_model'
-   **Version 1.1.1 - May 2023**
    -   New: Added localization changes.
    -   Changed: Modified plugin name to Financial Services Operations Integration with Jack Henry jXchange.
-   **Version 1.1.0 - February 2023**
    -   New: Added financial transaction records for financial accounts.
    -   Changed:
        -   Modified application name to Financial Services Operations integration with Jack Henry Associates jXchange
        -   Modified Product Model decision table
        -   Added B2B Certificate of Deposit Financial Account
-   **Version 1.0.0 - November 2022**

    Jack Henry is a leading payment processing company for the financial services industry. With this integration to Financial Services Operations, banks will save time and cost by pre-populating information for various services, while also gaining visibility on client profile data in just a click to provide seamless client servicing.
