# Utility Actions spoke release notes

Version history for the Integration Hub Utility Actions Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - June 2026**
    -   Security patch for non-glide ACLs
    -   Fixed: getRefRecord\(\) Scoping Bypass for Convert Records to Array of Objects and Convert Record to Object actions
-   **Version 1.3.1 - April 2026**

    Fixed Australia compatibility for the Build Zip \(OOB\) action.

-   **Version 1.3.0 - October 2024**

    Enabled Java 11 support for code signing and MID script files.

-   **Version 1.2.4 - August 2024**

    Fixed: Only applicable to Blue Prism Spoke's "Build File" action.

-   **Version 1.2.3 - September 2023**

    Fixed: 'licensable' field set to be true in plugin.xml.

-   **Version 1.2.2 - April 2023**

    Improved installation performance.

-   **Version 1.2.1 - March 2023**

    Fixed: Fixed the parsing the string with multiline properly in the 'Generate Encoded Query' action.

-   **Version 1.2.0 - August 2022**
    -   New:
        -   Added 4 new actions
            -   Convert Record to Object
            -   Convert Records to Array Object
            -   Generate Encoded Query
            -   Look up Latest Comment on Record
-   **Version 1.1.0 - November 2021**

    New: Minor release of Utility Actions Spoke for IntegrationHub. This version adds 2 new actions \(Generate Encoded Query, Look up Latest Comment on Record\).

-   **Version 1.0.1 - November 2020**

    Patch release of the Utility Actions spoke for IntegrationHub.

-   **Version 1.0.0 - July 2020**

    Provides actions to automate the management of zips and files in ServiceNow.
