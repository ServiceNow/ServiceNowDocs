# Oracle Cloud IAM spoke release notes

Version history for the Integration Hub Oracle Cloud IAM spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.4 - June 2026**

    New ACLs added

-   **Version 1.1.3 - September 2022**

    Patch version of Oracle Cloud IAM Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.1.1 - June 2022**

    Fixed: Patch release of Oracle Cloud IAM Spoke. This version includes a fix for a message with the output of 500 when incorrect details are provided within the action - Bulk Move Resources.

-   **Version 1.1.0 - October 2021**

    Fixed: Security bug.

-   **Version 1.0.5 - September 2021**

    Fixed: An error observed at action - 'Look up User Group Membership'.

-   **Version 1.0.4 - July 2021**

    Patch release of the Oracle IAM spoke for IntegrationHub. This version adds KMF modules for additional security of the password2 fields and Rome compatibility.

-   **Version 1.0.3 - November 2020**

    Provides actions to automate the management of Oracle Identity and Access Management.
