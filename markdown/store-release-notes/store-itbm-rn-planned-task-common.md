# Planned Task Common release notes

Version history for the Planned Task Common on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - December 2025**

    Incorporates platform features of read-only.

-   **Version 1.0.4 - May 2023**

    Updates to the localization support.

-   **Version 1.0.3 - July 2021**

    Fixed: Bug fixes.

-   **Version 1.0.2 - January 2021**

    Fixed: Bug fixes.

-   **Version 1.0.0 - August 2020**

    Initial release. Planned Task Common configures the Planned Task table for use with the Work Progress Status for Agile Teams and Work Progress Status for SAFe apps.
