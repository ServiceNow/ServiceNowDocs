# Workforce Optimization for ITSM Admin Center release notes

Version history for the Workforce Optimization for ITSM Admin Center application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - May 2025**

    Minor bug fixes.

-   **Version 1.2.2 - November 2024**

    No Major feature changes. Contains minor changes and bug fixes.

-   **Version 1.0.2 - August 2024**
    -   Workforce Optimization Admin center will help system admins to do the setup of Workforce Optimization. From Admin Center, admins can:
        -   Mark assignment groups as WFO groups
        -   Add or remove managers and members from WFO groups
        -   Add or revoke application access to WFO groups and managers
