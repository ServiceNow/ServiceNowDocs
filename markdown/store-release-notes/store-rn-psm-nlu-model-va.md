# Natural Language Understanding Models for Sourcing and Procurement Operations release notes

Version history for Natural Language Understanding Models for Sourcing and Procurement Operationson the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.5 - May 2023**

    Changed: Renamed this application to Natural Language Understanding Models for Sourcing and Procurement Operations. This application was formerly named Procurement Service Management NLU Model.

-   **Version 2.0.4 - September 2022**
    -   Minor bug fixes.
    -   This plugin activates the Natural Language Understanding \(NLU\) content pack for Shoppinghub, which improves the Virtual Agent experience for the employees. Ensure all dependencies are activated before enabling this plugin.
