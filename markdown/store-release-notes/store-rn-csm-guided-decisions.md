# Guided Decisions release notes

Version history for the Guided Decisions applications on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 25.0.0 - May 2023**
    -   New:
        -   Provides a visual no-code experience for decision tree authors called Decision Tree Builder. Decision tree authors can map their procedures or troubleshooting processes as decision trees within minutes.
        -   Once a decision tree is designed, decision tree authors can activate it with a click of a button.
        -   Decision tree authors can reuse already provided answers and navigate users through the tree.
        -   Decision tree authors can see a choice list based on a reference field.
-   **Version 24.0.1 - November 2022**

    Updates to accommodate error handling in Guided Decision execution.

-   **Version 23.0.0 - August 2022**

    Guided Decisions guides agents through a set of questions to determine the action that will help them resolve an issue or complete a task in the most optimal way. It asks them questions based on previous answers and gives guidance on the next step in the resolution process. Guided Decisions reduce the cognitive burden on agents and improves their productivity. Guided Decisions can be embedded in a Playbook or served as a Recommended Action.
