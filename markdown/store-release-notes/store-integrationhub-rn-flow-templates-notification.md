# Flow Templates for Notifications release notes

Version history for the Integration Hub Flow Templates for notifications on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.4 - June 2024**

    Fixed: Table ACL checks.

-   **Version 1.2.3 - January 2023**

    Fixed: Dependencies.

-   **Version 1.2.2 - October 2022**

    Fixed: Improve installation performance of application.

-   **Version 1.2.1 - September 2022**

    Fixed: Dependencies.

-   **Version 1.2.0 - August 2022**

    New: Added 2 new flow templates.

-   **Version 1.1.1 - October 2021**

    New: New template added to the app - "Create a Slack Channel and Zoom Meeting when a Major Incident is Created".

-   **Version 1.0.5 - August 2021**

    New: Minor change to application name \(capitalized "Notifications"\).

-   **Version 1.0.1 - June 2021**
    -   New:
        -   Notify a workspace when an Incident is created Triggered when a new Incident is created. The flow sends Incident details along with the message you provide to the recipient\(s\) you choose. Spokes: Microsoft Teams, Slack, Cisco Webex Teams, Twilio, Gmail, Email
        -   Notify a workspace when a Task is created in any Task Table Triggered when a record is created in the Task table/Task table extension you choose. The flow sends Task details to the recipient\(s\) you choose. Spokes: Microsoft Teams, Slack, Cisco Webex Teams
        -   Notify a workspace when a Task is created or updated in any Task table Triggered when a record is created or updated in the Task table/Task table extension you choose. The flow sends Task details to the recipient\(s\) you choose. Spokes: Microsoft Teams, Slack, Cisco Webex Teams
        -   Send a Platform Notification when a record is created Triggered when a new record is created. The flow sends a Platform Notification to the recipient\(s\) you choose. Spokes: N/A - uses ServiceNow Notifications.
