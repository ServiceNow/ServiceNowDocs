# Export to PowerPoint for Enterprise Architecture release notes

Version history for the Export to PowerPoint for Enterprise Architecture on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - June 2026**

    New: Added support for query\_range access control lists \(ACLs\).

-   **Version 1.0.2 - August 2025**
    -   Changed: Plugin name changed from Export to PowerPoint for Application Portfolio Management to Export to PowerPoint for Enterprise Architecture.
    -   Fixed: Issue while exporting data to PowerPoint.
-   **Version 1.0.1 - February 2024**

    Fixed downloading of ppt.

-   **Version 1.0.0 - September 2022**

    Export to PowerPoint is the base application to export data from ServiceNow.
