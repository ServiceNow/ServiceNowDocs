# Employee Experience Pack: Return to Workplace release notes

Version history for the Employee Experience Pack: Return to Workplace on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.1.0 - November 2022**
    -   Please note that version 3.1.0 of this Experience Pack is officially compatible with the San Diego release only.
    -   Included content details:
        -   Lifecycle Event for Employee Return to Workplace, including HR Service, HR Template, Record Producer, and Case Creation Service Configuration.
            -   5 Activity Sets:
                -   Notify Employees
                -   Get Ready to Return
                -   Final Prep
                -   Back at the workplace
                -   Follow Up
            -   7 Activities for the Employee Return to Workplace Lifecycle Event
                -   Employee Form with data mapping to target subsequent activities
                -   HR Task for HR to review employee concerns
                -   Fulfiller Activity for Facilities to reactivate badge and parking pass
                -   URL Task for the employee to review updated policies
                -   Checklist task for packing reminder to the employee before return
                -   Mark when complete task to return borrowed assets to IT
                -   Survey task to the employee to give feedback on the process
    -   Lifecycle Event for the employee's Return to Workplace Location Readiness, including HR Service, HR Template, Record Producer, and Case Creation Service Configuration.
        -   4 Activity Sets:
            -   Task Force Initiation
            -   Update space and equipment
            -   Final Readiness Check
            -   Location Reopens
        -   9 Activities for the employee's Return to Workplace Location Readiness Lifecycle Event
            -   Mark when complete task to schedule location team training
            -   Checklist task to order additional supplies for the supply room
            -   Mark when complete task to install protective materials and equipment in common areas
            -   Catalog Item to test devices in common areas
            -   Checklist task to request sanitation services
            -   Mark when complete task to schedule a safety inspection
            -   HR Task to create communication materials for specific location needs
            -   HR Service for Go / No Go Decision, including final readiness checklist and approval steps
            -   Mark when complete task for team standup on the first day back on site
    -   1 Campaign Return to Workplace for employee who is returning to the workplace
        -   Announce and Plan Return bundle
        -   Welcome back to the workplace bundle
        -   Audience for users returning to the workplace
        -   2 portal banner content items, 1 for each bundle
        -   2 Notification email content items, 1 for each bundle
    -   1 Survey: Employee Return to Workplace Readiness Feedback
    -   1 Visual Task Board: Return to Workplace Taskforce Planning
    -   2 standalone Notification Content items
        -   Employee Readiness Survey Notification email
        -   Employee returning to workplace sample Notification email
    -   4 Knowledge Articles for the HR Knowledge Base
        -   Company updates about returning to the workplace
        -   Manager guide to returning to work
        -   Social distancing and safety tips for common workspaces FAQ
        -   Workplace Health and Safety Policy FAQ
    -   2 standalone Styled Content items for use in portal in lieu of welcome banners:
        -   Get Ready to Return
        -   Welcome Back
