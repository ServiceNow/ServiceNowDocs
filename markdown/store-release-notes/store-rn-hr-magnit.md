# HR Service Delivery Integration with Magnit release notes

Version history for the HR Service Delivery Integration with Magnit application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.2 - February 2024**

    Minor fixes.

-   **Version 2.1.1 - January 2024**

    Fixed: Improved tables' security by fixing ACLs with empty roles.

-   **Version 2.1.0 - May 2023**

    New:

    -   Ability to sync the status of an onboarding item, with Magnit - WAND system.
    -   Ability to transfer attachments, which are assigned at the onboarding task level.
-   **Version 2.0.0 - February 2023**
    -   With HR Service Delivery and Magnit better together, facilitate importing contingent worker data and onboarding items from Magnit into ServiceNow.
    -   Subsequently, an HR profile record is created and tracked for fulfilment in ServiceNow using Lifecycle Events.
