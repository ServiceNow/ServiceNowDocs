# Confluent Kafka REST Proxy Spoke release notes

Version history for the Confluent Kafka REST Proxy Spoke on the ServiceNow Store.

## Version history

-   **Version 1.0.0 - March 2021**

    New: Confluent Kafka REST API Proxy spoke provides actions to automate inbound and outbound communication between ServiceNow and Kafka via the Confluent Kafka REST API proxy.
