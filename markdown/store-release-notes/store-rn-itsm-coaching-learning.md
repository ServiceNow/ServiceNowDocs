# Coaching with Learning release notes

Version history for the Coaching with Learning application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.4.2 - March 2026**

    Fixed Translations.

-   **Version 5.4.0 - December 2025**
    -   Minor bug fixes
    -   Security uptakes
-   **Version 5.3.4 - August 2025**

    Minor bug fixes.

-   **Version 5.3.3 - February 2025**

    Fixed: Translation bugs.

-   **Version 5.3.2 - November 2024**

    No Major feature changes. Contains minor changes and bug fixes.

-   **Version 5.2.2 - August 2024**

    Fixed accessibility issues.

-   **Version 5.2.0 - May 2024**

    Fixed: WCAG bugs.

-   **Version 5.1.2 - March 2024**
    -   New:
        -   Uptook SRP changes for the WFO SOW Overview page
        -   Fixed accessibility issues
    -   Fixed:
        -   Security bugs
        -   WCAG Issues
-   **Version 5.1.1 - February 2024**

    Fixed: WCAG issues and security bugs.

-   **Version 5.1.0 - November 2023**
    -   New: Uptook SRP changes for the WFO SOW Overview page Fixed Accessibility issues.
    -   Fixed: Fixed Security bugs
-   **Version 5.0.0 - August 2023**
    -   New: Added quiz type courses including telemetry Migration of UI pages to layout 3.0
    -   Fixed: i18n issues.
-   **Version 4.0.0 - February 2023**
    -   New: Discover, Self enroll, assigned courses and learning paths on the Service Operations Workspace side
    -   Changed: My Learning
-   **Version 2.2.2 - December 2022**

    Changed: Updated the dependency version 5.0.2

-   **Version 3.0.2 - August 2022**
    -   New:
        -   As a manager, you can create a learning path, add relevant skills and course items, and publish the path. When your agents complete the path, any skills that are associated with the path are automatically added to the agent's profile.
        -   As an agent, you can improve your skills set by completing learning tasks from internal or external learning content in Service Operations Workspace.
-   **Version 2.2.1 - May 2022**

    New: Enabled localization support.

-   **Version 2.0.2 - March 2022**

    -   New:
        -   As a coach, use Coaching with Learning to train your agents with internal learning content and external content like Pluralsight and Cornerstone. You can organize similar content in catalogs and assign learning tasks and track completion.
        -   As an agent, complete learning tasks by using Service Operations Workspace with Workforce Optimization for ITSM.
    "

-   **Version 1.0.3 - September 2021**

    Manage internal learning content and access external learning content, like Pluralsight and Cornerstone, from a central location.
