# Issue Auto Resolution for HR release notes

Version history for the Issue Auto Resolution for HR application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.1.0 - June 2026**

    New: Updated modal UIs in Now Assist for HRSD for Accessibility \(WCAG 2.2 AA requirements\) - Zoom 400% and reflow

-   **Version 4.0.3 - June 2024**

    Fixed security issues.

-   **Version 4.0.0 - August 2023**
    -   New:
        -   Updated criticality model for better accuracy
        -   Recommend Topic/Dialog flows to auto-resolve noncritical cases;\(limited availability only\)
        -   Updated recommendations preview on Email
-   **Version 3.0.0 - February 2023**
    -   New:
        -   Simplified Login Experience: This feature will allow users to access the recommended portal link without credentials through MFA, benefiting deskless/non-office workers \(Limited Time authentication to be pre-installed\)
        -   Agent Feedback: This feature allows agents to provide criticality feedback to HR Cases which can immensely help improve the ML models
    -   Changed: IAR Configuration: New Language model, pre-processing, and post-processing extensions have been added to the configuration and are active by default.
-   **Version 2.0.0 - November 2022**

    This version includes IAR for HR Dashboard feature

-   **Version 1.0.1 - August 2022**

    Issue Auto Resolution for HR enables AI-powered HR case resolution and escalation using pre-trained models. The application provides capabilities to auto-resolve high-volume, non-critical HR cases and escalate sensitive or critical cases for agent assistance.
