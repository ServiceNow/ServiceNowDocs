# Case lines and workflows release notes

Version history for the Case lines and workflows application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.5.1 - June 2026**
    1.  Non-Glide Cobalt Raven ACLs in Product Code - Security enhancements to query access controls for Customer Service Document Template.
    2.  Granular and Feature specific admin roles
-   **Version 4.3.0 - March 2026**

    Changed: Customers now have the ability to view or create case characteristics.

-   **Version 4.1.0 - November 2025**

    Changed: Minor improvements

-   **Version 4.0.0 - August 2025**
    -   New: Multiple entitlements support on Case lines
    -   Changed: Increased length of the Character value field
    -   Fixed: Minor access issues
-   **Version 2.2.0 - May 2025**
    -   New: Added support for Install base and Asset on Case lines.
    -   Changed: Added above fields to existing form and list views.
-   **Version 2.1.0 - February 2025**

    Contains minor bug fixes.

-   **Version 1.0.0 - November 2024**

    The customer service case line feature enables the Case table to contain many line items on a case record. The line items are product instances and can be extended for various entities such as orders, order lines, invoices, invoice lines, contracts, etc.
