# Security Case Management common PAD artifacts release notes

Version history for the Security Operations Security Case Management common PAD artifacts on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.12 - January 2026**
    -   Fixed:
        -   Fixed the subflow "Submit to Sandbox - ESCM Activity" opening issue.
        -   Added detailed descriptions for all the existing playbook activities.
-   **Version 1.0.6 - May 2025**

    Bug fixes with respect to workflow to flow.

-   **Version 1.0.3 - November 2023**
    -   Changed:
        -   Supports the Wrapper Process Generator on the Vancouver version.
        -   Added a column in the results table for backup flows.
        -   Added translations for all existing playbooks.
-   **Version 1.0.0 - February 2023**

    This application contains the artifacts related to Process Automation Designer \(PAD\) that will be used across Security Case Management Workspaces.
