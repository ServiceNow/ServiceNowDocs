# HR Service Delivery Portal UI Components release notes

Version history for the HR Service Delivery Portal UI Components integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.5 - May 2024**
    -   Fixed table field ACL misconfiguration.
    -   Fixed Localization in the Time-off balance widget.
-   **Version 1.0.4 - November 2022**

    Minor optimization and refactoring

-   **Version 1.0.3 - March 2022**

    Fixed: Accessibility issues found in some use cases.

-   **Version 1.0.2 - February 2022**
    -   Fixed:
        -   Moved down to record producer for Direct deposit widget
        -   Adjusted Chart size
        -   Change in screen reader read item as button/link when the item is not interactive
-   **Version 1.0.1 - November 2021**

    This application provides wigdets that can be connected to any HCM to easily display information around direct deposit, holiday calender via ESC
