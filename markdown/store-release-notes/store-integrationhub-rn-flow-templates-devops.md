# Flow Templates for DevOps release notes

Version history for the Flow Templates for DevOps on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.3 - April 2023**

    Fixed: Improve installation performance of spoke.

-   **Version 1.1.2 - January 2023**

    Fixed: Dependencies.

-   **Version 1.1.1 - September 2022**

    Fixed: Dependencies.

-   **Version 1.1.0 - September 2021**

    New: This release includes two new templates, both for GitHub repository management.

-   **Version 1.0.0 - July 2021**
    -   New:
        -   Easily automate DevOps in your custom applications with the Flow Templates for DevOps application. Rather than creating a new flow from scratch, users can choose the relevant template in App Engine Studio and follow the guided screens to create a new flow. These templates leverage IntegrationHub spokes to execute Jenkins and Ansible jobs and post updates to Jira based on record updates in ServiceNow.
        -   Setup screens guide you and indicate what information is required. Flows generated from templates provide additional explanations and can be customized as needed. The templates all have pre-defined flow triggers, but you can modify the flows and add additional conditions to control when they are triggered.
