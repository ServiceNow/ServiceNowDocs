# Lucidchart Diagramming Spoke release notes

Version history for the Security Operations Lucidchart Diagramming Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.1 - January 2024**
    -   New: Added Get Folders and Create Diagram subflow which supports individual user credentials for authentication.
    -   Fixed: Fixed authentication expiration issue.
-   **Version 1.1.0 - August 2023**
    -   New: Added a Get Folder action.
    -   Changed: Modified the Create Diagram action input parameter to create the chart in a specified folder.
-   **Version 1.0.0 - August 2022**

    Provides an action to create Lucidchart diagrams from a ServiceNow instance.
