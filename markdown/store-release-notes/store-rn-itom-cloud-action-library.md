# Cloud Action Library release notes

Version history for the Cloud Action Library application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - June 2026**

    Fixed: Manage access through predefined query access controls for all Cloud Action Library tables

-   **Version 1.4.0 - November 2024**

    Fixed: Applied security and policy updates.

-   **Version 1.3.9 - May 2023**
    -   Changed: The filter only shows AWS accounts when setting up assume role config
    -   Fixed: Issue in AWS VM collector to support scenario of VM without tags
-   **Version 1.2.1 - May 2022**

    Fixes and quality improvements

-   **Version 1.0.7 - March 2022**

    Changed: Plugin dependencies have been changed to include individual IntegrationHub plugins.

-   **Version 1.0.5 - February 2022**

    Cloud Action library provides ready-made Automation Engine actions and sub-flows used with store apps like the Cloud Configuration Governance \(CCG\) store app. Also, you can use the CAL actions in the in your wokflows to interact with the cloud resource. The CAL actions enable you to list the cloud resources, read resource details, turn-off the cloud VM, set encryption level, enable advanced monitoring, and more.
