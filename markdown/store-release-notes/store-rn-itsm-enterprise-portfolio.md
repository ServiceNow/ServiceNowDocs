# Enterprise Portfolio release notes

Version history for the ITSM Enterprise Portfolio application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - December 2025**

    Fixed: Security fixes

-   **Version 1.2.1 - February 2025**

    New: Added active flags to the enterprise taxonomy and node tables. \*\*note these aren't used / have no effect yet, and will be in an upcoming release.

-   **Version 1.1.5 - November 2024**

    Fixed: Updated compatibility for newer family versions.

-   **Version 1.1.4 - May 2024**

    New: Prepared the app for translation in a future release \(i18n\).

-   **Version 1.0.2 - August 2023**

    The Enterprise Portfolio plugin allows users to create categorized, hierarchical structures of anything \(e.g. business applications, application services, software, users, demands, etc.\) in your organization.
