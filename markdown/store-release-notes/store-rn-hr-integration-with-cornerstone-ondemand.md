# HR Service Delivery Integration with Cornerstone OnDemand release notes

Version history for the HR Service Delivery Integration with Cornerstone OnDemand on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - May 2026**
    -   New: Not provided for this release.
    -   Changed: Applied non-Glide Cobalt Raven ACL pattern across spoke actions to align with the latest platform security guidance.
-   **Version 1.2.0 - December 2025**

    Performance improvements.

-   **Version 1.1.6 - May 2025**

    Minor bug fixes.

-   **Version 1.1.5 - June 2024**
    -   Assign learning tasks to employees in Cornerstone OnDemand
    -   Sync users, learning courses, and transcripts from Cornerstone and present them as employee to-do's in the employee portal
    -   Seamlessly navigate, access, and complete learning courses between Cornerstone OnDemand and ServiceNow
-   **Version 1.1.4 - February 2024**

    Minor fixes.

-   **Version 1.1.3 - November 2022**

    Minor optimization and refactoring

-   **Version 1.1.2 - April 2022**

    Fixed: Fix for error encountered when trying to open "Trigger CSOD Sync" from Flow Designer

-   **Version 1.1.1 - November 2021**

    Fixed: Fix for spoke version mismatch for the integration

-   **Version 1.1.0 - May 2021**

    Changed: No functional change. Updates to incorporate changes added in HR Service Delivery Integrations Framework \(technical changes\)

-   **Version 1.0.5 - December 2020**

    Fixed: The time window in which modified records are fetched is made uniform for all the services.

-   **Version 1.0.3 - October 2020**
    -   The integration with Cornerstone OnDemand pulls learning tasks and to-dos from Cornerstone OnDemand into ServiceNow HR Service Delivery, so that employees can view all of their learning tasks in a single place, in the Employee Service Center.
    -   These learning tasks can be accessed from anywhere, on any device. Employees can find and complete their learning courses on-time, and HR and learning teams spend less time managing compliance and updating multiple systems.
