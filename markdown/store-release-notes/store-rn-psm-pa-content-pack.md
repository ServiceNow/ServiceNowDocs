# Performance Analytics for Sourcing and Procurement Operations release notes

Version history for the Performance Analytics on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.16 - June 2026**

    Defect fixes.

-   **Version 3.0.14 - March 2026**

    Changed minor defect fixes: Fixes an issue where pa\_filter records could be created without an associated breakdown source, causing diagnostic errors in Performance Analytics.

-   **Version 3.0.9 - August 2024**
    -   Removed:
        -   The SPO CoreUI classic dashboards are now deprecated and no longer supported or available for new activation. The following Next Experience dashboards provide the latest experience for this functionality, supporting a modern and intuitive analytics experience:
            -   Procurement Strategy &amp; Ops Dashboard
            -   Procurement Buyer Dashboard
            -   Procurement Team Performance Dashboard
            -   Procurement Case Management Dashboard
-   **Version 3.0.6 - May 2024**

    Minor fixes.

-   **Version 3.0.0 - August 2023**

    New: Provided four new dashboards in the seismic platform that will enable customers to leverage the new default platform capabilities like configurability and better performance. It provides an intuitive and modern analytics experience to customers that will add value with unified capabilities for all Servicenow data.​ The seismic platform enables customers to edit, copy, and duplicate the delivered dashboards, and even edit at the widget level. They can also share the dashboards with groups or individual users. These four SPO dashboards include Procurement Buyer Dashboard, Procurement Strategy &amp; Ops Dashboard, Procurement Team Performance Dashboard, and Procurement Case Management Dashboard. Note: Existing customers are not impacted as the old classic dashboards are still available at the same navigation path. The new dashboards are added in the performance analytics workspace pages.

-   **Version 2.0.4 - May 2023**

    Changed: Renamed this application to Performance Analytics for Sourcing and Procurement Operations. This application was formerly named Performance Analytics Content Pack for Procurement Service Management.

-   **Version 2.0.2 - February 2023**

    Minor fixes

-   **Version 2.0.1 - August 2022**
    -   New
        -   Introduced a new dashboard to show Service Level Agreements \(SLAs\) for the procurement organization, with visualizations for:
            -   Cycle times for purchase requistions, sourcing requests, negotiations, and cases
            -   SLA breach times versus allotted times
            -   Overall SLA performance of the procurement team
