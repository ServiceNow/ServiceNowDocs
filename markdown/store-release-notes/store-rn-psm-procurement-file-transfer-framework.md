# Procurement File Transfer Framework release notes

Version history for the Procurement File Transfer Framework application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.2 - May 2023**
    -   Minor fixes.
    -   Extends the Procurement Integration Framework with SFTP and FTP functionality, designed for integrating with systems that do not support REST or SOAP-based integration methods.
