# Performance Analytics content pack for Agile 2.0 release notes

Version history for the IT Business Management Performance Analytics content pack for Agile 2.0 on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.6 - December 2025**

    Performance improvements.

-   **Version 1.4.5 - June 2025**

    Fixed: WCAG Color contrast related issues for Agile dashboards and issues related to UI themes.

-   **Version 1.4.4 - February 2025**

    Fixed: Fixed a minor internationalisation issue.

-   **Version 1.4.3 - August 2024**
    -   Changed: The following dashboards are upgraded to the Next Experience UI:
        -   Agile 2.0 Epic Dashboard
        -   Agile 2.0 Sprint Dashboard
        -   Agile 2.0 Prior Sprint Dashboard
        -   Agile 2.0 Release Dashboard
        -   Agile 2.0 Prior Release Dashboard
        -   Agile 2.0 Team Dashboard
    -   Removed: Agile 2.0 Team Dashboard no longer has the Velocity by Type report
-   **Version 1.3.2 - September 2023**
    -   New: Support for dark theme
    -   Fixed: Issues with Time elapsed widget
-   **Version 1.3.1 - May 2023**

    Updates to the localization support.

-   **Version 1.3.0 - July 2021**
    -   New:
        -   Support for custom story states in area charts of all Agile 2.0 dashboards
        -   Added read-only access to all Agile 2.0 dashboards using sn\_agile\_read role
-   **Version 1.2.2 - January 2021**
    -   Bug fixes for the following issues:
        -   Agile 2.0 Sprint Dashboard: Sprint Cumulative Flow Diagram does not properly display stories
        -   Agile 2.0 Team Dashboard: Team Members tab does not filter on group members
-   **Version 1.2.1 - August 2020**
    -   New: Cycle Time for stories in Team, Release, and Epic dashboards
    -   Access the Story Cycle Time report by installing these dashboards from Solution Library once you upgrade the application. For more details, see Using Solution Library for Agile 2.0 dashboards in the documentation.
-   **Version 1.1.0 - June 2020**
    -   New:
        -   Added the following reports:
            -   Cumulative Flow Diagram for Epics, Sprints, and Releases
            -   Sprint Variance for teams
-   **Version 1.0.2 - April 2020**

    Fixed: Bug fixes.

-   **Version 1.0.0 - August 2019**
    -   Burndown charts for Sprints and Releases with forecasts on all series and drill-downs to scope change details
    -   Burnup charts for Releases and Scrum Epics with with forecasts on all series and drill-downs to scope change details
    -   Velocity charts for Teams with drill-down to details
