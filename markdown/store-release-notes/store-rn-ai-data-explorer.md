# AI Data Explorer release notes

Version history for the ServiceNow® AI Data Explorer application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 5.0.8 - June 2026**
    -   Improved keyboard navigation in explorations:
        -   Navigate explorations fully via keyboard, including text and action buttons.
        -   Move up/down buttons have been added under the drag-and-drop icon.
    -   The data visualization sparkle icon is no longer shown for data sources in a non-supported secure scope.
-   **Version 4.1.5 - May 2026**
    -   Added support for exploration of data from secure scopes
    -   Enhanced query generation capabilities such as:
        -   Support for global OR conditions
        -   Improved filtering on class field of CMDB tables \(support for InstanceOf operator\)
        -   Added ability to filter by sys\_created\_by/sys\_updated\_by fields
        -   Semantic segment tuning improvements \(ability to disable specific segment types\)
    -   Administration enhancements:
        -   Added module for AI Data Explorer logs \(includes error and performance logs\)
        -   Added module for workspace configuration for AI Data Explorer
-   **Version 4.0.5 - March 2026**
    -   New: Action recommendations skill added - allows you to instantly generate action recommendation base on exploration goal and content.
    -   Changed:
        -   All AI Data explorer skills are enabled by default upon plugin activation. Customer can deactivate skills if required.
        -   Minor usability and UI improvements
-   **Version 3.0.14 - March 2026**
    -   Added Compatibility with Australia release
        -   AI Data Explorer application to allow users to ask questions in natural language and see them turned into documented insights and visualizations.
        -   Easily edit or add your own insights and organize findings
        -   Suggested and open prompt followup questions
        -   Allow users to ask questions directly from the context of the application: any data visualization \(Next Experience\) or list \(Next Experience  or CoreUI\)
        -   Allow users to share explorations with other users with view and edit rights
        -   Real time collaboration
        -   Email notifications when a user is tagged or has an exploration shared with them
        -   Summarize explorations with help of AI
        -   Refine texts in exploration with the help of AI \(change tone, shorten, elaborate etc\)
        -   Advanced analysis options with record level analysis and extended analysis mode. New option bring you beyond just simple answers based in data visualization, and bringing to the user insights that are hidden in data to help make analysis more efficient.
        -   Support more data sources:
            -   Workflow data fabric: allows you analyse data that are not stored in ServiceNow instance
            -   Database views: allow answer more complex questions that requires join of the tables
        -   Allow save charts to a dashboard
        -   Duplication of the content and explorations
        -   Extended support for multi-table queries: related list conditions and subqueries
-   **Version 1.12.6 - January 2026**

    AI Data Explorer is an AI-powered exploration tool that helps business users seamlessly explore data, document findings, and collaborate on insights to drive data-informed decisions more effectively.
