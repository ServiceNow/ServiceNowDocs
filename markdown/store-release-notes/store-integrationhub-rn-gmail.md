# Gmail spoke release notes

Version history for the Integration Hub Gmail spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.3 - December 2025**

    Fixed - Action: Get Email Details

-   **Version 1.3.2 - September 2024**

    Fixed: security bug.

-   **Version 1.3.0 - May 2023**

    New: Authentication template for easier spoke setup.

-   **Version 1.2.1 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.1 - December 2019**

    Provides actions to automate email and label management in Google Gmail.
