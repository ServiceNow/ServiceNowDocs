# Supplier Lifecycle Operations Integration with SAP Ariba release notes

Version history for the Supplier Lifecycle Operations Integration with SAP Ariba application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.0 - June 2026**

    Changed: Updated the dependent Utility Action Spoke to version 1.3.1

-   **Version 1.0.3 - December 2025**

    Fixed: Added sn\_so as a hard dependency plugin for Supplier Lifecycle Operations Integration with SAP Ariba. This plugin now contains the Supplier Outbound table needed for SAP Ariba supplier job triggers, previously located in sn\_slm.

-   **Version 1.0.2 - November 2025**

    Fixed: Minor fixes

-   **Version 1.0.0 - August 2025**

    This application enables bidirectional integration between Supplier Lifecycle Operations \(SLO\) and SAP Ariba allowing you to create and update supplier data.
