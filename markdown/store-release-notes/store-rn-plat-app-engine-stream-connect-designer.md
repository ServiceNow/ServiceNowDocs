# Stream Connect Designer release notes

Version history for the Stream Connect Designer application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.0.1 - March 2026**

    This version adds the Australia release features, such as Topic Alias, and supports both Hermes and Direct Kafka.

-   **Version 4.0.0 - August 2025**

    The Stream Connect Dashboard provides visibility into activity by Topic, Producer, and Consumer, as well as data usage over a selected time period. This release introduces alert monitoring, enabling customers to identify and take action on integration issues more effectively.

-   **Version 3.0.1 - May 2025**

    The Stream Connect Dashboard enables customers to view activity per Topic, Producer, and Consumer over a selected time period. This version adds the capability to track data usage monthly and provides a Guided Setup to simplify the setup.

-   **Version 2.0.1 - February 2025**

    The Stream Connect Dashboard enables customers to view activity per Topic, Producer, and Consumer over a selected time period.

-   **Version 1.0.4 - September 2024**

    The Stream Connect Dashboard provides an overview of Stream Connect for Apache Kafka metrics for topics and consumers. View messages and bytes for the desired time period.
