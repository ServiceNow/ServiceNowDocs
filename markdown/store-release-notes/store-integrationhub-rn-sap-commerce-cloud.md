# SAP Commerce Cloud spoke release notes

Version history for the Integration Hub SAP Commerce Cloud Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - September 2023**

    Added authentication template.

-   **Version 1.0.1 - April 2023**

    Improved installation performance.

-   **Version 1.0.1 - May 2021**
    -   This is a lightweight spoke with three actions only:
        -   Look up Orders
        -   Look up Order Details
        -   Look up Shopping Cart
