# Private cloud orchestration release notes

Version history for the ServiceNow® Private cloud orchestration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.1.0 - June 2026**

    Accessibility and Zoom related changes

-   **Version 1.0.1 - March 2026 \(Australia\)**

    Added translations for couple of messages, which were not handled in earlier release

-   **Version 1.0.0 - December 2025**

    This allows customers to install and utilize the Windriver Orchestrator in ServiceNow.
