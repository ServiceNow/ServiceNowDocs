# Field Service Management Virtual Conferencing Integration release notes

Version history for the Field Service Management Virtual Conferencing Integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 30.0.0 - March 2026**

    Changed: Updated access controls

-   **Version 2.2.1 - January 2025**

    Update application Java Version to 17.

-   **Version 2.2.0 - August 2023**

    Uptake for polaris dark theming within virtual conferencing workflows within the agent mobile application.

-   **Version 2.1.0 - February 2023**

    New: Uptake for Next Experience theming within virtual conferencing workflows within the agent mobile application

-   **Version 2.0.1 - March 2021**
    -   New:
        -   Support for Microsoft Teams as a conferencing tool within the ServiceNow Mobile Agent application for use with Field Service Management
        -   Microsoft Teams conferencing tool is compatible from Paris release onwards 'Add Participants' as a footer action in conference details page
        -   Add peer agents, users and groups to conference
    -   Changed:
        -   Conferences moved as a related list of Work Order Task
        -   Invite customer is now available as one of the parameters in 'Add Participants' and 'Make a Call' screens
    -   Removed:
        -   'Invite Customer' footer action from Conference details page
-   **Version 1.2.0 - November 2020**

    New: View conference recordings initiated for work order tasks

-   **Version 1.1.0 - October 2020**
    -   Any experience that enables resolving customer issues faster is of utmost importance in field service. The ServiceNow® Field Service Management Virtual Conferencing Integration application enables field service technicians and mobile workers to engage better with customers by initiating Zoom meetings from ServiceNow's native mobile app. Technicians can conveniently use video and screen sharing options in Zoom and optionally record the meeting.
    -   Resolving issues over Zoom as a visual engagement channel can result in smarter and better customer experiences.
