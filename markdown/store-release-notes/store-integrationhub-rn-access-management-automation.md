# Access Management Automation release notes

Version history for the Access Management Automation on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - December 2023**
    -   Changed:
        -   Updating all flows in the app with the latest actions from Microsoft Entra ID Spoke v4.0 \(formerly known as Microsoft Azure Active Directory Spoke\).
        -   Updating all labels and descriptions that refer to Microsoft Azure Active Directory to Microsoft Entra ID.
-   **Version 2.0.1 - September 2023**

    Fixed: Adds missing roles to service portal widgets.

-   **Version 2.0.0 - May 2023**

    Changed: Fulfilment flows by removing actions from Microsoft AD Spoke for IntegrationHub and replaces them with actions from Microsoft Active Directory v2 Spoke.

-   **Version 1.0.2 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.1 - September 2021**

    Patch release of Access Management Automation.

-   **Version 1.0.0 - August 2021**
    -   New:
        -   Automate service catalog requests for external application access in minutes with the Access Management Automation application. Choose from catalog items covering common use cases integrating with Active Directory, Azure Active Directory, and Okta and quickly move from request to fulfillment.
        -   Each catalog item comes linked to a flow that includes an approval process and automated fulfilment and follows request fulfilment best practices. The flows reassign the task and create an incident or event \(or execute a custom subflow\) in the case of an automation error. Follow configuration steps to select approvers and other assignees and get up and running in minutes.
