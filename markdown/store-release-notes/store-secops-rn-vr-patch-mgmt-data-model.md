# Patch Management Data Model release notes

Version history for the Patch Management Data Model application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.2 - May 2025**

    Patch Management Data Model is a dependent plugin for Vulnerability Response Patch Orchestration.It is a standalone, free plugin containing only the data model previously used in the VR Patch Orchestration application. This will decouple the Patch Orchestration data model from VR, ensuring it remains independent. The plugin will include the necessary scripts to import data into the tables.
