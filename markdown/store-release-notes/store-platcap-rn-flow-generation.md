# Flow Generation release notes

Version history for the Flow Generation application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 24.0.4 - November 2023**

    Our objective with the Flow Generation Now Assist skill is to help authors by providing a soft start instead of a blank canvas for flow creation. This should increase the speed of flow authoring and empower users to build flow automation regardless of technical knowledge.
