# Performance Analytics for Configuration Compliance release notes

Version history for the Vulnerability Response Performance Analytics for Configuration Compliance application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.2 - December 2025**

    Changed: Updated dashboard table references to align with the standardized Unified Security Exposure Management \(USEM\) data model and modularized feature sets.

-   **Version 1.5.1 - May 2025**

    Fixed: Added VR categories to the Configuration Compliance PA dashboard.

-   **Version 1.5.0 - November 2024**

    Changed: Age-based Performance Analytics \(PA\) reports have been updated to include support for configurable age calculation.

-   **Version 1.4.0 - August 2024**

    Fixed: Minor improvements implemented in the Configuration Compliance \[PA\] dashboard.

-   **Version 1.3.2 - June 2024**

    Fixed: Minor fixes for this release.

-   **Version 1.3.1 - May 2024**
    -   Changed: Changes made to MTTR and Scan Coverage widgets.
    -   Fixed: The Next-Generation dashboard will be shown when you access the Configuration Compliance Dashboard Overview module in the classic environment.
-   **Version 1.2.0 - November 2023**

    The Performance Analytics for Configuration Compliance dashboard is now available in the Next Experience UI from the Vulnerability Response workspaces.

-   **Version 1.1.0 - May 2023**

    Fixed: Updated the "Configuration Compliance Overview dashboard" to support the dark theme.

-   **Version 1.0.1 - November 2022**

    New: The Configuration Compliance PA Dashboard has been introduced in this release.
