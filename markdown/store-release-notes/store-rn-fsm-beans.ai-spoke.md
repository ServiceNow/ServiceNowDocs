# Beans.ai Spoke release notes

Version history for the ServiceNow® Beans.ai Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 29.0.7 - March 2026**
    -   Beans.ai Spoke connects Beans.ai travel provider capabilities to the ServiceNow platform through IntegrationHub, making travel intelligence available directly within FSM workflows.
    -   Beans.ai operates under a Bring Your Own License model. Customers with an existing Beans.ai license can activate this integration without additional licensing through ServiceNow.
