# Now Assist for Impact release notes

Version history for the ServiceNow® Now Assist for Impact application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.5 - June 2026**
    -   Code Fix Agent: Enhanced Error Logging
    -   When the Code Fix Agent runs into a problem, you will now see a plain-language explanation of what went wrong and what to do next — right on the remediation record. Select the links navigate directly to the relevant error record, so there's no need to hunt through system logs.
-   **Version 3.1.3 - April 2026**
    -   Minor Fixes
        -   Resolved an issue where Next Steps were not consistently rendering for users.
        -   Fixed an issue where the Now Assist button was incorrectly displayed for Now Assist for Impact application scripts and files.
-   **Version 3.0.3 - March 2026**
    -   Resolve Scan Engine Findings with Now Assist\(for Developers\)
        -   Automatically generate AI-powered code fixes for platform health findings at scale, with side-by-side code comparisons and conversational interfaces to evaluate changes before applying.
        -   Streamlined review process to assess AI-generated changes before they are applied.
        -   Developers can now eliminate technical debt in minutes, not hours—keeping instances secure, fast, and upgrade-ready.
-   **Version 2.0.18 - December 2025**
    -   ServiceNow Impact is a customer success product powered by AI and human expertise that helps customers maximise the power of the AI Platform for business transformation. Impact provides customers with the support and expertise they need to get to value fast. Below are the AI features:
        -   Outcome status summarisation
        -   Consumption Report summary
        -   Code Fix AI Agent
