# Cloud Provisioning and Governance: IBM Cloud Connector release notes

Version history for the ServiceNow® IT Operations Management Cloud Provisioning and Governance: IBM Cloud Connector on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

**Note:** In Paris, this application was renamed from Cloud Management: IBM Cloud Connector to Cloud Provisioning and Governance: IBM Cloud Connector.

## Version history

-   **Version 1.0.3 - October 2020**
    -   New: This application is now compatible with the Paris release.
    -   Changed: This application is renamed to Cloud Provisioning and Governance: IBM Cloud Connector \(previously known as Cloud Management: IBM Cloud Connector\).
-   **Version 1.0.1 - June 2019**
    -   Support for IBM Cloud
        -   Ability to create ServiceNow Catalog offerings from Terraform configuration template
        -   Ability to provisioning IaaS and PaaS resources in IBM Cloud with governance and security policies applied along the provisioning path
        -   Integration with ITSM processes such as Approval and Change control
        -   Day-2 operations \(lifecycle operations\) on the provisioned and discovered resources in the cloud
