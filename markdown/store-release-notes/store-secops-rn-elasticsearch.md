# Elasticsearch integration for Security Operations release notes

Version history for the Elasticsearch integration for Security Operations on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.3.4 - November 2024**

    Changed: Migration of Workflows to Flow Designer for Elasticsearch integration.

-   **Version 10.3.3 - August 2023**

    Fixed: The Elasticsearch Integration for Security Operations plugin available as a part of the base system is not working for Sighting Search.

-   **Version 10.3.2 - February 2023**

    New: Supports Type changes to display on the Security Incident Response workspace.

-   **Version 10.3.0 - June 2020**

    New: Implementation Flow to support Integration capability framework v2.0.

-   **Version 9.0.1 - November 2019**
    -   New:
        -   Support for custom index search. Support for custom timestamp format.
        -   Support for custom timestamp format.
    -   Changed:
        -   Elastic Search configuration has two new fields: Elasticsearch Index, Date range field.
        -   Link URL field renamed to Kibana Base URL.
