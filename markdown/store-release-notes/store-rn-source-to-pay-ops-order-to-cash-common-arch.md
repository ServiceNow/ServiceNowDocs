# Order to cash common architecture release notes

Version history for the ServiceNow® Order to cash common architecture application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.5.0 - March 2026**
    -   The AR invoice \(sn\_otc\_invoice\) table stores the invoice data.
    -   The AR invoice line \(sn\_otc\_invoice\_line\) table stores the invoice line data.
    -   Access control lists for both tables
-   **Version 1.4.0 - December 2025**

    Order-to-cash common architecture provides backend invoice tables.
