# Change Management for Field Service release notes

Version history for the Change Management for Field Service application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.4 - July 2025**

    New: Fixed work notes translation issue for multiple languages

-   **Version 1.1.2 - January 2025**

    Update application Java Version to 17.

-   **Version 1.1.1 - August 2023**

    New: Uptake for Polaris dark theming within agent mobile application.

-   **Version 1.0.0 - September 2021**

    The ServiceNow® Change Management for Field Service plugin provides seamless integration between Change Management and Field Service Management applications. Use this capability to create work orders from a change request. Track the progress of work orders directly from their respective parent change requests, whenever the work order state changes or a work note is added to the work order.
