# Microsoft Azure DevOps Integrations Common release notes

Version history for the IT Business Management Microsoft Azure DevOps Integrations Common on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - June 2021**

    Changed: Support for multi-select field and other bug fixes.

-   **Version 1.4.2 - April 2021**

    Fixed: bug fixes.

-   **Version 1.4.1 - March 2021**

    Fixed: Enables support for SAFe Sprint synchronization with Azure DevOps.

-   **Version 1.3.0 - July 2020**

    Fixed: Performance bug fixes.

-   **Version 1.2.1 - May 2020**

    Fixed: Performance bug fixes.

-   **Version 1.1.0 - April 2020**

    New: Support for Iterations.

-   **Version 1.0.0 - February 2020**

    The app has common code that supports Microsoft Azure DevOps Integration for Agile Development.
