# Supplier Lifecycle Operations Integration with Oracle Financial Cloud release notes

Version history for the ServiceNow® Supplier Lifecycle Operations Integration with Oracle Financial Cloud application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.1 - March 2026**

    This application enables bidirectional integration between Supplier Lifecycle Operations \(SLO\) and Oracle Financial Cloud, allowing you to create and update supplier data.
