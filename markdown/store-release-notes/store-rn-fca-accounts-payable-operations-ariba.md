# Accounts Payable Operations integration with Ariba release notes

Version history for the Accounts Payable Operations integration with Ariba application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 0.2.0 - December 2025**
    -   Changed: Updated the source configuration to push invoices to SAP ECC.
    -   Fixed: Removed hard-coded timeout values to improve configurability and system flexibility.
-   **Version 0.1.3 - November 2025**

    This application provides you with the ability to perform below operations in Sourcing and Procurement Operations to SAP Ariba: Invoice Ingestion.

-   **Version 0.1.1 - August 2025**

    Minor Fixes: Updated the Script Include of the APO Ariba app, update all error messages to use gs.getMessage for translation purpose.

-   **Version 0.1.0 - January 2025**

    This application provides you with the ability to perform below operations in Sourcing and Procurement Operations to SAP Ariba: Invoice Ingestion.
