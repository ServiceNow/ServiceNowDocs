# Calendly spoke release notes

Version history for the Integration Hub Calendly spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.1 - March 2026**

    Fixed: Look Up Event action returned an emptyEvent Object in the output despite the event details being present in the API response body

-   **Version 1.3.0 - February 2026**

    Removed: Deprecated - Delete Invitee Data.

-   **Version 1.2.0 - August 2023**

    New: Data Compliance - Delete Invitee Data action

-   **Version 1.1.1 - June 2023**
    -   New: Added actions: Cancel Event, Look up User, Invite User to Organization, and Look up Event.
    -   Changed:
        -   Spoke Icon.
        -   Upgraded actions: Look up Organization, Remove User from Organization with latest design pattern.
    -   Removed: Deprecated actions: Look up Organization, Remove User from Organization.
-   **Version 1.1.0 - May 2023**
    -   New:
        -   Added 4 actions
            -   Cancel Event
            -   Invite User to Organization
            -   Look up User
            -   Look up Event
        -   Added 2 actions in liew of the deprecated ones
            -   Look up Organization
            -   Remove User from Organization
    -   Changed:
        -   Calendly spoke icon
        -   Updated actions
            -   Look up Organization Members Stream
            -   Look up Scheduled Events Stream
    -   Removed: Deprecated 2 actions
        -   Look up Organization
        -   Remove User from Organization
-   **Version 1.0.2 - December 2022**

    Fixed: Calendly icon view in action picker.

-   **Version 1.0.0 - May 2021**

    The Calendly spoke provides actions to view and analyze meaningful usage data for Calendly software subscriptions. Analyze usage for a Calendly account to determine which licenses are stale so that you can reclaim these licenses and realize your potential savings.
