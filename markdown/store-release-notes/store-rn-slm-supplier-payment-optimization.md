# Supplier Payment Optimization release notes

Version history for the Supplier Payment Optimization application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.0.0 - June 2026 \(Australia\)**
    -   Fixed:
        -   Improved access controls for better security.
        -   Fixed translation of user interface text to ensure accurate display of product capabilities across multiple languages, resolving reported issues.
-   **Version 1.4.1 - June 2026**
    -   Fixed:
        -   Improved access controls for better security.
        -   Fixed translation issues.
-   **Version 3.0.0 - June 2026 \(Zurich\)**
    -   Fixed:
        -   Improved access controls for better security.
        -   Fixed translation of user interface text to ensure accurate display of product capabilities across multiple languages, resolving reported issues.
-   **Version 2.0.1 - March 2026 \(Zurich+\)**

    Fixed: SLO email templates updated with Coral theme.

-   **Version 1.1.1 - March 2026 \(Yokohama\)**
    -   Changed: Updated email template layout to improve visual alignment.
    -   Fixed: Fixed View Supplier Propensity Score screen layout.
-   **Version 2.0.0 - December 2025**
    -   New:
        -   New case type Credit card enablement added, that allows updating the payment method to credit card and capturing the terminal ID
        -   Enhanced payment optimization dashboard
        -   Improved banking form
-   **Version 1.1.0 - December 2025 \(Yokohama\)**
    -   New:
        -   New case type Credit card enablement added that allows updating the payment method to credit card and capturing the terminal ID
        -   Enhanced payment optimization dashboard
        -   Improved banking form
-   **Version 1.0.0 - August 2025**

    Provide insights on the potential savings and rebates that can be achieved by paying suppliers using a credit card.
