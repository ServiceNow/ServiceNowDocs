# Content Governance release notes

Version history for the Content Governance application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 31.4.0 - June 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 31.3.0 - May 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 31.2.5 - March 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 31.2.0 - December 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 31.1.5 - August 2025**

    Fixed: Reduced load time for Content Operations Dashboard calendar tab.

-   **Version 31.1.2 - May 2025**
    -   The following bugs are fixed:
        -   Filter the Publishing calendar in the Content Operations Dashboard by both pages and widgets simultaneously.
        -   Capability to select the start and end dates in different calendar years in the Content Request form.
        -   Get topic level security in My Favorites widget and Conversational LLM support to resolve tasks and approvals in Employee Center.
        -   Get new Employee Communications capabilities with a Workplace Service Delivery integration, a calendar widget view, RSVP/Waitlisting tools, events notifications and more; Get improvements on experience feedback and feedback configurations; leverage notifications in news content with Employee Center Pro.
-   **Version 31.1.1 - February 2025**

    Fixed bugs related to the Content Operations Dashboard: Calendar now opens the list of scheduled content items Calendar side panel now correctly displays content item names.

-   **Version 31.0.6 - November 2024**

    Fixed security bug.

-   **Version 31.0.4 - August 2024**
    -   Introduced content operations dashboard to provide centralized oversight across the content lifecycle, highlighting requests and active content publishing.
    -   Introduced content publishing calendar for strategic planning and scheduling of communications activities.
    -   Introduced content ownership matrix for clearer visibility into ownership across content items, audiences, topics, and pages within the content platform.
-   **Version 29.0.1 - August 2023**

    The release primarily focuses on addressing minor issues and bug fixes.

-   **Version 28.0.0 - February 2023**

    Changed: Updated the workflow to allow content authors or content requests to include additional content approvers before publishing the content.

-   **Version 27.0.1 - November 2022**
    -   Changed
        -   Updated the content creation and approval flow for an improved authoring experience and interface with Content Library.
        -   Enables content creation from the content request as well as sending the content for review within the content library associated with a governance request.
-   **Version 26.0.2 - August 2022**

    Streamline the process of business stakeholders requesting communications and employee communication managers authoring content and routing it back to the business stakeholders for review and approval.
