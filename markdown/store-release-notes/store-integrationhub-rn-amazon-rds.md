# Amazon RDS spoke release notes

Version history for the Integration Hub Amazon RDS spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.5 - June 2025**

    Fixed - Action: List DB Certificates.

-   **Version 1.0.4 - September 2023**

    Fixed: The licensing requirement.

-   **Version 1.0.3 - September 2022**

    Patch release of Amazon RDS Spoke for IntegrationHub. This version includes a fix to improve installation performance of spoke.

-   **Version 1.0.2 - October 2021**

    Fixed: This version includes fix for Random Generate Password Action.

-   **Version 1.0.1 - December 2020**

    Patch release of the Amazon RDS spoke for IntegrationHub.

-   **Version 1.0.0 - April 2020**

    Provides actions to automate the management of database instances, database groups, database snapshots and events in Amazon RDS.
