# Microsoft Outlook Add-In for Legal Service Delivery release notes

Version history for the Microsoft Outlook Add-In for Legal Service Delivery application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - December 2025**

    Fixed: The “Required information” section in the Outlook add-in now displays fields with an improved background, making them easy to read.

-   **Version 1.4.1 - August 2025**

    New: Coral is now the default theme for new portal, web, and mobile experiences with Next Experience or Core UI enabled. This theme provides a fresh look and feel, featuring brand-neutral illustrations to improve your user experience. A dark theme option is available for web and mobile experiences.

-   **Version 1.3.0 - May 2025**
    -   Fixed:
        -   The attach file window does not close automatically after attaching file to a legal request from the Outlook Add-in. This issue has been fixed.
        -   Localization issues
-   **Version 1.2.0 - February 2025**
    -   Fixed:
        -   The "Microsoft Outlook Add-In for Legal Service Delivery" manifest failed to upload to Microsoft Outlook. This has been fixed
        -   Performance issue
        -   UI issue
-   **Version 1.1.3 - December 2024**

    Fixed: Security issues.

-   **Version 1.1.0 - November 2024**
    -   Fixed:
        -   Security fixes
        -   Localization fixes
-   **Version 1.0.0 - August 2024**

    The Microsoft Outlook Add-in for Legal Service Delivery brings capabilities such as of adding attachments to existing legal requests and create requests within the Microsoft Outlook application. This feature allows your legal fulfillers to quickly add attachments or employee requests received over emails.
