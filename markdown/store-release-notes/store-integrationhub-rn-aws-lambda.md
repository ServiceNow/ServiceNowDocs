# AWS Lambda spoke release notes

Version history for the Integration Hub AWS Lambda spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.3 - September 2023**

    Fixed: The license requirements.

-   **Version 1.1.2 - September 2022**

    Fixed: Patch version of AWS Lambda Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.1.1 - October 2021**

    Fixed: This version includes a fix for the pagination logic issue within List Functions action.

-   **Version 1.1.0 - August 2020**

    New: This version includes the Add Permission action and supports the Paris release.

-   **Version 1.0.1 - May 2020**

    Provides actions to automate the management of Functions and Aliases for AWS Lambda.
