# Amazon Elastic Container Service spoke release notes

Version history for the Integration Hub Amazon Elastic Container Service spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.2 - September 2023**

    Fixed: The license requirements.

-   **Version 1.0.1 - July 2021**

    Patch release of the Amazon Elastic Container Service spoke for IntegrationHub. This version includes a fix for non-responsive complex objects on Paris release, and adds Rome compatibility.

-   **Version 1.0.0 - October 2020**

    Provides actions to automate the management of instances, clusters, services, and tasks in Amazon Elastic Container Service.
