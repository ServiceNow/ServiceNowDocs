# ServiceNow Remote Instance spoke release notes

Version history for the Integration Hub ServiceNow Remote Instance spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.0 - June 2026**

    Fixed: Streamline ACLs.

-   **Version 2.2.9 - November 2025**

    Patch update with bug fixes.

-   **Version 2.2.7 - December 2024**

    Fixed: Create Account.

-   **Version 2.2.6 - October 2024**

    Fixed: The OOB Look up Remote Tables action calls the data stream action using the background script.

-   **Version 2.2.5 - June 2024**

    Fixed: Dependency updates for compatibility.

-   **Version 2.2.4 - September 2023**

    Patch version of ServiceNow Remote Instance Spoke for Integration Hub. This version includes a fix to improve installation performance of spoke.

-   **Version 2.2.2 - September 2022**

    Patch version of ServiceNow Remote Instance Spoke for IntegrationHub. This version includes a fix to improve installation performance of spoke.

-   **Version 2.2.1 - May 2022**

    New: Patch release of Service Remote Instance Spoke for Integration Hub. This release version contains fixes for actions that aren't publishable in a flow and updates to action descriptions. It also includes sample flows to assist with setup of attachment-related flows.

-   **Version 2.2.0 - October 2021**
    -   New:
        -   New Action: Look up Comments/Worknote.
        -   New Flow: Export Record and Comments.
    -   Fixed:
        -   Fix for not being able to export comments and work notes of incidents.
        -   Fix for increasing the length of input for table names.
        -   Fixes for issues with Export Record and Comments Flow, Look up Comments/Work notes action.
        -   Fix for Empty Error message at Import Set action.
-   **Version 2.1.2 - April 2021**

    Patch release of ServiceNow Remote Instance Spoke for IntegrationHub.

-   **Version 2.1.1 - February 2021**

    ServiceNow Remote Instance Spoke \(2.1.1\) is a patch release that has a fix related to updating the Description and Work notes in the remote instance with line breaks.

-   **Version 2.1.0 - October 2020**

    Patch release of the ServiceNow Remote Instance spoke for IntegrationHub.

-   **Version 1.0.0 - July 2020**

    Provides actions to handle tables and records in a remote ServiceNow instance.
