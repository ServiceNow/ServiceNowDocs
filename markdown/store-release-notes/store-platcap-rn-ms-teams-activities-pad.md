# MS Teams Activities for PAD release notes

Version history for the MS Teams Activities for PAD application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - January 2023**

    Fixed: No longer flushes system cache during installation and upgrade, which previously caused instances to temporarily slow down.

-   **Version 1.0.2 - December 2022**

    Fixed: No longer flushes system cache during installation and upgrade, which previously caused instances to temporarily slow down.

-   **Version 1.0.1 - August 2022**

    Send messages via Microsoft Teams with this out-of-the-box Process Automation Designer \(PAD\) content that leverages the Microsoft Teams spoke.
