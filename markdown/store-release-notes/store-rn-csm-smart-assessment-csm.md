# Smart Assessment for CSM release notes

Version history for the ServiceNow® Smart Assessment for CSM application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.0 - June 2026 \(Zurich\)**

    The Smart Assessment for CSM plugin enables Smart Assessment to be used in CSM as a horizontal plugin, it abstracts complex configuration steps to simplify the experience for business users, allowing intuitive creation, editing, and publishing of smart assessment templateswithin CSM/FSM configurable workspace.
