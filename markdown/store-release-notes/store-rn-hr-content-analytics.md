# Content Analytics release notes

Version history for the Content Analytics application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 31.0.0 - June 2026 \(Australia\)**

    Updated to support the latest version of the dependent apps.

-   **Version 30.4.1 - June 2026 \(Zurich\)**

    Updated to support the latest version of the dependent apps.

-   **Version 30.4.0 - June 2026 \(Australia\)**

    Updated to support the latest version of the dependent apps.

-   **Version 30.3.0 - December 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 30.2.10 - August 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 30.2.8 - May 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 30.2.7 - March 2025**

    Resolved the bug where the Content Analytics daily processing job was evaluating all non-campaign data, leading to performance issues.

-   **Version 30.2.6 - February 2025**
    -   Content Analytics provides analytical capabilities for Content Experiences \(formerly known as Content Automation or Campaigns\) and Content Publishing \(formerly known as Content Delivery\) based service portals. It is an out-of-the-box analytics solution that provides the following types of reports:
        -   Web portal analytics:Measure and report standard web traffic, unique users, visits, and top pages.
        -   Campaign engagement analytics: Measure and report campaign performance, top content items viewed, top content items clicked, and task completion.
