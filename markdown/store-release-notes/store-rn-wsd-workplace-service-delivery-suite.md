-   **Version 2.14.0 - November 2024**

    The release notes of the related plugins can be viewed on the respective plugin's release notes.

-   **Version 2.13.0 - August 2024**

    Removed: Upgrading Workplace Service Delivery Professional will no longer install or update Workplace Agent for mobile.

-   **Version 2.12.1 - May 2024**

    Changed: The application got renamed from Workplace Service Delivery Suite to Workplace Service Delivery Professional.

-   **Version 2.11.0 - February 2024**

    Removed: Upgrading Workplace Service Delivery Suite will no longer update Workplace Space Management and Workplace Move Management. Upgrading the apps must be done separately.

-   **Version 2.10.0 - November 2023**

    The release notes of the related plugins can be viewed on the respective plugin's release notes.

-   **Version 2.9.0 - August 2023**

    New: A new application, Workplace Connectors is introduced and is a generic framework using data from various workplace hardware/sensors \(such as badging systems, occupancy sensors etc.\) and can be brought into Workplace Service Delivery via respective spokes.

-   **Version 2.8.0 - May 2023**

    The release notes of the related plugins can be viewed on the respective plugin's release notes.

    The release notes of the related plugins can be viewed on the respective plugin's release notes.

-   **Version 2.5.0 - November 2022**

    New: A new application, Workplace Central is introduced to enable you to perform Scenario planning using stack plans. The parent application for Scenario planning is Workplace Space Management though it is hosted in the Workplace central application.

-   **Version 2.4.0 - August 2022**

    The release notes of the related plugins can be viewed on the respective plugin's release notes.

-   **Version 2.1.0 - December 2021**

    The release notes for the related plugins can be found in the respective plugins.

-   **Version 2.0.0- October 2021**

    New: A new application, Workplace Service Delivery for Mobile is introduced.

-   **Version 1.9.0 - September 2021**

    New: A new application Workplace Move Management has been introduced to manage move requests raised by employee to move to another location.

-   **Version 1.8.1 - July 2021**

    New: A new application, Workplace Reservations for Microsoft Outlook Add-in is introduced.

-   **Version 1.7.0 - June 2021**

    New: A new application, Workplace Space Management is introduced.

-   **Version 1.6.0 - May 2021**

    New: Synchronize registered visitors between Workplace Visitor Management and Safe Workplace application's Employee Health and Safety Status. The synchronization is bi-directional between both applications.

-   **Version 1.5.0 - March 2021**

    Fixed: minor fixes.

-   **Version 1.4.1 - January 2021**

    New: A new application, Workplace Calendar Synchronization, is added. Using the application, you can create or update reservations in the Workplace Service Delivery, and synchronize them to a calendar service \(Exchange Online\).

-   **Version 1.3.0 - December 2020**

    New: A new application, Workplace Calendar Synchronization, is added. Using the application, you can create or update reservations in the Workplace Service Delivery, and synchronize them to a calendar service \(Exchange Online\).

-   **Version 1.2.1 - November 2020**

    Changed: From Workplace Safety Management \(sn\_wsd\_core\) version 2.1, all updates, and new features are available only on the Workplace Service Portal. The workplace services provided by ServiceNow® are removed from the Service Portal from Workplace Safety Management \(sn\_wsd\_core\) version 2.1.

-   **Version 1.0.1 - October 2020**

    New: A new application Workplace Visitor Management to manage guests and off-site employees when they visit the organization
