# Pluralsight spoke release notes

Version history for the Pluralsight spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - May 2025**

    Bug fixes.

-   **Version 1.2.0 - March 2023**

    New: Authentication template for easier spoke setup.

-   **Version 1.1.3 - December 2022**

    Patch release of Pluralsight Spoke for IntegrationHub. This version includes localization fixes.

-   **Version 1.1.2 - September 2022**

    Patch version of Pluralsight Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.1.0 - May 2021**

    Minor release of the Pluralsight spoke for IntegrationHub. This version includes changes for Looking up courses \(Course Management\) and the addition of new action Learning Status

-   **Version 1.0.0 - January 2021**

    Provides actions to automate Pluralsight learning and skill management.
