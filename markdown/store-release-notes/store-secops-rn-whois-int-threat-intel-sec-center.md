# Threat Intelligence Security Center integration with Whois release notes

Version history for the Threat Intelligence Security Center integration with Whois application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.1.0 - June 2026**

    Changed: Implemented Query Range ACL.

-   **Version 5.0.4 - March 2025**

    Changed the application name from "WHOIS Integration for Threat Intelligence Security Center" to "Threat Intelligence Security Center integration with WHOIS".

-   **Version 5.0.3 - November 2024**

    Changed: The View Raw HTML response button should be present always. If there is no data in the raw result field, an error message is displayed on clicking the button.

-   **Version 5.0.2 - August 2024**
    -   New: Supports localization.
    -   Changed: Made changes to support the latest whois API.
-   **Version 5.0.0 - June 2024**

    The Threat Intelligence Security Center's WHOIS Integration empowers users to perform Whois lookups on domain names and URLs. This capability provides valuable context for observables, aiding users in making informed decisions regarding potential threats.
