# Performance Analytics - GRC: Risk Management release notes

Version history for the Performance Analytics - GRC: Risk Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 12.0.1 - September 2021**
    -   Fixed: Corrected the app dependencies.
    -   Removed: Removed the dependency on GRC: Risk Management
-   **Version 12.0.0 - March 2021**

    Fixed: Fixes for PRB1373924, PRB13755157. Explicit roles plugin causing invalid interactive filter references.

-   **Version 11.0.0 - October 2020**
    -   Changed: Risk Overview and Risk Overview \(PA\) should be hidden after the migration to Advanced Risk is completed.
    -   Fixed: Corrected licensing subscription model
-   **Version 10.1.0 - June 2020**

    Fixed: Fixed layout for Risk Event Metrics hint popup

-   **Version 9.0.1 - November 2019**

    This feature was introduced in Madrid and is still compatible with New York.
