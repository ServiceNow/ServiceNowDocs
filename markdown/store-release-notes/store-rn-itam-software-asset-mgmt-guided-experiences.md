# Software Asset Management Guided Experiences release notes

Version history for the Software Asset Management Guided Experiences application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 7.0.1 - March 2026 \(Australia\)**

    In this version, software lifecycle reporting and compliance management are streamlined through a guided playbook.

-   **Version 6.0.3 - August 2025**
    -   New:
        -   Adobe Guided Setup
        -   Get a prescriptive guidance for the tasks that you must perform setup Adobe integration with SAM Pro. This Guided Setup organizes the configuration activities into various categories for ease of use with a list of tasks to be performed.
        -   Enhancements are made in existing guided setups including SQL server, Windows server ,SAM and M365.
-   **Version 5.0.4 - February 2025**
    -   New:
        -   Software Asset Management \(SAM\) Guided Setup:
            -   Get step-by-step guidance on the activities that you must perform to onboard SaaS and on-premises products. The guided setup helps you create or associate success goals, configure product integrations, create software entitlements, and run reconciliation to get the most out of the Software Asset Management application.
        -   Microsoft 365
            -   Get a prescriptive guidance for the tasks that you must perform in the Software Asset Management application, Microsoft 365 Admin Center, and other applications to configure Microsoft 365. This Guided Setup organizes the configuration activities into various categories for ease of use with a list of tasks to be performed.
-   **Version 4.0.4 - August 2024**
    -   This software application incorporates guided setups for complex software\(s\), providing a structured method for accomplishing routine software asset management duties with detailed, step-by-step guidance. Furthermore, it integrates functionalities for performing in-depth cluster analysis via SAM Cluster 360.
    -   Some of the main features covered are:
        -   Microsoft SQL Server License Management Guided Setup provides a sequence of tasks that help you configure Microsoft SQL Server on Software Asset Management Pro.
        -   Microsoft Windows Server License Management Guided Setup provides a sequence of tasks that help you configure Microsoft Windows Server, Microsoft System Center and Microsoft Core Infrastructure Suite on Software Asset Management Pro.
        -   SAM Cluster 360 provides SAM Managers with a holistic and strategic analysis of hypervisor clusters. It offers a single pane of glass to view infrastructure, licensing, optimization &amp; health across the cluster on Software Asset Management Pro.
