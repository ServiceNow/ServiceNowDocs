# Care Team Operations for EVS release notes

Version history for the Care Team Operations for EVS on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.0 - June 2026**
    -   This release delivers platform-wide accessibility \(WCAG 2.2 AA, 400% zoom/reflow\) and security-directive access-control hardening for Care Team Operations for Environmental Services.
        -   Security and Compliance
            -   WCAG 2.2 AA accessibility checkers: Added to Care Team Operations for EVS to support automated accessibility validation.
            -   Automated WCAG 2.2 AA accessibility testing: Enabled across all Industry Verticals applications in this bundle.
            -   WCAG 2.2 AA compliance and 400% zoom/reflow support: Delivered across Care Team Operations for Environmental Services.
            -   Non-Glide access control hardening: Audited and merged non-Glide ACL changes across Care Team Operations for EVS.
-   **Version 1.4.0 - March 2026**
    -   Fixed:
        -   The Australia release introduces enhanced protections for read‑only fields across the ServiceNow AI Platform. These changes include a new “read\_only\_option” field with granular control levels, including “strict\_read\_only” and “client\_script\_modifiable". The changes occur in the back end and maintain backward‑compatible behavior. This update helps strengthen your instance security while preserving the flexibility you need.         -   If you have the feature administrator role you can now complete tasks that were initially reserved for users with the broader administrator role.
-   **Version 1.1.0 - December 2025**

    Fixed: When a care team member rejects resolution on a case, that case will now change from Resolved to Work in Progress.

-   **Version 1.0.0 - August 2025**

    The ServiceNow Care Team Operations for Environmental Services Healthcare Operations Core platform and includes case types, record producers, and workflows designed to help care teams report issues related to environmental services. For organizations usingField Service Management, the out-of-the-box workflows synchronize data between the case and the work order. This enables facilities technicians to manage and complete service requests efficiently, ensuring timely resolution of physical environment and operational infrastructure issues raised by care teams.
