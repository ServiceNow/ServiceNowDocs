# Incident Management for Field Service release notes

Version history for the Incident Management for Field Service application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.2 - July 2025**

    Fixed the translation issue for some languages on the work notes

-   **Version 1.1.1 - January 2025**

    Update application Java Version to 17.

-   **Version 1.0.0 - September 2021**

    The ServiceNow® Incident Management for Field Service plugin provides seamless integration between Incident Management and Field Service Management applications. Use this capability to create work orders from an incident. Whenever state changes or a work note is added to the work order, efficiently track the progress of work orders directly from their respective parent incidents.
