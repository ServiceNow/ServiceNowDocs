# Customer Service with Request Management release notes

Version history for the ServiceNow® Customer Service with Request Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.1.0 - April 2026**

    Fixed: Demo data fix.

-   **Version 2.0.0 - March 2026**

    Customer Service with Request Management provides an integration between Customer Service Management and the Request Management application. Enables users to create request records from a Customer Service case.
