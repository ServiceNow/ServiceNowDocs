# Security Operations Setup Assistant release notes

Version history for the Security Operations Setup Assistant on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.4.41 - June 2026**
    -   New:
        -   Added the sn\_secops\_setup.admin role for Setup Assistant access, automatically inherited by sn\_si.admin users.
        -   Setup Status table fields are now strict read-only to prevent unauthorized client-side modifications.
-   **Version 10.3.9 - May 2023**

    Changed: Uncontrollable package for setup assistant.

-   **Version 10.3.8 - February 2023**

    Changed: Updated React library package versions.

-   **Version 10.3.5 - November 2022**

    Changed: Updated React library package versions.

-   **Version 9.0.0 - November 2019**

    Fixed: Minor PRB fixes post the Q3 July release.

-   **Version 8.0.9 - June 2019**

    Refer to Security Incident Response release notes for product changes and updates in the Madrid release.
