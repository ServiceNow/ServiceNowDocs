# Chat Recommendation release notes

Version history for the Chat Recommendation application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.2 - August 2024**
    -   AI-powered recommended responses are designed​to reduce time spent writing responses and finding resolution, thereby improving agent productivity by providing:​
        -   AI-recommended responses based on KB articles, similar chats, and the context of the conversation in real time. ​
        -   Help me write whenever the agent needs help composing a message. ​
        -   Editing capabilities \(elaborate/shorten\) for messages created by the agent or composed by the AI.
