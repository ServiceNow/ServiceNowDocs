# Service Operations Workspace Admin Center release notes

Version history for the ITSM Service Operations Workspace Admin Center application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.2.0 - February 2024**
    -   New: Roles-based access support for admin configurations
    -   Changed: Updated admin landing page to include additional section for configuration of Service Operation Workspace apps
-   **Version 4.1.0 - November 2023**

    No changes. Released version due to a dependent app version upgrade.

-   **Version 4.0.2 - November 2023**
    -   New:
        -   Introduced a side panel for easier navigation
        -   Added the SRM configuration module
    -   Changed:Renamed Admin Console to Admin Center.
-   **Version 3.1.0 - September 2023**

    Changed: Version number update for dependent app version

-   **Version 3.0.3 - August 2023**

    Service Operations Workspace Admin Center is part of the Service Operations Workspace Core bundle. This application contains the Admin Console which surfaces the related landing page configurations for Service Operations Workspace.
