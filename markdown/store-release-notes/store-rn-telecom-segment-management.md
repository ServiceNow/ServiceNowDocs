# Segment Management release notes

Version history for the ServiceNow® Segment Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.3.0 - May 2026**

    Changed: security upgrades

-   **Version 1.1.0 - January 2026**

    New: Added read &amp; write access to the system property -sn\_seg\_segment\_activity.fields for the segment admin role.

-   **Version 1.0.0 - December 2025**

    Transform how you categorise and engage your partner ecosystem with segment management. This application provides a flexible framework to group channel partners based on value, performance, and strategic criteria—enabling differentiated support levels, targeted incentives, and tailored engagement strategies.
