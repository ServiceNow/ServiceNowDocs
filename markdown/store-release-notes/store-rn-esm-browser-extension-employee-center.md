# Browser Extension for Employee Center release notes

Version history for the Browser Extension for Employee Center application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - June 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 1.1.1 - December 2025**

    Fixed the security access for content in Quick links associated with a website table \(Removed admin overrides for ACLs on Quick links associated with websites to provide granular admin roles\).

-   **Version 1.0.1 - October 2025**

    Fixed the search bar positioning issue where the search bar was scrolling along with the cursor instead of remaining static at the bottom of the Browser Extension for Employee Center home page.

-   **Version 1.0.0 - August 2025**

    Employee Center Browser Extension provides all the support and guidance directly within third-party work applications through a draggable pop-up. It enables seamless access to Employee Center content and resources without switching browser tabs or windows.
