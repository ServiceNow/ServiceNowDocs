# Employee Travel Safety release notes

Version history for the Safe Workplace Employee Travel Safety on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

**Note:** To automatically install all the Safe Workplace applications at once, download the ServiceNow® Safe Workplace suite application.

-   **Version 1.22.0 - December 2025**
    -   Fixed:
        -   Updated read only fields to use new platform security features.

    Changed: Minor ACL updates

    Security fixes.

    Fixed: Security fix

-   **Version 1.10.0 - December 2020**

    New: Localization for features from the November 19 release

-   **Version 1.9.0 - November 2020**
    -   New:
        -   In the new My Safe Workplace page, there is a new Employee Travel Safety category in the 'Things you can request' module
        -   Localization for features from the November 5 release
-   **Version 1.8.0 - November 2020**
    -   New:
        -   Employees can cancel travel from the service portal view
        -   New catalog item icons
    -   Fixed: Travel request state changes are now shown on the travel request service portal page
-   **Version 1.7.0 - October 2020**
    -   New: Localization for features from the October 1 release
    -   Fixed: Minor bug fixes
-   **Version 1.6.2 - October 2020**
    -   New:
        -   Employees can view request details and status in a portal view
        -   Requesters can cancel travel requests, and are notified when the request is canceled
        -   If app\_imt\_mobile is installed, 'Request Permission to travel' is now removed from the **More** tab in the mobile app
        -   Localization for features from the September 16 release
-   **Version 1.1.0 - September 2020**
    -   New:
        -   Ability to send reminder emails for approvals
        -   Enable one-time health verification
        -   Localization for the September 3 release features
    -   Fixed: Run once/daily sys\_property should be below the health verification property
-   **Version 1.0.1 - September 2020**
    -   The Employee Travel Safety application gives organizations the ability to pre-authorize employees for business travel based on the safety status of the destination area. Employee Travel Safety enables employees to submit workplace travel requests, triggering automated outreaches for customers to complete questionnaires regarding their site location status.
    -   Managers can make informed decisions on travel requests upon reviewing location site information and leveraging local COVID-19 trend data, while keeping employee health and safety in mind. Employees are prompted to enroll in health verifications before and throughout the duration of the trip. Daily contact tracing check-in requests are also sent to employees while traveling to ensure safety guidelines are being followed.
