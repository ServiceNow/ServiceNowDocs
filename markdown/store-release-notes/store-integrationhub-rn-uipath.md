# UiPath spoke release notes

Version history for the Integration Hub UiPath spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.6.0 - June 2026**

    Fixed: Streamline ACLs.

-   **Version 2.5.1 - November 2025**

    Fixed: Read-Only Directive. Refer the Global Communication for the detailed notes

-   **Version 2.4.1 - October 2025**

    This spoke has been updated to use granular, feature-specific admin roles instead of relying on the broad admin role, ensuring more precise access control.

-   **Version 2.4.0 - September 2025**

    This spoke has been updated to use granular, feature-specific admin roles instead of relying on the broad admin role, ensuring more precise access control.

-   **Version 2.3.0 - October 2024**
    -   New:
        -   Added new action 'Get Folders'
        -   Enable Java 11 for UiPath Spoke
-   **Version 2.2.3 - July 2023**

    Fixed: Add Item to Queue action that displays unrelated cached dynamic choice options in the Folders and Queues input drop downs.

-   **Version 2.2.2 - March 2023**

    This version fixes the Add Item to Queue action displaying unrelated cached dynamic choice options in the Folders and Queues input lists.

-   **Version 2.2.1 - September 2022**

    Fixed: Improve installation performance of spoke.

-   **Version 2.2.0 - September 2022**
    -   New:
        -   Incoming webhook support via UiPath Webhook Registry and UiPath Webhook Decision Policy modules
        -   OAuth support with authentication templates
    -   Fixed: Naming upgrade for many actions
-   **Version 2.1.2 - February 2022**

    Fixed: Patch version of Ui Path Spoke for IntegrationHub. This version fixes icon issues within flows.

-   **Version 2.1.1 - December 2021**

    Fixed: Patch release of UiPath Spoke for IntegrationHub. This version adds a fix to flows that errors out with "Invalid Specific Content format" at action "Add Item To Queue" after upgrading.

-   **Version 2.1.0 - October 2021**

    Fixed: Security bug.

-   **Version 2.0.1 - July 2021**

    Patch release of the UiPath spoke for IntegrationHub. This version adds KMF modules for additional security of the password2 fields and Rome compatibility.

-   **Version 2.0.0 - October 2020**

    Major release of the UiPath spoke for IntegrationHub.

-   **Version 1.0.3 - September 2020**

    Patch release of UI Path spoke for IntegrationHub.

-   **Version 1.0.2 - March 2020**

    Provides actions to automate the management of Jobs and Robots in UiPath.
