# File Explorer for Security Incident Response release notes

Version history for the File Explorer for Security Incident Response application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.1 - June 2026**

    Fixed: Added read\_only option on few table fields.

-   **Version 1.3.0 - December 2025**

    Fixed: Resolved an issue where the Modified by field was being set to the actual user instead of the system user during SharePoint integration updates. The field now correctly reflects the system user for automated modifications.

-   **Version 1.2.3 - May 2024**

    Fixed: Empty ACLs and Report view ACLs.

-   **Version 1.2.2 - March 2022**

    Fixed: Updated the Default Template record table.

-   **Version 1.2.0 - March 2022**

    Fixed: Bug fixes.

-   **Version 1.1.1 - December 2021**

    Provides base infrastructure to integrate with any third-party file hosting provider such as Microsoft SharePoint.
