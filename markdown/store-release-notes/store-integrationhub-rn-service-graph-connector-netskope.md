# Service Graph Connector for Netskope release notes

Version history for the Service Graph Connector for Netskope application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.1 - June 2026**
    -   The following enhancements and changes support internal security directives:
        -   Query ACLs added to the Netskope SGC client attributes, client import, and user attributes tables to align with ServiceNow Platform Security guidance.
        -   Thirty system-populated fields across the Netskope client attributes and user attributes tables are read-only to prevent modification of imported asset data outside of the integration's own data stream actions.
-   **Version 2.0.0 - February 2025**

    This integration is one of the ServiceNow-developed Service Graph Connectors. The Service Graph Connector for Netskope imports hardware assets from the Netskope product into your ServiceNow Configuration Management Database \(CMDB\) quickly, seamlessly, and securely. You can use the Guided Setup to help you configure your integration.
