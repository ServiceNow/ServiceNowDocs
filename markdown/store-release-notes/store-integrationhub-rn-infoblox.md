# Infoblox spoke release notes

Version history for the Integration Hub Infoblox spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - June 2026**

    Non Glide Cobalt Raven ACLs added.

-   **Version 2.0.4 - March 2024**

    Fix for InfobloxUtils scirpt was not returning the expected query for Retrievesearchquery.

-   **Version 2.0.3 - September 2023**

    Fixed: 'licensable' field set to be true in plugin.xml.

-   **Version 2.0.2 - September 2022**

    Patch release of Infoblox Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 2.0.1 - July 2022**
    -   New:
        -   Patch version of Infoblox Spoke for IntegrationHub. This version adds an input "Use for Extensible Attributes Inheritance" for actions - Create Host Record, Update Host Record, Change IPs of Host Record and adds more attributes to Host record's IP addresses output for the actions - Look up Host Records Stream, Look up Records Stream, Look up Records Criteria Stream, Look up Host Record by Reference, Look up Record by Reference, Add IPs to Host Record, Create Host Record, Update Host Record, Change IPs of Host Record.
-   **Version 2.0.0 - July 2021**
    -   Added generic create, look up, update, and delete records with extensible attributes support
    -   Added create, look up, and update host records
    -   Naming convention clean up
-   **Version 1.0.2 - August 2020**

    Fixed: This is a patch version and includes a fix.

-   **Version 1.0.1 - August 2019**

    Licensing patch for the Infoblox spoke on Integration Hub

-   **Version 1.0.0 - May 2019**

    Initial release of Infoblox spoke for Integration Hub.
