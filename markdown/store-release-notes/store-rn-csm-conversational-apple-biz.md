# Conversational Integration with Apple Messages for Business release notes

Version history for the Conversational Integration with Apple Messages for Business application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - March 2026**

    Fixed: Internal security fixes as part of security directive.

-   **Version 1.1.7 - August 2025**

    This is an internal security fix.

-   **Version 1.1.6 - December 2024**

    Changed the internal API signature for creating outbound attachments. No customer impact.

-   **Version 1.1.5 - November 2024**

    Fixed: Added security fix to use glideRecordSecure instead of GlideSecure to avoid writes in sandbox mode.

-   **Version 1.1.3 - August 2024**

    Fixes for RTL language support in onboarding page and for timezone conversion issue in date time control.

-   **Version 1.1.1 - February 2024**

    Changed: Changes to onboarding landing page, fixes for knowledge management framework \(KMF\) issues.

-   **Version 1.0.2 - September 2023**

    Fixed: Issue with response to the "help" command.

-   **Version 1.0.1 - May 2023**

    ServiceNow Conversational Integration with Apple Messages for Business enables brands to engage with their customers using Apple Messages for Business.
