# Google Tasks spoke release notes

Version history for the Integration Hub Google Tasks spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - September 2025**

    Fixed - Required ACL\(s\) or Role\(s\) for agents.

-   **Version 1.3.1 - July 2025**

    Fixed: AI Agents - Changed custom to template.

-   **Version 1.3.0 - May 2025**

    Minor release with new AI agents.

-   **Version 1.2.2 - September 2024**

    Fixed: security bug.

-   **Version 1.2.0 - May 2023**

    New: Authentication template for easier spoke setup.

-   **Version 1.1.1 - September 2022**

    Patch release of Google Tasks Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.1 - December 2019**

    Provides actions to automate Task and Tasklist management in Google Tasks.
