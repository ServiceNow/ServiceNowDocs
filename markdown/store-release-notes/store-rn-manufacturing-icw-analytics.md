# ICW Analytics release notes

Version history for the ServiceNow® ICW Analytics application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.0 - June 2026**

    New: SQDC Dashboard, consisting of an overview page and individual tabs for Safety, Quality, Delivery, and Cost metrics and trends.

-   **Version 0.1.1 - March 2026**

    ICW Analytics enables data-driven decision-making by surfacing execution trends and quality signals directly to teams with no need to navigate to separate reporting tools.
