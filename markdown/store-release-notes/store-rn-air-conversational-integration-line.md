# Conversational Integration with LINE release notes

Version history for the Now Intelligence Conversational Integration with LINE on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.7 - February 2025**
    -   Evaluate Sandbox Access for Client-Callable Script Includes \(Virtual Agent\)
    -   Added translations for strings to be translated
-   **Version 2.0.3 - August 2023**
    -   New:
        -   Create an API to integrate LINE with ServiceNow Instance.
        -   UI designs and configuration for LINE integration.
    -   Fixed:
        -   Line Integration API - Additional Flag in API Response to support UI alerts.
        -   Bot name for line is getting changed if kept as default.
        -   Breadcrumb for contextual action is not as mentioned in the figma.
        -   Remove integration link under helpful resources is missing from the manage page.
        -   User is not getting an option to toggle on and off notification for Line on settings page.
-   **Version 2.0.1 - February 2023**

    Fixed: Minor fixes.

-   **Version 1.0.2 - December 2020**

    Enable brands to engage with their customers using the LINE messaging app.
