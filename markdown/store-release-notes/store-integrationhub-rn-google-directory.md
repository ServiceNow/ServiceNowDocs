# Google Directory spoke release notes

Version history for the Integration Hub Google Directory spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 1.5.2 - October 2024**

    Fixed: 1 bug.

-   **Version 1.5.1 - January 2024**

    Fixed: Report View ACLs, permissions for PW Module Access Policy.

-   **Version 1.4.1 - December 2022**

    Fixed: Improve installation performance of spoke and fix an issue related to empty roles in ACLs.

-   **Version 1.3.1 - February 2022**

    New: Added a sample remote table and a sample outbound flow for user provisioning.

-   **Version 1.1.3 - November 2021**

    Fixed: Patch release of Google Directory Spoke for IntegrationHub. This version includes fixes for file name warnings.

-   **Version 1.1.2 - August 2021**

    New: Patch release of the Google Directory spoke for IntegrationHub. This version adds KMF module for additional security of password2 fields and Generate Random Password action.

-   **Version 1.0.0 - April 2020**

    Provides actions to automate the management of Google Directory Users and Groups.
