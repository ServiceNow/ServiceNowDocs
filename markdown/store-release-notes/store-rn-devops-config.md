# DevOps Config release notes

Version history for the ITSM DevOps Config on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.0.0 - February 2024**
    -   New:
        -   Create all apps access role in devops config
        -   User should be able to assign any user group to an application library so that I needn't be part of that group to make assignments
-   **Version 4.1.0 - November 2023**
    -   New:
        -   CSV is now a supported format for uploading and exporting configuration data
        -   REST APIs added for managing Dynamic Mapping in PaCE
    -   Changed: Dynamically map policies to deployables using business conditions
    -   Fixed: The policy editor role can no longer activate/deactivate a policy
-   **Version 4.0.0 - August 2023**

    Bug fixes and improvements.

-   **Version 3.1.0 - May 2023**

    New:

    -   Shared Components - ability to use components across different applications.
    -   Config Data Analyzer - ability to analyze differences between data models of applications.
-   **Version 3.0.0 - February 2023**

    Fixed: Bug fixes and improvements

-   **Version 2.0.0 - August 2022**
    -   New: Changes along with DevOps Change Velocity product to be now able to link a snapshot to a change request
    -   Changed: Snapshot linkage to change request
