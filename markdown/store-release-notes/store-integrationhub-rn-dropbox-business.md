# Dropbox Business spoke release notes

Version history for the Integration Hub Dropbox Business spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.0.5 - March 2024**

    Fixed refreshing the OAuth token.

-   **Version 1.0.4 - September 2022**

    Patch release of Dropbox Business Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.3 - December 2021**

    Changed: Patch release of the Dropbox Business spoke for IntegrationHub. This version includes security-related changes.

-   **Version 1.0.2 - August 2020**

    Fixed: This version includes a fix for the Copy Dropbox File to Attachment action.

-   **Version 1.0.1 - June 2020**

    Provides actions to automate the management of files, folders, groups, and teams in Dropbox Business.
