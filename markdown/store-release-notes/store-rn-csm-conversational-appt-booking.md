# Conversational Appointment Booking release notes

Version history for the Conversational Appointment Booking application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.2 - June 2026**
    -   Changed:
        -   Conversational Appointment Booking now supports Now Assist Virtual Agent, enabling customers to schedule, reschedule, or cancel Walk-up appointments using native chat or Engagement Messenger.
        -   Agents can delegate appointment management to the Virtual Agent mid-conversation using Conversation Autopilot.
        -   Admins can configure appointment booking flows using the Appointment Booking Rich Control in Virtual Agent topics.
-   **Version 1.4.1 - March 2026**

    Fixed: Internal security fixes as part of security directive.

-   **Version 1.2.1 - August 2023**

    Fixed issue with cross scope application access.

-   **Version 1.2.0 - February 2023**

    New: Updated the app to allow appointment booking without needing to provide a location.

-   **Version 1.1.3 - June 2022**

    Fixed: Timezone bug. Users now see the dates correctly based on the timezone associated with the config/user/system default.

-   **Version 1.1.1 - April 2022**

    Fixed: Minor issue where the date format would be incorrectly displayed in the topic.

-   **Version 1.1.0 - February 2022**

    Changed: Updates to incorporate the Next Experience theming and branding changes.

-   **Version 1.0.4 - September 2021**

    Conversational Appointment Booking enables customers to manage appointments using the ServiceNow Virtual Agent via native chat capabilities. Virtual Agent topics are provided to support Customer Service Management Walk-up appointments.
