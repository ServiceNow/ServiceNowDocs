# Service Graph Connector for ServiceNow OT Discovery release notes

Version history for the ServiceNow® Service Graph Connector for ServiceNow® OT Discovery application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.7 - June 2026**
    -   New:
        -   SGC central support. SGC central enables to discover and install connectors, and then effectively manage the full life cycle of creating, editing, monitoring, and debugging connections.
        -   Download of containerized packages of OT Discovery console, OT Discovery Collector.
-   **Version 1.0.4 - March 2026**

    Integrate the ServiceNow OT Discovery engine with the ServiceNow Operational Technology Manager application to import and manage sites, devices and softwares discovered by ServiceNow’s OT Native Discovery.
