# Document Intelligence Admin release notes

Version history for the Document Intelligence Admin application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.3 - May 2025**

    No updates.

-   **Version 3.2.0 - February 2025**

    Fixed: Hide features and use cases based on version compatibility.

-   **Version 3.1.1 - November 2024**
    -   Fixed:
        -   Accuracy calculations in Accuracy charts
        -   Compatibility issues with Vancouver and Washington
-   **Version 3.1.0 - September 2024**
    -   New: Ability to select the OCR model when creating a use case
    -   Fixed: Issue with creating use cases in some locales where the comma is the primary delimiter
-   **Version 3.0.3 - August 2024**
    -   New:
        -   Monitoring dashboards
            -   Monitor the overall performance of Document Intelligence in the Document Intelligence monitoring dashboard
            -   Monitor performance at the use case level in the use case performance dashboard
        -   Data normalization improvements
            -   Determine a default interpretation of ambiguous values for date and number fields
            -   Create a Reference field type to match extracted field values with fields in a referenced table record
-   **Version 2.1.4 - March 2024**
    -   Changed: Updated information about supported languages on the Home page.
    -   Fixed: Errors in some help dialogs.
-   **Version 2.1.2 - February 2024**
    -   Fixed:
        -   Ability to process documents after copying a use case
        -   Remove the option to select a target field if a target table has not been set
        -   Various small UI issues
-   **Version 2.1.1 - September 2023**

    New: Ability to configure new field types \(date, integer, decimal, float\) in your use cases.

-   **Version 2.0.0 - August 2023**
    -   New: Ability to copy and export your use cases
    -   Changed: More granularity for straight-through processing mode, enabling you to define fields to consider for full automation
    -   Fixed: Multiple, minor usability issues
-   **Version 1.1.0 - March 2023**
    -   New: Embedded help directly within the application
    -   Changed: Ability to set up use cases that involve check box extraction
    -   Fixed:
        -   Inability to select target fields from the field group's target table
        -   Various user experience issues
-   **Version 1.0.2 - February 2023**

    Document Intelligence Admin provides a new administration experience for Document Intelligence. Built on the Next experience, it provides a better user experience for process owners to set up, configure, and monitor their document processing solutions​.
