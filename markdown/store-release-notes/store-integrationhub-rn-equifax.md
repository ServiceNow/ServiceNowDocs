# Equifax Spoke release notes

Version history for the on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.0 - August 2023**

    The Equifax spoke allows ServiceNow to connect with Equifax's API products and access important information about customer credit history, fraud alerts, digital identity verification, transaction screening, and other relevant data. These spoke actions help businesses evaluate customer creditworthiness, verify customer identities, comply with anti-money laundering requirements, and more. This ability enables businesses to make faster lending decisions, verify new employee information, obtain insurance, set up utility and telecommunications account acquisitions, to name a few examples of data utilisation.
