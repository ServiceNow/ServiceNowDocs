# Scope 3 emissions management release notes

Version history for the Scope 3 emissions management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.3.0 - June 2026 \(Australia\)**

    Changed: This release includes security enhancements that strengthen access controls across the application.

-   **Version 21.1.1 - December 2025 \(Zurich\)**

    New: Minor enhancement to expose 'Active' field configuration for Choice options.

-   **Version 21.0.0 - August 2025**

    Changed: Refactored existing metric definitions to leverage choice-type fields instead of free-form text, improving data consistency and validation.

-   **Version 20.1.0 - May 2025**

    Fixed: Accessibility improvements.

-   **Version 20.0.1 - February 2025**

    Changed: Performance improvements of the 'Emissions by supplier category' script.

-   **Version 19.0.2 - August 2024**

    The Scope 3 dashboard is designed to assist in calculating and monitoring scope 3 emissions, providing a comprehensive view of your organization's environmental footprint and helping to maintain compliance with changing regulations. Scope 3 emissions encompass indirect emissions within your value chain, such as those arising from the procurement of equipment.
