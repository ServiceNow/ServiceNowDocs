# Performance Analytics for Security Incident Response release notes

Version history for the Security Operations Performance Analytics for Security Incident Response on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.5.2 - May 2025**

    Fixed: "Daily data collection" was incorrectly configured with a "Relative start" of 15, and conditions for certain Indicator Sources are incorrect \("Closed on Yesterday", "Opened on Yesterday".\)

-   **Version 10.5.1 - November 2024**

    Fixed: Resolved the Incorrect conditions on the indicator source SI. Open in Performance Analytics for Security Incident Response App version 10.4.2.

-   **Version 10.4.2 - May 2024**
    -   Changed: Deprecated the old Performance Analytics dashboards
    -   Fixed: Updated indicator sources from today to yesterday
-   **Version 10.4.0 - February 2024**

    Changed: A few dashboard widgets have been migrated to Next Experience.

-   **Version 10.3.11 - January 2024**

    New: The Security Incident Manager \(sn\_si.manager\) role doesn't have access to view a few widgets on the Security Incident Manager Overview dashboard.

-   **Version 10.3.9 - December 2023**

    Fixed: The Security Incident Manager \(sn\_si.manager\) was unable to view widgets on the Security Incident Manager Overview dashboard on Utah or higher versions.

-   **Version 10.3.6 - May 2023**

    New: You can now view all the CISO and Manager Performance Analytics \(PA\) dashboards on the Security Incident Response Workspace.

-   **Version 10.3.4 - February 2023**

    New: Added supporting changes to display all the Performance Analytics \(PA\) dashboards on the Security Incident Response Workspace.

-   **Version 10.0.0 - March 2020**
    -   Fixed:
        -   Formula updated for calculating Closed Security Incidents per Analyst in the Security Operations Efficiency dashboard.
        -   To enable the solution, an administrator can navigate to Performance Analytics &gt; Guided Setup, click Get Started, and then scroll to the section for Security Incident Response. https://docs.servicenow.com/bundle/newyork-release-notes/page/release-notes/security-operations/secops-sir-rn.html
-   **Version 9.0.0 - November 2019**
    -   Fixed: Minor Bug fixes since July Q3 Release for Security Operations Efficiency Dashboard.
    -   To enable the solution, an administrator can navigate to Performance Analytics &gt; Guided Setup, click Get Started, and then scroll to the section for Security Incident Response.
-   **Version 8.0.11 - July 2019**
    -   New: Recertified for New York
    -   Fixed: Minor bug fixes
-   **Version 8.0.9 - June 2019**

    To enable the solution, an administrator can navigate to **Performance Analytics** &gt; **Guided Setup**, click **Get Started**, and then scroll to the section for Security Incident Response.
