# Timeline component release notes

Version history for the Timeline component application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 29.0.1 - June 2026**

    Fixed various minor defects

-   **Version 29.0.0 - March 2026**

    Changed: now utilizing updated Horizon design system tokens

-   **Version 28.0.1 - December 2025**

    Fixed various issues

-   **Version 27.0.4 - June 2025**

    Fixed: timeline breaking issue in last version.

-   **Version 27.0.3 - February 2025**
    -   New: New property to configure ARIA attributes
    -   Fixed:
        -   Synchronization issue when there are more than one Timeline component
        -   Various minor bugs
-   **Version 26.0.2 - August 2024**
    -   Changed: Updated the component icon in the UI builder toolbox.
    -   Fixed: Minor defects.
-   **Version 25.0.1 - February 2024**

    Fixed various minor issues.

-   **Version 24.0.5 - August 2023**
    -   New
        -   Added accessibility features to support 2.1AA compatibility.
        -   Added features that adds the ability to add single or multiple markers on timelines by providing the timestamps where the markers should render on-load.
        -   If the marker functionality is enabled, users can add, replace, or clear markers on their application via the context menu or by right-clicking on the timeline.
    -   Fixed
        -   Various fixes to address minor issues.
-   **Version 23.0.1 - August 2023**

    This version contains the ability to add single or multiple markers on timelines by providing the timestamps where the markers should render on-load. If the marker functionality is enabled, users can add, replace, or clear markers on their application via the context menu or by right-clicking on the timeline.

-   **Version 22.1.1 - February 2023**

    Fixed: Minor bug fixes

-   **Version 22.1.0 - November 2022**
    -   Changed:
        -   Updates to tool tips
        -   "No data state" property grouping and title changes
-   **Version 22.0.5 - August 2022**

    Show events and information across a timeline to contextualize history or important information for an object.
