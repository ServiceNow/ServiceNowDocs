# Knowledge API release notes

Version history for the Customer Service Management Knowledge API on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 29.0.1 - March 2026**

    No major changes in this release.

-   **Version 1.0.2 - May 2022**

    Fixed: Improved functionality in Knowledge API related to fetching attachments.

-   **Version 1.0.1 - December 2021**
    -   Fixed: Bug fix to allow configurability of the REST API end point.
-   **Version 1.0.0 - March 2020**

    Provides REST API endpoints that enable applications to integrate with the Knowledge Management application for searching, viewing, and fetching lists of most viewed and featured knowledge articles.
