# Oracle Financial Cloud spoke release notes

Version history for the Integration Hub Oracle Financial Cloud spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.0 - June 2026**
    -   Security patch for non-glide ACLs
    -   Fixed: Range-based pagination has been added to the Look Up Records Stream action, and a total count has been added to the Get Dynamic Response Schema \(Metadata\) action.
-   **Version 1.4.0 - March 2026**

    Added 1 action.

-   **Version 1.3.1 - February 2026**

    Fixed: Security fixes on missing OOB report view and JSON injection vulnerability

-   **Version 1.3.0 - January 2026**

    Added 5 new spoke actions.

-   **Version 1.2.0 - November 2025**
    -   Enhanced Create Record and Look up Records Stream action
    -   Added Update Record and Delete Record
-   **Version 1.1.0 - September 2023**

    Fixed: 'licensable' field set to be true in plugin.xml.

-   **Version 1.0.6 - September 2022**

    Patch version of Oracle Financial Cloud Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.4 - October 2021**

    Changed: Label changes in Get Oracle Cloud Financials Object Metadata.

-   **Version 1.0.3 - November 2020**

    Patch release of the Oracle Financial Cloud spoke for IntegrationHub

-   **Version 1.0.1 - September 2020**
    -   The Oracle Financial Cloud spoke provides a list of actions that wraps around the primary REST and SOAP API provided by Oracle Financial Cloud. In addition, this spoke provides three sample flows that demonstrate how to perform:
        -   Constant sync AP invoices and journals from ServiceNow to Oracle Financial Cloud
        -   Bi-directionally create and update AP invoices from ServiceNow to Oracle Financial Cloud
