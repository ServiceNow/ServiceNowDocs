# Admin Workspace for Service Providers \(SPs\) release notes

Version history for the Admin Workspace for Service Providers \(SPs\) application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.4 - August 2025**
    1.  Fixed:
        1.  Previously, at 400% zoom, carousel components on the landing page lacked contrast and blended into the background. This has been fixed components now appear with clear, contrasting colors for better visibility.
        2.  Buttons on the SP Admin landing page and Client Details page were missing discernible text in both light and dark coral themes. Accessible labels have now been added to drop-downs in the Trends Over Time and Related Links widgets, ensuring screen reader compatibility and improved accessibility.
-   **Version 1.1.2 - May 2025**

    Fixed: Minor changes related to UI theming.

-   **Version 1.1.0 - August 2024**
    -   New:
        -   Minor performance improvements on landing page
        -   Guided Setup quick link, to initiate a domain separated HR Guided Setup, under Client Details tab
-   **Version 1.0.2 - February 2024**

    The Admin Workspace designed for Service Providers \(SPs\) offers a unified perspective on service utilization and seamless access to configuration options. This enables service provider admins to swiftly identify and address potential service requirements and gaps by utilizing comprehensive trended metrics. Additionally, service provider admins can compare the individual needs of customers with those of peers and the entire collection within the instance.
