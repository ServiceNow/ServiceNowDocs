# Conversational SMS Integration with Twilio release notes

Version history for the Conversational SMS Integration with Twilio on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.2.0 - November 2024 \(Xanadu\)**

    Minor bug fixes.

-   **Version 4.1.0 - August 2024 \(Xanadu\)**

    Fixed: AI Search cards.

-   **Version 4.0.1 - August 2023 \(Vancouver\)**

    Changed: Dependency versions.

-   **Version 3.0.0 - February 2023**

    New: Adapter changes for SMS Authentication.

-   **Version 2.1.0 - November 2022**

    Fixed: Localization fixes.

-   **Version 2.0.3 - August 2022**

    Fixed: Issues with mask type and input text in the logs.

-   **Version 2.0.2 - February 2022**
    -   New: Unified logging framework has been enabled.
    -   Fixed: The unified logging backward compatibility condition.
-   **Version 2.0.1 - October 2021**

    Fixed: Display of large and small images in cards.

-   **Version 2.0.0 - August 2021**
    -   New:
        -   START/STOP custom commands \(contextual actions\) for users to receive SMS notifications and stop receiving SMS notifications.
        -   URL redirect to Service Portal page.
    -   Fixes
-   **Version 1.1.1 - February 2021**

    New: Conversational SMS integration with Twilio is added as a new messaging channel supporting Virtual Agent.
