# Shift Scheduling for Field Service release notes

Version history for the Shift Scheduling for Field Service application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 7.1.2 - June 2026**

    New: Added Holiday schedules implementation for FSM

-   **Version 7.1.0 - April 2026**

    Fixed: WFO API calls with Territory to improve performance

-   **Version 7.0.4 - March 2026**
    -   Description: Shift Planning allows managers and supervisors to configure Workforce Scheduling to enable agents to sign up for shifts.
    -   Key features: Ability to capture the location for the Agents Personal Events.
-   **Version 6.1.0 - January 2026**
    -   Shift Planning allows managers and supervisors to configure Workforce Scheduling to enable agents to sign up for shifts.
    -   Fixed: When a time-off event is created for a user, the shift markspan turns grey, indicating that no shift is assigned.
-   **Version 6.0.4 - August 2025**
    -   Description: Shift Planning allows managers and supervisors to configure Workforce Scheduling to enable agents to sign up for shifts.
    -   Key feature: Ability to capture the location for the Agents Personal Events.
-   **Version 5.1.2 - July 2025**

    This patch release addresses the issue PRB1898328 related to multi-day personal events. After this patch update, personal events created with Workforce Optimization \(WFO\) will be rendered properly and considered for Schedule Optimization executions.

-   **Version 5.1.1 - May 2025**
    -   Shift Planning allows managers and supervisors to configure Workforce Scheduling to enable agents to sign up for shifts.
    -   Key features:
    -   -   Enables agents to sign up for shifts
-   Enables managers and agents to get shift signup notifications
-   New table "Agent Schedule Attribute" enables managers to track agent location and view agent personal event details
-   **Version 5.0.0 - February 2025**
    -   Latest Manager Workspace dependencies uptake for FSM
    -   Performance improvements for shift planning API to retrieve WFO schedules
-   **Version 4.0.6 - December 2024**
    -   Performance improvement for the WFO Shift scheduling API
    -   Schedule optimization flexible break retrieval logic enhancement
    -   Internalization support for the landing page widgets
-   **Version 4.0.2 - August 2024**
    -   Accessibility improvements to adhere to WCAG 2.1AA conformant.
    -   Performance improvements for the FSM Dispatcher Workspace API to retrieve schedules.
-   **Version 3.0.0 - February 2024**
    -   Fixed:
        -   Changes to run APIs run for same input multiple times \(Capacity planning use case\)
        -   Performance changes to improve loading of Manager workspace.
-   **Version 2.0.5 - August 2023**

    Use Workforce Optimization for Field Service to manage agent schedules and breaks.

-   **Version 1.0.0 - February 2023**

    Shift Scheduling offers an easy-to-use, streamlined Manager Workspace experience that allows managers to create/edit work shifts, schedules, and monitor schedule adherence for all their team members.
