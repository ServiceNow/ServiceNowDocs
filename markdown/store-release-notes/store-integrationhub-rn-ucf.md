# UCF spoke release notes

Version history for the Integration Hub UCF spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - May 2023**

    Added Authentication Template for easy 1-click setup.

-   **Version 1.0.1 - September 2022**

    Patch version of UCF Spoke for IntegrationHub. This version includes a fix to improve installation performance of spoke.

-   **Version 1.0.0 - April 2020**

    Provides actions to automate content record management for governance and compliance.
