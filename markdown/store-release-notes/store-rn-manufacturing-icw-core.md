# ICW Core release notes

Version history for the ServiceNow® ICW Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.0.0 - June 2026**

    New: Vertical navigation view for Functional Locations

-   **Version 1.0.4 - May 2026**

    Changed: This patch updates internal references to an upcoming Innovation Labs release of Industrial Loss Elimination.

-   **Version 1.0.0 - March 2026**
    -   Work in industrial environments relies on key concepts like organization structure, equipment layout, work schedules, and common workflows. These elements form a framework for efficient, accountable, and contextually valid task definition and execution.The ICW Core application allows industrial workers and leaders to
        -   Define the organizational structure and layout of their shop floor environments
        -   Set up shift patterns for use when planning tasks
        -   Create and use an industrial knowledge base
        -   Quickly find and execute industrial tasks
