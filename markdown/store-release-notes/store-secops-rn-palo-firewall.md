# Palo Alto Networks - Firewall integration release notes

Version history for the Security Operations Palo Alto Networks - Firewall integration on the ServiceNow Store.

**Note:** This integration is superseded by the new application [Palo Alto Networks Next-Generation Firewall for Security Operations](https://store.servicenow.com/sn_appstore_store.do#!/store/application/031a46fb87401300bbc71c7df7cb0b23/5.0.3?referer=%2Fstore%2Fsearch%3Fq%3DPalo%2520Alto%2520Firewall) and is no longer available for new activations. Customers are encouraged to adopt this new application to take advantage of an expanded set of Security Incident Response use cases leveraging Palo Alto NGFW product capabilities.

## Version history

-   **Version 10.4.0 - February 2021**
    -   New: The SOC Admin now has the ability for custom Approval Group for EDL Entries. Earlier the SOC Admin could only enable/disable Approvals for EDL Entries.
    -   Fixed: Fixed an issue where the "Email FW Retrieval URL" UI action button was not working as expected.
-   **Version 10.3.2 - January 2021**

    Changed: With Key Management Framework plugin, developers will have an ability to manage keys used for Password2 fields through crypto module definition.

-   **Version 8.0.8 - July 2019**
    -   New: Recertified for New York
    -   Fixed: Query performance issues
