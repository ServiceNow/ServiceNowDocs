# iManage Spoke release notes

Version history for the Integration Hub iManage Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.6 - January 2026**

    Fixed: Resolved an issue that removes parsing of integer string to integer. The API now correctly accepts string values, ensuring smoother iManage integration for workspace creation.

-   **Version 1.1.3 - July 2024**

    Fixed: There was an error when uploading documents to legal request, that was set up to store documents in iManage as an external storage. This has been fixed.

-   **Version 1.5.0 - November 2023**
    -   New:
        -   Webhook management actions: Create Subscription Channel, Stop Subscription Channel
        -   Modules: Google Directory Webhook Registries
        -   Decision Table: Google Directory Webhook Decision Policy
        -   Sample Subflows: Process Google Directory User Webhook Subflow, Create Google Directory User Webhook Subflow
-   **Version 1.1.0 - November 2023**
    -   Fixes on existing iManage spoke.
-   **Version 1.0.0 - August 2023**
    -   The Integration Hub Box spoke provides native integration to iManage to easily manage files and folders from ServiceNow.
        -   Auto create and manage iManage Workspaces, folders and files for collaboration to facilitate document management.
        -   Control permissions by using ServiceNow as a system of engagement to add or remove collaborators for faster outcomes.
        -   Manage workspaces, folders to organize documents attached to Legal Requests and Matters in iManage.
