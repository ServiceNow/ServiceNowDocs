# Now Assist for App Engine release notes

Version history for the Now Assist for App Engine application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 29.2.3 - June 2026**

    Changed: Updated versions of dependent apps.

-   **Version 29.1.2 - April 2026**

    Changed: Maintenance release.

-   **Version 28.2.7 - December 2025 \(Zurich\)**

    New: Use the custom app record summarization skill to generate AI summaries for tables in custom applications. You can generate summaries in-product and through chat in the Now Assist panel.

-   **Version 28.1.1 - October 2025**

    Version bump to pull in updated dependencies.

-   **Version 28.0.1 - October 2025**

    Version bump to pull in updated dependencies.

-   **Version 26.4.0 - October 2025**

    Security fixes.

-   **Version 28.1.0 - September 2025**

    Version update for Zurich compatibility.

-   **Version 28.0.0 - August 2025**

    Version update for Yokohama compatibility.

-   **Version 26.3.0 - June 2025**

    This application enables creators to build AI agents and skills using the ServiceNow Platform's native generative and agentic AI capabilities and seamlessly integrate them into their custom App Engine applications.
