# sn-multipivot release notes

Version history for the sn-multipivot application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 23.0.8 - March 2023**

    Bug fixes

-   **Version 23.0.5 - February 2023**
    -   Support multiple data sources
    -   Allow sorting per column during consumption
    -   Improved responsiveness to optimize the use of horizontal space
    -   New settings to configure header transition/wrapping
    -   Bug fixes
    -   Pivot table is a part of the Data Visualization component. It presents ServiceNow or custom data in a multipivot view.
    -   Pivot table data visualizations can be used in Next Experience and are available from Next Experience Dashboards, the UI Builder, and the Visualization Designer.
    -   This application is installed automatically with the "Data Visualization Core" application.
