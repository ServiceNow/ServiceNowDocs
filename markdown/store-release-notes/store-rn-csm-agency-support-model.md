# Agency Support Model release notes

Version history for the Agency Support Model application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - August 2022**
    -   New:
        -   Support for agency services to connect constituents, businesses, and agencies on a single public sector vertical platform
        -   Agent workspace updates to support the 360-degree view of agency activity
        -   Government case types updates to support agencies
        -   Roles for government employees \(agents\) and contributors servicing agencies
