# Portal Next Experience Theme release notes

Version history for the Portal Next Experience Theme application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 24.3.2 - June 2026**

    Product Enhancement and defect fixes.

-   **Version 24.2.2 - March 2026**

    Fixed: Defect fixes.

-   **Version 24.1.1 - December 2025**
    -   Changed: Minor design enhancements
    -   Fixed: Bug Fixes
-   **Version 24.0.2 - September 2025**

    Bug fixes

-   **Version 23.7.0 - February 2025**

    New: Added theme-level changes to support the Business Portal \(business\_portal\).

-   **Version 23.6.2 - November 2024**

    Fixed: Accessibility-related issue.

-   **Version 23.5.0 - August 2024**
    -   Fixed:
        -   Fixed minor localization and translation issues.
        -   Minor changes to ensure that correct color is displayed for links in KB articles.
-   **Version 23.2.0 - May 2024**

    New: Added a Portal Polaris Footer, which displays contact information, social media links, and other common links at the bottom of portal pages when configured in the portal theme.

-   **Version 23.1.0 - February 2024**
    -   New:
        -   Access and navigate a wide range of menu items on mobile devices.
        -   Display cart, wish list, guided tour, and scripted list menu items, such as notifications, approvals, and surveys in the header menu.
-   **Version 23.0.6 - August 2023**

    Fixed: Fixed user interface and accessibility issues with the Portal Next Experience Theme.

-   **Version 23.0.5 - April 2023**

    Changed: This is a patch release to improve the packaging of our release artifacts. We have trimmed down the size of our artifact and eliminated duplicate components. No functional changes have been made as a part of this release.

-   **Version 23.0.4 - February 2023**

    The Portal Next Experience Theme application allows the Next Experience theme to be used with the Customer Support \(CSM\), Customer Service \(CSP\), and Knowledge \(KB\) portals.
