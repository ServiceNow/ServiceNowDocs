# Microsoft OneDrive Spoke for Document Service Framework release notes

Version history for the Microsoft OneDrive Spoke for Document Service Framework application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.2.0 - May 2026**

    Changed: Extended personal authentication for personal authentication in list API.

-   **Version 3.1.0 - December 2025**

    Fixed: Added compatibility support for Australia Glide changes

-   **Version 2.0.0 - February 2024**
    -   Added support for file management with Sharepoint to include the following capabilities:
        -   Create and Upload documents
        -   Document Collaboration
        -   Version Management
        -   Download a document or any version as an attachment
-   **Version 1.2.3 - October 2023**

    Fixes.

-   **Version 2.3.1 - September 2023**
    -   Fixed:
        -   Copy Attachment to OneDrive using Path not working when Source Type is Site
        -   Create Upload Session action not working when Source Type is Site
-   **Version 1.2.2 - April 2023**

    Patch release for Microsoft OneDrive Spoke for Document Service Framework. This version includes an update to compatibility with Family releases.

-   **Version 1.2.1 - February 2023**

    Fixed: Patch release of Microsoft OneDrive Spoke for Document Service Framework. This release includes these fixes - removal of fix script and to leverage upgrade event instead of run\_once false, removal of deletion of the versions in reconnect subflow, and changes to improve install performance of the application.

-   **Version 1.1.0 - August 2022**

    New: Minor release for Microsoft OneDrive Spoke for Document Service Framework. This version adds a new sub-flow to update document content, recipient type as input to collaborate document, ability to fetch document id from existing OneDrive account, and enables uploading large file into upload file sub-flow.

-   **Version 1.0.5 - March 2022**

    Patch release of Microsoft OneDrive Spoke for Document Service Framework for IntegrationHub. This version includes updates to sub-flow outputs and a fix to an issue with the deletion of version records.

-   **Version 1.0.4 - July 2021**

    New: Patch release of the Microsoft OneDrive Spoke for Document Service Framework. This version adds Rome compatibility.

-   **Version 1.0.2 - February 2021**

    New: This spoke provides capabilities to support Multi Provider Document Services \(released as part of Quebec release\) for integrating with Microsoft OneDrive. It creates all the necessary tables and scripts, used by sub-flows in this spoke to seamlessly integrate.
