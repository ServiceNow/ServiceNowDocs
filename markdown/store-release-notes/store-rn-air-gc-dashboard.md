# GC Dashboard release notes

Version history for the ServiceNow® GC Dashboard application on the ServiceNow Store.

-   **Version 2.0.5 - December 2025**

    The General Counsel Dashboard consolidates data from Legal Service Delivery, including Request Management, Matter Management, Content Review, Simple Contracts, Conflict of Interest, Digital Forensics, and the Performance Analytics Content Pack, providing legal leaders with clear visibility into case workload, contract activity, and compliance posture.
