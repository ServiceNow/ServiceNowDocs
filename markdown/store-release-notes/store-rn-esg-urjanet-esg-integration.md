# Urjanet Operational Sustainability Management integration release notes

Version history for the Urjanet Operational Sustainability Management integration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.3.1 - June 2026 \(Australia\)**
    -   Changed: This release includes security enhancements that strengthen access controls across the application.
    -   Fixed: Query ACL failure on credential table — Resolved a query ACL failure on the ESG Urjanet credential table that was preventing proper data access.
-   **Version 22.0.0 - March 2026**

    Changed: Updated the application name to "Operational Sustainability Integration with Urjanet"

-   **Version 21.1.1 - December 2025 \(Zurich\)**

    Fixed: Strengthened security controls related to read-only fields

-   **Version 21.0.0 - August 2025**

    Changed: Refreshed demo data to better align with current feature capabilities and use case scenarios.

-   **Version 20.1.0 - May 2025**

    Fixed: Important security fixes.

-   **Version 16.0.4 - February 2023**
    -   Fixed:
        -   Translation support for missing form labels and messages fixed
        -   Client callable script includes are secured with appropriate ACLs
        -   Enabled retry policy for integration hub actions
        -   Domain path column name in staging tables corrected
-   **Version 15.0.2 - November 2022**

    Urjanet ESG integration helps provides capabilities to integrate utility data from Urjanet into ESG management, thus eliminating the need for manual data collection.
