# GRC: Business Continuity Management - Core release notes

Version history for the GRC: Business Continuity Management - Core on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.1.4 - April 2021**
    -   Fixed:
        -   Fixed cross-site scripting vulnerabilities with Plan Viewer
        -   Fixed PRB1453595 TinyMCE expired error message
        -   Fixed PRB1401160 page hanging when saving from Plan Viewer
-   **Version 2.0.1 - March 2021**

    New: Added support for technical impact assessment and RPO calculation

-   **Version 1.0.3 - October 2020**

    The Business Continuity Management - Core plugin provides the core tables, ACLs, and configurations required by each of the Business Continuity Management applications.
