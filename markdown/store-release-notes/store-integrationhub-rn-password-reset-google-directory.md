# Password Reset integration with Google Directory release notes

Version history for the Password Reset integration with Google Directory application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - June 2023**

    Fixed: Issue related to password delivery mechanism and force password change on first login attributes.

-   **Version 1.0.2 - April 2023**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.1 - February 2022**

    Changed: Patch version of Password Reset integration with Google Directory. This version updates the dependency to the latest Google Directory Spoke version.

-   **Version 1.0.0 - April 2021**
    -   Application Components:
        -   New Credential Store Type record for Google Directory
        -   Sample Credential Store record
        -   Sample Process records for self-service and service desk
        -   New subflows and action to support the application automation:
            -   Subflow - Password Connection Test
            -   Subflow - Password Lock State
            -   Subflow - Password Reset
            -   Subflow - Password Unlock
            -   Action - Get Configuration Variables for Google Directory Reset Process
    -   Password Reset Features:
        -   Enable or disable password reset
        -   Choose whether to provide a public URL for the process
        -   Choose whether to display a CAPTCHA
        -   Choose user identification type
        -   Define minimum number of required verifications
        -   Define if the user can choose from multiple verification methods
        -   Enable or disable a standalone account unlock process
        -   Either email or display a new autogenerated password
        -   Schedule enrollment reminders for users
