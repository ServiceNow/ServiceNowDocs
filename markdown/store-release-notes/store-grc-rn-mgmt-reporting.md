# GRC: Management Reporting release notes

Version history for the GRC: Management Reporting application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.3.3 - June 2026 \(Australia\)**
    -   Changed:
        -   Business domain handling moved to a separate applicationBusiness domain handling has been moved to a separate application to improve consistency across reporting configurations. Plugin dependencies and navigation filters have been updated to reflect the new structure.
        -   After upgrade:
            -   Reinstall the updated manifest files.
            -   If your instance includes Risk reporting configurations, verify that the Business Domain column is set to Risk.
-   **Version 18.1.0 - June 2024**

    Changed: The application has been modified to be non-licensable.

-   **Version 18.0.0 - February 2024**

    The GRC: Management Reporting application provides the reporting capabilities for importing GRC-risk information into Microsoft Word documents using the Microsoft Office 365 plugin.
