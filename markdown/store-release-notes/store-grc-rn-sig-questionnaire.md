# GRC: SIG Questionairre release notes

Version history for the GRC: SIG Questionnaire Integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 14.1.1 - March 2022**
    -   Changed: Deactivation of all SIG 2018 and 2019 questionnaires
    -   Fixed: Minor security issue
-   **Version 13.0.0 - September 2021**

    Fixed: Security fix in demo data

-   **Version 11.0.0 - October 2020**
    -   Fixed:
        -   Risk scoring for SIG 2018,2019 questionnaires
        -   Demo data for SIG
-   **Version 10.1.0 - June 2020**

    Fixed: Fixed report\_view ACLs for sensitive platform tables

-   **Version 10.0.0 - March 2020**

    This application has been recertified for Orlando. It is still compatible with New York.

-   **Version 9.0.1 - November 2019**

    This feature was updated for Madrid and is still compatible with New York.

-   **Version 6.0.1 - October 2018**
    -   The SIG Questionnaire application includes SIG questionnaire content for the following versions:
        -   SIG Full 2018
        -   SIG Lite 2018
        -   SIG Core 2018
    -   The SIG Questionnaire application renames the existing SIG questionnaire content for 2017 versions from:
        -   SIG Full to SIG Full 2017
        -   SIG Lite to and SIG Lite 2017
