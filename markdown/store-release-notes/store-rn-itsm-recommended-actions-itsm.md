# Recommended Actions for ITSM release notes

Version history for the Recommended Actions for ITSM application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.3.0 - March 2026**
    -   New: AI Gradient changes for Recommended Actions cards
    -   Fixed: Minor defects
-   **Version 3.2.0 - January 2026**

    New: AI search cards include relevancy score.

-   **Version 3.1.1 - December 2025**
    -   New:
        -   Genius Result card updates include:
            -   Expandable KB summaries
            -   KB sources as clickable links that open a detailed KB article view
            -   Button to attach and link a KB article to the incident comments
            -   Button to copy KB summary
-   **Version 3.0.2 - August 2025**

    Fixed minor issues.

-   **Version 3.0.0 - May 2025**
    -   Changed:
        -   Enabled full-view search
        -   Reduced loading times for opening and attaching KB articles
        -   UI button for attaching KB articles renamed "Attach and add link in comment"
-   **Version 2.1.0 - February 2025**

    Changed: Improved performance and optimizations, including dynamic fetching of catalog items and removal of redundant processes, to ensure a smoother user experience.

-   **Version 2.0.0 - November 2024**

    Changed: Improved support for catalog item guidance cards in Recommended actions AI search panel.

-   **Version 1.0.0 - August 2024**

    Recommended Actions for ITSM provides contextual &amp; actionable real-time recommendations and a AI based search panel for Service Operations Workspace along with workspace configurations that show recommendations in the contextual side panel and form fields.
