# ServiceNow Store - ServiceNow AI Platform Capabilities release notes

The ServiceNow Store contains ServiceNow AI Platform Capabilities products.

Visit the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website to view all the available apps and for information about submitting requests to the store.

## Read ServiceNow AI Platform Capabilities documentation in one place

The ServiceNow Store documentation for ServiceNow AI Platform Capabilities is now in the release family sections of the ServiceNow Product Documentation site. To read about ServiceNow AI Platform Capabilities features from the ServiceNow Store and the family releases, navigate to the [Product Documentation site homepage](https://docs.servicenow.com) and select your release family.
