# Claims for reporting release notes

Version history for the Claims for reporting application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.3.1 - June 2026 \(Australia\)**

    Changed: This release includes security enhancements that strengthen access controls across the application.

-   **Version 21.1.0 - December 2025 \(Zurich\)**

    Fixed: Minor UI changes to improve user experience.

-   **Version 21.0.1 - August 2025**

    Claims for reporting help companies create a repository of narratives for reporting. By enabling reusability, enhancing discoverability, and promoting consistency, organizations can develop disclosures that build trust, drive engagement, and tell a cohesive story.
