# Security Center release notes

Version history for the Security Center on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.3.1 - April 2026**

    Defect fixes.

-   **Version 3.3.0 - March 2026**
    -   New:
        -   Security Center hardening baseline version 8.
        -   Accept risk for hardening settings.
    -   Fixed: Defect fixes.
-   **Version 3.2.3 - February 2026**
    -   New:
        -   Security Center hardening baseline version 7.
        -   New Access management console, giving users ability to look at access misconfigurations.
    -   Changed:
        -   Updated integration with Identity and Access Management \(IAM\) tools featured on Security Center.  Use the updated sections in the Security Center home to access IAM tools such as Machine Identity Manager, Access Analyzer, and Scripting Governance, as well as new Security Metrics for IAM.
    -   Fixed: Defect fixes.
-   **Version 3.2.2 - December 2025**
    -   New:
        -   Security Center hardening baseline version 7.
        -   New Access management console, giving users ability to look at access misconfigurations.
    -   Changed: Updated integration with Identity and Access Management \(IAM\) tools featured on Security Center.  Use the updated sections in the Security Center home to access IAM tools such as Machine Identity Manager, Access Analyzer, and Scripting Governance, as well as new Security Metrics for IAM.
    -   Fixed: bug fixes.
-   **Version 3.1.2 - August 2025**
    -   Improved: New Identity and Access Management \(IAM\) tools featured on Security Center. Use the new IAM section in Security Center to access IAM tools such as Machine Identity manager, Access Analyzer, and Scripting Governance.
    -   Updated: Bug fixes
-   **Version 3.0.5 - June 2025**
    -   New:
        -   Security Tasks allows administrators to create, prioritize, delegate and monitor progress for platform security tasks, in a unified dashboard, that are either automatically generated by the platform or manually created.
        -   Embedded videos to provide an easier way to learn how to best leverage key tools and navigate the user experiences.
    -   Improved:
        -   Improved tools overview pages for a consistent experience and easy access to common filtering options.
        -   Support Next Experience UI themes which allows to configure the variables for colors, shapes, fonts, and other aspects of the user experience.
    -   Updated:
        -   Updates to the Security Hardening tool to the latest Instance Security Hardening Settings V6.
        -   Bug fixes.
-   **Version 2.1.2 - June 2025**

    Bug fixes.

-   **Version 1.6.3 - June 2025**

    Only minor bugs were addressed in this release.

-   **Version 3.0.2 - May 2025**
    -   New:
        -   Security Tasks allows administrators to create, prioritize, delegate and monitor progress for platform security tasks, in a unified dashboard, that are either automatically generated by the platform or manually created.
        -   Embedded videos to provide an easier way to learn how to best leverage key tools and navigate the user experiences.
    -   Improved:
        -   Improved tools overview pages for a consistent experience and easy access to common filtering options.
        -   Support Next Experience UI themes which allows to configure the variables for colors, shapes, fonts, and other aspects of the user experience.
    -   Updated: Updates to the Security Hardening tool to the latest Instance Security Hardening Settings V6.
-   **Version 2.1.1 - April 2025**

    Bug fixes.

-   **Version 2.1.0 - February 2025**
    -   Update the Scanner tool's Auditor suite with additional scan checks.
    -   Adding 6 new Best Practices.
    -   Bug fixes and improvements.
-   **Version 1.6.2 - January 2025**

    Only minor bugs were addressed in this release.

-   **Version 2.0.4 - December 2024**

    Fix upgrade issue introduced from new fixscript.

-   **Version 2.0.1 - November 2024**
    -   Changed:
        -   Improved navigation and user experience:
            -   New Home page integrated in the new Admin Center navigation
            -   Three new pages that provides summary information for the three key security tasks with their own navigation to relevant tools:
                -   Security Posture
                -   Security Configuration
                -   Security Monitoring
        -   Added new security scan suite Security Controls Activation
            -   Review security controls that were recently released
        -   Updated Security Hardening tool with the latest Instance Security Hardening Settings baseline v5.0
-   **Version 1.6.1 - November 2024**
    -   New features:
        -   Security Banner Announcements - Helps admin to view urgent security issues sent from Servicenow.
        -   Multi-instance Security posture dashboard &amp; Single-instance Security posture dashboard: Helps admin to view their production environment and be able to view security posture information across that production instance and its subprod instances to quickly and easily understand their environment overall security posture.
        -   Security Event Notifications - Helps admin to get notified when an indicative of a potential risky security events occurs.
-   **Version 1.6.0 - August 2024**
    -   New:
        -   Security Banner Announcements - Helps admin to view urgent security issues sent from ServiceNow.
        -   Multi-instance Security posture dashboard &amp; Single-instance Security posture dashboard: Helps admin to view their production environment and be able to view security posture information across that production instance and its subprod instances to quickly and easily understand their environment overall security posture.
        -   Security Event Notifications - Helps admin to get notified when an indicative of a potential risky security events occurs.
-   **Version 1.5.4 - July 2024**
    -   New Features:
        -   Platform Security Best Practices manager V1 - Helps admin complete their configuration tasks effectively and efficiently by providing them a tool that organizes the many tasks involved in instance security configuration and allows them to keep track of the progress.
        -   Implement AppSec Security Hardening Baseline V4.
        -   Navigation improvements between Admin Center and Security Center.
    -   System requirements: Min. supported version: Washington DC
-   **Version 1.5.3 - May 2024**
    -   New Features:
        -   Platform Security Best Practices manager V1 - Helps admin complete their configuration tasks effectively and efficiently by providing them a tool that organizes the many tasks involved in instance security configuration and allows them to keep track of the progress.
        -   Implement AppSec Security Hardening Baseline V4.
        -   Navigation improvements between Admin Center and Security Center.
-   **Version 1.4.0 - February 2024**

    Minor fixes.

-   **Version 1.3.4 - January 2024**
    -   Added support for the Washington family release.
    -   Some minor fixes.
-   **Version 1.3.2 - November 2023**
    -   New: Added 42 new settings
    -   Changed:
        -   CVSS score change for following settings:
            -   Enable Account Recovery
            -   Enable Updated Version of Multi SSO Plugin
        -   Settings compliance status change for following settings:
            -   Set Minimal Password Length
            -   Enable Email OTP for Multi-Factor Authentication
            -   Disallow Infected Files Download
    -   Fixed: Pre-existing issues within security center
-   **Version 1.2.0 - August 2023**

    New: Metrics Tool added 'Adaptive Authentication' metrics category and metrics.

-   **Version 1.1.11 - July 2023**

    Fixed:

    1.  Removed 'Block Direct Inserts to MID ECC Queue for Code Signing Requirement' setting.
    2.  Fixed Postgres DB syntax error warnings.
-   **Version 1.1.9 - May 2023**

    ServiceNow Security Center is an application that consists of a set of purpose-built tools designed to help organizations maintain the security of their ServiceNow deployments. Using Security Center, organizations can improve their security posture, strengthen their compliance levels, and do so with a seamless user experience.
