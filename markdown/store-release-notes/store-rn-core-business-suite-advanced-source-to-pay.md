# Core Business Suite Advanced for Source to Pay release notes

Version history for the ServiceNow® Core Business Suite Advanced for Source to Pay application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.5 - June 2026**

    Core Business Suite Advanced for Source to Pay is now available for direct installation from the Product Hub. This enables one-click installation of all CBS Advanced S2P applications.

-   **Version 3.0.0 - April 2026**

    Core Business Suite Advanced for Source to Pay extends the capabilities of the Foundation version to support more complex enterprise workflows. It provides advanced AI capabilities for the Source to Pay department.
