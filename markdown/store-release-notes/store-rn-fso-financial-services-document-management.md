# Financial Services Document Management release notes

Version history for the Financial Services Operations Financial Services Document Management on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.3 - May 2021**

    Changed: Financial Services Core application dependency version has been updated

-   **Version 1.1.2 - April 2021**
    -   New:
        -   Added new roles from Insurance Policy Operations to document agent
        -   Added decision input for the insurance policy service cases
    -   Changed: Updated scripted ACL to read service definitions
-   **Version 1.1.1 - March 2021**

    Changed: Moved plugin dependencies to Financial Services Operations Core

-   **Version 1.1.0 - December 2020**
    -   New:
        -   Introduced inbound and outbound documents for the Case
        -   Decision engine to define inbound and outbound documents for the Case
        -   Capability to verify, defer, or seek exception on submission of an inbound document with the corresponding workflow
-   **Version 1.0.4 - September 2020**

    Provides capabilities to manage middle and back office tasks that are assigned to the Document Services team across any Financial Services workflow.
