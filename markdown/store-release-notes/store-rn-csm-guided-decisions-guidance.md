# Guided Decisions - Guidance release notes

Version history for the Guided Decisions - Guidance applications on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 24.0.1 - November 2022**
    -   Changes to the guidance experience to show error messages and notifications in case multiple users are acting on the same guidance
    -   Changes to show error messages if guidance fails during execution
-   **Version 23.0.0 - August 2022**

    A guidance is a flexible entity for defining an action \(using a subflow\) and the information pertaining to that action. A guidance is used in a Guided Decision or a Recommended Action that enables the agent to execute the action.
