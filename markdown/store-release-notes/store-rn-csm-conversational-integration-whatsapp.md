# Conversational Integration with WhatsApp \(powered by Twilio\) release notes

Version history for the Conversational Integration with WhatsApp \(powered by Twilio\) on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.12 - March 2026**

    Fixed: All system messages are now sent to the user \(e.g. AI disclaimer\).

-   **Version 2.0.11 - December 2025**

    Minor bug fixes

-   **Version 2.0.10 - May 2025**

    Minor bug fixes.

-   **Version 2.0.8 - November 2024**
    -   Fixed:
        -   AI Search cosmetic issues
        -   Agents downloading images
-   **Version 2.0.6 - August 2024**

    Fixed: Support for AI Search cards.

-   **Version 2.0.3 - August 2021**
    -   New:
        -   Agent-initiated conversations
        -   System-initiated conversations
        -   Rich messaging including support for emojis and formatting
        -   Consumer-grade search experience via AI Search applications.
-   **Version 1.0.1 - February 2021**

    New: Enable brands to engage with their customers using the WhatsApp messaging app. The app requires subscription to the Twilio services.
