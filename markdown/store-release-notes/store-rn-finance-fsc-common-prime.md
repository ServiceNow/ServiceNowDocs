# FSC Common - Prime release notes

Version history for the ServiceNow® FSC Common - Prime application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.0 - April 2026**

    The ServiceNow FSC Common Prime application delivers the shared suite of advanced AI-powered skills, agents, and orchestration components that underpin Accounts Payable Operations, Sourcing and Procurement Operations, and Supplier Lifecycle Operations. From intelligent document processing and cross-workflow routing to agentic capabilities that resolve exceptions autonomously — FSC Common Prime provides the platform-level building blocks that accelerate end-to-end source-to-pay operations while keeping humans in control at every step.
