# Workforce Optimization Configurable Workspace Core release notes

Version history for the Workforce Optimization Configurable Workspace Core on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

    -   Fixed minor defects

    The ServiceNow Workforce Optimization application enables you to efficiently route work to your team, manage your team's skills and schedules, and monitor their performance. Workforce Optimization was enhanced and updated in the release. and Fixed minor defects,
