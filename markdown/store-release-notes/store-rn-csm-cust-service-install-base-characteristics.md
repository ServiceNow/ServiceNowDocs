# Customer Service Install Base Characteristics release notes

Version history for the Customer Service Install Base Characteristics application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.4.0 - June 2026**

    Changed: Enhanced security by strengthening ACLs on the Install Base characteristics table.

-   **Version 2.3.0 - March 2026**

    Changed: Minor enhancements.

-   **Version 2.2.1 - December 2025**

    Changed: Minor bug fixes

-   **Version 2.2.0 - November 2025**

    Fixed: Minor enhancements - Added a esm\_agent role to read the install base characteristics records.

-   **Version 2.1.0 - May 2025**

    Fixed: Minor improvements.

-   **Version 2.0.1 - February 2025**

    Install base characteristics enables customers to capture characteristics for their install bases.
