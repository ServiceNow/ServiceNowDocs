# ITOM Mobile Agent release notes

Version history for the ITOM Mobile Agent® application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.0 - May 2025**

    ITOM Mobile Agent on the Now Agent Mobile application allows customers with Service Reliability Management access to respond to alerts and incidents and manage their on-call schedules on the go.
