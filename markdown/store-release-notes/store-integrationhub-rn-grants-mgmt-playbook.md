# Grants Management release notes

Version history for the Grants Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.31.1 - June 2026**
    -   Rolling grant application submissions and approvals let you submit funding or decline recommendations for any scored proposal subset, enabling incremental funding decisions without waiting for the entire proposal portfolio to complete review.
    -   Release result letters per proposal for rolling grant approval scenarios in Grants Management.
    -   Defect fixes
-   **Version 1.22.0 - May 2026**
    -   Defect fixes
    -   Updated missing translations for UI text
-   **Version 1.21.5 - April 2026**

    Defect fixes for localization, functional, and accessibility issues.

-   **Version 1.21.0 - February 2026**
    -   V1 Enhancements
        -   Applicant Experience Enhancements
        -   Program Manager Productivity
        -   Merit Review &amp; Scoring Improvements
        -   Flexible Funding Workflows
        -   Document Management Enhancements
        -   UI and System Updates
    -   Funding Programs: New top‑level container to align grants to legislated initiatives, centralize budgets, and report across multiple programs
-   **Version 1.2.0 - August 2025**
    -   Reviewer Service Portal: A dedicated interface for reviewers to score, track, and manage applications.
    -   Grants Management Portal &gt; Results Tab: Applicants can now view and download decision letters and merit summaries, and accept or decline awards directly in the portal.
    -   Playbooks: Visual, step-by-step setup guidance for Grant Program Managers.
        -   New Phases Added:
            -   Evaluate: Conduct merit review activities and build funding proposals.
            -   Decision: Communicate decisions using result letter templates, including merit review summaries and decision reasons.
        -   New Activities in Setup Playbook:
            -   Choosing reviewer groups.
            -   Selecting result letter templates.
-   **Version 1.0.6 - August 2025**

    Fixed: Coral Theme Uptake Fixes for Custom Activity UIs.

-   **Version 1.0.2 - May 2025**

    The Public Sector Digital Services \(PSDS\) Grants Management Playbook enables government agencies to simplify and streamline grant program set-up, management, application intake, and screening. This application helps alleviate many of the systemic challenges government grant programs face today, including outdated grant-making systems, data fragmentation, workflow bottlenecks, and end-to-end visibility.
