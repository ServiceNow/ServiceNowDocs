# Microsoft Endpoint Configuration Manager spoke release notes

Version history for the Microsoft Endpoint Configuration Manager spoke integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.10.2 - June 2026**

    Non Glide Cobalt Raven ACLs added.

-   **Version 1.9.0 - April 2026**

    New: Added actions and subflows for CSD Agents

-   **Version 1.8.1 - June 2025**

    Fixed: A subflow error.

-   **Version 1.8.0 - October 2024**

    Fix for memory exception error that was occurring when pulling large amount of records from Discovery Flow.

-   **Version 1.7.5 - September 2024**

    Fix: Out-of-memory exception for the Discovery flows.

-   **Version 1.7.4 - September 2023**
    -   Fixed:
        -   The license requirement
        -   Additional properties not getting populated in Look up Device Collections and Look up User Collections
-   **Version 1.7.1 - March 2023**

    Patch release. Adds dependency support for available actions.

-   **Version 2.0.1 - February 2023**

    New adapter released to support Microsoft Endpoint Configuration Manager \(MECM\).

-   **Version 1.7.0 - January 2023**

    Minor release of Microsoft Endpoint Configuration Manager Spoke for Integration Hub. This version adds four new actions - Run MECM Script, Look up MECM Script by Name, Look up MECM Script Response by Operation ID, Look up MECM Script Execution Response

-   **Version 1.6.0 - September 2022**

    Minor release of Microsoft Endpoint Configuration Manager Spoke for Integration Hub. This version adds three new actions - Look up Device by ID, Run CMPivot Query, Look up CMPivot Response by Operation ID.

-   **Version 1.5.0 - July 2022**

    Changed: Minor release of Microsoft Endpoint Configuration Manager spoke for IntegrationHub. This version includes a few actions migrated to REST API-based communication with the MECM server. This migration helps to minimize memory-related problems through paginated results and shorter response times.

-   **Version 1.0.1 - February 2022**

    Provides actions to automate the management of Devices and Deployments using Microsoft Endpoint Configuration Manager.
