# Advanced Recommended actions for ITSM release notes

Version history for the ITSM Recommended Actions for ITSM - Advanced application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 8.1.0 - September 2025**

    Changed: Ensured that updating the incident in 'Update incident attributes' flow works for authorized roles.

-   **Version 8.0.2 - August 2025**

    Fixed minor issues.

-   **Version 8.0.1 - May 2025**
    -   New:
        -   Using Task Intelligence:
            -   Recommends similar incidents to propose a major incident
            -   Recommends similar active major incidents which the current incident can be linked to
    -   Changed:
        -   Enabled full-view search
        -   Reduced loading times for opening and attaching KB articles
        -   UI button for attaching KB articles renamed "Attach and add link in comment"
-   **Version 7.0.0 - November 2024**
    -   New:
        -   New Recommended Action and scripted resource generator to surface similar Problems on Incident
        -   New Recommended Action and scripted resource generator to surface similar Change Requests on Incident
-   **Version 5.1.3 - August 2024**

    No changes. Bumped up for dependency change.

-   **Version 5.0.0 - May 2024**

    Minor fixes.

-   **Version 4.2.0 - February 2024**
    -   Changed:
        -   Now view one or more recommendations within the field on SOW
        -   Now Genius results and AI search results for KB articles can be seen in recommendations panel.
-   **Version 4.1.0 - November 2023**

    No changes. Release version due to a dependent app version upgrade.

-   **Version 4.0.0 - November 2023**
    -   New:
        -   Changes in Recommended Actions
        -   Changes in Task Intelligence Admin Console
-   **Version 3.1.0 - September 2023**

    Changed: Version number update for dependent app version

-   **Version 3.0.0 - August 2023**

    Recommended Actions for ITSM - Advanced app provides contextual and actionable real-time recommendations for Service Operations Workspace along with workspace configurations that show recommendations in the contextual side panel and form fields.
