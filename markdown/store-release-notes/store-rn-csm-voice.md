# ServiceNow Voice for CSM release notes

Version history for the ServiceNow Voice for CSM application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.8.1 - February 2025**

    Fix in this Release: Internal security fix.

-   **Version 3.4.0 - November 2023**

    Changed: Related to SRP uptake for SOW.

-   **Version 3.0.2 - August 2023**
    -   Changes include:
        -   Supporting Outbound Calls \(manual dial\)
        -   Show conversation panel on workspace to show Real Time Transcription.
        -   Supporting Real Time Transcription for call transfers.
-   **Version 2.0.1 - February 2023**
    -   New: Real-time transcription capability
    -   Changed: Phone log data for voice has now been consolidated to interaction
-   **Version 1.2.1 - December 2022**

    Minor fixes.

-   **Version 1.1.1 - May 2022**

    New: Voice for CSM is now available in the Configurable Workspace so that you can see voice transcipts and sentiments for calls.

-   **Version 1.1.0 - February 2022**

    Changed: Product name changed from Cloud Call Center to ServiceNow Voice.

-   **Version 1.0.14 - May 2021**

    Fixed: Incoming calls from a customer are now assigned to the contact field instead of the consumer field of an interaction record.
