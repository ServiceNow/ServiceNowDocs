# AI Discovery release notes

Version history for the ServiceNow® AI Discovery application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.7.0 - June 2026**
    -   New integrations released:
        -   Enables discovery of AI assets via integrations with: AWS Bedrock, SageMaker, Agent Core, Azure Foundry, Copilot Studio, GCP Vertex AI, Salesforce, n8n, Databricks, Snowflake, Hugging Face, LangGraph
        -   Uses Service Graph Connectors framework for scalable integrations
-   **Version 2.6.0 - May 2026**
    -   Enterprise AI discovery systematically identifies, catalogs\(meta-data\) AI agent related assets like models, tools and prompts across an organization’s AI landscape.
    -   Enterprises are increasingly adopting AIsolutions across various platforms, includingAWS Bedrock, Google Cloud Platform \(GCP\),and Microsoft Azure and other SaaSplatforms like Databricks, Salesforce etc. ​AI assets are scattered across multiple hyper scalers, leading to a lack of centralized visibility and control.
    -   The administrator can configure discovery integrations with hyper scaler and SaaSenvironments to offer a unified overview of enterprise AI assets and their connections. This enables the AI steward to monitor AI systems such as AIAgents being created and deployed within the enterprise ecosystem, as well as track associated assets such as models, prompts, and tool details.​
