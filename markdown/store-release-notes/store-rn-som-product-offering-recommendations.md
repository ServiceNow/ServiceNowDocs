# Product Offering Recommendations release notes

Version history for the Product Offering Recommendations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - June 2026**

    Fixed: Security directives addressed.

-   **Version 1.2.0 - December 2025**

    Fixed: Bug on non-display of offer recommendations.

-   **Version 1.1.0 - May 2025**

    Fixed: Bug fix around Deny ACL.

-   **Version 1.0.0 - February 2025**

    Product Recommendations provide agents with an efficient way to find complementary products that augment or add value to the products in quotes, or what the customer owns. To make this process more efficient for agents, a list of recommended product offerings identifies additional products that agents can upsell or cross-sell based on items in quotes.
