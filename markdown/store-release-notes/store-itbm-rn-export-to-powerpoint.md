# Export to PowerPoint release notes

Version history for the Export to PowerPoint application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.4.0 - June 2026**

    Fixed: ACL handling with Export to PowerPoint.

-   **Version 2.3.0 - December 2025**
    -   Fixed:
        -   Resolved translation defects within PowerPoint add-in.
        -   Rectified parsed log generation errors when templates are created by PPT user role.
-   **Version 2.2.0 - August 2024**

    New: Export Roadmap from Strategic Planning Workspace.

-   **Version 2.1.1 - November 2023**

    Fixes related to Repeater token.

-   **Version 2.0.0 - November 2022**

    New: User can export data in certain colors and shapes to the powerpoint based on conditions in formatter mapping.

-   **Version 1.0.1 - September 2022**

    Export to PowerPoint is the base application to export data from ServiceNow.
