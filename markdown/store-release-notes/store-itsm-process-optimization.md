# ITSM Process Mining Content Pack release notes

Version history for the ITSM Process Mining Content Pack on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.0 - March 2026**

    Minor fixes.

-   **Version 1.9.1 - August 2025**

    Minor bug fixes.

-   **Version 1.9.0 - April 2025**

    Minor bug fixes.

-   **Version 1.7.9 - February 2025**

    Minor bug fix.

-   **Version 1.7.0 - February 2024**

    Minor fixes.

-   **Version 1.5.2 - November 2023**

    Changed: Onboarded translations.

-   **Version 1.3.10 - September 2023**

    Fixed: The "No Transitions Found" error on the bottleneck analysis after mining.

-   **Version 1.6.0 - August 2023**

    Changed: The name "Process Optimization" has been changed to "Process Mining."

-   **Version 1.3.9 - July 2023**

    Fixed: Fixed the 'No Transitions Found' error on the bottleneck analysis after mining.

-   **Version 1.5.1 - May 2023**

    Fixed: Retired all summary and insights variants.

-   **Version 1.5.0 - February 2023**

    New: Created default dashboard for ITSM Process Mining.

-   **Version 1.3.7 - November 2022**

    Fixed: Fixed some issues in the ITSM Process Mining Content Pack. Uptake of new tables and UI fields from Process Mining.

-   **Version 1.3.5 - March 2022**

    Minor fixes.

-   **Version 1.2.0 - July 2021**
    -   New:
        -   Ability to configure cluster analysis for ITSM work items.
        -   Ability to configure KPIs and insights for ITSM work items in the Process Mining dashboard.
-   **Version 1.1.0 - January 2021**

    New: A new capability introduced in Quebec with ITSM Enterprise SKU.
