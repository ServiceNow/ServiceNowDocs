# Assignment Workbench release notes

Version history for the ServiceNow® Assignment Workbench application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.1.2 - June 2026**
    -   Fixed padding issues
    -   WCAG 2.2 conformance fixes for Assignment Workbench
    -   Assignment Workbench UI &amp; accessibility fixes
-   **Version 1.0.1 - March 2026**
    -   Assignment Workbench is now delivered as a ServiceNow Store application, enabling modular deployment, simplified upgrades, and improved lifecycle management for Customer self-service and omnichannel engagement solutions.
    -   This release migrates the previously global-scoped Assignment Workbench and Customer Service Social Media integration into a fully managed Store application. The migration ensures feature parity while aligning with modern ServiceNow application architecture standards.
    -   By moving to a Store-based model, customers benefit from:
        -   Simplified upgrade paths
        -   Improved maintainability
        -   Better scalability
        -   Enhanced modular deployment
        -   Future extensibility for omnichannel capabilities
    -   All existing Assignment Workbench capabilities and Customer Service Social Media integration features are preserved in this release.
