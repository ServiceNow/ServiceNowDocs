# Now Assist for Enterprise Architecture \(EA\) release notes

Version history for the Now Assist for Enterprise Architecture \(EA\) application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 7.3.1 - June 2026**
    -   New:
        -   Use the Enterprise Architecture query agent to ask natural language questions about your enterprise architecture portfolio. The agent supports multi-condition queries, quarterly trend comparisons, and impact analysis for scenarios such as application decommissioning, and suggests related questions after answering.
        -   Generate a business process map \(BPM\) diagram in the Enterprise Modeling &amp; Visualization by uploading an image of an existing process diagram.
-   **Version 7.1.0 - March 2026**
    -   New: Added support for AI Data Explorer and Query Generation skills to enable natural language querying and analysis of Enterprise Architecture Workspace dashboard widgets.
    -   Changed:
        -   Business application insights are now context-aware, automatically including relevant information from the source page from where you are generating insights.
        -   Diagram change analysis now allows comparison between any two diagram versions, not just adjacent versions.
-   **Version 7.0.3 - January 2026**

    Fixed: Enhanced AI Skill Stability - Improved the reliability of AI-powered features to ensure a smoother experience.

-   **Version 7.0.1 - December 2025**
    -   New:
        -   Added the diagram change analysis Now Assist skill to compare different versions of a diagram and summarize the changes
        -   Added the refine text Now Assist skill that helps to elaborate or shorten the text context of a record field
    -   Changed:
        -   View the business application source fields on the skills configuration page, based on which the business application insights are generated
        -   Role restrictions added to default roles associated with all Now Assist for EA skills.
-   **Version 6.0.1 - October 2025**

    Changed: Updated LLM prompts for ADR Doc Summarization and Actions skill.

-   **Version 1.1.2 - October 2025**

    New:

    -   Implemented security in Now Assist AI agents with access control lists \(ACLs\).
    Changed:

    -   Updated LLM prompts for ADR Doc Summarization and Actions skill.
    Fixed:

    -   Issue in fetching the related records for a business application, while creating a diagram using the Generate enterprise architecture diagram agent.
-   **Version 6.0.0 - September 2025**
    -   New:
        -   Generate insights for business applications using the Now Assist skill, Business Application Insights
        -   Refine Architectural Decision Record \(ADR\) summaries by providing prompts to generate more targeted, relevant, and actionable summaries
-   **Version 3.0.0 - August 2025 \(Yokohama\)**
    -   New:
        -   Support for third-party small or large language model with any Now Assist for Enterprise Architecture \(EA\) skill or AI agent. The supported LLMs are:
            -   Google Gemini
            -   AWS Claude
            -   Azure OpenAI
-   **Version 1.1.1 - June 2025 \(Xanadu\)**

    New: Support for Now LLM.

-   **Version 1.4.0 - May 2025**

    New: Support for Now LLM.

-   **Version 1.1.0 - March 2025**

    Generate and summarize Enterprise Modeling and Visualization diagrams for business applications hierarchy, using the Enterprise architecture diagrammer AI agent.

-   **Version 1.0.0 - February 2025**

    Now Assist for Enterprise Architecture \(EA\) drives customer value faster by utilizing the power of Generative AI \(GenAI\) to summarize, shorten, or elaborate architectural decision records \(ADR\) in the Enterprise Architecture Workspace.
