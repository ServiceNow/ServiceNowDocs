# Header App Shell release notes

Version history for the Header App Shell application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 23.0.0 - August 2023**
    -   Fixed
        -   Various fixes to address minor issues.
-   **Version 23.0.0 - June 2023**

    Fixed: The app shell is not loading due to the base experience shell being empty.

-   **Version 22.1.0 - November 2022**

    New: Added support for internationalization and localization

-   **Version 22.0.6 - September 2022**

    Fix the layout definitions and clean up unused slots - header, openframe, wsSidepanel that use to display the sn-canvas-viewport to render the notification tray.

-   **Version 22.0.0 - August 2022**

    A minimalistic app shell for experiences that don't need global navigation menus or tools. This is typically used for builder or wizard type experiences.
