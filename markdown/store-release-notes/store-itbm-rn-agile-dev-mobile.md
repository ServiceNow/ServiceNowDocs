# Agile Development v2 release notes

Version history for the IT Business Management Agile Development v2 on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - February 2023**
    -   New: Added support for Next Experience theme
    -   Fixed: Minor bug fixes
-   **Version 1.0.4 - February 2022**

    Fixed: Bug fixes.

-   **Version 1.0.3 - July 2021**

    Fixed: Bug fixes.

-   **Version 1.0.2 - November 2020**

    Fixed: Bug fixes.

-   **Version 1.0.1 - March 2020**

    The new Agile Development v2 mobile app delivers an out-of-the-box mobile sprint tracking experience for team members, scrum masters, and product owners. This app helps you track and update stories and scrum tasks of the current sprint.
