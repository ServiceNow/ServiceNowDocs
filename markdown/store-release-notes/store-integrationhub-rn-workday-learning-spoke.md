# Workday Learning spoke release notes

Version history for the IntegratioHub Workday Learning Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.4 - April 2024**

    Fixed: Invalid references available in pill expressions of REST-based actions.

-   **Version 1.1.1 - September 2023**
    -   Fixed:
        -   Configuration template functionality
        -   'licensable' field set to be true in plugin.xml
-   **Version 1.1.0 - August 2023**
    -   Added OAuth option
    -   Updated and added more course management and skill management actions
    -   Updated security
-   **Version 1.0.2 - April 2023**

    Removed the attached record in WS- Security Policy Username Profile.

-   **Version 1.0.1 - November 2022**
    -   ATTENTION: Use of this product requires a license for either IntegrationHub Enterprise, HR Service Delivery Enterprise, or Employee Growth and Development.
    -   Provides actions to integrate Workday Learning courses and user activity with ServiceNow.
