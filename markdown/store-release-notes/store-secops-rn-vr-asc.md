# Azure Security Center Integration for Security Operations release notes

Version history for the Azure Security Center Integration for Security Operations integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.8.0 - December 2025**

    Fixed: Minor fixes.

-   **Version 2.2.2 - November 2022**
    -   New:
        -   Added support for management groups.
        -   Added support to leave subscription group/management group blank - through which it would pull in data related to all subscription groups/management groups.
    -   Changed: Updated the cloud lookup rules to make them extensible to support various resource types.
-   **Version 2.1.1 - August 2022**

    Removed: Resources Integration has been removed. The assessments integration will now import resources too. This helps to improve the performance.

-   **Version 2.0.0 - May 2022**

    Azure Security Center Integration for Security Operations integrates with Azure Security Center \(ASC\) to determine if the cloud assets are configured according to security standards. It also identifies the resources that must be fixed to protect your cloud resources. These resources are prioritized and a list of recommendations are provided.
