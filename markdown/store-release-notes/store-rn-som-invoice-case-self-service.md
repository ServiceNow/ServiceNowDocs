# Invoice Case Self-Service release notes

Version history for the ServiceNow® Invoice Case Self-Service application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.3 - May 2026**

    Changed: Updated the position of field labels and added new fields to the list view based on the request source.

-   **Version 1.0.2 - March 2026**

    Provide secure and seamless access to invoice case management on the ServiceNow Business Portal, empowering B2B customers to easily view, track, and manage invoice‑related cases.
