# Field Service Mobile release notes

Version history for the ServiceNow® Field Service Mobile application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 29.2.1 - June 2026**

    New Features: Added mobile notifications for Shift assignment/Agent sign-up requests

-   **Version 29.1.5 - April 2026**
    -   Fixed:
        -   Fixed the issue that agent personal events were not visible in Myschedule calendar screen.
        -   In the Remove part flow, show assets only if the work order task has a location and that location matches with Assets in the In use state.
-   **Version 29.1.3 - March 2026**

    Manages the Field Service mobile components on an iOS or an Android mobile device. Field service agents can execute work order tasks, manage assets, and close work order tasks on their mobile device in online or offline mode.
