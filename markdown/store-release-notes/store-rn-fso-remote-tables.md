# Financial Services Remote Tables release notes

Version history for the Financial Services Remote Tables application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - March 2026**

    Updated: Replaced direct references to the admin role with feature-specific granular admin roles to align with least-privilege access principles.

-   **Version 1.4.0 - August 2025**

    New: Added report view ACL for Remote transaction and account

-   **Version 1.3.1 - November 2024**

    New: Added tables for Remote Financial Account and Remote Financial Transaction.

-   **Version 1.2.5 - September 2024**

    Fixed" FSO Remote Lookup was not working on Voltron. Added app route for remote lookup to work on Votron.

-   **Version 1.2.2 - June 2024**

    Fixed scoped admin issue to provide access to system admin.

-   **Version 1.2.1 - May 2024**

    Changed: Modified configuration for localization support.

-   **Version 1.2.0 - February 2024**
    -   Changed:
        -   Migrated the UIB lookup page to layout 3.0
        -   Upgraded snc-app-parent version to 5.1.0.16
        -   Modified configuration for localization support
-   **Version 1.1.2 - May 2023**
    -   New:
        -   Added commercial real estate loan viewer role.
        -   Added localization changes.
    -   Removed:
        -   Removed snc\_internal role from ACLs.
        -   Removed ACLs that does not have any roles assigned.
-   **Version 1.1.1 - November 2022**

    Fixed: Security updates

-   **Version 1.1.0 - August 2022**

    New: Added new lookup tables for individual life, group life insurance
