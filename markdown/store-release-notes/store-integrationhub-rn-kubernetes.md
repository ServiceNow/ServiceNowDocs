# Kubernetes spoke release notes

Version history for the Integration Hub Kubernetes spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - June 2026**

    Non Glide Cobalt Raven ACLs added.

-   **Version 1.3.0 - September 2025**

    Added required ACLs or Roles as per the AI Security Directive guidelines.

-   **Version 1.2.1 - August 2025**

    Fixed: Kubernetes Spoke AI Agents : Agents Renaming and Output transformation strategy set to None

-   **Version 1.2.0 - March 2025**

    Minor release with new AI agents.

-   **Version 1.1.2 - February 2024**

    Fix for Kubernetes: Inspect POD Action getting a failed error.

-   **Version 1.1.1 - April 2023**
    -   New: Authentication template for easier spoke setup
    -   Fixed: Improved installation performance of spoke
-   **Version 1.0.3 - August 2019**

    Licensing patch for the Kubernetes spoke on Integration Hub

-   **Version 1.0.2 - March 2019**

    Provides Actions to automate your DevOps processes with Kubernetes
