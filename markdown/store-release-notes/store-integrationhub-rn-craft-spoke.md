# Craft Spoke release notes

Version history for the IntegrationHub Craft Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - June 2026**

    Security patch on non-glide ACLs

-   **Version 1.0.0 - November 2024**
    -   Craft Spoke provides a list of actions that interact with Craft GraphQL APIs. It provides the foundation to synchronize ServiceNow and SAP S/4 HANA, such as looking up alerts, companies, and company details.
    -   Given that Craft's GraphQL APIs are highly customizable, Craft Spoke showcases the design pattern from a common GraphQL query into a Spoke Action, where customers can further extend and tailor the Action per their own integration requirements.
    -   With ServiceNow and Craft integration, risk management possibilities are endless.
