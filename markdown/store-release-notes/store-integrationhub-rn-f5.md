# F5 BIG-IP spoke release notes

Version history for the Integration Hub F5 BIG-IP spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - September 2025**

    Added required ACLs or Roles as per the AI Security Directive guidelines.

-   **Version 1.2.1 - August 2025**

    Fixed: F5 BIG IP Spoke AI Agents : Agents Renaming and Output transformation strategy set to None

-   **Version 1.2.0 - March 2025**

    Minor release with new AI agents.

-   **Version 1.1.2 - September 2023**

    Fixed: The license requirement.

-   **Version 1.1.1 - April 2023**

    Minor release of F5 BIG-IP Spoke for Integration Hub. This version includes an Authentication Template which helps in setting up the spoke easily from Flow Designer . It also includes Deduplication changes.

-   **Version 1.0.3 - August 2019**

    Licensing patch for the F5 spoke on Integration Hub

-   **Version 1.0.2 - March 2019**

    Provides a sample Flow and the necessary Actions to easily automate F5 Routers using Big-IP
