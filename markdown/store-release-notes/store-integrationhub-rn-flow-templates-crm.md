# Flow Templates for CRM release notes

Version history for the Flow Templates for CRM on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - April 2023**

    Fixed: Improved installation performance of spoke.

-   **Version 1.0.2 - February 2023**

    Fixed: Dependencies.

-   **Version 1.0.1 - October 2022**

    Fixed: Dependency cleanup.

-   **Version 1.0.0 - July 2021**
    -   New:
        -   Easily add CRM automation in your custom applications with the Flow Templates for CRM application. Rather than creating a new flow from scratch, users can choose the relevant template in App Engine Studio and follow the guided screens to create a new flow. These templates leverage IntegrationHub spokes to create and update Cases and Opportunities in Microsoft Dynamics CRM.
        -   Setup screens guide you and indicate what information is required. Flows generated from templates provide additional explanations and can be customized as needed. The templates all have pre-defined flow triggers, but you can modify the flows and add additional conditions to control when they are triggered.
