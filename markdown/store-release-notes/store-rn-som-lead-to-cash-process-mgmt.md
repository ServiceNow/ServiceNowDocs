# Lead-to-Cash Process Management release notes

Version history for the Lead-to-Cash Process Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.0 - June 2026**

    Fixed: Enhanced security by introducing non-glide Query Range ACLs.

-   **Version 2.2.3 - March 2026**

    New: As part of platform security best practices, all system properties in the Lead-to-Cash Process Management application now have defined read and write roles. This change does not affect existing functionality.

-   **Version 2.2.0 - December 2025**

    Some of the current fields have their property value set to read\_only. According to the platform’s security directive, these fields must be updated to use the new value strict\_read\_only to ensure compliance with enhanced security requirements. This change does not affect functionality.

-   **Version 1.0.0 - August 2025**

    View the associated relationships between different sales entities tied to the sales process record using a hierarchical node map view.
