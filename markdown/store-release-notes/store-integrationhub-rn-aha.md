# Aha! spoke release notes

Version history for the Integration Hub Aha! spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.9.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.8.0 - April 2026**

    Added 3 new external triggers.

-   **Version 1.7.2 - January 2026**

    Fixed: Look Up Product By ID action.

-   **Version 1.7.1 - August 2025**

    Fixed: Incoming Webhook Functionality

-   **Version 1.7.0 - November 2024**

    Evaluated and modified Sandbox Access for Client-Callable Script Includes as false.

-   **Version 1.6.4 - July 2024**

    Fixed: "Idea", when "New" is clicked, a new record is created without filling any field value.

-   **Version 1.6.1 - February 2023**
    -   New: Added actions that are beyond user management and subscription management use cases
    -   Fixed: Bug fixes
-   **Version 1.5.0 - September 2022**
    -   New:
        -   Added 50+ more actions
        -   Added bi-directional integration design pattern
        -   Added remote table
        -   Added authentication template
        -   Added sample outbound flow
-   **Version 1.0.1 - May 2021**

    Manage Aha! accounts using your ServiceNow instance. Analyse the data usage for Aha! and reclaim the idle accounts.
