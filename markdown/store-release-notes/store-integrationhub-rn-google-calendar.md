# Google Calendar Spoke release notes

Version history for the Integration Hub Google Calendar Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.7.0 - June 2026**

    Fixed: Streamline ACLs

-   **Version 2.6.1 - March 2026**

    Changed: Look up Events Stream action.

-   **Version 2.6.0 - September 2025**

    Fixed: Required ACL\(s\) or Role\(s\) for agents.

-   **Version 2.5.2 - July 2025**

    Fixed: AI Agents - Changed custom to template.

-   **Version 2.5.1 - June 2025**

    Fixed: Actions: Look up Access Control List Stream, Look up Calendars from Calendar List Stream, Look up Event Stream.

-   **Version 2.5.0 - May 2025**

    Minor release with new AI agents.

-   **Version 2.4.5 - September 2024**
    -   Changed: Changed references from Hangout to Meet
    -   Fixed: Security bug
-   **Version 2.4.3 - March 2023**

    Fixed: Added event scope to OAuth Entity Profile when creating a connection and credential for the spoke.

-   **Version 2.4.1 - September 2022**

    Fixed: Patch version of Google Calendar Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 2.4.0 - August 2022**

    New: The minor release of Google Calendar Spoke for IntegrationHub. This version adds the functionality to be able to make a reservation without blocking google calendar.

-   **Version 2.3.0 - May 2022**
    -   New: Added Look up Free or Busy Schedule action
    -   Changed: Objectized Create Event and Update Event actions' recurrence input field
-   **Version 2.1.0 - December 2021**
    -   New:
        -   Added authentication template for easier configuration
        -   Added 10 actions on Access Control and Calendar List Management
        -   Added a sample remote table
-   **Version 2.0.0 - September 2021**
    -   New: A new action for Look Up Event Stream
    -   Changed: Changes to existing actions - Create &amp; Update Event
-   **Version 1.0.3 - July 2021**

    Patch release of the Google Calendar spoke for IntegrationHub. This version includes a fix that now enables selection of Look Up Event Stream Action within Flow Designer.

-   **Version 1.0.2 - November 2020**

    Patch release of the Google Calendar spoke for IntegrationHub.

-   **Version 1.0.1 - July 2020**

    Patch release of the Google Calendar spoke for IntegrationHub.
