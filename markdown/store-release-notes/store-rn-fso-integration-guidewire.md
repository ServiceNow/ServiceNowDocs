# Financial Services Operations Integration with Guidewire release notes

Version history for the Financial Services Operations Integration with Guidewire application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.1 - May 2024**
    -   New: Added a dependency to Financial Services Operations Core
    -   Change: Modified configuration for localization support
-   **Version 1.0.0 - November 2023**
    -   Guidewire Software, Inc. \(GWRE\) is the global market leader for core technology in property and casualty \(P&amp;C\) insurance. Guidewire's ClaimCenter, PolicyCenter, and BillingCenter make up the Guidewire InsuranceSuite. The Financial Services Operations Integration with Guidewire application will enable insurers to integrate with the cloud version of Guidewire InsuranceSuite.
    -   This application lays down the framework for supporting any P&amp;C use case on ServiceNow which requires integration with GWRE. Although out of box subflows address certain primary integrations like retrieving claim details from ClaimCenter, customers can still expand the integration layer to support any business need.
