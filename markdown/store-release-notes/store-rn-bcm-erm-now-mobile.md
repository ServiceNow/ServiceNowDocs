# Safe Workplace for mobile release notes

Version history for the Safe Workplace for mobile application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

From the Safe Workplace for mobile app, users can access applications from the Emergency Response Management and Safe Workplace product suites.

**Note:** The Emergency Response Management for Now® Mobile app was renamed to Safe Workplace for mobile.

## Version history

-   **Version 2.10.5 - June 2026**

    Fixed: Remediated non-Glide ACLs

-   **Version 2.10.3 - August 2025**

    Changed: Minor ACL updates

-   **Version 2.10.1 - January 2024**

    Security fixes.

-   **Version 2.10.0 - May 2022**

    New: Localization for features

-   **Version 1.35.0 - January 2022**

    New: Localization for 22 languages other than English. Translations are available for features released up until the December 16, 2021 release.

-   **Version 2.7.0 - October 2021**

    New: Exemption request

-   **Version 2.6.0 - August 2021**

    Fixed: Minor bug

-   **Version 2.5.0 - July 2021**

    Changed: Updates to the translation engine

-   **Version 2.4.0 - May 2021**

    New: Additional translations

-   **Version 2.3.0 - April 2021**

    Fixed: Minor bug

-   **Version 2.2.0 - December 2020**

    New: Support for a common structure to request consent to various privacy policies

-   **Version 2.1.0 - November 2020**
    -   New:
        -   New catalog item icons
        -   Localization for features from the October 15 release
-   **Version 2.0.0 - October 2020**
    -   New: Localization for features from the October 1 release
    -   Changed: The application name was changed from Emergency Response Management for Now Mobile to Safe Workplace for mobile
-   **Version 1.2.0 - October 2020**
    -   New: Safe Workplace applications are accessible from the mobile app
    -   Fixed: Minor bug fixes
-   **Version 1.1.0 - August 2020**
    -   New: Localization for features from the June 4 release
-   **Version 1.0.1 - June 2020**
    -   New: Updated localization for:
        -   Japanese
        -   French
        -   German
        -   Spanish
        -   Italian
        -   Brazilian Portuguese
        -   Korean
        -   Simplified Chinese
-   **Version 1.0.0 - May 2020**

    The Emergency Response Management for Now Mobile application provides a mobile experience for the Emergency Response apps. In the initial version, users will be able to report on their health status. Managers will also be able to view the health status of their employees, report status on behalf of their employees as well as the ability to reassign tasks which are currently assigned to their direct reports.
