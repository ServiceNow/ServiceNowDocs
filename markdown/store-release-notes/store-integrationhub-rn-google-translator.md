# Google Cloud Translator Service spoke release notes

Version history for the Integration Hub Google Cloud Translator Service spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.2.8 - June 2026**

    Fixed: An issue with the Google Translation batching functionality has been resolved. Batch calls now process correctly, ensuring translations are handled as expected.

-   **Version 3.2.7 - July 2025**

    Fixed: As part of this patch release, we fixed an issue pertinent to batch translation.

-   **Version 3.2.6 - May 2025**

    Resolved issues related to smart batching for Google translations.

-   **Version 3.1.1 - July 2021**

    As part of this patch release, we fixed performance issues pertinent to batch translation and batch detection.

-   **Version 3.0.1 - January 2021**

    New: This new version of Google Cloud Translation Integration provides the ability to translate and detect the languages of multiple texts at once. Smart Batching support is added to minimize the no. of outbound calls in case of batch translation and language detection. This version of the store app is supported from the Quebec release.

-   **Version 1.0.2 - October 2019**

    ServiceNow's Dynamic Translation allows for the seamless translation of text from one language to another. This spoke enables the use Google Translate with Dynamic Translation and with Flow Designer. Available actions:

    -   Detect Language: Identifies the language that text is in
    -   Translate Text to Multiple Languages: Translates text from one language to another as specified
