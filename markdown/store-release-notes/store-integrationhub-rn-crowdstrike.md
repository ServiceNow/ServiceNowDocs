# CrowdStrike spoke release notes

Version history for the CrowdStrike spoke integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - February 2025**

    In this release, two actions have been added for pulling Weekly and Hourly Average consumption counts for endpoints or workloads \(including physical or virtual devices such as computers, servers, laptops, desktops, mobile devices, containers, pods, and virtual machine images\).

-   **Version 1.0.7 - November 2024**
    -   An issue with the data download file limit in the Look up Active Hosts Stream action is fixed.
    -   A new connection attribute for a pagination count is added.
-   **Version 1.0.6 - August 2023**

    In v1.0.6, Look up host details actions are fixed.

-   **Version 1.0.4 - March 2023**

    Fixed: Fixed the Look up Host Details action to handle the new API request from vendor side.

-   **Version 1.0.2 - September 2022**

    Fix for creating accurate CAL records while running CrowdStrike Download Host Sensor Information Subflow

-   **Version 1.0.1 - August 2022**

    New: In the 1.0.1 patch version, an additional header is added to the request details.

-   **Version 1.0.0 - February 2022**

    Integrating your ServiceNow instance with the CrowdStrike enables you to view CrowdStrike active host sensors information and check license compliance.
