# Dynamic Guidance release notes

Version history for the ServiceNow® Dynamic Guidance application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 28.2.15 - April 2026**

    Dynamic Guidance is a voice-powered AI assistant natively embedded in ServiceNow that delivers real-time, step-by-step help across the platform. Users can ask questions in natural voice, receive contextual answers grounded in ServiceNow documentation, and get guided through tasks while staying in their workflow. Key features include real-time conversation transcripts, pause and resume controls, support for extended 30-minute sessions, and admin-configurable documentation sources. Accessible directly from the Now Assist Panel and Help Center, it serves admins, agents, fulfillers, developers, and process creators.
