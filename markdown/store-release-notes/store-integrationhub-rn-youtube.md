# YouTube spoke release notes

Version history for the Integration Hub YouTube spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.5 - July 2025**

    Fixed: Query Metrics action throws errors due to missing variables.

-   **Version 1.0.4 - August 2023**

    Fixed duplicated internal Spoke names.

-   **Version 1.0.3 - May 2023**

    Patch release for YouTube Spoke for Integration Hub. This version fixes an issue to show correct duration for videos longer than 60 minutes.

-   **Version 1.0.2 - September 2022**

    Patch version of Youtube Spoke for Integration Hub. This version enables the state of Look up Groups to be Published and adds a fix to improve the installation performance of the spoke.

-   **Version 1.0.0 - August 2020**

    Provides actions to automate the management of analytics, captions, comments, groups, playlist, reports, subscriptions, and videos on YouTube.
