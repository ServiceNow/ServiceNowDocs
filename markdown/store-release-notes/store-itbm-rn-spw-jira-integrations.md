# SPW Jira Integrations release notes

Version history for the ServiceNow® SPW Jira Integrations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.1.0 - May 2026**

    Guided setup for SPW Jira integration updated to include manual webhook creation steps: retrieving the webhook callback URL from the Jira instance record in ServiceNow and registering it in Jira, and updated credentials configuration step to support two-way sync of work item updates with Jira

-   **Version 1.0.1 - March 2026**

    Guided Setup: Users can now successfully configure multiple projects in sequence using the Guided Setup.

-   **Version 1.0.0 - December 2025**
    1.  Achieve end-to-end visibility in SPM from strategy to execution with the SPW Jira Integrations. This integration synchs the following data:
        1.  Bi‑directional sync for Epics, Features,Capabilities, and Stories​
        2.  Sprints sync one‑way from EAP → Jira and mustbe created in EAP​
        3.  ​Sprint status updates flow back; creation stilloriginates in EAP by design
    2.  Simplify setup of this integration with guided steps using play books and flexible mappings for tables, fields, and choice values for each Jira project\(space\).
    3.  Diagnose any issues with inbuilt Diagnostics tool
