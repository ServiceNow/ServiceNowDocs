# AWS Elastic Load Balancing spoke release notes

Version history for the Integration Hub AWS Elastic Load Balancing spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.1 - September 2022**

    Fixed: Patch version of AWS Elastic Load Balancing Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.0 - July 2020**

    Provides actions to automate the management of load balancers, target groups, and tags in Amazon Elastic Load Balancer.
