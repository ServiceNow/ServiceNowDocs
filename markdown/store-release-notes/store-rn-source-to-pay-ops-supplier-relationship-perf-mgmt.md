# Supplier Relationship and Performance Management release notes

Version history for the Supplier Relationship and Performance Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 10.0.0 - June 2026 \(Australia\)**
    -   New:
        -   Enhanced Smart Assessments UI flow for better user experience.
        -   Improved assessment cancellation handling and resolved related errors.
    -   Changed:
        -   Fixed previous issues and errors in Smart Assessments.
        -   Upgraded the overall flow with UI enhancements.
        -   Improved system stability and assessment cancellation functionality compared to the previous release.
-   **Version 7.0.0 - June 2026 \(Zurich\)**

-   **Version 6.0.0 - March 2026 \(Zurich+\)**
    -   New:
        -   Gen-AI powered supplier performance summarisation: Introduces AI-generated summary of supplier performance scores using available KPI data, validated against a strict output format to ensure consistency and provides a structured narrative covering:
            -   Supplier Performance Score andTrend \(Improving / Stable / Declining\) based on failure count comparison across periods
            -   Narrative Insight summarizing overall supplier health in simple language
            -   Performance Details — failing KPIs, KPIs currently in an action plan, and at-risk KPIs approaching threshold
            -   Historical Context — action plans created, targets achieved, and previous focus areas
            -   Steps Ahead — recommended actions for review, monitoring, and tracking of flagged KPIs
    -   -   Action Plan — Activity and Notifications
    -   Activity Stream: The action plan page now includes an activity stream capturing all updates made to the plan, including the full chat history from agentic conversations, giving users a complete audit trail in one place.
    -   Email Notifications: Stakeholders are notified via email when updates are made to an action plan, ensuring visibility without requiring manual check-ins on the record.

-   **Version 5.0.0 - December 2025**
    -   New:
        -   Automated KPIs
            -   The automated KPI collection system improves SLO performance by enabling you to seamlessly manage performance monitoring by:
                -   Automated Data Collection: Pull KPI data directly from ServiceNow tables at a defined frequency
                -   Configurable Conditions: Set rules to extract and aggregate data accurately
                -   Exception Handling: Automatically revert to manual collection if automated processes fail
            -   The Automated KPIs feature reduces manual effort, ensures timely KPI updates, and provides a reliable fallback mechanism for uninterrupted performance tracking.
        -   Overall Dashboard
            -   The overall supplier dashboard provides complete visibility into your supplier base. You can now gain holistic insights and manage supplier performance more effectively through:
                -   Comprehensive Metrics: View overall supplier score, total spend, active suppliers, and supplier risk in one place
                -   KPI Trends: Track top KPIs across all suppliers
                -   Top Performers and Usage: Identify top-performing suppliers and see how many KPI templates are in use
                -   Interactive Filters and Tabs: Apply filters by supplier or segment and access a dedicated tab for action plans
            -   The overall supplier dashboard empowers you to monitor performance at scale, spot trends quickly, and drive strategic decisions with ease.
