# RPA sample template release notes

Version history for the RPA sample template application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

**Note:** The RPA Sample templates Store application is deprecated and is no longer supported or available for new activation. RPA Desktop Design Studio provides the latest experience for this functionality. For more information, see [Use sample automations in RPA Desktop Design Studio](https://www.servicenow.com/docs/bundle/xanadu-integrate-applications/page/product/rpa-studio/task/use-sample-automation.html).

## Version history

-   **Version 3.1.0 - October 2023**
    -   New:
        -   Oracle EBS Employee Definition Sample Template
        -   Oracle EBS Purchase Order Sample Template
-   **Version 2.3.2 - August 2023**

    With the newly added templates you can download, upload, and share files from OneDrive.

-   **Version 2.1.1 - June 2023**

    New:

    -   With the Clear Browser Cache Robot template, you can clear your browsers cache data using Microsoft Edge and Google Chrome.
    -   With the Copy Excel to SQL table data Robot template, you can copy the Microsoft Excel table data to SQL database table.
    -   With the Oracle E-Business Suite \(EBS\) Accounts Receivable \(AR\) Transaction Robot template, you can create the transaction records in EBS application.
    -   With the Convert Excel to XML template, you can convert the excel file to XML file.
-   **Version 2.0.1 - February 2023**
    -   New:
        -   With the Oracle E-Business Suite \(EBS\) Journal Robot template, you can create journal entries in the Oracle EBS General Ledger module.
        -   With the Oracle E-Business Suite \(EBS\) Procure to Pay Robot template, you can create Accounts Payable \(AP\) invoices in the Oracle Payables module.
-   **Version 1.0.2 - August 2022**

    RPA templates are prebuilt automations that enable customers to kickstart their RPA initiatives. This template provides automating user password reset functionality for Oracle E-Business Suite \(EBS\) where APIs aren't available. It guides the ServiceNow developer to implement password reset functionality.
