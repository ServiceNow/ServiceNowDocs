# Enterprise Architecture - Prime release notes

Version history for the ServiceNow® Enterprise Architecture - Prime application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.1 - June 2026**
    -   New:
        -   Generate a business process map \(BPM\) diagram in the Enterprise Modeling &amp; Visualization by uploading an image of an existing process diagram.
        -   Use the Enterprise Architecture query agent to ask natural language questions about your enterprise architecture portfolio. The agent supports multi-condition queries, quarterly trend comparisons, and impact analysis for scenarios such as application decommissioning, and suggests related questions after answering.
-   **Version 1.0.0 - April 2026**
    -   Enterprise Architecture Prime drives customer value faster by utilizing the power of Generative AI \(GenAI\).
        -   Summarize, shorten, or elaborate architectural decision records \(ADR\) in the Enterprise Architecture Workspace.
        -   Generate and summarize Enterprise Modeling and Visualization diagrams for business applications hierarchy, using the enterprise architecture diagrams AI agent.
            -   Enterprise architects, as well as non-enterprise architects will be able to invoke the AI Agent and ask it to generate a business hierarchy diagram for enterprise architecture entities.
            -   After the diagram is generated, the agent will suggest summarizing the diagram, listing entities in the diagram, and describing the relationships between them.
        -   Compare diagram versions to generate natural language summaries of changes.
        -   Elaborate or shorten text in form fields in Enterprise Architecture Workspace.
        -   Generate contextual insights for business applications in the Enterprise Architecture Workspace.
        -   Generate insights for Enterprise Architecture Workspace dashboards.
