# Universal Request for Source-to-Pay Operations release notes

Version history for the ServiceNow® Universal Request for Source-to-Pay Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.2.2 - June 2026**
    -   New:
        -   SPO‑native integration: Sourcing and Procurement configuration available within the Universal Request app.
        -   Cross‑department transfers: Fulfillers transfer Universal Requests across departments with automatic updates to assignment groups.
        -   Requester resolution and visibility: Requesters accept resolutions and complete follow‑up tasks directly from the ticket page.
-   **Version 1.1.2 - March 2026**

    Changed: The Universal Request for Source-to-Pay Operations application \(app‑fsc‑ur‑common\) has been converted from an XML‑based application into a Fluent‑based application.

-   **Version 1.0.2 - December 2025**

    Universal Request is a ServiceNow feature designed to provide a single, centralized entry point for handling service requests across multiple business units. It eliminates confusion about which department to contact by allowing users \(employees or suppliers\) to submit a generic request that the system intelligently routes to the correct team
