# HR Service Delivery for Healthcare release notes

Version history for the HR Service Delivery for Healthcare on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.4 - May 2025**

    Fixed a bug: Relocation assistance and work visa transfer request is missing in preboarding when the checkboxes are checked during onboarding.

-   **Version 1.0.3 - February 2025**
    -   Fixed:
        -   4 bugs have been fixed as a part of this release:
            -   Request medical scrubs - going to Awaiting acceptance instead of close complete Need to add case option - skip automatic user acceptance state
            -   Specialty field is not populated
            -   Stage1 of healthcare onboarding is getting errored out New Hire notification activity
            -   Duplicate RCAs are present in app-hr and app-hr-healthcare
-   **Version 1.0.2 - November 2024**

    Fixed: Minor bug fixes and minor security updates.

-   **Version 1.0.1 - August 2024**

    The HR Service Delivery for Healthcare application is a new app catering to Healthcare specific requirements of HR Service Delivery customers. The new app will contain features and capabilities bundled together for solving complex HR processes in Healthcare domain like Smooth Physician Onboarding, Capturing and Storing Healthcare Professional Profile and more.
