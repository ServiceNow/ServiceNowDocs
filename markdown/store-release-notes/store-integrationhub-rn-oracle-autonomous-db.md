# Oracle Autonomous DB spoke release notes

Version history for the Integration Hub Oracle Autonomous DB spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.6 - December 2022**

    Changed: App dependency.

-   **Version 1.0.5 - September 2022**

    Patch version of Oracle Autonomous DB Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.4 - March 2022**

    New: Includes the missing dependency on Oracle Cloud IAM Spoke.

-   **Version 1.0.3 - November 2021**

    Changed: Patch release of Oracle Autonomous DB Spoke for IntegrationHub. This version includes changes to randomly generated passwords.

-   **Version 1.0.2 - July 2021**

    Patch release of the Oracle Autonomous DB spoke for IntegrationHub. This version adds KMF modules for additional security of the password2 fields and Rome compatibility.

-   **Version 1.0.1 - November 2020**

    Provides actions to automate the management of Oracle Autonomous DB.
