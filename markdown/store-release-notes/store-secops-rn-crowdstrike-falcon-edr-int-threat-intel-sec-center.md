# CrowdStrike Falcon EDR Integration for Threat Intelligence Security Center release notes

Version history for the CrowdStrike Falcon EDR Integration for Threat Intelligence Security Center application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.1 - April 2025**

    Fixed:

    -   Update on Send to EDR for IPv6 Observable in CrowdStrike EDR.
-   **Version 3.0.0 - November 2024**

    CrowdStrike Falcon Host Integration for Threat Intelligence Security Center \(TISC\) enables Cyber Threat Intelligence \(CTI\) analysts to automatically push malicious or suspicious IP addresses, domains, file hashes, and more from the TISC to the CrowdStrike watchlist for continuous monitoring and real-time alerting.
