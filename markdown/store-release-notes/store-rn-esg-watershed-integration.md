# Watershed integration for Operational Sustainability Management release notes

Version history for the Watershed integration for Operational Sustainability Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.0.1 - March 2026**

    Changed: Updated the name of the application to "Operational Sustainability Integration with Watershed"

-   **Version 16.0.1 - February 2023**

    Fixed: Translation support for missing form labels and messages.

-   **Version 15.1.0 - November 2022**

    Changed: Minor changes in integration setup.

-   **Version 15.0.1 - August 2022**

    The ServiceNow ESG Management Integration Tool helps customers manage their Watershed data feeds by normalizing and mass loading data using flat file.
