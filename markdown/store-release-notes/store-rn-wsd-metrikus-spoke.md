# Metrikus Spoke release notes

Version history for the Metrikus Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.7 - June 2026**

    Fixed: While retrieving space occupancy batches, an incorrect batch size was used.

-   **Version 1.0.5 - February 2025**

    No changes.

-   **Version 1.0.3 - August 2024**

    Fixed: Issues related to security and user interface.

-   **Version 1.0.1 - May 2024**

    Metrikus is a data platform that integrates with 200+ vendors of different types of sensors \(occupancy sensor, temperature sensor, air quality sensor, noise sensor etc.\). This spoke app integrates with Metrikus to fetch occupancy status \(occupied, not occupied, count of people at a space\) of desks and meeting rooms fetched from Occupancy sensors. The app hosts APIs to fetch the occupancy status of spaces from Metrikus.
