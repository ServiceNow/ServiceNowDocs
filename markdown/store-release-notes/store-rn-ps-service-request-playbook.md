# Service Request Playbook release notes

Version history for the Service Request Playbook application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.12.0 - June 2026**

    Add translations for new features and defect fixes.

-   **Version 3.11.5 - April 2026**

    Fixed: Hardcoded strings in the Service Request Playbook are now exposed for translation via Playbook Localization, resolving i18n issues across Public Sector Digital Services \(PRB1968059\)

-   **Version 3.0.0 - February 2025**

    Changed: Updated application labels to support accessible rich internet applications \(ARIA\) accessibility requirements.

-   **Version 2.5.0 - November 2024**

    Changed: Updated application labels to support accessible rich internet applications \(ARIA\) accessibility requirements.

-   **Version 2.4.2 - November 2023**

    Changed: The "Check for duplicate cases" activity in the Service Request Playbook was updated to automatically display similar requests by location without having to search for the requests.

-   **Version 2.3.0 - August 2023**

    New: The 'Check for duplicate cases' activity in Service Request Playbook is updated to automatically display similar requests by location without having to search.

-   **Version 2.1.4 - July 2023**
    -   Fixed
        -   Service Request Playbook's trigger license record was added to avoid canceling playbook execution
-   **Version 2.1.0 - May 2023**
    -   Changed:
        -   Easily extend and develop digital service solutions using updated Constituent, Business, and Agency data models.
        -   Capture detailed constituent and business profile information through new and updated fields for profile definition.
        -   Streamline the support of digital services by using service definitions that associate government services with specific case types.
-   **Version 2.0.1 - February 2023**
    -   New:
        -   Mobile agent experience for government workers to resolve service requests on a mobile device
        -   311 Service Request service definition that enables government workers to easily create service requests from case type selector
