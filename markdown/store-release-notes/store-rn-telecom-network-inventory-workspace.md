# Network Inventory Workspace release notes

Version history for the Network Inventory Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.3 - March 2023**

    Fixed: Added additional validations to ensure the cross-impact of TNI Workspace and SOW Workspace.

-   **Version 1.2.1 - February 2023**
    -   Changed:
        -   Updated Navigation pane to support expanded network inventory data model.
        -   Defined managed network functions to manage and track the network functions that are delivered through Virtual Network Functions\(VNF\) and Cloud-native Network Functions \(CNF\).
        -   Defined the Server, Virtual machine, Kubernetes Cluster, and Kubernetes Pod to track and manage their CIs.
        -   Created various CI inventory groups and applied actions to the group of CIs using the Inventory Group in the TNI workspace.
-   **Version 1.0.1 - August 2022**
    -   New:
        -   Landing page with configurable widgets.
        -   Usage details \( available slots, interfaces\) on the Equipment.
        -   Equipment Model Images and Equipment relations on the Equipment details page.
