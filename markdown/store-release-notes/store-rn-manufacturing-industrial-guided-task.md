# Industrial Guided Task release notes

Version history for the ServiceNow® Industrial Guided Task application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.0.0 - June 2026**

    New: Work Set Tasks are generated from Work Set Standards and can contain one or more sub-tasks of type Industrial Guided Task or Industrial Action.

-   **Version 1.0.0 - March 2026**
    -   Release highlights:
        -   Design structured digital workflows faster with an enhanced guided task authoring experience.Evaluate task outcomes accurately using a smart assessment engine with scoring rules, conditionallogic, and validation.
        -   Improve frontline execution with usability features such as barcode or QR scanning, step‑by‑stepguidance, and offline support.
        -   Track guided tasks through their full lifecycle using intuitive workflow states that support clearergovernance and audit readiness.
        -   Extend task governance across the ICW suite through integrations with Industrial Standards, ICWMobile, and ICW Core.
-   **Version 0.4.3 - January 2026**
    -   Industrial Guided Tasks is an innovative solution for industrial workers who have repeating or ad-hoc work that requires guidance, tracking, and confirmation. Guided tasks are generated from Industrial Standards and contain relevant contextual information related to the operational environment. They also include detailed guidance to help the industrial worker successfully complete required activities in a timely and accurate manner, including tasks performed during every shift.Guided tasks support the industrial worker with:
        -   Detailed scheduling and priority information, helping to answer the question “what do I need to do next?”
        -   Step-by-step activity flows, including steps which only activate when required
        -   Guidance at the point of use, including images, text, and Knowledge
