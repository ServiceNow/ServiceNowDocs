# Software Asset Management Workspace release notes

Version history for the Software Asset Management Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 9.1.1 - October 2025**

    This version includes bug fixes for the AI activity log card and license usage.

-   **Version 9.1.0 - September 2025**
    -   New:
        -   New card 'AI activity log' added in SAM Workspace Homepage. This card reports the recent AI activity by agents. Users can also trigger the Create software reclamation rule and Evaluate software removal candidate agentic workflows via the suggestions button on this card.
        -   New menu 'Reclamation Rules' in License Operations section under Administration in the SAM Workspace. Users can trigger the Create software reclamation rule agentic workflow from this list view.
        -   New buttons in Publisher Result and Product Result pages have been introduced to review and trigger the Create software reclamation rule and Evaluate software removal candidate agentic workflows.Note: The Create software reclamation rule agentic workflow is only available to the SAM admin role and the Evaluate software removal candidate agentic workflow is available to SAM admin and SAM user roles.
-   **Version 9.0.4 - August 2025**

    Software Asset Workspace is an intuitive and streamlined user interface of the Software Asset Management application, to manage software licenses, compliance, and optimization.It is a unified medium with multiple views enabling you to create entitlements, software models, run and review reconciliation results, remediate non compliance, give visibility into your software assets, and provides access to analytics via dashboards.
