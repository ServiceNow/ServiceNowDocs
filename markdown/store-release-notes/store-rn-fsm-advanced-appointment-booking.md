# Advanced Appointment Booking release notes

Version history for the ServiceNow® Advanced Appointment Booking application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 30.0.4 - April 2026**

    Fixed: System properties shipped with the Advanced Appointment Booking app have been updated to require the appointment\_booking\_adminrole for read and write access. With this change, only users with the appointment\_booking\_adminrole can read and modify the system property values, while administrators and maintenance users retain access as per standard platform behavior.

-   **Version 30.0.0 - March 2026**
    -   Standard appointment scheduling works for straightforward operations. Advanced Appointment Booking is built for everything else.
    -   Organizations can define reusable appointment window configurations, model overlapping availability, and set scheduling rules at the territory level so regional operating patterns, working hours, and service constraints are reflected accurately, not approximated.
    -   Planners can configure work and travel durations at both schedule and override levels, apply territory-specific holiday calendars, and manage exceptions without rebuilding base configurations. When demand spikes, controlled overbooking gives teams a structured way to absorb volume without blowing up the schedule.
    -   Appointment recommendations guide customers and agents toward the best available options, balancing operational constraints with a straightforward booking experience.
