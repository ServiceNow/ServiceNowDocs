# Zero Copy Connectors Primary release notes

Version history for the ServiceNow® Zero Copy Connectors Primary application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.2.1 - June 2026**

    Provides AI-Native SKU support for all Primary connectors.

-   **Version 2.1.0 - May 2026**

    All the Primary connectors are included in the Zero Copy Connectors Primary plugin.

-   **Version 1.0.0 - May 2026 \(Zurich\)**

    The Zero Copy Connectors Primary plugin includes all primary connectors.

-   **Version 2.0.0 - April 2026**

    This profile includes all Primary connectors supported by ServiceNow.
