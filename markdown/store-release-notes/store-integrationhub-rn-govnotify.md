# GovNotify spoke release notes

Version history for the GovNotify spoke integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.2.0 - September 2025**

    Added required ACLs or Roles as per the AI Security Directive guidelines.

-   **Version 1.1.1 - August 2025**

    Fixed: GovNotify Spoke AI Agents: Agents Renaming and Output transformation strategy set to None

-   **Version 1.1.0 - March 2025**

    Minor release with new AI agents.

-   **Version 1.0.5 - September 2023**

    Fixed: 'licensable' field set to be true in plugin.xml.

-   **Version 1.0.4 - April 2023**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.2 - June 2022**

    Fixed: Patch release of GovNotify Spoke. This version includes a fix to remove invalidated properties from plugin.xml.

-   **Version 1.0.1 - February 2022**
    -   New:
        -   Added KMF submodule
        -   Added ACL to GovNotify tables
-   **Version 1.0.0 - December 2021**

    GovNotify is the UK government service that sends notifications to the users by emails, text messages, or physical letters. The notifications can be triggered from the central government, local authority, and NHS. ServiceNow, as the System of Engagement, can manage the service requests for the government forms and services. When the users have completed the forms and services, GovNotify Spoke can initiate GovNotify and send notifications to the end user's on the latest status update in real time.
