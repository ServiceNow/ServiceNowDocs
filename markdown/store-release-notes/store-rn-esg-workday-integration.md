# Workday integration for Operational Sustainability Management release notes

Version history for the Workday integration for Operational Sustainability Management on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 21.1.1 - December 2025**

    New: Minor improvement to expose 'Active' field configuration for Choice options.

-   **Version 21.0.0 - August 2025**

    Changed: Refreshed demo data to better align with current feature capabilities and use case scenarios.

-   **Version 16.0.3 - February 2023**
    -   -   Ability to configure Workday report source that would provide data for the metric definitions
-   Ability to schedule and pull HR data from the configured Workday report sources
-   The Workday ESG Integration is built by Bristlecone Inc.
-   The Workday ESG integration helps to automatically import HR data from Workday into ESG Management thus eliminating the need for manual data collection.
