# Process Mining Content Pack for FSM release notes

Version history for the Process Mining Content Pack for FSM application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - March 2026**
    -   Automatically create a copy of Field Service Management \(FSM\) content pack process configuration on content pack application installation.
    -   Uptake of the dependency
    -   As part of the incoming dependency automatically create a copy of the process configuration template when the FSM content pack is installed
-   **Version 1.4.0 - February 2025**

    Addressed data model changes coming from our core plugin \(sn\_po\).

-   **Version 1.2.1 - September 2024**

    Nothing has been changed. We have just updated the version of our dependency plugin \[sn\_po\].

-   **Version 1.2.0 - February 2024**

    Changed: Minor fixes and changes made to the data models of Process Mining for Field Service Management.

-   **Version 1.0.0 - February 2023**

    Process Mining for Field Service Management creates business process flows from the work order task data in audit trails, allowing process owners to perform in-depth analysis and discover process insights to improve business outcomes.
