# Cloud License Estimator release notes

Version history for the Cloud License Estimator application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.0 - August 2025**

    Cloud License Estimator \(CLE\) simplifies the process of estimating resource counts for cloud resources that are eligible for licensing.CLE validates your cloud account details and provides an estimate of the licensable resource count for ITOM and CCM based on the prevailing respective licensing rules.CLE supports AWS and Azure cloud environments.By providing the necessary details in the configuration, users can run the license estimator and can get a detailed report pdf. The pdf report gives user a deep insight for the ITOM and CCM licensable resources out of total cloud resources.Users have the option to download the report in a PDF format for future reference.If you're looking for a hassle-free solution to estimate your cloud resource count and licensing requirements, CLE is the perfect tool for you.Try it today and streamline your cloud licensing process like never before.
