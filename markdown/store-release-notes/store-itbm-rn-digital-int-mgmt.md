# Digital Integration Management release notes

Version history for the Digital Integration Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.1 - June 2026 \(Australia\)**
    -   New: Added support for query\_range access control lists \(ACLs\).
    -   Changed: Resolved accessibility issues in Digital Integration Management to meet WCAG 2.2 AA and Reflow compliance standards.
-   **Version 1.6.0 - June 2026**
    -   New: Added support for query\_range access control lists \(ACLs\).
    -   Changed: Resolved accessibility issues in Digital Integration Management to meet WCAG 2.2 AA and Reflow compliance standards.
-   **Version 1.5.0 - December 2025**

    Fixed: Fixed the issue of displaying closed digital integration requests in the My requests list of Self-Service.

-   **Version 1.4.2 - May 2025**
    -   Added support to allow 140 characters for Digital Interface name
    -   Added Architectural artifacts related list for a Digital Interface
-   **Version 1.4.1 - February 2025**

    Fixed: Added a new workspace view to the Digital Integration table to fix an issue with a preview of records.

-   **Version 1.4.0 - November 2024**

    Fixed: Added more validation and restriction logic to improves data quality and data integrity.

-   **Version 1.3.0 - August 2024**

    The increasing applications and promotion of micro-service architectures are resulting in a large increase in the number of applications and their integrations. With limited governance to connect applications and with the lack of information on how to track and how these applications interact, production issues are hard to troubleshoot. Digital integration management is a capability in Application Portfolio Management that helps manage all application integrations effectively by enabling modeling for APIs and integrations. Digital integration management provides detailed information about integrations such as apps names, versions, type, data flow direction, and middleware used. The model enables an easy way to document, visualize and understand the landscape of APIs and their related business applications.
