# Service Graph Connector for Microsoft Defender for IoT \(Azure\) release notes

Version history for the Service Graph Connector for Microsoft Defender for IoT \(Azure\) \(Azure\) on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.3 - February 2025**

    No changes.

-   **Version 2.0.1 - September 2024**
    -   New:
        -   Map the Hardware and OT Control Module classes into columns with the updated transform map configuration
        -   Certified for Xanadu
-   **Version 2.0.0 - August 2024**
    -   Map the Hardware and OT Control Module classes into columns with the updated transform map configuration
    -   Certified for Xanadu
-   **Version 1.0.2 - February 2024**
    -   Handling of guessed MAC addresses
    -   Support for importing control modules
    -   Troubleshooting capabilities within the SGC
    -   Support for the learning mode of sensors
    -   Filtering capabilities
    -   Several additional class mappings
    -   Minor updates to messages, guided setup, handing the mixed case of sensors, etc.
    -   Certified for Washington DC
-   **Version 1.0.1 - August 2023**

    Integrate Microsoft Defender for IoT \(Azure\) with the ServiceNow Operational Technology Manager application to import devices, and sensor appliances.
