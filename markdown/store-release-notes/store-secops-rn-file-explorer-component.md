# File Explorer Component for Security Operations release notes

Version history for the File Explorer Component for Security Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.11 - August 2025**

    Fixed: Resolved an issue in the SharePoint File Explorer Connector for SIR where long folder names were incorrectly truncated and replaced by a period \("."\), causing confusion during folder navigation.

-   **Version 1.2.4 - November 2024**

    Fixed: Resolved different accessibility issues under Collaboration tab in MSIM workspace, like navigation buttons etc.

-   **Version 1.2.2 - March 2022**

    Fixed: FE folder and files list was not getting loaded when the user toggles between other tabs.

-   **Version 1.1.0 - December 2021**

    Displays the files from File Explorer and allows the users to perform file operations.
