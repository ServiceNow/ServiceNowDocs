# FSO Process Mining Content Pack release notes

Version history for the FSO Process Mining Content Pack on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.8.2 - August 2025**

    Changed: Included additional translations to support localization for Q4.

-   **Version 1.7.0 - November 2024**

    Changed in FSO Card Operations - Card Disputes: Removed the breakdown of Card network and Reason code for Card disputes service cases.

-   **Version 1.6.0 - February 2024**
    -   New: Added breakdowns by Dispute category and Dispute reason code
    -   Changed:
        -   Upgraded snc-app-parent version to 5.1.0.16
        -   Modified configuration for localization support
    -   Removed: Removed breakdowns by Dispute reason and Fraud alert
-   **Version 1.5.2 - August 2023**
    -   Changed
        -   Updated application name, label, and description from "Process Optimization" to "Process Mining".
-   **Version 1.5.1 - May 2023**

    New: Added localization changes.

-   **Version 1.5.0 - February 2023**

    New: Added PO configuration for Intelligent Servicing for Fraud application.

-   **Version 1.4.0 - November 2022**
    -   New:
        -   Added PO projects for Commercial Lines Claims
        -   Added PO projects for Personal Lines Claims
-   **Version 1.3.0 - August 2022**

    New: Added new PO projects for Individual Life, Group Life insurance and claim operations

-   **Version 1.2.1 - March 2022**
    -   New:
        -   Process Mining project for Personal Lines Servicing
        -   Process Mining project for Commercial Lines Servicing
-   **Version 1.2.0 - February 2022**
    -   New:
        -   Added project for Financial Services Personal Deposit Operations
        -   Added project for Financial Services Business Deposit Operations
-   **Version 1.1.1 - August 2021**

    Fixed: Fixed the building project issue associated to Loan Operations and Payment Operations

-   **Version 1.1.0 - July 2021**
    -   New:
        -   Added Process Mining projects for the following Financial Services Operations applications:
            -   Complaint Management
            -   Business Lifecycle
            -   Client Lifecycle
            -   Treasury Operations
-   **Version 1.0.2 - May 2021**

    Changed: Financial Services Core application dependency version has been updated

-   **Version 1.0.1 - April 2021**
    -   Changed: Made it compatible for Quebec
    -   Removed: Removed compatibility for Paris
-   **Version 1.0.0 - March 2021**
    -   New:
        -   Process Mining projects for Financial Services Operations applications:
            -   Payment Operations
            -   Personal Loan Operations
            -   Business Loan Operations
