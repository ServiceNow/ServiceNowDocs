# Ethoca Spoke release notes

Version history for the Ethoca Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.1 - August 2025**

    New in the release: Added datastream and other IntegrationHub apps dependency.

-   **Version 3.0.0 - February 2025**

    New: Added spoke actions to Ethoca Spoke application to integrate with the Ethoca Alerts API Suite published by Ethoca Inc.

-   **Version 2.0.2 - May 2024**

    The Ethoca spoke allows ServiceNow to connect with Ethoca APIs, enabling customers to handle dispute management and fraud prevention more efficiently and securely. Customers can use the spoke to make API calls to the Ethoca service and process responses for further action. They can also access detailed merchant information and subscription control, view digital receipts, and monitor and manage their disputes. By integrating with the ServiceNow platform and other spokes, customers can securely handle their fraud prevention and dispute resolution needs.
