# Now Assist for Service Graph Connectors release notes

Version history for the Now Assist for Service Graph Connectors application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

**Note:** Now Assist for Service Graph Connectors is deprecated. All capabilities are now provided by Now Assist for Configuration Management Database \(CMDB\).

## Version history

-   **Version 1.1.0 - February 2025**
    -   New:
        -   Improved response format: Added improvements to the response displayed in the Now Assist panel for diagnosing issues and providing recommendations, including Issue and Diagnosis recommendation sections to make the response easier to understand.
        -   Support for multiple errors: Added support for handling up to three errors in cases of multiple errors for an import set.
        -   Additional error categories: Added support for additional error categories including Import errors, Flow errors, Partial payloads, and Incomplete payloads.
-   **Version 1.0.0 - November 2024**
    -   Service Graph Connectors accelerate and simplify integrations, providing stable, trustworthy, pre-built integrations with over 60 other IT systems. They enable you to quickly access the key data needed to complete projects in service operations, asset management, security, and various other areas, from operational technology to endpoint management.
    -   Now Assist for Service Graph Connectors utilizes ServiceNow's secure, purpose-built generative AI in combination with specially written knowledge articles and ServiceNow best practices to offer easy-to-understand assistance with setting up, managing, and troubleshooting Service Graph Connectors, reducing manual effort and decreasing the time required to succeed.
    -   It is available to any customer with access to other Now Assist skills.
