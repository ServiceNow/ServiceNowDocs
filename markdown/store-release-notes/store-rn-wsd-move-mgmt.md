# Workplace Move Management release notes

Version history for the Workplace Move Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.14.0 - June 2026**
    -   Changed: Unified record page experience in Workplace Central. Under the Move Management module, Move case and move project pages now use the standard record page layout for a consistent experience across Workplace products.
    -   Fixed: Security fixes - Add profile reader to Move writer and remove the read ACL role for move writer from workplace\_profile
-   **Version 1.13.3 - March 2026**

    Fixed: Improved localization of the date displayed on the calendar in a move project.

-   **Version 1.13.0 - December 2025**
    -   New: Enhancements to Bulk Moves via Excel upload experience on the Workplace Central Move Management module
    -   Fixed:
        -   From location not getting cleared in Desk change request when Requested For is changed as a user not having Workplace Profile
        -   Assign button is not working in list view of move cases on project in workplace central
        -   Translations
-   **Version 1.12.1 - August 2025**
    -   New: Ability to efficiently create and manage bulk move requests by uploading a spreadsheet containing essential move details such as employee information, current locations, and new location.
    -   Changed: Workplace Move Management dashboard list views have been relocated to the common page within Workplace Central.
    -   Fixed: Accessibility improvements
-   **Version 1.10.0 - May 2025**
    -   Changed: Workplace Move Management dashboard list views have been relocated to the common page within Workplace Central.
    -   Fixed: Accessibility improvements
-   **Version 1.9.2 - February 2025**

    Fixed: The "From" location was not getting auto-populated in the "Change desk" request form.

-   **Version 1.9.0 - November 2024**
    -   New: Employees can now submit desk change requests through MS Places.
    -   Fixed: Security Fixes.
-   **Version 1.8.5 - August 2024**

    Fixed: Accessibility improvements.

-   **Version 1.8.0 - May 2024**
    -   New:
        -   Using Space Optimization workspace in Workplace Central:
            -   Initiate employee relocations within and between buildings directly from the floor map.
            -   View employee seat assignments on the floor map.
            -   Access employee information with move actions on map cards in the side panel.
            -   Display indicators on the floor map and map cards to show ongoing move cases.
    -   Fixed:
        -   Performed general translation improvements.
        -   Performed security fixes.
-   **Version 1.7.5 - February 2024**
    -   Fixed:
        -   The protection policy of the non SNC Script Includes has been changed, allowing for customizations by lifting read-only restrictions.
        -   General translation improvements in Move Scheduler.
-   **Version 1.7.0 - November 2023**
    -   Fixed:
        -   You can now create or update workplace profiles with anomalies to avoid failures in move cases.
        -   Earlier, the child task tab in the Workplace Central move case form did not display the associated tasks. This issue has been resolved.
        -   General accessibility improvements have been made like tooltip support in the Move Project action buttons.
-   **Version 1.6.0 - August 2023**
    -   New:
        -   You can now track move cases and associated tasks seamlessly with the new Move management module in the Workplace Central workspace.
        -   The Workplace Central workspace provides you with the Move Schedule, a visual move planning tool, to help you group move cases into projects and create an organized and logical plan for multiple moves.;
    -   Fixed: Granular ACL improvements have been made with reporting to prevent unauthorized access to sensitive column data.
-   **Version 1.5.4 - May 2023**

    Fixed: Performance improvements in loading the desk change request form.

-   **Version 1.5.1 - February 2023**
    -   Changed
        -   The default email templates have been updated to match the next experience content and the style guidelines. The new templates are installed along with the old ones in case the old ones already exist on the instance.
        -   Updated experience to support the next experience for native mobile and responsive web.
-   **Version 1.4.0 - November 2022**

    New: Perform single-user move requests generated from a scenario plan deployment, and manage employee space assignment changes.

-   **Version 1.3.0 - August 2022**

    Changed: Added title field to the record producers to support translations accordingly.

-   **Version 1.2.1 - May 2022**

    New: You can now select an employee's location using the Workplace Indoor Mapping plugin.

-   **Version 1.1.3 - February 2022**

    Fixed:A workplace service item will now be available to all locations if it is not assigned to any specific location.

-   **Version 1.0.8 - December 2021**

    Changed: Security-related changes

-   **Version 1.0.5 - October 2021**

    New: Additional support has been added to support all the 22 languages that are currently available across the ServiceNow AI Platform.

-   **Version 1.0.2 - September 2021**

    ServiceNow® Workplace Move Management enables employees to collaborate, communicate, and easily manage one-off seat changes from start to finish. Both employees or their managers can make move request through a configurable, automated process so that the workplace managers can supervise the movement of employees and their belongings with minimal to no disruption.
