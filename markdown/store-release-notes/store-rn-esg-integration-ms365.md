# Operational Sustainability Management integration with Microsoft 365 release notes

Version history for the Operational Sustainability Management integration with Microsoft 365 application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 17.0.1 - August 2023**
    -   Fixed
    -   -   Reference field used in Calculated Metric Definition aggregation has been updated to improve consistency
-   **Version 16.0.1 - May 2023**
    -   New:
        -   Ability to configure different reporting items from various report sources like tables and reports per business domain.
        -   Ability to configure filter conditions and columns of interest.
        -   Ability to view and download the manifest file required for enabling the Microsoft add-in in Word.
        -   Ability to select the configured reporting items in the add-in under 'Data', 'Table' and 'Charts' tabs.
        -   Ability to view the inventory of links and navigate to the source data.
        -   Ability to refresh inserted content in word from ServiceNow.
    -   Microsoft 365 for ServiceNow Reporting provides reporting capabilities to organisational reporting managers to seamlessly report on ServiceNow data and reports through Microsoft Word and establish an auditable link between the document and the data.
