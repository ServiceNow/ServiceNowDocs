# @servicenow/now-vis-gauge release notes

Version history for the @servicenow/now-vis-gauge application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 23.0.8 - March 2023**

    Bug fixes

-   **Version 23.0.5 - February 2023**
    -   Gauge visualization is a part of the Data Visualization component. It presents a single score of ServiceNow or custom data as a position on a gauge/speedometer chart.
    -   The Gauge data visualization can be used in the Next Experience and is available from Next Experience Dashboards, the UI Builder, and the Visualization Designer.
    -   This application is installed automatically with the "Data Visualization Core" application.
    -   This data visualization is new/added in this version.
