# Card data security container release notes

Version history for the ServiceNow® Card data security container application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.1.1 - April 2026**

    Updated: Improved performance and accuracy when revealing card data by optimizing change detection and removing unnecessary throttling.

-   **Version 1.1.0 - March 2026**
    -   Enable dispute agents to access card data and sensitive documents within ServiceNow by embedding card information and documents in an iframe that leverages a tokenization system. This dedicated Seismic component facilitates this integration, ensuring compliance with security standards while maintaining user accessibility. This component is configured with the following modes to streamline card data interaction:
        -   Reveal card number
        -   Input card number
        -   View document
        -   Upload document
