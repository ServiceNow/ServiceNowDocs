# Magnit spoke release notes

Version history for the IntegrationHub Magnit Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - September 2023**

    Fixed: 'licensable' field set to be true in plugin.xml.

-   **Version 1.2.0 - May 2023**
    -   This minor version includes two new actions and a fix for exiting actions.
        -   New Actions:
            -   Send onboarding status to Magnit
            -   Verify onboarding status from Magnit
    -   Fixed: Exiting actions are also fixed to support xml content.
-   **Version 1.1.0 - December 2022**

    PRO Unlimited spoke is renamed to Magnit Spoke, as due to company's corporate rebranding.

-   **Version 1.0.0 - November 2022**

    ProUnlimited Spoke provides the user action to fetch the contingent worker details from PRO-WAND system. With the actions provided with the spoke, the user will be able to fetch either all the contingent worker details or the newly created and updated contingent worker details. These contingent worker details will be helpful in triggering the onboarding tasks for those workers from the ServiceNow side.
