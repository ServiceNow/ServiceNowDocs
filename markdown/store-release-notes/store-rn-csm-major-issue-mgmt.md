# Major Issue Management release notes

Version history for the ServiceNow® Major Issue Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 4.0.0 - June 2026**
    -   New: Added new reference field on Case \(sn\_customerservice\_case\) table: 'Suggested major case'
    -   Changed: Added logic to maintain 'Suggested major case' through a Major Case's lifecycle \(major\_case\_state as Proposed or Accepted\)
-   **Version 3.0.1 - April 2026**

    New: Enhanced major case creation to support consistent case types across related cases. Key actions include creating major cases with the same case type as the original, ensuring child cases inherit the major case's type.

-   **Version 2.0.0 - March 2026**

    A set of capabilities used to manage customer-facing communications and resolution processes for common issues.
