# Cloud Deployment Automation release notes

Version history for the Cloud Deployment Automation application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - December 2024**

    Fixed: Sandbox Access for Client-Callable Script Includes.

-   **Version 1.0.2 - April 2023**

    Fixed: Improve installation performance of application.

-   **Version 1.0.0 - August 2022**

    Automate service catalog requests for cloud deployment in minutes with the Cloud Deployment Automation application. Choose from catalog items covering common use cases integrating with AWS CloudFormation and quickly move from request to fulfillment. Each catalog item comes linked to a flow that includes an approval process and automated fulfillment and follows request fulfillment best practices. The flows reassign the task and create an incident or event \(or execute a custom subflow\) in the case of an automation error. Follow configuration steps to select approvers and other assignees and get up and running in minutes.
