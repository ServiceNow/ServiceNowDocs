# Vulnerability Response Integration with Claroty CTD release notes

Version history for Vulnerability Response Integration with Claroty CTD on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.4 - August 2025**

    Fixed in Zurich release: Updated with bug fixes

-   **Version 2.0.3 - May 2025**
    -   New:
        -   Support for Claroty CTD v5.1.
        -   New filter properties enable you to identify and import vulnerabilities according to your requirement.
    -   Changed:
        -   With Claroty CTD v5.1 you can import unlimited numbers of Common Vulnerabilities and Exposures \(CVEs\).
        -   If you are using Claroty CTD v5.1, the Minimum CVSS Score field \(available in v5.0 and earlier\) is replaced with CVSS V3 Score.
-   **Version 2.0.1 - November 2024**

    Fixed: Updated with various bug fixes.

-   **Version 1.0.5 - February 2024**

    Certified for Washington DC.

-   **Version 1.0.4 - October 2023**

    Refactored VIT import jobs to correctly handle closed VIT state \(PRB1698775\).

-   **Version 1.0.3 - August 2023**

    Import vulnerabilities from Claroty CTD into the ServiceNow Operational Technology Vulnerability Response application and take risk-based action with production process context.
