# Document Service Framework for OneDrive release notes

Version history for the Document Service Framework for OneDrive application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.2.0 - May 2026**

    Changed: Extended personal authentication for personal authentication in list API.

-   **Version 3.1.0 - December 2025**

    Fixed: Added compatibility support for Glide changes

-   **Version 3.0.0 - August 2025**
    -   Personal authentication support
    -   Treat missing collaboration id as error case in Collaborate Document V2
-   **Version 2.1.1 - October 2024**
    -   Changing dependency version
    -   Enabling subscription
-   **Version 2.1.0 - August 2024**
    -   Minor changes to integrate MultiProvider to DMS
    -   Minor defect fixes
-   **Version 2.0.1 - March 2024**

    Provides capabilities to support Document Services \(released as part of the Quebec release\) for integrating with Microsoft OneDrive and Sharepoint for File Management. It creates all the necessary tables and scripts, used by sub-flows in this spoke, to seamlessly integrate.
