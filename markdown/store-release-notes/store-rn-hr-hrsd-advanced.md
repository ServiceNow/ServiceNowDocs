# HRSD - Advanced release notes

Version history for the ServiceNow® HRSD - Advanced application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.0.2 - June 2026**

    Changed: App and plugin dependencies have been updated.

-   **Version 1.0.3 - April 2026**

    The HRSD Advanced plug-in acts as a digital identifier, specifically designed to represent and validate the customer's licensing tier.
