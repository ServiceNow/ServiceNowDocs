# Customer Request for Quote release notes

Version history for the ServiceNow® Customer Request for Quote application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.1 - March 2026 \(Zurich\)**
    -   Fixed:
        -   Names of all activities are displayed in the left navigation when the playbook experience is launched.
        -   Multiple boarder lines are no longer displayed in review details table.
        -   The Selected product activity is now loads in an expanded state.
-   **Version 1.0.0 - December 2025**

    Enables business-to-business \(B2B\) customers to submit a request for quote \(RFQ\) directly from the Business Portal for products requiring pricing inquiries. This app improves customer autonomy, shortens the sales cycle, and enhances operational efficiency by reducing manual intervention. Sales teams review RFQs, generate quotes, and make them available for customers to view in the portal.
