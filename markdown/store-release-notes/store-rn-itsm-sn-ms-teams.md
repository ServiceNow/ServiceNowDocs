# ServiceNow for Microsoft Teams release notes

Version history for the ServiceNow for Microsoft Teams integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.1.2 - February 2021**
    -   New:
        -   For calls that require higher-touch resolution, an agent can start a Microsoft Teams Call from a chat directly from Agent Workspace.
        -   The platform requires: Notify Connector for Microsoft Teams \(Store App\) Agent Chat \(Plugin\)
    -   Fixes.
    -   Removed: The Start Chat capabilities on the HR task table is removed for out-of-box new installations.
-   **Version 4.0.2 - February 2021**
    -   New:
        -   For calls that require higher-touch resolution, an agent can start a Microsoft Teams Call from a chat directly from Agent Workspace.
        -   The platform requires: Notify Connector for Microsoft Teams \(Store App\) Agent Chat \(Plugin\)
    -   Fixes.
    -   Removed: The Start Chat capabilities on the HR task table is removed for out-of-box new installations.
-   **Version 3.0.12 - December 2020**
    -   Fixed:
        -   Intermittent actionable approval notifications are not delivered.
        -   Employee with language set in user profile doesn’t receive actionable notifications.
        -   When NOW Actions is installed along with ServiceNow for Teams application, the responses to the actionable notification are not captured.
        -   When the On-Call scheduling plugin is not installed, Start Microsoft Teams Chat doesn’t respond.
-   **Version 3.0.11 - November 2020**

    ServiceNow for Microsoft Teams extends the Now Virtual Agent integration with Microsoft Teams to enable employees to more effectively request and receive service from within Microsoft Teams.
