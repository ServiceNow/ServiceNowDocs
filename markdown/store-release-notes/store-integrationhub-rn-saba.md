# Saba spoke release notes

Version history for the Saba spoke integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.2 - June 2026**

    Changed: Access control lists \(ACLs\) have been updated to align with current platform security standards.

-   **Version 1.1.0 - December 2025**

    Minor fixes.

-   **Version 1.0.4 - December 2022**

    Patch release of Saba Spoke for IntegrationHub. This version includes localization fixes.

-   **Version 1.0.3 - February 2022**

    Fixed: Patch release of Saba Spoke for IntegrationHub. This version fixes a security issue.

-   **Version 1.0.2 - November 2021**

    Provides actions to integrate the SABA courses and user activity with ServiceNow.
