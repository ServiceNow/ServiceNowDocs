# Now Assist for Hardware Asset Management release notes

Version history for the Now Assist for Hardware Asset Management on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.3.0 - June 2026**
    -   This release delivers UI modernization and quality improvements. The Now Assist for HAM interface has been upgraded to the latest UI framework, with improved parsing and rendering of Asset Summary responses. Accessibility improvements have also been applied.
    -   Improvements:
        -   Currency fields extracted from purchase order attachments now display correctly.
        -   A missing field configuration in the purchase order attachment flow has been added.
        -   A component reliability issue in the record review workflow has been fixed.
-   **Version 4.0.2 - May 2026 \(Zurich\)**

    The Now Assist panel for sourcing includes updates that enhance functionality, streamline the procurement process, and improve user experience and efficiency. These updates address key user feedback and align with broader product capability advancements for the Now Assist context menu.

-   **Version 4.0.3 - May 2026 \(Yokohama\)**

    This section outlines key improvements and adjustments made to ensure seamless integration with the YP13 system, including protocol enhancements and configuration updates.

-   **Version 4.1.0 - April 2026**

    Feature visibility is now automatically enforced based on the HAM SKU tier \(Foundation, Advanced, or Prime\), ensuring users only see capabilities included in their subscription.

-   **Version 4.0.0 - March 2026**
    -   Added a Gen AI-powered "Generate asset insights" feature for Hardware Asset Management, enabling asset managers to access real-time, comprehensive asset information aggregated from purchase orders, incidents, tasks, work orders, and financial records.
    -   Key actions include:
        -   Summarizing recent and active lifecycle events
        -   Generating chain of custody reports
        -   Displaying audit status, lifecycle dates, financial information, and identifying missing critical data.
    -   The insights are posted on asset records upon page load and triggered via UI actions or natural language prompts. This improves decision-making, operational efficiency, and asset oversight by addressing fragmented asset data and enabling timely, actionable insights.
-   **Version 3.0.1 - December 2025**
    -   In this release, users with the procurement\_user role can access the Help manage hardware asset requests agentic workflow including the following AI agents:
        -   Hardware asset management sourcing AI agent
        -   Transfer order creation AI agent
        -   Purchase order creation AI agent
        -   Hardware asset repair process \(patch\)
-   **Version 2.1.0 - October 2025**

    In this version, security directive changes and platform performance improvements have been implemented.

-   **Version 2.0.0 - September 2025**
    -   New in the Zurich release:
        -   Zurich patch 1
            -   Optimize hardware asset repair process with the suite of AI agents
            -   Automate the repair of bugive and out-of-warranty hardware assets by using AI agents in the Help repair hardware assets agentic workflow. These AI agents validate the repair requests, provide detailed troubleshooting and repair instructions, and complete the tasks on receiving user confirmation. By automating tasks in the asset repair process, AI agents help reduce delays in issue resolution, speeding up repairs, and minimizing manual intervention.
        -   Activation information
            -   Install Now Assist for Hardware Asset Management \(HAM\) by requesting it from the ServiceNow Store. Visit the ServiceNow Store website to view all the available apps and for information about submitting requests to the store. For cumulative release notes information for all released apps, see the ServiceNow Store version history release notes.
        -   Additional information
            -   The Now Assist for Hardware Asset Management \(HAM\) application requires the Hardware Asset Management Pro plus license.
        -   Related ServiceNow applications and features
            -   Now Assist
                -   Help improve the productivity and efficiency in your organization, deliver better self-service, recommend actions, provide answers, and empower your users to search more effectively.
            -   Now Assist Admin console
                -   Use the Now Assist Admin console for quick and effortless access to the important information that you need to set up, configure, and monitor Now Assist applications and features.
            -   Now Assist panel
                -   Use this conversational interface in ServiceNow Service Operations Workspace to summarize a chat, an incident, or resolution notes so that you can get the context of this information more quickly.
            -   Now Assist skills
                -   Use the ServiceNow Now Assist products to provide generative AI skills to meet the needs of users in different workflows, including case or incident summarization, chat summarization, resolution notes generation, and code generation.
-   **Version 1.2.0 - August 2025**

    Improved performance and support for third-party LLMs in the Help manage hardware asset requests agentic workflow.

-   **Version 1.1.0 - June 2025**

    With the Now Assist for Hardware Asset Management \(HAM\) application, hardware asset managers can streamline and automate the process of sourcing hardware assets by using AI agents included in the agentic workflow.
