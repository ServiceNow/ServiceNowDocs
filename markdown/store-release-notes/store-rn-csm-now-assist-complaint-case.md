# Now Assist for Complaint Case \(CSM\) release notes

Version history for the ServiceNow® Now Assist for Complaint Case \(CSM\) application on the ServiceNow Store.

-   **Version 2.1.5 - June 2026**

    Changed: No new features, enhancements or fixes. Since there are updates to the AI agents, the version of this application must also be updated.

-   **Version 2.1.1 - April 2026**

    No content changes, adding a wrapper for the AI agent collection.

-   **Version 2.0.2 - March 2026**

    New: Intelligent, state‑specific summaries for complaint casesConsistent, structured output across all complaint lifecycle stagesAutomatic surfacing of key issue details, actions, SLAs, and resolution contextDesigned to reduce manual review time and accelerate agent response

-   **Version 1.0.3 - February 2026**

    No Features delivered via this application, it is a wrapper for the Complaint Case AI Agent collection.

-   **Version 1.0.2 - January 2026**

    This application is intended to include Now Assist skills for Complaint Case. The current version serves as a wrapper application for the Complaint Case AI Agent collection and does not deliver any skills in this release.
