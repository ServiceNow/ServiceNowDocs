# Insurance claims release notes

Version history for the FSO Insurance claims application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - June 2026**

    New: Added child tables for indexing for Financial Case indexed source

-   **Version 1.2.3 - April 2026**

    Updated: Added new translations for the store app, improving language support and accessibility for users in different regions.

-   **Version 1.2.2 - March 2026**
    -   Updated:
        -   Updated the read-only option to 'Strict Read Only' for read-only fields
        -   Replaced direct references to the admin role with feature-specific granular admin roles to align with least-privilege access principles.
-   **Version 1.0.5 - February 2026**

    Changed: Added aria-label to incident details heading for improved screen reader accessibility

-   **Version 1.0.4 - August 2025**

    Fixed: Unexpected 'Continue' and 'Back' buttons visible on 'Collect inbound documents' activity for all FSO Apps \(Banking and Insurance\).

-   **Version 1.0.3 - February 2025**
    -   Changed:
        -   Changed the incident config table icon
        -   Changed view names to make them generic
        -   Updated the 'Create adjuster tasks for claim incidents' subflow to include service definition field
-   **Version 1.0.2 - November 2024**

    The ServiceNow Insurance claims application streamlines and automates the entire claims process with a flexible framework that adapts to any claim type without needing to create new case types. It allows for easy configuration of a comprehensive end-to-end claims workflow and supports complex claims requiring multiple adjusters. Additionally, it features customizable, persona-specific workspaces for improved efficiency.
