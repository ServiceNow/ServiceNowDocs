# Now Assist for RPA Hub release notes

Version history for the Now Assist for RPA Hub application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.0.4 - March 2026 \(Australia\)**

    New: Added support for Model provider version for RPA bot generation skill

-   **Version 3.3.0 - January 2026 \(Yokohama\)**

    New: Use Now LLM Long-Term Support \(LTS\) model as AI model provider for RPA bot generation skill in addition to Azure OpenAI, Google Gemini, Now LLM, and Anthropic Claude on AWS.

-   **Version 5.0.2 - December 2025**

    New: Use Now LLM Long-Term Support \(LTS\) model as AI model provider for RPA bot generation skill in addition to Azure OpenAI, Google Gemini, Now LLM, and Anthropic Claude on AWS.

-   **Version 2.1.0 - October 2025**

    Changed: Enhanced access controls for RPA bot generation skill. Access to RPA bot generation skill is now restricted to users with the RPA developer or RPA admin role. These roles contain the Now Assist Admin user role \(sn\_nowassist\_admin.user\).

-   **Version 4.0.1 - October 2025**

    Performance Improvements.

-   **Version 4.0.0 - September 2025 \(Zurich\)**

    Changed: Enhanced access controls for RPA bot generation skill.

-   **Version 3.2.0 - September 2025**

    Changed: Enhanced access controls for RPA bot generation skill. Access to RPA bot generation skill is now restricted to users with the RPA developer or RPA admin role. These roles contain the Now Assist Admin user role \(sn\_nowassist\_admin.user\).

-   **Version 3.1.0 - August 2025 \(Yokohama\)**

    New: Google Gemini and Anthropic Claude are available as AI model providers for RPA bot generation skill in addition to Azure OpenAI GPT and Now LLM.

-   **Version 2.0.0 - June 2025 \(Xanadu\)**

    New: Use the Robotic Process Automation \(RPA\) bot generation skill in RPA Desktop Design Studio to create and edit automations and activities through short text prompts and preview options, significantly accelerating automation development. It equips both newcomers and veteran users with the capability to rapidly develop and construct improved automations. Improve automation logic using the Build automation option, whether starting from predefined components or a blank slate, guided by intuitive text prompts.

-   **Version 3.0.2 - May 2025**

    Changed: Improved RPA bot generation by adding more components information. For more information about the list of supported components, see Supported components and connectors for Now Assist for RPA Hub in Yokohama release.

-   **Version 1.0.0 - February 2025**

    Use the Robotic Process Automation \(RPA\) bot generation skill in RPA Desktop Design Studio to create and edit automations and activities through short text prompts and preview options, significantly accelerating automation development. It equips both newcomers and veteran users with the capability to rapidly develop and construct enhanced automations. Improve automation logic using the Build automation option, whether starting from predefined components or a blank slate, guided by intuitive text prompts.
