# Reporting UI Component for Workspace release notes

Version history for the Reporting UI Component for Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.4.0 - June 2026**

    Changed: Implemented Query Range ACL

-   **Version 2.3.1 - April 2026**

    Fixed: An issue where the Publish button was non-functional in the TISC reporting feature.

-   **Version 2.3.0 - December 2025**

    New: Enhancements to enable support for intelligence reports within the Threat Intelligence Security Center.

-   **Version 2.2.0 - August 2025**

    Changed: Modified the Edit and Publish button Colour from Red to Blue.

-   **Version 2.0.7 - May 2025**

    Fixed: Fixed one bug related to localization.

-   **Version 2.0.6 - February 2025**

    New: Supports the Daily Status Reporting feature in both SIR Workspace and TISC.

-   **Version 2.0.4 - November 2024**

    Fixed: Fixed ARIA violations/accessibility issues.

-   **Version 2.0.3 - August 2024**

    New: Reporting UI Component for Workspace supports localization.

-   **Version 2.0.2 - May 2024**

    Threat Intelligence analysts can use the case reporting feature to generate reports related to cases, such as Post-Investigation summaries and Status reports, and share them with relevant stakeholders.
