# Snowflake Spoke release notes

Version history for the Snowflake Spoke application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.3 - September 2025**

    Fixed - Metadata actions

-   **Version 1.0.2 - February 2025**

    Snowflake Spoke provides a list of actions that interact with the Snowflake APIs. It provides the foundation to synchronize ServiceNow and Snowflake systems, such as creating, retrieving, updating, and deleting any Snowflake records, as well as executing custom queries. When ServiceNow and Snowflake collaborate together, the data warehousing integration possibilities are endless.
