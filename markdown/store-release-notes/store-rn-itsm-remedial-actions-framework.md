# Remedial Actions Framework release notes

Version history for the Remedial Actions Framework on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.1 - May 2023**
    -   New: New capability to support remediations.
    -   A flexible framework to enable remedial actions. By leveraging this framework, remedial actions can be triggered from different front end UIs and can follow different processes for execution of the remedial actions as defined.
    -   In addition to enhancing an incident with relevant metrics from your devices and servers, you can now add remediations to streamline investigations and accelerate incident remediation and resolution with this framework.
