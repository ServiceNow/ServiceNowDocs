# Mentoring release notes

Version history for the Mentoring application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.0 - June 2026**

    Fixed automation test cases.

-   **Version 2.2.1 - March 2026**

    New: Extract raw activity data from the Mentoring application for all employees under a manager

-   **Version 2.1.1 - December 2025**

    No customer facing changes.

-   **Version 2.0.1 - August 2025**

    Introduced Multiple mentoring programs: This capability allows mentors and mentees to enroll and participate in multiple concurrent mentoring programs based on criteria and descriptions defined by the admins. Admins can use this feature to create, manage, and organize mentoring relationships based on topic of interest, skills, or job roles.

-   **Version 1.4.0 - May 2025**
    -   New: Auto-close mentoring relationships after the grace period post due date.
    -   Changed: Change in APIs for Mentoring Activity Insights
    -   Fixed: Coral Theming
-   **Version 1.3.0 - February 2025**
    -   New:
        -   Data Model changes to store Mentoring Snapshot.
        -   Improved the way Mentoring snapshot is saved for Mentor Match Insights
        -   Performance Improvement changes for Mentoring Skill Activity Insights
    -   Fixed:
        -   Fixed demo data issue for Mentoring snapshot
        -   Fixed a11y violations in GP Mentorship card and enrollment
        -   Cosmetic UI fixes for Mentoring Filter flyout in Search Mentors
-   **Version 1.2.0 - November 2024**

    Mentoring offers improved networking opportunities and enables individuals to forge meaningful connections. It equips employees with tools such as facilitating the search for guidance and suitable mentors to expedite their learning and amplify their potential for career advancement.
