# Confluence Search Connector release notes

Version history for the Confluence Search Connector application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - November 2024**

    New: Ability to remove indexed content that has been deleted or archived on Confluence.

-   **Version 1.0.0 - August 2024**

    Confluence Search Connector allows your employees to search and discover content from ServiceNow. You can index pages from multiple spaces, enabling employees to find the right answers through Employee Service Center, Now Mobile, and Virtual Agent.
