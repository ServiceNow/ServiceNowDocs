# HR Service Delivery NLU Model for Virtual Agent Conversations release notes

Version history for the HR Service Delivery NLU Model for Virtual Agent Conversations on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.2.0 - November 2023**

    New: You can now select HR cases for Topic Recommendations.

-   **Version 22.1.2 - November 2022**

    NLU model changes with consistent intents, utterances, and entities.

-   **Version 22.0.1 - May 2022**
    -   New:
        -   401kBenefitsInquiry
        -   ChangeInBenefitsInquiry
        -   DownloadPayslip
        -   RequestEmploymentVerificationLetter
        -   EnrollNewBeneficiary
        -   HRAccountAccessRequest
        -   HRCaseFollowUp
        -   HSAInquiry
        -   TuitionReimbursementRequest

            **Note:** These new intents are only available in the English language.

    -   Changed: The existing GeneralHRInquiry intent is Deprecated \(Soft Deprecation\).
-   **Version 21.1.0 - September 2021**
    -   New: NLU model extends support for Dutch \(NL\), Simplfied Chinese \(ZH\), Brazilian Portuguse \(PB\)
    -   Changed: Modified utterances for Japanese\(JP\) and Spanish\(ES\) models
-   **Version 21.0.2 - July 2021**

    New: NLU models are trained to understand different statements that a user might make during a conversation and relate these statements to a task that the user might want to perform. The NLU models for HR provide NLU support to the HR Virtual Agent conversation topics.
