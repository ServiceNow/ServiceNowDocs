# Analytics Generation release notes

Version history for the ServiceNow® Analytics Generation application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 4.1.4 - April 2026**

    Use the data visualizations generation skill to create visualizations of table and indicator data from conversational interactions in the Now Assist panel. For example, you can ask for information about the number of open incidents by priority and get a bar chart. You can edit this chart or add it to a dashboard that you own. The data visualizations generation skill is available under Data and Analytics workflow in Now Assist Admin
