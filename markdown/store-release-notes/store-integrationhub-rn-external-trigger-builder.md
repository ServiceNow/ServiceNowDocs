# External Trigger Builder release notes

Version history for the External Trigger Builder application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - March 2026**

    New: Subscribe External Triggers in Trigger Builder.

-   **Version 1.0.3 - August 2025**

    New: Create and manage external event sources.

-   **Version 1.0.0 - February 2025**

    External Trigger Builder improves Trigger Builder with the capability to build and save your own webhook-based external triggers. External triggers can be used to trigger flows when an event happens in an external system. External triggers improve the experience for externally based inbound event related flows.
