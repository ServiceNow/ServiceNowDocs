# Data Separation release notes

Version history for the Data Separation on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.1 - August 2023**

    The Strategic Portfolio Management Data Separation application enables organizations to restrict access to sensitive data based on a lens hierarchy and its leaf node. The application supports records, related items, planning consoles, workbenches, and reports.

    New:

    -   Drive data separation based on the selected lens hierarchy and leaf node.
    -   Map user groups to various levels of the lens hierarchy to provide access in a hierarchical manner.
    -   Enable data separation for select entities that are supported out of the box.
