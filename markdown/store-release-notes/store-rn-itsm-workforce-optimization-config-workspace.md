-   **Version 1.0.0 - March 2022**

    Workforce Optimization for ITSM provides a set of applications that empowers the manager persona to optimize their work. By using the Manager Workspace enabled by this application, managers can go to a single place where they can perform all their tasks, such as view and change agents schedules, assign and review trainings and track groups and individual agents KPIs.
