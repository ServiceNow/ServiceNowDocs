# GRC: Crisis Management integration with Everbridge Notifications release notes

Version history for the GRC: Crisis Management integration with Everbridge Notifications application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 11.0.0 - June 2026 \(Australia\)**

    Fixed: ACLs and role assignments for tables, along with fix scripts to register and handle customization of those ACLs on install/upgrade.

-   **Version 9.1.1 - December 2025 \(Zurich\)**

    Fixed: Enhanced security on read-only fields to prevent from changing

-   **Version 8.1.0 - June 2025**

    Updated: implement admin override changes.

-   **Version 6.1.1 - July 2024**

    Fixed: Fixed app compatibility for the glide family releases.

-   **Version 6.1.0 - June 2024**

    Fixed: Security defect for GlideRecord encoded query.

-   **Version 3.0.2 - August 2023**

    New: Added crisis management with Everbridge notification integration to BCM next experience workspace.

-   **Version 2.0.0 - February 2023**

    Fixed: Draft notifications can still be sent when the event is in the pending approval state.

-   **Version 1.1.2 - December 2022**

    Fixed: Fixed parent POM in order to remove the redundant GLIDE-INF folder and content contained within.

-   **Version 1.1.1 - March 2022**

    New: Enabled multi-language support and included translations.
