# App Applaud UX release notes

Version history for the ITSM App Applaud UX application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - November 2024**

    No Major feature changes. Contains minor changes and bug fixes.

-   **Version 1.1.1 - August 2024**

    There are no major feature changes. There are a few accessibility bug fixes for this app in this release.

-   **Version 1.1.0 - May 2024**

    Fixed: WCAG bugs.

-   **Version 1.0.3 - February 2024**

    Fixed: Top applauded occasions report bug.

-   **Version 1.0.1 - November 2023**
    -   An employee recognition tool that enables Managers and Employees alike to appreciate their peers on their achievements. This app has notifications enabled out of the box. It also provides insights into recognitions patterns to the respective users and managers.
    -   This app is integrated with Service Operations Workspace and Manager workspace.
    -   Applaud can be used with Service Operations Workspace independent of Workforce Optimization.
