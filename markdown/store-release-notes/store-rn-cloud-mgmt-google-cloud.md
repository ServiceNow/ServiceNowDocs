# Cloud Provisioning and Governance: Google Cloud Connector release notes

Version history for the ServiceNow® IT Operations Management Cloud Provisioning and Governance: Google Cloud Connector on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

**Note:** In Paris, this application was renamed from Cloud Management: Google Cloud Connector to Cloud Provisioning and Governance: Google Cloud Connector.

## Version history

-   **Version 1.0.10 - May 2023**
    -   Fixed:
        -   Localization issues
        -   Fixes in packaging xml \(plugin.xml\)
-   **Version 1.0.7 - May 2022**

    Fixed: Cache flush performance improvement

-   **Version 1.0.5 - October 2020**
    -   New: This application is now compatible with the Paris release.
    -   Changed: This application is renamed to Cloud Provisioning and Governance: Google Cloud Connector \(previously known as Cloud Management: Google Cloud Connector\).
-   **Version 1.0.3 - June 2019**

    Support for Google Cloud Platform \(GCP\)

    -   Ability to create ServiceNow Catalog offerings from Google Deployment Manager template
    -   Ability to provisioning IaaS and PaaS resources in GCP with governance and security policies applied along the provisioning path
    -   Integration with ITSM processes such as Approval, Change control
    -   Near-real-time discovery of state changes to resources in GCP thru integration with StackDriver
    -   Day-2 operations \(life cycle operations\) on the provisioned and discovered resources in the cloud
