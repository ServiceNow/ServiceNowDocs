# User Surveys release notes

Version history for the ITSM User Surveys application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.4 - December 2025**

    Minor bug fixes

-   **Version 1.5.2 - November 2024**

    No Major feature changes. Contains minor changes and bug fixes.

-   **Version 1.4.0 - November 2023**

    Removed: Removed duplicate file name.

-   **Version 1.3.0 - August 2023**

    New: Uptake Applicable groups for User Surveys; Localization build changes.

-   **Version 1.2.0 - February 2023**
    -   New: Provide the ability to take survey in a new tab in that record view vs clicking on take survey UI action which takes me to a new form view
    -   Fixed: Security bug fix
-   **Version 1.1.2 - August 2022**

    This application is for customers who have installed Workforce Optimization. This feature enables team managers create and administer surveys to their teams to capture team satisfaction, in addition to other qualitative metrics. You can then associate results from surveys to Workforce Optimization KPIs.
