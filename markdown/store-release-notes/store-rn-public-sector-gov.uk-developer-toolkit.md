# GOV.UK Developer Toolkit release notes

Version history for the ServiceNow® GOV.UK Developer Toolkit application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.0 - June 2026**

    The GOV.UK Developer Toolkit \(GDS Service Portal\) is a library of reusable, GDS-compliant portal widgets and embeddable components that let ServiceNow developers and partners build citizen-facing portals for UK government customers fast, with minimal customization.
