# Credentials Core release notes

Version history for the Credentials Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.0 - June 2026**

    Added Non-Glide Cobalt Raven ACLs for Credential template and Credentials table.

-   **Version 1.1.0 - May 2025**
    -   New:
        -   Syncs the skills from Credly and associates it to badges.
        -   Adds the skills associated to the users achieved badges to users skills profile \(or\) nudges the user to refresh their skills profile if the skill already exists on user profile.
        -   Identifies new skills and pushes to the skills pipeline for Admin to harmonize.
        -   Captures the newly processed skills from credly source and associates back to the badges and the user profile.
        -   Showcases Credly Badges as part of the employee skill activity insights.
-   **Version 1.0.0 - February 2025**

    Credentials Core application is used to store various forms of digital credentials such as badges, awards, certifications etc.It holds the data about the user and their enrollment status in the badge provider entity along with data on the associated badges for the user. Credentials core application also helps in syncing user badge details from Credly if the spoke is installed.
