# ITOM Cloud Services Core release notes

Version history for the ITOM Cloud Services Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.1.15 - June 2026**

    Improved onboarding job logging: Enhanced log formatting with structured execution tracking for easier troubleshooting.Accessibility improvements: Updated the Diagnostics page and Classic UI dialogs to meet WCAG 2.2 AA standards.Security hardening: Tightened ACL role assignments for improved access control.

-   **Version 4.1.6 - April 2026**
    1.  Hermes records: Improved messaging service reliability on Zurich and Australia releases.
    2.  User check fix: Fixed Diagnostics page to correctly detect locked out service users.
    3.  ACC warning banner: Diagnostics page now displays a warning when ACC Agents are not connected.
-   **Version 4.1.5 - February 2026**
    -   Changed:
        -   Improved stream data continuity after upgrades
        -   Enhanced overall installation experience
-   **Version 4.1.0 - December 2025**
    -   Changed:
        -   Endpoint Check and Error Log Improvements for Onboarding JobEnhanced reliability of onboarding processes with improved endpoint validation and clearer, more accurate error logging.
        -   Security Enhancements
-   **Version 4.0.8 - November 2025**
    -   Fixed:
        -   Resolved an issue with the ACCFreshnessMetric script that could lead to performance issues in production when the number of agents increased significantly.
        -   Added ACLs to multiple Scripted REST APIs to enhance security.
-   **Version 3.4.3 - September 2025**
    -   Fixed:
        -   Resolved an issue with the ACCFreshnessMetric script that could lead to performance issues in production when the number of agents increased significantly.
        -   Added ACLs to multiple Scripted REST APIs to improve security.
-   **Version 3.4.0 - August 2025**

    Fixed: Minor bug fixes

-   **Version 3.3.0 - July 2025**

    Fixed: Minor bug fixes

-   **Version 3.2.1 - May 2025**

    New: Hyperlinks in the ICS diagnostics dashboard are now accessible to customers to help them easily follow root causes back another level.

-   **Version 3.1.1 - February 2025**

    Fixed bugs and implemented improvements for o11y.

-   **Version 3.0.1 - November 2024**

    Bug fixes for topic activation.

-   **Version 2.3.1 - August 2024**

    Changed: Improved topic activation.

-   **Version 2.2.1 - May 2024**

    Changed: Improvements to Agent data collection.

-   **Version 2.1.0 - February 2024**

    The plugin for ITOM Cloud Services generates relevant topics for tenants, enabling them to transmit data from their customer infrastructure to ServiceNow automatically. These topics are predominantly intended for use with Hermes.
