# Google Persistent Disk spoke release notes

Version history for the Integration Hub Google Persistent Disk spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.2 - September 2022**

    Patch version of Google Persistent Disk Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.1 - May 2020**

    Provides actions to automate the management of Google Persistent Disk.
