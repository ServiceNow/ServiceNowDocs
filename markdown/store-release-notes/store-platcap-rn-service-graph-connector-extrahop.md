# Service Graph Connector for ExtraHop release notes

Version history for the ServiceNow AI Platform capabilities Service Graph Connector for ExtraHop on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.3 - September 2020**
    -   ExtraHop provides real-time network visibility across the enterprise by implementing stream processing to transform your network data in flight into structured wire data. ExtraHop Integration for CMDB brings network visibility data into the ServiceNow Configuration Management Database \(CMDB\) at configurable intervals to enrich the discovered device data and establish relationships between devices based on network traffic flow.
    -   This integration relies on the ExtraHop REST API to query for recent device activity. The integration then transforms and maps the device and network traffic data into the appropriate CMDB Configuration Item \(CI\).
