# Virtual Agent for PPM release notes

Version history for the Virtual Agent for PPM on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.1 - May 2023**

    Changed: Updated for localization support.

-   **Version 1.0.0 - August 2020**

    Virtual Agent for PPM provides user assistance to project management professionals through conversations within a messaging interface. This helps project managers view business-related information on their projects quickly in a single step rather than having to navigate through various lists.
