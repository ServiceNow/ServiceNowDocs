# Software Asset Management integration with Tableau release notes

Version history for the Software Asset Management integration with Tableau application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.1 - August 2024**
    -   Software Asset Management application with the Tableau Cloud application enables you to track your software subscriptions and to reclaim stale licenses.
    -   The Tableau Integration provides visibility of user subscriptions and analyze meaningful usage data to identify and reclaim stale subscriptions.
