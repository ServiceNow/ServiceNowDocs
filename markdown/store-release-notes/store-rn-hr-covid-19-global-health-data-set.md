# COVID-19 Global Health Data Set release notes

Version history for the Safe Workplace COVID-19 Global Health Data Set application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

**Note:** To automatically install all the Safe Workplace applications at once, download the ServiceNow® Safe Workplace suite application.

## Version history

-   **Version 1.20.5 - June 2026**

    Fixed: Remediated non-Glide ACLs

-   **Version 1.20.3 - May 2024**

    Security fixes.

-   **Version 1.20.2 - January 2024**

    Security fixes.

-   **Version 1.20.1 - August 2022**

    New: Quality improvement

-   **Version 1.19.0 - March 2022**

    Fixed: Issue where the COVID-19 job sync trigger time was not randomized

-   **Version 1.18.0 - February 2022**

    New: Localization for features from the January 27 release

-   **Version 1.17.0 - January 2022**

    New: Localization for 22 languages other than English. Translations are available for features released up until the December 16, 2021 release.

-   **Version 1.16.0 - December 2021**

    Changed: Updates to the translation engine

-   **Version 1.14.0 - July 2021**
    -   Changed: Updates to the translation engine
    -   Fixed: Data feed bug
-   **Version 1.13.0 - June 2021**

    Fixed: Minor bug related to number of days

-   **Version 1.12.0 - April 2021**

    New: Localization for features from the April 15 release

-   **Version 1.11.0 - April 2021**

    New: Create data feed for Vaccination Status statistics

-   **Version 1.10.0 - March 2021**
    -   New: Localization for features from the February 18 release
    -   Fixed: Minor bug
-   **Version 1.9.0 - February 2021**

    Fixed: The data feed is now delta-syncing correctly

-   **Version 1.8.0 - December 2020**
    -   New: Localization for features from the December 3 release
    -   Fixed: Minor bug fixes
-   **Version 1.7.0 - December 2020**

    New: Ability to view case and death metrics per 100,000 people in a location

-   **Version 1.6.0 - November 2020**

    New: Localization for features from the October 15 release

-   **Version 1.5.0 - October 2020**

    Fixed:

    -   The data feed job now correctly syncs when the system date format is not in the default format
    -   Minor bug fixes
-   **Version 1.4.0 - September 2020**

    Fixed: HTTP 401 error when calling central instance

-   **Version 1.3.0 - September 2020**
    -   New: COVID-19 locations are visible to users without roles
    -   Fixed: The COVID-19 Global Health Data Set data is now completely synced, even when the data server is busy
-   **Version 1.2.0 - August 2020**

    Fixed: Corrected the issue where data sync misses records

-   **Version 1.1.0 - July 2020**

    New:

    -   Localization for features from the July 9 release
    -   Compatible with ServiceNow instances on the Paris release
-   **Version 1.0.9 - July 2020**

    New: Localization for features from the June 18 release

-   **Version 1.0.8 - June 2020**

    New:

    -   Ability to determine if one or more underlying data sources weren't refreshed for a given day
    -   Localization for COVID-19 Global Health Data Set \(v1.0.2\)
-   **Version 1.0.2 - June 2020**

    Inform key business decisions regarding employee health and safety by having access to comprehensive COVID-19 data. The data set is updated daily, and is sourced and aggregated from 4 publicly available data sources. The data set contains a geolocation map component that is used in the Safe Workplace Dashboard, and it can also be leveraged in your own solutions created on the ServiceNow AI Platform.
