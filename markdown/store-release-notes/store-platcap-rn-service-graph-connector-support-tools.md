# Service Graph Connector Support Tools release notes

Version history for the Service Graph Connector Support Tools application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.0 - February 2024**
    -   Service Graph Connector Support Tools has prebuilt dialogs that enable administrators resolve common implementation issues with Service Graph Connectors.
    -   The application helps troubleshoot issues with:
    -   -   Connections
-   Data
-   Performance
-   Mapping
