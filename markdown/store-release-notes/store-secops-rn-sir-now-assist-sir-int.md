# Now Assist for Security Incident Response integrations release notes

Version history for the ServiceNow® Now Assist for Security Incident Response integrations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.1 - June 2026**

    The Now Assist for SIR Integration Toolkit is designed to facilitate the creation of new integrations with the Security Incident Response application. It leverages the Now Assist code and spoke generation capabilities to simplify the setup process, enabling users to quickly develop and deploy integrations. This features an intuitive user interface, clear code samples, and automation-driven workflows, which collectively reduce setup time and complexity. This integration process is streamlined within ServiceNow Studio, allowing for efficient testing and deployment while maintaining security and compliance oversight.
