# Security Support Common Orchestration release notes

Version history for the Security Operations Orchestration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 12.13.4 - February 2025**

    Changed: Migrated base system workflows to Flow Designer flows.

-   **Version 12.13.2 - November 2024**

    Changed: Supporting changes for migrating workflow to flows using flow designer related to Sightings Search and Block Request capabilities.

-   **Version 12.13.1 - August 2024**

    Changed: UI improvements to support observables enrichment.

-   **Version 12.12.1 - May 2024**

    No new features or updates included with this version. This version ensures that right Security Support Common version is installed when Vulnerability Response is installed.

-   **Version 12.11.1 - March 2024**

    Changed: Updated the support common release version.

-   **Version 12.11.0 - February 2024**

    No new features or updates included with this version. This version ensures that features from the last release are compatible with the Washington DC family release.

-   **Version 12.10.0 - December 2023**

    No existing features were impacted with this version.

-   **Version 12.8.1 - November 2023**

    Changed: Updated the dependent version of Security Support Common \(sn\_sec\_cmn\).

-   **Version 12.7.1 - August 2023**

    Changed: Updated dependency application version.

-   **Version 12.6.2 - June 2023**

    Version 12.6.2 certified for Tokyo and Utah.

-   **Version 12.6.0 - May 2023**

    Version 12.6.0 certified for Utah, Tokyo, and San Diego.

-   **Version 12.5.1 - February 2023**

    No new features with this release. This version of the application is being released to maintain compatibility and its place in the hierarchy with other dependency applications.

-   **Version 12.4.4 - November 2022**

    Changed: \[Security support orchestration\] Added logic to migrate the Threat lookup trigger to Capability Framework v2.

-   **Version 12.3.0 - August 2022**

-   **Version 12.2.1 - May 2022**

-   **Version 12.1.0 - March 2022**

    Changed: Support for dependency hierarchy

-   **Version 12.0.0 - February 2022**
