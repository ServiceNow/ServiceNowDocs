# release notes

Version history for the ERP Semantic Mining application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 7.0.5 - May 2025**
    -   ERP Customization Mining highlights for the Yokohama Store 2 release: Control the machine learning training in the mining process.
    -   New in the Yokohama Store 2 release: Reset AI/ML processing manually The Reset AI/ML analysis option on the configuration tab allows you to reset the analysis so the flow can run again.
-   **Version 6.0.6 - February 2025**
    -   New in the Yokohama release:
        -   The home page provides a dashboard containing various charts and graphs showing metrics related to ERP Customization Mining.
        -   The SQLM and APPSTATS scan is now done monthly.
        -   Take guided tours within ERP Customization Mining to learn about features and to complete tasks step by step.
-   **Version 5.0.4 - November 2024 \(Xanadu\)**
    -   Export and import scan results by using the snapshot option.
    -   Re-trigger AL/ML execution on demand.
-   **Version 4.1.5 - August 2024**

    No changes for ERP Mining for the Xanadu release.

-   **Version 4.0.6 - May 2024**

    New: Support for notifications for successful and failed mining jobs.

-   **Version 1.0.14 - February 2023**

    ERP Customization Mining is a workspace that helps provide an overview of your ERP customizations and provide a foundation for replacing, improving and innovating workflows with low-code apps on ServiceNow while preserving your system of records.

    By mirroring any ERP data such as purchase documents, sales documents, data records etc. the app makes it possible to leverage the data on the apps you build using ServiceNow AI Platform.
