# Flow Templates for Cloud Services release notes

Version history for the Flow Templates for Cloud Services integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.2.3 - January 2023**

    Fixed: Dependencies.

-   **Version 1.2.1 - October 2022**

    Fixed: Improve installation performance of application.

-   **Version 1.1.0 - October 2021**

    Easily automate cloud services requests in your custom applications with the Flow Templates for Cloud Services application. Rather than creating a new flow from scratch, users can choose the relevant template in App Engine Studio and follow the guided screens to create a new flow. These templates leverage IntegrationHub spokes to manage virtual machines, virtual cloud networks and subnets, and cloud storage/DBs across a wide range of 3rd party providers.
