# Cisco AI Defense Integration for AI Security Exposure Management release notes

Version history for the ServiceNow® Cisco AI Defense Integration for AI Security Exposure Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.1 - June 2026**

    The Cisco AI Defense integration can be used to import AI security exposures such as model vulnerabilities and model validation findings \(automated red teaming alerts\) into AI Security Exposure Management and automate workflows for remediation.
