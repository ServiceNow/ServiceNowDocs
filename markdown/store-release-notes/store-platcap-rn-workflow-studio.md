# Workflow Studio release notes

Version history for the Workflow Studio application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 28.4.1 - May 2026 \(Zurich\)**

    Changed: Support for triggers as a service.

-   **Version 29.1.1 - April 2026 \(Australia\)**

    Changed: Updated version number. No major changes this release

-   **Version 29.0.5 - March 2026**

    New: Support flow\_admin role for more granular access to Workflow Studio modules and UI.

-   **Version 28.3.3 - March 2026 \(Zurich\)**

    Changed: There are no major changes in this release.

-   **Version 27.4.2 - January 2026**
    -   New: Improve memory performance for flow generation local storage
    -   Fixed: Fixed Custom schema Ui issues in Playbooks
-   **Version 28.2.3 - December 2025 \(Zurich\)**

    New: Support for the theme switcher that allows switching between light and dark themes.

-   **Version 28.1.4 - September 2025 \(Zurich\)**

    New: Support for generation of subflow inputs and outputs.

-   **Version 28.0.2 - August 2025 \(Zurich\)**

    New: Introduced a guided user journey in Workflow Studio to help users build the appropriate workflows by providing interactive assistance, tool descriptions, and decision-making support.

-   **Version 27.3.2 - August 2025 \(Yokohama\)**

    Create a flow from text directions by using Now Assist.

-   **Version 27.2.3 - May 2025 \(Yokohama\)**
    -   New:
        -   Implemented a guided user journey in ServiceNow Workflow Studio to help users build the appropriate workflows by providing interactive assistance, tool descriptions, and decision-making support
        -   Added dark theme support
    -   Fixed: Accessibility issues
-   **Version 27.1.4 - February 2025 \(Yokohama\)**
    -   New:
        -   Support 200% and 400% zoom for operations and integrations tabs
        -   Support 400% zoom for homepage
    -   Fixed:
        -   Expand/Collapse functionality issues in Operations tab
        -   Flow not generated when additional property "Run with roles" is used
        -   Generate flow with Now Assist not enabled until clicking on try example.
-   **Version 26.3.1 - February 2025 \(Xanadu\)**
    -   New:
        -   Show all conversational compatible actions and subflows in a new tab. These actions and subflows can be configured to be available in conversational interfaces.
        -   Show all conversational enabled actions and subflows in a new tab. These actions and subflows have been configured for one or more conversational interfaces.
    -   Fixed:
        -   Fixed showing the same record on both the Conversational Compatible and Conversational Enabled tabs.
        -   Fixed not displaying the Description field.
        -   Fixed hash tag functionality not working when AI Search is installed before an upgrade.
-   **Version 25.2.7 - January 2025 \(Washington DC\)**

    Fixed: Add new patch for PAD DEP Version. There is no new Workflow Studio functionality being shipped in this patch.

-   **Version 26.2.6 - November 2024 \(Xanadu\)**
    -   New: Performance improvements
    -   Changed: Cosmetic UI changes
    -   Fixed: Accessibility issues fixes
-   **Version 26.1.3 - August 2024 \(Xanadu\)**
    -   New:
        -   Add + button in tabs to create new assets in Workflow Studio
        -   Create custom categories for your workflow items
        -   Open Flow operations in Workflow Studio
        -   Show banner if new store versions available for Workflow Studio and its dependent builders
    -   Changed: Updated dashboards for flows and playbooks
-   **Version 25.2.5 - June 2024**

    Add new patch for decision builder. There is no new Workflow Studio functionality being shipped in this patch.

-   **Version 25.2.4 - May 2024**
    -   New:
        -   Added plus button in tabs to create new workflow objects.
        -   Added information banner to show when new Store app versions are available.
    -   Fixed:
        -   Open Flow operations in Workflow Studio
        -   Performance improvements
-   **Version 25.1.3 - March 2024**

    Fixed: Improved performance.

-   **Version 25.1.1 - February 2024**

    Workflow Studio is a single place for all things worklfow automation. Developers can create playbooks, flows, actions, spokes, and decisions using Workflow Studio.
