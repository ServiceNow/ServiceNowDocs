# Observation Framework release notes

Version history for the Observation Framework on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - April 2021**
    -   New: The observation framework invite-only app helps your analyst to:
        -   Observe an increase in incidents over time.
        -   Automatically group the records into topics with Predictive Intelligence Clustering.
        -   Provide insights and recommended actions for each topic to close the loop.
