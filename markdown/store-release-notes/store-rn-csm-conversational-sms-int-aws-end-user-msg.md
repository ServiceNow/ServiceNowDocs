# Conversational SMS Integration with AWS End User Messaging release notes

Version history for the Conversational SMS Integration with AWS End User Messaging application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.2 - February 2025**

    This app enables end users to interact with ServiceNow Virtual Agent and live agents through an SMS interface using the AWS End User Messaging service as SMS provider.
