# AWS Elastic Beanstalk spoke release notes

Version history for the Integration Hub AWS Elastic Beanstalk spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.2 - September 2023**

    Fixed: The license requirements.

-   **Version 1.0.1 - April 2023**

    Fixed: Improve installation performance of spoke.

-   **Version 1.0.0 - June 2020**

    Provides actions to automate the management of applications and environments in Amazon Elastic Beanstalk.
