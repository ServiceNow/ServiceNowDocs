# Employee Experience Pack: Promotions release notes

Version history for the Employee Experience Pack: Promotions on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.1.0 - October 2022**

    -   New
        -   These update sets contain a Promotion Lifecycle Event modeled after implementation best practices.
        -   Included content:
            -   1 Lifecycle Event consisting of 7 activity sets
                -   Change Requested
                -   HR Review and Validation
                -   Approvals
                -   Formal Communication to Employee
                -   Planning for Transition
                -   New Job Starts
                -   Integration into New Team
            -   27 Activities:
                -   2 Checklist \(including manager checklist and communication checklist\)
                -   15 Mark When Complete \(including communicating new job details to an employee, creating transition plans, etc.\)
                -   3 Take Survey \(pulse survey mid-way thru transaction, surveys after a promotion is finalized\)
                -   1 Email notification
                -   5 approvals
                -   1 e-signature
            -   3 HR services
                -   Promotion Request
                -   General Mobility Question
                -   Letter of Promotion
            -   16 Knowledge articles
            -   1 Guided Setup
            -   3 Flows and:
                -   Flow: Promotion - Create/Update Job Record
                -   Subflow: Promotion - Update Missing Fields
                -   Subflow: Promotion - Update LE Case Record
            -   1 Business Rule: Promotion - Set to Draft
    -   Changed
        -   Compatibility with the Rome release
        -   Fix for data mapping in the Promotion Letter document template is available in a separate update set included with the download.
    Please note that version 3.1.x of this Experience Pack is officially compatible with the San Diego release only.
