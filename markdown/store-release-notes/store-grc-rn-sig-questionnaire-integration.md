# GRC: SIG Questionnaire Integration release notes

Version history for the GRC: SIG Questionnaire Integration Integration on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.3.0 - June 2026 \(Australia\)**
    -   New:
        -   Added SIG Core 2026 with 17 security and compliance domains.
        -   Added SIG Lite 2026 smart assessment template.
        -   Added SIG Detail 2026 questionnaire template.
    -   Changed: Added comment and work notes read access for vendor assessment reviewer role on SIG questionnaires.
    -   Fixed:
        -   Ensured in-flight assessments continue without interruption when template version is retired.
        -   Removed business user role from vendor detail page widget to improve security.
-   **Version 22.0.0 - March 2026**

    Fixed: An issue where the admin role could not update certain fields has been resolved.

-   **Version 21.1.0 - December 2025 \(Zurich\)**

    Changes: Updates have been made to enable third-party risk management admin to do necessary updates to SIG.

-   **Version 21.0.2 - August 2025**

    New: 2025 SIG assessment templates based on Smart Assessment Engine have been added.

-   **Version 20.1.1 - May 2025**

    New: SIG 2025 Content \(All\) has been added.

-   **Version 18.1.2 - July 2024**

    Fixed: Fixed app compatibility for the glide family releases.

-   **Version 18.1.1 - May 2024**
    -   New: Added SIG 2024 questionnaire content.
    -   Fixed: Added report view ACL to table: sn\_sig\_asmt\_parsing\_status.
-   **Version 17.0.0 - August 2023**

    New: Added ability for Third-party Assessors to answer SIG questionnaires on behalf of the third-party.

-   **Version 16.0.2 - March 2023**

    New: Added SIG 2023 questionnaires.

-   **Version 16.0.1 - February 2023**

    New: Added SIG 2023 questionnaires.

-   **Version 15.0.2 - December 2022**

    Fixed: Minor reduction of the installation size of the application.

-   **Version 15.0.1 - September 2022**

    Fixed: Issues automatically created when the vendor submits the correct responses to SIG questionnaire questions.

-   **Version 15.0.0 - August 2022**
    -   Fixed:

        -   Certain dependencies showed up in incorrect order
        -   Set "Not applicable" option on SIG questions to 'On' by default
-   **Version 14.1.1 - March 2022**
    -   New: Added Sig Lite 2022, Sig Core 2022, and SIG Detail 2022
    -   Fixed:
        -   Stopped issues from being created mistakenly with SIG Lite 2021
        -   Fixed upgrade issue that would overwrite question changes in the template
-   **Version 13.0.0 - September 2021**

    Fixed: Added ability to configure document parser

-   **Version 12.0.1 - June 2021**
    -   Fixed: Issue that caused question mark character to be added to some of the metric questions in the SIG Full 2020 template.
    -   New: Added SIG 2021 Questionnaires
-   **Version 11.0.2 - October 2020**
    -   Fixed:
        -   Risk scoring for SIG 2020 questionnaires
        -   Demo data for SIG
    -   Changed: Disable SIG 2017 questionnaires
-   **Version 10.0.0 - March 2020**

    New: Support for SIG 2020

-   **Version 9.0.0 - November 2019**

    This feature was updated in Madrid and is still compatible with New York.
