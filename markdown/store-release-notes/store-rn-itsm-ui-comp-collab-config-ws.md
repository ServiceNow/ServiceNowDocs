# UI Components of Collaboration for Configurable Workspaces release notes

Version history for the UI Components of Collaboration for Configurable Workspaces on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 9.2.0 - June 2026**
    -   Changes:
        -   Updated plugin dependencies to ensure compatibility with the ServiceNow Australia and later releases
        -   Minor defect fixes
-   **Version 1.2.2 - March 2026**
    -   New: To support the overall workspace adoption, the following UI16 redirection is available from the classic \(UI16\) module menus to the SOW equivalent experience.
    -   For new instances, SOW redirection is ON by default, all users are redirected to the equivalent SOW experience.
        -   This setting can be switched only with ServiceNow consultation.
    -   For upgraded instances, a configuration setting is available at the record level for SOW redirection.
        -   Once the configuration is activated at a record level, all users of that record type isredirected to the equivalent SOW experience.
        -   The configuration setting can be toggled as required.
-   **Version 1.2.0 - December 2025**

    No changes.

-   **Version 1.1.0 - September 2025**

    New: Added Collaboration as a UIB component and enabled it in the component panel for use on any UIB page for call and chat features.

-   **Version 1.0.0 - August 2025**
    -   Notify UI component provides the conference call, telephony, and SMS capabilities within the Service Operations Workspace. Using the Notify UI component framework, you can:
        -   Initiate and manage conference calls on either Zoom, MS Teams, or Webex.
        -   Send an SMS to the required stakeholder.
        -   Initiate and manage telephony calls \(for example, Twilio\) from ServiceNow.
