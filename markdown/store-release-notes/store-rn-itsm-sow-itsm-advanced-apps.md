# Service Operations Workspace ITSM Advanced Applications release notes

Version history for the ServiceNow® Service Operations Workspace ITSM Advanced Applications application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 9.2.1 - June 2026**
    -   Changes:
        -   Updated plugin dependencies to ensure compatibility with the ServiceNow Australia and later releases.
        -   Minor defect fixes.
-   **Version 8.7.1 - June 2026**
    -   Changed:
        -   SOW applications are reorganized as per the following application packages:
            -   SOW ITSM Applications \(Incident, Request\)
            -   SOW ITSM Advanced Applications \(Change, Problem, Major Incident Management \(MIM\), On-Call Scheduling, Walk-up Experience\)
    -   For existing customers:
        -   Family release upgrade: All applications will be automatically upgraded to the default version available in the Australia family release.
        -   Store version upgrade: Customers installing SOW 8.7 &amp; above from the ServiceNow Store must install both application packages listed above.
-   **Version 9.1.1 - April 2026**

    Streamline IT Service Management workflows and processes for modern operations. You can effectively manage walk-up appointments, and streamline change management in Service Operations Workspace. All of these workflows leverage the power of capabilities such collaborations, AI based recommendations and on-call scheduling.
