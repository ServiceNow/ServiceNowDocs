# Sustainable IT release notes

Version history for Sustainable IT on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 22.3.0 - June 2026 \(Australia\)**
    -   Changed:
        -   ESG reader access to Sustainable IT tables — The ESG reader role now has read access to Sustainable IT tables, improving cross-module data visibility.
        -   This release includes security enhancements that strengthen access controls across the application.
-   **Version 21.1.0 - December 2025 \(Zurich\)**

    New: Minor enhancement to expose 'Active' field configuration for Choice options.

-   **Version 21.0.1 - August 2025**
    -   Changed: Refreshed demo data to align with current feature capabilities and use case scenarios.
    -   Fixed: Important security updates
-   **Version 20.1.0 - May 2025**
    1.  Added labels to dropdown present in IT footprint map, product drill down page, hardware assets and data center page.
    2.  Fixed scroll issue on datacenter tab for the Single score widget.
    3.  Added support for Coral theme and accessibility standards.
-   **Version 20.0.2 - February 2025**
    -   Fixed:
        -   Improved the component to meet WCAG 2.1 AA guidelines, including the ability to reflow zooming to 400%.
        -   Fixed a problem with breadcrumb labels.
        -   Fixed an issue with 'Data center efficiency' widget.
-   **Version 19.0.6 - August 2024**
    -   New:
        -   Ability to drill down from locations and map them to energy use and emissions of Hardware models.
        -   Ability to drill down from model categories and map them to energy use and emissions of Hardware models.
        -   Show the top 6 models based on energy and emissions on the new drill-down page
-   **Version 18.0.1 - February 2024**
    -   New: The following new tables are added to the sn\_esg\_sustain application:
        -   sn\_esg\_sustain\_map\_marker\_configuration
        -   sn\_esg\_sustain\_map\_data\_range\_configuration
    -   Changed: 'Top 3 greenest data centers 'vizualisation is now changed to 'Top 3 energy efficient data centers - DCIE' and the data is now displayed based on DCIE instead of Emission.
-   **Version 17.0.1 - August 2023**
    -   New
        -   Use the Sustainable IT application and purpose-built dashboard to effectively manage and monitor the emissions generated by your IT hardware assets and data centers.
        -   Track the proper disposal of your IT hardware assets after they reach the end of their lifespan.
        -   Track PUE, CUE and WUE indicators for each data center.
