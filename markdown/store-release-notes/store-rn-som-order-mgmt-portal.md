# Order Management Portal release notes

Version history for the Order Management Portal on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.0 - August 2025 \(Yokohama\)**

    Install the Sales Cart plugin \(sn\_sales\_cart\) to enable cart-related functionality in the Business Portal.

-   **Version 1.4.0 - July 2025**

    Order management business portal enable business-to-business \(B2B\) customers to view and configure products in product catalogs and place orders.
