# Cloud Security for Cloud Workspace release notes

Version history for the ServiceNow® Cloud Security for Cloud Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.3 - April 2026**

    Fixed: The Accounts with Critical Security Findings widget on the Compliance Dashboard in the Cloud Workspace now displays its border correctly.

-   **Version 1.0.1 - December 2025**
    -   Cloud Workspace provides a centralized, consistent experience for managing all ServiceNow cloud solutions as part of the Cloud Governance Suite. It uses a unified data model to ensure that different cloud features work seamlessly together and share the same information, reducing silos and improving visibility.
    -   One of the key features in Cloud Workspace is Vulnerability Response \(VR\), which is available through the Security Operations Vulnerability Response offering. VR focuses on exposing vulnerabilities and misconfigurations for assets on public cloud accounts.
    -   Why It Matters for CCoE
        -   Proactive Risk Management: Manage vulnerabilities early to prevent security breaches and compliance violations
        -   Unified Governance: Integrate vulnerability insights into the broader cloud governance framework for holistic control
        -   Operational Efficiency: Reduce manual effort by automating vulnerability management and remediation workflows
    -   Business Value for CCoE
        -   By embedding Vulnerability Response within Cloud Workspace, organizations can:
            -   Strengthen cloud security posture without adding complexity
            -   Ensure compliance with internal and external regulations
            -   Improve visibility and governance across multi-cloud environments
