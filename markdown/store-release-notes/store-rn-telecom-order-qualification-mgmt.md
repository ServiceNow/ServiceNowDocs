# Order Qualification Management release notes

Version history for the Order Qualification Management on the Order Qualification Management.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.5.6 - June 2026 \(Australia\)**

    Fixed: Enhanced security by introducing query range ACLs.

-   **Version 4.5.0 - June 2026 \(Zurich\)**

    Fixed: Minor security fixes

-   **Version 4.4.2 - May 2026**

    Fixed: A minor security fix for the read-only option

-   **Version 4.0.0 - February 2024**
    -   New: localisation support
    -   Fixed: External ID related issue fixed
-   **Version 3.0.0 - November 2023**
    -   New: Moved API code to new telecommunications application \(sn\_om\_tmt\)
    -   Fixed: Use state instead of status in TSQ POST response
-   **Version 2.0.0 - August 2023**

    Fixes.

-   **Version 1.0.0 - February 2023**
    -   New:
        -   Support for open API to ingest service qualification request
        -   Features to manage service qualification requests
        -   Support for triggering outbound service qualification requests to external systems
