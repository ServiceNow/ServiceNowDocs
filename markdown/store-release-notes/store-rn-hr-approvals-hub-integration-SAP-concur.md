# Approvals Hub integration with SAP Concur release notes

Version history for the Approvals Hub integration with SAP Concur application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - June 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 1.4.0 - March 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 1.3.2 - December 2025**
    -   Added the following enhancements:
        -   Accessibility enhancements with Approvals hub for Concur to ensure an inclusive experience for users.
        -   The 'Send Back Request' action includes updated labels on the form elements, addressing critical accessibility issues.
-   **Version 1.3.1 - November 2024**

    Uptake of latest version of dependent apps.

-   **Version 1.2.1 - February 2024**

    Added out-of-the-box configurations to enable approvers to filter SAP Concur approvals and show SAP Concur Approvals in Manager Hub.

-   **Version 1.1.1 - November 2023**

    Addressed minor issues such as allowing admin users to edit the scheduled jobs for pulling Concur reports and to show relevant error messages when Concur API failure occurs.

-   **Version 1.1.1 - November 2023**

    This app allows you to bring SAP Concur expense reports directly into ServiceNow Employee Center Pro - providing a centralized interface for seamless approvals. This includes the ability to view expense reports, expense entries, itemizations, and attachments. Approvers can approve or send back Concur expenses to the requestor right from Employee Center Pro.
