# Localization Workspace release notes

Version history for the Localization Workspace application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.1.2 - June 2026**
    -   New in this version:
        -   improved guided tours
        -   improved UI when importing glossaries
        -   improved UI when adding translated terms to a glossary
        -   exporting glossaries
        -   creating a role dedicated to terminology management
-   **Version 3.0.2 - March 2026**
    -   New in this release:
        -   Guided tours to set up language providers and language groups
        -   Language Asset Management: upload and edit glossaries of terms and their translations
        -   Cancel translation requests, delete draft requests, and review archived requests from the Home screen
        -   View the translation method from the translation request details
        -   Use intelligent due date suggestions based on the scope of your requests
-   **Version 2.0.2 - December 2025**
    -   Centralized Language Framework Configuration within the Workspace.
    -   Language Groups \(favorites\) simplify translation requests management in bulk.
    -   Bulk Select or Deselect Content rather than each item individually.
    -   Translation Item Search for specific content items to include or exclude them from your translation list.
    -   Optional Due Date applied to requests
    -   Guided Setup for Admins to help correctly configure the Localization Framework and Workspace.
-   **Version 1.1.0 - August 2025**
    -   Localization Workspace highlights for this release
        -   Dynamic artifact detection enables Localization Workspace to identify all translatable content, including customers' own artifacts.
        -   With status synchronization, translation requests in Localization Workspace reflect the same status as those in Localization Framework.
-   **Version 1.0.5 - May 2025**

    ServiceNow Localization Workspace is a targeted translation management experience for content owners and translation fulfillers. With Localization Workspace you can manage and deliver multilingual content at scale. Localization Workspace is a new application in the Yokohama release.
