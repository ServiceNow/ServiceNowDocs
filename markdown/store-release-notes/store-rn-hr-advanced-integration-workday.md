# HR Service Delivery Advanced Integration with Workday release notes

Version history for the HR Service Delivery Advanced Integration with Workday on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.6 - December 2025**

    Minor fixes.

-   **Version 2.2.2 - May 2025**

    Bug fixes.

-   **Version 2.2.1 - March 2025**

    New: Added OOB subflows which allows employees to update personal details such as email, address, and date of birth.

-   **Version 2.1.1 - November 2024**
    -   The delivered integrations between ServiceNow HR Service Delivery and Workday HR application include the following experiences:
        -   Request Time Off
            -   Simple \(Link to Workday Application\)
            -   Advanced \(Contextual Time Off Experience\)
            -   Legal Name Change \(VA Conversation\)
        -   Payroll Discrepancy
        -   Total Rewards Widget
        -   Holiday Calendar Widget
-   **Version 3.2.1 - November 2024**
    -   The delivered integrations between ServiceNow HR Service Delivery and Workday HR application include the following experiences:
        -   Employee synchronization
        -   Pull to-dos
-   **Version 2.0.1 - August 2024**
    -   The delivered integrations between ServiceNow HR Service Delivery and Workday HR application include the following experiences:
        -   Legal Name Change \(VA Conversation\)
        -   Request Time Off
            -   Simple \(Link to Workday Application\)
            -   Advanced \(Contextual Time Off Experience\)
        -   Payroll Discrepancy
        -   Total Rewards Widget
        -   Holiday Calendar Widget
-   **Version 1.2.2 - June 2024**
    -   The delivered integrations between ServiceNow HR Service Delivery and Workday HR application include the following experiences:
        -   Legal Name Change \(VA Conversation\)
        -   Request Time Off
            -   Simple \(Link to Workday Application\)
            -   Advanced \(Contextual Time Off Experience\)
        -   Payroll Discrepancy
        -   Total Rewards Widget
        -   Holiday Calendar Widget
-   **Version 1.2.1 - February 2024**

    Minor fixes.

-   **Version 1.2.0 - November 2023**
    -   New: Integrate HR Service Delivery Integration with Workday with localization tools.
    -   Fixed: Error when disabling Payroll COE when the HR Service Delivery Advanced Integration with Workday plugin is installed.
-   **Version 1.1.0 - August 2023**

    New: OAuth 2.0 Authentication.

-   **Version 1.0.4 - November 2022**

    Minor optimization and refactoring

-   **Version 1.0.3 - February 2022**
    -   Fixed:
        -   NLU Model and Intent mapping is missing from the VA topic "Workday Legal Name Change \(Template\)" for San Diego instances
        -   Payroll Use-case: Sum of all taxable wages should not be displayed
        -   Security fixes
        -   View Workday Payslip widget does not work as a standalone widget
        -   Request time off doesn’t work in San Diego
    -   Changed: Add Employee Service Center as dependency
-   **Version 1.0.2 - November 2021**

    Fixed: Fix for spoke version mismatch for the Advance integration

-   **Version 1.0.1 - July 2021**

    New: HR Service Delivery Advanced Integration with Workday
