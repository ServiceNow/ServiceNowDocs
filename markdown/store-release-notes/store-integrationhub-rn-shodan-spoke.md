# Shodan Spoke release notes

Version history for the Shodan Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.0 - February 2024**

    New: Added authentication templates, to help in the spoke setup using the Connections dashboard.

-   **Version 1.0.1 - September 2022**

    Patch version of Shodan Spoke for IntegrationHub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.0 - February 2021**

    New: Provides actions to automate the management of Shodan tasks.
