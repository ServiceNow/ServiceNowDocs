# IT Service Management for Microsoft 365 release notes

Version history for the IT Service Management for Microsoft 365 application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.12.3 - June 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 2.12.2 - March 2026**

    Updated to support the latest version of the dependent apps.

-   **Version 2.11.1 - December 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 2.9.1 - August 2025**

    Updated to support the latest version of the dependent apps.

-   **Version 2.8.0 - May 2025**

    Enable Copilot integration through the ServiceNow for Microsoft 365 from XP9 and YP3. Enhance login mechanism on the second load for the Employee Center tab in Servicenow for Teams \(V2\) for the self-configurable apps.

-   **Version 2.7.13 - November 2024**

    Uptake of latest version of the dependent apps.

-   **Version 2.7.6 - August 2024**

    Uptake of latest version of the dependent apps.

-   **Version 2.7.0 - May 2024**

    Replaced Business Rules based custom actionable notifications with VA notifications.

-   **Version 2.6.11 - February 2024**

    Minor fixes.

-   **Version 2.6.10 - November 2023**

    Employees get a connected and unified employee experience with direct access to Employee Center and Virtual Agent in Microsoft Teams, Outlook and Microsoft 365. Agents can better collaborate and communicate with individuals and groups on incidents and other tickets through Microsoft Teams.
