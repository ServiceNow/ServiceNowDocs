# Trello spoke release notes

Version history for the Integration Hub Trello spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.4.0 - November 2025**

    Security patch: Improve sample webhook registry security.

-   **Version 1.3.0 - November 2024**

    Evaluated and modified sandbox access for client-callable script includes as false.

-   **Version 1.2.2 - January 2024**

    Fixed: Misconfiguration of table/field ACLs within 'com.sn.trello.spoke' plugin.

-   **Version 1.2.1 - June 2023**
    -   Added about 50 new actions that align with Trello's core business functions
    -   Improve webhook security
    -   Created and applied report\_view ACLs
-   **Version 1.1.0 - February 2022**

    Fixes.

-   **Version 1.0.1 - August 2021**

    Made changes to support internationalization.
