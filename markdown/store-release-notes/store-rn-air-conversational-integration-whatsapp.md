# Conversational Integration with WhatsApp \(powered by Twilio\) release notes

Version history for the Now Intelligence Conversational Integration with WhatsApp \(powered by Twilio\) on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.5 - August 2023**

    Changed: Updated Twilio Spoke version dependency.

-   **Version 2.0.4 - February 2023**

    Fixed: Minor fixes.

-   **Version 1.0.0 - December 2020**

    Enable brands to engage with their customers using the WhatsApp messaging app. The app requires subscription to the Twilio services.
