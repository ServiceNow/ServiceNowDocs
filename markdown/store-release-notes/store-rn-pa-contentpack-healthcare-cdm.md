# Performance Analytics Content Pack for Healthcare CDM release notes

Version history for the Performance Analytics Content Pack for Healthcare CDM application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.0 - May 2024**

    New: Added minor performance improvements.

-   **Version 3.0.0 - February 2024**

    New: Add minor enhancements to true up to the Washington release.

-   **Version 2.0.0 - November 2023**

    Minor changes.

-   **Version 1.1.0 - August 2023**

    First release: Provides Performance Analytics capabilities and dashboard for healthcare clinical device management applications. We have provided a dashboard and reports that are useful for leaders in clinical engineering or biomedical engineering departments of hospitals, clinics and specialty chains. This dashboard and associated reports can be used to meet audit and compliance requirements.
