# UKG spoke release notes

Version history for the UKG spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.6.0 - June 2026**
    -   Changed:
        -   Added automatic OAuth token refresh to align with the latest UKG API requirements.
        -   Access control lists \(ACLs\) have been updated to align with current platform security standards.
-   **Version 3.5.0 - December 2025**

    Minor fixes.

-   **Version 3.4.0 - November 2024**

    Security fixes.

-   **Version 3.3.0 - August 2024**

    Added authentication template for one-click auth setup.

-   **Version 3.2.4 - March 2024**
    -   Fixed:
        -   Change Person Number to a String type for retrieve employee schedule action
        -   Added more output fields to retrieve employee schedule action
-   **Version 3.2.3 - September 2023**

    Fixed: 'licensable' field set to be true in plugin.xml.

-   **Version 3.2.2 - September 2022**

    Patch version of UKG Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 3.2.1 - December 2021**

    Changed: Patch release of UKG Spoke for IntegrationHub. The version includes security-related changes.

-   **Version 3.2.0 - June 2021**

    New: This is a minor release for UKG Spoke of IntegrationHub. It includes 5 new actions: Look up Manager Availability Requests, Look up Shift Swap Requests Manager, Look up Self Schedule Requests Manager, Look up Open Shifts Requests Manager, and Look up Shift Cover Requests as Manager.

-   **Version 3.1.0 - March 2021**

    New: The minor release for UKG spoke of IntegrationHub. This version includes a new action “Get Employee Accruals by Person Number” and minor changes to inputs for “Retrieve Manager Time Off Requests By Person Number, Retrieve Manager Time Off Requests By Employee ID” actions.
