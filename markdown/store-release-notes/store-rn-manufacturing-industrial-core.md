# Industrial Core release notes

Version history for the Industrial Core on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.0 - June 2026**

    New: Certified for Australia Patch 3

-   **Version 3.1.0 - March 2026**
    -   New:
        -   The OT Discovered Subnet \[sn\_ot\_discovered\_subnet\] table was added to the Industrial Core plugin
        -   Use the OT device naming strategy to create unique names for your OT devices
-   **Version 3.0.0 - December 2025**

    Changed: Minor Platform enhancement and a bug fix.

-   **Version 1.0.4 - September 2025**

    Fixed: Minor bug fixes.

-   **Version 1.0.3 - August 2025**

    Industrial Core plugin allows to package pre-requisite tables for integrations and OT licensing. The Industrial core plugin serves as a centralized solution for OT-specific utilities and data model changes. For example, the ot\_class\_mapping\_base is specifically required for OT SGC connectors who want to leverage the class mappings in a table.
