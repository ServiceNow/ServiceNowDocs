# @servicenow/now-score release notes

Version history for the @servicenow/now-score application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 23.0.8 - March 2023**

    Bug fixes

-   **Version 23.0.5 - February 2023**
    -   New:
        -   Added the possibility to add an icon to the single score
        -   Allow configuring conditional icon colors
    -   Fixed: Bug fix
    -   Single Score is a part of the Data Visualization component. It presents a view of a single score from any ServiceNow or custom data.
    -   The Single Score data visualization can be used in the Next Experience and is available in Next Experience Dashboards, the UI Builder, and the Visualization Designer.
    -   This application is installed automatically with the "Data Visualization Core" application.
