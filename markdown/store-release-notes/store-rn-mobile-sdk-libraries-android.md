# Mobile SDK Libraries - Android release notes

Version history for the Now® Mobile SDK Libraries - Android application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.22.0 - June 2026**
    -   Voice Agent support
    -   General stability improvements
-   **Version 2.21.1 - May 2026**

    General stability improvements

-   **Version 2.20.0 - April 2026**

    General stability improvements.

-   **Version 2.19.0 - March 2026**
    -   Fixed minor issues in the NowChat SDK
    -   Fixed a WebView zooming and caching issue in the NowWeb SDK.
-   **Version 2.18.0 - February 2026**

    Fixed minor UX issues in the virtual agent.

-   **Version 2.17.0 - January 2026**

    Fixed minor UX issues in the virtual agent.

-   **Version 2.16.0 - December 2025**

    Minor bug fixes.

-   **Version 2.15.0 - November 2025**

    Minor bug fixes.

-   **Version 2.11.0 - July 2025**

    Added new alert text icon color to NowChatThemeInterface.

-   **Version 2.10.0 - June 2025**

    General bug fixes and stability improvements.

-   **Version 2.9.0 - May 2025**

    Added support for on-demand theme changing.

-   **Version 2.8.0 - April 2025**

    Added callback to expose chat Session ID when chat session is ended and the chat window is closed.

-   **Version 2.7.0 - March 2025**
    -   Added ability to enable Now Assist in SDK
    -   Supported Theming for NowWeb
-   **Version 2.6.0 - February 2025**

    Updated the UI for the Virtual Agent component.

-   **Version 2.2.0 - May 2024**
    -   Provide ability to start a new conversation when entering Virtual Agent.
    -   Provide ability to hide the new conversation button in Virtual Agent.
    -   Provide ability to end on-going conversation when exiting chat.
-   **Version 2.1.0 - February 2024**
    -   Added multi attachment upload in the NowWeb SDK
    -   Added light and dark theme support in the NowWeb &amp; NowChat SDK
    -   Added Engagement Messenger in the sample app
-   **Version 2.0.0 - November 2023**
    -   The ServiceNow Mobile SDK enables you to use the capabilities of the ServiceNow platform within your standalone mobile apps.
    -   With the Mobile SDK you can:
        -   Allow third-party developers to access ServiceNow mobile platform APIs from iOS and Android apps.
        -   Query data and embed Virtual Agent.
        -   Provide simple authentication flows leveraging JSON Web Tokens for third-party apps to authenticated and unauthenticated users.
        -   Embed authenticated web views.
        -   Support Push Notifications triggered by ServiceNow AI Platform
