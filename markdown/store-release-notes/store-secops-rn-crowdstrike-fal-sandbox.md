# CrowdStrike Falcon Sandbox Integration for Security Operations release notes

Version history for the CrowdStrike Falcon Sandbox Integration for Security Operations.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 11.0.11 - June 2026**

    New: Introduced Query ACLs in CrowdStrike Falcon Sandbox.

-   **Version 11.0.10 - September 2024**

    Fixed: The issue where the report summary was appended to the external link, causing the report to load a blank page.

-   **Version 11.0.9 - May 2023**

    Fixed: Optional tag was missing in the capability filter xml file, which is now fixed.

-   **Version 11.0.7 - February 2023**

    New: Updated to support Security Incident Response workspace.

-   **Version 11.0.4 - January 2022**

    Changed: This integration is now compatible only with CrowdStrike Falcon X.

-   **Version 11.0.3 - June 2021**

    New: A filter has been added for MD5 Observables type which enables you to stop submissions to CrowdStrike Falcon Sandbox.

-   **Version 11.0.2 - February 2021**

    New: The CrowdStrike Falcon Sandbox for Security Operations integration allows you to submit files and URLs as part of the security incident response process to the CrowdStrike Falcon Sandbox and perform a detailed malware analysis. The results of this malware analysis submission are retained as part of the security incident record and can be used for further incident resolution steps and automation workflow activities.
