# CPQ Configurator release notes

Version history for the ServiceNow® CPQ Configurator application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.4.0 - June 2026**

    New: Enable flightpath debugging for the configurator. Flightpath property was added to the UI component.

-   **Version 1.3.0 - March 2026**

    CPQ Configurator is a Seismic component that hosts the configurator and integrates with all SOM applications that have a configuration experience.
