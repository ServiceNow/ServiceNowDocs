# Employee Relations integration with Navex Ethicspoint release notes

Version history for the ER integration with NAVEX application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.1.1 - February 2024**

    Minor fixes.

-   **Version 1.0.0 - August 2022**

    This bi-directional integration will empower the ER agent to use our employee relations workflow for a seamless experience in resolving ER cases. We intend in completing the experience for the customer by integrating Navex EthicsPoint with our Employee Relations. This integration will pull in ER cases from Navex EthicsPoint to ServiceNow.
