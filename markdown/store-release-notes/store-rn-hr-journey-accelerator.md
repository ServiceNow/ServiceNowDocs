# Journey Accelerator release notes

Version history for the HR Service Delivery Journey Accelerator on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 4.0.5 - January 2023**
    -   Changed:
        -   Minor fixes
        -   Security bug fixes
        -   RCS fixes
    -   Minor fixes
-   **Version 4.0.3 - October 2022**
    -   New:
        -   \(new in 4.0+\) Integration with Listening Posts which includes integrations with CSOD/Udemy/Pluralsight/Sumtotal/SABA \(No requirements of I-Hub License\)
        -   \(new in 4.0+\) Integration with Learning Posts
    -   Fixed:
        -   Duplicated task types appearing in 'task type' field
        -   Journey Accelerator breaks the LE Activity Configurations extension point by not implementing all methods
-   **Version 4.0.2 - February 2022**
    -   New:
        -   Integration with Listening Posts which includes integrations with CSOD/Udemy/Pluralsight/Sumtotal/SABA \(No requirements of I-Hub License\)
        -   Integration with Learning Posts
-   **Version 3.0.3 - September 2021**
    -   New:
        -   Managers can create and manage to-do templates
        -   Guided tour for managers
    -   Changed:
        -   Updates to the installation of Microsoft Exchange functionality
        -   Added 'edit' and 'delete' flows from Journey Accelerator to Exchange
-   **Version 2.0.2 - June 2021**

    Minor fixes.

-   **Version 2.0.1 - May 2021**
    -   New:

        -   Event-based to-do type that allows employees to schedule meetings in Outlook \(Exchange Online\) from Journey Accelerator; must configure Microsoft Exchange Online spoke, existing iHub subscription not required.
        -   Updated mentor permissions: plan creator can assign mentor 'edit' or 'view' permissions.
        -   Configurable plan header by persona and plan type.
    -   Changed: Usability changes of the list view including updated badging and color bars for better visual cues.
-   **Version 1.0.2 - January 2021**
    -   With Journey Accelerator, ServiceNow admins can easily create templates that enable managers to create personalized plans for key employee transitions, such as promotions, transfers, onboarding, offboarding, and more.
    -   Managers, mentors, and employees collaborate on plans from the Employee Service Center.
        -   Managers create customized plans and to-dos, assign mentors, and track progress.
        -   Mentors and employees view plans and update their progress.
