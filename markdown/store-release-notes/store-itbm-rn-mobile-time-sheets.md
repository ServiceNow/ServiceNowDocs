# Mobile Time Sheets release notes

Version history for the Mobile Time Sheets on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.0 - December 2025**

    New: Added support for the read-only fields.

-   **Version 2.2.0 - August 2023**
    -   New: Added support for dark theme
    -   Fixed: Fixed issue with rejected time sheet layout
-   **Version 2.1.1 - February 2023**
    -   New: Next Experience UI
    -   Changed: Merged 'Edit resource plan' button functionality into the 'Edit time card' functionality
    -   Fixed: Minor bug fixes
-   **Version 2.0.1 - July 2021**

    Changed: You can now view all the week days in a single screen when you edit a time card.

-   **Version 2.0.0 - April 2021**
    -   New:
        -   New list layout for the time sheets on the home screen.
        -   View the time sheets in all the states on the home screen.
        -   Add notes to your time cards.
        -   New indicators help you quickly view the days on a time card that have hours posted.
    -   Changed:
        -   Add assigned and unassigned tasks to your time cards easily from a single screen.
        -   View details, add notes, or delete a time card using a single action menu.
        -   Added new tasks categories.
        -   New icons to submit, approve, reject, or recall time sheets.
-   **Version 1.0.2 - November 2020**

    Fixed: Minor issues fixed.

-   **Version 1.0.0 - August 2020**

    The Mobile Time Sheet app enables time sheet users and managers to access their time sheets from a mobile device. Time sheet users can use the app to create, view, edit, enter notes, and submit their time cards from their mobile device. Time sheet project or user managers can view, approve, reject or recall a time sheet or time card from their mobile device.
