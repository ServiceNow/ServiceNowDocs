# Export to PowerPoint for Strategic Portfolio Management release notes

Version history for the Export to PowerPoint application for Strategic Portfolio Management on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.5.0 - June 2026**

    Fixed: ACL handling in Export to PowerPoint.

-   **Version 1.4.2 - December 2025**

    Fixed: Enhanced support of export to include filtering based on multi-values and string fields.

-   **Version 1.4.0 - August 2024**

    New: Ability to export Roadmap from Strategic Planning Workspace.

-   **Version 1.2.1 - November 2023**

    Fixed related to Repeater token.

-   **Version 1.2.0 - May 2023**

    New: Export a Portfolio \(from Strategic Planning Workspace\) status report as a Microsoft PowerPoint file.

-   **Version 1.1.0 - November 2022**
    -   New:
        -   Export key details of a project from Project Workspace as a Microsoft PowerPoint
        -   Define PowerPoint formatter mapping with conditions to replace the token value with a format style
-   **Version 1.0.0 - September 2022**

    Application will provide project managers the ability to export project status reports as PowerPoints. Users can create their own custom PowerPoint Templates and export their project status reports in these custom PowerPoint templates.
