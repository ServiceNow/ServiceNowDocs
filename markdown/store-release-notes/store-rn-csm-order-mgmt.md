# Order Management release notes

Version history for the Order Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 5.0.0 - February 2023**
    -   New
        -   Enable product agents to create catalog categories that organize product offerings into hierarchical groups, making it easier to find products.
        -   Pricing admins and managers can set Archive and Retire states for pricing entities.
        -   Enable order agents to reprice order lines by using the Reprice button.
-   **Version 2.0.0 - May 2022**

    New: Added Price Lists capabilities

-   **Version 1.0.1 - February 2022**

    The Order Management application allows order agents to create orders on behalf of customers and track fulfillment of those orders through order tasks. Orders can also be created by bringing in order data through API from third party systems.
