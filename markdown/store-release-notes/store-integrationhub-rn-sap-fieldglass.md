# SAP Fieldglass spoke release notes

Version history for the Integration Hub SAP Fieldglass spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - June 2026**

    Security patch for non-glide ACLs

-   **Version 1.2.1 - October 2023**
    -   Fixed:
        -   Improvements in the installation performance.
        -   Multiple flows configured to run as User instead of System.
-   **Version 1.2.0 - September 2023**

    Added authentication template.

-   **Version 1.1.5 - March 2023**

    Fixed: Fixed bugs in the Look up Pending Approvals, Get Work Order Offer, and Look up Reject Reasons actions.

-   **Version 1.1.1 - March 2022**

    Fixed: This version includes a fix for the crypto module error.

-   **Version 1.1.0 - October 2021**

    Fixed: Security bug.

-   **Version 1.0.0 - November 2020**

    The SAP Fieldglass Spoke provides a list of actions that interact with SAP Fieldglass. It provides the foundation to interact with Fieldglass's both Buyer and Supplier REST API. Given that SAP Fieldglass is a highly customizable system, SAP Fieldglass Spoke also showcases the design pattern that can support additional use cases beyond what is out of the box.
