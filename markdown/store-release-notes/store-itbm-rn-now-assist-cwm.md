# Now Assist for Collaborative Work Management release notes

Version history for the Now Assist for Collaborative Work Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.0.1 - June 2026**

    New: AI-powered import of Tasks and Stories into CWM Boards from spreadsheets, documents, and images using Now Assist.

-   **Version 5.0.3 - April 2026**

    Generate formula with AI for custom columns of the type Formula. Let AI help you build a formula to perform calculations on the data from available columns of your CWM Boards. This feature supports all math and string functions supported by the ServiceNow AI Platform.

-   **Version 5.0.1 - March 2026**
    -   New:
        -   Use Now Assist to:
            -   Generate scrum tasks for a story.
            -   Create a custom column of Formula type and use plain language to automatically generate the formula. The Formula column can help perform calculations and supports all math and string functions.
-   **Version 4.0.1 - December 2025**
    -   New: Agile team members can now use Now Assist to generate acceptance criteria for a story based on the short description and description
    -   Changed: Explicit button to create CWM Task automatically created in any specified CWM Space &amp; Board from any CWM Docs page content
    -   Fixed: Performance improvement of Now Assist skills in CWM
-   **Version 3.1.2 - October 2025**

    Fixed: Improvement in overall performances of Now Assist capabilities.

-   **Version 2.0.2 - October 2025**

    Includes security fixes.

-   **Version 3.1.0 - September 2025**
    -   New:
        -   New AI capabilities introduced in CWM Docs - Open Prompt Content Generation:
            -   Simply type a prompt, such as Write a job description for a Product Manager or Summarize this customer feedback and suggest any improvement, and Now Assist will generate high-quality draft content instantly.
            -   Whether you're starting from scratch or expanding existing content, Now Assist empowers you to draft professional-quality documents effortlessly.
            -   Stay within Docs while drafting, without toggling between multiple applications for AI content generation.
-   **Version 3.0.0 - August 2025**

    Enhanced: Support for third party LLMs for existing skills of Now Assist for Collaborative Work Management. The list of 3rd Party LLMs supported are Microsoft Azure, AWS Anthropic, and Google Gemini.

-   **Version 1.0.0 - June 2025 \(Xanadu\)**
    -   New:
        1.  Doc Summarization Skill: Stay informed with AI-driven document summarization directly in CWM Docs. Easily shorten, elaborate, or shorten content to match your needs.
        2.  Task Generation from Docs: Eliminate manual task creation with Now Assist's intelligent task generation. Create clear, detailed tasks for your Boards using your Doc content, with flexible options to split, combine, or adjust recommendations.
-   **Version 2.0.0 - May 2025**

    Enhance your productivity with the latest AI-powered capabilities in Collaborative Work Management \(CWM\) using Now Assist for Collaborative Work Management \(CWM\). Our new updates bring the ability to summarize, elaborate, and shorten your content within Docs, while also automatically generating clear, actionable tasks for your Boards helping teams save time, reduce manual effort, and drive better outcomes.Whether you're reviewing documents or preparing project plans, these intelligent skills help you stay informed, aligned, and efficient.
