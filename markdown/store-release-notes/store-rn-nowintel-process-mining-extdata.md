# Process Mining for external data release notes

Version history for the Process Mining for external data application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 29.5.0 - March 2026**
    -   New: Support pagination and sorting for external data table list
    -   Fixed:
        -   Ability to delete an external data source whose tables were not created
        -   Conflicting Date: Time Format on External Data Import
    -   Accessibility fixes: Show records fixed for analyst users
-   **Version 28.8.4 - September 2025 \(Zurich\)**

    No updates.

-   **Version 28.8.2 - August 2025 \(Zurich\)**

    Fixed: Analyst user issues with external data projects- Internationalization fixes- "Proceed to Integration hub" tab issue.

-   **Version 27.7.1 - July 2025 \(Yokohama\)**

    Fixed: Record data generation issue.

-   **Version 26.16.11 - July 2025 \(Xanadu\)**

    No additional fixes.

-   **Version 27.6.3 - May 2025 \(Yokohama\)**

    No additional changes.

-   **Version 26.16.8 - March 2025 \(Xanadu\)**

    There are no additional changes in Process Mining for external data.

-   **Version 27.6.1 - February 2025 \(Yokohama\)**
    -   New:
        -   Additional breakdown capacity for Process Mining on external data
        -   Ability to import a maximum of 10 breakdown fields enabling to segment and analyze specific subsets of the external process data using Process Mining.
-   **Version 26.16.6 - January 2025 \(Xanadu\)**
    -   Fixed:
        -   Ability to Edit dataset
        -   Ability to add relevant breakdowns from guided project setup \(Project builder\)
-   **Version 26.16.1 - November 2024 \(Xanadu\)**

    No fixes.

-   **Version 26.13.1 - August 2024**

    Minor bug fixes.

-   **Version 25.6.3 - February 2024**
    -   New:
        -   Ability to create multidimensional model\(multiple tables\) on external data connected to ServiceNow tables
        -   Ability to create a project with a preconfigured table once the external dataset is imported and processed
    -   Fixed: Dark theme and translation fixes
        -   Clustering on the external data table from the analyst workbench
        -   Edit dataset journey completion fix
-   **Version 24.0.4 - November 2023**
    -   Changed: Contextual changes
    -   Fixed: No major fixes
-   **Version 24.0.2 - September 2023**
    -   New:
        -   Bring your external audit data into the ServiceNow platform
        -   Guided import process
-   **Version 24.4.1 - August 2023 \(Vancouver\)**

    Today, optimizing processes is a time-consuming, expensive process, and delivers inconsistent results. Leaders are struggling to optimize the processes involved in service delivery. They need a new level of transparency to identify opportunities for improvement, or areas ripe for smart automation.
