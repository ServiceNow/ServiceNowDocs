# Marketplace Core release notes

Version history for the ServiceNow® Marketplace Core application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 30.1.0 - June 2026**

    New: Marketplace Core supports Non-Glide Cobalt Raven ACLs \(access control lists\). Administrators can review and merge platform security updates for Non-Glide Cobalt Raven ACLs in Marketplace Core, with upgrade and completed Zboot testing across Platform and Workspace interfaces.

-   **Version 30.0.1 - March 2026**

    Marketplace Core provides the foundational components required for FSM Contractor Marketplace functionality to work. Install this plugin before enabling any Contractor Marketplace features that depend on it.
