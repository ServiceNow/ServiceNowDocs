# Workplace PPE Inventory Management release notes

Version history for the Safe Workplace Workplace PPE Inventory Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

**Note:** To automatically install all the Safe Workplace applications at once, download the ServiceNow® Safe Workplace suite application.

## Version history

-   **Version 1.18.2 - June 2026**

    Fixed: Remediated non-Glide ACLs

-   **Version 1.18.0 - August 2025**

    Changed: Minor ACL updates

-   **Version 1.17.0 - November 2024**
    -   Changed: Several illustrations used in the record producers have been updated.
    -   Fixed: Security fixes
-   **Version 1.16.2 - February 2023**

    Fixed: Minor bug to update illustration.

-   **Version 1.16.0 - August 2022**
    -   Fixed:
        -   Minor bug to support localization
        -   Quality improvements
-   **Version 1.15.0 - January 2022**

    New: Localization for 22 languages other than English. Translations are available for features released up until the December 16, 2021 release.

-   **Version 1.14.1 - December 2021**

    Changed: Updates to the translation engine

-   **Version 1.13.0 - August 2021**

    Fixed: Minor bug

-   **Version 1.12.0 - August 2021**

    New: Localization for features from the July 22 release

-   **Version 1.11.0 - July 2021**

    New: Localization for features from the July 1 release

    -   Changed: Updates to the translation engine

    New: Localization for features from the February 18 release

    -   New: New catalog item icons

    -   New:

-   **Version 1.0.1 - June 2020**
    -   New: Localization support for:
        -   Japanese
        -   French
        -   German
        -   Spanish
        -   Italian
        -   Brazilian Portuguese
        -   Korean
        -   Simplified Chinese
-   **Version 1.0.0 - May 2020**

    The Workplace PPE Inventory Management application helps you manage and monitor workplace needs for safety equipment provided to your workforce. A dashboard provides a comprehensive view of inventory by each facility, an aggregate scope of the entire workplace, and historical data on how equipment levels have changed over time. Current inventory levels can be updated daily for accurate and real-time inventory management.
