# Threat Intelligence Security Center integration with Palo Alto Networks NGFW release notes

Version history for the Threat Intelligence Security Center integration with Palo Alto Networks NGFW application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.1 - March 2025**
    -   Fixed: Validations on "IP","Domain","URL" observables to add to the EDL.
    -   Changed: Modified the application from "Palo Alto Networks NGFW Integration for Threat Intelligence Security Center" to "Threat Intelligence Security Center Integration with Palo Alto Networks NGFW".
-   **Version 2.0.0 - November 2024**

    Palo Alto Networks NGFW Integration for Threat Intelligence Security Center \(TISC\) enables Cyber Threat Intelligence \(CTI\) analysts to manage External Dynamic Lists \(EDLs\) directly from TISC. Analysts can create or remove EDL entries, which the Palo Alto Networks NGFW can then consume to drive security actions.
