# Salesforce Marketing Cloud spoke release notes

Version history for the Salesforce Marketing Cloud Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.6.1 - June 2026**

    Security patch on non-glide ACLs

-   **Version 1.5.3 - March 2026**

    Added new field "Connection Alias" to the "Configurations" table and made changes in connection configuration template

-   **Version 1.5.1 - March 2025**

    Fixed: Configuration issue of the connection when the logged-in user is "sam\_integrator or admin" and when the user is within the global scope.

-   **Version 1.5.0 - November 2024**

    Evaluated sandbox access for client-callable script includes and marked sandbox callable as false.

-   **Version 1.4.1 - January 2024**

    Fixed: Making data extension input mandatory and misleading hint to be fixed in the configuration template.

-   **Version 1.4.0 - November 2023**

    Added OAuth Support.

-   **Version 1.3.1 - May 2023**

    Added about 40 new actions on campaign management, contact management, list management, webhook management, and activity management.

-   **Version 1.0.0 - March 2021**
    -   New:
        -   User Management
            -   Look up Users Stream
            -   Update User by User ID
        -   Data Extension Management
            -   Look up Data Extension Attributes
            -   Look up Data Extension Attributes for Bulk Records Stream
