# Google Drive Spoke release notes

Version history for the Integration Hub Google Drive Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.0 - September 2025**

    Fixed - Required ACL\(s\) or Role\(s\) for agents.

-   **Version 2.2.1 - July 2025**

    Fixed: AI Agents - Naming, Output Transformation.

-   **Version 2.2.0 - May 2025**

    Internal changes: updated actions/tools of drive agents released in March.

-   **Version 2.1.1 - March 2025**

    Minor release with new AI agents.

-   **Version 2.0.1 - October 2024**

    Fixed: Look up File action.

-   **Version 2.0.0 - May 2024**
    -   New: Added new actions equivalent to those deprecated with newer design patterns
    -   Removed: Deprecated older actions
-   **Version 1.4.1 - June 2023**

    Changed: Return additional output field: updated date for the Look up Files action.

-   **Version 1.4.0 - May 2023**

    New: Four new actions: Look up File Revisions Stream, Look up File, Copy File Revision as Attachment, Delete File Revision.

-   **Version 1.3.2 - September 2022**

    Fixed: Patch version of Google Drive Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.3.1 - August 2022**

    Fixed: Patch release of Google Drive Spoke for IntegrationHub. This version fixes an issue on being unable to upload a text file into google drive.

-   **Version 1.3.0 - March 2021**

    Minor release of Google Drive spoke for IntegrationHub. This version adds actions \(Look Up Files and Folders Stream, Look Up Permissions Stream\) and a configuration template for authentication.

-   **Version 1.2.0 - July 2020**

    Patch release of Google Drive spoke for IntegrationHub.

-   **Version 1.0.1 - September 2019**

    Provides actions to automate collaboration and file and folder management in Google Drive.
