# Wrike spoke release notes

Version history for the Wrike Spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - September 2025**

    Added required ACLs and Roles as per the AI Security Directive guidelines.

-   **Version 1.2.0 - May 2025**

    Minor release with new AI agents.

-   **Version 1.1.0 - March 2023**

    New: Authentication template for easier spoke setup.

-   **Version 1.0.1 - September 2022**

    Patch version of Wrike Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.
