# Access Analyzer release notes

Version history for the Access Analyzer on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 6.1.0 - March 2026**
    -   New: Access analyzer support for Agentic workflows and AI agents in compare user access and access simulation features.
    -   Fixed: Defects.
-   **Version 6.0.5 - January 2026**
    -   New: Access analyzer support for Agentic workflows and AI agents in Analyze Permissions feature
    -   Fixed: Defects
-   **Version 6.0.4 - December 2025**
    -   New: Access analyzer support for Agentic workflows and AI agents in Analyze Permissions feature
    -   Fixed: Bugs
-   **Version 5.0.1 - August 2025**

    New: Access analyzer support for Security data filters and related record list in ACL.

-   **Version 4.0.1 - February 2025**
    -   Access analyzer for access\_analyzer\_admin, Access insights.
    -   Fixed a bug.
-   **Version 3.1.0 - November 2024**
    -   New:
        -   Access analyzer support evaluation of Query operation \(Query Match &amp; Query range\) for all features \(Evaluate access, Compare user access &amp; Simulate access\)
        -   ESMA Directive - implemented improved security control for Access Analyzer tables
-   **Version 3.0.0 - August 2024**
    -   New: Simulate access: Simulate how access to the specified table would change once a role or group is assigned or removed from the user
    -   Changed: Deny unless ACL &amp; Empty ACL support for Access Analyzer.
-   **Version 2.0.1 - April 2024**

    Empty ACLs fix as part of hardening directive.

-   **Version 2.0.0 - February 2024**
    -   New: ServiceNow Access Analyzer V2 tool to compare the access of 2 users and determine the right level of access controls is provided for the users.
    -   Changed: Access analyzer version 1 supported with time limited role.
-   **Version 1.0.4 - October 2023**

    Various fixes as part of patching.

-   **Version 1.0.2 - August 2023**

    Access Analyzer allows for selecting a user, group, or role and evaluating access permission to a table, its field or record.
