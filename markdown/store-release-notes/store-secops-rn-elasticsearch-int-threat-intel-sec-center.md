# Threat Intelligence Security Center integration with Elasticsearch integration for release notes

Version history for the Threat Intelligence Security Center integration with Elasticsearch integration application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.0.4 - May 2025**

    Fixed: ElasticSearch integration logo was not displaying correctly.

-   **Version 3.0.3 - March 2025**

    Changed: Application name from "Elasticsearch integration for Threat Intelligence Security Center" to "Threat Intelligence Security Center integration with Elasticsearch".

-   **Version 3.0.2 - August 2024**

    New: Pushed changes to support localization.

-   **Version 3.0.0 - June 2024**

    The Threat Intelligence Security Center seamlessly integrates with Elasticsearch, by enabling comprehensive searches across log data within the Elasticsearch deployment. It efficiently incorporates pertinent sighting information into Observables for enhanced security analysis.
