# Customer Project Management release notes

Version history for the ServiceNow® Customer Project Management application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.1.0 - June 2026**

    Non-Glide Cobalt Raven ACLs for Customer Project Management - Security enhancements to query access controls for Case lines and workflows.

-   **Version 2.0.0 - March 2026**
    -   No new features have been delivered; migrated the application from family to store. Integrate Customer Service Management with the Project Portfolio Management application. This integration enables customers to manage complex projects with multiple tasks and provide end users with visibility into these projects.
    -   Customer Service Management provides an integration with the ServiceNow PPM Standard application, earlier known as Project Portfolio Suite with Financials.
    -   Customers can use this integration to create projects and project tasks for their customer accounts. Customers can also provide end users with visibility into their projects and tasks from the Customer Service Portal and enable them to request additional actions.
