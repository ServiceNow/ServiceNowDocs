# Field Service Territory Planning release notes

Version history for the ServiceNow® Field Service Territory Planning application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 30.2.4 - June 2026**

    New: Updated Territory planning console accessibility

-   **Version 30.0.3 - March 2026**
    -   Field Service Territory Planning allows organizations to create and manage territories and the work within the territories. Territories are built around geographies and resources allowing the organizations to allocate the work and technicians effectively.
    -   Territories can be updated to match the resource availability or demand . The plugin also evaluates defined conditions to match each work order task to the most appropriate territory and the best-positioned technician or crew to handle it.
