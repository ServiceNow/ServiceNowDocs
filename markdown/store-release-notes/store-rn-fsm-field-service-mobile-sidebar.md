# Field Service Mobile Sidebar release notes

Version history for the Field Service Mobile Sidebar application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.2 - March 2026**

    Updated the dependency to the Field Service mobile application since Field Service mobile migrated to the store from a family plugin.

-   **Version 1.0.5 - December 2025**

    Added a minor fix for showing save icon on the work order task screen to save the task as favorite.

-   **Version 1.0.3 - August 2024**

    Provide agents with sidebar collaboration on mobile to enable an in-app experience for chatting and collaborating with team members while preserving important information for future reference and historical context.
