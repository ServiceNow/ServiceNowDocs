# HR Service Delivery Integration with Ultimate Kronos Group release notes

Version history for the HR Service Delivery Integration with Ultimate Kronos Group on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.0.6 - June 2024**

    Fixed: Security issues.

-   **Version 2.0.4 - February 2024**

    Minor fixes.

-   **Version 2.0.3 - November 2022**

    Minor optimization and refactoring

-   **Version 2.0.2 - September 2021**
    -   Fixed:
        -   Precision of number of hours worked
        -   Issue related to the honouring of the Date format of Reportee Time Off Request as per user preference
        -   Fixed the vendor name
-   **Version 2.0.1 - June 2021**
    -   New:
        -   Retrieve information on the list of punches, view reportees' schedule changes and time off requests while conversating with the virtual agent.
        -   Retrieve location information along with the upcoming shift date and time.
    -   Changed:
        -   The template names have personalized in Version 2.0.1. Retrieve My Accrual Balance \(Template\) Retrieve My Hours Worked \(Template\) Retrieve My Employee Schedule \(Template\) Retrieve My Time off Requests \(Template\) Retrieve My Upcoming Shift \(Template\) Reportee’s Location Today \(Template\)
    -   Fixed: The file size issue of the ML solution has been fixed.
-   **Version 1.0.1 - March 2021**
    -   New:
        -   Retrieve information on leave balances, schedules, and upcoming shifts on-the-go.
        -   Build conversations that are based on keywords that an employee enters.
        -   Apply Natural Language Understanding \(NLU\) models, which enable the virtual agent to pick utterances for understanding, processing, and responding to what employees are saying during a conversation.
