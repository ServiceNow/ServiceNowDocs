# Agent-Initiated Messaging Interface release notes

Version history for the ServiceNow AI Platform capabilities Agent-Initiated Messaging Interface on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.22 - March 2026 \(Australia\)**

    Translation updates.

-   **Version 1.0.20 - December 2025 \(Zurich\)**

    Extended 'Add link in message' KB guidance action for Interaction type 'messaging'

-   **Version 1.0.19 - August 2025 \(Zurich\)**

    New: Added ability for CSM phone agents to search existing repositories of knowledge articles and then share the public article links with customers easily over SMS channel.

-   **Version 1.0.17 - February 2025 \(Yokohama\)**

    Fixed bug.

-   **Version 1.0.16 - November 2024 \(Xanadu\)**

    Changes: Bug fixes.

-   **Version 1.0.15 - August 2024 \(Xanadu\)**

    Bug fixes.

-   **Version 1.0.12 - August 2023 \(Vancouver\)**

    Bug fixes.

-   **Version 1.0.11 - February 2023**

    Bug fixes.

-   **Version 1.0.10 - August 2021**

    Enables administering agent-initiated messaging capabilities for messaging channels.

-   **Version 1.0.8 - July 2021**

    Enables administering agent-initiated messaging capabilities for messaging channels.

-   **Version 1.0.8 - June 2021**

    Minor fixes.

-   **Version 1.0.6 - January 2021**

    Provides configurations for agent-initiated messaging capabilities. This version is certified for Quebec.
