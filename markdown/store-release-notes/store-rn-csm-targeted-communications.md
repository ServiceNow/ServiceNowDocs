# Targeted Communications release notes

Version history for the ServiceNow® Targeted Communications application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 30.1.0 - May 2026**

    Changed: Introduced query‑range ACLs for Targeted Communications to enforce stronger query‑level data access controls.

-   **Version 30.0.0 - March 2026**
    -   Targeted Communications application allows organizations to send personalized messages, articles, and email notifications to specific internal and external audiences​
    -   Feature designed for proactive and reactive communication, such as notifying customers about service outages or product updates
