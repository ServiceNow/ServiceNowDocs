# POM - Prime release notes

Version history for the ServiceNow® POM - Prime application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.1.1 - June 2026**

    New: Enhancement to the automatic conversion of supplier emails into PO Exception cases, now supporting additional languages and ERP purchase order IDs for improved PO identification.

-   **Version 1.0.0 - April 2026**

    Advanced agentic capabilities to support efficient resolution of purchase order exceptions.
