# Google Docs spoke release notes

Version history for the Integration Hub Google Docs spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.3.0 - September 2025**

    Fixed - Required ACL\(s\) or Role\(s\) for agents.

-   **Version 1.2.1 - August 2025**

    Fixed - AI Agents - Renaming and Output transformation strategy

-   **Version 1.2.0 - March 2025**

    Minor release with new AI agents.

-   **Version 1.1.0 - June 2023**

    New: Authentication template to the spoke so that spoke setup is easier.

-   **Version 1.0.1 - December 2022**

    Fixed: Improve installation performance of spoke and fix an issue related to HTML tags and nbsp tag when space is used.

-   **Version 1.0.0 - July 2020**

    Provides actions to create, update and search in Google Docs.
