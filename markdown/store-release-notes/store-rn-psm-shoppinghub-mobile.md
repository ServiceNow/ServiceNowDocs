# Shopping Hub Mobile release notes

Version history for the Shopping Hub Mobile application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 8.0.1 - June 2026 \(Australia\)**

    Changed: Remediated non‑Glide Cobalt Raven ACLs in product code \(DIRS0000421\).

-   **Version 7.6.26 - June 2026**

    Defect fixes.

-   **Version 7.6.24 - August 2025**

    Minor bug fixes.

-   **Version 7.6.20 - May 2025**

    Minor fixes.

-   **Version 7.6.16 - November 2024**

    Addressed issues with quick checkout overlay not loading and not rendering approval to-dos.

-   **Version 7.6.12 - August 2024**

    Minor fixes.

-   **Version 7.6.6 - May 2024**

    Minor fixes.

-   **Version 7.6.0 - February 2024**

    Minor fixes.

-   **Version 7.1.0 - November 2023**

    Minor fixes.

-   **Version 7.0.0 - August 2023**

    Minor fixes.

-   **Version 6.0.2 - May 2023**

    Changed: Renamed this application to Shopping Hub Mobile. This application was formerly named ShoppingHub Mobile.

-   **Version 6.0.0 - February 2023**
    -   Changed: Updated the Next Experience theming to be applied to the mobile experience for the Now Native Mobile application. For the Shopping Hub mobile experience, the 'My tasks' and 'My purchases' sections will render using the Next Experience theming.
    -   **Note:** Existing customers will be impacted by this theming change where their mobile experience will adopt the Next Experience theming record.

-   **Version 5.0.0 - November 2022**

    Fixed accessibility issues from WCAG 2.1 AA audit findings on NOW Mobile.

-   **Version 4.4.1 - August 2022**

    Minor fixes.

-   **Version 4.3.1 - May 2022**

    Minor fixes.

-   **Version 4.2.0 - February 2022**
    -   New
        -   Approvers can now request clarification from ShoppingHub Mobile and shoppers can provide clarification from the same.
        -   Shoppers can now see the reasons for rejection and cancellation of a purchase from My Purchases on ShoppingHub Mobile.
