# Jenkins spoke release notes

Version history for the Integration Hub Jenkins spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.3.1 - June 2026**

    Fixed an issue where the Restart Jenkins and Rename Job actions were failing with an unknown error

-   **Version 2.3.0 - September 2025**

    Fixed - Required ACL\(s\) or Role\(s\) for agents.

-   **Version 2.2.1 - August 2025**

    Fixed - AI Agents - Renaming and Output transformation strategy

-   **Version 2.2.0 - March 2025**

    Minor release with new AI agents.

-   **Version 2.1.2 - September 2023**

    Fixed: The license requirement.

-   **Version 2.1.1 - April 2023**

    Fixed: Improve installation performance of spoke.

-   **Version 2.0.2 - August 2019**

    Licensing patch for the Jenkins spoke on Integration Hub

-   **Version 1.6.3 - March 2019**
    -   Integrate ServiceNow change management into the software delivery pipeline
    -   Easily author flows using Jenkins action to automate integration to CI/CD pipeline
    -   Promote codeless orchestration using natural language powered by the Integration Hub engine
    -   Automate the continuous integration of high-risk change requests that require approval
