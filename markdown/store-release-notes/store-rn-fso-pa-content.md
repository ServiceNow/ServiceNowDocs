# Performance Analytics Content Pack for FSO release notes

Version history for the Performance Analytics Content Pack for FSO application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.12.1 - March 2026**

    Updated: Replaced direct references to the admin role with feature-specific granular admin roles to align with least-privilege access principles.

-   **Version 1.11.2 - August 2025**

    Changed: Included additional translations to support localization for Q4.

-   **Version 1.10.0 - February 2025**
    -   Changed:
        -   Updated breakdown on Card network with Network ID for reports on Card disputes task and transaction.
        -   Removed breakdown on Card network and Reason code for Card disputes service case.
-   **Version 1.9.0 - November 2024**
    -   Changed PA for Dispute Management:
        -   Updated breakdown on Card network with Network ID for reports on Card disputes task and transaction.
        -   Removed breakdown on Card network and Reason code for reports on Card disputes service case.
-   **Version 1.8.1 - August 2024**

    Removed header component from FSO PA dashboard page.

-   **Version 1.8.0 - May 2024**
    -   Changed:
        -   Upgraded the analytics dashboards to Next Experience dashboards
        -   Modified configuration for localization support
-   **Version 1.7.0 - February 2024**
    -   New: Added new breakdowns by Dispute Category and Dispute Reason Code
    -   Changed:
        -   Upgraded snc-app-parent version to 5.1.0.16
        -   Modified configuration for localization support
    -   Removed: Removed breakdown by Dispute Reason
-   **Version 1.6.0 - November 2023**
    -   Changed:
        -   Migrated Financial Services Business Deposit Operations PA dashboard to Performance Analytics Dashboard
        -   Migrated Financial Services Business Lifecycle PA dashboard to Performance Analytics Dashboard
-   **Version 1.5.1 - May 2023**

    New: Added localization changes.

-   **Version 1.5.0 - February 2023**
    -   New: Added PA reports for Intelligent Servicing for Fraud.
    -   Fixed: Fixed drill down for Branch Manager dashboard score cards.
-   **Version 1.4.0 - November 2022**
    -   New:
        -   Added new dashboard for business location manager
        -   Added new PA dashboard for commercial lines claims manager
        -   Added new PA dashboard for personal lines claim
-   **Version 1.3.0 - August 2022**
    -   New: Added new reports for individual life, group life insurance, claims and updated complaint PA dashboard.
-   **Version 1.2.1 - March 2022**
    -   New:
        -   Dashboard and reports for Personal Lines Servicing
        -   Dashboard and reports for Commercial Lines Servicing
-   **Version 1.1.1 - August 2021**

    Changed: Changed label for Complaint Management dashboard

-   **Version 1.1.0 - July 2021**
    -   New:
        -   Added PA dashboards for the following Financial Services Operations applications:
            -   Complaint Management
            -   Business Lifecycle
            -   Client Lifecycle
            -   Treasury Operations
-   **Version 1.0.2 - May 2021**

    Changed: Financial Services Core application dependency version has been updated

-   **Version 1.0.1 - March 2021**
    -   New:
        -   PA dashboards and reports for Financial Services Operations applications
            -   Business Loan Operations
            -   Personal Loan Operations
            -   Payment Operations
