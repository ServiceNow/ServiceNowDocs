# Now Assist for Public Sector Digital Services \(PSDS\) release notes

Version history for the Now Assist for Public Sector Digital Services \(PSDS\) application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.2.2 - June 2026**
    -   Multilingual Enhancements
    -   ICM Case Narrative Ai Agent Support
-   **Version 2.1.1 - April 2026**

    ServiceNow is transitioning all SKUs to product tier packaging with three tiers: Foundation, Advanced, and Prime. This app was updated to add relevant subscription metadata for supporting this transition

-   **Version 2.0.3 - March 2026**
    -   New:
        -   Investigative case summarization
        -   Investigative case narrative refinement
        -   Document screening
-   **Version 1.2.4 - December 2025**
    -   Now LLM LTS Support
    -   Java 21 Compliant
    -   Security Directive Compliance
-   **Version 1.1.4 - November 2025**

    Added Fee Waiver Justification Skill

-   **Version 1.1.3 - October 2025**

    Performance Improvements

-   **Version 1.1.2 - September 2025**

    Changed Dependencies.

-   **Version 1.1.1 - August 2025**

    Dependency version changes.

-   **Version 1.1.0 - February 2025**
    -   Now Assist for AI search: Single-turn capability for legislation/policy summarization; agent-facing
    -   Now Assist Chat Summarization: Agent facing chat summarization
-   **Version 1.0.1 - August 2024**

    Now Assist for PSDS enables government agencies to condense case records into short summaries using the case summarization skill. Agents can use the resolution summarization skill to generate resolution information for a government case type, propose the resolution to the constituent/business contact, and add the information directly to the case record. Now Assist for PSDS brings generative AI case summarization and resolution notes generation into the Public Sector Digital Services experience.
