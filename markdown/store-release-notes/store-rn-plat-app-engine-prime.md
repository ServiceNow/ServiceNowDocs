# App Engine - Prime release notes

Version history for the ServiceNow® App Engine - Prime application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 29.1.5 - June 2026**

    Changed: Updated versions of dependent apps.

-   **Version 29.1.3 - April 2026**
    -   App Engine Prime is the highest-tier App Engine subscription, delivering the complete suite of AI-powered development capabilities on the ServiceNow platform. Building on Foundation capabilities, App Engine Prime unlocks Now Assist for App Engine and Now Assist for Creator — enabling developers to harness generative and agentic AI to build smarter applications faster than ever.
    -   With App Engine Prime, creators gain access to build agents with the Build Agent, an autonomous AI developer, alongside the full Now Assist skill suite for flows, catalog items, UI, playbooks, tests, and more. This is the subscription for organizations looking to lead with AI-native application development at scale.
