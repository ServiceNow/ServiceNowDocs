# Manage Order Operations release notes

Version history for the ServiceNow® Manage Order Operations application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.2 - December 2025**

    Designed for order operations teams, the Manage order operations application automates the intake process for order case creation. Instead of relying on manual requests, customers interact with a Virtual Assistant \(VA\) that captures inputs and automatically creates order cases. This reduces handling time, improves operational efficiency, and enhances customer satisfaction.
