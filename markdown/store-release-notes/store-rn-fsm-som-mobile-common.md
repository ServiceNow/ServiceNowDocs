# Sales Customer Relationship Management Mobile Common release notes

Version history for the ServiceNow® Sales Customer Relationship Management Mobile Common application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 29.1.2 - March 2026**

    This plugin provides the shared mobile components used by Sales and Order Management features within Field Service Management. It establishes a common foundation for FSM mobile experiences, ensuring consistent behavior across mobile use cases without requiring duplicate configuration. Install this plugin when enabling Sales and Order Management features that rely on shared FSM mobile functionality.
