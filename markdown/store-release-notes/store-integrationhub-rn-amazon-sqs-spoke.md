# Amazon SQS Spoke release notes

Version history for the Integration Hub Amazon SQS Spoke application on the ServiceNow Store.

## Version history

-   **Version 1.0.1 - January 2025**

    Fixed: Receive Message, Look up Queue Attributes actions.

-   **Version 1.0.0 - August 2023**

    Automate queue, message, and permission management of your Amazon SQS instance from the ServiceNow instance.
