# CDO Dashboard release notes

Version history for the ServiceNow® CDO Dashboard application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 2.0.5 - December 2025**

    The Chief Digital Officer Dashboard integrates data from IT Service Management and Performance Analytics Premium to provide a unified view of digital service performance, innovation progress, and business impact.
