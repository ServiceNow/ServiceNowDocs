# Amazon CloudWatch spoke release notes

Version history for the Integration Hub Amazon CloudWatch spoke on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.2 - December 2022**

    Changed: Spoke name.

-   **Version 1.0.1 - September 2022**

    Fixed: Patch version of Amazon Cloudwatch Spoke for Integration Hub. This version includes a fix to improve the installation performance of the spoke.

-   **Version 1.0.0 - August 2020**

    Provides actions to automate the management of alarms and metrics in Amazon Cloudwatch.
