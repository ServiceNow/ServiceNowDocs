# GitLab spoke release notes

Version history for the Integration Hub GitLab spoke on the ServiceNow.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.5.0 - June 2026**

    Fixed: Streamline ACLs.

-   **Version 2.4.1 - May 2026**

    Fixed: File Upload Issue in "Add New File to Repository" action

-   **Version 2.4.0 - November 2025**

    Fixed: Read Only Field. These changes are part of Internal Security Directive. Refer to the Global Communication shared to know more about this Directive.

-   **Version 2.3.2 - October 2025**

    New: AI Agents for GitLab Spoke.

-   **Version 2.2.0 - November 2024**

    Fixed: Sandbox access for client.

-   **Version 2.1.3 - August 2024**
    -   Fixed:
        -   \[Security Bug\] ACL bypass with 'GitLabWebhookGenerator' Script Include
        -   In GitLab Spoke, some data stream actions are failing with a red band error, some actions does not have help content and error messages
        -   OAuth Refresh token is saved with incorrect type in oauth\_credential table
-   **Version 2.1.2 - December 2023**

    Fixed: Gitlab - Not able to get the Authentication Token.

-   **Version 2.1.1 - October 2022**

    New: Added Token expiry functionality to Gitlab webhooks.

-   **Version 2.0.0 - August 2022**
    -   New:
        -   Added 25 more actions.
        -   Added sample outbound flow and inbound subflow.
        -   Added remote table.
        -   Added authentication template for easier Spoke setup.
-   **Version 1.0.5 - December 2021**

    Changed: Patch release of GitLab Spoke for IntegrationHub. This version includes security-related changes.

-   **Version 1.0.4 - May 2021**

    Patch release of the GitLab spoke for IntegrationHub.

-   **Version 1.0.3 - October 2020**

    Patch release of the GitLab spoke for IntegrationHub

-   **Version 1.0.2 - August 2020**

    Fixed: This is a patch version and includes fixes.

-   **Version 1.0.1 - February 2020**

    Added data stream dependency and fixed a typo in the description

-   **Version 1.0.0 - January 2020**

    Automate branch, group, issue, merge request, project, and source code management of your GitLab instance from the ServiceNow instance.
