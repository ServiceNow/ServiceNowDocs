# ServiceNow Voice for HR Service Delivery \(HRSD\) release notes

Version history for the ServiceNow® Voice for HR Service Delivery \(HRSD\) application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.5 - December 2025**

    Fixed: Creation of Amazon Lex Bot to setup ServiceNow Voice for HRSD.

-   **Version 1.0.2 - February 2024**
    -   Automatically classify, route, and prioritize HR cases directly from Interactive Voice Response \(IVR\) interactions
    -   Add recordings and transcripts directly in the case for reference and customer support
    -   Receive status updates and additional information to existing cases directly from phone interactions
