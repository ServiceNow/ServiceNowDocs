# Next Experience All Menu Editing release notes

Version history for the ServiceNow® Next Experience All Menu Editing application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 29.4.30 - June 2026**
    -   What’s New: Improved keyboard navigation: Create modules and applications using only a keyboard.
    -   Key Issues Fixed: Improved component resizing behavior and resolved visual design inconsistencies
-   **Version 29.3.14 - May 2026**
    -   What’s New: Enhanced localization in UI Builder - The "Code" and "Comment" fields for translation literals are now exposed in the properties pane. This allows developers to disambiguate English homonyms by providing unique composite identifiers \("Code" + "Key"\) and translator comments, improving translation accuracy.
    -   Key Issues Fixed: Other changes to improve accessibility
-   **Version 29.2.28 - April 2026**

    Next Experience All Menu Editing brings modern Next Experience lists, forms, and page configuration directly into Platform, enabling admins to create and edit application menus and modules quickly - without needing Workspaces or UI Builder.
