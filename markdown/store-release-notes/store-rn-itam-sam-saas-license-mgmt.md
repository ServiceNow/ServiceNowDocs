# Software Asset Management - SaaS License Management Foundation release notes

Version history for the IT Asset Management Software Asset Management - SaaS License Management Foundation application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 15.0.16 - July 2025 \(Yokohama\)**
    -   The record deletion policy for DEX Usage and aggregation has been moved to Data Cleaner.
    -   The CrowdStrike issue preventing the generation of the product usage table because of varying instance date formats has been resolved.
    -   The issue with false downgrade candidates being generated \(such as Microsoft 365 E3-E1\) has been resolved.
-   **Version 14.0.10 - July 2025 \(Xanadu\)**
    -   The record deletion policy for DEX Usage and aggregation has been moved to Data Cleaner.
    -   The CrowdStrike issue preventing the generation of the product usage table because of varying instance date formats has been resolved.
    -   The issue with false downgrade candidates being generated \(such as Microsoft 365 E3-E1\) has been resolved.
-   **Version 13.1.5 - June 2024 \(Washington DC\)**

    In v13.1.5, Adobe and Microsoft365 scheduled jobs activation issue is fixed.

-   **Version 12.0.6 - November 2023 \(Vancouver\)**
    -   Issues with Zoom integration are fixed.
    -   Archived users subscriptions are now fetched for G Suite integration.
    -   Users are mapped to the correct software model for Looker integration.
    -   Cisco Webex Meetings integration profile is deprecated.
-   **Version 11.1.2 - November 2023 \(Utah\)**

    In v11.1.2, Cisco Webex Meetings integration profile is deprecated.

-   **Version 10.0.10 - November 2023 \(Tokyo\)**

    In v10.0.10, Cisco Webex Meetings integration profile is deprecated.

-   **Version 7.0.2 - August 2021**
    -   Use the ServiceNow SaaS License Management application to leverage the new improved user interface for existing integrations:
        -   GoTo
        -   Trello
        -   Confluence
        -   Monday.com
        -   Miro
        -   Cisco Webex Meetings
    -   Use the ServiceNow SaaS License Management application to integrate with the following applications:
        -   Cisco Webex
        -   Roadmunk
        -   Adobe Workfront
        -   Zendesk
        -   Microsoft Dynamics 365
-   **Version 6.0.3 - July 2021**
    -   New: Consumption-based reconciliation Subscriptions to go through the reconciliation engine New and improved user interface
    -   Fixed: Bug fixes
-   **Version 5.0.1 - May 2021**

    Fixed: Minor updates

-   **Version 4.0.0 - March 2021**

    Fixed: Minor updates

-   **Version 3.0.0 - February 2021**

    New: The Software Asset Management - SaaS License Management plugin is now available on the ServiceNow Store.

-   **Version 2.0.2 - January 2021**

    It requires a Software Asset Management Professional plugin and provides core capabilities to manage licenses for SaaS products. Get visibility to your subscription spend and optimize it. This will be activated with Software Asset Management - SaaS License Management Integrations plugin.
