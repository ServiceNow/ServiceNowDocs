# Dispute Rules Content Pack for Nacha release notes

Version history for the Dispute Rules Content Pack for Nacha application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 1.0.0 - December 2025**

    This application provides a centralized reference for ACH return reason codes and the logic used to determine them, based on Nacha operating guidelines. It includes knowledge base articles outlining eligibility rules, required documentation, and timelines for each return code. Designed to support back-office and compliance teams, this tool helps streamline decision-making, ensure regulatory alignment, and reduce errors in dispute processing.
