# Data Model Navigator release notes

Version history for the ServiceNow® Data Model Navigator application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.4 - May 2026**
    -   Fixed:
        -   Files are moved to the appropriate folder in the search index.
        -   Specific read-only fields are changed to strict read-only fields.
-   **Version 1.0.3 - March 2026**

    Fixed: This release contains a fix related to the search index, the files were moved to the appropriate folder.

-   **Version 1.0.2 - January 2026**

    This application contains the framework which is used to capture all the details about the OOB data model, starting with the CMDB data model.
