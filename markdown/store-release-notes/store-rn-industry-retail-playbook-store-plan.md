# Retail Playbook for Store Plan release notes

Version history for the ServiceNow® Retail Playbook for Store Plan application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 3.4.0 - June 2026**
    -   New:
        -   Integrating questionnaire results UIB page opened via related list query.
        -   Demo data for Store Plan Authoring and Tracking workflows included.
-   **Version 3.3.0 - June 2026 \(Zurich\)**
    -   New:
        -   Integrating questionnaire results UI Builder \(UIB\) page opened via related list query.
        -   Demo data for Store Plan Authoring and Tracking workflows included.
-   **Version 3.0.0 - May 2026**

    New: Added demo data for published HQ Communications store plan

-   **Version 2.2.0 - May 2026 \(Zurich\)**

    New: Added demo data for the published HQ Communication store plan

-   **Version 2.1.1 - April 2026**

    Fixed: Resolved issue with the cancel button in the Plan Summary schedule during HQ Communications playbook configuration.

-   **Version 2.0.1 - April 2026 \(Zurich\)**

    Fixed: Resolved issue with the cancel button in the Plan Summary schedule during HQ communications playbook configuration.

-   **Version 2.1.0 - March 2026**

    The retail playbook for store plan enables HQ and regional managers to easily create structured operational plans with tasks and cases, linked to store locations and schedules.
