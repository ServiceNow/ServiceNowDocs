# Now Assist for CPQ release notes

Version history for the ServiceNow® Now Assist for CPQ application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

-   **Version 1.0.3 - April 2026**

    Maintenance release. Contains internal code updates with no impact to existing functionality or user-facing behavior.

-   **Version 1.0.1 - March 2026**

    Now Assist for Quote Management leverages generative AI and ServiceNow’s intelligent workflows to streamline complex quote management processes. Designed for enterprises managing high-volume quotes, this CPQ solution minimizes manual intervention, accelerates quote resolution and approval cycles, and improves overall customer satisfaction through greater accuracy, consistency, and speed.
