# Security Simulation and Training Integration for SecOps release notes

Version history for the Security Simulation and Training Integration for SecOps application on the ServiceNow Store.

**Important:** For details on system requirements and family compatibility, view the application listing on the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website.

## Version history

-   **Version 2.1.3 - May 2024**

    Changed: Modifications are made to the visualization of the phishing simulation chart in order to display the monthly trend.

-   **Version 2.0.2 - February 2024**

    The Security Simulation and Training Integration for SecOps framework provides a way to integrate third-party phishing simulation and training products with ServiceNow Security Operations. This allows organizations to ingest phishing simulation metrics from these products and leverage them in the Cyber Security Executive Dashboard.
