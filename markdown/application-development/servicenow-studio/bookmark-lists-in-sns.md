# Bookmark lists in ServiceNow Studio

Bookmark lists in ServiceNow Studio so you can return to them quickly from the Bookmarks tab in the Navigator panel.

## Before you begin

Role required: admin or delegated\_developer

## About this task

**Note:** Bookmarks and favorites are distinct ServiceNow capabilities. Bookmarks exist only within ServiceNow Studio, while favorites are accessible throughout your instance. Bookmarks do not automatically become favorites, and favorites do not automatically become bookmarks.

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Open the list you want to bookmark.

    A list in ServiceNow Studio can range from the entire File Categories list to a list of a few files.

3.  Select **Bookmark list** using one of the following options.

    -   At the top of the list, select **Bookmark list** next to the **Create** button.

        ![Bookmark the list from the top of the page.](../image/sn-studio-bookmark-top.png)

    -   Select the more options icon ![](../image/sn-studio-more-options-icon.png), then select **Bookmark list**. For example, open the list of all flows from the Automation file category and bookmark the list.

        ![Bookmark a list from more options menu in the Navigator panel.](../image/sn-studio-bookmark-list.png)

4.  Enter a name for the bookmark in the **Add to bookmarks** dialog.

5.  Select **Add**.

6.  Select **Close**.

7.  Refresh the Navigator panel to view your new bookmark on the **Bookmarks** tab.

8.  To rename the bookmark, select the rename bookmark icon, enter a new name, and select **Save**.
