# App details page

The App details page for each application in ServiceNow Studio contains valuable information about your app. You can create new files, update app settings, and publish your app for deployment, among other things.

## App details features

The App details page shows information about your app and provides access to common app actions.

1.  **Navigator panel**: View app files organized by category. Bookmark files or apps, open the full list of any file type, or filter the list to find specific files. For more information, see [Bookmark apps and app files in ServiceNow Studio](../task/bookmark-apps-and-app-files.md).
2.  **Files section**: View featured file types \(Data, User Interface, Automation, and Security\) or all file types. Filter or sort the list to find the files you want to work with. For more information, see [Working with metadata app file categories in the ServiceNow Studio Navigator](sn-studio-working-with-metadata.md).
3.  **Page header**: View app metadata, invite collaborators, or create files directly using the + Create option. Badges next to the app title — such as Creator Studio, AI, or Fluent — indicate where the app was created and how it must be deployed. For more information, see [App details page](app-details-page.md) and [Viewing app origination information in ServiceNow Studio](viewing-app-origination-information-in-sns.md).
4.  **App actions**: Select the more options icon ![](../image/sn-studio-more-options-icon.png)to link the app to source control, invite collaborators, access settings, or publish, refresh, or delete the app. If the app is linked to source control, available source control operations appear in a menu next to the **Publish** button. For more information, see [Working with applications in ServiceNow Studio](working-with-apps-in-servicenow-studio.md).
5.  **File records**: Select a file to open it, select **Add** to create files of that type, or bookmark files directly from the record. For more information, see [Create an app file in ServiceNow Studio](../task/sn-studio-create-app-file.md).
6.  **Scope and update set**: View the scope and update set associated with the current app at the bottom of the page. The scope switches automatically when you switch between tabs in ServiceNow Studio. Create an update set and associate app changes to it so your changes are ready to deploy. For more information, see [Open apps and app files across scopes in ServiceNow Studio](qs-open-apps-files-across-scopes.md) and [Working with update sets in ServiceNow Studio](working-with-update-sets-in-servicenow-studio.md).
