# Find an app or app file using Search

Search for and open an app or app file in ServiceNow Studio. You can also activate code search to search for snippets of code within select metadata types on your instance.

## Before you begin

Role required: admin or delegated developer

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Choose from one of the following options to select the Search bar.

    -   If you're on the ServiceNow Studio home page, select the Search icon \(\[Omitted image "sn-studio-search-icon.png"\] Alt text: Search icon\) in the Navigator panel, or select in the main Search bar.

        \[Omitted image "sn-studio-main-search.png"\] Alt text: Main Search bar in ServiceNow Studio

    -   If you're on any page other than the home page, such as Tools or Deployment, select the Search icon \(\[Omitted image "sn-studio-search-icon.png"\] Alt text: Search icon\) in the left side navigation.
3.  Activate code search by selecting the switch next to **Activate code search**.

    **Note:** Code search enables you to search for snippets of code within specific application file types. For more information about using code search, see [Find an app or app file using code search](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-development/servicenow-studio-classic/find-app-app-file-using-code-search.md).

4.  Enter your app or app file name.

    If you're familiar with the ServiceNow AI Platform search capabilities and want to navigate directly to a table, follow the instructions in [Navigate directly to a table in ServiceNow Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-development/servicenow-studio-classic/qs-navigate-directly-to-table.md).

5.  Select your app or file from the list that appears.

6.  To expand and filter your search results, select **Enter** to open the **Search results** tab, then select any of the filters to narrow your list of search results.

    \[Omitted image "sn-studio-search-filters.png"\] Alt text: Filters for files, tables, and apps in the search results list
