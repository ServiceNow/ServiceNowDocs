# ServiceNow Studio quick start

Complete these tasks to learn the core workflows in ServiceNow Studio — from finding apps and files to packaging and deploying changes.

## Where do I start in ServiceNow Studio?

Complete these tasks to learn the core workflows in ServiceNow Studio. For a step-by-step app creation tutorial, see [ServiceNow Studio and Build Agent tutorial](sns-ba-tutorial-landing.md).

|Task|Description|
|----|-----------|
|[Find an app or app file using Search](../task/qs-find-app-app-file-using-search.md)|Search from any page in ServiceNow Studio to find and open metadata records quickly.|
|[Find an app or app file using the Navigator panel](../task/qs-find-app-app-file-using-navigator-panel.md)|Browse the Navigator panel taxonomy to locate files and metadata records by type.|
|[Bookmark apps and files in ServiceNow Studio](../task/qs-bookmark-apps-files.md)|Bookmark frequently used apps and app files so you can access them from one location.|
|[Bookmark lists in ServiceNow Studio](../task/bookmark-lists-in-sns.md)|Bookmark lists you use often so you can return to them without searching.|
|[Access recently opened apps and app files in ServiceNow Studio](../task/qs-access-recently-opened-apps-files.md)|View and reopen apps and files you have worked on recently.|
|[Find info about an app in ServiceNow Studio](qs-find-info-about-an-app-servicenow-studio.md)|Open the App details page to view app metadata and create files.|
|[Navigate directly to a table in ServiceNow Studio](../task/qs-navigate-directly-to-table.md)|Open table views directly by entering shortcut commands such as **.list**, **.do**, and **.config** after a table name in the Search bar.|
|[Open apps and app files across scopes in ServiceNow Studio](qs-open-apps-files-across-scopes.md)|Open global and scoped apps and files in ServiceNow Studio.|
|[Focus the Navigator panel on an app or app file](../task/qs-focus-navigator-panel-on-app-app-file.md)|Focus the Navigator panel on any open file or application so the panel reflects your current context.|
|[Access integrated development tools and builders in ServiceNow Studio](open-and-switch-between-integrated-development-tools.md)|Open any file type from the File Categories menu to access integrated development tools.|
|[Create an update set in ServiceNow Studio](../task/qs-create-update-set.md)|Create an update set to package your app changes for deployment.|
|[Publish app changes to the Application Repository from ServiceNow Studio](../task/qs-publish-changes-to-app-using-app-repo.md)|Publish app changes to the Application Repository. If your company has a deployment pipeline configured, you can also deploy through the pipeline in ServiceNow Studio.|
