# Summarize a record through chat using Now Assist for App Engine

Quickly understand the contents of a record through a conversation with Now Assist.

## Before you begin

Role required: admin

Verify that the custom app record summarization skill is activated and configured. For more information, see [Activate the custom app record summarization skill](activate-custom-app-record-summarization-na-for-app-engine.md).

The following procedure describes how to summarize a record through chat in the Now Assist panel. You can also summarize a record in-product within Core UI or a custom workspace. For more information, see [Summarize a record in-product using Now Assist for App Engine](summarize-record-in-product-na-for-app-engine.md).

## Procedure

1.  Open the Now Assist panel by selecting the Now Assist icon \(![](../../app-engine-studio/image/now-assist-sparkle-icon-dark.png)\).

2.  In the Now Assist panel, select **Summarize a record** or enter `summarize a record`.

3.  Enter the record number for the record that you want to summarize.

4.  Select the Enter key.

    **Note:** Generating and displaying the summary may take several seconds.

5.  When you're finished summarizing a record, you can copy the summary or provide feedback about the summary.
