# Use Now Assist for App Engine

Use the skills within Now Assist for App Engine to add custom AI features to your custom applications.

Now Assist for App Engine contains many tools and skills, including the following:

-   **[AI Agent Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/enable-ai-experiences/ai-agent-studio.md)**

    Create flexible, adaptive AI agents that support App Engine use cases.

-   **[Now Assist Data Kit](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/now-assist-data-kit/na-data-kit-generate-data.md)**

    Build and maintain datasets and ground truths to evaluate your custom skills.

-   **[Knowledge article generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/servicenow-platform/now-assist-in-knowledge-management/now-assist-knowledge-management.md)**

    Use Now Assist to inspect your custom application data and generate knowledge base articles.

-   **[Now Assist in Document Intelligence](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/now-assist-in-document-intelligence/docintel-nowassist-landing.md)**

    Incorporate the capabilities of DocIntel into your flows and playbooks.

-   **[Now Assist Skill Kit](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/now-assist-skill-kit/now-assist-skill-kit-landing.md)**

    Develop custom skills to be used within your custom apps.

-   **[Now Assist in Virtual Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/conversational-interfaces/now-assist-in-virtual-agent/now-assist-in-va-landing.md)**

    Use Virtual Agent Designer with LLM enabled components to create chatbots.

-   **[Now Assist Platform skills](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/now-assist-skills/now-assist-on-now-platform.md)**

    Use platform skills to export dashboards and to help AI agents find answers from a variety of content.

-   **[Now Assist Navigation skills](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/intelligent-experiences/now-assist-skills/now-assist-global-navigation.md)**

    Use Now Assist to find records or tables, for example, ask for a list of incident records.

-   **[Search Q&amp;A \(+ multi-context synthesized answer generation\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/platform-administration/ai-search/now-assist-ais.md)**

    Search and summarize KB articles, policies, and guides that you publish with your application.


For a complete list of the tools and skills in Now Assist for App Engine, see the product listing in the [ServiceNow Store](https://store.servicenow.com/store/app/5d27eef41bd92a50396216db234bcb45).
