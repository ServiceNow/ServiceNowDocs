# Auto-generate ATF tests

Auto-generate ATF tests by selecting the auto-generate option either from the Auto-generate Tests module or Tests/Suites modules.

## Before you begin

Role required: admin

## Procedure

1.  Access the Auto-generate Tests feature using either of the following ways.

**Note:** The ATF Test Generator and Cloud Runner store app link shows up. The Auto-generate Tests module shows up only if you don’t have the ATF Test Generator and Cloud Runner store app installed.

2.  Select the **Check out the app on Store** option.

**Note:** The ATF Test Generator and Cloud Runner store app page shows up.

**Note:** You can also select **Suites** modules instead of **Tests**.

2.  Scroll down to select the **Auto-generate Tests** link.

**Note:** This link takes you to the Test Generator home page if you have the store app installed.
