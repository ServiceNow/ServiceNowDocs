# Supported metadata in Build Agent

Work with a range of ServiceNow metadata and app types in Build Agent.

Build Agent supports working with many types of ServiceNow metadata and app files, including the following:

-   Automated Test Framework \(ATF\) tests
-   Business rules
-   Client scripts
-   Client-side logic
-   Configuration
-   Email integration
-   List controls
-   Scheduled jobs
-   Script includes
-   Scripted REST APIs
-   Security
-   Server-side logic
-   Service Catalog items
-   Tables
-   UI actions
-   UI components
-   UI pages
-   UI policies
-   UI views
-   Workspaces
