# Test Agent

Use Test Agent to streamline the process of authoring, running, and troubleshooting tests autonomously from a single interface. Test Agent generates test coverage for code produced by Build Agent, executes tests, and performs root cause analysis on failures, enabling end‑to‑end build‑to‑test workflows within a single development session.
