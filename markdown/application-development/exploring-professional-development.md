# Exploring professional development

ServiceNow provides a single mobile and web application development platform to quickly build business applications to power digital transformation. Learn how anyone can automate, extend, and build digital workflow apps across the enterprise with the ServiceNow AI Platform.

This content covers the application development stages to help developers easily build applications on the ServiceNow AI Platform.

-   [Plan your application development](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-development/plan-your-app-development.md): Think through the application before building it.
-   [Build your application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-development/build-your-application.md): Create and configure the application and its components.
-   [Validate app functionality](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-development/validate-app-functionality.md): Perform functional testing and write test cases.
-   [Deploy your app](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-development/deploy-your-app.md): Move the application into production.
