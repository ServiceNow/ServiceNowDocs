# Analyze a flow

Have Now Assist scan a flow's execution details, identify errors, and suggest fixes.

## Before you begin

Role required:

-   now.assist.creator
-   flow\_designer or admin

Turn on the flow summarization skill. See [Turn on the Flow execution analysis skill](turn-on-the-flow-execution-analysis-skill.md).

## Procedure

1.  Navigate to **All** &gt; **Process Automation** &gt; **Workflow Studio**.

2.  Navigate to **Operations** &gt; **Flows**.

3.  Open the execution details of the flow that you want to analyze.

4.  From the Workflow Studio sidebar, select **Summarize**.

5.  From the Analysis pane, select **Analyze flow**.


## Result

Now Assist analyzes the flow, identifies errors, and suggests fixes.
