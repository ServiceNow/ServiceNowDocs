# Use the app generation skill to generate apps

By default, ServiceNow Studio uses Build Agent to create apps with AI. To use the app generation skill instead, disable the Build Agent setting in your ServiceNow Studio preferences.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Select the user preferences menu.

3.  Select **Preferences**.

4.  In the **Global behavior** section, toggle off the switch for using Build Agent.
