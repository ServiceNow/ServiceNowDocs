# Exploring Now Assist for Creator

Explore the AI capabilities, users, benefits, and workflow for using Now Assist for Creator.

## Now Assist for Creator overview

Now Assist for Creator includes generative and agentic AI capabilities that make developing on the ServiceNow AI Platform more efficient. The Now Assist for Creator AI capabilities are embedded across platform tools—including ServiceNow Studio, Workflow Studio, UI Builder, and the ServiceNow IDE—so developers can generate apps, flows, code, and UI components without leaving the tools they use every day. Now Assist for Creator also includes Build Agent, an autonomous AI agent that can vibe code full-stack applications through natural language conversations in the ServiceNow IDE.
