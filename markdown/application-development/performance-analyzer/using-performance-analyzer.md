# Testing applications with Performance Analyzer

Get information on how to test applications with Performance Analyzer.

-   **[Get metrics by duration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/performance-analyzer/filter-application-metrics-by-duration.md)**

    Filter metrics in the Performance Analyzer by duration from the last fifteen minutes to the last seven days.

-   **[Get application metrics](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/performance-analyzer/get-application-metrics.md)**

    Filter metrics by application in the Performance Analyzer.

-   **[Get page route metrics](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/performance-analyzer/get-page-route-metrics.md)**

    Filter metrics for an application by page route in the Performance Analyzer.

-   **[Get interaction metrics](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/performance-analyzer/get-interaction-metrics.md)**

    Filter metrics for an application's page route by interaction in the Performance Analyzer.
