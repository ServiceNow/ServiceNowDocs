# Install an application from the application repository

Install your application on an instance so that employees can start using the application that you developed.

## Before you begin

-   [Publish an application to the application repository](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-development/application-repository-self-hosted/t_PublishAppsToTheAppRepository.md).
-   Check the entitlement type of the application to ensure that your instance is entitled to the application. For more information, see [Manage application entitlements from the application repository](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/application-development/application-repository-self-hosted/manage-app-entitlements.md).

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Applications** &gt; **My Company Applications**.

2.  Find the application.

3.  Next to the application listing, select a version to install.

4.  Click **Install**.
