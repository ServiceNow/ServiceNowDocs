# Documentation site release notes

This product documentation site is continuously updated for not only content but also requested functionality. Please review the changes introduced to this site to make the most of your experience.

## Q2 2026 changes

Improvements in accessibility, mobile responsiveness, and AI discoverability of the ServiceNow product documentation site, plus the release of the Australia family of product documentation.

### Notable changes

-   ServiceNow product documentation is now available on GitHub at [https://github.com/ServiceNow/ServiceNowDocs](https://github.com/ServiceNow/ServiceNowDocs).
-   AI tools and integrations can now access docs directly without JavaScript rendering.
-   Australia family content is now prioritized in search results, search facets, and the product directory.
-   Accessibility and visual fixes: color contrast, missing navigation labels, and heading hierarchy.
-   The site is now responsive. It previously was optimized for desktop only.
-   Redirect issues affecting bookmarked URLs, in-product help links, and context-sensitive help links have been resolved.
-   When diagrams and screenshots include clickable regions, those interactive areas now work correctly for navigating to related content.

### Future changes

-   More redirect fixes are in progress for links from older versions of the site.
-   Broader AI discoverability for this site is in progress.

## Q1 2026 changes

We've rebuilt the doc site experience to improve how we deliver content and help you find what you need more easily.

Our updated site is designed to offer a more consistent and reliable experience across ServiceNow websites. Your existing bookmarks will continue to work through redirects, so you won't need to update saved links. Some features may look or behave differently as part of the improvements, and you may notice updates to the interface as you explore the site.

We're committed to improving the docs experience over time and listening to your feedback. Thank you for being part of our journey to improve how we deliver documentation.

### Available features

We've kept the core functionality you rely on while adding improvements to the overall experience. The site now includes:

-   A refreshed layout and navigation that aligns with the ServiceNow brand
-   An updated, shorter URL format for content pages. We're still at servicenow.com/docs and support your bookmarks with redirects, but the long URLs are deprecated in favor of a more concise format.
-   Enhanced search results, including filtering for videos and previewing results
-   Functionality you know and use:
    -   A family release version picker
    -   Integration of multimedia with content
    -   AI topic summaries
    -   AI code explanations
    -   Topic feedback at the bottom of the page
    -   Search filtering and clustering
    -   Product directory and release notes
    -   Customized release notes
    -   API documentation
    -   Downloadable full-publication PDFs
    -   Site-wide announcements

### Notable changes

Some features from the previous site are temporarily unavailable. We are actively working to bring back capabilities for features such as:

-   Printing PDFs on-the-fly

    As a workaround until this feature is added, you can download the full PDFs from each topic or use your browser's print functionality for single topics.

-   Subscribing to topics
-   Personalizing your site preferences
-   Dark mode support
-   Web responsiveness
-   Mobile support

### Future releases

The new docs experience gives us the flexibility to deliver improvements more quickly. While timelines and priorities may shift, here’s a preview of what we’re working on for future releases:

-   Optimized search results page, including layout, indexing, and content discoverability
-   Table of contents experience revision
-   Improved release notes experience
-   Improved UX on topic pages
-   Role-based experiences
-   Accessible PDFs
