# Track a store inquiry case from Retail portal

Track the status of a store inquiry case from the Retail portal.

## Before you begin

Role required: Store associate, Store manager

## Procedure

1.  In Retail Portal, select **Home** &gt; **Cases &amp; Tasks** from the menu bar.

2.  Select the case number to track the case.
