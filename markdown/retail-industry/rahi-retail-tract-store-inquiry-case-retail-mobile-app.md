# Track a store inquiry case from a Retail mobile app

Track the status of a store inquiry case from a mobile app.

## Before you begin

Role required: Store associate, Store manager

## Procedure

1.  On Retail mobile app, select **My Store** &gt; **All cases** from the menu bar.

2.  Select the case number to track the case.
