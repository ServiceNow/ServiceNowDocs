# Add a task to a retail case

Add tasks directly to your retail case using the Tasks related list.

## Before you begin

Role required: sn\_retail.associate, sn\_retail.manager\_contributor​, sn\_retail.support\_agent​

## Procedure

1.  Navigate to **All** &gt; **Retail Service Management** &gt; **Retail Cases**.

2.  Select the Retail case to add a task to.

3.  Within the retail case, navigate to the **Tasks** related list.

4.  Click **New** and fill in the following fields as needed.
