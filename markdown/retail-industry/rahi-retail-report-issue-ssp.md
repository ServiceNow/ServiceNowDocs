# Report an issue using the Business Location Service Portal

Use the Report an Issue catalog item in the Business Location Service Portal to submit a retail case support request for issues that can be resolved within your retail organization.

## Before you begin

Role required: sn\_retail.associate, sn\_retail.support\_agent, or sn\_retail.manager

## Procedure

1.  Navigate to the **Business Location Service Portal**.

2.  Select the **Report an issue** catalog item.

3.  Fill in the following fields.
