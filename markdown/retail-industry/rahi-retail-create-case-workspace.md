# Create a retail case support request in CSM/FSM Configurable Workspace

Use the CSM/FSM Configurable Workspace to create a retail case.

## Before you begin

Role required: sn\_retail.associate, sn\_retail.support\_agent, or sn\_retail.manager.

## Procedure

1.  In the CSM/FSM Configurable Workspace, navigate to **Lists** &gt; **Retail Cases** &gt; **My Cases**.

2.  Click **New**.

3.  Fill in the fields as needed.

4.  Click **Save**.
