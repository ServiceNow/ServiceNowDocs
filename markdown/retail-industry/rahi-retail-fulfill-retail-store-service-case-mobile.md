# Manage a store inquiry case from a mobile app

Manage a store inquiry case from a mobile app by accepting or rejecting a proposed solution.

## Before you begin

Role required: Store associate, Store manager

## Procedure

1.  On Retail mobile app, select **My Store** &gt; **All cases** from the menu bar.

2.  Select the case.

3.  Select **Actions** &gt; **Close case**

    If the HQ agent requests for more information, you can add notes or more details. If the HQ agent proposes a solution with the resolution, you get options to accept the solution or reject the solution.

4.  Select **Accept** to close the case.

5.  Select **Reject** to reopen the case.
