# Using Now Assist in mobile Virtual Agent

ServiceNow Virtual Agent for mobile enables you to quickly obtain information, make decisions, and perform common work tasks.

![standard chat window example with booking travel](../image/na-va-standard-chat.png "Standard chat")

Now Assist in Virtual Agent standard chat provides a conversational support experience within a static chat window. For more detailed information about how to use this skill, see [Standard chat](https://www.servicenow.com/docs/access?context=nava-standard-chat&version=yokohama&pubname=yokohama-conversational-interfaces&ft:locale=en-US).

![enhanced chat explaining how to book travel](../image/na-va-enhanced-chat.png "Enhanced chat")

Now Assist in Virtual Agent enhanced chat is a conversational support experience that includes the functionality for multiple active conversations and improved search capabilities. For more information on how to use this skill, see [Enhanced chat](https://www.servicenow.com/docs/access?context=nava-enhanced-chat&version=yokohama&pubname=yokohama-conversational-interfaces&ft:locale=en-US).

![suggested actions to check leave balance, block calendar, and notify teammates](../image/na-va-suggested-actions.png "Suggested actions")

Suggested actions prompt you with suggested actions related to your conversation that you may consider doing next. Selecting a suggested action will begin that action within the same chat conversation and carry over any relevant information from your current conversation.

![people citation showing contact info for an employee named Alisa Chinoy](../image/na-va-people-citations.png "People citations")

People citations enables you to search for information about a specific employee. Simply ask Now Assist who someone is and provide their name. The employee’s profile displays with information such as their full name, work location, email, and work phone.

![searching for how to book travel in the now assist chat window](../image/na-va-chat-search.png "Search results from chat (enhanced chat only)")

Launch the standard search results page from within enhanced chat to quickly find the information you need without leaving the chat. Simply tap the magnifying glass icon under the list of sources to launch a search.

![list of multiple active chats a user has open](../image/na-va-multiple-active-chats.png "Multiple active chats (enhanced chat only)")

Switch between multiple active chats and start where you left off. Tap the chat that you want to view to open it.
