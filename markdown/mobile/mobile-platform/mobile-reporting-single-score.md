# Single score reports for mobile

Learn to use a single score report. This report is a single value that you can use to share metrics or scores that are key to your business needs. For example, the number of high priority incidents opened by your team.

**Note:** For information on how to configure single score reports, you need to create mobile dashboard previews within a launcher screen. For more information, see [Launcher screens](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/mobile/mobile-platform/sg-mobile-applet-launcher.md) and [Mobile dashboard preview](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/mobile/mobile-platform/mobile-dashboard-preview.md).

Tap on the single score to see a complete list of records relating to the report. You can then tap on individual records for additional information.
