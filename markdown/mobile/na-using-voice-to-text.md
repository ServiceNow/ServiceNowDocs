# Using Voice Input for mobile

The Voice Input feature allows users to easily transcribe spoken words into written text. Voice-to-text can be accessed directly from the search bar or Now Assist chat.

![Now Assist search bar shows it's listening to the user's voice](../image/na-voice-input.png "Voice input")

To begin, tap the microphone button ![](../image/na-voice-icon.png) in your Now Assist chat search bar to start the transcription. To show that Now Assist is listening, the listening icon ![](../image/na-listen-icon.png) will display while you are speaking. Once your message has been transcribed, it will appear as text in your search bar.
