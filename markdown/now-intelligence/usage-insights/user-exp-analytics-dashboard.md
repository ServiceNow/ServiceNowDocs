# Overview of the User Experience Analytics dashboard

User Experience Analytics shows detailed KPI data on a dashboard and on list views. Product managers gain relevant insight into the usage and adoption of a web or mobile application.

Users with any User Experience Analytics viewer role can access the dashboard by navigating to **User Experience Analytics** &gt; **Dashboard**.

Counters in the User Experience Analytics dashboard contain aggregated user numbers.

**Note:** If the tracking consent policy named Disabled is selected user metrics aren't tracked.

The dashboard provides access to the following modules and controls.

\[Omitted image "dashboard-overview-page-annotated-mobile.png"\] Alt text: Dashboard - Overview screen.
