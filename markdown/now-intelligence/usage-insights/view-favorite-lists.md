# Bookmarking and viewing favorite user and session records

Manually select user and session records to view them in a favorites list.

## Before you begin

Role required: user

## About this task

Each user and session record in the dashboard has a star icon to the left of the user index or session date. Select the star icon to bookmark the record to a favorite list.

Select and deselect a star to add or remove a user or session record from a favorites list, and then view the favorite from the records you selected.

\[Omitted image "favorites-bookmarked.png"\] Alt text: Some records bookmarked as favourites in a sessions list.

## Procedure

1.  In either the user module or session module, bookmark a record by selecting the favorites icon \(\[Omitted image "favorites-icon.png"\] Alt text: Favorites icon.\) to the left of a record.

2.  View the records bookmarked for users or sessions by doing one of the following:

    -   In the modules menu, select **Users** &gt; **Favorites**. To filter the results of your favourites, see [Filter a users list](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/usage-insights/view-users-list.md).
    -   In the modules menu, select **Sessions** &gt; **Favorites**. To filter the results of your favourites, see [Filter a sessions list](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/usage-insights/filter-session-list.md).
