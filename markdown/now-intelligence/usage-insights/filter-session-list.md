# Filter a sessions list

Refine a sessions list to focus on data such as selected screens or events for your application.

## Before you begin

Role required: analytics\_admin, mobile\_analytics\_admin, web\_analytics\_admin, or portal\_analytics\_admin

## Procedure

1.  On the **Where** condition line, select a property from the filter menu.

2.  Select an operator and a value as applicable.

3.  To add another property, click the add property icon \(\[Omitted image "add-filter-icon.png"\] Alt text: Add property icon.\).

4.  -   To clear filters back to the original filter set for the sessions list, click **Reset**.
-   To remove a filter property, click the delete icon \(\[Omitted image "delete-filter-property.png"\] Alt text: Delete filter property icon.\) next to the property.
5.  Click **Search** to apply filters and refresh the list.
