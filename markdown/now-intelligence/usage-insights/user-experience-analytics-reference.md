# User Experience Analytics reference

Properties, KPIs, and components installed with User Experience Analytics.

Several components are installed with the User Experience Analytics plugin, including user roles, scheduled jobs, and tables.
-   **[User Experience Analytics related properties](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/usage-insights/all-analytics-properties.md)**  
Use system properties to configure User Experience Analytics in Platform Analytics, ServiceNow Mobile Platform, Service Portal, and the Conversational Analytics area of Virtual Agent.
-   **[User Experience Analytics KPIs](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/usage-insights/user-exp-analytics-KPIs.md)**
