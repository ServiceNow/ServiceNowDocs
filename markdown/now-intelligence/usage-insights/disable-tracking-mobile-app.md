# Enable or disable User Experience Analytics tracking

You can turn off or turn on tracking for Platform Analytics, ServiceNow Mobile Platform, Service Portal, and the Conversational Analytics area of Virtual Agent by updating the User Experience Analytics settings.

## Before you begin

Role required: analytics\_admin, mobile\_analytics\_admin, web\_analytics\_admin, now\_experience\_analytics\_admin, or portal\_analytics\_admin

## Procedure

1.  Navigate to **All** &gt; **User Experience Analytics** &gt; **Settings**.

2.  From the list, locate the record for the application you want to disable analytics tracking for.

    The application name is listed in the **Name** column.

3.  Open the record by selecting either **true** or **false** on the same row.

4.  On the form, do one of the following:

    -   Select the **Enabled** check box to turn on tracking.
    -   Clear the **Enabled** check box to turn off tracking.
5.  Select **Update**.
