# Exporting data

You can export data records to CSV format. Exporting user and session lists, and other analytics data to CSV is available wherever you see the download icon \(\[Omitted image "csv-download.png"\] Alt text:\).

Select the \(\[Omitted image "csv-download.png"\] Alt text: CSV export download icon.\) to begin automatic export of the data. The download location is specified in your browser settings.
