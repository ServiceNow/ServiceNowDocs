# Edit a cohort

Edit cohort session data to include in your analysis.

## Before you begin

Role required: Users with the analytics\_viewer, portal\_analytics\_viewer, mobile\_analytics\_viewer, core\_ui\_analytics\_viewer, now\_experience\_analytics\_viewer roles can create, edit and delete cohorts for the applications they have access to.

## Procedure

1.  Navigate to **Platform Analytics** &gt; **User Experience Analytics**.

2.  From the Applications list, select the application with the cohort you want to edit.

3.  Select **Analysis** &gt; **Cohort Analysis**.

4.  Select the Actions menu \[Omitted image "more-actions-menu-icon.png"\] Alt text: Actions menu icon of the cohort you want to edit and choose **Edit**.

5.  Change the name, description and actions as desired.

    \[Omitted image "uxa-edit-cohort.png"\] Alt text: Edit cohort dialog.

6.  Select **Save** to update the cohort configuration.
