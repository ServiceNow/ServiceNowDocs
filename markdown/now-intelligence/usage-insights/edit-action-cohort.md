# Edit a cohort

Edit cohort session data to include in your analysis.

## Before you begin

Role required: Users with the analytics\_viewer, portal\_analytics\_viewer, mobile\_analytics\_viewer, core\_ui\_analytics\_viewer, now\_experience\_analytics\_viewer roles can create, edit and delete cohorts for the applications they have access to.

## Procedure

1.  From the **Cohorts** module, select from the drop down the cohort you want to edit.

2.  Point to the cohort name to show the options, and click the edit \[Omitted image "edit-icon.png"\] Alt text: Edit cohort or funnel icon. icon.

3.  Change the cohort name, sessions to include, and set of user actions as desired.\[Omitted image "edit-cohort-wizard.png"\] Alt text: Edit cohort dialog.

4.  Click **Finish** to update the cohort configuration.
