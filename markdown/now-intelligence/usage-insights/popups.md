# Popups

View sessions of mobile app users who see a specific pop-up. Pop-up analytics help you easily identify bugs and errors your users encounter.

\[Omitted image "popup-screen.png"\] Alt text: Popups screen

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Most Viewed

</td><td>

Most viewed pop-up in the app by number of views and percentage over alls views of all pop-ups.

</td></tr><tr><td>

Highest Quit Rate

</td><td>

Shows screens which users quit the most, measured by total number of sessions ending in the screen relative to total app quits.

</td></tr><tr><td>

Popup

</td><td>

Name of the pop-up.To rename a pop-up, point to the pop-up name and click the edit \[Omitted image "edit-icon.png"\] Alt text:icon to rename the pop-up.

</td></tr><tr><td>

Text

</td><td>

Text the user reads in the pop-up.

</td></tr><tr><td>

Sessions

</td><td>

The percentage of sessions the pop-up has been viewed in. Click the Sessions list \(\[Omitted image "session-icon.png"\] Alt text:\) icon for a selected pop-up to see the list of sessions that contain the pop-up.

</td></tr><tr><td>

Options

</td><td>

Shows the options that were presented to the user in the pop-up. For example, ‘Confirm,’ or ‘Cancel’.Click the Sessions list \(\[Omitted image "session-icon.png"\] Alt text:\) icon to see the list of sessions filtered by options for the selected pop-up options.
