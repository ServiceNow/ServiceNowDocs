# Using Now Assist for Process Mining

Run work notes analysis on a transition to identify the work notes and generate insights about inefficiencies, challenges, and other reasons that cause reassignment of work.

This section covers the following topic:

-   **[Run work notes analysis](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/now-intelligence/process-mining/run-worknotes-analysis.md)**  
Generate a work notes analysis on a transition. 50 transition records are required to trigger work notes analysis, and 25 work notes must be eligible.
