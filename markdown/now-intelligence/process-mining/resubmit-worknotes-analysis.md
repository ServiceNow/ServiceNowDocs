# Resubmit worknotes analysis

If a worknotes analysis fails for some reason, you can resubmit it.

## Before you begin

Role required: sn\_process\_optimization\_analyst, sn\_process\_optimization\_power\_user, or sn\_process\_optimization\_admin

## Procedure

1.  Navigate to **All** &gt; **Workspaces** &gt; **Process Mining Workspace**.

2.  From the list of projects, select a project.

3.  From the Scheduled Tasks pane, select **Resubmit** for the Worknotes analysis that has failed.

    To view the error details, select **View error details**.
