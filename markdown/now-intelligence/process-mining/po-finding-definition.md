# Working with finding definitions

Use finding definitions to deliver discovered insights to your Summary and Insights dashboard.

Finding definitions display information gathered in Process Mining projects. These appear as a list within your Summary and Insights page. Use this information to see key information about your business processes and get insights on bottlenecks, route variation, and more.

\[Omitted image "summary-insights2.png"\] Alt text: Summary and Insights - quick insights examples

-   To create a new finding definition, select **Change the configuration** link.

    For information on creating finding definitions, see [Configuring improvement opportunities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/now-intelligence/process-mining/configuring-finding-definitions.md).

-   Select \[Omitted image "show-hide-find.png"\] Alt text: Show/hide icon to hide or show columns.
-   Select \[Omitted image "order-by-find.png"\] Alt text: Sort to sort the columns.
-   Select \[Omitted image "filter-finding.png"\] Alt text: Filter to filter the list of findings.
-   Select **Action**, and select any item from the list to see more details.
