# KPI dashboard

On the **KPI Dashboard** tab, see any KPI dashboard that you have configured for your project.

When a dashboard is configured for your business process and user needs, you can see quick snapshots from these dashboards.

You can configure KPI dashboards from two places:

-   From the project configuration: The KPI dashboards configured for your project are available only for the particular project. For more information, see [Set objectives](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/now-intelligence/process-mining/set-objectives.md).
-   From the process configuration: The KPI dashboards configured for a process table are available for all projects based on that table, by default. For more information, see [Configure impact metrics](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/now-intelligence/process-mining/impact-metrics.md).

**Note:** If you have configured a KPI dashboard from the project configuration, then that will take precedence over the one configured in the process configuration.
