# Summary and Insights page

The Summary and Insights page enables you to view opportunities for optimizing your process. Access the dashboard from a Process Mining project.

You can view the Project Metrics dashboard where you see some initial information about your mined data. This is showcased in two graphs:

-   Time series graph: Trendline to show the average time taken for completion of records over time.
-   Histogram: Distribution of records by the time taken to reach the process end.

\[Omitted image "proj-metrics.png"\] Alt text: Project Metrics

If you set up a KPI dashboard in the project configuration, you can view the KPI dashboard by selecting the Project Metrics drop-down. An administrator can also configure to view a process graph from the Process Mining Workspace. When a dashboard is configured for your business process and user needs, you can see quick snapshots from these dashboard sections.

The following figure shows the KPI Application Services dashboard, which was set up in the project configuration.

\[Omitted image "summ-insight-kpi.png"\] Alt text: KPI dashboard

## Improvement opportunities overview

On the Summary and Insights page, see the Improvement opportunities overview. Here you find an overview of project-related statistics that are generated on the project.

\[Omitted image "improve-opp.png"\] Alt text: Improvement opportunities

## Variation Analysis

On the Summary and Insights page, you can also see the variation analysis of your data.

Variation analysis enables you to compare different variants of a mined project and view the analysis of a selected variant.

\[Omitted image "variation-analysis-sum.png"\] Alt text: Variation Analysis from Summary and Insights page

-   Filter the routes using the filter icon.
-   Sort the results using the **Sort by** list.
-   Select **View process analysis** to view the graph for the selected variant in the Analyst Workbench.

## Opportunities

In the Opportunities section, you can view a list of rule-based and automated improvement opportunities associated with the project.

\[Omitted image "action-find.png"\] Alt text: Improvement Opportunities

You can view the following details:

-   Improvement opportunities that are new, under review, and archived.
-   List of improvement opportunities associated with the project with all the details.
-   Select \[Omitted image "details-finding.png"\] Alt text: finding details beside the title to see the details about the Improvement opportunities.
-   Select **Action**, and select any item from the list to see more details.

-   Select \[Omitted image "show-hide-find.png"\] Alt text: Show/hide icon to hide or show columns.
-   Select \[Omitted image "order-by-find.png"\] Alt text: Sort to sort the columns.
-   Select \[Omitted image "filter-finding.png"\] Alt text: Filter to filter the list of improvement opportunities.
-   Create an improvement opportunity by selecting the **Change the configuration** link. This link will automatically take you to the **Select improvement opportunities** tab in the Project Builder. For more information, see [Set improvement opportunities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/process-mining/improve-opportunities.md).

## Integration with Performance Analytics dashboards

Integrate Performance Analytics dashboards to include the KPIs and visualizations you want to see. Using the dashboard builder, you can create your own dashboards with custom widgets/KPIs specifically for your project. These dashboards can be linked at the project or process configuration level.
