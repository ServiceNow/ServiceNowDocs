# Run Process Mining evaluation project

Run Process Mining evaluation project to familiarize with improving your process with Process Mining capability.

## Before you begin

Role required: sn\_process\_optimization\_analyst, sn\_process\_optimization\_power\_user, or sn\_process\_optimization\_admin

## Procedure

1.  Navigate to **All** &gt; **Platform Analytics** &gt; **KPIs**.

2.  Select a KPI.

3.  Select **Run process analysis**.

    If the KPI is related to ITSM, then the sample mining starts.

    You will be able to view the records of the last 7 days and a maximum of 3600 records.
