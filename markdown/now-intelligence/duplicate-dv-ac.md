# Duplicate a visualization in the Visualization Designer

Duplicate a visualization so that you can use aspects of the source in a copy of the visualization.

## Before you begin

Role required: viz\_creator

## Procedure

1.  Navigate to **All** &gt; **Platform Analytics** &gt; **Library** &gt; **Data Visualizations**.

2.  Select the **Data Visualizations** tab.

3.  Select the visualization that you want to duplicate.

4.  From the More actions menu \(![](../../dashboards/image/icon-vert-3dot-p.png)\), select **Duplicate**.

5.  Give the duplicate a unique name and select **Duplicate**.


## Result

The visualization is copied with the new name. All other aspects of the visualization remain the same.

**Important:** The new copy of the visualization is automatically saved.
