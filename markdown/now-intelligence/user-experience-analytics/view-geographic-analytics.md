# Geographic analytics

See the regions, countries, and states mobile users access your application from.

Navigate to **Analytics** &gt; **Geographic** to see an app usage by country.

![Analytics module - Geographic screen](../image/analytics-module-geo-screen.png)

To drill down to a country's province, state, or territory breakdown, select the country in the map, or country name in the panel list.

'Other' indicates that it's unknown which countries users performed sessions from.

Select the **magnify** icon ![Magnify zoom icon.](../image/magnifier-zoom.png) next to a listed country to navigate to a filtered list of sessions from the country.

**Note:** Geographic analytics is only available up until the Xanadu release. Starting Yokohama, it no longer appears in its current visualization and is replaced by a sessions/users analysis breakdown.
