# UI Insights

The Insights module for UI Analysis on the User Experience Analytics dashboard shows a tabbed overview of screen KPIs.

![UI Analysis - Insights screen](../image/ui-analysis-insights-screen.png)

You can click a screen image on any of the following tabs to navigate to further analytics for the screen.

<table id="table_yms_lhj_xjb"><thead><tr><th>

Tab

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Quit Rate

</td><td>

**Screens with Highest Quit Rate** - Shows screens which users quit the most, measured by total number of sessions ending in the screen relative to total app quits.

</td></tr><tr><td>

User Interactions

</td><td>

**Screens with Most User Interactions** - Shows screens users engage with the most, measured by the number of gestures, for example, taps or swipes, performed in a screen relative to the total number of gestures performed by all users in the app.

</td></tr><tr><td>

Screen Popularity

</td><td>

**Most/Least Viewed Screens** - Shows screens with the highest and lowest view rates, measured by number of sessions that contain the screen relative to total number of app sessions.

</td></tr><tr><td>

User Engagement

</td><td>

**Screens with the Longest/Shortest Visit Duration** - Shows screens with the longest and shortest visit durations, with average visit durations in the screens.

</td></tr><tr><td>

Usability Problems

</td><td>

-   **Screens with Unresponsive Gestures** - Shows screens with the most unresponsive gestures, for example, screens that did not respond to user gestures. Measured by number of unresponsive gestures relative to the total number of gestures performed in the screen by all users.
-   **Screens with Slow Times to First Action** - Shows screens with the longest time to first action by showing average duration between the first screen appearance and the first gesture performed. These can indicate screens which are confusing to users.

</td></tr></tbody>
</table>**Parent Topic:**[UI analysis](ui-analysis.md)
