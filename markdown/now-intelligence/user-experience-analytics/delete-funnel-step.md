# Delete a funnel

Easily delete a funnel.

## Before you begin

Role required: Users with the analytics\_viewer, portal\_analytics\_viewer, mobile\_analytics\_viewer, core\_ui\_analytics\_viewer, now\_experience\_analytics\_viewer roles can create, edit and delete funnels for the applications they have access to.

## Procedure

1.  From the **Funnels** module, select to highlight from the funnel list the existing funnel you want to delete.

2.  Select the delete icon ![Delete icon](../image/delete-cohort-funnel-icon.png) to delete the funnel.

3.  Confirm that you want to delete the funnel by selecting **OK**.
