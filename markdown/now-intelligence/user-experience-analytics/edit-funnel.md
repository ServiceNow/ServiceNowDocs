# Edit a funnel

Edit funnel data to collect or steps to include in your analysis.

## Before you begin

Role required: Users with the analytics\_viewer, portal\_analytics\_viewer, mobile\_analytics\_viewer, core\_ui\_analytics\_viewer, now\_experience\_analytics\_viewer roles can create, edit and delete funnels for the applications they have access to.

## Procedure

1.  From the **Funnels** module, select from the existing list of funnels the funnel to edit.

2.  Select the edit ![](../image/edit-icon.png) icon.

3.  Change the funnel name, timeframe of sessions to include, and edit or add steps as applicable.

4.  Select **Finish** to update the funnel configuration.
