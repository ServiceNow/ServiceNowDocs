# Exploring User Experience Analytics

Access the User Experience Analytics application data via the user experiences analytics dashboard or by exporting the data to the UI Builder.

## User Experience Analytics overview

The ServiceNow® User Experience Analytics application provides dashboard views for monitoring usage analytics of your Next Experience web applications as well as Virtual Agent, ServiceNow mobile, and Service Portal applications.

## User Experience Analytics benefits

|Benefit|Feature|Users|
|-------|-------|-----|
|Enable or disable user experience analytics tracking|Link to configure UXA|Show now\_experience\_admin role|
|View domain separation data|both web and mobile| |
|View a session list|both web and mobile| |
|View a session event timeline|both web and mobile| |

-   **[Overview of the User Experience Analytics dashboard](../../concept/user-exp-analytics-dashboard.md)**  
User Experience Analytics shows detailed KPI data on a dashboard and on list views. Product managers gain relevant insight into the usage and adoption of a web or mobile application.
