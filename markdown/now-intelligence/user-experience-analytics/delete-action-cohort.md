# Delete a cohort

Easily delete a cohort report.

## Before you begin

Role required: Users with the analytics\_viewer, portal\_analytics\_viewer, mobile\_analytics\_viewer, core\_ui\_analytics\_viewer, now\_experience\_analytics\_viewer roles can create, edit and delete cohorts for the applications they have access to.

## Procedure

1.  From the **Cohorts** module, select from the drop down list the cohort you want to delete.

2.  Point to the cohort name to show the options, and click the delete ![Delete cohort or funnel icon.](../image/delete-cohort-funnel-icon.png) icon.

3.  Confirm that you want to delete the cohort by clicking **OK**.
