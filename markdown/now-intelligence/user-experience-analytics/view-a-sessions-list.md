# View a sessions list

View pre-filtered session lists, and search for session by additional properties.

## Before you begin

Role required: user

## Procedure

1.  From the Dashboard, navigate to **Sessions** and select the sessions list you want to view.

2.  |Sessions list option|Description|
|--------------------|-----------|
|All|Lists all user sessions.|
|First Sessions|Lists sessions for users that opened the app for the first time. Shows only the first session for each user.|
|Returning Users|Lists sessions for users who opened the app more than once. Shows only user sessions starting from their second session.|
|Loyal Users|Lists sessions for users who opened the app more than five times. Shows sessions starting from a user's fifth session and after.|
|Quick Abandons|Lists all sessions in which users spent less than 20 seconds in the first session.|
|Short Sessions|Lists sessions that lasted shorter than 20 seconds.|
|Long Sessions|Lists sessions that lasted longer than five minutes.|
|Offline Sessions|Lists all offline sessions.|
|Favorites|Lists individual sessions marked as favorites.|

3.  Define filters to refine the list results.

4.  View the list of sessions, or click an individual record to view more details of a specific user session.
