# Sessions and users list indicators

You can view and filter indicators and tags on a Sessions or Users list.

From the list filter, you can filter a user or sessions list by an available indicator or tag.

**Note:** Indicators added to records from within the Sessions or Users modules display only within that module. For example, a tag added to a user record from within the Users module displays only on the Users list, and not the Sessions list.
