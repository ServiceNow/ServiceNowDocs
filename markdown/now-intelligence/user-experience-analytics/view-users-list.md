# View a users list

View pre-filtered user lists, and search for users by more user or session properties.

## Before you begin

Role required: analytics\_admin, mobile\_analytics\_admin, web\_analytics\_admin, or portal\_analytics\_admin

## Procedure

1.  From the Dashboard, navigate to **Users**, then select the list you want to view.

2.  |Users list option|Description|
|-----------------|-----------|
|All|Lists all users.|
|New Users|Lists users that opened the app for the first time within the specified time period. Shows only the first session for each user.|
|Returning Users|Lists users that opened the app more than once across all devices.|
|Loyal Users|Lists users who opened the app more than five times. Shows sessions starting from a user's fifth session and after.|
|Never Returned|Lists users who opened the app only once.|
|Active Last Week|Lists users who opened the app within the last seven days.|
|Multi Platform|Lists users who have installed the app on both iOS and Android.|
|Multi Device|Lists users who have installed the app on multiple devices.|
|Favorites|Lists individual users marked as favorites.|

3.  Add filter conditions to further refine the list results.


## What to do next

Click a list item to view the selected user's session record.

## Filter a users list

Refine a users list to focus on data such as the web or mobile platform or the user's country, or add a custom filter.

### Before you begin

Role required: analytics\_admin, mobile\_analytics\_admin, web\_analytics\_admin, or portal\_analytics\_admin

### Procedure

1.  On the **Where** condition line, select a property from the filter menu.

2.  Select an operator and a value as applicable.

3.  To add another property, click the add property icon \(![Add property icon.](../image/add-filter-icon.png)\).

4.  -   To clear filters back to the original filter set for the user list, click **Reset**.
-   To remove a filter property, click the delete icon \(![Delete filter property icon.](../image/delete-filter-property.png)\) next to the property.
5.  Click **Search** to apply filters and refresh the list.
