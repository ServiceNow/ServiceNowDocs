# Bookmarking and viewing favorite user and session records

Manually select user and session records to view them in a favorites list.

## Before you begin

Role required: user

## About this task

Each user and session record in the dashboard has a star icon to the left of the user index or session date. Select the star icon to bookmark the record to a favorite list.

Select and deselect a star to add or remove a user or session record from a favorites list, and then view the favorite from the records you selected.

![Some records bookmarked as favourites in a sessions list.](../image/favorites-bookmarked.png "Records bookmarked as favourites in a sessions list")

## Procedure

1.  In either the user module or session module, bookmark a record by selecting the favorites icon \(![Favorites icon.](../image/favorites-icon.png)\) to the left of a record.

2.  View the records bookmarked for users or sessions by doing one of the following:

    -   In the modules menu, select **Users** &gt; **Favorites**. To filter the results of your favourites, see [Filter a users list](view-users-list.md#).
    -   In the modules menu, select **Sessions** &gt; **Favorites**. To filter the results of your favourites, see [Filter a sessions list](filter-session-list.md).
