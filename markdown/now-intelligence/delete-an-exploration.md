# Delete an exploration

Authorized users can delete an exploration in several ways.

## Before you begin

Role required: now\_assist\_explorer\_user and ownership of the exploration. Users with admin role can delete any exploration.

## Procedure

1.  Navigate to **All** &gt; **AI Data Explorer** &gt; **AI Data Explorer**.

2.  Expand the All explorations list.

    \[Omitted image "nowass-expl-show-explorations.png"\] Alt text: The Show explorations list button.

3.  To delete an exploration, either

    -   Delete the exploration from the All explorations list.

        \[Omitted image "nowass-expl-del-list.png"\] Alt text: Deletion option from the All explorations list.

    -   Open the exploration and delete it from the additional actions list in the toolbar.

        \[Omitted image "nowass-del-option.png"\] Alt text: Deletion option in the additional options menu.
