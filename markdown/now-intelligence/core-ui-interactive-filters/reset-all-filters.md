# Reset all interactive filters on a dashboard tab

Reset all applied filters on a dashboard tab to view the default value or the unfiltered data.

## Before you begin

Role required: none. You must have access to the dashboard.

## About this task

**Note:**

If there is a default value on a filter, resetting the filters returns that filter to the default setting rather than clearing it entirely.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Dashboards**.

2.  Select the dashboard and tab that you want to reset.

3.  Click the context menu \(\[Omitted image "Form\_MenuIcon.png"\] Alt text:\) and select **Reset Filters**.
