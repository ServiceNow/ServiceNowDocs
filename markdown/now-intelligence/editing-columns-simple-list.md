# Editing columns on a Simple List

You can select which columns are shown when viewing a Simple List. You cannot select more columns than the maximum number that the list is configured to show.

You can change which columns a List -Simple shows to you during your session. Your changes do not affect other users or persist after you close the list.

To change which columns a List - Simple shows, select the List actions cogwheel icon \[Omitted image "gear.png"\] Alt text: Cogwheel icon, then select **Edit columns**

\[Omitted image "ediet-columns.png"\] Alt text: Edit columns from the List actions menu.

You can select columns to add or remove selected columns by selecting the X. You can also drag selected columns to change the order in which they appear. You can also restore the default column selection and order.

\[Omitted image "edit-column-list.gif"\] Alt text: Editing the columns that appear in a list.

**Important:** You cannot add more columns than the maximum, which is set in the Simple List configuration. If you try to add more columns than this limit, your changes are ignored. You must either remove other columns, change the configuration, or ask an authorized colleague to do so. Changing the configuration affects all users, not only you. For more information, see [Create a simple list visualization in the Visualization Designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/now-intelligence/create-dv-simple-list-ac.md).
