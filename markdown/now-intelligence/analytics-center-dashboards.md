# Dashboards in Platform Analytics

Dashboards are reusable canvasses containing data visualizations, filters, and other visual components.

To access dashboards in Platform Analytics, navigate to **All** &gt; **Platform Analytics** &gt; **Library** &gt; **Dashboards**.

**Note:**

For information on the roles associated with Platform Analytics dashboards, see [Platform Analytics roles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/platform-analytics-roles.md).

For information about adding a Dashboards page to a configurable workspace, see [Add a dashboard to a Dashboards page](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/add-dashboard-to-workspace.md).

## Dashboards overview tiles

Each section of the Dashboards page shows up to 20 dashboard tiles at a time. You can filter the tiles on the Dashboards tab to show:

-   **Recent** shows the most recent Next Experience dashboards that you've viewed. The tile shows when you last opened the dashboard.
-   **Bookmarked** shows only your bookmarked dashboards. Bookmark dashboards that you have access to so that you can find them easily here. For more information about bookmarking dashboards, see [Bookmark a Platform Analytics dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/bookmark-dashboard-ac.md).
-   **Certified** shows only the dashboards that an administrator has certified. For more information, see [Certify a Platform Analytics dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/now-intelligence/certify-db-ac.md).
-   **Created by me** shows the dashboards that you’ve created.
-   **Shared with me** shows dashboards that other users have shared with you.
-   **All** shows all the dashboards that you have access to.
