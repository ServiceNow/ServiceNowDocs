# Delete a visualization in the Visualization Designer

You can delete a data visualization that is no longer useful.

## Before you begin

Inform any users who can view the visualization that you’re deleting it. No validation is executed to check if the visualization is used in a dashboard.

Role required: You can delete any data visualization that you created. If you have the admin or viz\_admin role, you can delete any data visualization.

## Procedure

1.  Navigate to **Platform Analytics** &gt; **Data visualizations**.

2.  Open the data visualization that you want to delete.

3.  From the More actions menu \(![More actions menu icon](../image/icon-vert-3dot-p.png)\), select **Delete**.

4.  Confirm the deletion.

    You cannot undo this action.
