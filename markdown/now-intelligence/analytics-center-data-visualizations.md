# Data visualizations in Platform Analytics

Data Visualizations use graphics to show table, indicator, and other data. They combine and extend the functions of reports and Performance Analytics widgets from the Core UI.

How to create a data visualization in the Visualization Designer. 

**Important:** You cannot place reports or Performance Analytics widgets on a Platform Analytics dashboard. You must create data visualizations instead.

To access data visualizations, navigate to **All** &gt; **Platform Analytics** &gt; **Library** &gt; **Data Visualizations**. Owners of a data visualization can access and share that visualization. Users whom the owner has shared a visualization with can access that visualization. Any user with access to data can create a data visualization based on that data.

![Analytics Center Data Visualizations tab](../../par-for-workspace/image/analytics-center-data-vis-tab.png "Analytics Center Data Visualizations tab features")

## List actions

A set of buttons enables you to take the following actions on the Data Visualization list:

-   **Refresh list ![The Refresh List button.](../../par-for-workspace/image/analytics-center-dv-refresh.png)**

    There can be a lag in the list of visualizations after you create or delete a visualization. Select **Refresh List** to update the list.

-   **Copy URL for data visualizations ![Copy URL button.](../../par-for-workspace/image/analytics-center-vd-copy-url.png)**

    Copies the data visualizations library URL to the clipboard.

-   **New**

    Create a data visualization, as described by visualization type in the subtopics of this section. For information on who can create data visualizations,see [Platform Analytics roles](../../par-for-workspace/reference/platform-analytics-roles.md).


## List filters

A search box lets you filter the list of data visualizations by name or description. The filter panel shows you which filters are being applied. By default, the filter Active = true is always applied. For example, filtering on the word Data shows the following filters in the panel:

-   Active = true
-   Name contains Data
-   Description contains Data

Alternatively, you can select one of these predefined filters:

-   **All** shows all the visualizations that you have access to.
-   **Bookmarked** shows only your bookmarked visualizations. Bookmark any data visualization that you have access to so that you can find it here easily. To bookmark a visualization, see [Bookmark a visualization in the Visualization Designer](../../dashboards/task/bookmark-dv-ac.md).
-   **Certified** shows only the visualizations that an administrator has certified. To certify a data visualization, see [Certify a data visualization in the Visualization Designer](../../dashboards/task/certify-dv-ac.md).
-   **Shared with me** shows the visualizations that have been shared with you as a user, or member of a group or role. To share a data visualization, see [Share a data visualization in the Visualization Designer](../../reporting/task/share-dv-ac.md#).
-   **Created by me** shows only the visualizations that you’ve created.
