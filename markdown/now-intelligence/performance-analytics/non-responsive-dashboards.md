# Working with non-responsive dashboards

From the Utah release, non-responsive dashboards are no longer available.

On upgrade, non-responsive dashboards are converted to [responsive dashboards](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/now-intelligence/performance-analytics/c_ResponsiveDashboards.md).
