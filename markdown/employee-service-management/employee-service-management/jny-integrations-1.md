# Integrations for Journey designer app

Extend the Journey designer application's capabilities by integrating other applications into the journey experience.

Integrating supported applications expands the features and capabilities of Journey designer. Supported applications have unique installation and configuration requirements. Supported integration applications can be installed before or after installing the Journey designer app.

-   [Learning Posts](learning-post-app-1.md) v5.0.0 \[sn\_lnp\]
-   [Listening Posts](listening-posts-1.md) v5.0.0 \[sn\_lp\]
-   [Manager Hub](manager-hub-overview.md) v1.0.0 \[sn\_mh\]
-   [Microsoft Exchange Online spoke](https://www.servicenow.com/docs/access?context=ms-exch-online-spoke&version=yokohama&pubname=yokohama-integrate-applications&ft:locale=en-US) v3.2.0 \[sn\_ex\_online\_spke\]
-   [Retry Handler Framework v1.0.1 \[sn\_handler\]](https://www.servicenow.com/docs/access?context=retry-handler-frmwrk&version=yokohama&pubname=yokohama-integrate-applications&ft:locale=en-US)

    The Retry Handler Framework is required when using the Microsoft Exchange Online spoke.


-   **[Integrating Learning Posts with Journey designer](jny-dsgnr-intgrt-lrnng-pst.md)**  
Integrate Learning Posts with Journey designer to create and access an omni-channel learning experience empowering employees with the right learning at the right time, making learning accessible.
-   **[Integrating Listening Posts with Journey designer](integrating-li-with-jny.md)**  
Use Listening Posts with Journey designer to embed pulse surveys to get insight into employees' experiences and share those insights with managers.
-   **[Integrating Manager Hub with Journey designer](jny-dsngr-manager-hub-intgr.md)**  
Integrate Manager Hub with Journey designer to facilitate daily activities among various cross-functional processes and departments, and automate processes such as new hire onboarding.
-   **[Integrating Microsoft Exchange Online spoke with Journey designer](jny-dsgnr-ms-exch-ol-spoke-intg.md)**  
The Microsoft Exchange Online spoke integration with Journey designer is achieved with Journey Accelerator.
-   **[Integrating HR Service Delivery Virtual Agent Conversations app with Journey designer](jny-dsgnr-virtual-agent.md)**  
Install the HR Service Delivery Virtual Agent Conversations app to enable Journey designer to send actionable notifications as nudges to employees and managers on configured communications channels.
