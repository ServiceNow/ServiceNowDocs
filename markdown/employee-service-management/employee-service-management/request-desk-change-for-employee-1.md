# Create a move request on behalf of an employee

Create a move request on behalf of an employee from the application directly instead of using the Workplace Service Portal.

## Before you begin

Role required: sn\_wsd\_move.manager

## About this task

Specify the current location or current desk that the employee is moving from, and the target location or target desk that the employee is moving to. Select the date of movement and the reason for moving. You can also provide extra information related to the move if required.

## Procedure

1.  Navigate to **All** &gt; **Workplace Move Management** &gt; **Move requests**.

2.  Click **Create new**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |On behalf of|Name of the employee for who you are raising the move request. If you are raising the move request for yourself, select your name.|
    |Requested move date|Date on which the movement must be completed. Select a date from the calendar.|
    |Move reason|Reason for moving from the current location. Select from the list of available reasons.|
    |**From:**|
    |Region|Region where the current desk is located.|
    |Site|Site where the current desk is located.|
    |Campus|Campus in which the current desk is located.|
    |Building|Building in which the current desk is located.|
    |Floor|Floor on which the current desk is located.|
    |Space|Space in which the current desk is located.|
    |**To:**|
    |Region|Region where the target desk is located.|
    |Site|Site where the target desk is located.|
    |Campus|Campus in which the target desk is located.|
    |Building|Building in which the target desk is located.|
    |Floor|Floor on which the target desk is located.|
    |Space|Space in which the target desk is located.|
    |What should be moved?|Items to be moved from the current desk to the target desk.|
    |Additional notes|Extra information regarding the move request.|

4.  Click **Submit**.


## Result

The move request is submitted to change the employee's desk. If there are personal items such as communication equipments, ergonomic furniture, or any personal belongings to be moved to the new location, then the **Move personal items** workplace service-related service activities are performed.

If the state is set to Closed incomplete, it means that the move request could not be completed due to various reasons. As a manager, you can review if there are any conflicts with the move request in the Conflicts review related list.
