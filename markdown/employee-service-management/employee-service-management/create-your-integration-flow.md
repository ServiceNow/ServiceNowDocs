# Create integration flow for ATS

Create the integration flow with your preferred ATS to send job requisition data to the ATS.

## Use Integration Hub

Use the Integration Hub to integrate Hiring Experiences with your preferred ATS. For more information, see [Integration Hub spokes](https://www.servicenow.com/docs/access?context=spokes-list&version=zurich&pubname=zurich-integrate-applications&ft:locale=en-US).

## Supporting actions

Create the supporting actions after the integration is done to establish a channel between your ATS and Hiring Experiences. The supporting actions vary depending on your ATS.

## Export and import data manually

You can export the list of job requisitions from the Job Requisition \[sn\_ta\_hiring\_core\_job\_requisition\] table and imported the data to an ATS manually.

For more information, see [Exporting data](https://www.servicenow.com/docs/access?context=c_ExportData&version=zurich&pubname=zurich-platform-administration&ft:locale=en-US).
