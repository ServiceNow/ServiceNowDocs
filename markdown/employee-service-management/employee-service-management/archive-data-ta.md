# Rules to archive data

Create rules to archive and delete data according to your organizational policy.

You have the option to archive and delete data after a specified time, instead of setting up purge policies.

Create archive and destroy rules to archive data and delete the archived records.

For more information, see [Data archiving](https://www.servicenow.com/docs/access?context=c_ArchiveData&version=zurich&pubname=zurich-platform-administration&ft:locale=en-US).
