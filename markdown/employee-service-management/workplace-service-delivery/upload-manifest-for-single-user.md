# Upload the manifest file for a single user

After you create the manifest file, make it available for usage within Microsoft Outlook. You can upload the add-in for a single user use.

## Before you begin

Ensure that you have created a manifest file for the add-in.

Role required: sn\_wsd\_msaddin.admin

## Procedure

1.  Open **Microsoft Outlook**.

2.  On the Outlook ribbon, click **Get Add-ins**.

3.  Select **My add-ins**.

4.  In the Custom add-ins section, select **Add a custom add-in** &gt; **Add from file**.

5.  Select the manifest file that you have created for the add-in.

6.  Click **Next** &gt; **Install**.


## Result

The Workplace Reservations for Microsoft Outlook Add-in is installed. The add-in appears while scheduling a meeting in the Outlook.
