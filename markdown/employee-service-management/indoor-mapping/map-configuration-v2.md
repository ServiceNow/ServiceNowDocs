# Map Configuration tab

Manage CAD layers, places, text, block, layer types, and place types using the Map Configuration preview layout.

Select the **Map Configuration** tab to open the Map configuration page. For more information, see [Configure CAD objects using the Map Configuration tab](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/manage-cad-map-config.md).

\[Omitted image "wsd-map-config-showing-left-panel.png"\] Alt text: Map configuration preview layout with places and layers option.

## Georeference

Georeference your floor plan and position it on the map to scale and align it on the World map \(OpenStreetMap\). For more information, see [Georeference your floor plan](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/georeference-floor-plan.md).

## Layers

Configure the layers in your CAD file and preview it on the Map Configuration preview layout. For more information, see [Configure layers](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/wsd-edit-layers.md).

\[Omitted image "wsd-layers-expanded-autocadv2.png"\] Alt text: Layer summary pill showing additional information for a selected layer type.

## Places

Configure place in your CAD file and preview it on the Map Configuration preview layout. For more information, see [Configure places](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/edit-places.md).

\[Omitted image "wsd-places-expanded-view-autocad-v2.png"\] Alt text: Places summary pill showing additional information for a selected place type.
