# Extension point for occupancy and environment data

Create a scripted extension point to process occupancy and environment data from the provider and convert it into a standard format. Confirm that each provider implements this extension.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Extension Points** &gt; **Scripted Extension Points**.

2.  Search for `sn_wsd_wc.OccupancyEnvironmentDataHandler`.

3.  On the form banner, select the **here** link to edit the record.

4.  Select the **Create Implementation** related link to create your occupancy and environment data specific extension point script.
