# Configure environmental sensor data for a workplace location

Create environmental sensor data for a workplace location in Workplace Connectors. The Occupancy and Environment Data \(sn\_wsd\_wc\_occ\_env\_data\) table checks for carbon dioxide value, ozone, sound, light, temperature, pressure, and humidity levels at a workplace location. You can also view the time-series data for each value that is retrieved from the MetricBase.

## Before you begin

**Note:** Seed data or out-of-the box table record is provided with the application and provides pre-filled values.

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Connectors** &gt; **Occupancy and Environment Sensors** &gt; **Occupancy and Environment Data**.

2.  On the form, review the field information for environmental data.

    For more information about Environmental form fields, see [Environmental data form for Workplace Connectors](../reference/wsd-environmental-data-form-fields.md).
