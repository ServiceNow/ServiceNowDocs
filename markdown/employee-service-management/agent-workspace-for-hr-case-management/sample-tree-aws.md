# View a sample decision tree in Agent Workspace for HR Case Management

View sample trees that triage general inquiry or payroll cases in Agent Workspace for HR Case Management.

## Before you begin

Role required: sn\_hr\_core.admin

## Procedure

1.  Navigate to **All** &gt; **Guided Decisions** &gt; **Decision Trees**.

2.  Select one of the following:

    -   **General Benefits Tree**
    -   **Pay issue guided decision**
3.  Select **Open in Builder** to view the tree in **Decision Tree Builder**.

    For more information, see [Configuring decision trees in Decision Tree Builder](https://www.servicenow.com/docs/access?context=decision-trees-in-gdb&version=xanadu&pubname=xanadu-customer-service-management&ft:locale=en-US).
