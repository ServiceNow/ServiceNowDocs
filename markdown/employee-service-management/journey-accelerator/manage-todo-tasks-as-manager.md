# Manage action plan to-do tasks and create templates

Create, modify, and manage to-do tasks for employees and mentors as part of your management responsibilities.

Journey Accelerator action plans can be customized for employee transitions. Part of that customization is a managers ability to add, delete, modify, and save to-do tasks. Save the to-do tasks you create as templates to save time and provide consistent experiences for employees and mentors.

-   **[Add or modify an action plan to-do](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/journey-accelerator/add-todo-task.md)**  
Create, modify, and save to-do tasks as part of an action plan. To-dos are assigned to employees, mentors, or managers to complete.
