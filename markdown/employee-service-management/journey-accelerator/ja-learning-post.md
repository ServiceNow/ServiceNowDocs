# Learning Posts with Journey Accelerator

Quickly create action plans that include to-do tasks for learning and training with Learning Posts.

With the Learning Posts application, you can embed learning courses from commonly used Learning Management Software \(LMS\) systems such as Saba, Udemy, Cornerstone, SumTotal, and others with included integrations. For more information, see [Learning Posts](learning-post-app-1.md).

Include courses focused on the employee needs. Managers can easily search for training and add it as a to-do task in an action plan. Create plans with to-do tasks for new employees, team members returning from leave, or employees taking on a new role. Employees access the learning directly from an action plan or to-do task.

Activate the [Learning Posts](learning-post-app-1.md) \[sn\_lnp\] application from the ServiceNow Store to automatically activate the Learning Core \[sn\_lc\] application. The Learning Core application contains the base set of tables, content, learning bases, roles, and access configuration that are used in the [Learning Posts](learning-post-app-1.md) \[sn\_lnp\] application.
