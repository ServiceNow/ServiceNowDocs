# Workplace Central troubleshooting

Troubleshoot issues that you face in the Workplace Central application.

## Condition

After transferring Workplace Central data between ServiceNow ServiceNow instances, some data and links are lost.

## Remedy

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Fix Scripts**.

2.  From the Fix Scripts list, select the **Load space details table- current state** script.

3.  On the Fix Script details page, select **Run Fix Script**.
