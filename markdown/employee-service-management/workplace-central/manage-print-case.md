# Managing print case

Filter workplace cases using criteria such as Short Description, Workplace Location, or other relevant fields to generate a consolidated report using the Workplace Central Case management workspace.

Use one of the following options to proceed:

-   Print Filter Selection – Apply filters and select this option to create a workplace task with the consolidated report attached.
-   Print Selected Records – Manually select one or more workplace cases and choose this option to view the consolidated report directly.

## Print filtered cases

Use filter criteria to filter multiple cases and generate a consolidated report based on the filtered cases. You can set the condition based on the Short description, workplace location, or any other fields. When you run the filter and select Print filter selection, a new workplace task is created with the consolidated report as an attachment.

### Before you begin

Role required: sn\_wsd\_case.case\_reader

### Procedure

1.  Navigate to **All** &gt; **Workplace Central** &gt; **Workplace Central**.

    You can also open Workplace Central from the Employee Center directly. Navigate to **Workspaces** &gt; **Workplace Central**.

    The Workplace Analytics dashboard opens.

2.  On the left pane, select the **Case Management** icon \(\[Omitted image "casemgmt-icon.png"\] Alt text: Case Management icon.\).

    The Case Management landing page opens.

3.  Select the filter icon and set the filter criteria.

    The results appear based on the filtered condition.

4.  Select **Print filter selection**.

    You can view the consolidated report with the filtered case details in a new Workplace Task page.


## Print selected cases

Print a consolidated report, including task details and associated services, by selecting one or more workplace cases.

### Before you begin

Role required: sn\_wsd\_case.case\_reader

### Procedure

1.  Navigate to **All** &gt; **Workplace Central** &gt; **Workplace Central**.

    You can also open Workplace Central from the Employee Center directly. Navigate to **Workspaces** &gt; **Workplace Central**.

    The Workplace Analytics dashboard opens.

2.  On the left pane, select the **Case Management** icon \(\[Omitted image "casemgmt-icon.png"\] Alt text: Case Management icon.\).

    The Case Management landing page opens.

3.  Select one or more workplace cases.

4.  Select **Print selected cases**.

    You can view the consolidated report with the selected case details in a new Workplace Task page.
