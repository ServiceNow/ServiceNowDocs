# Use the Now Assist for Legal Service Delivery \(LSD\) capabilities from the Now Assist panel

Use the contextual generative AI capabilities, such as a request summary by using the conversational interface in the Now Assist panel.

## Before you begin

Make sure that Next Experience is enabled in the instance. For more information, see [Next Experience UI](https://www.servicenow.com/docs/access?context=next-experience-landing-page&version=zurich&pubname=zurich-platform-user-interface&ft:locale=en-US).

Role required: sn\_lg\_gen\_ai.request\_fulfiller

## About this task

You can use the Now Assist panel to request a summary.

For more information about the Now Assist panel, see [Now Assist panel](https://www.servicenow.com/docs/access?context=now-assist-panel-overview&version=zurich&pubname=zurich-intelligent-experiences&ft:locale=en-US). For information about activating the Now Assist panel, see [Activate the Now Assist panel standard chat](https://www.servicenow.com/docs/access?context=activate-now-assist-panel&version=zurich&pubname=zurich-intelligent-experiences&ft:locale=en-US).

## Procedure

1.  Log in to an instance where the Now Assist for LSD application is installed.

2.  Navigate to **Workspaces** &gt; **Legal Counsel Center**.

3.  Request the generative AI capabilities in LSD for summarization by selecting the Now Assist icon ![Now assist icon.](../../../common/image/icon-ai-sparkle.png).

4.  After you request the generative AI capabilities, select **Summarize a record**.

5.  Enter the record number that you want to summarize.

    The summary is displayed.

    ![Now Assist panel with selections for generating a request summarization](../image/lsd-na-side-panel.png)
