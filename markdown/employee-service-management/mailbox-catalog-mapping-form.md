# Office Mailbox to Catalog Field Mappings form

The Office Mailbox to Catalog Field Mappings form helps you map an email field to a catalog item field.

<table id="table_jvk_wyw_pcc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Catalog item

</td><td>

Catalog item that the email fields must be mapped to.This field is automatically set to the Catalog item selected for the mapping configuration record.

</td></tr><tr><td>

Application

</td><td>

Application scope of the record.This field is automatically set to the application scope of your instance.

</td></tr><tr><td>

Catalog item field

</td><td>

Catalog item field that must be mapped to the email field.

</td></tr><tr><td>

Mailbox item property

</td><td>

Email field that must be mapped to the catalog item field.

</td></tr><tr><td>

Active

</td><td>

Option to activate the record.
