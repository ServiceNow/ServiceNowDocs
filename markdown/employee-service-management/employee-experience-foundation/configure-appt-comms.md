# Configure appointment communication

Configure the communication channels and location consent for appointment bookings.

## Before you begin

Role required: sn\_hr\_core.walkup\_admin, sn\_walkup.walkup\_manager

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **Appointment Booking** &gt; **Appointment Communication Configurations**.

2.  On the Location Consent list, select **New**.

3.  On the form, fill in the fields.

    For a description of the field values, see [Location Consent form](../reference/location-consent-form.md).


## What to do next

Link the communication configuration to the walk-up location queue that you want to use for appointment booking. For more information, see [Create or modify Walk-up Experience locations](https://www.servicenow.com/docs/access?context=configure-walkup-location&version=yokohama&pubname=yokohama-it-service-management&ft:locale=en-US).
