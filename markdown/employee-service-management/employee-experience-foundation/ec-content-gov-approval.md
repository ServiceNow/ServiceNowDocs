# Add or remove content approvers

A content admin can modify the content approvers list for a request.

## Before you begin

Role required: sn\_cd.content\_admin

## Procedure

1.  Navigate to **All** &gt; **Content Governance** &gt; **All requests**.

2.  Click the content request you want to modify.

3.  Scroll down to the content request item related list and select the request item.

4.  Click **Manager approvers**.

5.  Use the left and right arrows to add or remove users from the approvers list.

6.  Click **Submit**.

7.  Click **Update**.
