# Portal notification content form

The field descriptions provided here are required to configure your portal notification messages.

<table id="table_hqc_cqf_c2c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name for the notification content record.

</td></tr><tr><td>

Notification

</td><td>

Type of notification content.

</td></tr><tr><td>

Portals

</td><td>

Portals where the notifications show up.

</td></tr><tr><td>

Table

</td><td>

Record table for the notification type.This field is automatically set to the record table depending on the **Notification** type selected.

</td></tr><tr><td>

Active

</td><td>

Option to enable the notification content configuration.

</td></tr><tr><td>

Advance

</td><td>

Option to enable support for dynamic content templates and grouping of unseen notifications.

</td></tr><tr><td>

Content Configuration

</td><td>

Content template: Display message for the notification.The field supports variable data by setting and using different parameters.

</td></tr><tr><td>

Content Configuration

</td><td>

Content params: Parameters used to provide notification messages with variable data.

</td></tr><tr><td>

Content Configuration

</td><td>

Action url: URL that redirects to the record table of each notification type.

</td></tr><tr><td>

Icon Configuration

</td><td>

User custom icon: Option to enable customized icon for portal notifications.

</td></tr><tr><td>

Icon Configuration

</td><td>

Custom icon: Customizable icon for different notification content types.This section appears only when **User custom icon** is enabled.

</td></tr><tr><td>

Icon Configuration

</td><td>

Icon: List of icons to set for different notification content types.

</td></tr><tr><td>

Icon Configuration

</td><td>

Invert icon color: Option to enable invert icon color.

</td></tr><tr><td>

Icon Configuration

</td><td>

Icon background: Background colors for the notification icons.

</td></tr></tbody>
