# Defer load in Employee Center

Learn more about the heavy-load and high-traffic scenarios and ways to optimize the user experience and portal performance.

The Service Portal Designer **Defer widget loading** option helps configure the widgets load in a skeleton view. For more information, see [Create and edit a page using the Service Portal Designer](https://www.servicenow.com/docs/access?context=t_ConfigureAPage&version=yokohama&pubname=yokohama-platform-user-interface&ft:locale=en-US). The key benefits are:

-   Improve performance and reduce initial load times.
-   Enhance user experience with skeleton loaders.
-   Create a faster loading experience by enhancing user satisfaction.
-   By matching skeleton loaders to the final widget appearance, the loading process becomes visually coherent, improving the overall usability of the page.
-   Improve performance across browsers and devices.
