# User Criteria form

Add a user criteria record to decide which users, roles, and groups can access a quick link.

<table id="table_ehp_bkl_mqb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the user criteria.

</td></tr><tr><td>

Users

</td><td>

Users who can access the quick link when you apply the user criteria. Click the Unlock users icon to select users. Click the Add me icon to add yourself as a user. Click the Add email address icon to add a user's email address.

</td></tr><tr><td>

Groups

</td><td>

Groups who can access the quick link when you apply the user criteria. Click the Unlock Groups icon to select the groups.**Note:** Use Groups instead of the Advanced user criteria to get your data cached.

</td></tr><tr><td>

Roles

</td><td>

Roles who can access the quick link when you apply the user criteria. Click the Unlock Roles icon to select the roles.

</td></tr><tr><td>

Advanced

</td><td>

Option to create a script for the user criteria.**Note:** Use Groups instead of the Advanced user criteria to get your data cached.

</td></tr><tr><td>

Application

</td><td>

This field is automatically set to Employee Center.

</td></tr><tr><td>

Active

</td><td>

Option to activate this user criteria for use.

</td></tr><tr><td>

Companies

</td><td>

Companies who can access the quick link when you apply the user criteria. Click the Unlock Companies icon to select the companies.

</td></tr><tr><td>

Locations

</td><td>

Locations which can access the quick link when you apply the user criteria. Click the Unlock Locations icon to select the locations.

</td></tr><tr><td>

Departments

</td><td>

Departments who can access the quick link when you apply the user criteria. Click the Unlock Departments icon to select the departments.

</td></tr><tr><td>

Match All

</td><td>

Option to make every condition required when the user criteria is applied. The conditions are set in the previous fields, such as Companies, Locations, Departments, and so on.

</td></tr></tbody>
