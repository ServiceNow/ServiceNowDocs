# Footer Menus form

Use the Footer Menus form to create a new footer menu that you want to appear on your Employee Center or Employee Center Pro.

<table id="table_fgv_pyx_lqb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Footer

</td><td>

The name of the footer that you want to create menus for.

</td></tr><tr><td>

Menu

</td><td>

The main category under which you want the footer links displayed.

</td></tr><tr><td>

Link Type

</td><td>

How you want the links on the menu to appear in the footer on the Employee Center. Select any of the following options:-   **Navigation Links**: Displays links at the top of the footer. You can add links that let you navigate to different information resources and pages.
-   **Bottom Links**: Displays links at the bottom of the footer. You can add links to information such as, terms and conditions, privacy statement, and so on.
-   **Social Icons**: Displays social media icons, such as Twitter, Facebook, and LinkedIn.

</td></tr><tr><td>

Order

</td><td>

The order in which you want the menu to appear in the footer.

</td></tr></tbody>
