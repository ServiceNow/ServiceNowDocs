# Taxonomy form

Use the taxonomy form to create your own taxonomy for the Employee Center portal.

<table id="table_vdm_zcw_rqb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Unique name that identifies the taxonomy.

</td></tr><tr><td>

Managers

</td><td>

Managers or manager group of the taxonomy content.

</td></tr><tr><td>

Application

</td><td>

Application scope that the taxonomy belongs to.For example, Employee Center Pro or Employee Center Pro Kiosk.

</td></tr><tr><td>

Active

</td><td>

Indicates if the taxonomy content is active or not.

</td></tr><tr><td>

Domain

</td><td>

Domain the taxonomy content belongs to.

</td></tr><tr><td>

Description

</td><td>

Description of the taxonomy content.
