# Update the Employee Center home page theme

Enable the gradient background theme along with other UI changes on the Employee Center home page as an existing customer.

## Before you begin

Role required: sp\_admin

## About this task

The gradient background theme is available with Employee Center version 37.0 by default for new customers. It comes with several user experience enhancements like a gradient-themed background and search bar on top.

For existing customers, the following procedure must be followed to enable the theme after upgrading to Employee Center version 37.0.

## Procedure

1.  Navigate to **Service Portal** &gt; **Portals**.

2.  Select the Employee Center \(esc\) portal, update the **Homepage** field to the updated portal home page available by default, and select **Save**.

    **Note:** For Employee Center version 37.0, the updated home page provided with the gradient background theme is ec\_standard\_home.


## Result

Your Employee Center home page has the updated user interface.
