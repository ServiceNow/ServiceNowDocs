# Change the number of returned search results

Specify the maximum number of search results the system should return.

## Before you begin

Role required: esc\_admin

## About this task

-   Search content on the topic pages.
-   Support for Knowledge Base article, Learning content, and Service Catalog item content types are available with the feature.
-   Change the default number of results displayed according to your organizational requirements.

## Procedure

1.  Navigate to **All** &gt; **AI Search** &gt; **Search Experience** &gt; **Search Applications**.

2.  From the applications list, click **ESC Portal Default Search Application**.

3.  Change the **Search Results Limit** to your desired number from the base system value 12.
