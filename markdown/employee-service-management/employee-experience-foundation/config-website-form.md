# Website configuration form

The field descriptions provide a point of reference when you're configuring the Browser Extension for Employee Center for a website.

<table id="table_z1x_1tq_vfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Site name

</td><td>

Name of the website.

</td></tr><tr><td>

Parent website

</td><td>

Parent website helps with the logical grouping of certain related websites.

</td></tr><tr><td>

Enable resources search

</td><td>

AI search results, if applicable.

</td></tr><tr><td>

Site URL

</td><td>

URL of the website being configured.

</td></tr><tr><td>

Active

</td><td>

Option to enable the Browser Extension configuration for the website.

</td></tr><tr><td>

Apps Associated

</td><td>

List of active applications to be associated with access from the website through the Browser Extension.**Note:** Applications can also be added to the configured website from the **Associated Websites** tab on the application page.

</td></tr><tr><td>

Quick links Associated

</td><td>

List of quick links from the portal to be associated with access from the website through the Browser Extension.**Note:** Quick links can also be added to the configured website from the **Associated Websites** tab on the quick links page.

</td></tr><tr><td>

Topics Associated

</td><td>

List of topic pages from the portal to be associated with access from the website through the Browser Extension.**Note:** Topic pages can also be added to the configured website from the **Associated Websites** tab on the specific topic page.

</td></tr><tr><td>

Contents Associated

</td><td>

List of connected content from the portal to be associated with access from the website through the Browser Extension.**Note:** Content can also be added to the configured website from the **Associated Websites** tab on the specific content page.

</td></tr></tbody>
