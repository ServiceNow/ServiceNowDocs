# Configure search navigation tabs

In Employee Center, the AI search navigation tabs are enabled by default. You can configure a search navigation tab on the search results.

## Before you begin

Role required: admin

Ensure you understand the AI search and indexed sources. For more information, see [Create an indexed source](https://www.servicenow.com/docs/access?context=create-indexed-source-ais&version=xanadu&pubname=xanadu-platform-administration&ft:locale=en-US) and [Configuring navigation tabs in AI Search](https://www.servicenow.com/docs/access?context=config-nav-tabs-ais&version=xanadu&pubname=xanadu-platform-administration&ft:locale=en-US).

## About this task

The search navigation tabs help you to refine their search query results by search source.

## Procedure

1.  Navigate to **All** &gt; **AI Search** &gt; **AI Search Index** &gt; **Indexed Sources**.

2.  Use the available **AI Search Indexed Sources** or click **New**.

    By default, the following sources are available for Employee Center.

    -   Guided Self-Service
    -   Applications
    -   Articles
    -   News
    -   Events

        **Note:** Ensure you install the required applications.

3.  Click **Submit**.


## Result

The configured sources appear on AI search results in a tab. For more info on how tabs appear in AI search, see [Configure application visibility and access](applauncher-enable-availability.md).
