# Featured Content form

Use the featured content form to mark the associated connected content as featured content for a topic.

<table id="table_yrp_bbk_sqb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Connected Content

</td><td>

Connected content associated with the topic. Search for and select the connected content associated with the topic.**Note:** You can't add a quick link as a featured item.

</td></tr><tr><td>

Topic

</td><td>

Topic in which you want the featured content to appear.

</td></tr><tr><td>

Order

</td><td>

Order in which you want the featured content to appear.
