# Content Analytics dashboards

Content Analytics display data visualizations for portal data collected in the default tracking profile.

## Content Analytics dashboard

The Content Analytics dashboard displays portal engagement metrics, including unique users.

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Adoption

</td><td>

Tracks the number of new and returning \(unique\) users each day

</td></tr><tr><td>

Bounce Rate

</td><td>

The percentage of users who leave the portal after viewing only one page without further interaction

</td></tr><tr><td>

Average Session Duration

</td><td>

Average number of minutes a user spends in a session. The system tracks a session as either-   30 minutes after a user opens a page
-   Until the user closes the tab

</td></tr><tr><td>

Average Time per Page

</td><td>

Average number of minutes a user spends looking at a page

</td></tr><tr><td>

Engagement Metrics

</td><td>

Type of engagement \(page views and session\) by week represented with vertical bars, with a line representing average weekly page views

</td></tr><tr><td>

Visits by Browser

</td><td>

Number of views by browser type

</td></tr><tr><td>

Visits by Device

</td><td>

Number of views by device type

</td></tr><tr><td>

Unique Users
