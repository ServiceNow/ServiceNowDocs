# Content Analytics dashboards

Content Analytics display data visualizations for portal data collected in the default tracking profile.

## Content Analytics dashboard

The Content Analytics dashboard displays portal engagement metrics, including unique users.

|Widget|Description|
|------|-----------|
|Adoption|Tracks the number of new and returning \(unique\) users each day|
|Engagement Metrics|Type of engagement \(page views and session\) by week represented with vertical bars, with a line representing average weekly page views|
|Unique Users|Total number of users from the previous day|
|Daily Top Pages|Top 5 pages for the last number of days|
|Weekly Top Pages|Top 5 pages for the last number of weeks|

## Content Analytics- Basics dashboard
