# Content Library Overview dashboard

Content analytics displays data visualizations of performance metrics for content published to the portal or Now mobile app in the Content Library Overview dashboard.

## Dashboard overview

The Content Library Overview dashboard displays metrics for content published via a schedule or publish plan. This dashboard does not display content delivered via a campaign. For campaign content data visualizations, see [Campaign overview and Campaign analytics dashboards](ecpro-content-automation-content-pack.md).

|Visualization|Description|
|-------------|-----------|
|Open Content Tasks|Number of tasks by status \(such as open or closed complete\). Each bar in the chart links to a list of the tasks with that status.|
|Recent Content Task Closures|Vertical bar chart that shows the number of tasks that were closed by time period. Each bar in the chart links to a list of tasks closed within that time frame|
|Event Activity Over Time|Total number of user interactions with content.|
|Events Over Time by Content|Number of user interactions with content, where each line represents a piece of content. Mouse over a line to see metrics specific to a piece of content.|
|Events Over Time by Action|Number of user interactions with content, where each line represents a type of interaction. Mouse over a line to see metrics specific to a type of interaction.|
|Top Content|Bar chart showing the most accessed content. Clicking a bar shows metrics for the specific content and action.|
|Action Activity|Pie chart that shows the proportions of each type of interaction and impressions compared to the total number of actions.|
|Top Notifications|Bar chart showing the most clicked notifications for the specific time period.|

## Interaction tracking

Dashboards display data visualizations for specific parameters, such as user interactions with content.

**Note:**

-   An interaction is an action performed by the user, such as a download, link click, to-do completion, or video play.
-   An impression is an instance of content appearing for a user. Each time a user views the content, it is counted as an interaction.

Content Analytics tracks the following user interactions \(depending on content type\). When content is rendered in the portal, the system counts that as an impression. When an employee clicks on content, the system counts that as a clickthrough, download, or played \(depending on the content type\).

**Note:** The Rich Content Clickthrough feature counts any click within rich content areas, including links, but does not distinguish between link clicks and other clicks. For example, clicking a link or an image in the rich content area both count as clicks, but the feature does not specify which was clicked.

-   Banner clickthrough
-   Banner impression
-   Calendar download
-   Calendar impression
-   Email clickthrough
-   Event download
-   Event impression
-   Image-based link clickthrough
-   Image-based link impression
-   Link clickthrough
-   News article tab clickthrough
-   News article clickthrough
-   News article impression
-   Rich content tab clickthrough
-   Rich content clickthrough
-   Rich content impression
-   To-do completion
-   To-do impression
-   Video impression
-   Video played
