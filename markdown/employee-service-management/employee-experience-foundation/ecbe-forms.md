# Browser Extension for Employee Center forms

The Browser Extension for Employee Center forms provide information about various configurations required to set up the extension.

-   **[Website configuration form](../reference/config-website-form.md)**  
The field descriptions provide a point of reference when you're configuring the Browser Extension for Employee Center for a website.
-   **[Browser Extension policy form](../reference/ecbe-policy-form.md)**  
The Policy form provides the field descriptions to set up the Browser Extension policies on the supported browsers.
