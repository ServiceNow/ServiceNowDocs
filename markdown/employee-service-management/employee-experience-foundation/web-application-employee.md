# Access applications from App Launcher

Access all your applications from a central location in one-click.

As an employee, you can access the custom and third-party app integrations. Navigate to the applications from the associated topics in the Employee Center Pro from **My applications**. Use the **Favorites**, **Categories**, and **Recently opened** options for better usability.

\[Omitted image "ec-app-launcher.png"\] Alt text: Application launcher categories and topics

-   View and launch all apps that you have permissions to.
-   Quickly go to the applications organized into the **Featured**, **Favorites**, **Categories**, and **Recently opened** options.
-   Access the topic associated applications from a topic page.
-   View applications that are tagged under a category.
-   Select applications as favorites.
-   Navigate to **View all** for a full list of both SSO and non-SSO applications.

For more information, see [App Launcher](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/yokohama/markdown/yokohama/employee-service-management/employee-experience-foundation/web-application-employee-about.md)
