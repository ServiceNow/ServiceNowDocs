# Customize UI actions for Universal Request integration with Microsoft Teams

Customize the UI actions and extend them to the tables that you require. The `UniversalRequestMSTeamsUtils` script include controls the visibility of the UI actions.

The following UI actions are available by default. You can customise them to suit your needs.

-   Start Microsoft Teams Chat
-   Import Messages from Microsoft Teams

To customise the UI actions, you must modify the form fields. For a detailed understanding of the form fields, see [UI actions](https://www.servicenow.com/docs/access?context=t_EditingAUIAction&version=yokohama&pubname=yokohama-platform-administration&ft:locale=en-US)
