# Customize the Employee Center with widgets

You can configure the widget instance options to control how data is displayed within the widget.

Widgets are small cards that display dynamic content from ServiceNow® applications and plug-ins on the Employee Center portal. Employees can use widgets to view company announcements, complete to-dos, access SSO-integrated apps, or navigate to popular topics without leaving the portal. Portal admins can customize the widgets, rearranging their size and order and modifying their display.

To view how the widgets appear on the Employee Center portal, see [Employee Center Pro features and interface](emp-center-vs-ec-pro.md).

The Employee Center features several widgets that are pre-installed and ready to use. Other widgets are installed using plugins such as Content Publishing. For a comprehensive list of all the available widgets, see [Employee Center widgets](employee-center-widgets-list.md).

Use the following links to learn about the features that are delivered via widgets.

-   **[Recommended for you](rfy-ec.md)**  
Recommended for you displays catalog items and knowledge articles, enabling you to view information that is most relevant to you.
-   **[Quick links](emp-center-quick-link-config.md)**  
Quick links are a way to quickly access internal and external resources related to specific departments within your organization. You can create quick links, associate quick links to topics, and restrict access to quick links.
-   **[Popular topics widget](../task/config-popular-topics.md)**  
Popular topics widget lets you quickly view and access the most popular topics. You can configure the instance options to control the appearance of the widget, such as the title, number of topics, display size, and border color.
-   **[Modify the Quick links widget display](../task/config-quick-links-widget.md)**  
Design and manage the appearance of the widget, such as the title, display style, tile size, and content alignment.
-   **[Modify the my favorites widget display](../task/config-favorites-instance-options.md)**  
The **My Favorites** widget displays the favorited content for quick use. You can configure the instance options to control the appearance of the widget, such as the title and number of items to display you want to view.
-   **[Modify the employee profile widget display](../task/config-employee-profile-instance-options.md)**  
The employee profile widget displays the tabs and work-related information hierarchy. You can configure the instance options to control the number of tabs to display.
-   **[Modify the topic header widget display](../task/config-topic-widget.md)**  
The topic header widget displays the topic and its child topics, including all the connected content associated with the topics. You can configure the instance options to control the appearance of the topic header widget, such as the topic description and the topic banner and icon.
-   **[Modify the sub-topics widget display](../task/config-child-topic-widget.md)**  
The sub-topics widget displays all the child topics associated with the parent topic in your taxonomy. Click the child topic to open the page for the child topic. You can configure the instance options to show or hide the topic icons for the child topics in the widget.
-   **[Modify the org chart widget display](../task/config-orgchart-instanceoptions.md)**  
The org chart widget displays the employees and organization hierarchy. You can configure the instance options to control the number of employees.
-   **[Modify the topic content widget display](../task/config-topic-content-widget.md)**  
The topic content widget displays the knowledge articles and requests associated with the topic in your taxonomy. Click a knowledge article or a request to open it. You can configure the instance options to control the number of items to show per page and their sort order.
-   **[Modify banner width options](../task/change-default-width-of-employee-center-pro-home-page-banner.md)**  
Modify the width of the Employee Center Pro banner widget, on the home page, to suit your preference.
-   **[Use email notification layout](../task/activate-notification-template.md)**  
Use the email notification layout and template shipped with the Employee Experience Foundation \(com.snc.sn\_ex\_emp\_fd\) plugin to deliver notifications.
-   **[Modify the Upcoming shift widget display](../task/config-shift-instanceoptions.md)**  
The upcoming shift widget displays the shift information of the Employee Center Pro Kiosk user. You can configure the instance options based on your display preferences.
-   **[Modify the Time off widget display](../task/config-timeoff-instanceoptions.md)**  
The widget displays the time off balances. You can configure the instance options based on your display preferences.
-   **[Modify Send to self widget display](../task/config-sendtoself-instanceoptions.md)**  
The Send to self widget displays the options for sending useful information as an email or an SMS for the Employee Center Pro Kiosk user. You can configure the instance options based on your display preferences.
-   **[Modify the Browse help content widget display](../task/config-browse-help-topics-instanceoptions.md)**  
The widget displays the help content for the Employee Center Pro Kiosk user. You can configure the instance options based on your display preferences.
-   **[Modify the Support resources widget display](../task/config-support-resources-instanceoptions.md)**  
The widget displays the support resources for the Employee Center Pro Kiosk user. You can configure the instance options based on your display preferences.
-   **[Modify the Guided self-service widget display](../task/config-guided-help-gss-instance.md)**  
The widget displays the help content for the Employee Center Pro Kiosk user. You can configure the instance options based on your display preferences.
