# Activity Configuration form

Use the Activity Configuration form to do a basic configuration or an advanced configuration for an activity.

<table id="table_erv_qfv_nqb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Activity Name

</td><td>

Name of the activity.

</td></tr><tr><td>

Activity navigation

</td><td>

Internal or external resource you want to open when you click an activity. The choices are:-   **Internal**
-   **External**

</td></tr><tr><td>

Activity portal page

</td><td>

Internal portal page you want to open when you click an activity. This field appears only when **Internal** is selected from **Activity navigation** field.

</td></tr><tr><td>

External URL

</td><td>

URL of the external website you want to open when you click an activity. This field appears only when **External** is selected from **Activity navigation** field.

</td></tr><tr><td>

Application

</td><td>

Application that you want to configure the activity for.

</td></tr><tr><td>

Primary

</td><td>

Option to set the activity as primary. You can set only one activity as primary.

</td></tr><tr><td>

Active

</td><td>

Option to activate the activity for use.

</td></tr><tr><td>

Sort By

</td><td>

How you want to sort the activity. This field is set to **sys\_created\_on** by default. This field appears only for a basic configuration after you select the **Primary** option.

</td></tr><tr><td>

Sort Order

</td><td>

Order in which you want to sort the activity. The choices are:-   **Ascending**
-   **Descending**

This field is set to **Ascending** by default. This field appears only for a basic configuration after you select the **Primary** option.

</td></tr><tr><td>

Icon

</td><td>

Select a suitable icon for the Activity Name field from the available list.

</td></tr><tr><td>

Order

</td><td>

Order in which you want the activity to appear.

</td></tr><tr><td>

Advanced

</td><td>

Option to do an advanced configuration.

</td></tr><tr><td>

Summary view script

</td><td>

Script to read existing configurations and display the count of activities in the Summary view of the My Active Items widget. This field appears only when you select the **Advanced** option.

</td></tr><tr><td>

List view script

</td><td>

Script to read existing configurations and display tasks only for the primary activity. These tasks are shown in the List view of the My Active Items widget. This field appears only when you select the **Advanced** option.

</td></tr></tbody>
