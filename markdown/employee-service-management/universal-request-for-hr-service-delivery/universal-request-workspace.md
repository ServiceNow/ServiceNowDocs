# Universal Request in Agent Workspaces

Use Universal Request in Agent Workspaces to create, view, and work on the universal request.

Using Universal Request in the Agent Workspaces allows agents to:

-   Create universal request
-   Manage and triage universal request
-   Assign and route the universal requests to the appropriate department

Agents can view all universal requests in a list view based on the states, such as new, in progress, or closed along with the primary task association.
