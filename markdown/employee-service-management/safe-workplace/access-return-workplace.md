# Accessing the lifecycle events

Access the Employee Return to Workplace and Return to Workplace Location Readiness employee experiences from Lifecycle Events.

## Before you begin

Role required: sn\_hr\_le.activity\_writer

## Procedure

1.  Navigate to **All** &gt; **Lifecycle Events** &gt; **Administration** &gt; **Manage Lifecycle Events**.

2.  From the Manage Lifecycle Events list, select **Employee Return to Workplace** or **Return to Workplace Location Readiness**.
