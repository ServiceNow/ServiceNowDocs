# Delete a quick link created by an administrator

To delete an administrator created quick link in Journey designer, you must be an administrator.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Journey Designer** &gt; **Manage Journey Type Links**.

2.  Select the check box for the row that contains the link you want to delete.

3.  From the **Actions on selected row** list, select delete.

4.  Confirm that you want to delete the record.

    Deleting the record removes the quick link from all journeys of that **Journey Type**.
