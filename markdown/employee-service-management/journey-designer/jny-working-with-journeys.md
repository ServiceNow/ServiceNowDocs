# Managers, mentors, and employees work with journeys in the Employee Center

Managers, employees, and mentors access journeys from the Employee Center \(EC\).

From the Employee Center, users can create, modify, and complete journeys depending on their assigned permissions and roles.

-   Employees can: View their journey and complete assigned tasks.
-   Managers can: Create and modify journeys, assign mentors, modify user permission for a journey, and monitor journey analytics. Managers create and modify journeys, add additional mentors, add recommended learning, add quick links, and add personalized messages.
-   Mentors can: Help employees successfully complete a journey. Mentors with the assigned permission can modify journeys, add additional mentors, add recommended learning, add quick links, and add personalized messages.

**Note:**

-   Journeys are available from the Now Mobile app. See [Journeys on Now Mobile](journeys-mobile-now.md)
-   The total task count in the journey progress bar may be different from the **Journey overview** **All tasks** count. This total task count discrepancy is because the **Journey overview** count displays the task count for only those tasks the employee has access to.

Some features may require integration with a related application, see [Integrations for Journey designer app](jny-integrations-1.md).

-   **[Journey view for a manager](manager-journeys-view.md)**  
As a manager, here's how you can view all your journeys and your team's journeys in Journey designer on the Employee Center portal.
-   **[Create a journey](../task/jny-dsgnr-create-journey.md)**  
Users that are managers and have Modifier \[sn\_jny.writer\] permission create journeys for employees and assign mentors to help with the journey.
-   **[Create a task template](../task/jny-dsgnr-create-task-template.md)**  
Journey owners \(managers\) and mentors create templates of tasks, such as scheduling a meeting or taking a learning course, to assign to employees' journeys.
-   **[Add task template to journey](../task/jny-add-task-template-journey.md)**  
Use task templates to quickly add pre-configured tasks to an employee's journey, such as scheduling a meeting or taking a learning course.
-   **[Mentors can modify and manage journeys](../task/jny-dsgnr-mentor.md)**  
Mentors with the appropriate permission can modify journeys and add other mentors.
-   **[Add, edit, or delete journey quick links](../task/jny-dsgnr-mngr-mntr-quick-links.md)**  
Journey owners and mentors add, edit, and delete quick links in a journey.
-   **[Journey view for an employee](../task/view-journey-overview.md)**  
As an employee with journeys in Journey designer, view your journey overview and tasks associated with that journey in Employee Center.
-   **[Manage Journey designer task updates in bulk](jny-dsgnr-bulk-task-update.md)**  
Many journeys have similar task types for multiple users. You can update similar task types in bulk for journeys of the same type.
-   **[Send a personal message to journey users](../task/jny-dsgnr-personal-message.md)**  
Journey owners and mentors with the **Can add personalized message** permission can send a personal message to the employee assigned to a journey.
-   **[Display learning recommendations to the Manager creating a journey](jny-dsgnr-learning-recom-manager-ml.md)**  
Enable manager who is creating a journey for an employee, to add relevant learning recommendations for the employee in the Employee Center.
