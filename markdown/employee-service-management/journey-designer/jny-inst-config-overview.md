# Installation and configuration overview

The Journey designer app has multiple versions that have different installation instructions and configuration options.

-   **[Install Journey designer version 3](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/journey-designer/jny-install-version-3.md)**  
Install the Journey designer version 3 \[sn\_jny\] application to enable administrators and managers to create and manage various professional and personal events and journeys.
-   **[Install Journey designer version 2](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/journey-designer/jny-install-version-2.md)**  
Install the Journey designer version 2 \[sn\_jny\] application to enable administrators and managers to create and manage various professional and personal events and journeys.
-   **[Install Journey designer version 1](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/journey-designer/install-jny-dsgnr.md)**  
Install the Journey designer \[sn\_ja\] application to enable administrators and managers to create and manage various professional and personal events and journeys.
-   **[Add or repair demo data for applications and plugins](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/journey-designer/jny-dsgnr-install-repair-app-plugin.md)**  
If the demo data wasn’t added during the initial installation, administrators can add demo data to a plugin using **Repair** install.
-   **[Fix Restricted Caller Access \(RCA\) approval requests](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/journey-designer/jny-manual-rca-fix.md)**  
After you install or upgrade to the latest Journey designer application from ServiceNow Store, you might encounter RCA approval messages requesting for an update in the access request.
-   **[Configure Journey designer features](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/journey-designer/jny-dsgnr-configuration.md)**  
Administrators configure Journey designer for the needs of their organization.
