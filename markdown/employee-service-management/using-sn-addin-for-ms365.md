# Using ServiceNow Add-in for Microsoft 365

Use the ServiceNow Add-in for Microsoft 365 to request catalog items or create incidents and cases directly from your emails.

-   **[Get a custom add-in for Microsoft Outlook](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/custom-addin-outlook.md)**  
Get the custom add-in for your Microsoft Outlook client to use the features configured by your admin.
-   **[Raise a request using the ServiceNow Add-in for Microsoft 365](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/raise-request-addin-ms365.md)**  
Open a relevant catalog item or form to raise a request with pre-filled information from your email.
