# Create a question bank in Listening Posts

Create a repository of questions to use in a pulse survey.

## Before you begin

Role required: sn\_lp.creator

## Procedure

1.  Navigate to **Listening Posts** &gt; **Pulse Question bank**.

2.  Click **New**.

3.  On the form, fill in the fields:

    |Field|Description|
    |-----|-----------|
    |Name|Name for the question bank.|
    |Description|Description for the question bank.|

4.  In the **Questions** related list, click **New** to add questions to the question bank.

    For more information, see [Create a pulse question in Listening Posts](create-pulseq.md).
