# Create a pulse theme in Listening Posts

Theme is a group of pulse surveys with a common purpose. For example, onboarding an employee can be a theme, and Day 1, Week 1 can be a pulse survey.

## Before you begin

Role required: sn\_lp.creator

## Procedure

1.  Navigate to **Listening Posts** &gt; **Pulse Theme**.

2.  Click **New**.

3.  On the form, fill in the fields:

<table id="table_kt1_rmd_knb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Suitable name for the theme.

</td></tr><tr><td>

Type

</td><td>

Type of the theme: -   **Group pulse**: Audience, start date, and end date of the theme are fixed.
-   **Employee lifecycle**: Audience, start date, and end date of the theme vary based on criteria.

**Note:** Employee lifecycle is assigned to audience depending on the configured lifecycle event.

</td></tr><tr><td>

Description

</td><td>

Additional description for the theme.

</td></tr><tr><td>

Active

</td><td>

Option to indicate that the theme is active and available for use.

</td></tr><tr><td>

Filters

</td><td>

Filters in Pulse Dashboard that are used to refine employee responses.**Note:** The default filters are department, country, and title.

</td></tr></tbody>
</table>4.  Click **Submit**.


## What to do next

-   View the list of pulse surveys under a theme in the **Pulse Surveys** related list. To create a pulse survey under a theme, see [Create a pulse survey in Listening Posts](create-pulse-survey.md).
-   Add reports and specify order in which you want the reports associated to the theme to be displayed in Pulse Dashboard. To add a report under a theme, [Add a pulse report in Listening Posts](create-pulse-report.md).
-   View the list of response sharing rules in the **Response Sharing Rules** related list. To create a response sharing rule, see [Create a response sharing rule in Listening Posts](create-response-rule.md).

-   **[Add a pulse report in Listening Posts](create-pulse-report.md)**  
Capture more demographics and have relevant breakdowns by adding reports within a theme. Add a report by specifying an order in which the report must appear on the Pulse Dashboard.
-   **[Create a response sharing rule in Listening Posts](create-response-rule.md)**  
Create response sharing rules by defining groups with which survey responses are shared.
