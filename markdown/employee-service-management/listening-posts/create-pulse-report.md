# Add a pulse report in Listening Posts

Capture more demographics and have relevant breakdowns by adding reports within a theme. Add a report by specifying an order in which the report must appear on the Pulse Dashboard.

## Before you begin

Role required: sn\_lp.creator

## Procedure

1.  Navigate to **Listening Posts** &gt; **Pulse Theme**.

2.  In the **Reports** related list, click **New**.

3.  On the form, fill in the fields:

    |Field|Description|
    |-----|-----------|
    |Report|Report that must appear in Pulse Dashboard.|
    |Order|Order in which the report must appear in the Pulse Dashboard. A report with a lesser order appears first.|
    |Theme|Theme to which the report belongs.|

4.  Click **Submit**.
