# Create a response sharing rule in Listening Posts

Create response sharing rules by defining groups with which survey responses are shared.

## Before you begin

Role required: sn\_lp.creator

## Procedure

1.  Navigate to **Listening Posts** &gt; **Pulse Theme**.

2.  In the **Response Sharing Rules** related list, click **New**.

3.  On the form, fill in the fields:

<table id="table_pgp_pnd_knb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name for the response sharing rule.

</td></tr><tr><td>

Description

</td><td>

Description of the response sharing rule.

</td></tr><tr><td>

Demographic Condition

</td><td>

Only user responses that satisfy this condition are displayed to the viewer on the Pulse Dashboard.

</td></tr><tr><td>

User criteria

</td><td>

User groups with which you want to share the pulse survey responses.**Note:**

-   Only the **Users** and **Groups** fields are applicable in user criteria for response sharing rules in a pulse survey.
-   The **Advanced field** option isn’t applicable.
-   If the user criteria is empty, no notifications are sent.


</td></tr></tbody>
</table>4.  Click **Submit**.
