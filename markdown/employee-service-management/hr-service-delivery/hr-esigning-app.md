# E-signature applications of HR Service Delivery

E-signature applications of HR Service Delivery.

[HR e-signature](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/hr-service-delivery/hr-e-signature.md)

HR e-signature is a **non-embedded signature** capability for managed documents, and knowledge articles, attached to an HR task for fulfilment. It supports three signing types \(Acknowledgement, Credential, and Signature\) and two document types \(Managed Documents and Knowledge Articles\).

**Note:**

-   HR e-signature must be used for internal, non-legal acknowledgement use cases.
-   HR e-signature is a legacy product with no active development.

[HR document templates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/hr-service-delivery/c_HRDocumentTemplates.md)

HR document templates provides embedded signature and supports both internal and external e-signing use cases. It supports two document types, HTML and PDF.

**Note:** HR document templates is a legacy product and being prepared for deprecation. Use [Document Templates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/hr-service-delivery/document-templates-overview.md) instead.

[Document Templates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/hr-service-delivery/document-templates-overview.md)

Document Templates is an embedded signature capability for HTML and PDF document templates. It provides embedded signature, supports both internal and external e-signing use cases \(of legal types also\). It supports three signing types \(Signing using the AdobeSign application, Signing using the DocuSign application, and Signing using the ServiceNow application\) and supports two document types, HTML and PDF.
