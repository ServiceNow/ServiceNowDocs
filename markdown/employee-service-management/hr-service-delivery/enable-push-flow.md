# Enable push todos flow

Enable push todos flow to push existing todos to HR Service Delivery Integration with SuccessFactors.

## Before you begin

Role required: sn\_hr\_sf.admin

## Procedure

1.  Navigate to **All** &gt; **Flow Designer**.

2.  Open **Push Todos**.

3.  In the **Trigger** field, specify the time and interval at which you want to automatically run the scheduled flow.

4.  Click **Activate**.
