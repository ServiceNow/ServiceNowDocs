# Configure AWA chat groups for HR Service Delivery Agent Workspace

Adding your HR agents to chat groups to route chat requests to the correct agent.

## Before you begin

Role required: sn\_hr\_core.admin

## Procedure

1.  Navigate to **User Administration** &gt; **Groups**.

2.  Select a group.

    The base systems provides:

    |Groups|Groups|
    |------|------|
    |HR|HR Systems|
    |HR Activity Writers|HR Talent Management|
    |HR Admin|HR Tier 1|
    |HR Benefits|HR Tier 2|
    |HR Employee Relations|HR Tier 3|
    |HR Investigations|HR Tier 4|
    |HR Leadership|HR Total Rewards|
    |HR Onboarding|HR VIP Watchlist|
    |HR Payroll|HR Workforce Administration|
    |HR Recruiting|HRIT Operations|

3.  Select the **Group Members** tab.

4.  Select **Edit**.

5.  Add users from the **Collection** column to the **Group Members List**.

6.  Select **Save**.

7.  Select **Update**.
