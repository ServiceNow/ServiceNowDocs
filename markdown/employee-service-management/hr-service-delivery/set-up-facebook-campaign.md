# Set up Employee Campaigns for Workplace from Facebook

Complete the Employee Campaigns for Workplace from Facebook application set up by configuring the Workplace for Facebook spoke and activating a schedule job.

## Before you begin

-   Role required: sn\_fb\_wp\_spoke.workplace\_fb\_admin
-   Activate the Employee Campaigns for Workplace from Facebook application from ServiceNow Store. This application automatically activates the Content Delivery and Content Automation applications, and the Workplace for Facebook spoke.

## Procedure

1.  Set up the Workplace for Facebook spoke.

    The Workplace for Facebook spoke manages the setup and configurations related to credentials and linking of those credentials to the Workplace from Facebook instance. For more information, see Set up the Workplace for Facebook spoke.

2.  Activate the synchronize Facebook Workplace Groups scheduled job by setting the value to **True**.

    The schedule job synchronizes groups from Workplace from Facebook to ServiceNow instance.
