# Edit personal details

Review your personal details and edit them if necessary.

## Before you begin

Role required: sn\_asc.alumni

## Procedure

1.  Navigate to **All** &gt; **Alumni Service Center** &gt; **Profile**.

2.  Select Personal details.

3.  Edit your personal details in these fields.

    |Fields|Description|
    |------|-----------|
    |Preferred name|Your name.|
    |Preferred pronoun|The pronoun you prefer.|
    |Date of birth|Your date of birth.|
    |Personal mobile phone|Your personal mobile number.|
    |Home phone|Home phone number, if you have any.|
    |Address|Your residential address.|
    |City|The city you reside in.|
    |State|The state you belong.|
    |Postal code|Your postal code.|
    |Country|Your country.|

    Use the Public or Private option for each field to control whether other alumni can view these details.

4.  Select **Save**.
