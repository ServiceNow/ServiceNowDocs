# Sending an email notification after reassignment

After you reassign an agent to an HR case, you can send an email to notify the agent.

There are multiple ways to notify other agents that you have assigned them to an HR case.

![HR Agent Workspace assign an agent](../reference/images/agent-ws-hr-assign-agent.png)

1.  Select ![Search for Record icon](../image/magnifying-glass.png) next to the **Assigned to** field.
2.  Select an agent.
3.  Click the Open record icon \(![Open Record icon](../reference/images/icon-global-search.png)\) next to the agent's name.
4.  The agent's details are opened in a new tab.
5.  In the Email field, click the Email icon\(![Email icon](../reference/images/envelope-outline.png)\).
6.  Add a **Subject** line and compose the mail in the space provided.
7.  Click **Send**.

![HR Agent Workspace email](../reference/images/agent-ws-hr-email.png)

You can also create an email from the ![More icon](../reference/images/more-icon.png) More icon. For more information, see [Sending an email in HR Agent Workspace](../task/send-an-email-hr-agent-ws.md).
