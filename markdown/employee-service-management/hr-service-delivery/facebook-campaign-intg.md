# Employee Campaigns for Workplace from Facebook

The ServiceNow® Employee Campaigns for Workplace from Facebook application enables you to package your content into a campaign and publish the content to groups on Workplace from Facebook.

**Important:**

Employee Campaigns for Workplace from Facebook is now deprecated and no longer supported or available for new activation.

For deprecation details, see the [Deprecation Process \[KB0867184\]](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB0867184) article in the Now Support knowledge base.

ServiceNow Employee Campaigns is a multi-channel communications tool for Service Portal, mobile, and email. The Employee Campaigns for Workplace from Facebook application extends the capability of Employee Campaigns to publish content via post on Workplace from Facebook groups. The key features are:

-   New content category and content type to support posting rich content on Workplace from Facebook, which includes message, image, and URL.
-   Content published as a Workplace from Facebook post to the targeted groups. The targeted groups can be one or more groups including open and closed.

## Set up the application

First, you must activate the Employee Campaigns for Workplace from Facebook application from ServiceNow Store. This application automatically activates the Content Delivery and Content Automation applications, and the Workplace for Facebook spoke. Next, you must set up the Employee Campaigns for Workplace from Facebook application.

**Important:** The Employee Campaigns for Workplace from Facebook application can be activated only if you have the Employee Service Center license and have upgraded to the Paris Patch 1 release of Employee Campaigns.

For detailed instructions on how to use the Employee Campaigns for Workplace from Facebook application to publish content on Workplace from Facebook, refer to Use Employee Campaigns for Workplace from Facebook.

## Example of how content is published on Workplace from Facebook

Following is an example of an open enrollment campaign.

An HR administrator wants to run an outreach campaign for an upcoming open enrollment, and wants to reach out to employees through the Workplace from Facebook channel. The HR administrator creates Workplace from Facebook content with a message, an image, and a link to the Employee Benefits site, adds the Workplace from Facebook Content to the campaign bundle within a campaign, and publishes the campaign.

A schedule job runs and publishes content to groups on Workplace from Facebook. ServiceNow Spoke is the name by which the post is made on Workplace from Facebook groups.
