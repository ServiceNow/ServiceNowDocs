# HR Service Delivery Integration with Magnit

The ServiceNow® HR Service Delivery Integration with Magnit application provides a seamless experience to track and manage onboarding activities of contingent workers.

## Key features

-   Track and manage onboarding activities of contingent workers. Pull contingent workers from the Magnit application into a ServiceNow AI Platform instance and create onboarding tasks for contingent workers from ServiceNow AI Platform instance.
-   Create custom flows for various use cases. For example, trigger a life cycle event for creating HR cases and notifications in ServiceNow AI Platform instance to manage contingent workers onboarding.
-   Provide seamless onboarding experience for contingent workers.

## Activation information

First, activate the HR Service Delivery Integration with Magnit application from ServiceNow Store. Activating the application automatically activates the Magnit spoke, Enterprise Service Management Integrations Framework and Human Resources Scoped App: Lifecycle Events for Enterprise.

Next, [Set up Magnit spoke](https://www.servicenow.com/docs/access?context=magnit-spoke&version=xanadu&pubname=xanadu-integrate-applications&ft:locale=en-US).

## Request apps on the Store
