# Employee Journey Management

Employee Journey Management integrates HR Service Delivery features and applications to enable employees, managers, and the organization to initiate and manage journeys to drive employee engagement and productivity.

There are several applications, plugins, integrations, and features that are available for Employee Journey Management. Each application or feature may have requisites and prerequisites that must be met before applications or features works properly. Prior to installing and configuring any features, read the related documentation.

## Requisite plugins and applications

Requisite plugins and applications must be installed or activated so that related features and application work properly.

-   **[Content Publishing](../../employee-center/concept/ec-publish-content.md)**

    Content Publishing allows a multitude of contents to be created and published to various channels such as portal, email, mobile and more. Content Publishing allows you to communicate with your employees using different methods.

-   **[Employee Center](../../employee-center/concept/employee-center-landing-page.md)**

    The ServiceNow® Employee Center is a standard multi-department, dynamic portal for service delivery and employee engagement. The Employee Center portal also enables customers to easily scale their service solutions across departments, such as IT, HR, Workplace, Legal, and Procurement with digital workflows.

-   **Human Resources Scoped App: Core \[com.sn\_hr\_core\]**

    Provides case and knowledge management for HR Service Delivery. Standardizes the documentation, interaction, and fulfillment of employee inquires and requests while having visibility into the quantity and type of cases coming in.

-   **[Lifecycle Events Enterprise](../task/activate-hr-lifecycle-events-for-enterprise-1.md)**

    You can activate the Human Resources Scoped App: Lifecycle Events for Enterprise plugin \[com.sn\_hr\_lifecycle\_ent\] if you have the admin role. This plugin includes demo data and activates related plugins if they are not already active.


## Key applications

Read the installation and configuration documentation for key applications to identify any prerequisites.

-   **[Journey designer](jny-dsgnr-landing-page-1.md)**

    The ServiceNow® Journey designer application enables employees and managers in your organization to initiate and manage journeys to drive employee engagement and productivity.

-   **[Journey Accelerator](journey-accelerator-1.md)**

    Administrators can use the ServiceNow® Journey Accelerator application to create templates for plans and to-do tasks. Managers then use the templates to create and publish customized plans for key employee transitions. Managers, employees, and mentors can access the plans from the Employee Center \(EC\).


## Applications and feature integrations

Read the installation and configuration documentation for any application or integration that you intend to install. Some of the applications and integrations have detailed prerequisites that must be completed before the integration or application works properly.

-   **[Learning Posts](learning-post-app-1.md)**

    With the ServiceNow® Learning Posts application, create an omni-channel learning experience for your employees empowering them with the right learning at the right time of need.

-   **[Listening Posts](listening-posts-1.md)**

    The ServiceNow® Listening Posts application enables you to create simple pulse surveys to capture employee touch points, gain insights from employee feedback, and improve overall employee experience.

-   **[Machine Learning for HR Service Delivery](predictive-intelligence-for-hr-service-delivery-1.md)**

    Machine learning solutions for HR Service Delivery help in auto determining assignment groups and services for HR cases, auto creating cases from emails, and discovering knowledge articles that help in faster resolution of cases and tasks.

-   **[Manager Hub](manager-hub-overview.md)**

    The ServiceNow® Manager Hub increases managers self-service and proactive engagement with their team by providing insights and recommended actions for what's most urgent and important to drive team success. It enables managers to grow as leaders through curated and personalized resources.

-   **[Microsoft Exchange Online](ja-ms-exchange-online-setup-1.md)**

    **Note:** If an administrator intends to use the Microsoft Office 365 scheduling capabilities with Journey Accelerator, the administrator must install and configure the related Microsoft plugins and integrations before installing Journey Accelerator and Journey designer.

    By integrating the Journey Accelerator scheduling feature with the Microsoft Office 365 calendar function, users can schedule plans and to-do tasks.
