# Import users

Import users from consumer system of records into a provider instance.

## Before you begin

Role required: admin

**Note:** Set up the required ServiceNow integrations on your provider instance.

## Procedure

1.  Import users from consumer system of records \(by methods supported by ServiceNow\) to provider instance.

2.  Associate the correct consumer company to the users who are imported into provider instance.
