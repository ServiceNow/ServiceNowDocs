# Run a scheduled job

Manually run the schedule job to pull data from Oracle HCM to the ServiceNow platform.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Integrations framework** &gt; **Sources**.

2.  Open the Oracle HCM record.

3.  Click **Run job**.
