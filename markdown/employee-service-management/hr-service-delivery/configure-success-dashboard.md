# Configure HR Success Dashboard indicators

Set up the HR Success Dashboard so that HR leadership teams and process owners can start using the dashboard.

## Get started with HR Success Dashboard indicators

1.  [Install the HR Success Dashboard indicators](../task/install-success-dashboard.md)
2.  The dashboard tracks the Human Resources General knowledge base. To change the tracked knowledge base or add additional knowledge bases, follow the steps to: [Configure tracked Knowledge Base for the self-solved KPI](../task/configure-knowledge-sdb.md)
3.  If you have topics other than the base system topics, configure the Virtual Agent deflection nodes: [Configure the Virtual Agent topics](../task/instrument-virtual-agent-sdb.md)
4.  [Set fulfillment automation level of catalog item](https://www.servicenow.com/docs/access?context=set-fulfillment-automation-level-sdb&version=australia&pubname=australia-it-service-management&ft:locale=en-US)
5.  Configure the dashboard key performance indicators \(KPIs\), add contributing indicators, or update the More information card
    1.  [Configure Success Dashboard indicators KPIs](https://www.servicenow.com/docs/access?context=config-kpis-sdb&version=australia&pubname=australia-it-service-management&ft:locale=en-US).
    2.  [Add contributing indicators](https://www.servicenow.com/docs/access?context=add-contributing-indicators&version=australia&pubname=australia-it-service-management&ft:locale=en-US).
    3.  [Update more information cards](https://www.servicenow.com/docs/access?context=update-sidepanel-more&version=australia&pubname=australia-it-service-management&ft:locale=en-US).
6.  [Run Update formula script for HR Success Dashboard indicators](../task/run-the-update-formula-script.md)
7.  [Activate Performance Analytics jobs](../task/activae-pa-indicator-jobs-sdb.md)
