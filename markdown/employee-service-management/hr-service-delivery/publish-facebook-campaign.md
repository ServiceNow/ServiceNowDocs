# Customize the name and logo for Workplace from Facebook

Customize the name and logo by which you want to post content on Workplace from Facebook. By default, ServiceNow spoke is the name by which the post is made on Workplace from Facebook.

## Before you begin

Role required: sn\_fb\_wp\_spoke.workplace\_fb\_admin

## Procedure

1.  Log into the Workplace from Facebook instance.

2.  Select **Integrations** in the Admin panel.

    The Settings for ServiceNow spoke window appears.

3.  In the **Integrations Name** field, enter the name by which you want the post to appear on Workplace from Facebook.

4.  In **Integration Logo**, add the logo by which you want the post to appear on Workplace from Facebook.

5.  Save your changes.
