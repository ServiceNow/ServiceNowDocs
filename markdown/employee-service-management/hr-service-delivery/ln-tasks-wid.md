# Learning tasks widgets

The Learning tasks and Learning tasks – Completed \(pre-configured widgets\) help to display learning tasks in the To-dos page in Employee Center.

For more information on these learning widgets, navigate to **Employee Center** &gt; **To-dos Configuration**.
