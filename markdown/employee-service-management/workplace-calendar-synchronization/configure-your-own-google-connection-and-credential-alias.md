# Configure your own Google connection and credential alias

You can configure your own Google connection and credential alias if you do not want to use the default alias **Google\_calendar**.

After you authenticate Google with ServiceNow, instead of configuring the connection and credential alias using the Flow designer connection configuration, you can also create your own connection and credential.

If you want to use the default alias, refer to [Create a Connection and credential record for Google](../task/create-connection-configuration-with-google.md).

1.  [Setup OAuth connectivity with Google Calendar](../task/setup-oauth-connectivity-with-google.md)  
Create an application registry for Google Calendar with ServiceNow to synchronize reservations. Perform this app registration if you want to create your own connection and credential alias for Google Calendar.
2.  [Create your own connection and credential alias for Google](../task/create-own-connection-credential-alias-for-google.md)  
Instead of using the default **Google\_Calendar** alias, you can create your own connection and credential alias to use it with your Google Calendar provider.
