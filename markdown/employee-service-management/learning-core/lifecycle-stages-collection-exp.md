# Life-cycle stages of a content collection in Learning Core

The State transition of a collection guides you through the stages of its life cycle.

|Life-cycle stage status|Description|
|-----------------------|-----------|
|Draft|Default value for a new collection.|
|In Review|Collection that is pending review.|
|Scheduled for Publish|Collection that is scheduled for publish.|
|Published|Collection that is published.|
|Pending Retirement|Collection that is pending for retirement.|
|Retired|Collection that is retired.|
