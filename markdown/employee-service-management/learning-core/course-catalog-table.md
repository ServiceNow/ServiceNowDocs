# Course catalog form

Use the course catalog form to create a new learning course catalog.

|Field|Description|
|-----|-----------|
|Title|Name of the catalog.|
|Active|Option to indicate that the catalog is available for use.|
|Owner|The owner of the course catalog.|
|Description|Description of the catalog.|
|Publish flow|Flow for publishing the course. Use **Insta publish flow** for immediate publishing of course catalogs.|
|Retire flow|Flow for publishing the course. Use **Insta retire flow** for immediate retirement of course catalogs.|
|Management|
|Admins|Suitable administrator role for the course catalog.|
|Content properties|
|Extension duration \(days\)|Number of days until which any course within the catalog can be extended.|
|Pending retirement duration \(days\)|Number of days remaining for the course to retire.|
