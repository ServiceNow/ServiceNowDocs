# Create user achievements

Create user achievements to project the accomplishments of course takers on My Learning.

## Before you begin

Role required: sn\_lep.achievement\_manager

## Procedure

1.  Navigate to **All** &gt; **Achievements** &gt; **User Achievements**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |User|User whose achievements are projected.|
    |Achievement|Associated achievement definition.|
    |Status|Status of the achievement: **current**, **at risk**, or**expired**.|
    |Achieved on|Date on which the achievement is achieved.|
    |Expiry date|Date on which the achievement is due for expiry.|

4.  Select **Submit**.
