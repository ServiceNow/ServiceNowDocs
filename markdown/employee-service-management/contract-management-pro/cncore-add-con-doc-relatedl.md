# Add Contract documents tab to the contract repository record

Enable display of contract documents in a contract repository record by adding the Contract documents tab to it.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Tables**

2.  Select the table configured as a contract repository.

    For more information, see [Create a contract configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/contract-management-pro/cncore-contract-config.md).

3.  Right-click the context menu icon \[Omitted image "context-menu.png"\] Alt text: Context menu icon and select **Configure** &gt; **Related Lists**.

4.  In the **View** list, select your workspace.

5.  Add the Contract documents tab in contract repository records by moving **Contract documents** from the Available list to the Selected list.

    \[Omitted image "cmpro-enable-contract-doc.png"\] Alt text: Moving contract documents from Available list to Selected list.

6.  Select **Save**.

7.  Select **Update**.


## Result

The Contract document tab is available your workspace to view the contract documents and it's versions.

\[Omitted image "cmpro-bu-add-doc-rl-result.png"\] Alt text: The contract document related list is available in your record
