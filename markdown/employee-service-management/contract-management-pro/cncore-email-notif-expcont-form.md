# Expiring Contracts Condition form fields

The Expiring Contracts Condition form is used to create a rule or update an existing rule.

|Field|Description|
|-----|-----------|
|Name|Name of the condition.|
|Application|The application where the condition is created and used.|
|Compliant state|System field. Do not use.|
|Order|Order in which the conditions are evaluated.|
|Event name|Name of the event this condition triggers.|
|Expiration Condition|Condition that must be true for the **Condition check** field to be set to this value. Add as many conditions as are needed. For more information on condition check, see [Define a condition check](https://www.servicenow.com/docs/access?context=t_DefineAConditionCheck&version=xanadu&pubname=xanadu-it-asset-management&ft:locale=en-US).|
|Domain|Domain of the condition.|
|Condition check|Associated condition check for the expiration condition.|
|Table|Table associated with the condition check.|
