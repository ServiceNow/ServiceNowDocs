# Reclassify supporting documents in the contract request

As a contract fulfiller, reclassify the supporting documents submitted in the contract request. For example you can reclassify the supporting document as a contract document.

## Before you begin

Role required: sn\_cm\_core.contract\_fulfiller

## About this task

You can reclassify supporting documents only when the State is Work in progress.

## Procedure

1.  Navigate to your workspace and open the contract request for which you want to reclassify the documents.

2.  In the **Supporting Documents** tab, select **Reclassify**.

3.  In the **Select document** drop-down, select the document that you want to reclassify.

4.  In the **Reclassify to** drop-down list, select the new contract type.

    Only active contract types are displayed in the list.

5.  Add more information about the document in the **Work notes** field.

6.  Select **Reclassify** to update the classification.

7.  Select **Save** to save the record.
