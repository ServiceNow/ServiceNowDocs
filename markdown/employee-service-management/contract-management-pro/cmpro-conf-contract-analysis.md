# Configuring contract analysis

Configure use cases with associated field groups and fields, and map them to clauses and expected responses. Now Assist uses the applicable use case to analyze a contract document and identify non-standard and missing clauses.

-   **[Create use cases for contract analysis](../task/cmpro-na-usecase-ca.md)**  
Create a use case for contract analysis that is used by Now Assist to identify non-standard and missing clauses in a contract document.
-   **[Map a field group to a clause](../task/cmpro-na-map-fieldgrp-clause.md)**  
Map field groups of a use case to clause variations of a clause library. Now Assist uses the mapped clause content to display suggestions for a missing or non-standard clause.
-   **[Map a field to an expected response](../task/cmpro-na-exp-res-mapping.md)**  
Map a field of a use case to an expected response. Now Assist executes the fields \(prompt questions\) on a contract document to identify a non-standard clause.
-   **[Map a use case for contract analysis](../task/cmpro-na-usecase-mappings-ca.md)**  
Map a use case to specific tables, and define conditions to apply the use case for contract analysis.
