# Configure Contract Management Pro

Install Contract Management Pro, set up the requirements for managing contract requests.

-   **[Install Contract Management Pro](../task/cncore-install-cmpro.md)**  
Install the Contract Management Pro application. The application installs related ServiceNow® Store applications and plugins if they aren’t already installed.
-   **[Configure the Microsoft Word add-in for ServiceNow Contracts](../task/cncore-config-word-addin.md)**  
As an admin, configure the Microsoft Word add-in for ServiceNow Contracts.
-   **[Set up Contracts Core](cncore-setup-cmpro.md)**  
As a contract configurator, create and manage a library of clauses, clause variations and contract templates that will be used to generate contract document when a contract request is submitted.
-   **[Configure contract templates for a contract request](cncore-document-templates.md)**  
Create contract templates to easily generate Microsoft Word contract documents based on request types.
-   **[Configure document templates for standard letters or documents](../task/cncore-wdt-other-bu.md)**  
Create Word Document Templates to create standard letters or documents that can be signed by configured participants.
-   **[Add and configure contract request functionality](cncore-uptake-steps.md)**  
As contract administrator, add Contract Management Pro functionality application.
-   **[Configuring Contract Workspace with UI Builder components](cncore-conf-cntrct-wrkspc.md)**  
Configure and customize Contract Workspace by using the UI Builder web user interface builder.
-   **[Configure Obligation Management notifications](../task/cncore-config-ob-mgmt-notf.md)**  
Activate Obligation Management notifications to notify users when an obligation task is created, assigned, submitted, approved, or rejected.
