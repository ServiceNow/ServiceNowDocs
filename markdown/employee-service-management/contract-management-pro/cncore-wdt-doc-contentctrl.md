# Content controls in Word Document Templates

Word Document Templates supports content controls that act as placeholder for the content.

**Note:** You must add content controls in the Microsoft Word document before you [Configure document templates for standard letters or documents](../task/cncore-wdt-other-bu.md). Content controls must be added using a Windows system only.

The following types of content controls are supported.

|Content control type|Description|
|--------------------|-----------|
|Field\_|This content control is used to indicate meta data.|
|Signature\_|This content control is a placeholder for electronic signature.|
|Sign\_Date\_|This content control is used to indicate the date when the signer signs the document electronically.|

While creating content control in the word document template, you can prefix the tag with the appropriate content control type. For example, if you want to include company name, you can specify as Field\_CompanyName. The Locking settings can be set to prevent the content control from being deleted and edited.

For more information on how to work with content controls, refer to the [Microsoft support](https://support.microsoft.com/en-us) site.
