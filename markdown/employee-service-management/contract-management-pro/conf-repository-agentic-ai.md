# Activate business rules for the Manage contract repository agentic workflow

Activate the Agentic AI - Set reminders for contract and the Agentic AI Set reminders for Econtract business rules to autonomously run the Manage contract repository agentic workflow.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Business Rules**.

    The Business Rules table appears.

2.  In the Name column, search for `Agentic AI - Set reminders for contract`.

3.  Open the **Agentic AI - Set reminders for contract** rule.

4.  Select the **Active** check box.

5.  Select **Update**.

    The Business Rules table appears.

6.  In the Name column, search for `Agentic AI Set reminders for Econtract`.

7.  Open the **Agentic AI Set reminders for Econtract** rule.

8.  Select the **Active** check box.

9.  Select **Update**.


## Result

The Manage contract repository agentic workflow is activated for Now Assist in Contract Management.
