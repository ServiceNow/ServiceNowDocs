# Create document template categories

As a contract configurator, create document template categories to easily identify similar documents and find the correct document template more easily. For example, a document template category of NDA-Japan could group non-disclosure agreements for Japan.

## Before you begin

Role required: sn\_cm\_core.contract\_config

## Procedure

1.  Navigate to **All** &gt; **Document Templates** &gt; **Document Template Categories**.

2.  In the **Name** field, enter the name for the category.

3.  Select **Active** to make the category active.

4.  Select **Submit** to save the category.
