# UFX Add on Event mapping form

The UFX Add on Event mapping form to configure the display of declarative action in your application.

|Field|Description|
|-----|-----------|
|Event mapping name|Name for your map. For example, Initiate contract|
|Active|Option to activate the mapping.|
|Description|Description for the action that's displayed in the Action Assignments list.|
|Source component|The component where the button will appear. For example, the Action bar.|
|Controller|A controller that handles the target event. For example, Form.|
