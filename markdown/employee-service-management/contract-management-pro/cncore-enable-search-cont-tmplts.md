# Enable keyword search for contract templates

Enable keyword search in contract templates.

## Before you begin

Role required: sn\_cm\_core.contract\_config

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Text Index Configuration**.

2.  From the list, select the record with the table value **Word Template \[sn\_doc\_word\_template\]**.

3.  Select **Generate Text Index** from the Related Links

4.  Select **Ok** in the Generate Text Index dialog box.


## Result

The table will be indexed. For more information on how to search for a keyword in a contract template, see [Search for keywords in contract templates](cncore-search-in-doct.md)
