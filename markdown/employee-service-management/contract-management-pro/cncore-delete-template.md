# Delete a contract template

Delete contract template when you no longer need it.

## Before you begin

Role required: sn\_cm\_core.contract\_config

## Procedure

1.  Navigate to **All** &gt; **Contracts Core** &gt; **Contract Administration** &gt; **Contract Templates**.

2.  Open a contract template from the list.

3.  Select **Delete**.

4.  In the Confirmation pop-up, select **Delete**.
