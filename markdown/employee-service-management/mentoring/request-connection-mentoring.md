# Request a connection

Request a connection with a mentor who seems like the right fit for you.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **Your career** &gt; **Mentoring** &gt; **Overview**.

2.  On a mentor's profile, in the Suggested mentors section, select **Connect**.

    **Note:** You see suggested mentors only after setting your preferences.

3.  From the Mentoring duration list, select the duration.

4.  In the **Message** field, add a request message.

5.  Select **Send**.
