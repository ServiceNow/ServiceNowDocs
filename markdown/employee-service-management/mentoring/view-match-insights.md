# View match insights

View the Match Insights score that is calculated based on the match score of the preferences.

## Before you begin

Enroll as a mentee. See [Enroll as a mentor or mentee](enroll-mentoring.md).

Add mentee preferences. See [Add mentee preferences](add-preferences-mentee.md).

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **Your career** &gt; **Mentoring** &gt; **Overview**.

2.  In the **Suggested mentors**, click the hamburger menu on a suggested mentor.

3.  Select **View match insights** and view the following match information about the suggested mentor or mentee.

    -   Focus areas
    -   Skills
    -   Language
4.  Select **Close** to close the match insights card view.
