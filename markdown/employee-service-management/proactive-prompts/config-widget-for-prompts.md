# Add and configure the prompt widget

Add and configure the prompt widget to access Proactive Prompts on a portal.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Pages**.

2.  Search and select a page to add the prompt widget.

    You can also create a new page and then add the widget. For more information, see [Create and edit a page using the Service Portal Designer](https://www.servicenow.com/docs/access?context=t_ConfigureAPage&version=xanadu&pubname=xanadu-platform-user-interface&ft:locale=en-US) and [Service Portal widgets](https://www.servicenow.com/docs/access?context=service-portal-widgets&version=xanadu&pubname=xanadu-platform-user-interface&ft:locale=en-US).

3.  Select the **Open in Designer** related link.

    The Service Portal Designer is displayed.

4.  In the Widgets tab, search for prompts and drag the prompts widget on to the Edit area.

5.  Select the ![Pencil icon](../../../reuse/icons/product-icons/pencil-outline-24.svg) edit icon on the top-right corner of the widget.

6.  Edit the form.

    |Field|Description|
    |-----|-----------|
    |Prompt categories|Categories to pull and add category-specific prompts on the prompt widget. For information on signal categories, see [Configure signal categories in Proactive Prompts](proactive-prompts-signal-catergories.md).|
    |Persona|Persona to filter and display the prompts on the widget. The options are, Manager, or Employee.|
    |Title|Title to be displayed on the widget.|
    |Show Menu|Displays the Manage Preferences menu on the widget that enables you to turn on or off the display of the prompts individually.|
    |Max prompt count|The maximum number of prompts displayed on the widget.|
    |Prompts per row|The number of prompts displayed at a time on the widget. The next set of prompts can be accessed with a slider on the widget.|

7.  Select **Save**.
