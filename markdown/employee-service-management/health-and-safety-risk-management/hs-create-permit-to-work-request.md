# Create a permit-to-work request

The permit-to-work request is submitted by the worker who carries out the job with significant risk.

## Before you begin

Role required: sn\_hs\_rm.permit\_requestor

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **Health and Safety** and select **Permit to work**.

2.  Select the **Request a permit to work** card or select the quick link to request a permit-to-work.

3.  On the form, fill in the fields.

    For a description of the field values, see [Permit-to-work form](../reference/hs-permit-to-work-form.md).

    **Note:** Additional fields must be filled on the permit to work form depending on the type of the permit selected.

4.  Select acknowledgment and then select **Submit**.

    The permit request is created where you can view the permit details.

5.  Select **My requests** to open the permit.

6.  Select the checklist tab of the permit request and complete the checklist as applicable to the task.

    The checklists are also displayed under the **Open** tab of **My tasks.**

7.  Select **Save** to save changes.

    If necessary, the checklist can be edited.

8.  Select **Mark Complete** when the checklist items are completed.

    **Note:** The checklist can’t be edited after it’s marked complete.

9.  From **Actions**:

    -   Select **Print permit** to print the permit request.
    -   Select **Print checklist** to print the completed checklist.
    **Note:** The printed permit and checklist can be signed and attached in the attachment tab for this permit. The attached documents can be viewed in the activity stream.
