# Issue a permit to work

A permit coordinator can issue a permit after the permit approver approves a request.

## Before you begin

Role required: sn\_hs\_rm.permit\_coordinator

## Procedure

1.  Navigate to **All** &gt; **Health and Safety** &gt; **Health and Safety Workspace**.

2.  Select the risk management icon \(![Risk assessment icon](../image/icon-risk-assessment.png)\).

3.  Open the **Permit to work** list and select **All** and then select a permit to review.

    The permit coordinator can:

    -   Create a Health and Safety action.
    -   Review the approvers
    -   Review the checklist
4.  Select **Issue permit** if all safety requirements are met.

    The permit status changes to active and the permit requestor can print the permit through the employee center.

5.  Select **Close permit** when the job is complete.

    If necessary, the permit coordinator can perform further actions after closing the permit:

    -   Select **Cancel permit** in the UI action menu, if there are unsafe conditions.
    -   Select **Suspend permit** in the UI action menu, if a danger is expected after the permit is issued. The permit can be suspended until the inspection is completed.
