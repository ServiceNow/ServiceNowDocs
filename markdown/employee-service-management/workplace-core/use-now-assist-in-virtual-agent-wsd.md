# Using Now Assist in Virtual Agent topics for Workplace Service Delivery

The Now Assist in Virtual Agent assists Workplace Service Delivery hybrid employees to book meeting spaces more conveniently and quickly.

Now Assist in Virtual Agent provides workplace requesters to have a streamlined, conversational experience based on generative AI. Workplace employees can submit a workplace reservation request to reserve a space and add services to a reservation. Now Assist in Virtual Agent enhances the user experience by combining AI Search with generative AI chat skills.

The generative AI capabilities can provide a human-like conversational experience for Workplace Reservation Management request submissions.

The Now Assist in Virtual Agent retrieves details from Workplace Reservation Management and provides workplace users with the following:

-   Reserving a space for meeting
-   Adding co-workers to a reservation
-   Adding visitors or guests to a reservation
-   Adding a Zoom meeting link for remote joining
-   Ordering and adding catering services
