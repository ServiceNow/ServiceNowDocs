# Create workplace contracts in Core UI

As a lease administrator or manager, you can create workplace contracts and also add assets, terms and conditions, contract locations, and documents to a contract.

## Before you begin

Role required: sn\_wsd\_la.manager or sn\_wsd\_la.admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Lease Administration** &gt; **Workplace Contracts**.

2.  Select **New**.

3.  In the **Contract Model** field, select **Lease**.

4.  In the **Terms and conditions** section, specify the legal information in the contract.

    For more information, see [Add and build contract terms and conditions in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/lease-add-terms-and-conditions.md)

5.  On the form, fill in the rest of the fields.

    For description of the field values, see [Workplace contract new record form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/workplace-contract-form.md).

6.  Right-click the form header and select **Save**.

7.  Add contract services, contract locations, assets, or documents to the contract.

    For more information, see the following topics:

    -   [Add a workplace contract service to a contract in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-lease-contract-service.md)
    -   [Add a workplace contract location in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-lease-contract-location.md)
    -   [Add assets to a workplace contract in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-an-asset-to-a-lease-contract.md)
    -   [Add a document to a workplace contract in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-document-to-workplace-contract.md)
8.  Save and exit the contract or submit it for review.

    -   To save and exit the contract, select **Update**.

    -   To send a notification to the approver, select **Submit for Review**.

        For information about the approval process, see [Approve or reject a workplace contract in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/approve-lease-contract.md).


-   **[Add a workplace contract service to a contract in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-lease-contract-service.md)**  
Workplace contract services are the common services offered as part of the lease contract. You can add and track the services along with their associated costs.
-   **[Add a workplace contract location in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-lease-contract-location.md)**  
Add a location that is covered by the contract to a workplace contract.
-   **[Add assets to a workplace contract in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-an-asset-to-a-lease-contract.md)**  
Add assets to a workplace contract, including software licenses, to specify what the contract legally covers.
-   **[Add workplace contract clauses in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-workplace-contract-clauses.md)**  
Add workplace contract clauses, including insurance clause, parking clause, warranty clauses to store the lease contract clauses.
-   **[Add workplace contract options in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-workplace-contract-options.md)**  
Add workplace contract options such as option to renew, option to sub lease, option to terminate to store the lease contract options.
-   **[Add workplace contract responsibilities in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-workplace-contract-responsibilities.md)**  
Add workplace responsibilities to indicate who \(landlord or tenant\) is responsible for different activities such as lawn maintenance, painting, repairing.
-   **[Add and build contract terms and conditions in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/lease-add-terms-and-conditions.md)**  
Add and build terms and condition when a contract is in the Awaiting Review, Rejected, or Renewal Rejected substate.
-   **[Add a document to a workplace contract in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-lease-administration/add-document-to-workplace-contract.md)**  
Add documents to the workplace contract to maintain a record of all relevant documents and different versions of those documents.
