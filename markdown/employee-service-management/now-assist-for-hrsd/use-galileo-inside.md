# Use Now Assist for HR - Galileo Inside to answer HR-related questions

Query the Galileo library to generate a response based on articles and research from The Josh Bersin Company.

## Before you begin

Role required: sn\_hr\_na\_galileo.read

## Procedure

1.  Open a new chat in the Now Assist panel.

2.  Enter your question containing any of the following keywords to trigger a response from Galileo Inside:

    -   Josh Bersin
    -   Galileo
    -   HR Practices
    -   HR Policy
    -   Ask Bersin
    -   HR Trends
    -   HR Guidelines
    -   HR Framework
    -   HR Standards
    If you do not include one of the keywords, Now Assist provides two responses: the first response is generated by the Now LLM and the second is retrieved from Galileo Inside and includes source citations with links to the original article.
