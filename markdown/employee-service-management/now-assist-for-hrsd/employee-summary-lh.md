# View employee summary reports

View the Employee Summary Virtual Agent topic in Employee Center to access employee contribution summaries, view consolidated profiles, and track employee progress.

## Before you begin

You must have one of the following plugins to view this topic:

-   Manager Hub \(sn\_mh\)
-   Leader Hub \(sn\_egd\_lh\)

Role required: sn\_egd\_lh.leader or sn\_mh.manager\_hub\_user

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

2.  Select the Open chat window icon \(\[Omitted image "chat\_icon.png"\] Alt text: Open chat window\).

3.  Enter `Employee Summary`.

4.  Select the employee that you want to view the summary for.

    **Note:** You can view summaries only for your direct and indirect reports.

5.  Select a start and end date from the calendar.

    \[Omitted image "emp-sum-3.png"\] Alt text: Start and end date calendar

    **Note:** The duration between the start and end dates must not exceed one year.

6.  Select **Make changes** if you want to make changes, otherwise select **Looks good**.


## Result

The Employee summary report is available to review. You can enter follow-up questions, or close the chat.
