# Add Employee Center to the Now Assist in Virtual Agent display experience

Enable Now Assist in Virtual Agent to appear in the Employee Center so you can converse with AI agents during a process that's facilitated by an agentic workflow.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Conversational Interfaces** &gt; **Assistants**.

    The Assistants page appears.

2.  Select **Now Assist in Virtual Agent \(default\)**.

3.  On the navigation panel, select **Display experience**.

    The Choose how to display page appears.

4.  In the Portals widget, select **Employee Center** from the **Add portal** drop-down list.

    The Add chat experience dialog box appears.

5.  Select the **Allow the search bar to open into a full-page chat experience** check box.

6.  Select **Add**.

7.  Select **Save and continue**.
