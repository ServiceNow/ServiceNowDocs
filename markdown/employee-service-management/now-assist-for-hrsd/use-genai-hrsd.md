# Submit an HR request with Gen AI Virtual Agent

Place an HR request using the conversational and streamlined experience offered by Gen AI Virtual Agent.

## Before you begin

Role required: None

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

2.  Click **Open Chat**.

    Gen AI Virtual Agent appears.

3.  Enter your query and select the required catalog item.

    For certain catalog items, KB articles are provided for assistance, for some catalog items you will be directed to a different page to place the request, and for certain catalog items Gen AI Virtual Agent creates an HR case for you.
