# Use the Now Assist panel in Agent Workspace for HR Case Management

Generate chat and case summaries and resolution notes by using the Now Assist panel in Agent Workspace for HR Case Management or Core UI.

## Before you begin

Role required:

-   sn\_hr\_core.case\_writer: Agent Workspace for HR Case Management
-   sn\_hr\_core.basic: Core UI

## About this task

## Procedure

1.  Open an HR case that is assigned to you.

2.  Select the Now Assist icon ![Now Assist icon.](../../../common/image/icon-ai-sparkle.png).

3.  Select a skill from the menu:

    -   To generate a chat summary, select **Chat summarization**. For more information, see To generate a chat summary.
    -   To generate resolution notes, select **Generate resolution notes**. For more information, see To generate resolution notes.
    -   To generate a case summary, select **Summarize a record**. For more information, see To generate a case summary.
    ![Now Assist panel menu in Workspace.](../image/now-assist-icon.png)
