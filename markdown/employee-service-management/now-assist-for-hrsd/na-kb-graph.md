# Use Knowledge Graph in Now Assist for HRSD

Create a personalized knowledge model as a Knowledge Graph schema.

## Before you begin

An employee uses Now Assist in Virtual Agent to request an employee visa. The Virtual Agent uses the assigned Knowledge Graph schema to resolve the query and create an Employee Travel Visa Request. Here’s an example of how a Knowledge Graph schema assists a Now Assist in Virtual Agent user. Here’s an example of how a Knowledge Graph schema assists a Now Assist in Virtual Agent user.

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

2.  Select **Open chat window**.

3.  Query the following: `Need to raise a travel visa request`.

4.  Select **Start** below **Employee Self Travel Visa Request**.

5.  The query is processed by Knowledge Graph and generates the following fields:

    -   Employee name
    -   Country of birth
    -   Country traveling to
    -   Travel start date
    -   Trip duration
    -   Business justification
6.  Fill in the required fields.

    Based on the Knowledge Graph schema selected, the Knowledge Graph searches all edges and nodes to provide available outputs such as **Employee name** and **Country of birth.**

7.  Edit the provided information before confirmation if needed.

    When confirmed, the request is processed with minimal effort from the user.


## Result

Knowledge Graph leverages the existing information that is available in the internal databases and auto-populates it to reduce efforts while making the entire experience seamless.
