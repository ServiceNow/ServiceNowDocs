# Summarize an ER case interview using Now Assist for HRSD

Use the ER interview summarization skill to obtain a comprehensive overview of the ER interview, which includes key details such as quick recap, key assertions and responses.

## Before you begin

Role required: sn\_hr\_er.case\_writer

## About this task

You can use the ER interview summarization skill in either Core UI or Agent Workspace for HR Case Management.

-   In Core UI, the summary appears in a banner in the Interview record.
-   In Agent Workspace for HR Case Management, the summary is generated in the **Interview** tab.

**Note:** For information on how to configure the ER interview summarization skill, see [Configure Now Assist for HR Service Delivery \(HRSD\)](configure-now-assist-hr.md).

## Procedure

1.  Navigate to **HR Agent Workspace**.

2.  In Workspace, open an ER case that is assigned to you.

    Alternatively, you can navigate to **All** &gt; **HR Cases** and select an ER case.

3.  Open the **Interview** related list.

4.  Select **Summarize**.

5.  **Note:** Generating and displaying the summary may take several seconds.

6.  Review the summary details.

    A summary of the interview record is displayed with a quick recap, key assertions, and responses.

    ![ER interview summary in HR Agent Workspace](../image/int-summarize-aws.png "ER interview summary in HR Agent Workspace") ![ER interview summary in HR Agent Workspace](../image/int-summarize-aws-output.png "ER interview summary in HR Agent Workspace")

    ![ER interview summary in Core UI](../image/int-summarize-coreui.png "ER interview summary in Core UI") ![ER interview summary in Core UI](../image/int-summarize-coreui-output.png "ER interview summary in Core UI")

7.  When you finish summarizing the interview record, you can add it to the interview notes, expand or collapse it, provide feedback, copy it, or view information about it.
