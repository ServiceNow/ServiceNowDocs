# Print workplace cases

Workplace agents can print the workplace case at any time on the go. The case reports contain reservation details, services, and all other relevant information.

## Before you begin

Verify that you have installed the required plugins.

Role required: sn\_wsd\_case.workplace\_agent

## Procedure

1.  Log in to Workplace Agent for mobile application.

    The launch screen opens with your **Work view**.

2.  On the **Work view** page, go to any one of the following sections to open the case to which you want to add a child task.

    -   My work overview
    -   My work
    -   Group work
    -   Unassigned work
3.  Select the case.

    The case details page of the selected case opens separately.

4.  Select the \[Omitted image "wsdagentmob-actions.png"\] Alt text: Menu icon on the top right of the header.

5.  Select **Print case**.

    A confirmation message appears.

6.  Select **Yes**.

    The downloaded document displays the workplace case details.
