# Print workplace tasks

Workplace agents can print the workplace task details related to a workplace case. The task reports contain reservation details, services, and all other relevant information.

## Before you begin

Verify that you have installed the required plugins.

Role required: sn\_wsd\_case.workplace\_agent

## Procedure

1.  Log in to Workplace Agent for mobile application.

    The launch screen opens with your **Work view**.

2.  On the **Work view** page, go to any one of the following sections to view the cases and tasks.

    -   My work overview
    -   My work
    -   Group work
    -   Unassigned work
3.  If you know the parent case of the task, select the case.

    The case details page opens separately.

4.  Select the task.

    The task details page opens separately.

5.  Select **Print task**.

    A confirmation message appears.

6.  Select **Yes**.

    The downloaded document displays the workplace task details.
