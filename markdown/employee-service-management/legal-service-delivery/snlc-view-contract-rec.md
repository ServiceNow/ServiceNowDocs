# View a contract repository record

View the history of a legal contract to track changes, review the audit trail, and update fields. Easily see the date a contract was modified and who modified it.

## Before you begin

Role required: sn\_lg\_cnt.contract\_fulfiller

## Procedure

1.  Navigate to **All** &gt; **Legal Counsel Center**.

2.  Select the List icon \(\[Omitted image "lsd-lcc-list-icon.png"\] Alt text: List icon\).

3.  View all contracts by navigating to **Legal Contracts** &gt; **All Contracts**.

4.  View expired contracts or contracts that will expire soon.

    -   To view contracts that will expire within 30 days, select **Expiring contracts**.
    -   To view expired contracts, select **Expired contracts**.
5.  View a previous version of the contract.

    1.  Select the contract from the list.

    2.  In the **Contract History** related list, select a date in the **Contract Starts** column to view the earlier version.
