# Raise an employee parking request

Submit a request for an employee parking space through either the Workplace Service Portal or the Employee Center. You can request short-term or long-term parking spaces, visitor or guest parking spaces, parking spaces for an electric car, or other parking issues.

## Before you begin

Role required: sn\_wsd\_core.workplace\_user

## About this task

Use the **Employee Parking** request for visitor a parking space, electric parking space, or any other parking issues at your workplace.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

2.  On the Employee Center home page, select the **Workplace Services** &gt; **Site safety**.

3.  Select **Employee Parking** request.

4.  On the Employee Parking form, fill in the fields.

    For a description of the field values, see [Employee Parking form](../reference/employee-parking-form.md).

5.  Select **Add attachments** to attach any required files.

6.  Select **Submit**.


## Result

The Employee Parking request is submitted.

The state of the request changes to Awaiting approval for approval by the respective manager. After the request is approved, the state of the request changes to Work in progress. Depending on the progress of the background tasks that are assigned to fulfill the request, the state of the request changes further.
