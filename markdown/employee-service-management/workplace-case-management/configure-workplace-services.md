# Configure workplace services topic

Configure catalog items for the child topics under the Workplace service topic within the Workplace Service Delivery integration with Microsoft Places application.

## Before you begin

Role required: sn\_wsd\_case.admin or sn\_wsd\_case.manager

## Procedure

1.  Navigate to **All** &gt; **Content Taxonomy** &gt; **Topics**.

2.  Search for the Workplace services topic.

3.  Select the Workplace service topic that you want to modify to add more catalog items to the Workplace Service Delivery integration with Microsoft Places application.

4.  Select the topic or category for which you want to add the new catalog item.

5.  On the form, scroll down and click **New** under the **Connected Content** tab.

6.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Content type|Type of the taxonomy content configuration for which you want to add the catalog item.|
    |Topic|The name of the topic. This field is preloaded with the selected topic name.|
    |Catalog item|The catalog item that you want to add to the topic. Ensure to select the item that is related to the topic.|

7.  Click **Submit**.


## Result

The Workplace service catalog item is added. The catalog item will be applied to the specific topic within the Workplace Service Delivery integration with Microsoft Places application.
