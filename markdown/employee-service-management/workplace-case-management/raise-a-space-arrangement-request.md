# Raise a space arrangement request

Submit a space arrangement request to plan and configure office spaces for meetings, events, or collaborative work. Select the existing reservation and specify the space layout details.

## Before you begin

Role required: sn\_wsd\_core.workplace\_user

## About this task

Use the **Space arrangement** workplace service to request a specific room configuration for meetings and events. You can also specify the required space arrangement such as courtyard, rooftop, if the space you want to reserve is not available from the list of options.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

2.  On the Employee Center home page, select the **Workplace Services** &gt; **Maintenance**.

3.  Select **Space arrangement** request.

4.  On the form, fill in the fields.

6.  Select the **Space layout** for the space arrangement.

7.  Select the required **Layout Option** from the available list.

    The banner displays the space arrangement pattern based on the selected layout option.

8.  Enter the number of chairs required for the event or meeting.

9.  Enter any additional information or instructions for the space arrangement.

10. Select **Add attachments** to attach any required files.

11. Select **Submit**.


## Result

The **Space arrangement** request is submitted.

The state of the request changes to Awaiting approval for approval by the respective manager. After the request is approved, the state of the request changes to Work in progress. Depending on the progress of the background tasks that are assigned to fulfill the request, the state of the request further changes.
