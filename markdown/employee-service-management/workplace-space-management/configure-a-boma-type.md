# Configure a BOMA type

Define a BOMA type using which you can classify a space based on its BOMA standard.

## Before you begin

Ensure that Workplace Space Management application is installed.

Role required: sn\_wsd\_core.admin and sn\_wsd\_spcmgmt.manager

## About this task

Classify a space based on Building Owners and Managers Association \(BOMA\) standards. Based on the BOMA type of a space, the organization derives what type of a space it is, plans real-estate, calculates chargebacks, performs vendor management and more. Similar to Workplace Core Space type configuration, additionally, you can categorize a space with respect to the BOMA standard and map both.

## Procedure

1.  Navigate to **All** &gt; **Workplace Core** &gt; **Administration** &gt; **BOMA type configuration**.

2.  Click **New**.

3.  On the BOMA type form, enter a **Name** for the BOMA type.

4.  Select **Active** to activate the BOMA type.

5.  Click **Submit**.


## Result

The BOMA type is configured. To map the BOMA type with a Space type, refer to [Map a space type with BOMA type](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-space-management/map-a-space-type-with-boma-type.md).
