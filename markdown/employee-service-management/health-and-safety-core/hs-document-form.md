# Health and Safety document version form

Safety team such as safety agent and contractor coordinator can use the Health and Safety document version form to add or modify a safety-related document.

For more information, see [Add a new Health and Safety related document](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/health-and-safety-core/add-hs-related-document.md).

<table id="table_xgm_2zy_bdc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Document

</td><td>

Name of the document this version is associated with.This field is auto-populated with the document name this version is created from.

</td></tr><tr><td>

Version state

</td><td>

State of the document version.Only a version in the Published state is available for use in Health and Safety records.

</td></tr><tr><td>

File type

</td><td>

File type of the document.-   **Attachment**: A document uploaded and attached in this record.
-   **Hyperlink**: A link for the document stored at another location.
-   **Knowledge article**: A knowledge article stored at another location.

</td></tr><tr><td>

URL

</td><td>

Web link of the document location.This field appears only when **Hyperlink** is selected from **File type**.

</td></tr><tr><td>

Attachments

</td><td>

Document attached in this record. Select **Select file** to upload the document version.This field appears only when **Attachment** is selected from **File type**.

</td></tr><tr><td>

Knowledge article

</td><td>

Knowledge article record.This field appears only when **Knowledge article** is selected from **File type**.

</td></tr><tr><td>

Expiry date

</td><td>

Date when the document expires.

</td></tr><tr><td>

Notes

</td><td>

Information related to the document version.
