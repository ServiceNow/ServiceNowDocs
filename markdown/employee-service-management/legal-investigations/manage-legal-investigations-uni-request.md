# Triaging universal requests for legal investigation complaints

When employees observe and report an ethics violation or similar incident, the resulting universal requests are triaged for prioritization and resolution.

Due to the sensitivity of such cases, you can view and work on these universal requests for complaints based on the roles you've been assigned.

-   **Universal Request routing agent**

    With the sn\_uni\_req.sensitiveinfo\_agent role for handling sensitive requests, you can review and triage these universal requests.

-   **Legal fulfiller**

    With the sn\_lg\_investigate.fulfiller role, you can create an ethics legal matter from the universal request.

-   **HR agent**

    With the sn\_hr\_er.case\_writer role, you can create a Report Misconduct HR case from the universal request.
