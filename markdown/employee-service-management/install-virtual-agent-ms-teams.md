# Access ServiceNow for Microsoft Teams app

Enables users to interact with the agents from Microsoft Teams application.

## Before you begin

Role required: admin

## About this task

To receive notifications from Virtual Agent, users must link their accounts to Microsoft Teams. Guest users can't receive notifications.

## Procedure

1.  Launch Microsoft Teams application.

2.  Click **Apps**.

3.  In the apps search field, enter `ServiceNow for Microsoft Teams`.

4.  Click **ServiceNow for Microsoft Teams** app.

5.  Click **Open**.


## Result

The **ServiceNow for Microsoft Teams** app is successfully installed in your Microsoft Teams application.
