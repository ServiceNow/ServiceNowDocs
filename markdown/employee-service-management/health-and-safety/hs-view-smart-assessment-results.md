# View the smart assessment result using the Mobile agent app

You can view the smart assessment that was completed as part of the safety inspection or audit.

## Before you begin

Role required: sn\_hs\_rm.safety\_inspection\_writer

## Procedure

1.  Tap the Home icon \(![home icon](../../../reuse/images/home-page-icon.png)\).

2.  Select **See all** for the **Inspections** or **Audits** section.

3.  Tap the inspection or audit record.

4.  Tap the **Smart assessment results** button.

5.  Select **Next** to view all the sections of the assessment.

6.  Select **Close**.
