# Working with Skill feedback in Talent Feedback

Skill feedback is one of the main features of Talent Feedback that can be used by managers to request feedback for their employees, and by employees to provide feedback for their peers.

-   [Request skill feedback](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/talent-feedback/request-skill-feedback.md)

    As a manager, request skill feedback from an employee.

-   [View feedback requests](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/talent-feedback/view-feedback-request.md)

    As a manager, view and manage all the feedback requests raised for your employees.

-   [View skill feedback submitted](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/talent-feedback/view-skill-feedback.md)

    As an employee, view and submit feedback requested from you.


-   **[Request skill feedback](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/talent-feedback/request-skill-feedback.md)**  
Using Talent Feedback you can request feedback for one or more skills from your employees.
-   **[View and submit feedback](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/talent-feedback/view-skill-feedback-requested.md)**  
You can view any feedback request sent to you and submit the feedback on Talent Feedback.
-   **[Decline a feedback request](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/talent-feedback/accept-deny-feedback.md)**  
You can decline a feedback request sent you if you do not wish to provide feedback for any reason.
-   **[View skill feedback submitted](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/talent-feedback/view-skill-feedback.md)**  
As a manager, view all the skill feedback submitted by employees.
-   **[View feedback requests](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/employee-service-management/talent-feedback/view-feedback-request.md)**  
You can view and feedback requests platform under the Talent Feedback module.
