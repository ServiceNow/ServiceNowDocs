# Add Talent Development Aspirations Types

As an Talent admin you can add or manage **Aspiration Types**, which are used to identify employee aspirations.

## Before you begin

Role required: sn\_egd\_core.admin or admin

## Procedure

1.  Navigate to **All** &gt; **Talent Development** &gt; **Aspirations** &gt; **Aspiration Types**.

2.  Select **New**.

3.  Complete the **Aspiration Type New record** form.

    To see details about the different fields, see [Talent Aspiration types form](../reference/egd-aspiration-type-form.md).

4.  Select **Submit**
