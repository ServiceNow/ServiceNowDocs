# Talent Activity Category reference

Administrators create activity categories that are used by managers and employees.

|Field|Description|
|-----|-----------|
|**Active**|Select **Active** to make the activity category available for use in Growth plans.|
|**Description**|The description displayed in the user interface \(UI\) for the activity category.|
|**Label**|The UI visible text for the category.|
|**Name**|The name of the activity category.|
|**Order**|A numeric value used to determine the display order of activity categories.|
|**Value**|The value used to track the activity category.|
