# Add Talent Development Interests

As a Talent Development Core admin, you can add **Interests** that help employees understand and create an aspiration.

## Before you begin

Role required: sn\_egd\_core.admin or admin

## Procedure

1.  Navigate to **All** &gt; **Talent Development Core** &gt; **Aspirations** &gt; **Interests**.

2.  Select **New**.

3.  Complete the **Interests** form.

    To see details about the different fields, see [Talent Interest form](../reference/egd-interests-form.md).

4.  Select **Submit**.
