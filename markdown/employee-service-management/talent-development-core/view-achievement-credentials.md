# View credentials in Achievements and Credentials

View all type of user credentials in Achievements and Credentials.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; ****Achievements and Credentials**** &gt; **Achievement Credentials**.

2.  On the achievement credentials page, view the details stored in the following format.

    **Note:** You can select the user, credential template, state, and URL options to view more information about these fields.

    |Field|Description|
    |-----|-----------|
    |User|Name of the user.|
    |Credential template|Achievement credential template associated with the user.|
    |State|Current state of the credential \(whether it was accepted or rejected\).|
    |Public|Visibility of the credential.|
    |Expiry date|Date and time on which the credential has expired \(if expired\).|
    |URL|URL of the source of the credential. For example: Credly account URL.|
    |External ID|External ID corresponding t o the user.|
    |Issued date|Date and time when the credential was issued.|
    |Created|Date and time when the credential was created.|
    |Created by|Information about the role that created the credential.|
    |Domain|Domain information of the credential|
    |Tags|Option to add tags to a credential.|
    |Updated|Date and time when the credential was last updated.|
    |Updated by|Information about the role that made the last update to the credential.|
    |Updates|Information about the most recent updates made to the credential.|
