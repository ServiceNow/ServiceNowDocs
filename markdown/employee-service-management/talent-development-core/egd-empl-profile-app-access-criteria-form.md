# Employee Application Access Criteria reference

Talent Development uses the **Employee licensing** feature in the Employee Profile app to license users.

|Field|Description|
|-----|-----------|
|**Application**|The scope for creating the access criteria.|
|**Criteria**|The criteria that are evaluated to license users for the application. Users must meet all the conditions in order to be licensed.|
|**Employee Licensed Application**|The application that the licensing is for.|
|**Name**|A descriptive name for the access criteria.|
