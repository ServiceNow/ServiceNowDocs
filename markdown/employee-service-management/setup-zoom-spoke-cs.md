# Set up Zoom spoke

Integrate your Zoom account with your  ServiceNow® instance. Create a custom OAuth application in Zoom and authenticate requests from  ServiceNow.

For complete information about setting up a Zoom spoke, see [Zoom spoke](https://www.servicenow.com/docs/access?context=zoom-spoke&version=xanadu&pubname=xanadu-integrate-applications&ft:locale=en-US).
