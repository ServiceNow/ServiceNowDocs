# Export a quiz scorecard as an image

You can export scorecards as images.

## Before you begin

Role required: assessment\_admin or admin

## Procedure

1.  Click the menu icon and select **Save as PNG** or **Save as JPEG** as the download format.

2.  When the export is complete, select **Download** to save the scorecard image to a storage location.
