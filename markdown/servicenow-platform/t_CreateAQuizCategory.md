# Create quiz categories

Create quiz categories, which represent a theme for evaluating a specific element of the quiz topic and contain questions pertaining to that theme.

## Before you begin

Role required: assessment\_admin or admin

## About this task

When you create a quiz, the system creates a default category using the name of the quiz. You can use this category, modify it, or create additional categories as needed. To have any results, a category must contain scored questions.

## Procedure

1.  Navigate either **Quizzes** &gt; **Quiz Designer** or **Quizzes** &gt; **Quizzes** and click **Quiz Designer** in the list header.

2.  Enter the name of the quiz in the **Name** field.

    The system uses this name as the name of the quiz and the first category.

3.  Configure the category.

    1.  Click the gear icon in its title bar.

    2.  In the **Properties** dialog box, set or modify the properties.

        You can change the name of the category, add a description for it, and enter text in the **Details** field that introduces or explains the category to the recipients. The system updates the category as you type.

    3.  Click the X icon to close the category properties dialog box and save your settings.

    \[Omitted image "QuizDesignerCatProp.png"\] Alt text: Quiz designer category properties

4.  Add a new category by clicking the **+** icon in the title bar of an existing category.
