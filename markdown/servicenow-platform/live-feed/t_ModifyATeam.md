# Modify a team

If you are team administrator, you can modify an existing team.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Live Feed** &gt; **Live Feed**.

2.  In the sidebar, click your profile picture, then select the **Teams** tab.

3.  From the team list, select the team name.

4.  Click the edit icon \(\[Omitted image "GearIcon.png"\] Alt text: Edit icon\).

5.  Modify the team properties as needed.

6.  Click **Save**.
