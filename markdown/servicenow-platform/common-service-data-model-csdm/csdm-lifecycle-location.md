# Life cycle of location entities

The values for the location life-cycle process reflect the locations used by your organization and are visible only in the common data locations table.

## Life-cycle values for location entities

For definitions of the values, see [Definitions of life-cycle values for location entities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/common-service-data-model-csdm/csdm-lifecycle-df-location.md).

\[Omitted image "csdm-lifecycle-vp-location.png"\] Alt text: Relationships between CSDM stages and life cycle values.

**Note:** The \[life\_cycle\_control\] table uses the type of CI \(tangible/physical, document and contract, location and so on\) to determine which *life cycle stage status* values are available for each *life cycle stage*.

For additional information on how you can benefit from implementing life-cycle value pairs for CMDB entities, see the ['Map existing status values to CSDM life-cycle value pairs' section in the 'Foundation domain' topic](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/common-service-data-model-csdm/foundation-domain.md).
