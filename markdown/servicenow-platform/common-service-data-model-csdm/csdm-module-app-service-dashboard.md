# Monitoring and managing Configuring service instance settings

The Application Services dashboard enables you to monitor and manage service instances \(service instances were formerly called application services\) to ensure that they are fully configured and are populated in the CMDB.

## Working in the dashboard

A report on the dashboard, for example, can direct you to a service instance that is not configured with a service population method so you can repair it. For information on working in the dashboard, see [Use Application Services dashboard to monitor health](../../configuration-management/task/app-service-dashboard.md).
