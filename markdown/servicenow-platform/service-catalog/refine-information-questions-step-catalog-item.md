# Refine content in the Questions step of a catalog item

You can refine the generated responses of a catalog item. Refine content, such elaborate or shorten it, in the Instructions and the Conversational fields in the Questions step.

## Before you begin

Role required: catalog\_builder\_editor and now.assist.creator

## Procedure

1.  Navigate to **All** &gt; **Service Catalog** &gt; **Catalog Builder**.

2.  Open the catalog item for which you want to refine the content.

3.  Open the Questions step.

4.  If you want to edit content in the **Instructions** field, select the Annotation tab, or if you want to edit content in the **Conversational label** field, select the Question tab.

5.  Select the corresponding refine \(![The refine icon](../image/refine-icon.png)\) icon for these fields.

6.  Select any of these options:

    -   Elaborate: To regenerate the existing content in more detail.
    -   Shorten: To shorten the existing content.
7.  Select **Insert**.
