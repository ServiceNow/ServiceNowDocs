# IP Address

Use the IP address variable to specify the IPv4 and IPv6 data for a catalog item.

Under the **Validation Scripts** submodule, a validation script type associated with this variable is available.

**Note:** When you set the default value for the **IP Address** variable, or specify its value in the g\_form.getValue\(\) and g\_form.setValue\(\) functions, the delimiter is \(.\).
