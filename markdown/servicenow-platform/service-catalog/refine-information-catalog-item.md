# Refine content in the Details step of a catalog item

You can refine the generated responses of a catalog item. Regenerate, elaborate, or shorten the short description and description in the Details step.

## Before you begin

Role required: catalog\_builder\_editor and now.assist.creator

## Procedure

1.  Navigate to **All** &gt; **Service Catalog** &gt; **Catalog Builder**.

2.  Open the catalog item for which you want to refine the content.

3.  Open the Details step.

4.  If you want to refine the content in the Details step, select the refine \(![The refine icon](../image/refine-icon.png)\) icon for **Short description** or the **Description** field.

5.  Select any of these options:

    -   Elaborate: To regenerate the existing content in more detail.
    -   Generate: To regenerate the content based on the other content available in the Details step.
    -   Shorten: To shorten the existing content.
6.  Select **Insert**.
