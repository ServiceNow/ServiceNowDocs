# Creating a catalog item using Now Assist

Create a catalog item by using Now Assist and make the item available in Service Catalog for catalog item requests. Easily refine the short description and description.

## Before you begin

Role required: catalog\_builder\_editor and now.assist.creator

## Procedure

1.  Navigate to **All** &gt; **Service Catalog** &gt; **Catalog Builder**.

2.  Select the **Dashboard** tab and select **Create a new catalog item**.

3.  Review the steps involved in the creation process and select **Continue**.

4.  Select a template from the list and select **Use this item template**.

5.  In the Now Assist step, describe the catalog item you want.

    For more information about describing your catalog item, see [How to describe a catalog item](../reference/how-to-describe-catalog-item.md).

6.  Select **Generate with Now Assist**.

    Now Assist generates content for the Details and Questions steps of the catalog item.

7.  Verify the content in the Details and Questions steps before you submit the item.

    **Note:**

    -   The AI sparkle \(![The AI Sparkle icon](../../../common/image/icon-ai-sparkle.png)\) icon displayed for these steps and their fields denotes that the content in the fields is AI-generated. If you change the content, the sparkle is removed.
    -   You can refine the content in the **Short description** and **Description** fields of the Details step. You can also refine the content in the **Instructions** and the **Conversational label** fields in the Questions step.
8.  After verifying and providing all the information required for your catalog item, submit the form.


-   **[Refine content in the Details step of a catalog item](refine-information-catalog-item.md)**  
You can refine the generated responses of a catalog item. Regenerate, elaborate, or shorten the short description and description in the Details step.
-   **[Refine content in the Questions step of a catalog item](refine-information-questions-step-catalog-item.md)**  
You can refine the generated responses of a catalog item. Refine content, such elaborate or shorten it, in the Instructions and the Conversational fields in the Questions step.
