# Multi Provider Document Services Framework

Enable your team to collaborate in real time and add versions, store, copy, delete, restore, and upload documents by integrating document providers with your ServiceNow instance.

-   **[Activate Multi Provider Document Services Framework](../task/activate-multi-provider-doc-framework.md)**  
You can activate the Multi Provider Document Services Framework plugin \(com.snc.multiprovider\_documents\) for Document Services if you have the admin role.
-   **[Installed with Multi Provider Document Services Framework](../reference/installed-with-multi-provider-framework.md)**  
Several types of components are installed with activation of the Multi Provider Document Services Framework plugin, including tables and user roles.
-   **[Microsoft OneDrive integration with Document Services](one-drive-integration.md)**  
Use Microsoft OneDrive to copy, delete, restore, and get version actions on documents directly in your ServiceNow instance.
-   **[Google Drive integration with Document Services](google-drive-integration.md)**  
Use Google Drive to manage and sync your documents, and get version actions on documents directly in your ServiceNow instance. You can also collaborate on documents and manage attachments hosted within Google Drive.

Create provider details for multi provider documents.

Create document actions for multi provider documents.
-   **[Create provider permissions](../task/create-provider-permissions-mp.md)**  
Create provider permissions for multi provider documents.
-   **[Configure permissions](../task/configure-permissions-mp.md)**  
Configure permission for roles and groups for multi provider documents.
