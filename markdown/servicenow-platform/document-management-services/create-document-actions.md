# Create document actions

Create document actions for multi provider documents.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Multi Provider Document** &gt; **Admin** &gt; **Document Actions**.

2.  Select **New**.

3.  Enter the Document Action Name.

4.  Select Provider permission, Subflow, and Provider Details.

5.  In the Description field, enter the description.

6.  Select **Submit**.
