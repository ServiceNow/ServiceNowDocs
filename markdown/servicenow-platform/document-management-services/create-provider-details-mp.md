# Create provider details

Create provider details for multi provider documents.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Multi Provider Document** &gt; **Admin** &gt; **Permission Configurations**.

2.  Select **New**.

3.  In the Attributes field, enter Name and Value.

4.  In the Provider details name field, enter the name.

5.  Select the Provider Name from the drop-down.

6.  Select **Submit**.

    **Note:** Once the provider details are created, it automatically creates the document actions and provider permissions.
