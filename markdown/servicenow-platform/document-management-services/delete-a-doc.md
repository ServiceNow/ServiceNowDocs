# Delete documents

Delete documents in the document repository.

## Before you begin

Role required: admin with admin access

or

Access permission: owner

or

User: owner

## Procedure

1.  Navigate to **All** &gt; **Documents** &gt; **Documents**.

2.  Either delete a single document or multiple documents.

    -   To delete a single document, open the document record and select **Delete**.
    -   To delete multiple documents, select the document record entries, and from the Actions on selected rows drop-down menu, select **Delete**.
