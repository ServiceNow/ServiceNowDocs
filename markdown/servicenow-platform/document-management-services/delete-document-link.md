# Delete a document link

Delete a shared link to remove existing user access to the document.

## Before you begin

Role required: owner or writer

## Procedure

1.  Navigate to **All** &gt; **Documents** &gt; **Documents**.

2.  Select the record that you want to share the link for.

3.  In the record, select **Share Document**.

4.  Select the delete icon \(\[Omitted image "delete-icon-link.png"\] Alt text: Delete icon.\) to delete the shared link.

5.  In the dialog box, select **Delete** to confirm.
