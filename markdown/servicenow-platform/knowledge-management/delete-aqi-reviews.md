# Delete AQI reviews

Delete an AQI review that was created by mistake or is no longer required.

## Before you begin

Roles required:

-   To delete an AQI in a submitted state: admin or knowledge admin
-   To delete an AQI not in a submitted state: knowledge admin or the reviewer who initiated the AQI

## Procedure

1.  Navigate to **All** &gt; **Knowledge** &gt; **Articles** &gt; **My Pending AQI Checklists**.

2.  Select all checklist reviews you would like to delete.

3.  Click **Delete**.
