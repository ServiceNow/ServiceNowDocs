# Define a datasource

The datasource is an entity that has associated data that you intend to display. For example, a community post or a user.

## Before you begin

Role required: admin or evam\_admin

## Procedure

1.  Navigate to **All** &gt; **Entity View Action Mapper \(EVAM\)** &gt; **Datasources**, and then click **New**.

2.  On the form, fill in the fields.
