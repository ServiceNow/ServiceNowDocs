# Save or load a Dependency Views map

In the **View Map** module, use the menu icon to save and load Dependency Views maps.

## Before you begin

Role required: None

## Procedure

1.  Navigate to **All** &gt; **Dependency Views** &gt; **View Map**.

2.  Click the View menu icon \(\[Omitted image "MenuIcon.png"\] Alt text: View menu\).

3.  Select **Save View**, **Load View**, or **Last View**.
