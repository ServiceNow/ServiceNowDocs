# Display the help tips on the forms

Display the help tips on the forms by updating the preferences. Help tips provide extra details about the fields and their related actions.

## Before you begin

Role required: sn\_oper\_res.manager

## Procedure

1.  Navigate to **** &gt; **Preferences** in the user profile.

    The user profile is shown in the example.

    ![User profile.](../image/user-profile-pref.png)

2.  Select **Display**.

3.  Enable the **Show help tips on forms** option.

    The option is displayed in the example.

    ![Show the option.](../image/show-help-tips-option.png)

    The help tips are displayed on the form as shown in the example.

    ![Help tip.](../image/help-tip-on-the-forms.png)
