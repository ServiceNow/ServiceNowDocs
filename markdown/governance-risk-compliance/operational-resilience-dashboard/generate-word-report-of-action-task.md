# Install Document designer with Word

Install the ServiceNow Document designer with Word \(sn\_grc\_doc\_design\) application. It extracts metadata \(including fields, related lists, and reference fields with their associated elements\) from ServiceNow® tables and enables the insertion of repeating content blocks \(for example, 10 blocks for 10 issues\) based on record count. When applied to specific records, the template generates a Microsoft Word document.

## Before you begin

Role required: sn\_oper\_res.admin

## Procedure

1.  Navigate to **System Applications** &gt; **All Available Applications** &gt; **All**.

2.  Find the ServiceNow Document designer with Word application using the filter criteria and search bar.

3.  Select **Install**.

    The ServiceNow Document designer with Word application is installed in your ServiceNow® instance.
