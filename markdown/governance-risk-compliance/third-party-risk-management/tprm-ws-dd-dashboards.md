# TPRM Dashboards page

The dashboards page displays the Third-party insights dashboard and TPRM custom analytics dashboard as well as any other dashboards you and your team create.

## Accessing the dashboards

You can view the Dashboards page by navigating to **Workspaces** &gt; **Vendor Management Workspace** and selecting the dashboard page icon ![](../image/icon-tprm-ws-dashboard.png). From the dashboards page you can view two pre-existing dashboards, with the option to create more.

![Dashboards page. For the text description, refer to the text that precedes and follows this example.](../image/tprm-ws-dashboard-page.png "Viewing dashboards")

Select the down arrow next to the name of the dashboard to open the drop-down menu and view all of your accessible dashboards. For more information on TPRM dashboards, see [Monitoring assessment data using TPRM dashboards](../../grc-vendor-risk/concept/tprm-monitor-dashboards.md).
