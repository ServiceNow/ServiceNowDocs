# TPRM issue summarization skill

The issue summarization skill in Now Assist for Third-party Risk Management \(TPRM\) uses generative AI to create concise summaries of vendor-related issues, helping assessors quickly review risk details, improve consistency, and accelerate remediation.

## Overview of an AI-generated issue summary

If you have the Third-party assessment reviewer \[sn\_vdr\_risk\_asmt.vendor\_assessment\_reviewer\] role, you can create a summary for an issue. You can learn the details of a third-party risk issue from the summary generated by the TPRM issue summarization skill. By reading the issue summary, you can get an overview of vendor-related issues that are often high in volume. By default, issues in all states are eligible for summarization.

To learn more about how you can summarize an issue, see [Generate a summary of a TPRM issue](../task/create-a-summary-of-issue.md).

## Benefits of summarizing an issue

-   Provides a concise view of vendor risk details by extracting key information from complex records.
-   Helps reduce the effort required to review full issue documentation while retaining essential context.
-   Promotes uniform remediation guidance and aids in prioritizing multiple risk items.

## Components of an issue summary

After generating a summary for an issue, the summary displays the following sections:

-   Issue summary
-   Resolution
-   Related tasks
-   Key takeaways

The following table provides more information on the sections associated with an issue summary:

|Section|Description|
|-------|-----------|
|Issue summary|A concise overview of the issue, including context, impact, and vendor details.|
|Resolution|Steps taken or planned to remediate the issue and resolve associated risks.|
|Related tasks|Linked tasks or actions required to complete remediation or follow-up.|
|Key takeaways|Critical insights, lessons learned, or important notes for future reference.|

The following fields are used to generate issue summaries:

-   Short description
-   Description
-   State
-   Priority
-   Assigned to
-   Recommendation
-   Explanation
-   Applies to
-   Third party
-   Engagement
-   Assessment
-   Related tasks

**Note:** Summaries are generated based on prompts configured for the issue summarization skill. For more information, see [Activate the TPRM issue summarization skill](../task/tprm-summarize-an-issue.md).
