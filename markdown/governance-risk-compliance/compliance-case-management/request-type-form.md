# Request type form

Use the Request Type form in the Compliance Case Management application to categorize the compliance requests by their type.

The following table lists the descriptions of the field values.

|Field|Description|
|-----|-----------|
|Label|Label to be displayed for the request type.|
|Name|Name of the request type. For example, Policy update/enquiry.|
|Category|Category of the request type. This field is automatically set to **Compliance case \[sn\_comp\_case\_compliance\_request\]**.|
|Active|Option to select the request type as active or inactive.|
|Description|Description about this request type. This description is for the administrator to understand the request type.|
|State Model|
|State Model|Workflow states for the request type. For more information, see [Create a state model to define the workflow states and transitions](../task/create-state-model.md).|
|Template Configuration|
|Document templates|Template to generate Portable Document Format \(PDF\) reports for compliance requests associated with this request type.|
