# Review and close an action task

Review and close an action task after an action task owner provides responses and observations for the case in the GRC: Compliance Case Management application.

## Before you begin

Role required: sn\_compliance\_ws.corporate\_compliance\_analyst

## Procedure

1.  Navigate to **All** &gt; **Policy and Compliance** &gt; **Compliance Workspace**.

2.  In the Tasks module, open the task to review.

3.  Review the Action task and then select one of the following options.
