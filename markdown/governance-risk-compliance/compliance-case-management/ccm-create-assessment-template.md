# Create an assessment template

Create an assessment template to request responses from the assessor or reviewer. You can create this template by using the assessment metric type form in the GRC: Compliance Case Management application.

## Before you begin

Role required: sn\_compliance\_case\_admin

## About this task

You can create a questionnaire template instead of using the default template to assess your compliance cases. After you set up the assessment template, it’s available in the case task section for that case type.

## Procedure

1.  Navigate to **All** &gt; **Compliance Case Management** &gt; **Assessment Templates**.

2.  Select **New**.

3.  On the form, fill in the fields.

    For a description of the field values on the Assessment Metric Type form, see [Assessment Metric Type form](../reference/assessment-metric-type-form.md).

4.  Select **Submit**.


-   **[Assessment Metric Type form](../reference/assessment-metric-type-form.md)**  
Use the Assessment Metric Type form to create an assessment questionnaire template by using the GRC: Compliance Case Management application.
