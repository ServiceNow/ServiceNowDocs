# Create a compliance request in the Compliance Workspace

Raise a compliance request regarding organization's policies, inquiries, and guidelines by using the Compliance Case Management application. By seeking guidance, clarification, or approval, you contribute to maintaining a culture of compliance within the organization.

## Before you begin

Role required: sn\_compliance\_ws.corporate\_compliance\_analyst and sn\_compliance\_ws.corporate\_compliance\_manager

## Procedure

1.  Navigate to **All** &gt; **Compliance Workspace**.

2.  Select the Compliance case management icon \(![Case management icon.](../image/compliance-case-launcher.png)\).

3.  On the Compliance Case Management landing page, select **Create**.

4.  From the options available in the Create list, select **Request**.

5.  On the form, fill in the fields.

    For a description of the field values on the Create a Compliance Case form, see [Compliance request form](../reference/compliance-request-form.md).

6.  Select **Save**.


-   **[Compliance request form](../reference/compliance-request-form.md)**  
Use the Create new compliance request form in the GRC: Compliance Case Management application to raise a compliance request.
