# Configure knowledge graph tags for AI reporting assistant

Map knowledge graph tags to a business domain to improve the accuracy of AI reporting assistant results when querying ServiceNow data. This configuration is optional.

## Before you begin

**Note:** Complete this task only if AI reporting assistant results lack accuracy.

Role required: sn\_grc\_doc\_design.admin

## Procedure

1.  Enter `sn_doc_design_ai_reporting_config.LIST` in the application navigator, and open **AI reporting configurations**.

2.  Select **New**.

3.  Select the business domain in the **Business domain** field.

4.  Select one or more knowledge graph tags in the **Knowledge graph tags** field.

5.  Select the **Active** check box.

6.  Select **Submit**.
