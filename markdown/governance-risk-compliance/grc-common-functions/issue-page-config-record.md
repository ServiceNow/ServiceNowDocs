# Issue Page Configuration module

The Issue Page Configuration module displays the configurations related to the Issues overview landing pages in various workspaces. The data displayed in the Issues overview landing page is configured using the Issue Page Configurations module in the classic user interface. The Issues overview landing page includes the Issues, Issue triages, Tracking status of the issues, and Audit observations pages.

The Issues Overview Landing page is available in the following workspaces:

-   Audit workspace
-   Compliance workspace
-   Privacy workspace
-   Risk workspace

GRC administrators can configure, monitor, and report issues for an active workspace by mapping the states, filter conditions, and tables using the Issue Page Configuration module.

Based on the configurations performed in the Issue Page Configuration module, workspace users can view the data that is assigned to them in the Issues overview landing page in the workspace.

Administrators can perform the following tasks using the Issue Page Configuration module:

-   Create new Issue page configurations records in the classic user interface that are displayed in the Issues overview page in the workspace.
-   Configure the issues related to the tables, tracking, issue triages, and observations for the selected tables and track policy exceptions, evidence, and so on.

The homepage in each workspace displays the issues section that provides the following information:

-   The My issues section displays the Issues and Triage issues sections in a number format. Issues subsection displays the details such as Assigned to, Managing, Remediation assigned, To review, Overdue. Triage issues subsection displays the details such as Assigned to and To review.
-   The My group's issues section displays the Issues and Triage issues sections in a number format. Issues subsection displays the details such as Assigned to, Managing, Unassigned manager, To review, Overdue. Triage issues subsection displays the details such as Assigned to, Unassigned owner, To review, and Unassigned reviewer.
-   The Watching section displays the details such as Triage issues and Remediation tasks in a number format.
