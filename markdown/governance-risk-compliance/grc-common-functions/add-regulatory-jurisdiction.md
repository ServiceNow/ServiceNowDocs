# Add a jurisdiction to an agency

Add a jurisdiction to a regulatory agency in Compliance Workspace to define the geographical scope for regulatory applicability.

## Before you begin

Role required: compliance\_case\_manager

## About this task

As a compliance\_case\_manager, you can associate one or more jurisdictions with a regulatory agency. This ensures that regulatory changes are correctly mapped to applicable regions.

## Procedure

1.  Navigate to **All** &gt; **Compliance Workspace** &gt; **Lists** &gt; **Regulatory information** &gt; **Regulatory agencies**.

2.  Select the agency record you want to update.

3.  In the **Jurisdictions** related list, select **Add**.

4.  Choose one or more jurisdictions from the list \(for example, North America, EU, APAC\).

    For more information on creating jurisdictions, see [Map the taxonomy](../../grc-rcm/task/manage-taxonomy.md).

5.  Select **Save**.
