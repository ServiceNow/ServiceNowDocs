# Link existing documents to a model record

Link documents from your repository to a model record. This feature helps you manage all model-related documents in one place, ensuring version control, audit readiness, and traceability.

## Before you begin

Role required: sn\_model\_risk\_mgmt.executive\_reader, sn\_model\_risk\_mgmt.model\_validator, sn\_model\_risk\_mgmt.model\_risk\_coordinator, sn\_model\_risk\_mgmt.model\_governance

Users on the Xanadu and Yokohama releases must install the Document Management plugin manually. Users upgrading from an earlier release to Zurich must also install it manually. For new installations, the plugin is installed automatically.

## Procedure

1.  Navigate to **All** &gt; **Model Risk Workspace**.

2.  Select the list icon ![](../../grc-compliance-case-mgmt/image/list-icon.png).

3.  From the list, navigate to **Models** &gt; **All**.

4.  Select the model and navigate to the **Documents** tab.

5.  Select **Link Documents**.

6.  Select the document that you want to link.

7.  Select **OK**.


## Result

The document is linked in the model risk record.
