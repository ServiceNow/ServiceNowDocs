# Add key stakeholders to a privacy case

Add key stakeholders to a privacy case to identify the key members who are responsible for the data that is lost during the breach. You can then send the respective breach assessments to the stakeholders based on their responsibilities.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_analyst

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace**.

2.  Select the list ![List icon](../../grc-compliance-case-mgmt/image/list-icon.png) icon.

3.  Select **All Cases**.

4.  Open the case for which you want to add a key stakeholders.

5.  Click the Key stakeholders related list.

6.  Click **Add**.

    1.  Select the stakeholders to add.

    2.  Click **Add**.

        The key stakeholders are added.
