# Add or create an issue for a privacy case

Identify and manage issues related to the impacted areas for the reported privacy case. You can also create issues from the Privacy Case Management landing page.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_analyst

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace**.

2.  Select the list ![List icon](../../grc-compliance-case-mgmt/image/list-icon.png) icon.

3.  Select **All Cases**.

4.  Open the case for which you want to add an issue.

5.  Select the Issues related list and do one of the following.
