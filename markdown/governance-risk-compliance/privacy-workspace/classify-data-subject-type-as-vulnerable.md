# Classify data subject type as vulnerable

Classify a data subject type as vulnerable. When you mark a data subject type as vulnerable, the criticality score is calculated as High.

## Before you begin

Role required: sn\_privacy.manager

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace**.

2.  In the Privacy Workspace, select the List icon \(![](../../grc-cam-workspace/image/ws-list-icon.png)\).

3.  Navigate to **Privacy elements** &gt; **Data subject type**.

4.  Select and open the data subject type you want to classify as vulnerable.

5.  Select **Vulnerable** to mark the data subject type as vulnerable.

6.  Select **Save**.


## Result

The data subject type that you classified as vulnerable contributes to the criticality score being High.
