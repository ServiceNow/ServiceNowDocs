# Create an information object

Create information objects manually to associate the right data subject types with business processes or applications.

## Before you begin

Role required: sn\_privacy.admin

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace**.

2.  In the Privacy Workspace, select the List icon \(![List](../../grc-workspace-vrm/image/ws-list-icon.png)\).

3.  Navigate to **Privacy elements** &gt; **AllInformation objects**.

4.  Select **New**.

5.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the information object category. For example, `Financial data`.|
    |Owned by|User who owns the information object.|
    |Data Classification|Data classification of the information object.|
    |Business Unit|Business unit that uses this information object.|
    |Department|Department that uses this information object.|
    |Description|Description of the types of information objects the category contains.|

6.  Select **Save**.

    The information object is created.


## Result

Once the information object is created, you can see the Processing activities and Information object categories related lists.
