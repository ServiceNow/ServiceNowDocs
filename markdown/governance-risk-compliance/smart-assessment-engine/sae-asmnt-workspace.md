# Accessing templates in the Assessment Workspace

You can see all your assessment template-related information on the Assessment Workspace landing page. You can review the published and unpublished assessment templates and create assessment templates by using the Smart Assessment Engine application.

## Smart assessment workspace

You can view all your assessment template-related information by navigating to **Workspaces** &gt; **Assessment Workspace**. If you have the sn\_smart\_asmt.assessment\_admin or sn\_smart\_asmt.template\_manager roles, you can create the assessment templates or update the unpublished assessment templates in the Assessment Workspace. A template manager's ability to create or update the assessment templates is determined by their category role. Any team member with the Template reader \[sn\_smart\_asmt.template\_reader\] role can view the specific templates for their category role. For more information on the assessment categories, see [Create an assessment template category](../task/sae-asmnt-template-category-create.md). You can also access the Assessment Workspace landing page by selecting the home icon ![Home page icon.](../../grc-workspace-vrm/image/icon-tprm-ws-home.png).

As shown in the following example, you can view all of your published and unpublished assessment templates with their names, categories, and descriptions.

![Assessment Workspace that lists all the assessment templates.](../image/sae-assess-workspace.png)

## Creating assessment templates

You can create a template by selecting **New template** and filling out the Template details form. For more information, see [Create an assessment template](../task/sae-asmnt-template-create.md) and [Using the template designer](sae-template-designer.md).
