# Copy an assessment template

Make a copy of an existing assessment template in the Smart Assessment Engine to help reduce the amount of manual input needed. The copied template includes all questions, sections, instructions, and existing configurations from the original template, which you can then customize to meet new requirements or scenarios.

## Before you begin

Role required: sn\_smart\_asmt.template\_manager and sn\_smart\_asmt.assessment\_admin

## About this task

You can copy an assessment template that is in the Draft or Published state. The user who copies the template is marked as the creator and the duplication time is added as the creation time.

## Procedure

1.  Navigate to **Workspaces** &gt; **Assessment Workspace** to access the Assessment Workspace landing page.

2.  Open the existing assessment template that you want to copy.

3.  Select the copy ![](../image/copy-icon.png) icon.

    -   The user is navigated to the copied template, and the copying process begins, which may take some time.
    -   While the copy is in progress, both the source and copy template are locked for editing.
    -   After the copying is completed, you can update the assessment template details as needed.

## Result

The system generates a copied template in the Draft state. When you open the template, you can see the all questions, sections, instructions, and existing configurations from the original template. You can update it as required.
