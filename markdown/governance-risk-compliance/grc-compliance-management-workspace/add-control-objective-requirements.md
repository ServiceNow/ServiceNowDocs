# Add control objective requirements

Control objective requirements can be manually added or selected from existing requirements when configuring a control objective.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Policy and Compliance** &gt; **Compliance Workspace**.

2.  Select the List icon.

3.  Under Compliance Library, select **Control objectives**.

4.  Select the control objective for which you want to add control objective requirements.

5.  On the control objective page, navigate to **Control objective requirements** tab.

6.  To add existing control objective requirements, do the following:

    1.  Select **Add**.

        The Control objective requirements list appears. These are ones that are not already associated with the control objective.

    2.  Select the control objective requirements from the list, and select **Add**.

7.  To create new control objective requirements, do the following:

    1.  Select **New**.

        The Create new control objective requirements page appears.

    2.  In the Reference field, enter a unique numerical identifier for the control objective requirement \(for example, you can use source or reference of the control objective and a number\).

    3.  In the Description field, enter a description for the control objective requirement.

    4.  Select **Save**.
