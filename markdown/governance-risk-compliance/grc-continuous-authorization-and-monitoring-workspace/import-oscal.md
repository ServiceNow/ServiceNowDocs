# Importing in OSCAL format

CAM supports importing Catalog and System Security Plan \(SSP\) models in OSCAL \(Open Security Controls Assessment Language\) format. This feature streamlines data integration, enhances collaboration, and helps reduce manual effort so your security control information is consistent and up to date.

For more information on the OSCAL import error and control catalog, see the [OSCAL Import \[KB1794095\]](https://hi.service-now.com/kb_view.do?sysparm_article=KB1794095) article in the Now Support Knowledge Base.

-   **[Import OSCAL catalog](../task/import-oscal-cam-ws.md)**  
From the OSCAL Import landing page, import catalog OSCAL data into the CAM Workspace.
-   **[Import OSCAL SSP](../task/import-oscal-ssp-cam-ws.md)**  
From the OSCAL Import landing page, import SSP OSCAL data into the CAM Workspace. This action enables you to seamlessly integrate security control assessments and compliance data, streamlining the process of managing and assessing security controls within your organization.
