# Monitor and manage your NIST security posture

Access the CAM Overview, AO Overview, and SCA Overview dashboards from the CAM Workspace to monitor and analyze data and view CAM reports.

## Before you begin

Role required: sn\_irm\_cont\_auth.admin, sn\_irm\_cont\_auth.authorization\_official, sn\_irm\_cont\_auth.info\_system\_sec\_manager, sn\_irm\_cont\_auth.info\_system\_sec\_officer, sn\_irm\_cont\_auth.sec\_control\_assessor, sn\_irm\_cont\_auth.system\_owner

## About this task

## Procedure

1.  Navigate to **All** &gt; **CAM Workspace** and then select the dashboard icon \(\[Omitted image "feedback-dashboard-icon.jpg"\] Alt text: Dashboard icon.\).

    The dashboard appropriate for your user role opens.

2.  Select the change dashboard list icon \(\[Omitted image "change-dashboard-icon.png"\] Alt text: Change dashboard list icon.\) to select the dashboard to view.

    -   **CAM Overview dashboard**

        The CAM Overview dashboard provides multiple tabs with reports on critical aspects of your CAM security posture.

        Users with the CAM Administrator \[sn\_irm\_cont\_auth.admin\] role can view the dashboard.

        See [CAM Overview dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/continuous-risk-monitoring/cam-overvw-pa-db.md).

    -   **AO Overview dashboard**

        The Authorization Packages tab on the AO Overview dashboard provides central visibility into the state of all packages. The reports on the Baseline Controls tab provide a general overview of the baseline controls in your system.

        The reports on the Baseline Controls tab on the AO Overview dashboard provide a general overview of the baseline controls in your system.

        Users with the Authorization Official \[sn\_irm\_cont\_auth.authorization\_official\] role can view the dashboard.

        See [AO Overview dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/continuous-risk-monitoring/reports-ao-overview.md).

    -   **SCA Overview dashboard**

        The SCA Overview dashboard enables you to view and manage control assessments as they occur.

        Users with the CAM Administrator \(sn\_irm\_cont\_auth.admin\) or Security Control Assessor \(sn\_irm\_cont\_auth.sec\_control\_assessor\) role can view the dashboard.

        See [SCA Overview dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/continuous-risk-monitoring/reports-sca-ov.md).
