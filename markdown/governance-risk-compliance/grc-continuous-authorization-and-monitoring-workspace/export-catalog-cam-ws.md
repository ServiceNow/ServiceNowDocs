# Export OSCAL catalog

From the Control objective list view page in the CAM Workspace, export the catalog in the OSCAL JSON format for the selected control objectives. This action enables you to export your control objectives from CAM.

## Before you begin

Role required: sn\_irm\_cont\_auth.authorization\_official, sn\_irm\_cont\_auth.info\_system\_sec\_manager, sn\_irm\_cont\_auth.info\_system\_sec\_officer, sn\_irm\_cont\_auth.system\_owner, and sn\_irm\_cont\_auth.reader.

## Procedure

1.  Navigate to **Workspaces** &gt; **CAM Workspace**.

2.  In the primary navigation, select the List icon \(![List](../../grc-workspace-vrm/image/ws-list-icon.png)\).

3.  In the Compliance list, select Control objectives.

4.  From the list view, select one or more control objectives records that you want to export.

5.  Select **OSCAL Export**.

    ![S](../image/cam-export-oscal-control-objective.gif)

6.  In the **OSCAL Export** pop-up window, enter the title and the version.

7.  Select **Export**.

    **Important:** Verify that the pop-up blocker is turned off for the URL so the Catalog JSON file downloads to your local repository automatically.
