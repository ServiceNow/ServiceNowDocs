# Create a smart assessment template for risk identification

Create a smart assessment template for risk identification using the Smart Assessment Engine application.

## Before you begin

Role required: sn\_risk.admin

## Procedure

1.  Navigate to **All** &gt; **Assessment Workspace**.

2.  On the Assessment workspace landing page, select **New template** and then fill in the Template details form.

</td></tr><tr><td>

Description

</td><td>

Text that helps others to understand what the template will be used for.

</td></tr><tr><td>

Assessment template categories

</td><td>

Category that the new assessment template should be a member of. You must select `Risk identification category`.

Template purposes enforce data segregation for templates. A purpose controls which users can view a template. Each assessment template is associated with a purpose. To view a template within a specific purpose, you must have a category role associated with that purpose.

See [Create an assessment template category](../../assessment-engine/task/sae-asmnt-template-category-create.md)**Note:** You can create an assessment only from a published assessment template.

</td></tr><tr><td>

Assessment target

</td><td>

The table records selected to be assessed, otherwise known as the assessment scope. Multiple tables can be added as the assessment target making the scope a combination of records from each table. You must select `Entity`.
