# Add a risk event cause to the cause library

A cause library is a centralized library of the possible causes that can lead to a risk event. Adding causes helps to identify the reason for a risk event and prevent future events.

## Before you begin

Role required: sn\_risk.manager and sn\_risk.admin

## About this task

Add a risk event cause to the cause library to help with risk analysis, risk prediction, and risk prevention.

## Procedure

1.  Navigate to **All** &gt; **Risk Events** &gt; **Administration** &gt; **Causes**.

2.  Click **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the cause. For example,`Inadequate IT security`.|
    |Parent|Parent of the cause. The parent refers to the group under which the cause is categorized. For example, for Inadequate IT security, the parent cause can be Process.|
    |Description|Brief description of the cause.|

4.  Click **Submit**.


## Result

The risk event cause is added to the library.
