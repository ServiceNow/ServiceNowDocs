# Assign project risks to stakeholders for assessment

Assign the project risks to the relevant stakeholder for assessment. You must manually assign the risks to the stakeholders.

## Before you begin

Role required: it\_project\_manager

## About this task

If the Project Risk integration Configuration form has **Stakeholders** selected as an assessor, then the project manager must manually assign the assessment to one of the stakeholders of the project.

## Procedure

1.  Navigate to **All** &gt; **Project** &gt; **Projects** &gt; **All**.

2.  Select the project for risk assessment.

    Ensure that risks for this project are present. See [Add risks for a project](https://www.servicenow.com/docs/access?context=add-risks-for-project&version=zurich&pubname=zurich-it-business-management&ft:locale=en-US).

3.  Click the Risks related list.

4.  Select the risk you want to assess.

5.  Click **Assess Risk**.

    The values are shown based on the integration configuration.

6.  In the **Approver** field, select an assessor for the project risk.

7.  Click **Submit**.


## Result

The risk moves to the risk assessor's workflow.
