# Create an action item in the risk response task

Create multiple strategies with various action items for each risk response task. Action items help you to understand and focus on specific steps needed to manage risks.

## Before you begin

Role required: sn\_risk\_advanced.ara\_assessor

## About this task

Action items are specific, granular tasks defined within a risk response task to address and manage risks effectively. They’re elevated from the broader risk response task and can be assigned to different users to ensure that the required response plan is executed.

**Note:** You can create action items for:

-   Risk responses except Risk Acceptance tasks
-   Risk responses that are in Draft or Work in Progress

## Procedure

1.  Navigate to **All** &gt; **Risk** &gt; **Risk Workspace**.

2.  Select the task ![Task icon.](../image/task-icon.png) icon.

3.  Open a risk response task for which you want to add an Action item.

4.  Select the **Action items** related list.

5.  Select **New**.

6.  On the form, fill in the fields.

    For a description of the field values on the Create New Action Item form, see [Create New Action Item form](../reference/create-new-action-item-form.md).

7.  Select **Save**.

8.  To send the action item to the assignee, select **Send to Assignee**.

    The action item moves to the Assigned state.


-   **[Create New Action Item form](../reference/create-new-action-item-form.md)**  
Use the Create New Action Item form to define the plan of action, assign responsibilities, set priorities, and establish milestones for the action item.
