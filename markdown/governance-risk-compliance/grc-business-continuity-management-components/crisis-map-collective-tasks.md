# Structured workflows for Crisis map

You can use the Crisis map interface in BCM Configurable Workspace to receive alerts on possible threats for the business operations of your organization. You can then manage the alerts by initiating response workflows.

Typically, the BCM program manager performs the following tasks to manage the alerts from the Crisis map interface:

-   Manage the alerts in Crisis map. For more information, see [Manage alerts from the map interface](../task/manage-alerts-in-crisis-map-interface-uib-ws.md).
-   Set controls in Crisis map to customize the alerts. For more information, see [Set controls to customize the alerts](../task/set-controls-and-customize-alerts-on-map-uib-ws.md).
-   View resource layer clustering in Crisis map. For more information, see [Select resource layer clustering](../task/view-resource-layer-clustering-uib-ws.md).
-   Monitor the assets within the impacted area in Crisis map. For more information, see [Monitor assets within the impacted areas](../task/view-assets-at-risk-within-impacted-area.md).
-   Initiate response workflows for crisis management. For more information, see [Initiate the response actions](../task/notifystakeholders-initiate-Crisis-Management-workflows-uib-ws.md).

-   **[Adopting UIB page for improved performance](crisis-map-migration.md)**  
Beginning with the Yokohama release, the Crisis map application is now built with a UI Builder \(UIB\) page. Previously, the single-use component limited the ability to customize the user interface \(UI\), add filters, or modify the design in the Crisis map. By adopting the UIB page, you can gain full control over the implementation, enabling you to extend support for your specific use cases.
-   **[Manage alerts from the map interface](../task/manage-alerts-in-crisis-map-interface-uib-ws.md)**  
Manage alerts from the Crisis map interface to identify threats and their geo-locations relative to your assets. You can open alerts for details, dismiss them when resolved, adjust the impacted area boundaries, or undo recent changes.
-   **[Set controls to customize the alerts](../task/set-controls-and-customize-alerts-on-map-uib-ws.md)**  
Set controls in the Crisis map interface in BCM UIB Workspace. You can then customize the alerts for assets, locations, and so on.
-   **[Select resource layer clustering](../task/view-resource-layer-clustering-uib-ws.md)**  
Select resource layer clustering in the Crisis map within the BCM Configurable Workspace to view assets or resources on the map.
-   **[Monitor assets within the impacted areas](../task/view-assets-at-risk-within-impacted-area.md)**  
Monitor at-risk assets in the Crisis map within the BCM Configurable Workspace when alert threats appear near your business locations. This functionality enables you to protect resources in impacted areas and help prevent significant losses to your organization.
-   **[Initiate the response actions](../task/notifystakeholders-initiate-Crisis-Management-workflows-uib-ws.md)**  
Initiate the response workflows for your recovery teams in BCM UIB Workspace. You can then notify the stakeholders and help them respond to the threat and take the necessary actions.
