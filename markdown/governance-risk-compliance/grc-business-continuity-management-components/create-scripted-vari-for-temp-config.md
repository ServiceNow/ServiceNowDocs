# Define the scripted variables

Define data for the scripted variables using the Template Configurations module for creating a report template in the BCM application. You can configure the data in either text or HTML format.

## Before you begin

Role required: sn\_grc\_doc\_design.admin and sn\_bcm.admin

## Procedure

1.  Navigate to **All** &gt; **Business Continuity** &gt; **General Administration** &gt; **Template Configurations**.

2.  Open the template for which you want to create scripted variables.

    The Scripted variables related list is shown.

    ![Scripted variables related list.](../image/temp-config-scri-var-rel-list.png)

3.  In the Scripted variables related list, select **New** to create a scripted variable.

    The Scripted variable new record is shown.

    ![Scripted variable new record.](../image/temp-config-scri-var-rel-list-new-record.png)

4.  On the form, add the name, type, and the custom script for the scripted variable.

5.  Select **Submit**.

    The Scripted variables for the template, such as Asset dependencies and Attachments, are shown in the example.

    ![Scripted variables related list.](../image/scripted-varia-rel-list-template-config.png)

6.  To edit an existing scripted variable, select it from the related list, make the changes, and select **Update**.
