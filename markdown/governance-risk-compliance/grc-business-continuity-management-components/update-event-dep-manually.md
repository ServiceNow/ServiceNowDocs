# Update the event dependencies

Update the event dependencies manually by selecting the **Update dependencies** UI action. You can update the dependencies in an active event.

## Before you begin

Role required: sn\_bcm.program\_manager, sn\_bcm.planner, sn\_bcm.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Business Continuity Workspace**.

2.  Open an event in the **Pending** state.

3.  To update the dependencies manually, select **Update dependencies**.

    A UI message is displayed on the screen: `Dependencies are being updated.`

4.  To save the changes to the event, select **Save**.

    The plan owners and BCM lead can update the event dependencies by saving the changes to the event.
