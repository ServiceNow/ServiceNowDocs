# Recovery Tier record form

Use the Recovery Tier record form to configure a recovery tier with a set of business applications by using BCM UIB Workspace.

## Recovery Tier record form

For description of the field values, see the table.

|Field|Description|
|-----|-----------|
|Name|Name for the recovery tier level.|
|Recovery time objectives|Metric that calculates how quickly the business should recover an element after a disaster.|
