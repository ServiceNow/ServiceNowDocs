# Notification Template form

Use the Notification Template form to create a template with pre-defined information for sending out the notifications.

## Notification Template form

|Field|Description|
|-----|-----------|
|Name|Name of the notification template.|
|Description|Short note about the purpose of the template.|
|Delivery channels|Communication channels that are used for sending the notification.|
|Subject|Title of the notification.|
|Message body|Introductory text message for the notification.|
|Foot note|Closing message or a foot note for the notification.|
|SMS text|Text message sent through SMS.|
