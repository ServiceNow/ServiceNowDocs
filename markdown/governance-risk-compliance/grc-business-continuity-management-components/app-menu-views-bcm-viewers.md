# Application menu options for BCM users

The table lists the application menu options that are available for the main users of the business continuity management application to view and navigate.

## BCM application menu options

For detailed information on user roles, see [Roles installed](installed-with-bcm.md#section_wly_yyh_wdb) table.

BCM Admin

</td><td>

As a BCM administrator, you can access all menu options from the Business Continuity menu:-   Business Continuity Workspace
-   Business Impact Analysis
-   Planning
-   Crisis Management
-   BIA Configuration
-   Plan Configuration
-   General Administration

</td></tr><tr><td>

</td><td>

BCM program manager

</td><td>

As a BCM program manager you can view and access the menu options:-   Business Continuity Workspace
-   To create an impact analysis: **Business Continuity** &gt; **Business Impact Analysis** &gt; **Impact Analysis**
-   To create a plan: **Business Continuity** &gt; **Planning** &gt; **Plans**
-   To create an event: **Business Continuity** &gt; **Crisis Management** &gt; **Events**

</td></tr><tr><td>

![BCM planner](../image/BCMPlanner.png)

-   To create an impact analysis: **Business Continuity** &gt; **Business Impact Analysis** &gt; **Impact Analysis**
-   To create a plan: **Business Continuity** &gt; **Planning** &gt; **Plans**
-   To create an event: **Business Continuity** &gt; **Crisis Management** &gt; **Events**
