# Contact Import Rules form

Use the Contact Import Rules form to create a contact import rule that you can apply on the User table.

## Contact Import Rules form

|Field|Description|
|-----|-----------|
|Name|Name of the import rule.|
|Table|User table \[sys\_user\] for the rule.|
|Filter condition|Condition applied on the User table to filter the records.|
|Record type|Default category of the record type.|
|User mapping|Unique user reference from the Users table.|
|Contact ID mapping|Unique ID of each contact that is used to sync with Everbridge.|
