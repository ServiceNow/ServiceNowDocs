# Grid configurations and categories

You can set up the grid configuration to render the BIA dependency assessment grid with configured columns. You can define the grid category such as Dependency Assessment for which the grid configuration is applicable.

The following example shows the grid configuration in the Business Continuity Management application. ![Grid configuration.](../image/new-grid-configurations.png)

**Note:** Grid configuration is used in BCM classic Workspace only.

For more information on how to configure the grid in the Business Continuity Management application, see [Set up grid configuration for BIA dependency assessment](../task/setup-bcm-grid-config.md).

The following example shows the configuration of a grid category in the Business Continuity Management application. ![Configuration of a grid category.](../image/new-grid-category.png)

**Note:** Grid categories are used in BCM classic Workspace only.
