# IP Address form

The IP Address form represents the active CMDB record for an address, created either from an allocated IP slot or at the subnetwork level. The fields on this form are listed below.

|Field|Description|
|-----|-----------|
|IP Address|The address value. For records created using the From allocated IPs method, this is the host address \(for example, 10.0.0.1\). For records created using the At subnet level method, this is the subnetwork’s full CIDR \(for example, 10.0.0.0/26\).|
|IP version|Either IPv4 or IPv6.|
|Owned By Configuration Item|Optional reference to the configuration item that owns this address.|
|Netmask|Optional.|
