# Create a country code

Create, review, update, or delete a country code by using the Telecommunications Network Inventory application. A country can have multiple phone formats and phone validations.

## Before you begin

Role required: sn\_ni\_core.inventory\_admin

## About this task

Create a country code and add conditions to it so that you can manage, review, or update it.

## Procedure

1.  Navigate to **Workspaces** &gt; **Network Inventory Workspace**.

2.  Select the list icon \(![List icon.](../image/ni-workspace-list-icon.png)\), and then go to **Administration** &gt; **Country code**.

3.  **Note:** The list view of the country code includes almost all country codes.

    Select **New**

4.  On the **Details** tab, on the form, fill in the fields.

    To learn more about the fields, see [configure a territory phone display rule](https://servicenow.com/docs/bundle/utah-platform-administration/page/administer/field-administration/task/t_ConfigATerritoryPhoneDisplRule.html).

5.  Add the attachments, such as the graphics or documents, by selecting the attachment icon \(![Attachment icon.](../image/attachments-icon.png)\) in the right panel.

6.  Select **Save**

    Related tabs appear next to the **Details** tab. To learn more, see [configure a territory phone display rule](https://servicenow.com/docs/bundle/utah-platform-administration/page/administer/field-administration/task/t_ConfigATerritoryPhoneDisplRule.html).

7.  If you want to delete a country code, navigate to that country code, select the options icon \(![Options icon.](../image/options-icon.png)\), select **Delete**, and select **OK** when you see the confirmation window.


## What to do next

Create an area code. For information, see [Create an area code](create_an_area_code.md).
