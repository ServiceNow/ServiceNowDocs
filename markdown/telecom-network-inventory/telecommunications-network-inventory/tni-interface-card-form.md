# Card form

The Card form enables you create, review, and modify the network card details.

|Field|Description|
|-----|-----------|
|Name|Name of this network interface card. The ServiceNow AI Platform uses this name to identify it in your network inventory.|
|Ports|Total number of ports on this network interface card.|
|Ports in use|Number of ports in use on this network interface card.|
|Slots|Total number of slots on this network interface card.|
