# Attribute pack in CI records

Use an attribute pack to capture the attributes that you define for a configuration item \(CI\) record in the Telecommunications Network Inventory application.

## Before you begin

Role required: sn\_ni\_core.inventory\_agent

## Procedure

1.  Navigate to **Workspaces** &gt; **Network Inventory Workspace**.

2.  Select the list icon \(![List icon.](../image/ni-workspace-list-icon.png)\), and then open a new inventory class.

3.  Select **New**.

4.  Select **Add Packs**.

5.  Select **Submit**.


## Result

The attribute pack tables are added on the **Packs** tab.

## What to do next

You can update the fields in the pack table that you’ve added. To learn more, see [Update a pack for a CI record](review-update-pack-table.md).
