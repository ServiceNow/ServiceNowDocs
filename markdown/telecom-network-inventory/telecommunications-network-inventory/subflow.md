# Telecommunications Network Inventory subflows

You can use Workflow Studio subflows in the Telecommunications Network Inventory application to automate the network inventory's Design and Assign process.

-   **[Logical Connection Creation subflow](logical-connection-creation-subflow.md)**  
The Logical Connection enables you to create a logical connection record in the Telecommunications Network Inventory application based on the input that you receive when you instantiate an inventory.
-   **[Physical Connection Creation subflow](physical-connection-creation-subflow.md)**  
The Physical Connection Creation subflow enables you to create a physical connection record in the Telecommunications Network Inventory application based on the input that you receive when you instantiate an inventory.
