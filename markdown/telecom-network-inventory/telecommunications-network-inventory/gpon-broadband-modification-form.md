# GPON broadband modification form

The GPON Broadband form captures the service and network configuration details required to modify a GPON broadband services.

|Fields|Description|
|------|-----------|
|Model|Change model that is associated with the Telecommunications Network Inventory change request.|
|Customer Site|Network site.|
|New ONT equipment template|Equipment template.|
|Splitter|Equipment.|
|Splitter port|Port interface.|
|Existing VLAN|Your existing VLAN connection.|
