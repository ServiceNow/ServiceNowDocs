# Inventory models additional tabs

The additional tabs of the inventory models appear after a model is created. These tabs display all the related information of a model. You can always view, update, save, and delete the information as required.

<table id="table_dbc_1kj_25b"><thead><tr><th>

Name

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Assets

</td><td>

List of assets related to the model.

</td></tr><tr><td>

Configuration items

</td><td>

List of all the Configuration Items \(CIs\) created for the model.

</td></tr><tr><td>

Model components

</td><td>

List of components that constitutes the model.

</td></tr><tr><td>

Model life cycles

</td><td>

List of life cycles of the models.

</td></tr><tr><td>

Vendor catalog items

</td><td>

List of vendor catalog items associated to the product model.

</td></tr><tr><td>

Hardware model lifecycles

</td><td>

List of life cycle phases and life cycle type of a hardware model.

</td></tr><tr><td>

Network model relationships

</td><td>

List of all network model relationships of the model.**Note:** This field is not applicable to the connection models.

</td></tr><tr><td>

Model images

</td><td>

Images of the model. The images uploaded here are shown in the Configuration Items created for this model.**Note:** This field is applicable only to **Equipment models**.

</td></tr><tr><td>

Bandwidth compatibilities

</td><td>

List of bandwidths that are compatible to the respective connection. Further, you can create and update the associated connection model and bandwidth group.**Note:** Bandwidth compatibilities field is applicable only to the connection models.

</td></tr></tbody>
