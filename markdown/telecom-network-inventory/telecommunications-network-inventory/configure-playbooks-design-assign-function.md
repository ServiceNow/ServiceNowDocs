# Design and Assign function for your network services

Create and configure a Design and Assign function in the Telecommunications Network Inventory application. You can use it to design your network service and assign the network resources.

The Design and Assign function is created and configured in the Workflow Studio using the playbook experience. Some of the configuration is performed in ServiceNow AI Platform. These settings include creating record producers, Change model, and request definitions. You can fully automate your Design and Assign function by linking flows, actions, and subflows together in a playbook.

Follow the steps to create and configure a Design and Assign function.

1.  [Create a Change Model](../task/create-change-model-design-assign-playbook.md).
2.  [Create record producers.](../task/create-record-producer-design-assign-playbook.md)
3.  [Add input variables to the record producer](../task/create-variables-record-producer-design-assign-playbook.md).
4.  [Add variable set to the record producer](../task/add-variable-set-record-producer-design-assign-playbook.md).
5.  [Create subflows](../task/create-subflow-design-assign-playbook.md).
6.  [Create request definitions](../task/create-request-definition-design-assign-function.md).
7.  [Create a playbook for design and assign function](../task/create-playbook-design-assign-function.md).
8.  [Update the system property to configure the Design and Assign home page](../task/update-system-property-configure-design-assign-home-page.md).

On completing these activities, a UI action displays on the Design and Assign home page to launch the function. Select the UI action to launch your new Design and Assign function.
