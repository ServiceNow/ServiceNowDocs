# Configure the Now Mobile Agent application

Configure the Now Mobile Agent application and customize it for Industrial Connected Workforce \(ICW\) users.

## Before you begin

Role required: sn\_icw.application\_admin

## Procedure

1.  Navigate to **All** &gt; **System Applications** &gt; **Studio**.

2.  In the Select Application screen, select **ICW Mobile**.

3.  Customize the application to display the desired widgets and fields on the mobile application instance.

4.  Configure the Now Mobile Agent application.
