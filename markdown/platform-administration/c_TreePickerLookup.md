# Tree picker lookup

The reference lookup can be rendered in the tree picker format by modifying the dictionary and adding the attribute **tree\_picker**.

You cannot customize the label names used in the tree picker. The label names are taken from the values in the table.

**Note:** The tree display hierarchy does not appear in reference fields when using mobile applications.
