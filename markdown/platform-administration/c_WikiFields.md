# Wiki field type

Wikitext fields use the basic wiki markup language and support links to external URLs and to sources within the system.

Some fields in the base system support Wikitext and administrators can add Wikitext fields on any form in the system.

-   **[Supported wiki tags](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/platform-administration/r_SupportedWikiTags.md)**  
Basic tags are supported for Wikitext fields in the system.
-   **[Create a Wikitext field](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/platform-administration/t_CreateAWikitextField.md)**  
You can create a Wikitext field on any form.
-   **[Extend the functionality of a Wikitext field](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/platform-administration/t_ExtendAWikitextFieldFunction.md)**  
Configure the Wikitext field to link to other sources within the system.
