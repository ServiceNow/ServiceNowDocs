# View a schedule item

You can view scheduled items, which are individual instances of a scheduled job, from the system scheduler.

## Before you begin

Role required: system\_scheduler\_admin

## About this task

## Procedure

1.  Navigate to **All** &gt; **System Scheduler** &gt; **Today's Scheduled Jobs**.

    **Note:** It is inadvisable to modify the schedule items themselves. It is best to modify the scheduled jobs that contain them.

    Each schedule item appears on the table listing.
