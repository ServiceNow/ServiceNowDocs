# Installing applications, plugins, and products

Application Manager enables you to install any applications, plugins, or products that are included in your license or that have been procured from the ServiceNow Store.

## Overview of using the Application Manager to install

You can view all applications, plugins, and products available for your instance by navigating to the Application Manager and selecting the **Available for you** tab. From there, you can install new applications, products, and plugins to your instance.

**Note:**

Some applications and plugins can't be uninstalled after they've been installed.
