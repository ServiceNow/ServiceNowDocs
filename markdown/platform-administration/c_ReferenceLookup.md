# Reference lookup

On forms, the reference lookup icon \(\[Omitted image "SearchIconHeisenberg.png"\] Alt text: Core UI reference lookup icon\) appears by editable reference fields. Clicking the reference lookup icon displays a list of records on the referenced table.

When List v3 is enabled, the list appears in a popover. Otherwise, the list appears in a pop-up window.

\[Omitted image "ReferenceLookupListUI16.png"\] Alt text: The reference lookup list opens in a popover when List v3 is enabled

\[Omitted image "ReferenceLookupListUI15.png"\] Alt text: The reference lookup list opens in a pop-up window

-   **[Tree picker lookup](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/xanadu/markdown/xanadu/platform-administration/c_TreePickerLookup.md)**  
The reference lookup can be rendered in the tree picker format by modifying the dictionary and adding the attribute **tree\_picker**.
