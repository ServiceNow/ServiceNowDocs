# Estimating document volume for source systems

Source system tools allow you to estimate the number of documents available for retrieval by external content connectors. By estimating the available document count for a source system, you can determine whether you need to apply crawl scope restrictions when configuring an external content connector for that source system.

-   **[Estimate document volume for Atlassian Confluence Cloud](../task/estimate-doc-volume-acc.md)**  
Estimate the number of documents included in your Atlassian Confluence Cloud source system. Use this information to determine crawl scope settings needed for your Atlassian Confluence Cloud external content connector.
-   **[Estimate document volume for Microsoft SharePoint Online](../task/estimate-doc-volume-mspo.md)**  
Estimate the total number of documents included in your Microsoft SharePoint Online source system and the document counts for individual sites. Use this information to determine crawl scope settings needed for your Microsoft SharePoint Online external content connector.
