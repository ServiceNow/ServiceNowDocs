# Disable synonyms for a table

By default, Zing uses all active synonym dictionaries when a user performs a text search. Search administrators can disable synonym dictionary usage on a per-table basis.

## Before you begin

Role required: ts\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Text Indexes**.

2.  Select the table for which you want to change search behavior.

    The system displays the Text index record for the table.

3.  Select **Disable synonym**, so the check appears.

4.  Select **Update**.
