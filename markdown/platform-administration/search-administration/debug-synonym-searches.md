# Debug synonym searches

You can view debug messages to determine if synonym searches are working as expected.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Diagnostics** &gt; **Session Debug** &gt; **Debug Text Search**.

2.  Perform a search using a term in a synonym dictionary.

3.  View the session debug output.

    **Note:** You can only view the debug output on a table view \(\_list.do\). You can't view it on a UI page itself.
